# 尺规作图证明

- Conversation ID: `692f0adb-a55c-832d-b27d-b3881a2a9785`

## Question

尺规作图问题。有\triangle XYZ，XY上有一点A且XA=\frac{1}{3}XY，YZ上有一点B且YB=\frac{1}{3}YZ，ZX上有一点C且ZC=\frac{1}{3}ZX。
现在\triangle XYZ消失了，只剩下了A,B,C三点。求用尺规作图恢复\triangle XYZ。

我给出一种做法：
一。以A为圆心AB长为半径画圆Circle_A，以B为圆心BC长为半径画圆Circle_B，以C为圆心CA长为半径画圆Circle_C。
二。连接AB,BC,CA。AB所在的直线和Circle_A交于点D，同理BC所在的直线和Circle_B交于点E，CA所在的直线和Circle_C交于点F。
三。连接DC，AE，BF。这三条线组成的三角形即所求。

请证明这种做法是正确的。

## Answer

好题，很漂亮的做法。下面我用向量（或坐标）方法把你给的尺规作图严格证明一遍——思路很直观：构造出的点 $D,E,F$ 分别是 $B,C,A$ 关于 $A,B,C$ 的对称点（在各自直线上），然后代数地证明直线 $DC,\,EA,\,FB$ 恰好是原三角形的三条边 $ZX,\,XY,\,YZ$。

把三角形顶点记为向量（或坐标） $X,Y,Z$。由题意

```math
A\in XY,\quad XA=\tfrac13\,XY\quad\Longrightarrow\quad
A=X+\tfrac13(Y-X)=\tfrac23X+\tfrac13Y,
```

```math
B=\tfrac23Y+\tfrac13Z,\qquad C=\tfrac23Z+\tfrac13X.
```

按照你的作法，点 $D$ 是以 $A$ 为圆心、半径 $AB$ 在直线 $AB$ 上与 $B$ 对称的交点（即在直线 $AB$ 上除 $B$ 外满足 $AD=AB$ 的点）。代数上这等价于

```math
D=2A-B.
```

同理

```math
E=2B-C,\qquad F=2C-A.
```

下面验证这些构造的直线与原来三角形的对应边重合。

计算 $D$ 与 $C$ 的差：

```math
D-C=(2A-B)-C
=2\Big(\tfrac23X+\tfrac13Y\Big)-\Big(\tfrac23Y+\tfrac13Z\Big)-\Big(\tfrac23Z+\tfrac13X\Big).
```

化简得

```math
D-C = X-Z.
```

因此向量 $D-C$ 与 $X-Z$ 同向，说明直线 $DC$ 与直线 $XZ$ 共线（即同一条直线）。于是直线 $DC$ 就是原三角形的边 $ZX$（或它的延长线）。

同样计算

```math
E-A=(2B-C)-A
= (2(\tfrac23Y+\tfrac13Z)-(\tfrac23Z+\tfrac13X))-(\tfrac23X+\tfrac13Y)
= Y-X,
```

故直线 $AE$ 与直线 $YX$ 共线，即 $AE$ 是原边 $XY$。

再计算

```math
F-B=(2C-A)-B = Z-Y,
```

故直线 $BF$ 与 $ZY$ 共线，即 $BF$ 是原边 $YZ$。

因此你构造出的三条直线 $DC,\,AE,\,BF$ 恰好分别与原三角形的三边 $ZX,\,XY,\,YZ$ 重合。它们的三条两两交点就是原三角形的顶点 $X,Y,Z$。（换句话说，三条直线所围成的三角形正是原来的 $\triangle XYZ$。）

补充一句直观的几何解释（不用繁复代数的直观）：步骤一里每个圆的另一交点是对应已知点关于中点的对称点（例如 $D$ 与 $B$ 关于 $A$ 对称），所以你得到了三个“对称点” $D,E,F$；把这些对称点与原来的分点 $A,B,C$ 连起来，等价于把原来的三边平移/伸缩到了这些线，最终恢复出原来的三条边。代数证明给出了严格细节。

结论：你的构法正确，并且上面的向量推导给出完整证明。👏
