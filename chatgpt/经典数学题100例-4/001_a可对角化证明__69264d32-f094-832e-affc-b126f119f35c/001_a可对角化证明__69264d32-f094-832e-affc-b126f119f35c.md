# A可对角化证明

- Conversation ID: `69264d32-f094-832e-affc-b126f119f35c`

## Question

设A \in \mathbb{C}^{3 \times 3}，tr(A)=3，tr(A^2)=5，tr(A^3)=9。证明：A可对角化。

## Answer

令 $A$ 的特征值为 $a,b,c$（按重数计），由迹的定义有

```math
a+b+c=\mathrm{tr}A=3,
```

并且

```math
a^2+b^2+c^2=\mathrm{tr}(A^2)=5,
```

```math
a^3+b^3+c^3=\mathrm{tr}(A^3)=9.
```

由平方和与和的关系

```math
a^2+b^2+c^2=(a+b+c)^2-2(ab+bc+ca)
```

代入已知值得

```math
5=3^2-2(ab+bc+ca)=9-2(ab+bc+ca),
```

从而

```math
ab+bc+ca=2.
```

再用立方和的恒等式

```math
a^3+b^3+c^3=(a+b+c)^3-3(a+b+c)(ab+bc+ca)+3abc,
```

代入数值得到

```math
9=3^3-3\cdot3\cdot2+3abc=27-18+3abc=9+3abc,
```

于是

```math
3abc=0\Longrightarrow abc=0.
```

因此 $abc=0$ 意味着三个特征值中有一个是 $0$。记另外两个为 $x,y$，则

```math
x+y=3,\qquad xy=2,
```

它们是二次方程 $t^2-3t+2=0$ 的根，所以

```math
\{a,b,c\}=\{0,1,2\},
```

三者互不相等。

特征值互不相等则其最小多项式没有重根（是 $\lambda(\lambda-1)(\lambda-2)$），因此矩阵 $A$ 在 $\mathbb C$ 上可对角化。∎
