# 尺规作图证明

- Conversation ID: `69307435-0b1c-8333-8770-86b24436e2a0`

## Question

尺规作图问题。有两个圆⊙A（较大）和⊙B（较小），它们外切于C点。请构造一个圆，使其与⊙A外切，同时与⊙A,⊙B的两条外公切线也相切。

我这里给出一种作法。
为了方便描述，我们假设把两个圆水平放置。⊙A在左边，⊙B在右边。
1。以B为圆心，BA长为半径作圆⊙B'交⊙A于D,E两点，D在上方，E在下方。
2。连接EC交⊙B'于F。连接DF交BA延长线于G。
3。以G为圆心，作⊙A的外切圆，⊙G即所求。

请证明此作法的正确性。

## Answer

好的 — 我把你的作法用坐标/代数方式严格证明一下，步骤清楚且代数化地验证“以 $G$ 为圆心作的圆确实满足题目条件”。

设定坐标与记号（横放，两圆外切于点 $C$）

设大圆 $\odot A$ 的圆心为 $A=(0,0)$，半径为 $r_A$。

小圆 $\odot B$ 的圆心为 $B=(d,0)$，半径为 $r_B$。

因为两圆外切，$d=AB=r_A+r_B$，切点为 $C=(r_A,0)$。
（如你所述，取 $D$ 为上交点，$E$ 为下交点。）

#### 1. 计算 $D,E$ 的坐标

交点满足

```math
\begin{cases}
x^2+y^2=r_A^2 &(\text{圆 } \odot A)\\
(x-d)^2+y^2=d^2 &(\text{圆 } \odot B'，\text{半径 }d)
\end{cases}
```

两式相减得

```math
-2dx = -r_A^2 \quad\Rightarrow\quad x=\dfrac{r_A^2}{2d}.
```

于是

```math
y^2 = r_A^2 - x^2,\qquad D=\Big(x,\;+\sqrt{r_A^2-x^2}\Big),\quad E=\Big(x,\;-\sqrt{r_A^2-x^2}\Big).
```

为简洁记号，记 $x=\dfrac{r_A^2}{2d}$ 和令 $y=\sqrt{r_A^2-x^2}$。

#### 2. 求点 $F$（直线 $EC$ 与圆 $B'$ 的另一交点）

用参数 $t$ 表示过 $C$ 指向 $E$ 的线：

```math
P(t)=C+t(E-C).
```

把 $P(t)$ 代入圆 $B'$：方程的两根 $t$ 中一个是 $1$（即点 $E$），另一个解为

```math
t_2 = -\frac{d}{r_A}.
```

于是得到了 $F=P(t_2)$。

（代数解出 $t_2$ 的过程如上；可直接代代算得 $F$ 的坐标。）

#### 3. 求点 $G$（直线 $DF$ 与 $BA$ 延长线，即 $x$-轴 的交点）

用参数 $s$ 表示 $D+s(F-D)$，令其 $y$-坐标为 $0$（即与 $x$-轴相交），解得

```math
s = -\frac{r_A}{r_B}.
```

代入得

```math
G_x = x + s(F_x-x) = -\frac{r_A d}{r_B},\qquad G=(G_x,0).
```

所以

```math
AG = |G_x-0|=\frac{r_A d}{r_B}.
```

因此以 $G$ 为心与 $\odot A$ 外切的圆半径为

```math
R_G = AG - r_A = \frac{r_A d}{r_B} - r_A = \frac{r_A^2}{r_B}.
```

（这是一个非常简洁、重要的代数表达：圆心 $G$ 的半径为 $\dfrac{r_A^2}{r_B}$。）

#### 4. 验证该圆同时与 $\odot A,\odot B$ 的两条外公切线相切

写出两圆的上（或下）外公切线的显式形式。设上外公切线为 $y=mx+b$（取上侧，则 $b>0$）。外公切线须满足到两个圆心的距离分别为半径：

```math
\frac{|b|}{\sqrt{1+m^2}}=r_A,\qquad \frac{|md+b|}{\sqrt{1+m^2}}=r_B.
```

对上侧两式取同号并相减，解得

```math
m^2=\frac{(r_A-r_B)^2}{4r_A r_B},\quad\text{即}\quad m=\pm\frac{r_A-r_B}{2\sqrt{r_Ar_B}}.
```

取上外切线对应的符号（例如取负号），再计算

```math
\sqrt{1+m^2}=\frac{d}{2\sqrt{r_A r_B}},\qquad
b=r_A\sqrt{1+m^2}=\frac{r_A d}{2\sqrt{r_A r_B}}.
```

现在计算点 $G=(G_x,0)$ 到这条直线的距离：

```math
\mathrm{dist}(G,\;y=mx+b)
=\frac{|mG_x+b|}{\sqrt{1+m^2}}.
```

把 $G_x=-\dfrac{r_A d}{r_B}$、上面的 $m,b,\sqrt{1+m^2}$ 代入，整理得到

```math
mG_x+b
= \frac{r_A^2 d}{2r_B\sqrt{r_A r_B}},\qquad
\sqrt{1+m^2}=\frac{d}{2\sqrt{r_A r_B}}.
```

因此

```math
\frac{|mG_x+b|}{\sqrt{1+m^2}}
= \frac{r_A^2 d}{2r_B\sqrt{r_A r_B}}\Big/\frac{d}{2\sqrt{r_A r_B}}
= \frac{r_A^2}{r_B} = R_G.
```

这说明：从 $G$ 到（任一）外公切线的距离恰好等于我们求得的半径 $R_G$。即以 $G$ 为心、半径 $R_G=\dfrac{r_A^2}{r_B}$ 的圆与两条外公切线都相切。

另外显然这个圆与 $\odot A$ 外切（因为我们用了 $AG-r_A$ 作为半径）。

#### 结论

按你的构造取到的点 $G$（直线 $DF$ 与 $BA$ 延长线的交点）满足

以 $G$ 为心，半径 $R_G=AG-r_A=\dfrac{r_A^2}{r_B}$ 的圆与 $\odot A$ 外切；

同时到 $\odot A,\odot B$ 的两条外公切线的距离均等于 $R_G$，因此与这两条外公切线也都相切。

所以你的作法是正确的。上面给出的代数推导可以看作对构造的严格证明；若你希望我把证明改写成纯几何（利用相似三角形与中心齐次变换/相似中心的论证）形式，我也可以把代数步骤翻译成对应的几何论证。要哪种风格？
