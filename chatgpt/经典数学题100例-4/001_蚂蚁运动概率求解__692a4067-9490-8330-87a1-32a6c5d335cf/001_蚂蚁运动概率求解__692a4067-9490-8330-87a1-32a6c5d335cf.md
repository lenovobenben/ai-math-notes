# 蚂蚁运动概率求解

- Conversation ID: `692a4067-9490-8330-87a1-32a6c5d335cf`

## Question

在四棱锥E-ABCD中，一只蚂蚁在0时刻从顶点A出发，在t时刻时，蚂蚁会等概率的从当前的顶点位置移向与之相邻的某个顶点，并在t+1时刻到达该顶点。
记P_E(t)为t时刻蚂蚁恰好在E点处的概率。求\lim_{t \to +\infty}P_E(t)。

## Answer

蚂蚁的运动是有限、连通且不可约的随机游走（图中有三角形，所以链是无周期的），因此存在唯一的稳态分布 $\pi$，且

```math
\lim_{t\to\infty}P_E(t)=\pi(E).
```

在无向图上的简单随机游走，稳态分布与顶点度成正比。顶点度为：

$\deg(E)=4$（与四个底点相连），

每个底点（$A,B,C,D$）的度为 $3$（与两个相邻底点和顶点 $E$ 相连）。

所以

```math
\pi(E)=\frac{\deg(E)}{\sum_{v}\deg(v)}=\frac{4}{4+4\cdot3}=\frac{4}{16}=\frac14.
```

也可用稳态方程验证：设底点的稳态概率为 $y$（四个相等），顶点 $E$ 为 $x$。有

```math
x=4\cdot\frac13 y,\qquad x+4y=1,
```

解得 $x=\tfrac14$。

因此答案为

```math
\boxed{\displaystyle\lim_{t\to\infty}P_E(t)=\frac14.}
```
