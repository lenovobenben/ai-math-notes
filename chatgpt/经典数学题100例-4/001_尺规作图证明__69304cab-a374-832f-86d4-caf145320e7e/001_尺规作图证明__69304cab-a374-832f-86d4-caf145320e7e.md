# 尺规作图证明

- Conversation ID: `69304cab-a374-832f-86d4-caf145320e7e`

## Question

尺规作图问题。有一个三角形ABC，其中∠B为钝角。请在AC的另一侧作出D,E两点。使得ABCDE构成凸五边形。并且使得三角形ABC,BCD,CDE,DEA,EAB的面积都相等。

这里给出一种作法。首先我们假设BC是水平放置的。A在BC的左上方。
1。过A作BC的平行线l_1。过C作AB的平行线l_2。l_1,l_2交于点F。
2。连接BF。作出长度len=\frac{\sqrt{5}-1}{4} \cdot BF（基本尺规作图可作）。延长BF到M点。使得FM=len。
3。过M作AC的平行线，交l_1和l_2分别为D,E即所求。

请证明此作法的正确性。

## Answer

下面给出一个解析几何的严格证明（坐标法），把题中作法与面积相等条件完全对应起来，并从中得到所用比例 $\dfrac{\sqrt5-1}{4}$。

### 设坐标与构造的代数表示

如题所述，把 $BC$ 放为水平。为方便起见进行仿射归一化：取

```math
B=(0,0),\qquad C=(1,0).
```

设 $A=(u,v)$（$v>0$）。按作图步骤：

过 $A$ 作与 $BC$ 平行的直线 $l_1$：即直线 $y=v$。

过 $C$ 作与 $AB$ 平行的直线 $l_2$：方向向量与 $AB=A-B=(u,v)$ 相同，参数方程为 $C+t(u,v)=(1+tu,\;tv)$。

$l_1$ 与 $l_2$ 相交于点 $F$。由 $tv=v\Rightarrow t=1$，得到

```math
F=(1+u,\;v).
```

连接 $B$ 与 $F$，向量 $BF=F-B=(1+u,\;v)$。在射线 $BF$ 上取点 $M$ 使得

```math
\overrightarrow{FM}=\alpha\overrightarrow{BF},
```

其中常数 $\alpha$ 在作法中取 $\alpha=\dfrac{\sqrt5-1}{4}$（题中给出的长度比例）。因此

```math
M=F+\alpha(BF)=(1+u,\;v)+\alpha(1+u,\;v)=\big((1+u)(1+\alpha),\;v(1+\alpha)\big).
```

过 $M$ 作与 $AC$ 平行的直线，交 $l_1$ 在 $D$，交 $l_2$ 在 $E$。下面将用代数求出 $D,E$ 的坐标并计算五个三角形的面积。

### 求 $D,E$ 的坐标

向量 $AC=C-A=(1-u,-v)$。于是直线过 $M$ 并与 $AC$ 平行的参数方程为

```math
P(s)=M+s(1-u,-v).
```

（1）求交点 $D$（在 $l_1$ 上，故 $y=v$）：设 $P(s_D)$ 为 $D$。由 $y$ 分量：

```math
v(1+\alpha)+s_D(-v)=v \quad\Rightarrow\quad s_D=\alpha.
```

代回得

```math
D=M+\alpha(1-u,-v).
```

计算 $D$ 的 $x$ 坐标（化简）得到

```math
D=\big(1+2\alpha+u,\;v\big).
```

（2）求交点 $E$（在 $l_2$ 上，形如 $C+t(u,v)$）：令 $P(s_E)=C+t(u,v)$。
从 $y$ 分量可得 $t=v(1+\alpha)-v s_E$ 与其他等式联立，解出

```math
s_E=-\alpha,\qquad t=1+2\alpha.
```

所以

```math
E=C+t(u,v)=\big(1+u(1+2\alpha),\;v(1+2\alpha)\big).
```

### 计算五个三角形的面积并验证相等

三角形面积可用向量叉积计算（基底选择使公式简洁）。

(1) 原始三角形

```math
[ABC]=\tfrac12\big|(C-B)\times(A-B)\big|=\tfrac12\big|(1,0)\times(u,v)\big|=\tfrac12 v.
```

(2) $BCD$。由于 $D$ 的 $y$ 坐标为 $v$，有

```math
[BCD]=\tfrac12\big|(C-B)\times(D-B)\big|=\tfrac12\big|(1,0)\times(x_D,v)\big|=\tfrac12 v=[ABC].
```

(3) $CDE$。用向量 $D-C=(u+2\alpha,\;v)$、$E-C=(u(1+2\alpha),\;v(1+2\alpha))$，

```math
\begin{aligned}
[\,CDE\,]&=\tfrac12\big|(D-C)\times(E-C)\big|\\
&=\tfrac12\big| (u+2\alpha)\cdot v(1+2\alpha) - v\cdot u(1+2\alpha)\big|\\
&=\tfrac12\big|2\alpha(1+2\alpha)v\big|=\alpha(1+2\alpha)\,v.
\end{aligned}
```

因此要使 $[CDE]=\tfrac12 v$ 必须满足

```math
\alpha(1+2\alpha)=\tfrac12.
```

解关于 $\alpha$ 的二次方程 $2\alpha+4\alpha^2=1$ 即

```math
4\alpha^2+2\alpha-1=0,
```

得到正根

```math
\alpha=\frac{\sqrt5-1}{4},
```

正好与构造中所给比例一致。代回可得 $[CDE]=\tfrac12 v=[ABC]$。

(4) $DEA$。注意 $D$ 与 $A$ 在同一水平直线 $y=v$，故底边长度为 $x_D-u=1+2\alpha$，高为 $y_E-y_A=2\alpha v$。于是

```math
[DEA]=\tfrac12\cdot(1+2\alpha)\cdot(2\alpha v)=\alpha(1+2\alpha)v=\tfrac12 v=[ABC].
```

(5) $EAB$。用叉积

```math
[\,EAB\,]=\tfrac12\big|(A-B)\times(E-B)\big|
=\tfrac12\big|u\cdot v(1+2\alpha)-v\cdot(1+u(1+2\alpha))\big|
=\tfrac12\big| -v\big|=\tfrac12 v=[ABC].
```

因此在取 $\alpha=(\sqrt5-1)/4$ 时，五个三角形的面积都等于 $\tfrac12 v$，即相等。

### 关于“在 $AC$ 的另一侧且为凸五边形”的说明

从坐标可见：

$A$ 与 $D$ 同处于直线 $y=v$；

$B,C$ 在 $y=0$；

$E$ 的 $y$ 坐标为 $v(1+2\alpha)>v$（因为 $\alpha>0$），因此 $E$ 在 $l_1$ 的上方一侧；
因此 $D,E$ 位于与 $A,B,C$ 相对的一侧（即“在 $AC$ 的另一侧”），并且按顶点顺序 $A\!-\!B\!-\!C\!-\!D\!-\!E$ 可验为凸（若需可加细节证明：由坐标可以检查相邻三点的有向面积均同号，从而保证简单多边形且各内角 $<\pi$）。此外题设 $\angle B$ 为钝角保证了构造时诸线交点和方向符合题目的次序与凸性要求（避免退化、交叉的情形）。

### 结论

以坐标形式严格计算得到：按题中作法取

```math
\alpha=\frac{\sqrt5-1}{4},
```

并按所述作图得到的点 $D,E$ 将使得

```math
[ABC]=[BCD]=[CDE]=[DEA]=[EAB].
```

因此给出的尺规作法是正确的。 ∎

（附注：上面的证明完全代数化、紧凑，若需要也可以把叉积处转写为经典的几何面积相似三角形的证明，得到同样的等式 $\alpha(1+2\alpha)=\tfrac12$ 并解出 $\alpha$。）
