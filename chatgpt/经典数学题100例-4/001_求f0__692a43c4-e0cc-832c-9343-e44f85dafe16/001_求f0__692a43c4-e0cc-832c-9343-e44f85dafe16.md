# 求F'(0)

- Conversation ID: `692a43c4-e0cc-832c-9343-e44f85dafe16`

## Question

设F(x)=\int_{0}^{x}(\sin \frac{1}{t})^4\mathrm{d}t。求F'(0)=?

## Answer

先把导数用极限定义写出来：

```math
F'(0)=\lim_{x\to0}\frac{1}{x}\int_0^x\big(\sin\frac{1}{t}\big)^4\,dt .
```

令 $x>0$。把积分变量换成 $u=\frac{1}{t}$（即 $t=\frac{1}{u},\ dt=-\frac{1}{u^2}du$），得

```math
F(x)=\int_0^x\sin^4\!\frac{1}{t}\,dt=\int_{1/x}^{\infty}\frac{\sin^4 u}{u^2}\,du .
```

于是

```math
\frac{F(x)}{x}= \frac{1}{x}\int_{1/x}^{\infty}\frac{\sin^4 u}{u^2}\,du.
```

设 $a=1/x$（当 $x\to0^+$ 时 $a\to\infty$），需要计算

```math
\lim_{a\to\infty} a\int_a^{\infty}\frac{\sin^4 u}{u^2}\,du.
```

把 $\sin^4 u$ 展开为常项加余弦项：

```math
\sin^4 u=\frac{3-4\cos2u+\cos4u}{8}.
```

所以

```math
a\int_a^{\infty}\frac{\sin^4 u}{u^2}\,du
= a\left(\frac{3}{8}\int_a^{\infty}\frac{du}{u^2}-\frac{1}{2}\int_a^{\infty}\frac{\cos2u}{u^2}du+\frac{1}{8}\int_a^{\infty}\frac{\cos4u}{u^2}du\right).
```

第一项恰好等于 $a\cdot\frac{3}{8}\cdot\frac{1}{a}=\frac{3}{8}$. 对余弦项，利用分部积分可得（以一般形式说明）

```math
\Big|\int_a^{\infty}\frac{\cos(ku)}{u^2}\,du\Big|\le \frac{C}{a^2}\quad(k\gt{}0)
```

（例如分部积分后得到上界 $O(1/a^2)$），因此乘以 $a$ 后这些振荡项都趋于 $0$ 当 $a\to\infty$。

综上极限为常项贡献：

```math
\lim_{x\to0}\frac{F(x)}{x}=\frac{3}{8}.
```

因此 $F'(0)=\dfrac{3}{8}$。（左右极限相同，因为 $\sin^4(1/t)$ 关于 $t$ 的符号是偶函数，左侧同样得到 $3/8$。）
