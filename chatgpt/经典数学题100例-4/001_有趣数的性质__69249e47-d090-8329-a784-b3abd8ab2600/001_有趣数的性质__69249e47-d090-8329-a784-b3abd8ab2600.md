# 有趣数的性质

- Conversation ID: `69249e47-d090-8329-a784-b3abd8ab2600`

## Question

若存在正整数n,使得k=\sqrt{n+1} + \sqrt{n},则称k是有趣的.证明：若k是有趣的,则k^{2022}也是有趣的.

## Answer

令

```math
x=\sqrt{n+1}+\sqrt{n}\quad(n\in\mathbb Z_{\gt{}0}).
```

注意到

```math
x^{-1}=\sqrt{n+1}-\sqrt{n},
```

且 $x+x^{-1}=2\sqrt{n+1}$.

记

```math
S_m:=x^m+x^{-m}\qquad(m\in\mathbb Z_{\ge0}).
```

由恒等式（或使用 Chebyshev 多项式的性质）

```math
S_{m+1}=(x+x^{-1})S_m-S_{m-1}
```

可知对任意正整数 $m$，$S_m$ 是以 $x+x^{-1}=2\sqrt{n+1}$ 为变量的整系数多项式的值。于是当 $m$ 为偶数时，$S_m$ 实际上是以 $(x+x^{-1})^2=4(n+1)$ 为变量的整系数多项式的值，因而 $S_{2t}\in\mathbb Z$（这是因为 $4(n+1)$ 是整数）。（换句话说，$(x^{2t}+x^{-2t})/2$ 是关于 $n+1$ 的整多项式，因此为整数。）进一步，容易看到 $S_0=2$、$S_2=4(n+1)-2=2(2n+1)$ 都是偶数，由递推关系可推出所有偶数项 $S_{2t}$ 都是偶数。于是对 $m=2022$（偶数），有

```math
S_{2022}=x^{2022}+x^{-2022}\in 2\mathbb Z.
```

把 $A:=\tfrac{1}{2}S_{2022}\in\mathbb Z$。那么

```math
x^{2022}=\frac{S_{2022}+\bigl(x^{2022}-x^{-2022}\bigr)}{2}
=\frac{S_{2022}+\sqrt{S_{2022}^2-4}}{2}
=A+\sqrt{A^2-1}.
```

注意到 $A>0$，且 $A^2=(A^2-1)+1$，因此

```math
x^{2022}=A+\sqrt{A^2-1}=\sqrt{A^2}+\sqrt{A^2-1}.
```

令 $m:=A^2-1\in\mathbb Z_{>0}$，便得到

```math
x^{2022}=\sqrt{m+1}+\sqrt{m},
```

即 $x^{2022}$ 也是“有趣的”。

因此，若 $k=\sqrt{n+1}+\sqrt n$ 为有趣数，则 $k^{2022}$ 仍是有趣数。∎
