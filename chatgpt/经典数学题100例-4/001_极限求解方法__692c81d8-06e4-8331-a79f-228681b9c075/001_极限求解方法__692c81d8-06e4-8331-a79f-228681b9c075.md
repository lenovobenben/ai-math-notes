# 极限求解方法

- Conversation ID: `692c81d8-06e4-8331-a79f-228681b9c075`

## Question

设x_0=y_0=1，且x_{n+1}=x_n+y_n，y_{n+1}=x_{n+1} +x_n。求 \lim_{n \to \infty}\frac{y_n}{x_n}

## Answer

```math
\displaystyle \sqrt2
```

证明（用比值不动点法）：令 $r_n=\dfrac{y_n}{x_n}$.
由递推得

```math
r_{n+1}=\frac{y_{n+1}}{x_{n+1}}=\frac{2x_n+y_n}{x_n+y_n}=\frac{2+r_n}{1+r_n}.
```

若极限存在且为 $r$，则满足不动点方程

```math
r=\frac{2+r}{1+r}\quad\Longrightarrow\quad r(1+r)=2+r\quad\Longrightarrow\quad r^2=2.
```

因此不动点为 $r=\pm\sqrt2$。由于 $r_0=1>0$ 并且递推保正，极限取正根，故

```math
\lim_{n\to\infty}\frac{y_n}{x_n}=\sqrt2.
```

（也可用矩阵特征向量法：矩阵 $A=\begin{pmatrix}1&1\\2&1\end{pmatrix}$ 的最大特征值为 $1+\sqrt2$，对应特征向量的比值为 $(1+\sqrt2)-1=\sqrt2$。）
