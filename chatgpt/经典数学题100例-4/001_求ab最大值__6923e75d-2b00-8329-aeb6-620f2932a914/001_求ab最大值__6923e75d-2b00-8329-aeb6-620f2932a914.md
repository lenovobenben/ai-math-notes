# 求ab最大值

- Conversation ID: `6923e75d-2b00-8329-aeb6-620f2932a914`

## Question

多项式f(x)=x^4+ax^3+bx+c的根均为实数.求ab的最大值.

## Answer

设四个实根为 $r_1,r_2,r_3,r_4$。由

```math
x^4+ax^3+bx+c=\prod_{i=1}^4 (x-r_i)
```

得对称和

```math
s_1=\sum_{i}r_i=-a,\qquad s_2=\sum_{i\lt{}j} r_ir_j=0,\qquad s_3=\sum_{i\lt{}j\lt{}k} r_ir_jr_k=-b.
```

所以要求的量为

```math
ab=s_1s_3.
```

把平均值记为 $m=\dfrac{s_1}{4}$，并设

```math
r_i=m+y_i\quad(i=1,\dots,4),
```

则 $\sum_i y_i=0$。由 $s_2=0$ 可得

```math
0=s_2=\tfrac12\big(s_1^2-\sum_i r_i^2\big)\implies \sum_i r_i^2=s_1^2=16m^2.
```

又 $\sum_i r_i^2=4m^2+\sum_i y_i^2$，因此

```math
\sum_{i=1}^4 y_i^2=12m^2.
```

计算 $s_3$（用 Newton 恒等式 或 展开）得

```math
s_3=-8m^3+\tfrac13\sum_{i=1}^4 y_i^3.
```

于是

```math
ab=s_1s_3=4m\Big(-8m^3+\tfrac13\sum_i y_i^3\Big)
= m^4\Big(\tfrac{4}{3}\sum_i z_i^3-32\Big),
```

其中 $z_i:=y_i/m$（当 $m=0$ 时上式显然给出 $ab=0$），并且由 $\sum y_i=0,\ \sum y_i^2=12m^2$ 可得

```math
\sum_i z_i=0,\qquad \sum_i z_i^2=12.
```

于是问题归结为在约束 $\sum z_i=0,\ \sum z_i^2=12$ 下，最大化

```math
\displaystyle Z=\sum_{i=1}^4 z_i^3
```

用拉格朗日乘子法可知在极值处每个 $z_i$ 满足二次方程，因而至多取两种不同值，代入约束可解得极大值对应的构型为三个相等、一个不同。解得

```math
(z_1,z_2,z_3,z_4)=(3,-1,-1,-1)\ \text{（及其置换）},
```

此时

```math
Z_{\max}=3\cdot(-1)^3+3^3=-3+27=24.
```

因此对任意可行的 $z$ 有 $Z\le 24$，代回得

```math
ab=m^4\Big(\tfrac{4}{3}Z-32\Big)\le m^4\big(\tfrac{4}{3}\cdot24-32\big)=0.
```

所以 $ab\le 0$。并且当 $m=0$（即 $s_1=0$，相当于 $a=0$）时可达等号；或者当 $z=(3,-1,-1,-1)$ 时任意 $m$ 都使等号成立（例如取根为 $(t,0,0,0)$ 的情形，或多重零根的情形）。

综上，所求最大值为

```math
\boxed{0},
```

等号可以由例如 $f(x)=x^4$（四重根 0）或 $f(x)=x^3(x-t)=x^4-tx^3$（根 $t,0,0,0$）等多种多项式达到。
