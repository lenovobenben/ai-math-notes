# 最大值求解

- Conversation ID: `69248403-67a0-8326-b92c-041ff4e39ba9`

## Question

```math
有实数a,b,c,d,e满足a^2+b^2+c^2+d^2+e^2=1,求|a-b|+|b-c|+|c-d|+|d-e|+|e-a|的最大值
```

## Answer

设向量 $v=(a,b,c,d,e)^\top$ 且满足 $\|v\|_2^2=a^2+b^2+c^2+d^2+e^2=1$。记差分向量

```math
u=(a-b,\;b-c,\;c-d,\;d-e,\;e-a)^\top = Bv,
```

其中 $B$ 为循环差分矩阵，矩阵作用是 $(Bv)_i=v_i-v_{i+1}$（下标按模5循环）。

有

```math
S:=|a-b|+|b-c|+|c-d|+|d-e|+|e-a|=\sum_{i=1}^5|u_i|.
```

对任意取值为 $\pm1$ 的向量 $s$，可选取使得 $|u_i|=s_i u_i$。于是

```math
S=s^\top u = s^\top B v=(B^\top s)^\top v \le \|B^\top s\|_2\;\|v\|_2
```

（由Cauchy不等式）。因为上式对任意 $s\in\{\pm1\}^5$ 都成立，所以

```math
S \le \max_{s\in\{\pm1\}^5}\|B^\top s\|_2\;\|v\|_2.
```

当 $\|v\|_2=1$ 时只需计算 $\max_{s}\|B^\top s\|_2$。注意

```math
(B^\top s)_i = s_i - s_{i-1}\in\{-2,0,2\}.
```

取例如 $s=(1,-1,1,-1,-1)$，可以算出

```math
B^\top s=(2,-2,2,-2,0),
```

其二范数为 $\sqrt{2^2+(-2)^2+2^2+(-2)^2+0^2}=\sqrt{16}=4$。不难检查没有更大的可能（各分量最大绝对值为2，最多四个分量非零），因此

```math
\max_{s}\|B^\top s\|_2=4.
```

综上对任意满足条件的 $v$ 都有

```math
S\le 4\|v\|_2=4.
```

并且上界可取：当

```math
v=\frac{1}{4}B^\top s=\frac{1}{4}(2,-2,2,-2,0)=(\tfrac12,-\tfrac12,\tfrac12,-\tfrac12,0)
```

时有 $\|v\|_2=1$ 且

```math
S=|1/2-(-1/2)|+|{-1/2-1/2}|+|1/2-(-1/2)|+|{-1/2-0}|+|0-1/2|=4,
```

因此最大值为 $4$。

答：最大值为 $\boxed{4}$。
