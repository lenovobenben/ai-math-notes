# 求BE最大值

- Conversation ID: `6916fc82-d1ac-8321-ae34-3cf4b61bb7e1`

## Question

在\triangle ABC中,AC=\sqrt{2}, BC=2\sqrt{5},将AB绕点A逆时针旋转90°得AD,连接BD,CD.若点E是CE的中点,连接BE,求BE的最大值.

## Answer

把它坐标化来做会最干脆——答案是 6。下面给出完整解法。

把 $A$ 置于原点，且令 $AC$ 在 $x$-轴上。于是

```math
A=(0,0),\quad C=(\sqrt2,0).
```

设 $B=(x,y)$。由题意有

(1)

```math
(x-\sqrt2)^2+y^2=(2\sqrt5)^2=20.
```

将线段 $AB$ 以点 $A$ 逆时针旋转 $90^\circ$，点 $B(x,y)$ 的像为

```math
D=(-y,\;x).
```

若点 $E$ 为 $CD$ 的中点，则

```math
E=\Big(\frac{\sqrt2+(-y)}2,\;\frac{0+x}2\Big)=\Big(\frac{\sqrt2-y}2,\;\frac{x}{2}\Big).
```

因此

```math
\begin{aligned}
BE^2&=\Big(\frac{\sqrt2-y}{2}-x\Big)^2+\Big(\frac{x}{2}-y\Big)^2\\
&=\tfrac14\big[(x-2y)^2+(2x+y-\sqrt2)^2\big].
\end{aligned}
```

为求 $BE$ 的最大值，等价求

```math
F(x,y)=(x-2y)^2+(2x+y-\sqrt2)^2
```

在约束 (1) 下的最大值。用拉格朗日乘数法：
设 $G=(x-\sqrt2)^2+y^2-20=0$，求解

```math
\nabla F=\lambda\nabla G.
```

解得两个驻点

```math
(x,y)=(-2\sqrt2,\ \ \sqrt2)\quad\text{和}\quad (x,y)=(4\sqrt2,\ -\sqrt2).
```

将它们代回得到对应的 $BE^2$ 分别为 $16$ 和 $36$。因此最大值为 $BE^2_{\max}=36$，即

```math
\boxed{BE_{\max}=6}.
```
