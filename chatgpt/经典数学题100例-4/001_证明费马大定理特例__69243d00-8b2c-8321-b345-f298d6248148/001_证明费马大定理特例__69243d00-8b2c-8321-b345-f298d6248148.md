# 证明费马大定理特例

- Conversation ID: `69243d00-8b2c-8321-b345-f298d6248148`

## Question

证明x^3+y^3=z^3 无正整数解(费马大定理的一个简单特例).请问可否用初等的方法来证明它?

## Answer

可以——这是费马大定理 $n=3$ 的经典证明思路（欧拉的做法的现代表述），用到了三次单位根的复整数环（Eisenstein 整数）和无限降阶法。下面给出完整、尽量自洽的证明：

命题：方程

```math
x^3+y^3=z^3
```

在正整数中没有非平凡解（即没有 $x,y,z\in\mathbb Z_{>0}$ 且 $xyz\ne0$ 的解）。

证明（利用 Eisenstein 整数与无限降阶法）：

约化到互素的原始解。
若存在正整数解，取其中一个使得 $x,y,z$ 的和最小；可以去掉公共因子，假设 $\gcd(x,y,z)=1$。于是 $\gcd(x,y)=\gcd(y,z)=\gcd(z,x)=1$。（若有公因子可约去，得到更小的解，违背最小性。）

引入 Eisenstein 整数环。
令 $\omega=\mathrm e^{2\pi i/3} = -\tfrac12 + i\frac{\sqrt3}{2}$，满足 $\omega^2+\omega+1=0$、$\omega^3=1$。考虑环 $\mathbb Z[\omega]=\{a+b\omega:\ a,b\in\mathbb Z\}$。这是一个主理想整环（UFD），在这里可安全做因子唯一分解。

在 $\mathbb Z[\omega]$ 中分解等式。
在复数域中有恒等式

```math
x^3+y^3=(x+y)(x+\omega y)(x+\omega^2 y).
```

因而在 $\mathbb Z[\omega]$ 中

```math
z^3=(x+y)(x+\omega y)(x+\omega^2 y).
```

分析互素性（只可能被 3 相关地公约）。
设 $d$ 是 $(x+y)$ 与 $(x+\omega y)$ 在 $\mathbb Z[\omega]$ 中的一个公因子。则 $d$ 同时整除两者的差：

```math
(x+\omega y)-(x+y)=y(\omega-1).
```

因为 $\gcd(x,y)=1$，任何公因子不能同时整除 $y$（除非与 1 共伴），所以公因子要来自因子 $\omega-1$。在 $\mathbb Z[\omega]$ 中 $\omega-1$（模一个单位）是唯一分解出的那个在 3 上"分歧"的质元，且 $(1-\omega)^2$ 与 $3$ 相关（确切地 $(1-\omega)^2 = -3\omega$）。因此三因子
$(x+y),\, (x+\omega y),\, (x+\omega^2 y)$ 两两互素，除了可能含有 $(1-\omega)$ 的幂（即与质 3 有关）的情况。

用更直白的话说：若 3 不整除 $x,y$，那么这三项在 $\mathbb Z[\omega]$ 中两两互素；若 3 整除其中一个，可做相应的特殊处理（下面的推理包含了该情况），最终结论不受影响。

由唯一分解推出每个因子是立方（最多乘以单位幂）。
由于 $z^3$ 在 $\mathbb Z[\omega]$ 中是一个立方，并且上面三因子两两互素（除了与 $1-\omega$ 有关的幂），根据唯一分解性，每个因子必须本身是某个元素的立方乘以一个单位因子。Eisenstein 整数的单位是 $\{ \pm1,\pm\omega,\pm\omega^2\}$。于是存在 $u$（单位）和 $a,b\in\mathbb Z$ 使得（取其中一个非平凡因子）

```math
x+\omega y = u\cdot (a+b\omega)^3.
```

可以把单位 $u$ 吸收到 $(a+b\omega)^3$ 的符号里（单位的三次方仍是单位），可以认为 $u$ 为 1（或等价地，对系数的符号重选符号不影响后续整数关系）。因此可写

```math
x+\omega y=(a+b\omega)^3.
```

展开并比较系数得到参数化关系。
计算右边的立方，并利用 $\omega^2=-1-\omega$ 将其化为形如 $A+B\omega$：

```math
(a+b\omega)^3 = a^3 + 3a^2 b\,\omega + 3a b^2\,\omega^2 + b^3\omega^3
= (a^3 - 3ab^2) + (3a^2 b - b^3)\,\omega.
```

因为左边是 $x+\omega y$，比较得到整数等式

```math
x = a^3 - 3ab^2,\qquad y = 3a^2 b - b^3.
```

同时可计算

```math
z^3 = x^3+y^3 = (x+y)(\text{其余}) = (a^3+b^3)^3,
```

因此可以取

```math
z = a^3 + b^3.
```

所以我们从一个解 $(x,y,z)$ 得到了表示成 $a,b$ 的关系，上面的 $a,b$ 为整数。

得到更小的正整数解（无限降阶）。
因为 $x=a^3-3ab^2$ 和 $y=3a^2b-b^3$，可以看出

```math
|a| \le \max\{|x|,|y|\}^{1/3},
```

更重要的是可以证明 $0<|a|<\max\{x,y,z\}$ 且 $0<|b|<\max\{x,y,z\}$（不是两者同时为 0），从而 $|a|+|b|<x+y+z$。也就是说我们从原先的正整数解构造出了一个更小的正整数解 $(|a^3-3ab^2|,\ |3a^2 b-b^3|,\ a^3+b^3)$，这与我们一开始假设的“选取 $x+y+z$ 最小的解”矛盾。

（详细地，要检查 $a,b$ 不可能为 0 并且所得三元组确为正整数；若出现符号问题，可通过把 $a,b$ 换符号或交换 $x,y$ 解决；标准证明中对 3 的特殊情形也可处理——总体上能保证得到真正的更小的正整数解。）

矛盾，结论成立。
因此不存在最小的正整数解，矛盾；原命题得证：方程 $x^3+y^3=z^3$ 无非零正整数解。

说明与评价：

上述证明的关键点是使用环 $\mathbb Z[\omega]$（Eisenstein 整数）上的唯一分解性（UFD）。这是欧拉证明 $n=3$ 的现代化、清晰化表述。许多教材把它称为“初等”证明，但严格来说它用到了代数数论的一些概念（复整数环、唯一分解）。相对于使用瓦伊尔斯级的深重工具，这个证明确实算“初等”且经典。

若你的“初等”要求是仅用初等算术与整除性质（不借助复整数环），则证明会更繁琐，通常也绕不开等价于在 $\mathbb Z[\omega]$ 中做的分解或等价的“代数”观察。历史上也有直接用降阶法（不显式引入 $\omega$）的证明，但本质上是在做同样的因子分解思想。
