# 方程正整数解

- Conversation ID: `6923d2e9-c7d4-8331-91b3-fa6f5b565995`

## Question

```math
求方程(n+1)^k-1=n^n的正整数解(n,k)
```

## Answer

我们要求解正整数解 $(n,k)$ 满足

```math
(n+1)^k-1=n^n,\qquad n,k\in\mathbb Z_{\gt{}0}.
```

先做简单检查：$n=1$ 时有 $2^k-1=1$，故 $2^k=2$，得 $k=1$。因此 $(1,1)$ 是一个解。

下面证明没有其它解。

首先看模 $n+1$ 的约束。由方程得

```math
(n+1)^k\equiv n^n+1\pmod{\,n+1\,}.
```

由于 $n\equiv-1\pmod{n+1}$，有 $n^n+1\equiv(-1)^n+1\pmod{n+1}$。若 $n$ 为偶数且 $n\ge2$，则 $(-1)^n+1=2$，所以 $n+1\mid2$，这只可能发生在 $n=1$。因此除 $n=1$ 外必须有 $n$ 为奇数。

令 $n\ge3$ 且为奇数。把方程写成

```math
n^n+1=(n+1)^k.
```

注意右边的素因子都来自 $n+1$。但根据 Bang—Zsigmondy（原始因子定理）的结论：对互质正整数 $a>b>0$ 且奇数指数 $m>1$，数 $a^m+b^m$ 存在一个“原始素因子”——即存在素数 $p$ 除 $a^m+b^m$ 但不除 $a^t+b^t$ 对任意 $t<m$（所有在该定理的例外情况里，均不符合我们这里 $a=n,b=1,m=n$ 的情形；特别地没有例外会发生在我们所需的参数上）。把 $a=n,b=1,m=n$ 代入可得：存在素数 $p$ 使得 $p\mid n^n+1$ 且 $p\nmid n^1+1=n+1$。

对这样的素数 $p$，由于 $n^n\equiv-1\pmod p$，可得到 $n^{2n}\equiv1\pmod p$ 且 $n^t\not\equiv1\pmod p$ 对任何 $t<n$（否则会与“原始”矛盾），因此模 $p$ 下 $n$ 的阶为 $2n$。由费马小定理可知阶必须除 $p-1$，于是 $2n\mid p-1$，即 $p\ge2n+1>n+1$。这与（2）中要求 $n^n+1=(n+1)^k$ 的素因子全部来自 $n+1$ 相矛盾（因为我们找到了一个素因子 $p$ 既不在 $n+1$ 的素因子中又在左边）。

综上，对 $n\ge2$ 不可能存在解，因此唯一的正整数解就是先前找到的

```math
\boxed{(n,k)=(1,1).}
```
