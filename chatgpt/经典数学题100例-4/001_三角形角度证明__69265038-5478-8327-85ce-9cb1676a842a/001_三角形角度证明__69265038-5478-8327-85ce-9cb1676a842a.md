# 三角形角度证明

- Conversation ID: `69265038-5478-8327-85ce-9cb1676a842a`

## Question

在\triangle ABC中，\frac{\sin A +\sin B +\sin C}{\cos A + \cos B + \cos C} = \sqrt{3} 。
证明：这个三角形中至少有一个内角为60°

## Answer

把条件用复数/向量想法来做会很干净。

设复数

```math
S=e^{\mathrm i A}+e^{\mathrm i B}+e^{\mathrm i C}=(\cos A+\cos B+\cos C)+\mathrm i(\sin A+\sin B+\sin C).
```

题给条件是

```math
\frac{\sin A+\sin B+\sin C}{\cos A+\cos B+\cos C}=\sqrt3,
```

即虚部/实部 = $\sqrt3$。所以 $S$ 的辐角为 $\arctan\sqrt3=\pi/3$，即存在实数 $R>0$ 使得

```math
S=R e^{\mathrm i\pi/3}.
```

两边同时乘以 $e^{-\mathrm i\pi/3}$ 得到实数：

```math
e^{\mathrm i(A-\pi/3)}+e^{\mathrm i(B-\pi/3)}+e^{\mathrm i(C-\pi/3)}\in\mathbb R,
```

因此该复数的虚部为 $0$，即

(1)

```math
\sin\!\bigl(A-\tfrac{\pi}{3}\bigr)+\sin\!\bigl(B-\tfrac{\pi}{3}\bigr)+\sin\!\bigl(C-\tfrac{\pi}{3}\bigr)=0.
```

记

```math
x=A-\tfrac{\pi}{3},\quad y=B-\tfrac{\pi}{3},\quad z=C-\tfrac{\pi}{3}.
```

因为 $A+B+C=\pi$，所以 $x+y+z=0$。由(1) 等价为

(2)

```math
\sin x+\sin y+\sin z=0,\qquad x+y+z=0.
```

用 $\sin x+\sin y=2\sin\frac{x+y}{2}\cos\frac{x-y}{2}$ 将前式变形（且 $x+y=-z$）：

```math
\sin x+\sin y=-2\sin\frac{z}{2}\cos\frac{x-y}{2},
```

所以

```math
\sin x+\sin y+\sin z=2\sin\frac{z}{2}\Bigl(\cos\frac{z}{2}-\cos\frac{x-y}{2}\Bigr)=0.
```

因此要么

```math
\sin\frac{z}{2}=0\quad\Rightarrow\quad z=0,
```

要么

```math
\cos\frac{z}{2}=\cos\frac{x-y}{2}.
```

若 $z=0$ 则 $C-\tfrac{\pi}{3}=0$，即 $C=\tfrac{\pi}{3}$（即 $60^\circ$），得证。

若 $\cos\frac{z}{2}=\cos\frac{x-y}{2}$，则由 $\cos p=\cos q$ 得

```math
\frac{x-y}{2}=\pm\frac{z}{2}+2k\pi\quad(k\in\mathbb Z).
```

由于 $x,y,z$ 都来自三角形内角减去 $\pi/3$，取值在区间 $(-\tfrac{\pi}{3},\tfrac{2\pi}{3})$（即绝对值小于 $\pi$），所以只能取 $k=0$。于是

```math
x-y=\pm z.
```

若 $x-y=z$，联立 $x+y+z=0$ 得 $2x=0\Rightarrow x=0$；若 $x-y=-z$，联立得 $2y=0\Rightarrow y=0$。两种情形都推出 $x=0$ 或 $y=0$，即 $A=\tfrac{\pi}{3}$ 或 $B=\tfrac{\pi}{3}$。

综上，不论哪种情况，三角形 $ABC$ 中至少有一个内角等于 $\tfrac{\pi}{3}$（即 $60^\circ$）。证毕。 □
