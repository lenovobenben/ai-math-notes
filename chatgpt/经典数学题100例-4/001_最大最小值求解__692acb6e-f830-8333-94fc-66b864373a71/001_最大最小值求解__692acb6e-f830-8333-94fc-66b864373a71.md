# 最大最小值求解

- Conversation ID: `692acb6e-f830-8333-94fc-66b864373a71`

## Question

设x \ge y \ge z \ge \frac{\pi}{12}，且x+y+z=\frac{\pi}{2}。求\cos x \sin y \cos z 的最大值和最小值。

## Answer

把条件写清楚：$x\ge y\ge z\ge\frac{\pi}{12},\;x+y+z=\frac{\pi}{2}$。记

```math
F(x,y,z)=\cos x\sin y\cos z.
```

把 $z$ 固定，设 $S=\frac{\pi}{2}-z$，则 $x+y=S,\;x\ge y\ge z$ 等价于 $y\in[z,\,S/2]$ 且 $x=S-y$。于是对单变量

```math
h(y)=\cos(S-y)\sin y=\cos x\sin y
```

求极值。直接求导得

```math
h'(y)=\cos(S-2y).
```

在区间 $y\in[z,S/2]$ 上，$S-2y\in[S-2(S/2),\,S-2z]=[0,\,S-2z]\subset[0,\tfrac{\pi}{2}]$（因为 $S=\frac{\pi}{2}-z<\frac{\pi}{2}$），所以 $h'(y)\ge0$。因此对固定的 $z$：

$h(y)$ 在左端 $y=z$ 取最小值；

在右端 $y=S/2$（即 $x=y$）取最大值。

把这两种情形代回原式：

当 $y=z,\;x=\frac{\pi}{2}-2z$ 时

```math
F=\cos\!\Big(\frac{\pi}{2}-2z\Big)\sin z\cos z
=\sin(2z)\cdot\frac{1}{2}\sin(2z)=\frac{1}{2}\sin^2(2z).
```

在 $z\in\big[\tfrac{\pi}{12},\tfrac{\pi}{6}\big]$ 上，$2z\in\big[\tfrac{\pi}{6},\tfrac{\pi}{3}\big]$，所以 $\sin(2z)$ 单调增加，因而

```math
F_{\min}=\frac{1}{2}\sin^2(2z)\ \text{在}\ z=\frac{\pi}{12}\ \text{处取得最小值}
=\frac{1}{2}\sin^2\!\frac{\pi}{6}=\frac{1}{2}\Big(\frac{1}{2}\Big)^2=\frac{1}{8}.
```

对应点为 $\displaystyle (x,y,z)=\Big(\frac{\pi}{3},\frac{\pi}{12},\frac{\pi}{12}\Big)$。

当 $x=y=\dfrac{\pi/2-z}{2}$ 时

```math
F=\cos z\cdot\sin x\cos x=\cos z\cdot\frac{1}{2}\sin(2x)
=\frac{1}{2}\cos z\cdot\sin\!\big(\tfrac{\pi}{2}-z\big)
=\frac{1}{2}\cos^2 z.
```

在 $z\in\big[\tfrac{\pi}{12},\tfrac{\pi}{6}\big]$ 上，$\cos z$ 单调减小，故此表达在 $z=\tfrac{\pi}{12}$ 取得最大值：

```math
F_{\max}=\frac{1}{2}\cos^2\!\frac{\pi}{12}.
```

用 $\cos\frac{\pi}{12}=\cos15^\circ=\dfrac{\sqrt6+\sqrt2}{4}$，得

```math
\cos^2\frac{\pi}{12}=\frac{2+\sqrt3}{4},\qquad
F_{\max}=\frac{1}{2}\cdot\frac{2+\sqrt3}{4}=\frac{2+\sqrt3}{8}.
```

对应点为 $\displaystyle (x,y,z)=\Big(\frac{5\pi}{24},\frac{5\pi}{24},\frac{\pi}{12}\Big)$（即 $37.5^\circ,37.5^\circ,15^\circ$）。

综上，

```math
\boxed{\,F_{\min}=\frac{1}{8}\,,\text{ achieved at }(x,y,z)=\big(\tfrac{\pi}{3},\tfrac{\pi}{12},\tfrac{\pi}{12}\big);\qquad
F_{\max}=\frac{2+\sqrt3}{8}\,,\text{ achieved at }(x,y,z)=\big(\tfrac{5\pi}{24},\tfrac{5\pi}{24},\tfrac{\pi}{12}\big).\,}
```
