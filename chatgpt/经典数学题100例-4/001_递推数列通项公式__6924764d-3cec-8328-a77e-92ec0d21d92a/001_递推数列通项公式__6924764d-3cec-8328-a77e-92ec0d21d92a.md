# 递推数列通项公式

- Conversation ID: `6924764d-3cec-8328-a77e-92ec0d21d92a`

## Question

实数数列具有递推关系式a_{n+1}=\sqrt{\frac{a_n+1}{2}},求其通项公式.友情提示:需要对a_1的大小分情况讨论.

## Answer

先说明定义域：要求 $(a_n+1)/2\ge0$，即每项必须满足 $a_n\ge-1$。因此只讨论 $a_1\ge-1$ 的情况（若 $a_1<-1$ 则递推式无意义）。

我们把情况分为三类讨论并给出通项（并证明形式上的递推成立）。

##### 1. 特殊情形 $a_1=1$

显然 $a_{n+1}=\sqrt{\dfrac{1+1}{2}}=1$，所以

```math
\boxed{a_n\equiv1.}
```

##### 2. 情形 $-1\le a_1\le1$

记 $\theta_1=\arccos a_1$（取值在 $[0,\pi]$），则 $a_1=\cos\theta_1$。利用余弦半角公式

```math
\cos\frac{\theta}{2}=\sqrt{\frac{1+\cos\theta}{2}},
```

可见如果 $a_n=\cos\theta_n$，则

```math
a_{n+1}=\sqrt{\frac{a_n+1}{2}}=\sqrt{\frac{1+\cos\theta_n}{2}}=\cos\frac{\theta_n}{2},
```

即 $\theta_{n+1}=\theta_n/2$. 由此递推得

```math
\theta_n=\frac{\theta_1}{2^{\,n-1}},
```

所以对 $n\ge1$

```math
\boxed{\,a_n=\cos\!\Big(\dfrac{\arccos a_1}{2^{\,n-1}}\Big)\,,\quad\text{当 }-1\le a_1\le1.\,}
```

（特别地，若 $a_1=-1$ 则 $\theta_1=\pi$，得到 $a_n=\cos(\pi/2^{\,n-1})$，与直接计算 $a_2=0$ 等一致。）

##### 3. 情形 $a_1>1$

对 $a>1$ 有双曲余弦半角关系

```math
\cosh\frac{u}{2}=\sqrt{\frac{\cosh u+1}{2}}.
```

记 $u_1=\mathrm{arccosh}a_1$（即 $a_1=\cosh u_1,\;u_1>0$），若设 $a_n=\cosh u_n$ 同理可得 $u_{n+1}=u_n/2$，从而

```math
u_n=\frac{u_1}{2^{\,n-1}},\qquad a_n=\cosh\!\Big(\dfrac{\mathrm{arccosh}a_1}{2^{\,n-1}}\Big).
```

因此当 $a_1>1$ 时

```math
\boxed{\,a_n=\cosh\!\Big(\dfrac{\mathrm{arccosh}a_1}{2^{\,n-1}}\Big)\,,\quad\text{当 }a_1\gt{}1.\,}
```

（可用 $\mathrm{arccosh}x=\ln\!\big(x+\sqrt{x^2-1}\,\big)$ 写出显式形式。）

##### 结论与性质

综上，对所有定义域内的初值 $a_1\ge-1$：

```math
a_n=
\begin{cases}
1, & a_1=1,\\[4pt]
\cos\!\big(\dfrac{\arccos a_1}{2^{\,n-1}}\big), & -1\le a_1\le1,\\[6pt]
\cosh\!\big(\dfrac{\mathrm{arccosh}a_1}{2^{\,n-1}}\big), & a_1\gt{}1.
\end{cases}
```

不论 $a_1\in[-1,\infty)$，都有 $\displaystyle\lim_{n\to\infty}a_n=1$。
