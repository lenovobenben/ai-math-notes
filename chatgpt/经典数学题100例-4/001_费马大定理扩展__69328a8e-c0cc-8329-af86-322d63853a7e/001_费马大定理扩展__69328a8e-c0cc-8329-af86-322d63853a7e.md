# 费马大定理扩展

- Conversation ID: `69328a8e-c0cc-8329-af86-322d63853a7e`

## Question

著名的费马大定理，我们改造一下。设 x^n+y^n=z^n 其中x,y,z是正有理数，对于任意 n\ge 3是否有解？

## Answer

### 结论（简短）

没有。若 $n\ge3$，方程

```math
x^n+y^n=z^n
```

在正有理数 $x,y,z$ 中无非平凡解。

### 证明要点（清晰版）

设有正有理数解 $x,y,z>0$，写成分数形式

```math
x=\frac{a}{q},\qquad y=\frac{b}{q},\qquad z=\frac{c}{q}
```

（把三个分母同时乘到一个公分母 $q$ —— 可以取三个分母的最小公倍数）。代入得

```math
\left(\frac{a}{q}\right)^n+\left(\frac{b}{q}\right)^n=\left(\frac{c}{q}\right)^n.
```

两边乘以 $q^n$ 得整数等式

```math
a^n+b^n=c^n,
```

且 $a,b,c$ 都为正整数，不全为零 —— 这是一个非平凡的整数解。

但这是与费马大定理（Fermat’s Last Theorem）直接矛盾的：费马大定理告诉我们对于任意整数 $n\ge3$，不存正整数的非平凡解 $a^n+b^n=c^n$。因此原假设（存在正有理数解）不成立。故对任意 $n\ge3$，方程在正有理数中无解。

（费马大定理的证明由安德鲁·怀尔斯等人在1994年前后给出，已被数学界接受。）

## Question

那我再改造一次。假设x,y,z依然为正有理数，\left \{  \right \} 代表其小数部分（10进制）。
那么对于\left \{ x^n \right \} + \left \{ y^n \right \} = \left \{ z^n \right \}。n\ge 3是否有解？

## Answer

短答：有解。而且不只“平凡”解（即所有三者为整数）——存在很多非平凡的有理数解。

下面把问题整理并给出若干构造和具体例子。

### 等价条件

记十进制小数部分为 $\{t\}=t-\lfloor t\rfloor$。方程

```math
\{x^n\}+\{y^n\}=\{z^n\}
```

等价于

```math
x^n+y^n-z^n\in\mathbb Z.
```

若把三者用同一公分母表示（可取公分母 $q$），令

```math
x=\frac{a}{q},\quad y=\frac{b}{q},\quad z=\frac{c}{q}\quad (a,b,c,q\in\mathbb Z,\ q\ge1),
```

则上式等价于

```math
\frac{a^n+b^n-c^n}{q^n}\in\mathbb Z
\quad\Longleftrightarrow\quad
a^n+b^n\equiv c^n\pmod{q^n}.
```

所以问题变成在整数间寻找满足模 $q^n$ 同余的三元组 $(a,b,c,q)$。

### 一类容易得到的“平凡/半平凡”族

若 $x,y,z$ 都是整数，则 $\{x^n\}=\{y^n\}=\{z^n\}=0$，显然成立（这是最平凡的情形）。

取任意有理数 $x$ 和任意整数 $y$，令 $z=x$。因为 $\{y^n\}=0$ 且 $\{z^n\}=\{x^n\}$，所以 $\{x^n\}+\{y^n\}=\{z^n\}$。这给出无穷多解，且 $x$ 可以是非整数 —— 例如 $x=\tfrac12,y=1,z=\tfrac12$：对任意 $n$ 都成立（左边为 $(1/2)^n+0$，右边为 $(1/2)^n$）。

这些构造虽简单，但满足题意（$x,y,z$ 都是正有理数）。

### 非平凡例子（都不是整数，也不完全相等）

我们也可以找到真正非平凡的例子（即三个数都不是整数，且并非通过简单恒等 $z=x$ 得到）。例如取 $n=3$：

```math
x=\frac{1}{4},\qquad y=\frac{1}{2},\qquad z=\frac{25}{4}.
```

计算：

```math
x^3=\left(\tfrac14\right)^3=0.015625,\quad y^3=\left(\tfrac12\right)^3=0.125,\quad z^3=\left(\tfrac{25}{4}\right)^3=390.140625.
```

于是

```math
\{x^3\}=0.015625,\ \{y^3\}=0.125,\ \{z^3\}=0.140625,
```

并且确实 $\{x^3\}+\{y^3\}=0.015625+0.125=0.140625=\{z^3\}$. 所以这是一个非平凡解。

我也用计算找到了其它非平凡例子（不同 $n$ 下均可找到），表明这类解并非罕见。

### 构造思路（如何找更多解）

要构造解，关键是解模 $q^n$ 的同余

```math
a^n+b^n\equiv c^n\pmod{q^n}.
```

方法包括：

直接枚举小的 $q,a,b$，寻找满足同余的 $c$（计算机搜索很容易找到许多例子）。

利用“让某一项成为整数”的技巧（例如使 $y$ 为整数或使 $x=z$）得到简单族。

在需要严格的“全部非整数且互不相等”情形时，也可以通过选择合适的 $q$ 和将 $c$ 取作 $a+mq$（调整 $m$ 以满足到 $q^n$ 的同余）来构造，实际操作等价于求某个整数的 $n$ 次方在模 $q^n$ 下等于给定值，这是可用 Hensel 提升或直接枚举解决的。

### 小结

如果允许 $x,y,z$ 为整数或允许 $z=x$ 的“半平凡”情况，肯定有很多解（包括无穷多非整数解，见 $x=\tfrac12,y=1,z=\tfrac12$）。

更有意思的完全非平凡例子也存在（例如 $n=3$ 的 $\tfrac14,\tfrac12,\tfrac{25}{4}$），用同余条件 $a^n+b^n\equiv c^n\pmod{q^n}$ 可以系统地搜索或构造此类解。

## Question

请严格证明它有无穷多解（特指小数部分都不为零的情况）

## Answer

好的——下面给出一个严格且结构清晰的证明：对任意固定的整数 $n\ge3$，方程

```math
\{x^n\}+\{y^n\}=\{z^n\},\qquad x,y,z\in\mathbb{Q}_{\gt{}0},
```

且要求三者的小数部分都非零，有无限多解。

证明分三步：

### 第一步（把问题转成模同余）

写 $x=\dfrac{a}{q},\ y=\dfrac{b}{q},\ z=\dfrac{c}{q}$（取公分母 $q\ge2$），其中 $a,b,c\in\mathbb Z_{>0}$ 且假设 $q\nmid a,b,c$（这样保证小数部分非零）。则

```math
\{x^n\}+\{y^n\}=\{z^n\}
```

当且仅当

```math
\frac{a^n+b^n-c^n}{q^n}\in\mathbb Z
\quad\Longleftrightarrow\quad
a^n+b^n\equiv c^n\pmod{q^n}.
```

因此要构造满足条件的有理数三元组，只需构造整数四元组 $(q,a,b,c)$ 满足

$q\ge2,\;1\le a,b,c<q^n$（或至少 $a,b,c$ 不被 $q$ 整除，以保证分数非整数），

```math
a^n+b^n\equiv c^n\pmod{q^n}
```

### 第二步（先在素数模 $p$ 上找到“种子解”，再用 Hensel 提升到 $p^n$）

我们先在素数模 $p$（即在域 $\mathbb F_p$）上寻找非平凡解，然后利用 Hensel 引理把模 $p$ 的解提升到模 $p^n$。

选择素数 $p$ 满足 $p\nmid n$ 且 $\gcd(n,p-1)=1$.
（下面说明这样的素数确实存在：把 $M$ 记作 $n$ 的所有不同素因子的乘积。我们只需找一个整数残类 $r\pmod M$ 使得对每个素因子 $q\mid n$ 有 $r\not\equiv1\pmod q$（即 $r-1$ 与 $n$ 互素），并且 $\gcd(r,M)=1$。按中国剩余定理总能找到这样的 $r$；对模 $M$ 的无穷算术级数上有无限多素数（Dirichlet 定理），因此存在素数 $p\equiv r\pmod M$。对这样的 $p$ 有 $\gcd(n,p-1)=1$.）
因此在域 $\mathbb F_p$ 上，映射

```math
\phi:\mathbb F_p^\times\to\mathbb F_p^\times,\qquad \phi(t)=t^n
```

是一个双射（因为 $\gcd(n,p-1)=1$ 意味着乘方在乘法群上是一个置换）。于是对任意给定的 $a,b\in\{1,\dots,p-1\}$，值 $a^n+b^n\in\mathbb F_p$（若为 $0$ 也可处理），必定是某个 $c^n$ 的值，即存在 $c\in\{1,\dots,p-1\}$ 使得

```math
a^n+b^n\equiv c^n\pmod p.
```

因为我们可任选 $a,b$ 为非 0（模 $p$ 意义），因此在模 $p$ 上总有非平凡解（并且可以取 $a,b,c$ 都不为 0 模 $p$，保证分母 $p$ 不整除这些分子）。

把模 $p$ 的解提升到模 $p^n$（Hensel 提升）：
设 $F(t)=t^n-(a^n+b^n)$。我们在模 $p$ 上有根 $t\equiv c\pmod p$ 且 $c\not\equiv0\pmod p$。因为 $p\nmid n$ 且 $c\not\equiv0\pmod p$，所以导数

```math
F'(t)=n t^{\,n-1}
```

在 $t=c$ 处不为 0 模 $p$。根据 Hensel 引理（标准结果），这个模 $p$ 的单根可以唯一提升为模 $p^n$ 的根，即存在整数 $\tilde c$（唯一模 $p^n$）使得

```math
\tilde c^n\equiv a^n+b^n\pmod{p^n},
```

而且 $\tilde c\not\equiv0\pmod p$。于是我们得到一个四元组 $(q,p,a,b,\tilde c)$（把 $q=p$）满足

```math
a^n+b^n\equiv \tilde c^n\pmod{p^n},
```

且 $p\nmid a,b,\tilde c$。这就给出了一个“基础解”：$x=a/p,\ y=b/p,\ z=\tilde c/p$ 满足 $\{x^n\}+\{y^n\}=\{z^n\}$，且三者小数部分都非零。

（注：上面“存在素数 $p$ 使得 $\gcd(n,p-1)=1$”是用数论常用工具 —— 通过中国剩余定理取合适残类，再用 Dirichlet 定理保证该残类里有无限多素数 —— 从而能找到这样的素数 $p$。这是标准而且弱条件的存在性事实。）

### 第三步（从一个基础解构造出无限多互异解）

一旦有了一个基础解 $(q,a,b,c)$ 满足

```math
a^n+b^n\equiv c^n\pmod{q^n},
\qquad q\nmid a,b,c,
```

我们可以把 $a$（或 $b,c$）替换成 $a+t q^n$（任意整数 $t\ge0$），依然保持同余关系，因为二项式展开的每一项（除 $a^n$ 那一项）都带有因子 $q^n$。更精确地，

```math
(a+tq^n)^n \equiv a^n \pmod{q^n}.
```

因此对每个不同的整数 $t$，三元组

```math
\Big(\frac{a+t q^n}{q},\ \frac{b}{q},\ \frac{c+t q^n}{q}\Big)
```

仍然是方程 $\{x^n\}+\{y^n\}=\{z^n\}$ 的解；而且因为 $q\nmid a$，加上 $tq^n$ 不会把分子整除以 $q$（分数的分数部分保持与原来的 $a/q$ 相同），所以三个小数部分仍旧非零。不同的 $t$ 给出不同的有理三元组（分子不同），因而解的个数是无限的。

### 小结（结论）

对任意固定 $n\ge3$，我们先构造（或保证存在）某个素数 $p$ 使得 $\gcd(n,p-1)=1$（按上面的 CRT+Dirichlet 的选择法）；

在 $\mathbb F_p$ 上映射 $t\mapsto t^n$ 是双射，于是存在 $1\le a,b,c\le p-1$ 使得 $a^n+b^n\equiv c^n\pmod p$；

用 Hensel 引理把模 $p$ 的解提升到模 $p^n$，得到基础解且保证分子不被 $p$ 整除（因此小数部分非零）；

最后通过在分子上加任意倍的 $p^n$（保持模 $p^n$ 同余不变）可得到无限多互异的有理三元组，且三者小数部分均非零。

因此，对于任意 $n\ge3$，满足 $\{x^n\}+\{y^n\}=\{z^n\}$ 且三者小数部分都不为零的正有理数解确实有无限多个。

下面把你的结论用更严谨且简练的形式表述并给出要点证明（便于发表或记笔记）。

### 定理（精炼表述）

对任意固定整数 $n\ge3$，方程

```math
\{x^n\}+\{y^n\}=\{z^n\},\qquad x,y,z\in\mathbb Q_{\gt{}0},
```

在满足 $\{x^n\},\{y^n\},\{z^n\}\neq0$（即 $x,y,z$ 皆非整数）的约束下有无限多个解。

### 证明要点（简洁）

等价变换：若取公分母 $q\ge2$ 并写 $x=a/q,\ y=b/q,\ z=c/q$（且 $q\nmid a,b,c$），则小数部分条件等价于整数同余

```math
a^n+b^n\equiv c^n\pmod{q^n}.
```

选素数 $p$ 使得 $\gcd(n,p-1)=1$。这样的素数存在（可用中国剩余定理选残类，再用 Dirichlet 素数算术进列定理保证该残类中有素数）。对该 $p$，乘法群 $(\mathbb F_p^\times,\cdot)$ 上 $t\mapsto t^n$ 为双射，因此任取 $a,b\in\{1,\dots,p-1\}$ 都存在 $c$ 使得

```math
a^n+b^n\equiv c^n\pmod p,
```

且可取 $p\nmid a,b,c$。

用 Hensel 引理：因为 $p\nmid n$ 且所选 $c\not\equiv0\pmod p$，方程 $t^n-(a^n+b^n)\equiv0\pmod p$ 的根 $t\equiv c\pmod p$ 是简单根，可唯一提升为模 $p^n$ 的根 $\tilde c$ 满足

```math
\tilde c^n\equiv a^n+b^n\pmod{p^n},
```

且 $\tilde c\not\equiv0\pmod p$。于是 $x=a/p,\ y=b/p,\ z=\tilde c/p$ 是一个满足条件且三者非整数的有理解。

生成无限解：固定上述基础解 $(p,a,b,\tilde c)$。对任意整数 $t\ge0$，

```math
\Big(a+t p^n\Big)^n\equiv a^n\pmod{p^n},
```

因而三元组

```math
\Big(\frac{a+t p^n}{p},\ \frac{b}{p},\ \frac{\tilde c+t p^n}{p}\Big)
```

仍满足模 $p^n$ 的同余关系，并且分子不被 $p$ 整除，故对应小数部分均非零。不同 $t$ 给出不同的有理三元组，因而解无限多。

证明只需知道 Dirichlet 定理（保证某剩余类含无穷多素数）和 Hensel 引理（提升模 $p$ 根到模 $p^k$），两者都是标准数论工具。

若不愿引用 Dirichlet，亦可用计算或更弱的存在性论证找到合适的素数 $p$；实际上对于许多具体 $n$ 可以显式取到小素数实现构造。
