# 最小群阶

- Conversation ID: `692cd269-d4d8-8333-bbc9-f729d6729810`

## Question

设G是有限群，H是G的子群，[G:H]=n，若对任意x,y \in G \setminus H 都有xH \cap Hy \ne \emptyset 。求 |G|的最小值。

## Answer

把 $|H|=m$，又 $[G:H]=n$，故 $|G|=nm$。取任意 $g\in G\setminus H$。由题意对任意 $x,y\in G\setminus H$ 有 $xH\cap Hy\ne\varnothing$，所以以某个固定的 $g\notin H$ 为代表的双陪集 $HgH$ 必然包含所有不属于 $H$ 的元素。于是双陪集数为 2，且

```math
HgH=G\setminus H,
```

所以

```math
|HgH|=|G|-|H|=nm-m=m(n-1).
```

另一方面常用双陪集的计数公式（把 $d:=|H\cap gHg^{-1}|$）有

```math
|HgH|=\frac{|H|\cdot|H|}{|H\cap gHg^{-1}|}=\frac{m^2}{d}.
```

将二者相等得

```math
\frac{m^2}{d}=m(n-1)\quad\Longrightarrow\quad n-1=\frac{m}{d}.
```

于是

```math
m=d(n-1),\qquad |G|=nm=n\cdot d(n-1).
```

因为 $d=|H\cap gHg^{-1}|\ge1$ 且为整数，故

```math
|G|=n\cdot d(n-1)\ge n(n-1).
```

即对任意满足题意的 $G$，有下界 $|G|\ge n(n-1)$。

剩下要说明下界能否达到。取等号当且仅当 $d=1$，也就是对任意 $g\notin H$ 都有

```math
H\cap gHg^{-1}=\{e\},
```

这恰好说明 $H$ 与其任何非平凡共轭仅在单位元处相交——这是 Frobenius 补的情形。此时 $G$ 是一个 Frobenius 群，存在正规 Frobenius 核 $K$ 且 $|K|=n$，并且

```math
G\cong K\rtimes H,\qquad |G|=n(n-1).
```

这样的构造在 $n$ 为素幂时总是存在：令 $n=q=p^r$ 为素幂，取域 $\mathbb{F}_q$ 的仿射线性群

```math
\mathrm{AGL}(1,q)=\mathbb{F}_q\rtimes\mathbb{F}_q^\times,
```

其中补子 $H=\mathbb{F}_q^\times$ 的阶为 $q-1=n-1$，指数为 $q=n$，且满足上面的无交条件，从而 $|G|=q(q-1)=n(n-1)$。

综上：

必有 $|G|\ge n(n-1)$；

等号 $|G|=n(n-1)$ 当且仅当存在满足 $H\cap gHg^{-1}=\{e\}$（对任意 $g\notin H$）的结构，即 $G$ 为一个有核阶为 $n$ 的 Frobenius 群；这在 $n$ 为素幂时可由 $\mathrm{AGL}(1,n)$ 给出实例，因此对素幂 $n$ 下界可达到。

（若题目只要求最小可能值，则答案为下界 $n(n-1)$，并可补充说明何时可取到该值——例如当且仅当存在上述 Frobenius 结构，素幂 $n$ 是标准的可取情形。）
