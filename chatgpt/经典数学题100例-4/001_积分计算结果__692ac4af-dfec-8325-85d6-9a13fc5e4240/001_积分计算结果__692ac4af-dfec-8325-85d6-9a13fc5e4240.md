# 积分计算结果

- Conversation ID: `692ac4af-dfec-8325-85d6-9a13fc5e4240`

## Question

令S=\int_{0}^{2024}\min \left ( \left \{ \frac{x}{8} \right \}, \left \{ \frac{x}{4} \right \} \right ) \mathrm{d}x，则 \left \lfloor S \right \rfloor 是多少？

## Answer

我们做代换 $t=\dfrac{x}{8}$，则 $x=8t,\ dx=8\,dt$，积分区间 $t\in[0,253]$。于是

```math
S=\int_{0}^{2024}\min\!\Big(\big\{\tfrac{x}{8}\big\},\big\{\tfrac{x}{4}\big\}\Big)\,dx
=8\int_{0}^{253}\min\big(\{t\},\{2t\}\big)\,dt.
```

因为被积函数对 $t$ 以 1 为周期，

```math
\int_{0}^{253}\min(\{t\},\{2t\})\,dt=253\int_{0}^{1}\min(u,\{2u\})\,du.
```

在 $u\in[0,1)$ 上，$\{2u\}=2u$ 当 $u\in[0,1/2)$，$\{2u\}=2u-1$ 当 $u\in[1/2,1)$。因此

```math
\int_{0}^{1}\min(u,\{2u\})\,du=\int_{0}^{1/2}u\,du+\int_{1/2}^{1}(2u-1)\,du
=\tfrac{1}{8}+\tfrac{1}{4}=\tfrac{3}{8}.
```

所以

```math
S=8\cdot253\cdot\frac{3}{8}=253\cdot3=759.
```

因此 $\lfloor S\rfloor=759$。
