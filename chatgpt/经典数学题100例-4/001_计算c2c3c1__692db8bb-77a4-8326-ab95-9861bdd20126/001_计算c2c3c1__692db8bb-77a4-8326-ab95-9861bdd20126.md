# 计算c2+c3/c1

- Conversation ID: `692db8bb-77a4-8326-ab95-9861bdd20126`

## Question

设正实数c_1,c_2,c_3使得对任意定义在[-\pi,\pi]上的三次多项式f(x)均有\frac{1}{2\pi}\int_{-\pi}^{\pi}f(x)\mathrm{d}x=c_1f(-\pi) + c_2f(0) + c_3f(\pi)。则\frac{c_2 + c_3}{c_1}是多少？

## Answer

由题意对任意三次多项式成立，取基函数 $1,x,x^2,x^3$。设

```math
I(f)=\frac{1}{2\pi}\int_{-\pi}^{\pi}f(x)\,dx=c_1f(-\pi)+c_2f(0)+c_3f(\pi).
```

代入 $f(x)=1$ 得

(1)

```math
c_1+c_2+c_3=1.
```

代入 $f(x)=x$（积分为0）得

(2)

```math
- \pi c_1+\;0+\pi c_3=0\implies c_3=c_1.
```

代入 $f(x)=x^2$，有

```math
\frac{1}{2\pi}\int_{-\pi}^{\pi}x^2dx=\frac{\pi^2}{3}=c_1\pi^2+c_2\cdot0+c_3\pi^2=\pi^2(c_1+c_3).
```

由此 $c_1+c_3=\tfrac{1}{3}$. 联合 (2) 得 $2c_1=\tfrac{1}{3}$，故 $c_1=\tfrac{1}{6},\;c_3=\tfrac{1}{6}$。

再由 (1) 得 $c_2=1-c_1-c_3=1-\tfrac{1}{6}-\tfrac{1}{6}=\tfrac{2}{3}$。

于是

```math
\frac{c_2+c_3}{c_1}=\frac{\tfrac{2}{3}+\tfrac{1}{6}}{\tfrac{1}{6}}=\frac{\tfrac{5}{6}}{\tfrac{1}{6}}=5.
```

答案：$\boxed{5}$。
