# 证明对称性

- Conversation ID: `692c4883-947c-832a-82e1-2dc092243c7c`

## Question

对正整数n,k,d，记m(d,k,n)为满足下面两个条件的(a_1,a_2,\dots,a_d)的组数：
1，0 \le a_1 \le a_2 \le \dots \le a_d \le k。
2，a_1 + a_2 + \dots + a_d = n。
求证：m(d,k,n)=m(k,d,n)。

## Answer

把问题用「杨图（Ferrers diagram）」和共轭分块的语言说明是最直接的。

定义回顾。
$m(d,k,n)$ 计数的对象是长度为 $d$ 的整数序列

```math
0\le a_1\le a_2\le\cdots\le a_d\le k,\qquad a_1+\cdots+a_d=n.
```

构造双射。
给定一个满足条件的序列 $(a_1,\dots,a_d)$，把它看作 $d$ 行的左对齐点阵（杨图）：第 $i$ 行放 $a_i$ 个方格（或点），行下标按从上到下或从下到上都行——关键是每行长度不超过 $k$，行数为 $d$。整个点阵显然位于 $d\times k$ 的矩形内，且点的总数为 $n$。

定义新的长度为 $k$ 的序列 $(c_1,\dots,c_k)$ 为：
对于每个列 $j=1,\dots,k$，令

```math
c_j=\#\{i: a_i\ge k+1-j\},
```

也就是第 $j$ 列中有多少格子（等价于原杨图第 $k+1-j$ 列的高度）。容易看出

$0\le c_1\le c_2\le\cdots\le c_k\le d$。（因为随着列号从 1 到 $k$ 向左移动，列高单调不减，且任何列的高度最多为行数 $d$）

$\displaystyle\sum_{j=1}^k c_j=$（图中所有格子的总数）$=\sum_{i=1}^d a_i=n$。

因此 $(c_1,\dots,c_k)$ 是一个满足定义的长度为 $k$ 的序列——也就是计入 $m(k,d,n)$ 的对象。

反过来，从任一满足 $0\le c_1\le\cdots\le c_k\le d$ 且和为 $n$ 的序列出发，把它看作位于同一 $d\times k$ 矩形内的杨图（按列给出列高），对图做转置（共轭），就得到行长度为 $a_1\le\cdots\le a_d$ 的杨图；这给出了逆映射。因此上述构造是一个双射。

结论。
由于存在从满足 $m(d,k,n)$ 的对象到满足 $m(k,d,n)$ 的对象的双射（即杨图取共轭），有

```math
m(d,k,n)=m(k,d,n).
```

（也可以用生成函数或分块论的语言证明同样的对称性；上述杨图共轭给出了最直观的组合双射。）
