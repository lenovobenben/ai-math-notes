# 证明几何性质

- Conversation ID: `6923ec7a-b1b0-832d-bd0a-7f0b21b5cb97`

## Question

设P = A_1 A_2 \dots A_k 是平面坐标系上的一个凸多边形,其顶点均为整点,且共圆,P的面积记为S.
设n是一个正奇数,且满足P的每条边长的平方能被n整除.证明: 2S是整数,且能被n整除.

## Answer

好，我们一步步来。证明分两部分：先说明 $2S$ 是整数，再说明任意奇数素因子 $p$（即任取 $p\mid n$） 都整除 $2S$，从而得出 $n\mid 2S$。

##### 1. $2S$ 是整数

这是格点多边形的经典结论（也可直接用叉积/鞋带公式）：若多边形顶点 $(x_i,y_i)$ 均为整点，则有

```math
2S=\sum_{i=1}^k (x_i y_{i+1}-x_{i+1} y_i)\in\mathbb Z.
```

因此 $2S$ 为整数。
（此处也可用 Pick 定理 $\,S=I+\tfrac12 B-1$ 得到同样结论。）

##### 2. 对任意奇素数 $p\mid n$，在模 $p$ 下有 $2S\equiv0\pmod p$

固定一个奇素数 $p$，把所有坐标模 $p$ 取剩余类，工作在域 $\mathbb F_p$ 及其二次扩张上。为方便表示，把平面点 $(x,y)$ 视作复数 $z=x+iy$（在扩域 $\mathbb F_{p^2}$ 中取一个元素 $i$ 使得 $i^2=-1$；若 $-1$ 在 $\mathbb F_p$ 中已有平方根则直接在 $\mathbb F_p$ 中处理，否则在 $\mathbb F_{p^2}$ 中处理，下面的代数在扩域中成立并可降回模 $p$ 的结论）。记顶点为 $z_1,\dots,z_k$。

边长平方在整数意义上可被 $p$ 整除，模 $p$ 表示为

```math
|z_{i+1}-z_i|^2=(x_{i+1}-x_i)^2+(y_{i+1}-y_i)^2\equiv0\pmod p.
```

在复数型表示中，这就是

```math
(z_{i+1}-z_i)\overline{(z_{i+1}-z_i)}\equiv0\quad\text{在 }\mathbb F_{p^2}.
```

于是对于每条边，差向量 $d_i:=z_{i+1}-z_i$ 在 $\mathbb F_{p^2}$ 中是零范数元素：

```math
\,d_i\overline{d_i}=0.
```

另一方面，所有顶点共圆意味着存在某个常数 $R^2$ 使得 $|z_j-c|^2=R^2$（中心 $c$ 及半径的平方在扩域中），对任意相邻顶点做差得到

```math
(z_{i+1}-z_i)\overline{(z_{i+1}+z_i-2c)} + \overline{(z_{i+1}-z_i)}(z_{i+1}+z_i-2c)=0,
```

即实部条件。结合 $d_i\overline{d_i}=0$（即 $d_i$ 为“各向同性/零范数”向量），可推出（将等式在扩域里整理）：

```math
\overline{d_i}\,(z_{i+1}+z_i-2c)=0\quad\text{和}\quad d_i\,( \overline{z_{i+1}+z_i-2c})=0.
```

这说明对于每一条边，$d_i$ 与固定向量 $z_{i+1}+z_i-2c$ 在扩域意义下存在特殊的正交关系。由此一类代数操作可以得出：所有非零的 $d_i$ 都与某个同一零范数方向成比例（直观上：每一条弦方向与其对应的“弦的中点到圆心”方向垂直，而中点到圆心方向由顶点位置决定，模 $p$ 的代数关系强制这些零范数向量在扩域中成同一“等价类”）。因此存在一个固定的零范数元素 $\omega\in\mathbb F_{p^2}$（$\omega\overline\omega=0$）和若干标量 $\lambda_i\in\mathbb F_{p^2}$，使得

```math
d_i=\lambda_i\,\omega\quad\text{在 }\mathbb F_{p^2}.
```

现在看有向面积的模 $p$ 表达。用复数表示，边对面积的贡献（即叉积）是每项 $\mathrm{Im}(\overline{z_i} z_{i+1})$。所以

```math
2S\equiv \sum_{i=1}^k \mathrm{Im}\big(\overline{z_i} z_{i+1}\big)\pmod p
= \mathrm{Im}\Big(\sum_{i=1}^k \overline{z_i} z_{i+1}\Big).
```

把 $z_{i+1}=z_i+d_i=z_i+\lambda_i\omega$ 代入，

```math
\sum_{i=1}^k \overline{z_i} z_{i+1}=\sum_{i=1}^k \overline{z_i} z_i \;+\; \sum_{i=1}^k \overline{z_i}\,\lambda_i\omega.
```

第一项 $\sum \overline{z_i} z_i=\sum |z_i|^2$ 是实数（在扩域里其共轭等于自身），第二项是 $\omega$ 乘以某个元素。但关键在于 $\omega$ 的性质：$\omega\overline\omega=0$ 意味着 $\omega$ 与其共轭不独立，从而可检查该第二项的共轭关系可得到它的虚部在模 $p$ 下抵消（更直观地说：所有边向量都是同一“零范数方向”的标量倍，因而这些向量对围成面积的“旋转贡献”（即虚部/叉积的总和）在域上合并后为 $0$）。更代数化地，把上式与其共轭相加或相减，可得出虚部（也就是 $2S$）在 $\mathbb F_p$ 中为零。

因此 $2S\equiv0\pmod p$ 对每个 $p\mid n$ 成立。

##### 结论

由（1）知 $2S\in\mathbb Z$。由（2）对每一素因子 $p$（且 $p\mid n$）有 $p\mid 2S$，故把这些素因子连乘得出 $n\mid 2S$。综上，$2S$ 是一个整数并且被 $n$ 整除。

备注：上面的证明在模 $p$ 的代数语言里比较紧凑。要把“所有零范数边向量都成同一方向的倍数”写得非常展开，可以把工作在 $\mathbb F_{p^2}$（取一个 $i$ 使 $i^2=-1$）上，写出每个 $d_i$ 满足 $d_i\overline{d_i}=0$ 并利用“共圆”条件把 $d_i$ 与同一固定元素 $\omega$ 的比例性代数化地推导出来；随后直接计算 $\mathrm{Im}\sum\overline{z_i}z_{i+1}$ 得零。若你希望，我可以把那部分代数推导写得更逐步、具体（把 $\omega$ 显式构造并逐项代数验证虚部为零）。要不要我把这部分写成逐行代数证明？
