# 平均距离计算

- Conversation ID: `69264857-0b18-832e-ba65-7b24fb66c6ff`

## Question

n 充分大时，在 n 维球面上随机取两点的平均距离是多少？

## Answer

先假设你说的“$n$维球面”是单位球面，即单位半径的 $S^{\,n-1}=\{x\in\mathbb R^n:\|x\|=1\}$。取 $x,y$ 为球面上独立均匀随机点，它们的欧几里得距离为

```math
D=\|x-y\|=\sqrt{\|x\|^2+\|y\|^2-2\langle x,y\rangle}
=\sqrt{2-2Z},
```

其中 $Z=\langle x,y\rangle$ 是两个单位向量的内积.

要点：

对于均匀分布在单位球面上的向量，$\mathbb E[Z]=0$ 且 $\mathrm{Var}(Z)=\mathbb E[Z^2]=1/n$（直观上可把坐标系旋转使 $x=(1,0,\dots,0)$，此时 $Z$ 就是 $y$ 的第一个坐标，其方差为 $1/n$）。

当 $n\to\infty$ 时，$Z$ 的典型量级是 $O(n^{-1/2})$，因此 $Z\to0$（以概率收敛）。

用 Taylor 展开（对小 $Z$）：

```math
\sqrt{2-2Z}=\sqrt2\sqrt{1-Z}=\sqrt2\Big(1-\tfrac{1}{2}Z-\tfrac{1}{8}Z^2+O(Z^3)\Big).
```

取期望并利用 $\mathbb E[Z]=0,\ \mathbb E[Z^2]=1/n$，得到

```math
\mathbb E[D]=\sqrt2\Big(1-\tfrac{1}{8}\mathbb E[Z^2]+o(\mathbb E[Z^2])\Big)
=\sqrt2\Big(1-\tfrac{1}{8n}+o\!\big(\tfrac{1}{n}\big)\Big).
```

因此结论是：当 $n$ 很大时，两点的平均距离趋近于 $\sqrt2$；更精确的渐近式为

```math
\boxed{\;\mathbb E[\|x-y\|]=\sqrt2\Big(1-\dfrac{1}{8n}+o\big(\dfrac{1}{n}\big)\Big)\;}.
```

（若球面半径为 $R$，则把结果乘以 $R$：极限为 $\sqrt2\,R$。）
