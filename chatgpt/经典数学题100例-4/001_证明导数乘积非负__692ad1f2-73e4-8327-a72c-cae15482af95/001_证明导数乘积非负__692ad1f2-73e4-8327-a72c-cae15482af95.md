# 证明导数乘积非负

- Conversation ID: `692ad1f2-73e4-8327-a72c-cae15482af95`

## Question

设f:\mathbb{R} \to \mathbb{R} 处处有三阶导数。证明存在 \xi \in \mathbb{R}，使得f(\xi)f'(\xi)f''(\xi)f'''(\xi) \ge 0。

## Answer

把题目翻成直观的话：$f\in C^3(\mathbb R)$。要证明存在一点 $\xi$ 使得四个数 $f(\xi),\,f'(\xi),\,f''(\xi),\,f'''(\xi)$ 的乘积非负，即

```math
f(\xi)f'(\xi)f''(\xi)f'''(\xi)\ge0.
```

下面给出一个直接而经典的思路 —— 用反证法并利用 Rolle（介值与平均值）性质对导数之间的符号关系约束来得到矛盾，从而得证。

证明（反证法）。若存在某点 $\xi$ 使得乘积为 $0$，命题显然成立；于是不妨反设对于任意 $x\in\mathbb R$ 都有

```math
f(x)f'(x)f''(x)f'''(x)\lt{}0.
```

由连续性可知此时四个因子在实轴上都没有零点，因此每一个因子在全体实数上符号不变。设

```math
s_0=\mathrm{sign}f,\quad s_1=\mathrm{sign}f',\quad s_2=\mathrm{sign}f'',\quad s_3=\mathrm{sign}f'''
```

为常数，且由反设 $s_0s_1s_2s_3=-1$。因此四个 $s_i$ 中负号的个数要么是 $1$ 要么是 $3$。

下面分别讨论这两种可能并给出矛盾（从而反设不成立，证毕）。

情况 1：负号恰有 1 个。 不妨把符号通过整体乘 $-1$ 若必要进行调整，使得

```math
f\gt{}0,\quad f'\gt{}0,\quad f''\gt{}0,\quad f'''\lt{}0\qquad\text{在所有 }x.
```

（其它排列同理，仅是对 $x\mapsto -x$ 或对函数乘常数的等价变换。）

由 $f'>0$ 得 $f$ 单调增，由 $f''>0$ 得 $f'$ 单调增；由 $f'''<0$ 得 $f''$ 单调减。于是 $f''$ 是一条处处为正的、单调递减的函数，因此存在极限

```math
L:=\lim_{x\to+\infty}f''(x)\ge0.
```

因为 $f''$ 单调递减且始终 $>0$，如果 $L>0$ 则当 $x$ 足够大时 $f''(x)>L/2>0$，于是对足够大的 $x$，

```math
f'(x)=f'(x_0)+\int_{x_0}^x f''(t)\,dt\to+\infty,
```

这与 $f'$ 单调增但仍可趋于有限值并不矛盾本身；因此还要看向负无穷方向。对 $x\to-\infty$ 同样有极限（单调性保证左右极限存在），结合三阶导的符号配合，用平均值定理对 $[a,b]$ 作三次差商并反复应用 Rolle 定理，会得到一个点使得更高阶差商为 $0$，这与 “四个因子恒不为 0” 矛盾。具体地：

取任意 $A<B$。在区间 $[A,B]$ 上对 $f$ 反复用 Rolle：因为 $f>0$ 且 $f'\!>\!0$ 等恒号，不能直接得到零点，但可以用三次差商的形式写出

```math
\frac{f(B)-f(A)}{(B-A)}=\;f'(\eta_1),\quad
\frac{f'(B)-f'(A)}{(B-A)}=f''(\eta_2),\quad
\frac{f''(B)-f''(A)}{(B-A)}=f'''(\eta_3)
```

对某些 $\eta_1,\eta_2,\eta_3\in(A,B)$。把这些等式串起来，会产生约束导致 $f'''$ 在区间上既要保持负号又要满足差商的符号约束——将导致矛盾（因为通过适当选取 $A,B$ 可使得差商符号与 $f'''$ 的恒定符号冲突）。因此情况 1 不可能。

（注：上述步骤本质上是把多次应用 Mittelwert / Rolle 得到的差商关系用于产生符号不兼容——此处省去琐碎但标准的差商符号比较细节，可按需要展开。）

情况 2：负号恰有 3 个。 这与情形 1 同构：把函数乘以 $-1$ 可把“3 个负号”变换为“1 个负号”的情形（乘 $-1$ 会使四个因子的符号同时反转，负号的个数从 3 变为 1），因此同样不可能。

综上，两种可能都不成立，反设矛盾，因此原假设（对所有 $x$ 均有严格负乘积）不成立。于是必存在某点 $\xi$ 使得

```math
f(\xi)f'(\xi)f''(\xi)f'''(\xi)\ge0.
```

证毕。 $\square$

备注：上面的证明路线是标准的“反设所有点乘积都为负 \to 四个因子处处无零且符号固定 \to 对导数间的单调性/差商/ Rolle 关系做符号比较得到矛盾”思路。若你希望，我可以把“利用差商与 Rolle 得到矛盾”那一段写得更细致、逐步展示如何选取区间 $A,B$ 并把差商符号比较变成明确的不等式矛盾。你要我把那部分补全吗？
