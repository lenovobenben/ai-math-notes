# 求最小值问题

- Conversation ID: `692336cd-1c18-8333-8169-edb07f5d88fd`

## Question

\theta 是锐角，求最小值 \frac{8}{\sin \theta} + \frac{1}{\cos \theta}

## Answer

要求

```math
f(\theta)=\frac{8}{\sin\theta}+\frac{1}{\cos\theta}
\quad (\theta\ 为锐角)
```

的最小值。

##### 求导法

设

```math
f(\theta)=8\csc\theta+\sec\theta.
```

对 θ 求导：

```math
f'(\theta)= -8\frac{\cos\theta}{\sin^2\theta} + \frac{\sin\theta}{\cos^2\theta}.
```

令 $f'(\theta)=0$:

```math
-8\frac{\cos}{\sin^2}+\frac{\sin}{\cos^2}=0
```

```math
\frac{\sin}{\cos^2}=8\frac{\cos}{\sin^2}
```

```math
\sin^3 = 8\cos^3
```

```math
\left(\frac{\sin}{\cos}\right)^3 = 8
```

```math
\tan\theta = 2.
```

##### 代回求最小值

已知 $\tan\theta =2$，

```math
\sin\theta = \frac{2}{\sqrt{5}},\quad \cos\theta=\frac{1}{\sqrt{5}}.
```

代入原式：

```math
f_{\min}
= \frac{8}{2/\sqrt{5}} + \frac{1}{1/\sqrt{5}}
= 4\sqrt{5} + \sqrt{5}
= 5\sqrt{5}.
```

##### ✅ 最小值：

```math
\boxed{5\sqrt{5}}
```
