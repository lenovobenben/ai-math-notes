# 尺规作图证明

- Conversation ID: `6925a4c5-74b8-832e-9f82-65a22b687790`

## Question

尺规作图问题。有不共线的A,B,C三点，构造一条穿过点C的线，与A,B两点等距。

我提供一种作法。
此题有两个解：
第一个解是过C做AB的平行线。
第二个解是连接CB，CA。过A作BC的平行线m，过B作CA的平行线n。m,n交于点D。连接CD即所求。

请帮我证明其正确性。

## Answer

好 — 我来把你给出的两种尺规作法用清晰、严谨但直观的方式证明一下。目标是：构造一条过点 $C$ 的直线 $\ell$，使得点 $A$ 到 $\ell$ 的（垂直）距离等于点 $B$ 到 $\ell$ 的距离。

### 第一解：过 $C$ 做与 $AB$ 平行的直线

令 $\ell$ 为过 $C$ 且与 $AB$ 平行的直线。因为 $A$ 和 $B$ 都在同一条直线 $AB$ 上，而 $\ell$ 与 $AB$ 平行，所以从 $A$ 到 $\ell$ 与从 $B$ 到 $\ell$ 的距离等于“平行直线间的距离”，而同一条直线上任意两点到某一平行直线的垂直距离相等。因此

```math
\mathrm{dist}(A,\ell)=\mathrm{dist}(B,\ell).
```

所以该 $\ell$ 满足要求。

（直观地想：把 $AB$ 当作水平线，$\ell$ 也是水平线；$A$ 和 $B$ 的竖直坐标相同，因此到 $\ell$ 的竖直差相同。）

### 第二解：作两条平行线交点构造 $D$，取 $CD$

构造步骤回顾（你已给出）：

连接 $CA$、$CB$。

过 $A$ 作一条与 $BC$ 平行的直线 $m$。

过 $B$ 作一条与 $AC$ 平行的直线 $n$。

令 $D=m\cap n$。断言直线 $CD$ 即为所求：对 $\ell=CD$，有 $\mathrm{dist}(A,\ell)=\mathrm{dist}(B,\ell)$。

下面给出证明（用向量/面积的做法，简洁又严谨）。

第一步 — 确定 $D$ 的向量表达。

把点 $A,B,C$ 看作向量 $\mathbf a,\mathbf b,\mathbf c$。直线 $m$（过 $A$，与 $BC$ 平行）上的任一点可写为 $\mathbf a + t(\mathbf b-\mathbf c)$；直线 $n$（过 $B$，与 $AC$ 平行）上的任一点可写为 $\mathbf b + s(\mathbf c-\mathbf a)$。交点 $D$ 满足

```math
\mathbf a + t(\mathbf b-\mathbf c)=\mathbf b + s(\mathbf c-\mathbf a).
```

解得唯一的交点就是

```math
\mathbf d=\mathbf a+\mathbf b-\mathbf c.
```

（可直接代入检验：$\mathbf d-\mathbf a=\mathbf b-\mathbf c$ 与 $BC$ 平行；$\mathbf d-\mathbf b=\mathbf a-\mathbf c$ 与 $AC$ 平行，符合构造。）

于是

```math
\mathbf d-\mathbf c=(\mathbf a-\mathbf c)+(\mathbf b-\mathbf c).
```

第二步 — 用叉积（或面积）计算点到直线的距离并比较。

在平面上，一点 $P$ 到通过 $C$ 的直线方向向量 $\mathbf v$ 的距离可以用叉积表示（模长除以 $|\mathbf v|$）：

```math
\mathrm{dist}(P,\, \text{直线 } C\!+\!t\mathbf v)=\frac{|(\mathbf p-\mathbf c)\times\mathbf v|}{|\mathbf v|}.
```

现在取 $\mathbf v=\mathbf d-\mathbf c$。代入 $P=A$ 与 $P=B$ 得到的分子分别为

```math
|(\mathbf a-\mathbf c)\times(\mathbf d-\mathbf c)|,\qquad |(\mathbf b-\mathbf c)\times(\mathbf d-\mathbf c)|.
```

但由 $\mathbf d-\mathbf c=(\mathbf a-\mathbf c)+(\mathbf b-\mathbf c)$ 得

```math
(\mathbf a-\mathbf c)\times(\mathbf d-\mathbf c)
=(\mathbf a-\mathbf c)\times(\mathbf a-\mathbf c)+(\mathbf a-\mathbf c)\times(\mathbf b-\mathbf c)
=0+(\mathbf a-\mathbf c)\times(\mathbf b-\mathbf c).
```

同理

```math
(\mathbf b-\mathbf c)\times(\mathbf d-\mathbf c)
=(\mathbf b-\mathbf c)\times(\mathbf a-\mathbf c)+0=-(\mathbf a-\mathbf c)\times(\mathbf b-\mathbf c).
```

因此两者的模相等：

```math
\big|(\mathbf a-\mathbf c)\times(\mathbf d-\mathbf c)\big|
=\big|(\mathbf b-\mathbf c)\times(\mathbf d-\mathbf c)\big|.
```

除以同一个 $|\mathbf d-\mathbf c|$（即直线方向向量的模）后，就得到

```math
\mathrm{dist}(A,CD)=\mathrm{dist}(B,CD).
```

这正是要证明的结论。

（几何直观：因为 $D$ 的构造等价于把向量 $\overrightarrow{CA}$ 与 $\overrightarrow{CB}$ 叠加得到 $\overrightarrow{CD}$，所以 $(\overrightarrow{CA})\wedge(\overrightarrow{CD})$ 与 $(\overrightarrow{CB})\wedge(\overrightarrow{CD})$ 在绝对值上相等——也就是以 $CD$ 为底的三角形 $\triangle ACD$ 与 $\triangle BCD$ 面积相等，从而到 $CD$ 的高（即点到直线的距离）相等。）

### 小结

第一解：显然且直接——过 $C$ 作一条与 $AB$ 平行的直线，$A$ 与 $B$ 都在同一条平行线 $AB$ 上，因此到该直线的垂直距离相等。

第二解：构造的交点 $D$ 等于向量和 $\mathbf d=\mathbf a+\mathbf b-\mathbf c$，从而 $\overrightarrow{CD}=\overrightarrow{CA}+\overrightarrow{CB}$，用叉积（或面积）立即得到 $\mathrm{dist}(A,CD)=\mathrm{dist}(B,CD)$。
