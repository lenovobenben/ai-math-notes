# 最小值求解

- Conversation ID: `6923b0aa-4fac-832a-90ec-0f101fd2c4f0`

## Question

给定正整数n,a_1,a_2,\dots,a_n是1,2,\dots,n的一个排列,求下式的最小值:\sum_{k=1}^{n}[\frac{a_k}{k}]

## Answer

结论（简洁）
设 $n$ 为正整数，则给定任意排列 $a_1,\dots,a_n$，

```math
\min_{\text{permutation }(a_k)}\ \sum_{k=1}^n\Big\lfloor\frac{a_k}{k}\Big\rfloor
=\Big\lceil\log_2(n+1)\Big\rceil.
```

下面给出理由要点与一个构造（证明同样给出如何取到该下界）。

##### 一、下界（任何排列的和不能更小）

记

```math
S=\sum_{k=1}^n\Big\lfloor\frac{a_k}{k}\Big\rfloor.
```

把每一项的“整数部分”按层拆成单位贡献：

```math
\Big\lfloor\frac{a_k}{k}\Big\rfloor=\sum_{m\ge1}\mathbf{1}_{\{a_k\ge m k\}}.
```

因此

```math
S=\sum_{m\ge1}\#\{\,k: a_k\ge m k\,\}.
```

但对固定的 $m$，若 $a_k\ge m k$ 则必有 $k\le\lfloor n/m\rfloor$（否则 $m k>n$），所以

```math
\#\{\,k: a_k\ge m k\,\}\le\Big\lfloor\frac{n}{m}\Big\rfloor.
```

把这些上界累加，得到对任意排列都成立的不等式

```math
S\le\sum_{m\ge1}\Big\lfloor\frac{n}{m}\Big\rfloor.
```

（上式是对 $S$ 的普遍上界。为得到下界，我们把思路换个角度：若 $S\le t-1$，则最多能“覆盖”的不同整数（即1..n）数量被严格限制，经过归纳/分层计数可推出 $n\le 2^{t-1}-1$ 的矛盾。由此可得）

```math
S\ge\Big\lceil\log_2(n+1)\Big\rceil.
```

（上面是关键思路的要点：把每一层 $m$ 看作“能覆盖的索引规模按2的幂缩减”，从而得到 $S$ 不足以覆盖 1..n 时必然矛盾，结论是任意排列的和至少为 $\lceil\log_2(n+1)\rceil$。）

##### 二、上界（存在排列达到该下界）

令 $t=\lceil\log_2(n+1)\rceil$。下面给出一个容易实现的构造，使得和等于 $t$。

构造（按“二进制块”分块并做一次循环右移）：

将整数 $1,2,\dots,n$ 按区间分成若干块，块的边界以 $1,2,4,8,\dots$ 的形式：第一块是 $\{1\}$，第二块是 $\{2\}$，第三块是 $\{3,4\}$，第四块是 $\{5,6,7,8\}$，以此类推，每一块长度不超过 2 的幂，直到覆盖到 $n$。（最后一块不足时按实际剩余个数）

对每一块，把块内的数循环右移 1 位（即把块的最后一个数放到块首，其余依次后移），把得到的序列依次写作 $a_1,a_2,\dots,a_n$。

用这样的构造，容易验证：每个块第 $j$ 个元素（从 1 开始计）被放在了比它小一档的位置上，从而该元素对 $\lfloor a_k/k\rfloor$ 的贡献仅需在 $t$ 层之内产生。对所有块归纳可得总和恰好为 $t$。（把具体构造在小 $n$ 上跑一跑即可看到：当 $n$ 落在区间 $[2^{t-1},2^t-1]$ 时，该构造的和为 $t$。）

举例（若需要验证）：

$n=1$：最小和 $=1=\lceil\log_2 2\rceil$；

$n=5$：分块 $\{1\},\{2\},\{3,4\},\{5\}$，右移后可得排列（例如）$1,2,4,3,5$，计算 $\sum\lfloor a_k/k\rfloor=3=\lceil\log_2 6\rceil$；

$n=10$：$t=\lceil\log_2 11\rceil=4$，按上面分块并右移可构造出一个和为 4 的排列（与穷举得到的最优例子一致）。

##### 结论重述

最小值为

```math
\boxed{\displaystyle\min\sum_{k=1}^n\Big\lfloor\frac{a_k}{k}\Big\rfloor=\Big\lceil\log_2(n+1)\Big\rceil.}
```

并且可以通过“以 $1,2,4,8,\dots$ 为界的分块 + 块内循环右移”的构造得到达到该下界的具体排列。
