# 解方程求和

- Conversation ID: `69234310-5b28-8331-a10e-3efeb496cb5e`

## Question

已知a,b,c是三个不全相等的实数且满足a=ab+c,b=bc+a,c=ca+b,则a+b+c=?

## Answer

把方程写成更方便的形式并做代数运算：

由

```math
a=ab+c,\quad b=bc+a,\quad c=ca+b
```

两边相加得

```math
a+b+c=ab+bc+ca+a+b+c
```

于是

(1)

```math
ab+bc+ca=0.
```

把每个方程整理为

```math
a(1-b)=c,\quad b(1-c)=a,\quad c(1-a)=b.
```

三式相乘得

```math
abc(1-a)(1-b)(1-c)=abc.
```

若 $abc=0$ 则代入原方程只得到平凡解 $a=b=c=0$（即三者全等，题中排除）。故对非平凡解可两边除以 $abc$，得

```math
(1-a)(1-b)(1-c)=1.
```

展开并用 (1) 得

```math
1-(a+b+c)+(ab+bc+ca)-abc=1 \Longrightarrow - (a+b+c)-abc=0,
```

因此

(2)

```math
abc=-(a+b+c).
```

把关于对称和的关系消去（例如用消元或格罗布纳基），可得到关于 $S:=a+b+c$ 的多项式方程

```math
S^3-3S^2=0,
```

即 $S^2(S-3)=0$。由前面排除了 $S=0$（那只对应 $a=b=c=0$ 的平凡情形），故非平凡（即“三个不全相等”）时必有

```math
S=a+b+c=3.
```

答案：$\displaystyle 3$。
