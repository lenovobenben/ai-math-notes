# 正七边形证明

- Conversation ID: `6924743d-6ba8-832a-93e0-c3d21e66467e`

## Question

已知正7边形A_1A_2A_3A_4A_5A_6A_7.求证\frac{1}{A_1A_3} + \frac{1}{A_1A_4} = \frac{1}{A_1A_2}

## Answer

把正七边形的边长用圆的弦长表示，令外接圆半径为 $R$，设

```math
x=\frac{\pi}{7}.
```

则

```math
A_1A_2=2R\sin x,\qquad A_1A_3=2R\sin(2x),\qquad A_1A_4=2R\sin(3x).
```

所要证明的等式变为

```math
\frac{1}{2R\sin(2x)}+\frac{1}{2R\sin(3x)}=\frac{1}{2R\sin x},
```

两边同时乘以 $2R$，等价于

```math
\frac{1}{\sin(2x)}+\frac{1}{\sin(3x)}=\frac{1}{\sin x},
```

即

```math
\frac{\sin(2x)+\sin(3x)}{\sin(2x)\sin(3x)}=\frac{1}{\sin x}.
```

两边交叉相乘得到

(1)

```math
\sin2x\sin3x=\sin x\big(\sin2x+\sin3x\big).
```

用倍角与三倍角公式展开（或直接用 $\sin2x=2\sin x\cos x,\ \sin3x=3\sin x-4\sin^3 x$），把等式 (1) 化为关于 $c=\cos x$ 的代数方程。代入得到（去掉公共因子 $\sin^2x$）：

```math
2c(3-4\sin^2x)=2c+3-4\sin^2x.
```

用 $\sin^2x=1-c^2$ 化简可得

(2)

```math
8c^3-4c^2-4c+1=0.
```

现在注意 $x=\pi/7$ 满足 $\cos(7x)=\cos\pi=-1$。把 $\cos7x$ 用 $c=\cos x$ 展开得到七次多项式

```math
\cos7x=64c^7-112c^5+56c^3-7c,
```

于是 $\cos7x+1=0$ 等价于

```math
64c^7-112c^5+56c^3-7c+1=0.
```

这个七次多项式可以分解（按已知三次最小多项式分解），其中一个因子就是右边的三次多项式

```math
8c^3-4c^2-4c+1.
```

因此 $c=\cos(\pi/7)$ 必定满足 (2)。于是上面的推导全部成立，回代即可得原等式成立：

```math
\frac{1}{A_1A_3}+\frac{1}{A_1A_4}=\frac{1}{A_1A_2}.
```

（证明思路：用弦长转为正弦，化为三角恒等式，再化为关于 $\cos(\pi/7)$ 的代数方程；利用 $\cos7x=-1$ 得到该代数方程成立。）
