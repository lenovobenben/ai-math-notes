# 证明连续子段和相等

- Conversation ID: `69241885-4e7c-8328-9107-d41018c1698b`

## Question

a_1,a_2,\dots,a_m, b_1,b_2,\dots,b_m \in \{1,2,\dots,m\}.
证明:\exists 1 \le p \le q \le m,1 \le s \le t \le m 使得 a_p + \dots +a_q = b_s + \dots + b_t

## Answer

把问题用「段和（连续子段的和）」的计数和抽屉原理来做：

对序列 $a_1,\dots,a_m$ 的非空连续子段，其和的可能个数不超过子段的数量，而子段的数量是

```math
\frac{m(m+1)}2.
```

同理，序列 $b_1,\dots,b_m$ 的非空连续子段的和的数量也不超过 $\dfrac{m(m+1)}2$。

任一非空连续子段的和都在整数区间 $\{1,2,\dots,m^2\}$ 中（因为每项最小为1，最大为$m$，段长最多为 $m$，所以最大和为 $m\cdot m=m^2$）。因此两序列所有非空连续子段和的取值总数最多为 $m^2$。

但两序列的非空连续子段总数之和为

```math
\frac{m(m+1)}2+\frac{m(m+1)}2=m(m+1).
```

注意 $m(m+1)>m^2$（两边相差 $m$），所以把这 $m(m+1)$ 个子段的和放进只有 $m^2$ 个“抽屉”（可能的和值）里，按抽屉原理必有至少一个和值被两个不同的子段占用。若这两个子段分别来自 $a$ 和 $b$，则结论成立；若不是（即两个相同的和来自同一序列），那就说明该序列在自身内部已出现重复和，这并不妨碍下面的计数推理 —— 由于总体子段数量超过了可能的和值的总数，不可能出现“所有 $a$ 的和值集合”与“所有 $b$ 的和值集合”完全不相交（否则并集大小为 $m(m+1)>m^2$，与和值上界矛盾）。因此两集合必有交集，亦即存在一个和值既是某个 $a$ 的连续段和，也是某个 $b$ 的连续段和。

综上，存在 $1\le p\le q\le m,\;1\le s\le t\le m$ 使得

```math
a_p+\cdots+a_q=b_s+\cdots+b_t.
```

（$m=1$ 时显然成立。）
