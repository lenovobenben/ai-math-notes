# 最小值计算

- Conversation ID: `6916ea68-603c-8332-b6fd-4d40665011bc`

## Question

正方形ABCD边长为1,AD上有一点E,CD上有一点F, AE=CF, EH \perp BF于H.求CH的最小值.

## Answer

把正方形放在坐标系中：设

```math
A=(0,0),\;B=(1,0),\;C=(1,1),\;D=(0,1)
```

设点 $E\in AD$ 使 $AE=e$（即 $E=(0,e),\;0\le e\le1$），点 $F\in CD$ 满足 $CF=AE=e$，于是 $F=(1-e,1)$。

直线 $BF$ 上点 $H$ 为点 $E$ 到 $BF$ 的垂足（即 $EH\perp BF$）。对 $H$ 做投影计算（向量投影或直线投影），可得

```math
H\Big(\frac{1-e^{2}}{1+e^{2}},\;\frac{2e}{1+e^{2}}\Big).
```

于是

```math
CH^{2}=(1-x_H)^{2}+(1-y_H)^{2}=\frac{5e^{2}-4e+1}{e^{2}+1}.
```

令 $f(e)=CH^{2}=\dfrac{5e^{2}-4e+1}{e^{2}+1}$（在区间 $[0,1]$ 上）。对 $e$ 求导并解临界点，得到唯一落在 $[0,1]$ 的临界点

```math
e=\sqrt2-1.
```

代回可得最小值

```math
CH^{2}_{\min}=3-2\sqrt2=(\sqrt2-1)^{2},
```

因此

```math
\boxed{CH_{\min}=\sqrt2-1\approx0.41421356.}
```
