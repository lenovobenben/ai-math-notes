# 证明欧拉常数

- Conversation ID: `691f58c1-a290-832c-8f13-92e9080f0ce9`

## Question

证明 {\begin{aligned}\gamma = \lim _{s\to 0}{\frac {\zeta (1+s)+\zeta (1-s)}{2}}\end{aligned}}

## Answer

证明思路很直接：利用黎曼ζ函数在 $s=1$ 处的洛朗展开（极点与有限项），把奇异项相互抵消即可。

记号与已知事实
黎曼ζ函数在 $s=1$ 处有一个一阶极点，且存在洛朗展开

```math
\zeta(s)=\frac{1}{s-1}+\gamma+O(s-1)\qquad(s\to1),
```

其中常数项就是欧拉常数 $\gamma$。等价地，

```math
\gamma=\lim_{s\to1}\Big(\zeta(s)-\frac{1}{s-1}\Big).
```

计算极限
把 $s\to0$ 时的表达写成 $1$ 点附近的偏移：考虑 $\zeta(1+s)$ 与 $\zeta(1-s)$。由洛朗展开得

```math
\zeta(1+s)=\frac{1}{s}+\gamma+O(s),\qquad
\zeta(1-s)=\frac{1}{-s}+\gamma+O(s)=-\frac{1}{s}+\gamma+O(s).
```

将二者相加并取平均，

```math
\frac{\zeta(1+s)+\zeta(1-s)}{2}
=\frac{1}{2}\Big(\frac{1}{s}-\frac{1}{s}+2\gamma+O(s)\Big)
=\gamma+O(s).
```

令 $s\to0$ 即得

```math
\lim_{s\to0}\frac{\zeta(1+s)+\zeta(1-s)}{2}=\gamma.
```

这就是所要证明的结论：对称地取 $1$ 两侧的值可以把简单极点的奇异项抵掉，剩下的常数项正好是欧拉常数 $\gamma$。 ∎
