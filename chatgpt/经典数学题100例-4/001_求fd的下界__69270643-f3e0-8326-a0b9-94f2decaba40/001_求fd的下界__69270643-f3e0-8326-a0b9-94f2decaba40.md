# 求f(d)的下界

- Conversation ID: `69270643-f3e0-8326-a0b9-94f2decaba40`

## Question

设E\subset [0,2\pi]，E的Lebesgue测度为d，且n是正整数。 有\int_{E}|\cos(nx)|dx \ge f(d)，求f(d)。

## Answer

把问题写成找下界

```math
f(d)=\inf_{\substack{E\subset[0,2\pi]\\|E|=d}}\int_E|\cos(nx)|\,dx,\qquad 0\le d\le2\pi.
```

注意函数 $|\cos(nx)|$ 在 $[0,2\pi]$ 上关于 $x$ 的分布与 $|\cos x|$ 相同（只是做了 $n$ 倍的伸缩和平铺），因此下述计算与 $n$ 无关。

令 $a\in[0,1]$，记

```math
F(a)=\big|\{x\in[0,2\pi]:\;|\cos(nx)|\le a\}\big|.
```

设 $\theta=\arccos a\in[0,\pi/2]$。在一个长度为 $\pi$ 的周期内，$|\cos t|\le a$ 对应区间长度为 $\pi-2\theta$，在 $[0,2\pi]$ 上总共有 $2$ 个这样的 $\pi$ 周期，所以

```math
F(a)=2\pi-4\theta=2\pi-4\arccos a.
```

给定测度 $d$，取使得 $F(a)=d$ 的阈值 $a$，即

```math
\arccos a=\frac{2\pi-d}{4}\quad\Longrightarrow\quad a=\cos\!\Big(\frac{2\pi-d}{4}\Big).
```

对该阈值 $a$ 上的下位集计算积分：

```math
\int_{\{|\cos(nx)|\le a\}}|\cos(nx)|\,dx
=2\int_{0}^{\pi}\mathbf{1}_{\{| \cos t|\le a\}}|\cos t|\,dt
=4\big(1-\sin\theta\big)
=4\big(1-\sqrt{1-a^2}\big).
```

用 $\theta=\tfrac{2\pi-d}{4}=\tfrac{\pi}{2}-\tfrac{d}{4}$，有 $\sin\theta=\cos(\tfrac{d}{4})$。因此得到简洁的下界表达式

```math
\boxed{\,f(d)=4\big(1-\cos\frac{d}{4}\big),\qquad 0\le d\le 2\pi.\,}
```

取等号时可令 $E=\{x:\;|\cos(nx)|\le \cos\frac{2\pi-d}{4}\}$，即把测度 $d$ 的集合放在 $|\cos(nx)|$ 最小的那些点附近（例如围绕所有的零点做对称小区间并在各周期中均匀分配）。
