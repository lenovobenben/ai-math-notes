# 求大球半径

- Conversation ID: `69264ad9-8f78-8332-a326-0f67443c5c7d`

## Question

半径分别为6,6,6,7的四个球，两两外切。同时它们都内切与一个大球。求大球的半径

## Answer

把球看作三维“曲率”（curvature）问题，用三维推广的Descartes（Soddy–Gosset）定理：

若五个互相相切的球的曲率（带符号，外切为正，包含其它球并内切为负）为 $k_1,\dots,k_5$，则

```math
\bigg(\sum_{i=1}^5 k_i\bigg)^2=3\sum_{i=1}^5 k_i^2.
```

小球半径分别 $6,6,6,7$，所以对应曲率

```math
k_1=k_2=k_3=\tfrac{1}{6},\;k_4=\tfrac{1}{7}
```

设外包大球的曲率为 $k_5$（因为大球把小球包在内、内切，所以取负号，最后应得负值），代入上式。

先算已知四个的和与平方和：

```math
S=k_1+\cdots+k_4=\tfrac{1}{6}\cdot3+\tfrac{1}{7}=\tfrac{1}{2}+\tfrac{1}{7}=\tfrac{9}{14},
```

```math
Q=k_1^2+\cdots+k_4^2=3\cdot\frac{1}{36}+\frac{1}{49}=\frac{1}{12}+\frac{1}{49}=\frac{61}{588}.
```

代入方程 $(S+k_5)^2=3(Q+k_5^2)$，解得 $k_5=-\tfrac{1}{14}$（另一根为正，对应另一个内含小球的解，不取）。

因此大球半径

```math
R=-\frac{1}{k_5}=14.
```

答案：大球的半径为 $\boxed{14}$。
