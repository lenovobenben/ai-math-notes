# 积分计算方法

- Conversation ID: `692dba88-f37c-832f-8f78-a1364fd01923`

## Question

求\int_{0}^{1}\left ( \frac{1}{x} - \left \lfloor \frac{1}{x} \right \rfloor  \right ) \mathrm{d}x

## Answer

把区间按 $\lfloor 1/x\rfloor$ 的值分段。若 $x\in\big(\tfrac{1}{n+1},\tfrac{1}{n}\big]$（$n=1,2,\dots$），则 $\lfloor1/x\rfloor=n$，所以被积函数为 $1/x-n$。于是

```math
\begin{aligned}
I&=\int_0^1\Big(\frac1x-\big\lfloor\frac1x\big\rfloor\Big)\,dx
=\sum_{n=1}^{\infty}\int_{1/(n+1)}^{1/n}\Big(\frac1x-n\Big)\,dx\\
&=\sum_{n=1}^{\infty}\Big[\ln\frac{n+1}{n}-\frac{1}{n+1}\Big]
=\lim_{N\to\infty}\Big(\ln(N+1)-\sum_{n=1}^{N}\frac{1}{n+1}\Big)+1\\
&=1+\lim_{N\to\infty}\big(\ln(N+1)-H_{N+1}\big)=1-\gamma,
\end{aligned}
```

其中 $H_{N+1}$ 是调和数，$\gamma$ 为欧拉—马歇罗尼常数。

数值上 $\gamma\approx0.5772156649$，因此

```math
\int_0^1\Big(\frac1x-\big\lfloor\frac1x\big\rfloor\Big)\,dx=1-\gamma\approx0.4227843351.
```
