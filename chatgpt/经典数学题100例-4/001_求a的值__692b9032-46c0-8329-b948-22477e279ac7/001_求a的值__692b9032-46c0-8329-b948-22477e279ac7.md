# 求a的值

- Conversation ID: `692b9032-46c0-8329-b948-22477e279ac7`

## Question

设a,b \in \mathbb{R}满足\int_{0}^{2\pi}(x^2-a\cos x - b\sin x)^2\mathrm{d}x取得最小值。求a的值。

## Answer

把被积函数看作在 $L^2[0,2\pi]$ 中把 $f(x)=x^2$ 最小二乘投影到 $\{\cos x,\sin x\}$ 所得的线性组合。最小化时残差与 $\cos x,\sin x$ 正交，即

```math
\int_0^{2\pi}(x^2-a\cos x-b\sin x)\cos x\,dx=0,
```

```math
\int_0^{2\pi}(x^2-a\cos x-b\sin x)\sin x\,dx=0.
```

由于 $\int_0^{2\pi}\cos^2 x\,dx=\int_0^{2\pi}\sin^2 x\,dx=\pi$ 且 $\int_0^{2\pi}\cos x\sin x\,dx=0$，得

```math
a=\frac{1}{\pi}\int_0^{2\pi}x^2\cos x\,dx,\qquad
b=\frac{1}{\pi}\int_0^{2\pi}x^2\sin x\,dx.
```

计算

```math
\int_0^{2\pi}x^2\cos x\,dx
```

分部积分：设 $u=x^2,\ dv=\cos x\,dx$，得

```math
\int x^2\cos x\,dx=x^2\sin x- \int 2x\sin x\,dx
= x^2\sin x+2x\cos x-2\sin x.
```

代入上下限 $0$ 与 $2\pi$：
在 $x=2\pi$ 时值为 $0+4\pi-0=4\pi$，在 $x=0$ 时值为 $0$。因此积分为 $4\pi$，所以

```math
a=\frac{1}{\pi}\cdot 4\pi=4.
```

（顺带：$\int_0^{2\pi}x^2\sin x\,dx=-4\pi^2$，故 $b=-4\pi$。）

答案：$\displaystyle a=4$。
