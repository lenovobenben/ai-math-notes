# 尺规作图-十字架

- Conversation ID: `691d671c-4bd4-832a-8020-6d1040b2fd4a`

## Question

尺规作图问题。有4个点分别是某个正方形的四条边上的点。现在正方形消失了。请作出这个正方形。

我给一种作法。
假设四个点分别是ABCD。并且使得ABCD的顺序连起来是个凸四边形。过B做AC的垂线l，在l上做点E使得BE=AC（B，E两点分属线段AC两侧）。连接DE。DE所在直线就是目标正方形的一条边。然后分别过A，C做DE的垂线。过B做DE的平行线。所得矩形即为目标正方形。

## Answer

你的作法是对的 —— 我把它用向量/旋转的方式做个简洁严谨的说明，并给出注意事项（和退化情形）。

设定与符号
假设原正方形的一个顶点为 $P$，沿某一边方向的单位向量为 $\mathbf{u}$，再顺时针旋转 $90^\circ$ 得到的单位向量为 $\mathbf{v}$。正方形边长为 $s$。于是四个顶点为

```math
P,\; P+\;s\mathbf{u},\; P+s\mathbf{u}+s\mathbf{v},\; P+s\mathbf{v}.
```

题中四个给定点分别位于这四条边上，按顺序记作 $A,B,C,D$，则可写成（用参数 $t_1,t_2,t_3,t_4\in[0,1]$）：

```math
\begin{aligned}
A&=P+t_1 s\mathbf{u},\\
B&=P+s\mathbf{u}+t_2 s\mathbf{v},\\
C&=P+s\mathbf{u}+s\mathbf{v}-t_3 s\mathbf{u}=P+(1-t_3)s\mathbf{u}+s\mathbf{v},\\
D&=P+s\mathbf{v}-t_4 s\mathbf{v}=P+(1-t_4)s\mathbf{v}.
\end{aligned}
```

计算并说明构造为什么有效
先看 $AC$：

```math
\overrightarrow{AC}=C-A = s\mathbf{v} + (1-t_3-t_1)s\mathbf{u} = s\bigl(\mathbf{v}+\alpha\mathbf{u}\bigr),
\quad\text{其中 }\alpha=1-t_1-t_3.
```

把向量 $\mathbf{v}+\alpha\mathbf{u}$ 旋转 $90^\circ$（记作 $R$）得到

```math
R(\mathbf{v}+\alpha\mathbf{u}) = -\mathbf{u}+\alpha\mathbf{v}.
```

注意旋转不改变长度，所以 $\bigl|R(\overrightarrow{AC})\bigr|=|\overrightarrow{AC}|$.

题中在通过 $B$ 的、且与 $AC$ 垂直的直线上取点 $E$，使 $BE=|AC|$，并取在 $AC$ 的另一侧（即取 $E=B+R(\overrightarrow{AC})$ 的那一侧）。代入上面的表达式可得

```math
\begin{aligned}
E&=B+R(\overrightarrow{AC})
= \bigl(P+s\mathbf{u}+t_2 s\mathbf{v}\bigr) + s\bigl(-\mathbf{u}+\alpha\mathbf{v}\bigr)\\
&=P + s\bigl((t_2+\alpha)\mathbf{v}\bigr).
\end{aligned}
```

因此 $E$ 的位置是 $P$ 在 $\mathbf{v}$ 方向上某一点（注意 $u$ 分量消去）——也就是说 $E$ 与 $D$ 在同一条平行于 $\mathbf{v}$（即原正方形一条边方向）的直线上。于是直线 $DE$ 与 $\mathbf{v}$ 共线，正好平行于原正方形的一条边（或即为该边的延长线）。

既然得到了边方向（DE），按照你后续的步骤（过 $A,C$ 作 $DE$ 的垂线；过 $B$ 作 $DE$ 的平行线）得到的四边形边方向正好是 $\mathbf{u}$ 与 $\mathbf{v}$ 两个正交方向，且位置由三个给定点确定，必然复原出原正方形（或其平移/重合）。所以你的作法是正确的。

直观要点

关键在于：把 $AC$ 旋转 $90^\circ$ 并从 $B$ 出发，长度取为 $|AC|$，会把 $B$ 映到与 $D$ 共线的一点 $E$。因此 $DE$ 给出正方形的一条边方向。

然后用得到的方向画垂线和平行线即可完全确定正方形。

注意的特殊/退化情形

若某两个点恰好位于同一条边上或点落在顶点（参数取 0 或 1），构造仍可做但要注意区分哪条边；若四点不按边的自然循环顺序给出，先要重排成沿正方形边的顺序。

若 $A$ 与 $C$ 重合（或 $AC=0$），上面用到的“在 $B$ 上做与 $AC$ 垂线并取长度 $|AC|$” 将退化（零长度），需用别的方法处理（但实际上若 $A=C$ 表明两点在同一边或顶点，问题必须先分情况讨论）。

若给出的四点连成非凸四边形或顺序错误，构造可能失败或给出错误结果——题中你已假设是凸四边形并按边顺序 $A,B,C,D$，这是必要的假设。
