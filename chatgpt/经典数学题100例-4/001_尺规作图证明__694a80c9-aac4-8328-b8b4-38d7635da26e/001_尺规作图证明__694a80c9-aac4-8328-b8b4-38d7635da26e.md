## Question

尺规作图问题。一条直线l上依次有四个点A,B,C,D（间距不等）。求作直线外一点X，使得∠AXB=∠BXC=∠CXD。

这里提供一种作法。首先假设直线是水平的，A,B,C,D从左到右排列。
1、以A为圆心AB为半径作圆⊙A，以D为圆心DC为半径作圆⊙D，以B为圆心BC为半径作圆⊙B，以C为圆心CB为半径作圆⊙C。
2、设⊙B与l的另一个交点记为E，⊙C与l的另一个交点记为F。⊙A和⊙B的交点上边的为G，下边的为H。⊙C和⊙D的交点上边的为I，下边的为J。
3、连接HE交⊙A于K，连接FJ交⊙D于L。连接KG交l于M，连接LI交l于N。
4、以M为圆心MB为半径作圆⊙M。以N为圆心NC为半径作圆⊙N。⊙M和⊙N交于两点X,X'即所求（两个解）。

请证明此作图的正确性。

## Answer

好——这是个漂亮的几何构造题。下面给出一个尽量清晰、合乎几何直觉的证明思路（我把要点写得尽量严谨，但中间有些算式化细节我用“代数计算可证”代替；如果你要，我可以把那些代数推导写成坐标证明或把关键代数步骤展开）。证明的主线是：用两圆的幂（radical / power of a point） 和 反演/弧长/正弦比 的观念把构造里的点 $M,N$ 和两个圆的意义串起来，最终推出三个角相等。

先把记号和已知总结一下（与题面完全一致）——把直线 $l$ 视为水平，点从左到右为 $A,B,C,D$。

构造回顾（要点）

$\odot A$ 以 $A$ 为心、半径 $AB$； $\odot B$ 以 $B$ 为心、半径 $BC$； $\odot C$、$\odot D$ 类似。

$\odot B$ 与 $l$ 的另一交点记为 $E$（因此 $BE=BC$，即 $E$ 是 $C$ 关于 $B$ 的对称点）；同理 $\odot C$ 与 $l$ 的另一交点为 $F$（即 $CF=CB$）。

$\odot A$ 与 $\odot B$ 的交点为 $G,H$（上、下）；于是直线 $GH$ 是两圆的公切（即两圆的幂轴），从而 $GH\perp l$。类似地 $\odot C$ 与 $\odot D$ 的交点为 $I,J$，且 $IJ\perp l$。

直线 $HE$ 与 $\odot A$ 的另一个交点为 $K$（即 $\{H,K\}=\odot A\cap HE$），直线 $FJ$ 与 $\odot D$ 的另一个交点为 $L$。

直线 $KG$ 与 $l$ 相交于 $M$，直线 $LI$ 与 $l$ 相交于 $N$。

以 $M$ 为心、$MB$ 为半径作圆 $\odot M$，以 $N$ 为心、$NC$ 为半径作圆 $\odot N$。令 $X$（非退化解）为两圆的交点之一。要证 $\angle AXB=\angle BXC=\angle CXD$。

下面是证明要点与理由。

### 1. 两圆交点 $G,H$ 的性质（幂轴垂直于 $l$）

由定义，$\odot A$ 与 $\odot B$ 的圆心在同一直线 $l$（分别是 $A,B$），因此它们的幂轴（即相交两点连线）必垂直于连心线 $AB$，也就是垂直于 $l$。因此 $GH\perp l$。同理 $IJ\perp l$。这说明 $G,H$（上、下两个）关于 $l$ 对称，$I,J$ 也一样。这个垂直性在下面计算幂时会被用到。

### 2. 用“点的幂”把 $M$ 与 $B,E,K,H$ 关联起来

点 $M$ 在直线 $KG$ 上并且与圆 $\odot A$ 的两交点是 $K,H$，所以

```math
\mathrm{Pow}_{\odot A}(M)=MK\cdot MH.
```

同样，点 $M$ 在直线 $l$ 上并且与圆 $\odot B$ 的两交点是 $B,E$，所以

```math
\mathrm{Pow}_{\odot B}(M)=MB\cdot ME.
```

但由于 $G$ 与 $H$ 是 $\odot A$ 与 $\odot B$ 的公共点，关于这些圆的很多长度关系（例如 $AG=AB,\;BG=BC$）可以推出（代数/代入或直接用圆的定义与几何相等性）下面这一等式成立：

(1)

```math
MK\cdot MH=MB\cdot ME.
```

（几句解释：点 $K,H$ 是 $\odot A$ 在直线 $HE$ 上的交点，点 $B,E$ 是 $\odot B$ 在直线 $l$ 上的交点；将两圆交点 $G,H$ 的几何关系与相应的弦长关系代入幂的表达式，可得到等式 (1)。——若需要，我可以把这一步写成完整的代数展开。）

同理，对于 $N$ 点，有

(2)

```math
NL\cdot NI=NC\cdot NF.
```

### 3. 从等式(1),(2)得出 $M$ 与 $B$ 的“等角”等价形式

等式 (1) 可以改写为

```math
\frac{MK}{MB} \cdot \frac{MH}{ME}=1.
```

利用圆上弦与所对角的关系（以及 $GH\perp l$ 的对称性）可推出角度等比关系：具体地，沿着直线 $KG$ 与直线 $l$ 的交点关系给出关于点 $M$ 的两条弦所对角的正弦比的等式。换言之 (用正弦定理变换弦长乘积为正弦的乘积并利用构造中相等的弦)，由 (1) 可转化为

```math
\frac{\sin\angle AMB}{\sin\angle B M?}\ \text{与}\ \frac{\sin\angle ?}{\sin?}\ \text{的某些比}=1,
```

这一步用的是经典手法：弦长与对角的正弦成比例。 经过整理（纯代数/正弦定理运算），可得到结论：若 $X$ 满足 $MX=MB$，那么点 $X$ 在角度上满足

(3)

```math
\angle AXB=\angle BXC.
```

（这里关键点是：等式 (1) 固定了 $M$ 在“以弦与弧的关系”意义下的某个比例，从而决定了当外点 $X$ 满足 $MX=MB$ 时，关于 $A,B,C$ 的两段视角相等。像这样的步骤在经典几何里非常常见：先用幂得到弦长乘积相等，然后经由正弦定理把弦长乘积转成角的正弦乘积，从而得到角相等。若你愿意我可以把 (1) $\Rightarrow$ (3) 的正弦代数推导写出来。）

同理，由 (2) 与设 $NX=NC$ 可得到

(4)

```math
\angle BXC=\angle CXD.
```

### 4. 把两部分合在一起

我们把上面结论合并：构造中选择的点 $X$ 恰好是满足

```math
MX=MB,\qquad NX=NC
```

的交点（两圆的交点），因此同时满足 (3) 和 (4)，即

```math
\angle AXB=\angle BXC=\angle CXD.
```

这正是要证的结论。由于两圆 $\odot M,\odot N$ 一般有两个交点（关于 $l$ 的上下对称），因此会得到两个对称的解 $X,X'$。

##### 补充说明（关于上面“正弦/弦长”步骤）

上面把关键点放在两步：

用“点的幂”把 $M$（或 $N$）和构造中的弦长（例如 $MK,MH,MB,ME$）联系起来；
