# 地图缩小重合点

- Conversation ID: `6924815d-c444-832f-8508-8f4fb4949c31`

## Question

将一张地图按比例缩小后放入原地图中。证明：有且仅有一点与其原来位置重合

## Answer

把题目按更精确的数学语言描述一下：把平面上的一张地图按相同的比例 $0<k<1$ 缩小后放到原地图上（允许平移和旋转），那么小图与大图上有且只有一个点恰好落回它原来的位置。下面给出证明（含直观与代数两种说明）。

##### 直观说明（几何意义）

把整张地图看作平面上的点集，缩小并放上去的过程就是对平面做一个相似变换（即先以某点作中心做倍率为 $k$ 的同心缩放，之后可能再做一个旋转和平移——合起来仍是一个相似变换）。任何非平凡的相似变换（倍率 $k\neq1$）在几何上都有一个不动点（变换把它自己映成自己），这个点就是“相似中心”（如果没有旋转则是缩放的中心；有旋转时仍有一个唯一的固定点）。该点在原图上和缩小后的图上位置相同，且唯一。

##### 代数证明（唯一性与存在性）

把平面当作 $\mathbb R^2$。任意由缩放、旋转和平移组成的相似变换 $f:\mathbb R^2\to\mathbb R^2$ 都可以写成

```math
f(x)=kR x + b,
```

其中 $0<k<1$（缩放因子），$R$ 为 $2\times2$ 的正交矩阵（表示旋转或恒等），$b\in\mathbb R^2$ 表示平移向量。

要找固定点 $x$，解方程 $f(x)=x$，即

```math
kR x + b = x \quad\Longleftrightarrow\quad (I - kR)x = b.
```

注意矩阵 $I-kR$ 是可逆的，因为 $R$ 的特征值在复数域上是模为 $1$ 的复数 $e^{i\theta}$，所以 $1-k e^{i\theta}\neq0$（因为 $k\neq1$ 且 $0<k<1$），从而 $\det(I-kR)\neq0$。因此线性方程有且仅有一个解

```math
x=(I-kR)^{-1}b.
```

这给出了存在且唯一的固定点。

##### 结论

因此，缩小并放入原图所对应的相似变换在全平面上有且只有一个固定点——即存在且仅有一点在缩放放回后仍与原来位置重合。▢

（补充直观理解：如果没有旋转，固定点就是缩放的中心；加上旋转后固定点是“组合操作”的不动心——仍然唯一。）
