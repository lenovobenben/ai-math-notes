# 最大值求解

- Conversation ID: `6916ec5f-00a4-8328-94f8-beaf4c69d045`

## Question

边长为1的正方形ABCD,点P在BC延长线上运动.连接PA,PD,求 \frac{PA}{PD}的最大值.

## Answer

把正方形放坐标系：设 $A=(0,0),\;B=(1,0),\;C=(1,1),\;D=(0,1)$。点 $P$ 在直线 $BC$（即 $x=1$）上，设 $P=(1,y)\,(y\in\mathbb R)$。

计算

```math
PA=\sqrt{(1-0)^2+(y-0)^2}=\sqrt{1+y^2},\qquad
PD=\sqrt{(1-0)^2+(y-1)^2}=\sqrt{1+(y-1)^2}.
```

为求最大值，考虑平方比

```math
f(y)=\Big(\frac{PA}{PD}\Big)^2=\frac{1+y^2}{1+(y-1)^2}=\frac{y^2+1}{y^2-2y+2}.
```

对 $y$ 求导并解临界点：

```math
f'(y)=\frac{2(-y^2+y+1)}{y^4-4y^3+8y^2-8y+4},
```

令分子为零得

```math
-y^2+y+1=0\iff y=\frac{1\pm\sqrt5}{2}.
```

代入可得

```math
f\Big(\frac{1+\sqrt5}{2}\Big)=\left(\frac{1+\sqrt5}{2}\right)^2,\qquad
f\Big(\frac{1-\sqrt5}{2}\Big)=\left(\frac{1-\sqrt5}{2}\right)^2.
```

因此最大值对应 $y=\dfrac{1+\sqrt5}{2}$，且

```math
\max\frac{PA}{PD}=\sqrt{f\big(\frac{1+\sqrt5}{2}\big)}=\frac{1+\sqrt5}{2}.
```

结论：最大值为黄金比 $\displaystyle\frac{1+\sqrt5}{2}$，当且仅当 $P=(1,\tfrac{1+\sqrt5}{2})$（即在 $C$ 之外，沿 $BC$ 延长线到该点）时取得。
