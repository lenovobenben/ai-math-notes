# 定积分不等式

- Conversation ID: `692005d4-1f98-832d-b6cc-1edfda5c8b2e`

## Question

实值函数f在[0,1]上非负且可积.
求证 (\int_{0}^{1}xf(x)\mathrm{d}x)(\int_{0}^{1}(f(x))^2\mathrm{d}x) \ge \frac{4}{9}(\int_{0}^{1}f(x)\mathrm{d}x)^3

## Answer

把题目改写成更方便处理的形式，并给出一个清晰的变分/投影证明（同时指出等号情形）。

证明。 设

```math
A=\int_0^1 f(x)\,dx,\qquad B=\int_0^1 x f(x)\,dx,\qquad C=\int_0^1 f(x)^2\,dx,
```

且 $f\ge0$ 可积。若 $A=0$（即 $f\equiv0$）不难看出原不等式成立（两边均为0）。下面设 $A>0$，归一化

```math
p(x)=\frac{f(x)}{A},
```

于是 $\int_0^1 p(x)\,dx=1$。用 $m$ 和 $s$ 表示 $p$ 的一阶矩和二次范数：

```math
m=\int_0^1 x p(x)\,dx,\qquad s=\int_0^1 p(x)^2\,dx.
```

由定义有 $B=A m,\; C=A^2 s$。原不等式等价于

```math
A m\cdot A^2 s \ge \frac{4}{9}A^3 \quad\Longleftrightarrow\quad m s \ge \frac{4}{9}.
```

所以只需对所有概率密度 $p$ 在 $[0,1]$ 上证明

```math
\bigg(\int_0^1 x p\bigg)\bigg(\int_0^1 p^2\bigg)\ge\frac{4}{9}.
```

考虑在 $L^2[0,1]$ 中由基函数 $1,x$ 张成的二维子空间。对任意给定的第一、零阶矩

```math
\langle p,1\rangle=\int_0^1 p=1,\qquad \langle p,x\rangle=\int_0^1 x p=m,
```

将 $p$ 在该二维子空间上作正交投影可以降低二范数（这是正交分解的标准事实）：

```math
\int_0^1 p(x)^2\,dx \ge \int_0^1 q(x)^2\,dx,
```

其中 $q(x)=\alpha+\beta x$ 是与 $p$ 具有相同矩（即满足 $\int q=1,\ \int x q=m$）的线性函数。由此，对于任意给定的 $m$ ，在满足 $\int p=1,\ \int x p=m$ 的函数中，$\int p^2$ 的最小值由适当的线性函数给出。解出 $\alpha,\beta$：

```math
\int_0^1 (\alpha+\beta x)\,dx=1,\qquad \int_0^1 x(\alpha+\beta x)\,dx=m,
```

解得

```math
\alpha=4-6m,\qquad \beta=12m-6.
```

于是该线性函数的二范数为

```math
\int_0^1(\alpha+\beta x)^2\,dx=12m^2-12m+4=12\Big(m-\tfrac12\Big)^2+1.
```

因此对任意概率密度 $p$ 都有

```math
s=\int_0^1 p^2 \ge 12\Big(m-\tfrac12\Big)^2+1.
```

因此

```math
m s \ge m\Big(12\Big(m-\tfrac12\Big)^2+1\Big).
```

把右端当作关于 $m\in[0,1]$ 的函数考察，其导数为零时 $m=\tfrac13$，计算得到

```math
m\Big(12\Big(m-\tfrac12\Big)^2+1\Big)\Big|_{m=1/3}=\frac{4}{9},
```

并且在允许的 $m$ 范围内该表达式的最小值就是 $\frac{4}{9}$。因此对任意概率密度 $p$ 都有

```math
m s \ge \frac{4}{9}.
```

回到原量，得到

```math
(\int_0^1 x f)(\int_0^1 f^2)=A^3\,m s \ge A^3\cdot\frac{4}{9},
```

即

```math
\Big(\int_0^1 x f(x)\,dx\Big)\Big(\int_0^1 f(x)^2\,dx\Big)\ge\frac{4}{9}\Big(\int_0^1 f(x)\,dx\Big)^3.
```

最后说明等号情形：在上面的推导中，当且仅当 $p$ 恰好等于线性函数并且使得 $m=\tfrac13$ 时取得下界。解出对应的概率密度为

```math
p(x)=2(1-x)\quad(0\le x\le1),
```

所以等号当且仅当 $f(x)=\lambda(1-x)$（$\lambda\ge0$）时成立。

证毕。 □
