# 积分计算方法

- Conversation ID: `6926fa10-db68-8333-8ba4-454c7ac1fb3e`

## Question

求\int_{0}^{1} \int_{0}^{1} \int_{0}^{1}\max \{ x+y,x+z,y+z \}\mathrm{d}x\mathrm{d}y\mathrm{d}z

## Answer

把被积函数化简：对任意 $x,y,z$,

```math
\max\{x+y,x+z,y+z\}=(x+y+z)-\min\{x,y,z\},
```

因为 $x+y=x+y+z-z$ 等。于是积分为

```math
\iiint_{[0,1]^3}(x+y+z)\,dV-\iiint_{[0,1]^3}\min\{x,y,z\}\,dV.
```

第一项 $\displaystyle\iiint x+y+z=\sum_{v\in\{x,y,z\}}\int_0^1 v\,dv=3\cdot\frac12=\frac32$.

第二项为三个独立 $U(0,1)$ 变量的最小值的期望。若 $M=\min\{x,y,z\}$，则

```math
P(M\gt{}t)=(1-t)^3\ (0\le t\le1),
```

所以

```math
\mathbb{E}[M]=\int_0^1 P(M\gt{}t)\,dt=\int_0^1(1-t)^3\,dt=\frac14.
```

因此原积分为

```math
\frac32-\frac14=\frac54.
```

答案：$\boxed{\dfrac{5}{4}}$.
