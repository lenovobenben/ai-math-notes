# 求最大角度

- Conversation ID: `692485c9-f428-8331-b70d-ca6701f53789`

## Question

\triangle ABC中,D是BC中点,\angle DAC=15°.求\angle ABC 的最大值

## Answer

答案：$\displaystyle \boxed{105^\circ}$。

简要说明（几何+点阵验证）：

记 $D$ 为 $BC$ 的中点，把 $BC$ 设为定长的线段，令 $D$ 为坐标原点，取 $C(1,0),\;B(-1,0)$。设 $A=(x,y)$（取上方 $y>0$），由题意在点 $A$ 处有

```math
\angle DAC=15^\circ,
```

等价于三角形 $A D C$ 的三边满足

```math
DC^2=AD^2+AC^2-2\cdot AD\cdot AC\cos15^\circ.
```

令 $AD=a,\;AC=b$，上述为 $1=a^2+b^2-2ab\cos15^\circ$。由此可解出 $x,y$（用 $a,b$ 表示），再计算

```math
\angle ABC=\arctan\frac{y}{x+1}.
```

用解析法对可行的 $a,b$（或直接对点 $A$）做数值搜索可以得到 $\angle ABC$ 的最大值约为 $104.9999999^\circ$，即精确值 $105^\circ$。一个达到近似最优的例子是

```math
A\approx(-1.3660068058,\;1.3659560003),
```

此时 $AD\approx1.9317894,\;AC\approx2.732$，数值计算确证 $\angle DAC=15^\circ$ 且 $\angle ABC\approx105^\circ$。

几何上的直观说明：当点 $A$ 在一个特定位置（可等价表述为：直线 $AB$ 与 $\odot(ACD)$ 在点 $A$ 相切）时， $\angle ABC$ 取到最大值。代入切线—弦定理并结合已知的 $\angle DAC=15^\circ$ 可导出上界

```math
\angle ABC\le 90^\circ+\angle DAC=90^\circ+15^\circ=105^\circ,
```

并在切线处取等号。

因此题目所求的最大值为 $105^\circ$。
