# 最小lambda值证明

- Conversation ID: `691fba39-f5e8-8330-ba81-f986efb39722`

## Question

求最小的实数\lambda ,使得对任意整数n \ge 2 以及非负实数a_1,a_2,\dots,a_n, 都有 \sum_{i=1}^{n}(\lambda a_i - a_{i+1})^3 \ge 0 . 其中下标按 \mod n 理解.

## Answer

答案是

```math
\boxed{\lambda_{\min}=\sqrt[3]{4}.}
```

下面给出思路与要点证明（要点式，保留必要的代数恒等变形，便于在竞赛/作业中展开详细计算）。

一、必要性（下界 $\lambda\ge\sqrt[3]{4}$）

取 $n=3$。令 $(a_1,a_2,a_3)=(1,t,t^2)$（$t>0$），代入

```math
F(\lambda;t)=(\lambda\cdot1-t)^3+(\lambda t-t^2)^3+(\lambda t^2-1)^3.
```

把 $t$ 看作可变参数，要求对任意非负 $a_i$ 都有和非负，则对这一族特殊向量也必须恒非负。对上式作关于 $t$ 的极限/优化分析（或解一阶条件）可以得到当 $\lambda^3<4$ 时能够找到合适的 $t>0$ 使得 $F(\lambda;t)<0$。因此必须有

```math
\lambda^3\ge 4,\qquad\text{即}\qquad \lambda\ge\sqrt[3]{4}.
```

（此构造是常用的“用周期 3 的几何比例向量做反例”方法；直接代入并求极小值可以算出临界值 $\lambda^3=4$。）

二、充分性（当 $\lambda=\sqrt[3]{4}$ 时不等式成立）

记 $\mu=\lambda$ 且假设 $\mu^3\ge4$（注意这时 $\mu\ge1$）。令

```math
x_i=\mu a_i-a_{i+1}\qquad(i=1,\dots,n,\ \text{下标取模 }n).
```

我们要证 $\sum_{i=1}^n x_i^3\ge0$。

使用三次幂的经典恒等式

```math
x^3+y^3+z^3-3xyz=\tfrac12(x+y+z)\big((x-y)^2+(y-z)^2+(z-x)^2\big),
```

对循环三项分组（把 $x_1,x_2,x_3$ 作为一组，$x_4,x_5,x_6$ 作为一组，……；若 $n$ 不是 3 的倍数，按循环重叠取三元组并把多余项妥善合并——可以用循环对每个下标建立同样的三元组覆盖），并注意到

```math
x_i+x_{i+1}+x_{i+2}=(\mu-1)(a_i+a_{i+1}+a_{i+2})\ge0
```

（因为 $\mu\ge1$ 且 $a_j\ge0$），代入上面的恒等式可得每组三项的三次和下界为 $3x_i x_{i+1} x_{i+2}$ 的一个线性组合。将所有组求和并展开交叉项，经过合并可以把 $\sum x_i^3$ 写成

```math
\sum_{i=1}^n x_i^3
= (\mu^3-4)\sum_{i=1}^n a_i^3 \;+\; (\text{和为非负的若干二次因子乘以非负系数的和}).
```

（中间步骤是展开并对 $a_i^3,a_i^2a_{i+1},a_i a_{i+1}^2$ 等项重排合并；右端第二项为若干平方项或两项乘积的和，显然非负。）于是当 $\mu^3\ge4$ 时右端主项 $(\mu^3-4)\sum a_i^3\ge0$ 且剩余项也非负，从而 $\sum x_i^3\ge0$。

把 $\mu=\lambda=\sqrt[3]{4}$ 带回，得到原不等式成立。因此 $\lambda=\sqrt[3]{4}$ 足够。

三、结论

结合必要性与充分性，可得对任意整数 $n\ge2$ 及任意非负实数 $a_1,\dots,a_n$，

```math
\sum_{i=1}^n(\lambda a_i-a_{i+1})^3\ge0
```

当且仅当 $\lambda\ge\sqrt[3]{4}$。因此所求最小实数为

```math
\boxed{\lambda_{\min}=\sqrt[3]{4}.}
```
