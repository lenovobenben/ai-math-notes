# 求最小正整数值

- Conversation ID: `692dbf94-061c-8332-89eb-6bab58c6d9f8`

## Question

设f(x)是整系数多项式，且f(2013)=4，f(2012)=3，f(2011)=2，求f(2018)能取到的最小的正整数值。

## Answer

把已知差值“消掉”。令

```math
q(x)=f(x)-(x-2011).
```

则 $q(2011)=q(2012)=q(2013)=2$。于是

```math
q(x)-2
```

是整系数多项式，且以 $x-2011,\ x-2012,\ x-2013$ 为因子，所以存在整系数多项式 $g(x)$ 使得

```math
q(x)=2+(x-2011)(x-2012)(x-2013)g(x).
```

因此

```math
f(x)=x-2009+(x-2011)(x-2012)(x-2013)g(x).
```

代入 $x=2018$ 得

```math
f(2018)=2018-2009+(7\cdot6\cdot5)g(2018)=9+210\cdot t,\quad t\in\mathbb Z.
```

所以 $f(2018)$ 对 $210$ 同余 $9$，能取到的正整数中最小的就是 $9$（取 $g\equiv0$ 时达到）。答：$\boxed{9}$。
