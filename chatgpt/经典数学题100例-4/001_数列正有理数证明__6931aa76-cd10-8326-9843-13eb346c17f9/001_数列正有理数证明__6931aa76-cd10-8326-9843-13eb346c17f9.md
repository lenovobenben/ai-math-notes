# 数列正有理数证明

- Conversation ID: `6931aa76-cd10-8326-9843-13eb346c17f9`

## Question

定义一个数列：a_0=1，并且a_{2n+1}=a_n,a_{2n+2}=a_n+a_{n+1}，其中 n \ge 0。
证明这个数列将包含全部的正有理数：\left \{ \frac{a_{n-1}}{a_n}:n \ge 1 \right \} = \left \{ \frac{1}{1},\frac{1}{2},\frac{2}{1},\frac{1}{3},\frac{3}{2},\dots \right \}

## Answer

下面给出一个清晰的证明 —— 这实际上就是著名的 Calkin–Wilf 树 / Stern 二项序列 的一个变形表述。

记

```math
r_n=\frac{a_{\,n-1}}{a_n}\qquad(n\ge1).
```

我们要证明集合 $\{r_n:n\ge1\}$ 刚好是所有正有理数，且每个正有理数以最简形式恰好出现一次。

第一步 — 推导分数的二进制递推关系。

由数列的定义（$a_0=1$, 且对 $n\ge0$ 有 $a_{2n+1}=a_n,\; a_{2n+2}=a_n+a_{n+1}$），先把 $a_{2n}$ 用 $a$ 的较低下标写出：

```math
a_{2n}=a_{(n-1)\cdot2+2}=a_{n-1}+a_n\qquad(n\ge1).
```

因此对 $n\ge1$：

```math
\begin{aligned}
r_{2n}&=\frac{a_{2n-1}}{a_{2n}}=\frac{a_{n-1}}{a_{n-1}+a_n}=\frac{r_n}{1+r_n},\\
r_{2n+1}&=\frac{a_{2n}}{a_{2n+1}}=\frac{a_{n-1}+a_n}{a_n}=1+\frac{a_{n-1}}{a_n}=1+r_n.
\end{aligned}
```

并且初始值 $r_1=\dfrac{a_0}{a_1}=1/1$。

所以得到对所有 $n\ge1$ 的递推：

```math
r_{2n}=\frac{r_n}{1+r_n},\qquad r_{2n+1}=1+r_n,\qquad r_1=1.
```

这正是构造 Calkin–Wilf 二叉树 的典型规则：若某节点的值是 $x$，那么它的 左子节点 为 $x/(1+x)$，右子节点 为 $1+x$。根节点为 $1$。从根出发按层次（或按按二进制索引 $n$ 的自然顺序）依序展开，就得到序列 $\{r_n\}$。

第二步 — 所有正有理数都能出现（满射）。

给定任意最简正有理数 $\dfrac{p}{q}$（$p,q\in\mathbb Z_{>0}$，$\gcd(p,q)=1$）。若 $p=q=1$ ，它就是根 $1/1=r_1$。否则：

若 $p>q$，则 $\dfrac{p}{q}=(1+\dfrac{p-q}{q})$，说明它是右子节点，父节点是 $\dfrac{p-q}{q}$；

若 $p<q$，则 $\dfrac{p}{q}=\dfrac{\,p\,}{p+(q-p)}=\dfrac{\,\frac{p}{q-p}\,}{\,1+\frac{p}{q-p}\,}$，说明它是左子节点，父节点是 $\dfrac{p}{q-p}$。

在这两种情形中，父节点的分子分母之和 $p+q$ 会严格减小（$p+q \mapsto p+(q-p)=q$ 或 $p+q\mapsto (p-q)+q=p$），因此反复取父节点，经过有限步必能回退到根 $1/1$。于是每个 $\dfrac{p}{q}$ 都可以从根通过一条有限的“左右”路径得到，也就是说存在某个索引 $n$ 使得 $r_n=\dfrac{p}{q}$。因此 $\{r_n\}$ 覆盖所有正有理数（满射）。

第三步 — 唯一性（单射）。

上面的反向操作（把分数映回父节点）在最简分数情形下是唯一的：若分数 $p/q$ 满足 $p>q$ 则父节点必须是 $(p-q)/q$；若 $p<q$ 则父节点必须是 $p/(q-p)$。因此从任意给定的最简分数回退到根的路径是唯一的。而二叉树中每一条从根到某节点的路径对应唯一的自然数索引 $n$（把 $n$ 的二进制表示去掉首位 1 后，用 0/1 表示左右），所以每个最简正有理数只会出现一次（单射）。

结论。

由上述两步的满射与单射，集合 $\{r_n=\tfrac{a_{n-1}}{a_n}:n\ge1\}$ 恰好是所有正有理数的集合，并且每个正有理数以最简形式恰好出现一次。即

```math
\left\{ \frac{a_{n-1}}{a_n}:n\ge1\right\}=\left\{\text{所有正有理数}\right\},
```

这正是题中所要证明的结果（序列按 $n=1,2,3,\dots$ 的顺序给出 $1/1,1/2,2/1,1/3,3/2,\dots$ 这样的枚举）。

（本证明实际上把题中数列生成的分数序列识别为 Calkin–Wilf 二叉树，对应的父子关系由上面的代数递推给出；用欧几里得式的反向操作可证明存在且唯一。） ✔
