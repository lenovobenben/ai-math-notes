# 圆上的有理点个数

- Conversation ID: `692ffffe-4574-8325-bda8-25c4a4fdbf7f`

## Question

以无理点为圆心的圆周上能否有无限个有理点？ 如果不能，则最多有多少个？最少有多少个？（有理点是x,y均为有理数的点，否则则为无理点）

## Answer

结论先说一句话：
不能有无限多个。至多 2 个；最少 0 个。

下面给出清楚的证明与构造例子。

### 证明 — 不可能有无限多个（且最多 2 个）

设圆 $C$ 的圆心为 $(a,b)$（其中 $a,b$ 中至少有一个是无理数），圆上若有一个点 $(x,y)$ 称为“有理点”当且仅当 $x,y$ 都是有理数。

关键事实（常用且易证明）：

若一个圆上有 三个互不共线的有理点，则这个圆的圆心坐标必定是有理数。

证明要点：设三个有理点为 $P_1,P_2,P_3$。圆心是这三点两条弦中点的垂直平分线的交点。垂直平分线由两个有理点确定，其解析式（斜率与截距）都是有理数，所以两条这样的直线交点（圆心）必为有理坐标。因此圆心为有理点。
（所以如果圆心是无理点，则不可能存在三个有理点。）

由此可得：若圆心无理，则圆上至多有 2 个有理点（因为 3 个会强制圆心为有理，与“圆心无理”矛盾）。因此不可能有无限多个。

### 可以达到的极值与构造

最多：2 个。
上面已证明不能有 \ge3 个，所以最大是 2。下面给出能取得 2 的构造（圆心无理，圆上有两个有理点）。

例子：取两个有理点 $P_1=(0,0)$、$P_2=(1,0)$。这两点的垂直平分线是竖直线 $x=\tfrac12$。在这条直线上任选一个无理的点作为圆心即可，例如取圆心

```math
O=\Big(\tfrac12,\ \sqrt2\Big).
```

以 $O$ 为心、到 $P_1$ 的距离为半径构成的圆，通过计算可见 $P_1,P_2$ 都在该圆上，且若再有第三个有理点则导致圆心为有理，与 $O$ 无理矛盾。因此该圆上恰有两个有理点。

可以只有 1 个有理点。
例子：取圆心 $O=(\sqrt2,0)$，半径 $r=\sqrt2$。圆的方程为

```math
(x-\sqrt2)^2+y^2=2.
```

检查：点 $(0,0)$ 在圆上且是有理点。若存在另一有理点 $(x,y)$（$x\neq 0$ 或 $y\neq 0$），则由方程

```math
x^2+y^2=2\sqrt2\, x
```

得到 $\sqrt2=\dfrac{x^2+y^2}{2x}$（当 $x\neq0$），右端为有理数，这与 $\sqrt2$ 无理矛盾；若 $x=0$ 则 $y^2=0$ 已是原来的点 $(0,0)$。所以该圆上 唯一的有理点 是 $(0,0)$。

可以没有有理点（0 个）。
例子：取圆心 $O=(\sqrt2,0)$，半径 $r=1$。圆方程

```math
(x-\sqrt2)^2+y^2=1
```

若存在有理点 $(x,y)$（且 $x\neq0$），同样可解出 $\sqrt2$ 为有理数（矛盾）；若 $x=0$ 则得到 $y^2+2-0=1\Rightarrow y^2=-1$ 无实解。所以该圆没有任何有理点。

### 小结

若圆心是无理点，则圆上的有理点不可能无限多；最多 2 个。

可取到的情况有 0 个、1 个、2 个（每种情况都有具体构造例子）。
