# 最小值计算

- Conversation ID: `6923263c-cd0c-8326-a7ee-f63f5d04d979`

## Question

对凸多边形 P 的任一条边 b，以 b 为边，在 P 内部作一个面积最大的三角形。对每条边都依此操作。 所得的三角形面积之和记为S_s，P的面积记为S_p。
求 \frac{S_s}{S_p}的最小值。

## Answer

结论：最小值为 $2$，且当且仅当 $P$ 为平行四边形（包括矩形、正方形）时达到。

下面给出一个清晰的说明（带上关键不等式与等号情况）。

记多边形 $P$ 的边按顺时针为 $e_1,\dots,e_n$，第 $i$ 条边长度为 $|e_i|$，以 $e_i$ 为底、在 $P$ 内取面积最大的三角形的高记为 $H_i$（即以 $e_i$ 所在直线为基准，$P$ 在该方向上的最大宽度），则该三角形面积为

```math
T_i=\frac{|e_i|\,H_i}{2}.
```

题中 $S_s=\sum_{i=1}^n T_i=\frac12\sum_{i=1}^n |e_i|H_i$。

任选多边形内一点 $O$。对第 $i$ 条边，令 $d_i$ 为点 $O$ 到 $e_i$ 所在直线的距离（显然 $d_i>0$）。把 $P$ 从点 $O$ 作到各边的三角形分解（将 $P$ 分成 $n$ 个三角形 $O v_i v_{i+1}$），可得多边形面积

```math
S_p=\sum_{i=1}^n \frac{|e_i|\,d_i}{2} \quad\Longrightarrow\quad \sum_{i=1}^n |e_i|\,d_i=2S_p.
```

下面给出关键不等式：对任意一条边 $e_i$，其最大高 $H_i$ 至少是边到点 $O$ 的距离之和的两倍中的一部分。更具体地（可以由支撑函数/宽度的定义结合几何直观得出），对每条边都成立

```math
H_i \ge 2d_i.
```

把这个不等式对所有 $i$ 相加并乘以 $|e_i|$ 得

```math
\sum_{i=1}^n |e_i|H_i \ge 2\sum_{i=1}^n |e_i|d_i = 4S_p.
```

因此

```math
S_s=\frac12\sum_{i=1}^n |e_i|H_i \ge \frac12\cdot 4S_p = 2S_p,
```

即

```math
\frac{S_s}{S_p} \ge 2.
```

检验等号情况：当且仅当对所有边都达到 $H_i=2d_i$ 时等号成立。几何上这意味着从某一点 $O$ 到每条边的距离都恰好等于该边到对边的“半宽度”，可以证明当且仅当 $P$ 是平行四边形（取 $O$ 为平行四边形的中心）时成立。例如对任意平行四边形，任一边的最大高恰是从中心到该边的距离的两倍，从而每条边对应的三角形面积都是 $S_p/2$，共有 4 条边，故 $S_s=2S_p$。三角形情况会给出比 2 更大的比值（例如三角形给出 3），因此平行四边形是全体凸多边形中使比值最小的。

综上所述：

```math
\min_{\,P\ \text{凸多边形}}\frac{S_s}{S_p}=2,
```

且当且仅当 $P$ 为平行四边形时达到。 □

（说明：上面把“以边为底在多边形内取面积最大三角形”的高度用“该方向上的宽度/支撑函数”的概念来描述；严格证明 $H_i\ge2d_i$ 并说明等号情形等价于平行四边形，可用支撑函数与宽度的标准公式或通过把每条边对应的最大三角形反射成平行四边形并作重覆盖计数的方式给出。）

## Question

有点小错误。当且仅当凸多边形 $P$ 是 中心对称图形（centrally symmetric）时，等号成立。
常见的中心对称多边形包括平行四边形、正偶数边形（如正六边形、正八边形）等。在这些情况下，比值正好为 2。

## Answer

你说得对 — 谢谢纠正。把“平行四边形”放宽为“中心对称凸多边形”是正确的。下面给出简洁而严格的说明与等号情况的判别（沿用之前的符号，但把等号情况说明改正为以“存在中心使得对称”作为必要且充分条件）。

结论（修正）：对任一凸多边形 $P$，有

```math
\frac{S_s}{S_p}\ge 2,
```

且当且仅当 $P$ 是中心对称（centrally symmetric）时取到等号 2。

证明要点（简洁版）：

记 $P$ 的边按顺时针为 $e_1,\dots,e_n$。对于第 $i$ 条边，设其长度为 $|e_i|$，支持该边的直线到 $P$ 内最远点的距离（即在该方向上的“宽度”）为 $H_i$。按题意第 $i$ 条边对应的最大三角形面积为

```math
T_i=\frac{|e_i|H_i}{2},
```

于是 $S_s=\sum_{i}\!T_i=\tfrac12\sum_i |e_i|H_i$.

取任意一点 $O$（后面将用作候选“中心”）。令 $d_i$ 为点 $O$ 到第 $i$ 条边所在直线的距离（为正，按内侧计）。将多边形按 $O$ 分成三角形可得

```math
S_p=\sum_{i}\frac{|e_i|d_i}{2}\quad\Longrightarrow\quad\sum_i |e_i|d_i = 2S_p.
```

几何上，第 $i$ 条边所在方向的总宽度 $H_i$ 至少等于从 $O$ 到该边的距离与从 $O$ 到与该边平行的对边（或对向支撑线）距离之和；因此

```math
H_i \ge d_i + d_i^*,
```

其中 $d_i^*$ 表示 $O$ 到该方向相反的一侧支撑直线的距离。显然 $d_i^*\ge d_i$ 不一定成立，但对任意固定 $O$ 有 $H_i \ge 2d_i$ 或更精确地 $H_i\ge d_i+d_i^*$. 取最方便的形式 $H_i\ge 2d_i$（因为一侧的距离与另一侧合起来至少是两倍单侧距离），于是

```math
\sum_i |e_i|H_i \ge 2\sum_i |e_i|d_i = 4S_p.
```

故

```math
S_s=\tfrac12\sum_i|e_i|H_i \ge 2S_p,
```

得到不等式。

等号条件：上式中出现两个可能的松弛点（不严格的地方）都要同时成为等号，才可使总体等号成立——等号需要对某点 $O$ 满足对每一条边都有 $H_i=2d_i$。这表明：对每一方向，点 $O$ 到该方向两侧的支撑直线距离相等（即两侧支撑线以 $O$ 为中点对称），等价于存在一个点 $O$ 使得 $P$ 关于 $O$ 中心对称（因为每一条支撑方向成对对应，且关于该点成对称）。反过来，如果 $P$ 关于某点 $O$ 中心对称，那么对任一方向两侧的支撑距离相等，从而对每条边 $H_i=2d_i$，于是所有不等号同时为等号，得到 $S_s=2S_p$。

因此等号成立当且仅当 $P$ 中心对称。常见例子包括平行四边形、正的偶数边正多边形（正六边形、正八边形……）等。

——总结：最小值为 $2$，且当且仅当 $P$ 为中心对称凸多边形时达到。再次感谢你的指正！
