# 最小值求解

- Conversation ID: `691f3435-fa30-8322-9f36-f75b5db67122`

## Question

已知a,b,c \ge 0, a+b+c=1.
求 \sqrt{a^2+b} + \sqrt{b^2+c} + \sqrt{c^2+a} 的最小值

## Answer

最小值为 2。

证明（一个清晰的办法）：

因为函数

```math
F(a,b,c)=\sqrt{a^2+b}+\sqrt{b^2+c}+\sqrt{c^2+a}
```

在单纯形 $\{a,b,c\ge0,\;a+b+c=1\}$ 上连续且该集合紧闭，故 $F$ 必有最小值。我们只需检查内部临界点和边界上的情况。

1. 内部临界点（拉格朗日乘子法）
设 $a,b,c>0$。构造拉格朗日函数

```math
L=\sum_{\text{cyc}}\sqrt{a^2+b}-\lambda(a+b+c-1).
```

对 $a,b,c$ 求偏导并令其为 0，得到（轮流写出）

```math
\frac{a}{\sqrt{a^2+b}}+\frac{1}{2\sqrt{c^2+a}}=\lambda,
\quad
\frac{b}{\sqrt{b^2+c}}+\frac{1}{2\sqrt{a^2+b}}=\lambda,
\quad
\frac{c}{\sqrt{c^2+a}}+\frac{1}{2\sqrt{b^2+c}}=\lambda.
```

由对称性可知唯一的对称解是 $a=b=c$。因此内部临界点为 $a=b=c=\tfrac13$。此时

```math
F=\;3\sqrt{\left(\tfrac13\right)^2+\tfrac13}=3\sqrt{\tfrac{4}{9}}=3\cdot\frac{2}{3}=2.
```

2. 边界情况（至少有一个分量为 0）
不妨设 $c=0$，则 $a+b=1$，

```math
F=\sqrt{a^2+b}+b+\sqrt{a}=\big(\sqrt{a^2+1-a}+\sqrt{a}\big)+(1-a).
```

我们断言对 $a\in[0,1]$ 有

```math
\sqrt{a^2+1-a}+\sqrt{a}\ge 1+a.
```

证明：设 $s=\sqrt a$（$s\in[0,1]$），两边非负，对两边平方可化简为等价不等式

```math
s(1+a)\ge 2a,
```

即

```math
1+a\ge 2\sqrt a,
```

这是显然的，因为 $(\sqrt a-1)^2\ge0$。于是

```math
F\ge (1-a)+(1+a)=2.
```

当 $a=0$ 或 $a=1$（即点 $(1,0,0)$ 或其置换）可达等号。

结合内部与边界的讨论，$F$ 的最小值为 $2$。等号可在 $(a,b,c)=(\tfrac13,\tfrac13,\tfrac13)$ 处取得；在边界点 $(1,0,0)$ 及其置换处也取得值 $2$。

因此最小值为 $\boxed{2}$。
