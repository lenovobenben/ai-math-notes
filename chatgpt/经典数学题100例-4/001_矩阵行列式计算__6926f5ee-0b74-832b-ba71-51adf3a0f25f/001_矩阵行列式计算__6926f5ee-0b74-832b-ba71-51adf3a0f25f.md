# 矩阵行列式计算

- Conversation ID: `6926f5ee-0b74-832b-ba71-51adf3a0f25f`

## Question

在\mathbb{R}^3中，向量u,v,w的模长分别为1,2,2，两两夹角为60°，求 \lfloor \det (I + uu^T + vv^T + ww^T) \rfloor 的值。

## Answer

先把问题转成矩阵形式。令 $U=[\,u\ v\ w\,]$（把三向量作为列），则

```math
A=uu^{T}+vv^{T}+ww^{T}=U U^{T}.
```

由矩阵行列式引理（或事实 $\det(I+UU^{T})=\det(I+U^{T}U)$），

```math
\det\big(I+uu^{T}+vv^{T}+ww^{T}\big)=\det\big(I+U^{T}U\big),
```

而 $U^{T}U$ 是向量 $u,v,w$ 的 Gram 矩阵，其分量为点积 $G_{ij}=x_i\cdot x_j$。

由题意 $|u|=1,\ |v|=2,\ |w|=2$，两两夹角为 $60^\circ$，所以点积为

```math
u\cdot u=1,\quad v\cdot v=4,\quad w\cdot w=4,
```

```math
u\cdot v=1,\quad u\cdot w=1,\quad v\cdot w=2.
```

因此

```math
G=U^{T}U=\begin{pmatrix}1&1&1\\[4pt]1&4&2\\[4pt]1&2&4\end{pmatrix},
\quad
I+G=\begin{pmatrix}2&1&1\\[4pt]1&5&2\\[4pt]1&2&5\end{pmatrix}.
```

计算行列式（直接展开或用标准计算工具）得

```math
\det(I+G)=36.
```

故题目要求的下取整为 $\lfloor 36\rfloor=36$。
