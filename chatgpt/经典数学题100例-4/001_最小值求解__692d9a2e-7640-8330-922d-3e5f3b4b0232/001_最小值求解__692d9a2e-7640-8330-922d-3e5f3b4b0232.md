# 最小值求解

- Conversation ID: `692d9a2e-7640-8330-922d-3e5f3b4b0232`

## Question

设p,q,r>0，且p+q+r=1。求p\log_{2}p + q\log_{2}q + r\log_{2}r的最小值。

## Answer

令函数 $f(p,q,r)=p\log_2 p+q\log_2 q+r\log_2 r$ ，约束 $p+q+r=1$，且 $p,q,r>0$。

方法一（拉格朗日乘数）：
构造拉格朗日函数

```math
\mathcal{L}=p\log_2 p+q\log_2 q+r\log_2 r+\lambda(p+q+r-1).
```

对 $p,q,r$ 求偏导并令其为 0（用自然对数换底：$\log_2 x=\frac{\ln x}{\ln 2}$），得

```math
\frac{\partial\mathcal{L}}{\partial p}=\frac{\ln p+1}{\ln 2}+\lambda=0,\quad
\frac{\ln q+1}{\ln 2}+\lambda=0,\quad
\frac{\ln r+1}{\ln 2}+\lambda=0.
```

由此得到 $\ln p=\ln q=\ln r$，所以 $p=q=r$。结合约束得

```math
p=q=r=\frac{1}{3}.
```

代回原式得最小值

```math
f_{\min}=3\cdot\frac{1}{3}\log_2\frac{1}{3}=\log_2\frac{1}{3}=-\log_2 3.
```

方法二（熵的观点或凸性/Jensen）：函数 $x\mapsto x\log_2 x$ 在 $(0,1]$ 上为凸函数，故在概率简单形上值在重心（即均匀分布）取得最小值，结论同上。

数值近似：$-\log_2 3\approx -1.58496$。

答：最小值为 $-\log_2 3$，在 $p=q=r=\tfrac{1}{3}$ 处取得。
