# 计算积分结果

- Conversation ID: `692c5945-2df4-8333-be2d-8ab288a13e36`

## Question

设g(x)=\int_{x}^{\pi}\frac{(\sin t)^5}{t}\mathrm{d}t，记S=\int_{0}^{\pi}g(x)\mathrm{d}x。则 \left \lfloor 100S \right \rfloor = ？

## Answer

把积分次序交换：

```math
S=\int_{0}^{\pi}\Big(\int_{x}^{\pi}\frac{(\sin t)^5}{t}\,dt\Big)dx
=\int_{0}^{\pi}\int_{0}^{t}dx\;\frac{(\sin t)^5}{t}\,dt
=\int_{0}^{\pi}(\sin t)^5\,dt.
```

再计算 $\displaystyle\int_{0}^{\pi}\sin^5t\,dt$。利用对称性和降幂公式，

```math
\int_{0}^{\pi}\sin^5t\,dt=2\int_{0}^{\pi/2}\sin^5t\,dt,
```

且递推关系给出

```math
\int_{0}^{\pi/2}\sin^5t\,dt=\frac{4}{5}\int_{0}^{\pi/2}\sin^3t\,dt
=\frac{4}{5}\cdot\frac{2}{3}\int_{0}^{\pi/2}\sin t\,dt
=\frac{4}{5}\cdot\frac{2}{3}\cdot1=\frac{8}{15}.
```

因此

```math
S=2\cdot\frac{8}{15}=\frac{16}{15}.
```

计算 $100S=\frac{1600}{15}=\frac{320}{3}=106\frac{2}{3}$，所以

```math
\lfloor 100S\rfloor=106.
```
