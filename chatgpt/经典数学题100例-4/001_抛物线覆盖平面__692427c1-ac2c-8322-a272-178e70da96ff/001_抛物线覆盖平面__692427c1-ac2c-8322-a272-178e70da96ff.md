# 抛物线覆盖平面

- Conversation ID: `692427c1-ac2c-8322-a272-178e70da96ff`

## Question

有限个抛物线的内部区域能否覆盖全平面？

## Answer

不行。有限个抛物线所限定的“内部”区域的并不能覆盖整个平面。下面给出一个简明而严格的说明（用极坐标做主线）。

把任一条抛物线看作二次不等式

```math
Q(x,y)=ax^2+bxy+cy^2+dx+ey+f\le0,
```

且抛物线的判别式满足 $b^2-4ac=0$（这是抛物线的特征），因此二次齐次部分是一个平方：

```math
a x^2+bxy+ c y^2=\lambda(\alpha x+\beta y)^2
```

对某些常数 $\lambda,\alpha,\beta$（不是同时为零）。

在极坐标 $x=r\cos\theta,\;y=r\sin\theta$ 下，

```math
Q(r\cos\theta,r\sin\theta)
= r^2\lambda(\alpha\cos\theta+\beta\sin\theta)^2
+ r\big(d\cos\theta+e\sin\theta\big)+f.
```

对于固定的角度 $\theta$，当 $r\to\infty$ 时主项为 $r^2\lambda(\alpha\cos\theta+\beta\sin\theta)^2$。

若 $\alpha\cos\theta+\beta\sin\theta\neq0$，则主项为正（或负，视 $\lambda$ 符号而定）；不论如何 $r^2$ 项的绝对值会支配其它项，使得当 $r$ 足够大时 $Q(r\cos\theta,r\sin\theta)>0$（即该方向上的远处点不在不等式成立的一侧）。只有满足

```math
\alpha\cos\theta+\beta\sin\theta=0
```

的角度（至多两个角度）才有可能使 $r^2$ 项消失，从而允许在该角附近的很窄的角区间内存在被包含的远处点；并且这类角区间的角宽随 $r$ 增大按 $O(1/r)$ 缩小。

因此对于任意给定的一条抛物线，它在半径为 $r$ 的大圆周上占据的角度量最多是 $O(1/r)$；有限条抛物线的并占据的角度量也是 $O(1/r)$。当 $r$ 足够大时，$O(1/r)\ll 2\pi$，所以圆周上必然存在未被覆盖的角度 —— 即存在半径很大的一点不在这些抛物线内部的并集中。这说明有限个抛物线的内部并不能覆盖整个平面。

（直观上每条抛物线只“开口”朝向一个方向，远处只在很窄的方向扇区内能被包含，有限条合并后也不足以“包围”所有方向的远处。）
