# 圆弧期望方差

- Conversation ID: `692c7bc5-f6d0-8326-bea0-8a17a5db8933`

## Question

在单位圆x^2+y^2=1上按均匀分布随机取n个点(n \ge 2)，这n个点可以把单位圆分成n段圆弧。求包含点(1,0)的圆弧的长度的数学期望和方差。

## Answer

把单位圆周长记为 $2\pi$（单位弧长为弧度）。把圆上随机取的 $n$ 个点的弧长坐标归一化为在 $[0,1)$ 上的 $n$ 个独立均匀点。它们把圆周分成 $n$ 段，段长向量
$(L_1,\dots,L_n)$ 在单位和约束下服从 Dirichlet$(1,\dots,1)$（即在单纯形上均匀分布）。固定点 $0$（对应点 $(1,0)$），落在第 $i$ 段的概率等于该段长度 $L_i$，因此“包含 $0$ 的那一段长度”是对这些段长度的按长度加权（size-biased）抽样。

由可交换性可得该被挑中的段的归一化长度 $S\in[0,1]$ 的分布为

```math
f_S(s)=n\cdot s\cdot (n-1)(1-s)^{\,n-2}=n(n-1)\,s(1-s)^{n-2},
```

即 $S\sim\mathrm{Beta}(2,n-1)$。

所以实际弧长（以弧度计）为 $L=2\pi S$。由 Beta 分布的矩刻画，

```math
\mathbb{E}[S]=\frac{2}{n+1},\qquad
\mathrm{Var}(S)=\frac{2(n-1)}{(n+1)^2(n+2)}.
```

因此所求的弧长的数学期望和方差为

```math
\boxed{\mathbb{E}[L]=2\pi\cdot\frac{2}{n+1}=\frac{4\pi}{\,n+1\,}}
```

```math
\boxed{\mathrm{Var}(L)=(2\pi)^2\cdot\frac{2(n-1)}{(n+1)^2(n+2)}
=\frac{8\pi^2 (n-1)}{(n+1)^2(n+2)}.}
```

（检验：当 $n$ 很大时 $\mathbb{E}[L]\sim 4\pi/n$，与直觉一致。）
