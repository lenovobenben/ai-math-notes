## Question

平面几何的托勒密不等式是什么？如何证明？

## Answer

### 托勒密不等式（Ptolemy 不等式）及证明

结论（平面上任意四点）
设平面上任意四点为 $A,B,C,D$，记两点间的欧氏距离为例如 $|AB|$ 或写作 $AB$。则有

```math
AC\cdot BD \le AB\cdot CD + AD\cdot BC .
```

并且当且仅当四点共线或共圆（凸四边形时等号当且仅当四点共圆，即为托勒密定理的等号情形）时取等号。

下面给出两个常见而简洁的证明方法：代数（复数/向量）证明和圆内（托勒密定理）几何证明。

#### 一、复数（代数）证明（简洁而自然）

把平面视为复平面，把四点对应为复数 $a,b,c,d$。注意恒等式

```math
(a-c)(b-d)=(a-b)(c-d)+(a-d)(b-c).
```

（展开两边可直接验证相等。）对两边取模并利用三角不等式得

```math
|a-c|\,|b-d| = \big| (a-b)(c-d)+(a-d)(b-c)\big|
\le |a-b||c-d| + |a-d||b-c|.
```

把模换回点间距离，即得

```math
AC\cdot BD \le AB\cdot CD + AD\cdot BC,
```

这正是托勒密不等式。

等号成立当且仅当复数 $(a-b)(c-d)$ 与 $(a-d)(b-c)$ 在复平面上同向（即两向量共正实数倍），这等价于四点共线或共圆（这是复数表示与圆几何的常见判定），从而给出等号的判定条件（共线亦视作退化的“共圆”情形）。

#### 二、几何上的经典说明：托勒密定理（共圆时为等号）

若四点 $A,B,C,D$ 共圆且依次在圆上（形成凸四边形），托勒密定理说明

```math
AC\cdot BD = AB\cdot CD + AD\cdot BC.
```

常见证明用相似三角形：作对角线交点或做辅助弦，利用圆周角相等得到若干相似三角形，从而推出上面的等式。因篇幅不长，这里给出要点：

在圆上，连接四边形的两条对角线 $AC,BD$ 以及必要的弦，并利用对应角相等得到两个三角形相似；

由相似得到比例关系，消去系数后得上面的等式。

（读者若需要我可以把相似三角形的完整作图与逐步推导写出来。）

从托勒密定理到不等式的思路：任意四点，可令三点 $A,B,C$ 决定一个圆，把第四点 $D$ 投影或移动到该圆上得到 $D'$ 使得 $D'$ 与 $D$ 在与三点相关的构造上保持某些角度关系，然后应用托勒密定理对 $A,B,C,D'$ 得到等式，再用三角不等式（或射线/弦上的单调性）比较 $BD'$ 与 $BD$、$AD'$ 与 $AD$ 等，最终得到不等式。代数证明更短且统一处理了共圆与一般情形。

#### 小结与备注

复数证明短小精悍：把点看作复数后利用恒等式与三角不等式立刻得到结论；同时能直接看出等号条件（向量同向 \to 共圆或共线）。

共圆时等号为经典的托勒密定理。

托勒密不等式在几何不等式、圆锥坐标法、复变几何和凸几何中都非常有用。
