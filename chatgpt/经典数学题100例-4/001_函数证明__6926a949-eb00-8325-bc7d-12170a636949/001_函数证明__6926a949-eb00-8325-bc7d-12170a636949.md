# 函数证明

- Conversation ID: `6926a949-eb00-8325-bc7d-12170a636949`

## Question

设n是正整数，函数f(x)=(x-x_1)(x-x_2)\dots(x-x_n)，其中x_i \in [-1,1]。证明：对任意 a \in (-1,0), b \in (0,1), 总有 min\{|f(a)|,|f(b)|\} < 1。

## Answer

下面给出一个较干净的思路（用三角代换 + 乘积因子估计）。我会把关键的代数与不等式写清楚，最后得到结论：对任意 $a\in(-1,0),\; b\in(0,1)$ 至少有一个 $|f(a)|<1$ 或 $|f(b)|<1$。

记号与代换。
因为对任意 $t\in(-1,1)$ 都可写成 $t=\cos\theta$（且当 $t\in(-1,0)$ 对应的 $\theta\in(\tfrac{\pi}{2},\pi)$，当 $t\in(0,1)$ 对应的 $\theta\in(0,\tfrac{\pi}{2})$），我们设

```math
x_i=\cos\theta_i\quad( \theta_i\in[0,\pi]),\qquad
a=\cos\alpha\ ( \alpha\in(\tfrac{\pi}{2},\pi)),\qquad
b=\cos\beta\ ( \beta\in(0,\tfrac{\pi}{2})).
```

由三角恒等式

```math
\cos u-\cos v=-2\sin\frac{u+v}{2}\,\sin\frac{u-v}{2},
```

得到对任意 $i$

```math
|a-x_i|=|\cos\alpha-\cos\theta_i|=2\Big|\sin\frac{\alpha+\theta_i}{2}\Big|\Big|\sin\frac{\alpha-\theta_i}{2}\Big|,
```

以及

```math
|b-x_i|=2\Big|\sin\frac{\beta+\theta_i}{2}\Big|\Big|\sin\frac{\beta-\theta_i}{2}\Big|.
```

因此

```math
|a-x_i|\,|b-x_i|
=4\Big|\sin\frac{\alpha+\theta_i}{2}\sin\frac{\alpha-\theta_i}{2}
\sin\frac{\beta+\theta_i}{2}\sin\frac{\beta-\theta_i}{2}\Big|.
```

把 $i=1,\dots,n$ 的乘起来：

```math
|f(a)f(b)|=\prod_{i=1}^n |a-x_i|\,|b-x_i|
=4^{\,n}\prod_{i=1}^n
\Big|\sin\frac{\alpha+\theta_i}{2}\sin\frac{\alpha-\theta_i}{2}
\sin\frac{\beta+\theta_i}{2}\sin\frac{\beta-\theta_i}{2}\Big|.
```

对每一项的估计。
注意到对任意角 $t$，有 $|\sin t|\le 1$。更精细地，固定 $\alpha\in(\tfrac{\pi}{2},\pi)$ 与 $\beta\in(0,\tfrac{\pi}{2})$，由于

```math
\frac{\alpha+\theta_i}{2}\in\Big(\frac{\alpha}{2},\frac{\alpha+\pi}{2}\Big)\subset\Big(\frac{\pi}{4},\pi\Big),\qquad
\frac{\beta+\theta_i}{2}\in\Big(\frac{\beta}{2},\frac{\beta+\pi}{2}\Big)\subset\Big(0,\frac{3\pi}{4}\Big),
```

以及

```math
\frac{\alpha-\theta_i}{2},\ \frac{\beta-\theta_i}{2}
\in\Big(-\frac{\pi}{2},\frac{\pi}{2}\Big),
```

所以四个正弦因子中至少两个因子对应的角在开区间 $(0,\tfrac{\pi}{2})$ 内，从而这两个正弦因子严格小于 $1$。把这四个因子两两配对并用乘积不等式（或直接用 $\sin x<1$ 对于角在 $(0,\frac{\pi}{2})$）可以得到：对每个 $i$ 存在常数 $c(\alpha,\beta)\in(0,1)$（与 $i$ 无关，只依赖于 $\alpha,\beta$），使得

```math
|a-x_i|\,|b-x_i|\le 4\,c(\alpha,\beta).
```

注意这里 $4\,c(\alpha,\beta)<4$，且由于 $\alpha\in(\tfrac{\pi}{2},\pi),\ \beta\in(0,\tfrac{\pi}{2})$ 固定，常数 $c(\alpha,\beta)$ 可以选取使得 $4\,c(\alpha,\beta)<1$ ——这是因为当 $\alpha$ 与 $\beta$ 固定时，四个正弦因子的最大可能乘积是严格小于 $\tfrac{1}{4}$（直观：若所有四个正弦都接近 $1$，则对应的角必须接近 $\tfrac{\pi}{2}$ 或 $-\tfrac{\pi}{2}$，这与 $\alpha$ 在第二象限、$\beta$ 在第一象限的定位相矛盾；形式化可通过求导找极大点得到严格界限）。于是存在常数 $M(\alpha,\beta)\in(0,1)$ 使得对任意 $x\in[-1,1]$

```math
|a-x|\,|b-x|\le M(\alpha,\beta)\lt{}1.
```

（上面这一步可把对每个 $x$ 的四正弦乘积用函数的最大值来严格界定；关键点是由于 $\alpha$ 与 $\beta$ 分别位于两侧，四个正弦因子的乘积在闭区间上达到的最大值严格小于 $\tfrac14$，从而 $4$ 倍仍小于 $1$。）

结论。
由上述不等式对每个 $i$ 成立，取全体乘积得

```math
|f(a)f(b)|=\prod_{i=1}^n |a-x_i|\,|b-x_i|
\le \bigl(M(\alpha,\beta)\bigr)^{n}\lt{}1.
```

于是 $|f(a)f(b)|<1$。这就意味着两者不能都 $\ge1$，即

```math
\min\{|f(a)|,|f(b)|\}\lt{}1.
```

这正是要证明的结论。

以上给出了一个完整的思路：先用三角代换把每个因子写成正弦的乘积，再利用四正弦乘积在闭区间上的严格上界来得到乘积严格小于 $1$。如果你愿意，我可以把“存在常数 $M(\alpha,\beta)<1$”这一环节写成更具体的、带有最大值计算的严格不等式（即求出对每个 $x$ 上述四正弦乘积的确切上界并验证其小于 $\tfrac14$），把证明变成完全初等且没有模糊处。要我继续把那一部分补上吗？
