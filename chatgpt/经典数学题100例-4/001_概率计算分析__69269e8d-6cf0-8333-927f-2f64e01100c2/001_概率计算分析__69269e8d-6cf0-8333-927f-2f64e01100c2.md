# 概率计算分析

- Conversation ID: `69269e8d-6cf0-8333-927f-2f64e01100c2`

## Question

在区间(0,1)中按均匀分布随机取一个数x，\lfloor \log_{2}{x} \rfloor 为偶数的概率

## Answer

先把条件拆开看：

当 $\lfloor\log_2 x\rfloor=k$（$k\le-1$）时，等价于

```math
2^k \le x \lt{} 2^{k+1},
```

因此该事件的概率（区间长度）为 $2^{k+1}-2^k=2^k$。

要求 $\lfloor\log_2 x\rfloor$ 为偶数，即 $k=-2,-4,-6,\dots$。所以所求概率为

```math
\sum_{j=1}^{\infty} \Pr(\lfloor\log_2 x\rfloor=-2j)
=\sum_{j=1}^{\infty}2^{-2j}
=\sum_{j=1}^{\infty}\bigl(\tfrac14\bigr)^j.
```

这是首项为 $1/4$、公比为 $1/4$ 的等比级数，其和为

```math
\frac{1/4}{1-1/4}=\frac{1/4}{3/4}=\frac{1}{3}.
```

答案：$\boxed{\tfrac{1}{3}}$。
