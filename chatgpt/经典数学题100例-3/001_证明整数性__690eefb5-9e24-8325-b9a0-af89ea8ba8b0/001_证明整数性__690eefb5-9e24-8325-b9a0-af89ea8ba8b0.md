# 证明整数性

- Conversation ID: `690eefb5-9e24-8325-b9a0-af89ea8ba8b0`

## Question

证明\frac{n^5-n}{10}是整数

## Answer

要证明 $\dfrac{n^5-n}{10}$ 是整数，等价于证明 $n^5-n$ 能被 $10$ 整除。因为 $10=2\cdot5$，只需证明 $n^5-n$ 同时被 $2$ 和 $5$ 整除。

模 $2$：
若 $n$ 为任意整数，$n^5$ 与 $n$ 同奇偶性（因为 $5$ 为奇数），所以

```math
n^5-n\equiv 0\pmod2.
```

模 $5$：
考虑 $n$ 在模 $5$ 下的余数，只需检验 $n\equiv0,1,2,3,4\pmod5$ 的情况：

若 $n\equiv0\pmod5$，显然 $n^5-n\equiv0\pmod5$；

若 $n\not\equiv0\pmod5$，由费马小定理（Fermat）有 $n^4\equiv1\pmod5$，因此

```math
n^5-n=n(n^4-1)\equiv n(1-1)\equiv0\pmod5.
```

所以对所有整数 $n$，都有 $n^5-n\equiv0\pmod5$。

既然 $n^5-n$ 同时被 $2$ 和 $5$ 整除，而 $\gcd(2,5)=1$，可得 $n^5-n$ 被 $10$ 整除。即

```math
\frac{n^5-n}{10}\in\mathbb{Z}.
```

（也可直接逐个列出模 $5$ 的 5 种情形验证 $n^5\equiv n\pmod5$，结论相同。）
