# 复数求解方法

- Conversation ID: `6909dbbe-31f8-832a-be66-137d0816c844`

## Question

已知a,b,c,d\in \mathbb{C} ，且满足条件：
```math
a + b + c + d = 2，
```
\frac{1}{a} + \frac{1}{b} + \frac{1}{c} + \frac{1}{d} = 2。
求解：\frac{1}{a-1} + \frac{1}{b-1} + \frac{1}{c-1} + \frac{1}{d-1}

## Answer

做法：把 $a,b,c,d$ 视为多项式

```math
P(x)=(x-a)(x-b)(x-c)(x-d)=x^4-e_1x^3+e_2x^2-e_3x+e_4,
```

其中 $e_1=a+b+c+d,\ e_2=\sum_{i<j}ab,\ e_3=\sum_{i<j<k}abc,\ e_4=abcd$。

已知 $e_1=2$。又

```math
\frac1a+\frac1b+\frac1c+\frac1d=\frac{e_3}{e_4}=2\implies e_3=2e_4.
```

利用对数导数恒等式

```math
\frac{P'(x)}{P(x)}=\sum_{t\in\{a,b,c,d\}}\frac{1}{x-t},
```

在 $x=1$ 处有

```math
\sum_{t}\frac{1}{1-t}=\frac{P'(1)}{P(1)},
```

所以

```math
\sum_{t}\frac{1}{t-1}=-\frac{P'(1)}{P(1)}.
```

计算 $P(1),P'(1)$：

```math
P(1)=1-e_1+e_2-e_3+e_4=1-2+e_2-2e_4+e_4=e_2-e_4-1,
```

```math
P'(1)=4-3e_1+2e_2-e_3=4-6+2e_2-2e_4=2(e_2-e_4-1).
```

于是

```math
\sum_{t}\frac{1}{t-1}=-\frac{2(e_2-e_4-1)}{e_2-e_4-1}=-2,
```

前提是分母不为 $0$（即没有根等于 $1$，否则各项中会出现 $\tfrac{1}{0}$ 的未定义项）。

结论：如果 $a,b,c,d\neq1$ 且满足题中条件，则

```math
\boxed{-2}.
```

（若有某个根等于 $1$，表达式不定义。）
