# 求解双射函数

- Conversation ID: `6909f6bb-f07c-8333-b695-9fa5a3624256`

## Question

严格单调递增双射函数f:\mathbb{R} \to \mathbb{R}。满足f(f(x))=f(x) + 12x。求f(x)

## Answer

答案：$\displaystyle f(x)=4x$。

证明（要点）：

先求不动点。令 $x=0$，由方程得

```math
f(f(0))=f(0).
```

设 $t=f(0)$。则 $f(t)=t$。把 $x=t$ 代回原方程得

```math
f(f(t))=f(t)+12t\Rightarrow t= t+12t\Rightarrow 12t=0\Rightarrow t=0.
```

所以 $f(0)=0$。

迭代与线性递推。记 $a_n=f^{(n)}(x)$（$a_0=x,\;a_1=f(x)$），则由方程

```math
a_{n+1}=a_n+12a_{n-1},\qquad n\ge1,
```

其特征方程为 $r^2-r-12=0$，根为 $r=4,\;r=-3$。因此一般解为

```math
a_n=\frac{f(x)+3x}{7}\,4^n+\frac{4x-f(x)}{7}\,(-3)^n.
```

利用严格单调性去掉振荡项。因为 $f$ 为严格单调双射且 $f(0)=0$，对任意 $x>0$ 有 $a_0=x>0\Rightarrow a_1=f(x)>0$。由递推可见对 $x>0$ 有

```math
a_{n+1}-a_n=12a_{n-1}\gt{}0,
```

故数列 $(a_n)$ 对任意 $x>0$ 是严格递增的（对 $x<0$ 则严格递减）。但若 $\dfrac{4x-f(x)}{7}\neq0$，则存在 $(-3)^n$ 项造成序列项在奇偶项间符号或大小振荡，这与单调性矛盾。因此必须有

```math
\frac{4x-f(x)}{7}=0\quad\text{对任意 }x,
```

即 $f(x)=4x$ 对所有实数成立。（同样 $x=0$ 情况已检验。）

验证：显然 $f(x)=4x$ 满足

```math
f(f(x))=4(4x)=16x=f(x)+12x.
```

因此唯一满足条件的严格单调递增双射是 $f(x)=4x$。 ∎
