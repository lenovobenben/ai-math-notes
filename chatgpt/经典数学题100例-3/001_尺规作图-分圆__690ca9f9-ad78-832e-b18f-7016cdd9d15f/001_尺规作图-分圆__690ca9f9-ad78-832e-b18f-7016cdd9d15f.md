# 尺规作图-分圆

## Question

尺规作图如何分圆为面积相等的n等分（不要求形状相同）？

## Answer

好问题 —— 如果不要求每一份形状相同，用尺规作图把一个圆划成面积相等的 $n$ 份，最简单、通用且可用直尺与圆规完成的方法是把圆沿半径方向分成同心圆环（annuli），使第 $k$ 个同心圆的半径满足面积累积比例。下面给出原理与可执行的尺规构造步骤。

#### 原理（非常短）

设原圆半径为 $R$。要把圆分成 $n$ 份面积相等，则应使第 $k$ 个子圆（半径为 $r_k$）的圆面积为原圆面积的 $\dfrac{k}{n}$：

```math
\pi r_k^2 = \frac{k}{n}\pi R^2 \quad\Rightarrow\quad r_k = R\sqrt{\frac{k}{n}}\qquad(k=1,\dots,n-1).
```

因此问题归结为：用尺规构出长度 $R\sqrt{k/n}$（对每个 $k$）。

#### 用尺规构造 $r_k=R\sqrt{k/n}$ 的具体步骤（可重复用于每个 $k$）

下面给出一种常见且标准的尺规方法：先用半圆法得到 $\sqrt{k n}$，再用相似三角形做乘除，得到 $\sqrt{k/n}$ 并乘以 $R$。

在一条直线上取一点 $A$。以任意单位长度刻出线段 $AB$ 长为 $n$（即把整数 $n$ 表示为长度；任选实际尺度，只要对所有构造一致）。在点 $B$ 右侧再在同一直线上记录点 $C$，使得 $BC=k$。于是 $AC=n+k$。

以 $AC$ 为直径画半圆。在点 $B$ 作垂直于 $AC$ 的直线，交半圆于点 $D$。根据直径上的几何关系，垂线段 $BD$ 的长度等于 $\sqrt{AB\cdot BC}=\sqrt{n\cdot k}$。

（这是经典的半圆与直角三角形的几何：若直径被分成 $a$ 与 $b$，对应的高是 $\sqrt{ab}$。）

现在得到长度 $BD=\sqrt{kn}$。于是

```math
\frac{BD}{n}=\sqrt{\frac{k}{n}}.
```

我们要得到 $r_k = R\cdot\sqrt{k/n} = R\cdot\frac{BD}{n}$。

用相似三角形做乘除：给定三段长度 $BD$、 $n$（记为分母）和 $R$，可以用相似三角形构出 $\dfrac{BD}{n}\cdot R$。（标准做法：画一条射线，从射线上一点按比例标出长度 $n$ 与 $BD$，连结与另一条通过 $R$ 的线，作平行线，得到按比例缩放的段——这是尺规可行的基本乘除构造。）——结果即为所需长度 $r_k$。

以圆心 $O$（原圆心）作半径 $r_k$ 的圆，即可得到第 $k$ 个同心圈。把原圆分成从 $r_{k-1}$ 到 $r_k$ 的同心圆环，每个环的面积为 $\dfrac{1}{n}$（首环是 $0$–$r_1$）。

重复上述对每个 $k=1,\dots,n-1$ 的构造，就把圆分成 $n$ 个面积相等的环（最内层为半径 $r_1$ 的圆盘）。

#### 说明与优点

该方法对任意正整数 $n$ 都适用（不受“能否作出 $360^\circ/n$”这种正多边形可构性限制）。

构造中只用到加减、相似三角形（乘除）与半圆求几何平均（开方），这些都是尺规允许的操作（开平方是允许的）。

得到的 $n$ 片是同心圆环（形状不相同，但面积相等），满足题目“不要求形状相同”的条件。

若你偏好“径向扇形（扇形）”而不是同心环：把圆分为 $n$ 个等面积扇形等价于把圆心角等分为 $n$ 份（每份角度 $360^\circ/n$）。能否用尺规直接构出角度 $360^\circ/n$ 取决于 $n$：仅当正 $n$-边形可作（构造数论条件）时才可行。因此对任意 $n$，同心圆环法更稳妥通用。

#### 小结（快速步骤回顾）

理论：需要 $r_k=R\sqrt{k/n}$。

用半圆法得到 $\sqrt{kn}$（把线段分为 $n$ 与 $k$，半圆高为 $\sqrt{kn}$）。

用相似三角形把 $\sqrt{kn}$ 除以 $n$、再乘以 $R$，得到 $r_k$。

以原心画半径 $r_k$ 的同心圆，分别得到面积为 $1/n$ 的 $n$ 份同心圆环。

## Answer

非常好！你给出的这种构造方法非常巧妙，而且确实能实现用尺规作图将圆分成面积相等的 $n$ 份，尽管各部分形状非常不同、呈现波浪形的花纹状。下面我给出这个方法的几何原理解释和为什么它能保证面积相等的分析。

#### ✅ 你的构造步骤（复述确认）

作圆，过圆心作直径 $AB$。

把直径 $AB$ 等分为 $n$ 份，等分点记为 $P_1, P_2, \dots, P_{n-1}$。

对每个等分点 $P_i$：

以 $A$ 和 $P_i$ 为直径端点，作上半圆弧。

以 $B$ 和 $P_i$ 为直径端点，作下半圆弧。

擦去直径 $AB$。

得到的区域（上、下半圆弧交错）刚好把整个圆分为 $n$ 个面积相等的部分（如你图中同色的部分）。

#### 📐 原理说明

这个构造其实利用了一个非常漂亮的面积等式：

对于以同一直线上的不同端点为直径端点所作的圆弧，它们在同一半圆中的“弓形”面积与直径的平方成正比。

更具体地说：

设原圆直径为 $AB = D$。
在直径上取点 $P_i$，以 $A P_i$ 为直径作半圆弧。
则对应“弓形”面积（上半部分的波浪区域面积）与 $(A P_i)^2$ 成比例。

因为你把 $AB$ 等分了，即每个段长都是 $D/n$，
于是：

```math
(A P_i)^2 \propto i^2,
```

从而相邻两弓形的面积差 $\propto (i^2 - (i-1)^2) = 2i - 1$，
这恰好使得上下两部分交错相加时，总面积均匀等分为 $n$ 份。
（直观上：上半圆的第 $i$ 个弧形部分与下半圆的第 $n-i$ 个弧形部分面积互补。）

#### 🎯 为什么面积相等

若把圆关于直径 $AB$ 对称，考虑从左往右的等分点 $P_i$，
每一对相邻弧 $P_{i-1}P_i$ 所围成的波浪区域（上弧加下弧的组合）刚好覆盖整个圆一次，且每对的面积相同。
这可以从几何上用“圆弓面积相减法”或积分思路看出（如果在解析几何里计算，用 $y=\sqrt{R^2-x^2}$ 表示上半圆，面积与 $x$ 的平方关系成立）。

因此，这种构造是一个完全正确、可尺规实现的“面积等分”法。
