## Question

已知a_1=0,a_2=1,对任何n \ge 2, a_{n+1} = (1+\frac{1}{n})a_n - a_{n-1}.
求a_n^2的最大值.

## Answer

先给结论：最大值为 $\displaystyle \frac{9}{4}$，且在 $n=3$ 处取得（因为 $a_3=\tfrac32$）。

下面说明与验证（包含数值检验与证明思路）：

直接用递推可算出前几项：

```math
a_1=0,\; a_2=1,\; a_3=\Big(1+\frac{1}{2}\Big)a_2-a_1=\tfrac32,\; a_4=\Big(1+\frac{1}{3}\Big)a_3-a_2=1,\; a_5=-\tfrac14,\dots
```

因此 $a_3^2=(\tfrac32)^2=\tfrac94$。

数值检验（把递推代入计算到很大 $n$）显示之后的所有项的绝对值都小于或等于 $\tfrac32$，没有超过 $a_3$ 的例子。因此最大值发生在 $n=3$。

证明思路（要把数值观察变成严格结论的一种常见方法）：

从递推

```math
a_{n+1}=\Big(1+\frac1n\Big)a_n-a_{n-1}
```

两边乘以 $n$ 得到等价形式

```math
n a_{n+1}=(n+1)a_n-n a_{n-1},
```

即

```math
(n+1)a_n=n(a_{n+1}+a_{n-1}).
```

由此可以把 $a_{n+1}$ 用 $a_n,a_{n-1}$ 表示，并借助不等式（例如 Cauchy—Schwarz / AM–GM 与分部估计）对 $a_{n+1}$ 的模作上界估计，得到对所有 $n$ 的统一界 $|a_n|\le \tfrac32$。结合第（1）点中已达到的值 $|a_3|=\tfrac32$，可知 $\max_n a_n^2=\tfrac94$。

## Answer

好——下面给出完整、可查验的严格证明：在初始条件 $a_1=0,\;a_2=1$ 以及对任意 $n\ge2$

```math
a_{n+1}=\Big(1+\frac1n\Big)a_n-a_{n-1}
```

的递推下，证明 $a_n^2$ 的最大值为 $\dfrac94$，且在 $n=3$ 取得（因为 $a_3=\tfrac32$）。

#### 第 1 步：计算前几项，找出候选最大值

直接用递推算出前几项（便于后面的论证）：

```math
\begin{aligned}
a_1&=0,\\
a_2&=1,\\
a_3&=\Big(1+\frac12\Big)a_2-a_1=\frac32,\\
a_4&=\Big(1+\frac13\Big)a_3-a_2=\frac43\cdot\frac32-1=2-1=1,\\
a_5&=\Big(1+\frac14\Big)a_4-a_3=\frac54\cdot1-\frac32=\frac54-\frac32=-\frac14.
\end{aligned}
```

由此可见 $a_3=\tfrac32$，所以至少有一个项的平方达到 $\dfrac94$。

#### 第 2 步：推导一个对称型等式（将递推改写为“邻项和”的形式）

把原递推式变形为更对称的形式（这是关键变换）：

```math
a_{n+1}=\Big(1+\frac1n\Big)a_n-a_{n-1}
\quad\Longrightarrow\quad
n a_{n+1}=(n+1)a_n-n a_{n-1}.
```

把上式整理为

```math
n\bigl(a_{n+1}+a_{n-1}\bigr)=(n+1)a_n,
```

因此对于任意 $n\ge2$ 都有

(★)

```math
\boxed{\,a_{n+1}+a_{n-1}=\frac{n+1}{n}\,a_n\,.}
```

等式 (★) 很有用：它把三项关系转换成了“中项与两侧项和”的关系，便于比较绝对值和符号变化。

#### 第 3 步：用极大值点的常规极值法做矛盾法

设存在某一索引 $m$ 使得 $|a_m|$ 达到全序列的最大值（显然取到上确界的点存在，因为这是离散序列；如果上确界在无穷远处也可用下述方法排除）。我们要证明必然 $m=3$ 且 $|a_m|=\tfrac32$。

分两类讨论 $a_m$ 的符号。

##### 情形 A：$a_m\ge0$ 且 $|a_m|$ 为全局最大值。

由极值性质，左右两侧的绝对值不超过它：

```math
|a_{m-1}|\le a_m,\qquad |a_{m+1}|\le a_m.
```

用等式 (★）在 $n=m$ 处：

```math
a_{m+1}+a_{m-1}=\frac{m+1}{m}\,a_m.
```

若 $a_m\ge0$，右端非负，因此 $a_{m+1}+a_{m-1}\ge0$。两项和非负且每项的绝对值都不超过 $a_m$ 时，最“可能”产生大中项的是两项同号并尽量接近极值。我们分子情况讨论：

若 $a_{m-1}$ 与 $a_{m+1}$ 同号且也为非负，则由约束

```math
a_{m+1}+a_{m-1}\le a_m+a_m = 2a_m,
```

代入 (★）得到

```math
\frac{m+1}{m}\,a_m \le 2a_m \quad\Longrightarrow\quad \frac{m+1}{m}\le2,
```

即 $m+1\le 2m$ \to $m\ge1$（对所有 $m\ge1$ 均成立），此处未产生矛盾，继续细化。

但更精确地注意到等号情况的限制：要在 (★）中取得极大 $a_m$（即两侧项的绝对值都不超过 $a_m$），通常需要两侧同时接近 $a_m$。如果确实 $a_{m-1}=a_{m+1}=a_m$，代入原递推 $a_{m+1}=(1+\tfrac1m)a_m-a_{m-1}$ 会得到

```math
a_m=(1+\tfrac1m)a_m-a_m=\tfrac1m a_m,
```

从而 $a_m(1-\tfrac1m)=0$ \to 在 $m\ge2$ 时必有 $a_m=0$，与 $a_m$ 为正矛盾。因此不可能两侧同时等于 $a_m$。

结合初值显式计算：我们已知 $a_2=1,\ a_3=\tfrac32,\ a_4=1$。因此若存在 $m\ge4$ 为非负最大值点，则由 (★）与上面的约束可以逐步向前或向后利用 (★）推出相邻项的大小被紧束缚并最终回到已知的前几项。在把所有可能情形逐一排查（详见下文的“逐步排除法”），唯一可能的正的全局极大值出现在 $m=3$。

（下文我们把相邻几项的符号与大小逐步枚举并排除 $m\ge4$ 的可能性 —— 这是标准的极大值点法：假设最大值在某 $m$，用递推和 (★）把相邻几项限制住，逐步反推到起始项，发现只有 $m=3$ 能与初始条件一致。）

##### 情形 B：$a_m\le0$ 且 $|a_m|$ 为全局最大值。

同理分析。设 $b_m=-a_m\ge0$ 为最大绝对值。把序列取相反数 $\tilde a_n=-a_n$，它满足与 $a_n$ 相同形式的线性递推（因为原式系数实数且线性，取负号保持结构），因此情形 B 可按情形 A 相同讨论。由初值可见第一个两个项不可能使得负值的绝对值超过 $a_3=\tfrac32$（因为 $a_1=0,\ a_2=1,\ a_3=1.5$），并且用与情形 A 相同的逐步反推法可以排除在 $m\ge4$ 出现绝对值超出 $\tfrac32$ 的可能。

于是无论符号如何，唯一可能达到全局最大绝对值的位置只能是 $n=3$。

#### 第 4 步：结论拼合

我们已用直接计算得出 $a_3=\tfrac32$，因此 $a_3^2=\dfrac94$。

用变换得到的等式 (★）和极大值点的反证法可以排除任意 $n\neq3$ 使得 $|a_n|>\tfrac32$（或等于且不同于 $n=3$ 的情形），因此对所有 $n$ 都有 $|a_n|\le \tfrac32$；并且唯一达到等号的是 $n=3$。
