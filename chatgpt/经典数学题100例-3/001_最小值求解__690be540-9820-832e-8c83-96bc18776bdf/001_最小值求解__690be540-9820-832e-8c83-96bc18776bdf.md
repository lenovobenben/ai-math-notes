# 最小值求解

- Conversation ID: `690be540-9820-832e-8c83-96bc18776bdf`

## Question

已知a,b为正实数，且满足ab(a+8b)=20。求a+3b的最小值

## Answer

用拉格朗日乘数法解（也可做替代法）。设 $a,b>0$，约束

```math
F(a,b)=ab(a+8b)-20=0,
```

要求最小化 $G(a,b)=a+3b$。

构造拉格朗日函数

```math
\mathcal L=a+3b+\lambda\big(ab(a+8b)-20\big).
```

对 $a,b$ 求偏导并令其为 0：

```math
\begin{cases}
\partial\mathcal L/\partial a=1+\lambda\big(2ab+8b^2\big)=0,\\[4pt]
\partial\mathcal L/\partial b=3+\lambda\big(a^2+16ab\big)=0.
\end{cases}
```

由两式消去 $\lambda$：

```math
\frac{1}{2b(a+4b)}=\frac{3}{a^2+16ab}.
```

化简得

```math
a^2+10ab-24b^2=0.
```

令 $t=\dfrac a b$，则

```math
t^2+10t-24=0\quad\Rightarrow\quad t=2\ (\text{舍去负根}).
```

所以 $a=2b$。代入约束

```math
ab(a+8b)=2b\cdot b\cdot(10b)=20b^3=20\implies b^3=1\implies b=1,
```

于是 $a=2$。此时

```math
a+3b=2+3\cdot1=5.
```

检查可知这是极小值。

答案：最小值为 $\boxed{5}$，当且仅当 $(a,b)=(2,1)$ 时取得。
