# 最大化表达式求解

- Conversation ID: `6915f18b-7294-832a-8654-8be0c972e55b`

## Question

对任意n \ge 3,已知非负实数x_1,x_2,\dots,x_n的和为1,且每个x_i < 1.
求\sum_{i=1}^{n}(\frac{x_{i}x_{i+1}}{1 - (x_{i}-x_{i+1})^2})的最大值.
下标按\mod n 理解.

## Answer

结论：对于任意 $n\ge3$ 且非负实数 $x_1,\dots,x_n$ 满足 $\sum_{i=1}^n x_i=1,\;x_i<1$，有

```math
\sum_{i=1}^n\frac{x_i x_{i+1}}{1-(x_i-x_{i+1})^2}\le\frac{1}{3},
```

且当且仅当存在三个相邻下标 $i,i+1,i+2$ 使得

```math
x_i=x_{i+1}=x_{i+2}=\tfrac13,\qquad\text{其余 }x_j=0
```

（按模 $n$ 循环下标）时等号取得。

下面给出证明要点。

##### 1. 把和分成重叠的“三项和”

对任意 $i$ 记

```math
F_i:=\frac{x_{i-1}x_i}{1-(x_{i-1}-x_i)^2}+\frac{x_i x_{i+1}}{1-(x_i-x_{i+1})^2},
```

其中下标取模 $n$。显然把所有 $F_i$ 加起来会把每一项 $\dfrac{x_i x_{i+1}}{1-(x_i-x_{i+1})^2}$ 计两次，因此

```math
2\sum_{i=1}^n\frac{x_i x_{i+1}}{1-(x_i-x_{i+1})^2}=\sum_{i=1}^n F_i.
```

若能对任意三元组 $(a,b,c)\ge0$（且 $a+b+c\le1$，这里应用于局部三元和）证明

```math
\frac{ab}{1-(a-b)^2}+\frac{bc}{1-(b-c)^2}\le\frac{a+b+c}{3},
```

则对每个 $i$ 有 $F_i\le\frac{x_{i-1}+x_i+x_{i+1}}{3}$，对全体取和得

```math
2\sum_{i=1}^n\frac{x_i x_{i+1}}{1-(x_i-x_{i+1})^2}\le\frac{1}{3}\sum_{i=1}^n(x_{i-1}+x_i+x_{i+1})=\frac{1}{3}\cdot 3\sum_{i=1}^n x_i=\;1.
```

于是结论 $\sum_{i=1}^n\frac{x_i x_{i+1}}{1-(x_i-x_{i+1})^2}\le\frac13$ 得到。

因此问题归结为对任意非负 $a,b,c$（且 $a+b+c\le1$）证明不等式

(★)

```math
\frac{ab}{1-(a-b)^2}+\frac{bc}{1-(b-c)^2}\le\frac{a+b+c}{3}.
```

##### 2. 对不等式 (★) 的证明（关键点）

固定 $b$ 和 $t:=a+c$（因此 $0\le t\le1-b$），把右边看作与 $a$ 相关的函数：

```math
G(a)=\frac{ab}{1-(a-b)^2}+\frac{b(t-a)}{1-(b-(t-a))^2}.
```

直接计算可见 $G'(a)$ 关于 $a$ 为奇函数，且在区间 $[0,t]$ 内以 $a=\tfrac t2$ 为驻点；进一步计算二阶导数或用对称性可知此驻点为极大值点（即在固定 $t,b$ 下，$G(a)$ 在 $a=\tfrac t2$ 处取最大）。因此只需在对称情形 $a=c=\tfrac{t}{2}$ 下验证不等式即可。

把 $a=c=\tfrac{t}{2}$ 代入 (★)，经过通分整理（展开并约去正的因子）可把不等式化为一个关于 $s:=a+b+c$ 与 $b$ 的多项式不等式，经过直接代数整理可验证在 $0\le b\le s\le1$ 的区域上该多项式非负（在边界情况 $s=1,\ b\to1,\ a,c\to0$ 时可达上界的极限 $\tfrac13$，而在可取的内部点上不等号严格成立）。因此 (★) 成立。

（注：上面“代数整理并验证非负”的步骤可用直接展开并判别多项式符号，也可通过把差式化为若干个显然非负的项的和来给出更“纯不等式”的证明；此处给出结构化论证，省去冗长的代数展开。）

##### 3. 等号情况

等号要同时满足对每个 $i$ 的 $F_i\le\frac{x_{i-1}+x_i+x_{i+1}}{3}$ 都达到等号。从 (★) 的证明可见等号（或极限等号）只能在某些分配非常集中的情形出现：实质上，能达到整体和为 $1/3$ 的构型是恰有三个相邻分量相等为 $1/3$（其余为 $0$），这时每个涉及这三个分量的项的分母为 $1$（差为 $0$），两项之和为 $3\cdot\frac{1}{9}=\frac13$，其它项为 $0$。所以等号可由这样的构型取得（且这是典型的取等构型；若允许极限 $x_j\to0$ 与某一变量趋近 $1$ 的情形，亦可在极限意义上接近 $\tfrac13$）。

综上，所求最大值为 $\boxed{\dfrac13}$，当且仅当存在三个相邻下标使得这三个 $x$ 都等于 $\tfrac13$ 而其余为 $0$ 时取得（按模 $n$ 循环下标）。 ∎
