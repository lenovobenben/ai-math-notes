# 内接三角形边长最小值

- Conversation ID: `690b8004-c1c4-8332-9f9b-f60174bbb9fa`

## Question

在直角\triangle ABC中，已知\angle A=60°，\angle C=90°，AC=4。求\triangle ABC的内接正三角形的边长的最小值。

## Answer

把题意坐标化并代数化解会最简洁。

设直角 $\triangle ABC$ 中点 $C=(0,0)$，令 $AC$ 在 $x$ 轴、$BC$ 在 $y$ 轴上。由已知 $\angle A=60^\circ,\angle C=90^\circ,AC=4$，可用正弦定理求得

```math
BC=4\sqrt3,\qquad AB=8.
```

于是

```math
A=(4,0),\; B=(0,4\sqrt3),\; C=(0,0)
```

记在边 $AC$ 上的一点为 $P=(t,0)$（$0\le t\le4$），在边 $BC$ 上的一点为 $Q=(0,u)$（$0\le u\le4\sqrt3$）。以 $P,Q$ 为两顶点作正三角形，第三个顶点有两种可能，计算可得（取使第三顶点落在斜边上的那种）第三顶点为

```math
R=\Big(\frac{t}{2}+\frac{\sqrt3}{2}u,\;\frac{u}{2}+\frac{\sqrt3}{2}t\Big).
```

要使 $R$ 落在斜边 $AB$（直线方程 $y=-\sqrt3 x+4\sqrt3$）上，代入得条件

```math
2u+\sqrt3\,t=4\sqrt3,
```

即

```math
u=\frac{\sqrt3}{2}(4-t).
```

正三角形的边长为

```math
s=\overline{PQ}=\sqrt{t^2+u^2}
=\sqrt{\,t^2+\frac{3}{4}(4-t)^2\, }.
```

令 $f(t)=s^2=\frac{7}{4}t^2-6t+12$。这是关于 $t$ 的凸二次函数，极小值处

```math
t=\frac{6}{2\cdot(7/4)}=\frac{12}{7}.
```

代回可得最小值

```math
s_{\min}=\sqrt{f\!\left(\tfrac{12}{7}\right)}=\sqrt{\frac{48}{7}}=\frac{4\sqrt3}{\sqrt7}=\frac{4\sqrt{21}}{7}.
```

数值近似为 $s_{\min}\approx2.618614$。

因此，内接正三角形边长的最小值为 $\boxed{\dfrac{4\sqrt{21}}{7}}$（约 $2.6186$）。
