# 最大化R与C比值

- Conversation ID: `6913453d-0388-8331-82be-bb29cc0f3246`

## Question

已知在 $2025 \times 2025$ 方格表的每个方格中写有一个非负实数，满足每一行的所有数之和为 1，且每一列的所有数之和为 1。记 $r_i$ 为第 $i$ 行中所有数的最大值，并记 $R = r_1 + r_2 + \cdots + r_{2025}$。类似地，记 $c_j$ 为第 $j$ 列中所有数的最大值，并记 $C = c_1 + c_2 + \cdots + c_{2025}$。

求 $\frac{R}{C}$ 的最大可能值。

## Answer

把问题推广到一般的 $n\times n$ 情形（这里 $n=2025$），设每行和每列之和都为 $1$。记第 $i$ 行的最大元为 $r_i$，第 $j$ 列的最大元为 $c_j$，有

```math
R=\sum_{i=1}^n r_i,\qquad C=\sum_{j=1}^n c_j.
```

下面给出上界并给出构造使该上界达到（当 $n$ 为完全平方数时）。

#### 上界（思路与证明）

把行按照“其行最大元所在的列”分成若干类。设恰好有 $m$ 列包含至少一个行的“行最大元”，对第 $j$ 个非空类记其行集合为 $S_j$，并记

```math
s_j=\sum_{i\in S_j} r_i,\qquad t_j=|S_j|.
```

因为第 $j$ 列的所有这些行在该列上的这些最大元都位于同一列，而这一列的所有条目之和是 $1$，所以

```math
s_j=\sum_{i\in S_j} r_i \le 1\qquad(\forall j).
```

并且显然 $\sum_{j=1}^m s_j=R$ 以及 $\sum_{j=1}^m t_j=n$。

对列的最大值 $c_j$ 至少为对应类中最大的一项，所以对于非空类有

```math
c_j \ge \max_{i\in S_j} r_i \ge \frac{s_j}{t_j},
```

而对于空类（没有行把它作为行最大元的列）来说，该列仍然必须有列和为 $1$，因此其列最大元至少为 $\frac1n$。于是

```math
C=\sum_{j=1}^n c_j \;\ge\; \sum_{j=1}^m \frac{s_j}{t_j} \;+\; (n-m)\cdot\frac1n .
```

固定 $m$ 与 $R=\sum s_j$ 时，由于函数 $x\mapsto 1/x$ 在正半轴上是凸函数（所以在 $t_j$ 等分时使得 $\sum s_j/t_j$ 最小），当各组大小尽量平均（即 $t_j\approx n/m$）且把 $R$ 均匀分配到这 $m$ 组时，可得下界的近简洁形式（严格证明可用凸性或交换不等式）：

```math
\sum_{j=1}^m \frac{s_j}{t_j} \ge \frac{mR}{n}.
```

因此

```math
C \ge \frac{n-m}{n} + \frac{mR}{n} = \frac{n + m(R-1)}{n}.
```

由分组定义显然 $R=\sum r_i\le m$（每组的 $s_j\le1$），于是为了使 $\dfrac{R}{C}$ 尽可能大，最优时可取 $m=\lceil R\rceil$。把上式代入并对 $m$ 优化，放宽为连续变量可求最优 $m\approx\sqrt{n}$。若 $n$ 恰好是完全平方数 $n=k^2$（本题 $2025=45^2$），把 $m=k$ 代入上界得到

```math
C \ge \frac{n-k}{n} + \frac{kR}{n} \quad\Longrightarrow\quad
\frac{R}{C}\le \frac{nR}{\,n + k(R-1)\,}\le \frac{nk}{\,n + k(k-1)\,}
=\frac{n}{\,2k-1\,}.
```

最后一步取 $R=k$（这是可行的——下面给出构造）得到上界

```math
\boxed{\displaystyle \frac{R}{C}\le \frac{n}{2k-1}=\frac{n}{2\sqrt n -1}.}
```

#### 构造（当 $n=k^2$ 时可达到上界）

令 $n=k^2$。把行分成 $k$ 组，每组恰有 $k$ 行；从 $n$ 列中选出 $k$ 列为特殊列。定义矩阵元素如下：

若列是某一组的对应特殊列，且该行属于该组，则在该列上写入 $1/k$；

所有非特殊列上，对每一行都写入相同的数 $1/k^2$；

其它情形（行不属于该组但列为该组的特殊列）写入 $0$。

检查可得：

每行：在其所属组对应的特殊列上有一项 $1/k$，其余 $n-k=k^2-k$ 个非特殊列上各有 $1/k^2$，总和为

```math
\frac{1}{k} + (k^2-k)\cdot\frac{1}{k^2}=\frac{1}{k}+\frac{k-1}{k}=1.
```

每列：任一特殊列来自恰好 $k$ 个同组的行，各项都是 $1/k$，列和为 $k\cdot\frac1k=1$。任一非特殊列从所有 $n$ 行每行得到 $1/k^2$，列和为 $n\cdot\frac{1}{k^2}=1$。

于是矩阵满足双守恒条件。此矩阵有

```math
r_i=\max(\tfrac{1}{k},\tfrac{1}{k^2})=\frac1k\quad(\forall i),
```

所以

```math
R=n\cdot\frac1k=k^2\cdot\frac1k=k.
```

列的最大元：$k$ 个特殊列的最大元是 $1/k$，其余 $n-k$ 个非特殊列的最大元是 $1/k^2$，因此

```math
C=k\cdot\frac{1}{k}+(n-k)\cdot\frac{1}{k^2}=1+\frac{n-k}{k^2}=2k-1.
```

于是

```math
\frac{R}{C}=\frac{k}{2k-1} \cdot k=\frac{k^2}{2k-1}=\frac{n}{2\sqrt n -1},
```

达到上界。

#### 回到题目数值

题中 $n=2025=45^2$，代入上式得到最大值

```math
\boxed{\frac{R}{C}_{\max}=\frac{2025}{2\cdot45-1}=\frac{2025}{89}.}
```

（上界证明与上面的构造合在一起说明该值是可达且为最大。）
