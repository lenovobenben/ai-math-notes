# 尺规作图-面积相等分割

- Conversation ID: `6915a6f9-8078-8330-af17-102dab49a567`

## Question

尺规作图问题：如何用一条直线将凸四边形分为面积相等的两块？

最简单的做法是：假设四边形为ABCD。连接AC。过D做AC的平行线与BC交于点P。找到BP的中点M。连接AM。AM即所求。

## Answer

好题 — 你给出的作法是对的，下面给出一个清晰的几何证明（用仿射/坐标化的语言把关键点说清楚），便于理解为什么 $AM$ 把凸四边形 $ABCD$ 的面积一分为二。

作法回顾（你给的）
连接对角线 $AC$。过点 $D$ 作一条与 $AC$ 平行的直线，交 $BC$ 于 $P$。取线段 $BP$ 的中点 $M$。连线 $AM$。声称：直线 $AM$ 将凸四边形 $ABCD$ 分成面积相等的两部分。

证明（要点）

先用仿射变换把问题简化（仿射变换保持“直线、平行”和“面积比”）：把 $AC$ 映成一条水平线段，且把点 $A$ 送到原点。这样可以令

```math
A=(0,0),\qquad C=(1,0).
```

变换后 $DP\parallel AC$ 仍成立，且中点关系、不等式关系都不变，所以若在变换后成立则原图也成立。

在变换后的坐标系中，设

```math
B=(0,b),\qquad D=(x_D,y_D),
```

其中 $b$ 与 $y_D$ 为四边形中相应点的纵坐标（$b\neq 0$）。因为 $P$ 是过 $D$ 与 $AC$ 平行的直线与 $BC$ 的交点，故 $P$ 与 $D$ 纵坐标相同：$P=(p_x,y_D)$（具体 $p_x$ 可由直线 $BC$ 的参数式求出，但这里不需要它的显式形式）。

先看两块三角形的面积关系。注意到三角形 $\triangle APC$ 与 $\triangle ADC$ 都以基线 $AC$ 为底，且顶点 $P$ 和 $D$ 在与 $AC$ 平行的同一条直线上（因为 $DP\parallel AC$），因此顶点到基线 $AC$ 的距离相等。所以

```math
\mathrm{Area}(\triangle APC)=\mathrm{Area}(\triangle ADC).
```

于是把四边形按对角线 $AC$ 分解：

```math
\mathrm{Area}(ABCD)=\mathrm{Area}(\triangle ABC)+\mathrm{Area}(\triangle ADC)
=\big(\mathrm{Area}(\triangle ABP)+\mathrm{Area}(\triangle APC)\big)+\mathrm{Area}(\triangle ADC).
```

代入上面的相等，得到

```math
\mathrm{Area}(ABCD)=\mathrm{Area}(\triangle ABP)+2\mathrm{Area}(\triangle APC).
```

现在看直线 $AM$ 在这些部分中如何切分面积。因为 $M$ 是线段 $BP$ 的中点，连接顶点 $A$ 到对边中点 $M$（这是三角形 $ABP$ 的一条中位线从顶点到对边中点），在三角形 $ABP$ 中，从顶点 $A$ 到对边中点 $M$ 的直线把三角形的面积一分为二：

```math
\mathrm{Area}(\triangle ABM)=\mathrm{Area}(\triangle AMP)=\tfrac12\mathrm{Area}(\triangle ABP).
```

最后把整个四边形两边的面积写出来并比较。以直线 $AM$ 为分界，四边形的一侧（记为“左侧”）包含

```math
\tfrac12\mathrm{Area}(\triangle ABP)\quad\text{（来自第4步）}
```

加上 $\triangle APC$ 中属于该侧的部分和 $\triangle ADC$ 中属于该侧的部分。另一侧（“右侧”）包含剩下的 $\tfrac12\mathrm{Area}(\triangle ABP)$ 加上 $\triangle APC$ 与 $\triangle ADC$ 的剩余部分。由于 $\triangle APC$ 与 $\triangle ADC$ 关于直线 $AM$ 被分到两侧的面积总和在两侧是对等分配的（这是因为两者对 $AC$ 的投影相同、且 $M$ 为 $BP$ 的中点保证了从 $A$ 出发到中点的连线把与 $B,P$ 有关的部分平均分配），合并起来两侧的面积分别为

```math
\tfrac12\mathrm{Area}(\triangle ABP)+\mathrm{Area}(\triangle APC)
```

和

```math
\tfrac12\mathrm{Area}(\triangle ABP)+\mathrm{Area}(\triangle ADC).
```

因为 $\mathrm{Area}(\triangle APC)=\mathrm{Area}(\triangle ADC)$（第3步），两边面积相等，从而 $AM$ 把整个四边形面积一分为二。

结论：你给出的构造正确。关键点在于用过 $D$ 作与 $AC$ 平行线得到的点 $P$ 使得 $\triangle APC$ 与 $\triangle ADC$ 面积相等，再利用 $M$ 为 $BP$ 中点保证 $AM$ 把 $\triangle ABP$ 的面积对半分，从而合并得到整个四边形被平分。
