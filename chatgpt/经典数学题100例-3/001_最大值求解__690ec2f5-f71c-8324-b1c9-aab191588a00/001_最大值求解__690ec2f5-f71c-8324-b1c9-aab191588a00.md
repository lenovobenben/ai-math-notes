# 最大值求解

- Conversation ID: `690ec2f5-f71c-8324-b1c9-aab191588a00`

## Question

正方形ABCD边长为4。以B为圆心作半径为2的圆。点P在圆上运动。求PA-PC的最大值，并求此时PD的长度。

## Answer

把方形 $ABCD$ 放在坐标系中：设

```math
A(0,0),\;B(4,0),\;C(4,4),\;D(0,4)
```

以 $B$ 为圆心、半径 $2$ 的圆上任意点记为

```math
P=(4+2\cos\theta,\;2\sin\theta)\quad(0\le\theta\lt{}2\pi).
```

记函数

```math
f(\theta)=PA-PC=\sqrt{(4+2\cos\theta)^2+(2\sin\theta)^2}
-\sqrt{(2\cos\theta)^2+(2\sin\theta-4)^2}.
```

对 $\theta$ 求极值并解方程 $f'(\theta)=0$，可得极大值对应的一组解为

```math
\cos\theta=\frac{\sqrt{8-2\sqrt7}}{4},\qquad
\sin\theta=\frac{1+\sqrt7}{4}.
```

把这个 $\theta$ 代回可以化简得到极大值

```math
\boxed{\,\max(PA-PC)=2\sqrt2\,.}
```

对应的点 $P$ 的精确坐标为

```math
P=\Bigl(4+\tfrac{1}{2}\sqrt{8-2\sqrt7},\;\tfrac{1+\sqrt7}{2}\Bigr).
```

此时到点 $D(0,4)$ 的距离为

```math
PD=2\sqrt{\,8-\sqrt7+\sqrt{8-2\sqrt7}\,}\approx5.2915009136.
```

总结答案：

最大值 $PA-PC=2\sqrt2$。

此时 $PD=2\sqrt{8-\sqrt7+\sqrt{8-2\sqrt7}}\approx5.2915$。

## Question

PD可以进一步化简为2\sqrt{7}

## Answer

你说得对，$PD$ 可以进一步化简为 $2\sqrt7$。下面给出简洁的代数证明。

我们仍取与之前相同的参数表示（见上一步推导）：

```math
P=(4+2\cos\theta,\;2\sin\theta),
```

并且已得到

```math
\cos\theta=\frac{\sqrt{8-2\sqrt7}}{4},\qquad
\sin\theta=\frac{1+\sqrt7}{4}.
```

先化简 $\cos\theta$：
注意

```math
(\sqrt7-1)^2=7-2\sqrt7+1=8-2\sqrt7,
```

因而 $\sqrt{8-2\sqrt7}=\sqrt7-1$（两边均为正），所以

```math
\cos\theta=\frac{\sqrt7-1}{4}.
```

计算 $PD^2$：

```math
\begin{aligned}
PD^2&=(4+2\cos\theta)^2+(2\sin\theta-4)^2\\
&=16+16\cos\theta+4\cos^2\theta+4\sin^2\theta-16\sin\theta+16\\
&=36+16(\cos\theta-\sin\theta)\quad(\cos^2\theta+\sin^2\theta=1).
\end{aligned}
```

代入 $\cos\theta=\dfrac{\sqrt7-1}{4},\ \sin\theta=\dfrac{1+\sqrt7}{4}$，得

```math
\cos\theta-\sin\theta=\frac{\sqrt7-1-(1+\sqrt7)}{4}=\frac{-2}{4}=-\frac12.
```

于是

```math
PD^2=36+16\Big(-\tfrac12\Big)=36-8=28,
```

所以

```math
\boxed{PD=\sqrt{28}=2\sqrt7}.
```

因此最终结果与您的结论一致：在 $PA-PC$ 取得最大值时，$PD=2\sqrt7$。
