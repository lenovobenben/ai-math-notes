# 求x四次方和

- Conversation ID: `690d80b0-ba78-832d-bc0b-a73f9a6fdd88`

## Question

已知\left\{\begin{matrix}
```math
x+y+z=1 \\
x^2+y^2+z^2=4 \\
x^3+y^3+z^3=9
```
\end{matrix}\right.
求x^4+y^4+z^4=?

## Answer

我们用牛顿恒等式（Newton's identities）把幂和与对称多项式联系起来。

设 $e_1=x+y+z,\; e_2=xy+yz+zx,\; e_3=xyz$，并记 $p_k=x^k+y^k+z^k$。已知

```math
p_1=e_1=1,\; p_2=4,\; p_3=9
```

牛顿恒等式给出（对 $k=2,3$）：

```math
p_2=e_1p_1-2e_2,\quad p_3=e_1p_2-e_2p_1+3e_3.
```

由第一个式子：

```math
4=p_2=e_1^2-2e_2=1^2-2e_2\implies -2e_2=3\implies e_2=-\tfrac32.
```

代入第二个式子：

```math
9=p_3=e_1p_2-e_2p_1+3e_3=1\cdot4-(-\tfrac32)\cdot1+3e_3
=4+\tfrac32+3e_3.
```

于是

```math
3e_3=9-5.5=3.5\implies e_3=\tfrac{7}{6}.
```

对 $k=4$（注意这里只有三元，所以 $e_4=0$），牛顿恒等式为

```math
p_4-e_1p_3+e_2p_2-e_3p_1=0
```

即

```math
p_4=e_1p_3-e_2p_2+e_3p_1.
```

代入数值：

```math
p_4=1\cdot9-(-\tfrac32)\cdot4+\tfrac{7}{6}\cdot1
=9+6+\tfrac{7}{6}=15+\tfrac{7}{6}=\tfrac{97}{6}.
```

答案：

```math
\displaystyle x^4+y^4+z^4=\frac{97}{6}.
```
