# 求和计算方法

- Conversation ID: `690f6ddd-f2e4-8326-bed5-eb7d1154734e`

## Question

求 \sum_{k=1}^{\infty } \frac{k^2}{2^k}

## Answer

我们要求的是

```math
S = \sum_{k=1}^{\infty} \frac{k^2}{2^k}.
```

##### 第一步：先回忆常用母函数

已知对 $|x| < 1$：

```math
\sum_{k=1}^{\infty} kx^{k} = \frac{x}{(1-x)^2}.
```

##### 第二步：对上式求导以得到 $k^2$ 项

对两边关于 $x$ 求导：

```math
\frac{d}{dx}\left(\sum_{k=1}^{\infty} kx^{k}\right)
= \sum_{k=1}^{\infty} k^2 x^{k-1}
= \frac{d}{dx}\left(\frac{x}{(1-x)^2}\right).
```

右边求导：

```math
\frac{d}{dx}\left(\frac{x}{(1-x)^2}\right)
= \frac{(1-x)^2 - x \cdot 2(1-x)(-1)}{(1-x)^4}
= \frac{(1-x)^2 + 2x(1-x)}{(1-x)^4}
= \frac{1 - 2x + x^2 + 2x - 2x^2}{(1-x)^4}
= \frac{1 - x^2}{(1-x)^4}.
```

所以：

```math
\sum_{k=1}^{\infty} k^2 x^{k-1} = \frac{1 - x^2}{(1-x)^4}.
```

乘上 $x$：

```math
\sum_{k=1}^{\infty} k^2 x^{k} = \frac{x(1 - x^2)}{(1-x)^4} = \frac{x(1+x)}{(1-x)^3}.
```

##### 第三步：代入 $x = \frac{1}{2}$

```math
S = \frac{\frac{1}{2}(1+\frac{1}{2})}{(1 - \frac{1}{2})^3}
= \frac{\frac{1}{2} \cdot \frac{3}{2}}{(\frac{1}{2})^3}
= \frac{\frac{3}{4}}{\frac{1}{8}}
= \frac{3}{4} \times 8 = 6.
```

✅ 最终答案：

```math
\boxed{6}.
```
