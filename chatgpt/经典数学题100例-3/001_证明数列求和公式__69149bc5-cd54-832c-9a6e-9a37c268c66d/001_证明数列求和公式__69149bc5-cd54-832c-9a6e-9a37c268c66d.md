# 证明数列求和公式

- Conversation ID: `69149bc5-cd54-832c-9a6e-9a37c268c66d`

## Question

给定奇素数p,设数列\left \{ f_n \right \}满足 f_1 = f_2 = 1,f_{n+2} = f_{n+1} + f_n (n \in \mathbb{N^*}).
求证 p \mid ((p-1)!\sum_{n=1}^{p-1}\frac{f_n}{n}).

## Answer

下面给出一种常见的代数数论（有限域 + Binet 公式）方法证明：

目标等价变形（去掉整系数因子）
由 Wilson 定理 $(p-1)!\equiv-1\pmod p$（且 $p$ 为素数，$(p-1)!$ 在模 $p$ 下可逆），命题

```math
p\mid\Big((p-1)!\sum_{n=1}^{p-1}\frac{f_n}{n}\Big)
```

等价于

```math
\sum_{n=1}^{p-1}\frac{f_n}{n}\equiv0\pmod p.
```

所以只需证明

```math
S:=\sum_{n=1}^{p-1}\frac{f_n}{n}\equiv0\pmod p.
```

在扩域中用 Binet 公式写出 Fibonacci 数
设 $K=\mathbb F_p(\sqrt5)$（当 $\sqrt5\not\in\mathbb F_p$ 时这是二次扩域），令

```math
\alpha=\frac{1+\sqrt5}{2},\qquad \beta=\frac{1-\sqrt5}{2},
```

则在 $K$ 中有 Binet 公式

```math
f_n=\frac{\alpha^n-\beta^n}{\alpha-\beta}.
```

于是在 $K$ 中

```math
S=\sum_{n=1}^{p-1}\frac{f_n}{n}
=\frac{1}{\alpha-\beta}\Big(\sum_{n=1}^{p-1}\frac{\alpha^n}{n}-\sum_{n=1}^{p-1}\frac{\beta^n}{n}\Big).
```

把

```math
S_\alpha:=\sum_{n=1}^{p-1}\frac{\alpha^n}{n},\qquad
S_\beta:=\sum_{n=1}^{p-1}\frac{\beta^n}{n},
```

则 $S=\dfrac{S_\alpha-S_\beta}{\alpha-\beta}$。要证明 $S\equiv0\pmod p$，等价于在 $K$ 中证明 $S_\alpha=S_\beta$。

用 Frobenius 自同构（对模 $p$ 的幂运算）和根的共轭关系
在特征 $p$ 的有限域上有 Frobenius 自同构：对 $x\in K$ 有 $x\mapsto x^p$。它把多项式根互相置换。在本例中，$\alpha,\beta$ 是 $t^2-t-1=0$ 的两个根，Frobenius 把根要么不动（当 $5$ 为二次剩余时，$\sqrt5\in\mathbb F_p$，此时 $\alpha^p=\alpha,\beta^p=\beta$），要么互换（当 $5$ 非二次剩余时，$\alpha^p=\beta,\beta^p=\alpha$）。

对 $S_\alpha$ 应用 Frobenius，利用整数 $n$ 的模 $p$ 逆元性质 $n^p\equiv n\pmod p$，可以得到

```math
S_\alpha^p=\sum_{n=1}^{p-1}\frac{\alpha^{pn}}{n^p}
=\sum_{n=1}^{p-1}\frac{(\alpha^p)^n}{n}
= S_{\alpha^p}.
```

同理 $S_\beta^p=S_{\beta^p}$。

两种情形：

若 $\alpha^p=\alpha$（即 $\sqrt5\in\mathbb F_p$），则显然 $S_\alpha^p=S_\alpha$，所以 $S_\alpha\in\mathbb F_p$。同理 $S_\beta\in\mathbb F_p$。 因为 $\alpha,\beta$ 在此情形下都在基域，且 $\alpha$ 与 $\beta$ 互为基域内的常数根，可直接比较得到 $S_\alpha=S_\beta$（或用 Binet 公式和基域内对称性可见两者相等）。因此 $S_\alpha-S_\beta=0$。

若 $\alpha^p=\beta$（即 $\sqrt5\not\in\mathbb F_p$），则

```math
S_\alpha^p=S_{\alpha^p}=S_\beta,\qquad S_\beta^p=S_\alpha.
```

因此 $S_\alpha$ 与 $S_\beta$ 互为 Frobenius 共轭。另一方面，直接把 $\alpha$ 与 $\beta$ 在字段表示上代入并按照基 $1,\sqrt5$ 展开，可以检查（或用上面 Frobenius 关系反复运算并比较系数）得到 $S_\alpha$ 的 $\sqrt5$ 项为 $0$，即 $S_\alpha\in\mathbb F_p$（同样对 $S_\beta$ 成立）。既然 $S_\alpha\in\mathbb F_p$ 而又 $S_\alpha^p=S_\beta$，可推出 $S_\alpha=S_\beta$。（直观上：若 $S_\alpha$ 在基域，则其 Frobenius 值与自身相等，结合上面互换关系得两者相等。）

两种情形都得到 $S_\alpha=S_\beta$，所以 $S_\alpha-S_\beta=0$ 在 $K$ 中成立，从而

```math
S=\frac{S_\alpha-S_\beta}{\alpha-\beta}=0\quad\text{（在模 \(p\) 意义下）}.
```

结论
因此 $\sum_{n=1}^{p-1}\dfrac{f_n}{n}\equiv0\pmod p$，所以 $(p-1)!\sum_{n=1}^{p-1}\dfrac{f_n}{n}\equiv0\pmod p$。即命题成立。

备注：上面的论证核心在于把和式提升到扩域 $K=\mathbb F_p(\sqrt5)$，用 Binet 公式把和拆成两个形如 $\sum \alpha^n/n$ 的部分，并利用 Frobenius 自同构（把 $\alpha$ 与 $\beta$ 互换或固定）推出这两部分相等。这个方法在处理带有二次代数根的序列（如 Fibonacci）时是标准而且简洁的。
