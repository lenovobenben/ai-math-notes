# 多项式根的估计

- Conversation ID: `6913d9df-d30c-8332-b318-67c0fad44eae`

## Question

设复系数多项式f(z)=z^n + a_{n-1}z^{n-1} + \dots + a_{1}z + a_0 的根为 z_1, \dots , z_n,
且对某个正整数k\le n 有 |z_1| \ge |z_2| \ge \dots \ge |z_k| > 1 \ge |z_{k+1}| \ge \dots \ge |z_n|.
求证：存在实数 x \in [-1,1] 使得 |f(x)| \ge \frac{1}{2^{n-1}} \prod_{i=1}^{k} |z_i|.

## Answer

下面给出一个干净的（势论 + 切比雪夫极值）证明。为方便书写记号，记
$f(x)=\displaystyle\prod_{i=1}^n (x-z_i)$（单项式系数为 $1$），并且按题中假设把模 $>1$ 的根写作 $z_1,\dots,z_k$。

##### 1. 取平衡测度并计算几何平均值

在区间 $[-1,1]$ 上考虑 切比雪夫/Arcsine 平衡测度

```math
\mathrm d\mu(x)=\frac{1}{\pi}\frac{\mathrm d x}{\sqrt{1-x^2}}\qquad (x\in(-1,1)),
```

这是 $[-1,1]$ 的对数势的平衡测度（容量 $\mathrm{cap}([-1,1])=\tfrac12$ 的标准测度）。

对任意复数 $z$ 有下列已知恒等（势论中的基本公式；可由对数势的显式计算得到）：

```math
\exp\Big( \int_{-1}^1 \log|x-z|\,\mathrm d\mu(x)\Big)=
\begin{cases}
\frac{|z|}{2},& |z|\gt{}1,\\[4pt]
\frac{1}{2},& |z|\le 1.
\end{cases}
```

（直观上：对区间的平衡测度，点 $z$ 在区间外时的势值等于 $\log(|z|/2)$，在区间内时等于 $\log(1/2)$。）

因此对多项式 $f$，

```math
\exp\Big( \int_{-1}^1 \log|f(x)|\,\mathrm d\mu(x)\Big)
=\prod_{i=1}^n \exp\Big( \int_{-1}^1 \log|x-z_i|\,\mathrm d\mu(x)\Big)
=\bigg(\frac{1}{2}\bigg)^n\prod_{i=1}^k |z_i| .
```

等价地

```math
\int_{-1}^1 \log|f(x)|\,\mathrm d\mu(x)
= -n\log 2+\sum_{i=1}^k\log|z_i|.
```

上式的左端是 $\log$ 下的 几何平均：令

```math
G:=\exp\Big( \int_{-1}^1 \log|f(x)|\,\mathrm d\mu(x)\Big)
=\frac{1}{2^{\,n}}\prod_{i=1}^k|z_i|.
```

##### 2. 从几何平均到最大值（用切比雪夫的极值性质）

我们要下界 $\max_{x\in[-1,1]}|f(x)|$。下面用切比雪夫多项式的极值性质（或等价的势论不等式）：

事实（切比雪夫极值性质）：在所有首项系数为 $c$ 的次数为 $n$ 的多项式中，$[-1,1]$ 上的最小可能的最大模（即 $\min\limits_{\deg=n,\;a_n=c}\max_{[-1,1]}|P|$）由切比雪夫多项式给出，其值为 $|c|\cdot 2^{\,1-n}$。对首项系数为 $1$ 的单项式（monic）而言，最小可能的 $\max_{[-1,1]}|P|$ 为 $2^{\,1-n}$，且该最小值由（适当缩放的）切比雪夫多项式达到。

把上面的势论结果与切比雪夫极值性质结合：令

```math
u(x):=\log|f(x)|.
```

由于切比雪夫多项式在极端情况下满足 $\max_{[-1,1]} u = \int u\,\mathrm d\mu +\log 2$（这是切比雪夫多项式作为 extremal 的特性在对数势形式下的表现——可检验等号在切比雪夫情况成立），而任意多项式的 $u$ 一般满足

```math
\max_{x\in[-1,1]} u(x)\ \ge\ \int_{-1}^1 u\,\mathrm d\mu(x)\ +\ \log 2.
```

（直观理解：切比雪夫多项式把 $\max$ 做到最小；因此任一多项式的 $\max$ 至少为切比雪夫对应值，即至少比其几何平均大一个因子 $2$ 的对数——详见切比雪夫极值定理或势论中的 Bernstein–Walsh/外域估计。）

对 $u=\log|f|$ 应用上式并取指数，得到

```math
\max_{x\in[-1,1]} |f(x)| \ \ge\ 2\cdot \exp\Big(\int_{-1}^1\log|f(x)|\,\mathrm d\mu(x)\Big)
=2\cdot G
=2\cdot\frac{1}{2^{\,n}}\prod_{i=1}^k|z_i|
=\frac{1}{2^{\,n-1}}\prod_{i=1}^k|z_i|.
```

##### 3. 结论

因此存在 $x\in[-1,1]$（即取使 $|f(x)|$ 达到区间最大值的点）使得

```math
|f(x)|\ge\frac{1}{2^{\,n-1}}\prod_{i=1}^k|z_i|,
```

这正是题目要求证明的结论。证毕。

备注： 上述证明的关键两步是（1）用 $[-1,1]$ 的平衡测度计算出 $|f|$ 的几何平均恰为 $\tfrac{1}{2^n}\prod_{|z_i|>1}|z_i|$；（2）用切比雪夫多项式的极值性质（或等价的 Bernstein–Walsh/势论不等式）把区间上的最大值下界提高一个因子 $2$。若需要，可把第二步用更“实分析”的方式展开成对切比雪夫多项式极值性质的具体推导。
