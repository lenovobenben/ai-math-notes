# 导函数不可积例子

- Conversation ID: `6911ed51-a9d4-8327-ab9c-4f0a10d67220`

## Question

存在原函数连续可导，其导函数却不可积的实函数吗？

## Answer

要看你说的“不可积”具体指什么。结论和例子很简单：

如果“不可积”是指在全体实数上不可积（即对 $\mathbb R$ 的不定积分不收敛 / 不属于 $L^1(\mathbb R)$），那么可以。举个最简单的例子：

```math
f(x)=\frac{1}{1+|x|},\qquad F(x)=\int_0^x\frac{1}{1+|t|}\,dt.
```

函数 $F$ 在 $\mathbb R$ 上处处可微且 $F'(x)=f(x)$，所以 $F\in C^1(\mathbb R)$。但

```math
\int_{-\infty}^{\infty}\frac{1}{1+|x|}\,dx=2\int_0^\infty\frac{1}{1+x}\,dx=\infty,
```

因此 $f\notin L^1(\mathbb R)$（全局不可积）。

如果“不可积”是指在某个闭区间上不可积（例如在 $[a,b]$ 上不可积，按黎曼意义），那不可能：若 $F\in C^1([a,b])$ 则 $F'$ 连续，连续函数在紧区间上必有界且黎曼可积。因此导函数在任意闭有界区间上都是可积的（既是黎曼可积也是勒贝格可积）。

所以归结一句话：导函数若是连续的，则在任意紧区间上可积；但它完全可以在整个实线上不可积（即 $\int_{\mathbb R}|F'(x)|\,dx=\infty$），上面的 $1/(1+|x|)$ 就是具体例子。需要我再举其它类型的例子（比如属于 $L^p$ 的/不属于某个 $L^p$ 的）吗？

## Question

这个呢？
f(x)=\left\{\begin{matrix}
x^2\sin \frac{1}{x}, x \ne 0  \\
```math
0,x = 0
```
\end{matrix}\right.

## Answer

很好——我们来仔细看这个例子。

定义

```math
f(x)=\begin{cases}
x^2\sin\frac{1}{x},&x\ne0,\\[4pt]
0,&x=0.
\end{cases}
```

可导性与 $f'(0)$。
对 $x\ne0$ 直接求导，

```math
f'(x)=2x\sin\frac{1}{x}+x^2\cos\frac{1}{x}\cdot\left(-\frac{1}{x^2}\right)
=2x\sin\frac{1}{x}-\cos\frac{1}{x}.
```

在 $0$ 点，差商

```math
\frac{f(h)-f(0)}{h}=\frac{h^2\sin(1/h)}{h}=h\sin(1/h)\to0\quad(h\to0),
```

因此 $f'(0)=0$。所以 $f$ 在全体 $\mathbb R$ 可导。

导数在 $0$ 处的连续性。
当 $x\to0$ 时，项 $2x\sin(1/x)\to0$ 而 $-\cos(1/x)$ 在 $0$ 附近不停振荡（无极限），故

```math
\lim_{x\to0}f'(x)\ \text{不存在}.
```

因此 $f'$ 在 $0$ 处不连续，$f$ 不是 $C^1$。

关于“可积性”（分不同含义）：

在任意有界闭区间（例如 $[-1,1]$）上，$f'$ 是有界的（因为 $|f'(x)|\le 2|x|+1$），并且仅在 $0$ 处可能不连续。有限不连续点的有界函数是黎曼可积的，因此 $f'$ 在任意紧区间上是（黎曼及勒贝格）可积的。

但在整个实线上，$f'(x)$ 并不衰减：当 $x\to\infty$，$\cos(1/x)\to1$，所以 $f'(x)\to-1$。因此 $|f'(x)|$ 在 $\mathbb R$ 上的积分（绝对积分）发散，即 $f'\notin L^1(\mathbb R)$。换句话说，作为整域上的可积函数（对 $\mathbb R$ 的绝对可积性/可积性）它不成立。

总结一句话：这个 $f$ 在全体 $\mathbb R$ 上可导且 $f'(0)=0$，但 $f'$ 在 0 处不连续（所以 $f\notin C^1$）。在任意紧区间上 $f'$ 是可积的，但在整个实线上 $f'$ 不是（不属于 $L^1(\mathbb R)$）。如果你想要一个满足“原函数 $C^1$（即导数连续）但导数在 $\mathbb R$ 上不可积”的例子，我可以再给出一个。需要吗？
