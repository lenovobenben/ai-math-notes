# 约瑟夫环定义解释

- Conversation ID: `6916a3b6-dd70-832f-bc90-24231b102a16`

## Question

编程算法中的约瑟夫环是什么？

## Answer

约瑟夫环（Josephus Problem） 是一个经典的编程与数学问题，常用于考察算法思维、数据结构（如链表）、数学推理及时间复杂度优化。

#### 🔹 问题定义（通俗版）

有 n 个人（编号 0～n−1）围成一个圈，从第 0 个人开始报数：

报到 第 k 个 的人被淘汰；

被淘汰的人出圈，圈变小；

从下一个人继续从 1 开始报数；

重复此过程，直到只剩下最后一个人，他的编号就是答案。

这就是约瑟夫环问题，也叫“约瑟夫问题（Josephus Problem）”。

#### 🔹 数学表达

给出 n 和 k，求最后留下的人的编号记作 f(n, k)。

递推公式如下（最常用于算法题）：

```math
f(1, k) = 0
```

```math
f(n, k) = ( f(n - 1, k) + k ) \bmod n
```

#### 🔹 示例

例如：n = 5，k = 3。

淘汰顺序是：

```
编号: 0 1 2 3 4
报数: 1 2 3(淘汰)
剩下: 0 1 3 4  （从 3 的下一个人 4 开始）
```

继续执行后最终剩下的编号会是 3。

#### 🔹 常见解法

##### 1. 模拟（循环链表）

使用数组或链表模拟循环报数；

时间复杂度 O(n·k)，不适合大 n、k。

##### 2. 数学递推法

使用上面的递推公式；

时间复杂度 O(n)，可以处理很大 n。

##### 3. 递推的迭代形式（最常见）

```python
def josephus(n, k):
    res = 0
    for i in range(2, n + 1):
        res = (res + k) % i
    return res
```
