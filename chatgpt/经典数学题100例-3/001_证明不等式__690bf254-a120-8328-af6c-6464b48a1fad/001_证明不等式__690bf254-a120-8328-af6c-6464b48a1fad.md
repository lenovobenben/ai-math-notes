# 证明不等式

- Conversation ID: `690bf254-a120-8328-af6c-6464b48a1fad`

## Question

某三角形的三条边分别是a,b,c，面积为S。证明：a^2+b^2+c^2 \ge 4\sqrt{3} S

## Answer

### 证明

设三角形的三个角为 $A,B,C$，对任意边有余弦定理与正弦关系，但这里用外接圆半径 $R$ 的表示更简洁：

```math
a=2R\sin A,\qquad b=2R\sin B,\qquad c=2R\sin C,
```

三角形面积也可写为

```math
S=2R^2\sin A\sin B\sin C.
```

因此不等式

```math
a^2+b^2+c^2 \ge 4\sqrt3\,S
```

等价于（两边除以 $4R^2>0$）

```math
\sin^2A+\sin^2B+\sin^2C \ge 2\sqrt3\,\sin A\sin B\sin C.
```

记

```math
x=\sin A\sin B\sin C\ge0.
```

由算术–几何平均不等式（AM–GM）有

```math
\sin^2A+\sin^2B+\sin^2C \ge 3(\sin^2A\sin^2B\sin^2C)^{1/3}
=3x^{2/3}.
```

于是若能证明对所有可能的 $x$ 有 $3x^{2/3}\ge 2\sqrt3\,x$，则原不等式成立。令 $t=x^{1/3}\ge0$，不等式变为

```math
3t^2 \ge 2\sqrt3\,t^3 \quad\Longleftrightarrow\quad t^2(3-2\sqrt3\,t)\ge0.
```

只需证明 $t\le \dfrac{\sqrt3}{2}$。因为在固定角和为 $\pi$ 的条件下，函数 $f(A,B,C)=\sin A\sin B\sin C$ 在等角（即 $A=B=C=\tfrac{\pi}{3}$）处取得最大值，所以

```math
x=\sin A\sin B\sin C \le \big(\sin\tfrac{\pi}{3}\big)^3=\Big(\frac{\sqrt3}{2}\Big)^3=\frac{3\sqrt3}{8}.
```

于是

```math
t=x^{1/3}\le\frac{\sqrt3}{2},
```

从而 $3-2\sqrt3\,t\ge0$，即 $3t^2\ge 2\sqrt3\,t^3$。连结上面的不等式链得到

```math
\sin^2A+\sin^2B+\sin^2C \ge 3x^{2/3} \ge 2\sqrt3\,x,
```

即

```math
\sin^2A+\sin^2B+\sin^2C \ge 2\sqrt3\,\sin A\sin B\sin C,
```

从而回代得原不等式成立：

```math
a^2+b^2+c^2 \ge 4\sqrt3\,S.
```

等号当且仅当三角形为等边三角形（$A=B=C=\tfrac{\pi}{3}$，即 $a=b=c$）时成立。 ∎
