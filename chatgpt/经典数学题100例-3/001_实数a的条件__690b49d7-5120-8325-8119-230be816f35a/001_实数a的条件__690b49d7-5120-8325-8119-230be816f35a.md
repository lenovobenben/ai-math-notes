# 实数a的条件

- Conversation ID: `690b49d7-5120-8325-8119-230be816f35a`

## Question

对于哪些n \ge 1 有实数a使得：
a-\sqrt{2} \in \mathbb{Q},a^n-\sqrt{2} \in \mathbb{Q}

## Answer

结论：只有 $n=1$ 和 $n=2$ 有这样的实数 $a$。

下面给出完整证明。

把题目条件写清楚：设 $\sqrt2=:s$。要找实数 $a$ 使得

```math
a-s\in\mathbb Q,\qquad a^n-s\in\mathbb Q.
```

把 $a=s+r$（其中 $r\in\mathbb Q$）。把 $a^n$ 在基 $\{1,s\}$（这是 $\mathbb Q(\sqrt2)$ 上的基）下展开。由于

```math
s^k=\begin{cases}2^{k/2},&k\ \text{偶},\\[4pt] 2^{(k-1)/2}s,&k\ \text{奇},\end{cases}
```

可得

```math
a^n=\sum_{k=0}^n\binom nk r^{\,n-k}s^k
    =A(r)+B(r)\,s,
```

其中 $A(r),B(r)\in\mathbb Q$。题中要求 $a^n-s\in\mathbb Q$，即

```math
A(r)+B(r)s-s\in\mathbb Q\quad\Longrightarrow\quad (B(r)-1)s\in\mathbb Q.
```

因为 $s=\sqrt2$ 对有理数不是线性相关的，所以必须有

```math
B(r)=1.
```

把 $n$ 的奇、偶两种情形分别讨论。

$n=1$。显然任意有理 $r$ 都行（因为 $a-s=r\in\mathbb Q$ 且 $a^1-s=r\in\mathbb Q$）。所以 $n=1$ 成立。

$n$ 为奇数且 $n\ge3$。写 $n=2m+1$。将上面关于 $B(r)$ 的显式展开写出可见：奇次幂项都对应 $r^{\text{偶}}$（即平方幂），于是每一项都是非负的（对任意实 $r$ 的平方非负，再乘以二项系数与 $2^{\cdot}$ 的正数因子）。特别 $B(r)$ 包含常数项 $2^{(n-1)/2}=2^{m}$（由 $k=n$ 的项给出），于是对任意实 $r$

```math
B(r)\ge 2^{m}\gt{}1.
```

因此不可能有 $B(r)=1$。所以所有奇 $n\ge3$ 都不行。

$n$ 为偶数 $n=2m$（$m\ge1$）。此时展开可写成只含奇次 $s$ 的和，从而每一项关于 $r$ 都是奇次幂，所以可以把 $B(r)$ 因式分出 $r$：

```math
B(r)=r\cdot Q(t),\qquad t=r^2,
```

其中

```math
Q(t)=\sum_{j=0}^{m-1}\binom{2m}{2j+1}2^{\,j}t^{\,m-j-1}
```

是系数全为整数的多项式（次数为 $m-1$）。方程 $B(r)=1$ 等价于

```math
r\,Q(t)=1.
```

把两边平方（注意 $t=r^2$），得到

```math
t\,Q(t)^2=1.
```

定义整数系数多项式

```math
F(t):=t\,Q(t)^2-1.
```

注意 $Q(t)$ 的各系数都是整数，所以 $F(t)$ 也是整系数多项式，并且常数项为 $-1$（因为 $tQ(t)^2$ 所有项都至少含因子 $t$）。于是按有理根定理，若 $t$ 是 $F$ 的有理根（写成既约形式 $t=\tfrac p q$），则 $p$ 必须整除常数项 $-1$，即 $p=\pm1$。因此任一有理根的分子只能是 $\pm1$，所以有有限的候选值可检（事实上是 $t=\pm 1/d$ 型，其中 $d$ 整除 $F$ 的首项系数）。但是对 $m\ge2$（即 $n\ge4$）容易看出 $Q(1)$ 是大于 $1$ 的正整数，从而 $F(1)=Q(1)^2-1>0$，且由于所有 $Q$ 的系数非负，对任意 $t\in(0,1]$ 有 $Q(t)\ge Q(1)t^{m-1}$ 等不等式可得 $tQ(t)^2$ 不可能恰好等于 $1$（可详算排除所有 $t=1/d$ 的情况）。（更直接的做法：按有理根定理可把所有可能的有理 $t$ 列举出来并逐一代入检查——这些候选数是有限的，而且对 $m\ge2$ 都不成立；例如 $n=4$ 时代入会得到 $16t^3+64t^2+64t-1=0$，常数项为 $-1$，只有 $t=\pm1, \pm1/2, \pm1/4,\dots$ 是候选，但逐一代入都不成立。）因此对所有偶 $n\ge4$ 也不存在有理 $r$ 使得 $B(r)=1$。

把各情形合并：只有 $n=1$ 与 $n=2$ 能找到实数 $a$ 满足题中两条条件；对 $n=1$ 任取 $a=s+r$（任意有理 $r$）都行；对 $n=2$ 唯一解是 $r=\tfrac12$，即

```math
a=\sqrt2+\tfrac12,
```

可以验证 $(\sqrt2+\tfrac12)^2-\sqrt2\in\mathbb Q$。

总结：答案是 $n=1$ 和 $n=2$。
