# 求解函数条件

- Conversation ID: `690d6447-8cb0-8321-9b98-e07d69401137`

## Question

求出所有函数 f: \mathbb{R^+} \to \mathbb{R^+}，使得对任意正数x都恰有一个正数y满足 xf(y) + yf(x) \le 2

## Answer

答案：唯一满足条件的函数是

```math
\boxed{f(x)=\dfrac{1}{x}\quad(\forall x\gt{}0).}
```

证明：

令关系 $R$ 在正实数集上定义为

```math
xRy\iff xf(y)+yf(x)\le2.
```

题意要求：对任意 $x>0$，集合 $\{y>0:\;xRy\}$ 恰有一个元素。于是对每个 $x$ 存在唯一的 $p(x)$ 使得 $xRp(x)$。由 $xRp(x)$ 当且仅当 $p(x)Rx$（因为不等式在 $x,y$ 之间对称），所以 $p$ 是一个无固定点以外的对合（involution）配对：若 $p(x)=y$ 则 $p(y)=x$。

（1）先证 $p(x)=x$ 对所有 $x$ 都成立。若存在 $x$ 使 $p(x)=y\ne x$，则

```math
xf(y)+yf(x)\le2.
```

但因为对 $x$ 来说唯一满足的元素是 $y$，所以对于 $t=x$（注意 $t\ne y$）必须有

```math
2x f(x)\gt{}2\quad\Rightarrow\quad f(x)\gt{}\frac{1}{x}.
```

同理对 $y$ 有 $f(y)>\frac{1}{y}$。由此

```math
xf(y)+yf(x)\gt{}\;x\cdot\frac{1}{y}+y\cdot\frac{1}{x}=\frac{x}{y}+\frac{y}{x}\ge2,
```

且当 $x\ne y$ 时严格 $>\,2$，这与 $xf(y)+yf(x)\le2$ 矛盾。故不可能有 $p(x)\ne x$，即对任意 $x$ 都有 $p(x)=x$。

（2）由 $p(x)=x$，代入不等式得

```math
2x f(x)\le2\quad\Rightarrow\quad f(x)\le\frac{1}{x}\qquad(\forall x\gt{}0).
```

（3）再证不能有严格不等式。假设存在 $a>0$ 使 $f(a)<\dfrac{1}{a}$。写 $c=f(a)$，则 $ac<1$。对于任意 $t>0$ 取 $y=ta$，由（2）对任意 $y$ 有 $f(y)\le\frac{1}{y}=\frac{1}{ta}$，于是

```math
a f(y)+y f(a)\le a\cdot\frac{1}{ta}+ta\cdot c=\frac{1}{t}+ac\,t=:h(t).
```

函数 $h(t)=\frac{1}{t}+ac\,t$ 在 $t=1/\sqrt{ac}$ 处取得最小值

```math
h_{\min}=2\sqrt{ac}\lt{}2
```

（因为 $ac<1$）。因此存在 $t\ne1$ 使 $h(t)<2$，从而对相应的 $y=ta$ 有

```math
a f(y)+y f(a)\le h(t)\lt{}2,
```

这与对 $a$ 的唯一性（即只有 $y=a$ 可使不等式成立）矛盾。故不存在 $a$ 使 $f(a)<\dfrac{1}{a}$。

由（2）和（3）得到对所有 $x>0$ 必有

```math
f(x)=\frac{1}{x}.
```

检验：显然当 $f(x)=1/x$ 时

```math
xf(y)+yf(x)=\frac{x}{y}+\frac{y}{x}\ge2,
```

且等号当且仅当 $y=x$ 成立，因此对于每个 $x$ 恰有唯一的 $y=x$ 满足不等式（即等号）。因此该函数满足题意。

综上，唯一解为 $f(x)=1/x$（对一切正实 $x$）。 □
