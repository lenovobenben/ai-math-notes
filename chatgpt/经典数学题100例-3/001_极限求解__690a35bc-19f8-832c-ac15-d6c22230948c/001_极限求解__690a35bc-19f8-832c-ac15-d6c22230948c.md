# 极限求解

- Conversation ID: `690a35bc-19f8-832c-ac15-d6c22230948c`

## Question

令x_{0} \in (0,\pi)，有递推关系x_{n+1}=\sin x_{n}。求 \lim_{n \to \infty} \sqrt{n} x_{n}

## Answer

因为 $x_0\in(0,\pi)$，且 $x_{n+1}=\sin x_n>0$，可见序列单调下降且收敛于 0。为求极限，作如下展开与变形。

对小的 $x$ 有泰勒展开

```math
\sin x = x - \frac{x^3}{6}+o(x^3)\quad (x\to0).
```

考虑

```math
\frac{1}{x_{n+1}^2}-\frac{1}{x_n^2}
=\frac{x_n^2-x_{n+1}^2}{x_{n+1}^2x_n^2}
=\frac{(x_n-\sin x_n)(x_n+\sin x_n)}{\sin^2 x_n\;x_n^2}.
```

令 $x=x_n$，代入上面的展开并取极限得到

```math
\lim_{n\to\infty}\Big(\frac{1}{x_{n+1}^2}-\frac{1}{x_n^2}\Big)
=\lim_{x\to0}\frac{(x-\sin x)(x+\sin x)}{\sin^2 x\;x^2}
=\frac{1}{3}.
```

由 Stolz–Cesàro 定理（或将上式对 $n$ 求和并归纳）可得

```math
\lim_{n\to\infty}\frac{1/x_n^2}{n}=\frac{1}{3},
```

即 $n x_n^2\to 3$。于是

```math
\lim_{n\to\infty}\sqrt{n}\,x_n=\sqrt{3}.
```

答：

```math
\displaystyle \lim_{n\to\infty}\sqrt{n}\,x_n=\sqrt{3}.
```
