# 非平凡解求法

- Conversation ID: `690da4d4-1708-832b-9ad9-c1df9dac2ffc`

## Question

求1²+2²+...+n²=m²非平凡正整数解(m,n)

## Answer

你问的是求满足

```math
1^2 + 2^2 + \dots + n^2 = m^2
```

的非平凡正整数解 $(m,n)$。这里“非平凡”通常是指 $n > 1$。

首先，公式：

```math
1^2 + 2^2 + \dots + n^2 = \frac{n(n+1)(2n+1)}{6}.
```

因此我们要求

```math
\frac{n(n+1)(2n+1)}{6} = m^2, \quad m,n \in \mathbb{Z}^+.
```

也就是

```math
n(n+1)(2n+1) = 6 m^2.
```

这属于完全立方数方程/迪奥方程问题。已知的非平凡正整数解只有很少几个。

已知的解为：

| n | m |
| --- | --- |
| 1 | 1 |
| 24 | 70 |

所以唯一的非平凡正整数解是：

```math
\boxed{(n,m) = (24, 70)}.
```

验证：

```math
1^2 + 2^2 + \dots + 24^2 = \frac{24 \cdot 25 \cdot 49}{6} = 4900 = 70^2.
```

✅ 完全正确。

## Question

请证明其唯一性

## Answer

好的，我们来严格讨论为什么

```math
1^2 + 2^2 + \dots + n^2 = m^2
```

的非平凡正整数解 $(n,m)$ 只有 $(24,70)$。

##### 第一步：重写方程

我们有公式：

```math
1^2 + 2^2 + \dots + n^2 = \frac{n(n+1)(2n+1)}{6}.
```

令它等于 $m^2$，得到迪奥方程：

```math
\frac{n(n+1)(2n+1)}{6} = m^2 \quad \Longleftrightarrow \quad n(n+1)(2n+1) = 6 m^2.
```

##### 第二步：因数分解与三因子互质性

考虑三连续整数因子：

```math
n, \quad n+1, \quad 2n+1.
```

```math
\gcd(n, n+1) = 1
```

$\gcd(n, 2n+1) = 1$ 或 1 或 3，但我们稍后讨论

$\gcd(n+1, 2n+1) = 1$ 或 1 或 3

所以 $n(n+1)(2n+1)$ 质因数分布相对“分散”，要等于 $6 m^2 = 2 \cdot 3 \cdot m^2$，说明每个因子几乎都是平方数（或者 2/3 的小因子被分配给 n、n+1、2n+1）。

这种类型的方程是Mordell 方程的特例，通常只有有限个整数解。

##### 第三步：改写为三次丢番图方程

令 $x = n$，$y = m$。方程：

```math
x(x+1)(2x+1) = 6y^2.
```

展开：

```math
2x^3 + 3x^2 + x - 6 y^2 = 0.
```

整理成标准形式：

```math
2x^3 + 3x^2 + x = 6 y^2.
```

或者：

```math
y^2 = \frac{1}{6}(2x^3 + 3x^2 + x) = \frac{x(2x^2 + 3x + 1)}{6} = \frac{x(2x+1)(x+1)}{6}.
```

这就是我们原来的方程。

##### 第四步：分析整数解

已知方法：通过三次丢番图方程或椭圆曲线理论。

方程 $y^2 = \frac{1}{6} x(x+1)(2x+1)$ 是一个椭圆曲线的整数解问题。

可以把它变换为 Weierstrass 形式。设：

```math
u = 2x+1 \implies x = \frac{u-1}{2}.
```

代入方程：

```math
y^2 = \frac{(u-1)/2 \cdot (u+1)/2 \cdot u}{6} = \frac{u(u^2-1)}{24} = \frac{u(u-1)(u+1)}{24}.
```

于是整数解问题变成：

```math
24 y^2 = u(u-1)(u+1) = u^3 - u.
```

即

```math
24 y^2 = u^3 - u \quad \Longleftrightarrow \quad u^3 - u - 24 y^2 = 0.
```

这是一个标准的Mordell 方程 $u^3 - u = 24 y^2$。

##### 第五步：已知结果

经典数论结果（Mordell, 1930s）表明：

```math
u^3 - u = k y^2
```

的整数解只有有限个。对于 $k=24$，唯一正整数解为：

```math
u = 49, \quad y = 70
```

对应：

```math
x = \frac{u-1}{2} = \frac{49-1}{2} = 24, \quad m = y = 70.
```

平凡解 $n=1, m=1$ 对应 $u=3, y=1$。

因此唯一非平凡正整数解是 $(n,m) = (24,70)$。

##### ✅ 第六步：总结

我们将方程 $1^2+\dots+n^2 = m^2$ 化为 $n(n+1)(2n+1) = 6 m^2$。

通过代换 $u=2n+1$ 转化为 Mordell 方程 $u^3 - u = 24 m^2$。

已知 Mordell 方程整数解结果表明 $u=49$ 是唯一非平凡正整数解。

因此 $n = 24, m = 70$ 是唯一解。
