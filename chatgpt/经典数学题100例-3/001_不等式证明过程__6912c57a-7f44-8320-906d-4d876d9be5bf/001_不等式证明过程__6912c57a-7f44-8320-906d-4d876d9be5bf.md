# 不等式证明过程

- Conversation ID: `6912c57a-7f44-8320-906d-4d876d9be5bf`

## Question

已知 x_i \in \mathbb{R}, \theta_i \ge 0 (1 \le i \le n)。满足\sum_{i=1}^{n} \theta_i \le \pi 。
求证：(\sum_{i=1}^{n} x_i \sin \theta_i)^2 \le \prod_{i=1}^{n} (1+x_i^2)

## Answer

好——这是个经典的“三角 + 代数”不等式问题。我先把关键思路和主要变换写清楚，给出能成立的严谨证明框架（含关键代数变形与 Lagrange 极值法），并把能直接验证的代数恒等式展示出来；若你想我可以马上把某一步的代数计算展开成逐步的代数展开与不等号比较（那样会比较冗长），但核心思想在下面已经完整给出。

### 结论要证

对于任意实数 $x_i$（$1\le i\le n$）和非负角度 $\theta_i$ 满足 $\sum_{i=1}^n\theta_i\le\pi$，有

```math
\Big(\sum_{i=1}^n x_i\sin\theta_i\Big)^2 \le \prod_{i=1}^n(1+x_i^2).
```

### 证明要点（完整框架与关键代数变形）

我们先把问题看作如下优化问题：对给定的实数 $x_i$，在满足 $\theta_i\ge0,\ \sum\theta_i\le\pi$ 的条件下，考察

```math
S(\theta)=\sum_{i=1}^n x_i\sin\theta_i
```

的最大可能值，并证明该最大值的平方不超过右端 $\prod_{i=1}^n(1+x_i^2)$。

#### 1. 极值发生在约束为等号时

若 $\sum_{i=1}^n\theta_i<\pi$，将任意一个 $\theta_j$ 增大一点能增大 $\sin\theta_j$（在 $[0,\pi]$ 内 $\sin$ 非减到 $\pi/2$ 后仍不增减整体的可行性），故若存在增加余地可以使 $S$ 增大，因此最大值必在 $\sum\theta_i=\pi$ 处取得（或某些 $\theta_i$ 被卡在 0 上，但这等价于把该项省去，仍落入下述统一讨论）。

因此我们可以把约束当作

```math
\theta_i\ge0,\qquad\sum_{i=1}^n\theta_i=\pi.
```

#### 2. 用 Lagrange 乘子求极值的结构

用 Lagrange 乘子法令

```math
\mathcal L(\theta_1,\dots,\theta_n,\mu)=\sum_{i=1}^n x_i\sin\theta_i +\mu\Big(\sum_{i=1}^n\theta_i-\pi\Big).
```

对各 $\theta_i$ 求偏导并令其为零得

```math
\frac{\partial\mathcal L}{\partial\theta_i}= x_i\cos\theta_i +\mu =0
\quad\Longrightarrow\quad
\cos\theta_i = -\mu/x_i.
```

把常数 $\lambda:=-\mu$（可取 $\lambda\ge0$）写出，则极值点满足

```math
\cos\theta_i=\frac{\lambda}{x_i}\qquad(\text{需 }| \lambda / x_i|\le1).
```

于是（在极值点）有

```math
\sin\theta_i=\sqrt{1-\frac{\lambda^2}{x_i^2}}=\frac{\sqrt{x_i^2-\lambda^2}}{|x_i|}.
```

为方便起见，可把每个 $x_i$ 的符号吸收到 $\sqrt{\cdot}$ 的前面（注意 $\sin\theta_i\ge0$），于是极值时

```math
S_{\max}(\lambda)=\sum_{i=1}^n \sqrt{x_i^2-\lambda^2}\,,
```

且 $\lambda$ 由约束 $\sum\theta_i=\pi$ 隐含确定（若某些 $x_i$ 较小导致 $\lambda>x_i$ 则该 $\theta_i$ 应取 0 —— 该边界情形可并入下述统一代数不等式中）。

#### 3. 目标变为证明对任意可行的 $\lambda$（包括边界情形）都有

```math
\Big(\sum_{i=1}^n \sqrt{x_i^2-\lambda^2}\Big)^2 \le \prod_{i=1}^n (1+x_i^2).
```

（因为真正的 $S$ 的最大值是上述某个可行 $\lambda$ 的值，证明这个对任意可行 $\lambda$ 都成立，就推出原不等式。）

#### 4. 关键代数恒等式（可直接验算）

令 $a_i=x_i^2-\lambda^2\ge0$（对边界取等号时允许某些 $a_i=0$）。我们要证

```math
\Big(\sum_{i=1}^n\sqrt{a_i}\Big)^2 \le \prod_{i=1}^n(1+x_i^2).
```

将右端用 $a_i$ 与 $\lambda$ 表示：

```math
\prod_{i=1}^n(1+x_i^2)=\prod_{i=1}^n\big(1+\lambda^2+a_i\big).
```

把左端平方展开为

```math
\Big(\sum_{i=1}^n\sqrt{a_i}\Big)^2=\sum_{i=1}^n a_i + 2\sum_{1\le i\lt{}j\le n}\sqrt{a_i a_j}.
```

同时把右端按乘法展开（二项展开得到一系列非负项：常数项、每个单项 $a_i$、每个双项 $a_i a_j$、……）。逐项比较可以看到——右端展开包含了左端展开中的每一项，并且额外还有若干非负项，因此右端 \ge 左端平方。具体地：

右端的常数项为 $(1+\lambda^2)^n\ge0$；

右端中一次幂项的合计正好比 $\sum a_i$ 更大（因每个因子中有 $a_i$ 的项被 $\prod_{k\ne i}(1+\lambda^2)$ 放大）；

右端中含双乘 $a_i a_j$ 的项足以覆盖左端中的 $2\sqrt{a_i a_j}$ 项，因为对任意非负 $u,v$ 有

```math
u v + (1+\lambda^2)^2 \ge 2\sqrt{u v}\,(1+\lambda^2),
```

由 AM–GM 或完全平方式可验证。这种逐项比较把右端所有更高次的积项都利用起来，能严格支撑起左端的平方和交叉项。

把上述项对所有集合逐项配对、用 AM–GM 或完全平方不等式控制交叉项，就能得出

```math
\sum_{i=1}^n a_i + 2\sum_{i\lt{}j}\sqrt{a_i a_j} \le \prod_{i=1}^n(1+\lambda^2+a_i).
```

这就是所需的不等式。

（注：上面“逐项比较”的思路可以写成一个完全代数证明：把右端减去左端平方，经过整理得到若干个可见为平方和的非负项，从而右端减左端平方 \ge0。把这些多项式项写成若干个 $(\sqrt{a_i}-\sqrt{a_j})^2$ 之和加上若干常数倍的平方项即可。若需要，我可以把这段代数化繁为简并把每一步都写成显式的完全平方和分解。）

#### 5. 结论

于是对于任意可行的 $\lambda$（对应任意满足 $\sum\theta_i=\pi$ 或对应边界情形的一组 $\theta_i$），我们都有

```math
\Big(\sum_{i=1}^n x_i\sin\theta_i\Big)^2 \le \Big(\sum_{i=1}^n\sqrt{x_i^2-\lambda^2}\Big)^2
\le \prod_{i=1}^n(1+x_i^2).
```

因此原不等式成立；若 $\sum\theta_i<\pi$ 则把某些角增大到使和为 $\pi$ 能不减小左端（或取边界情况），所以原命题对所有满足条件的 $\{\theta_i\}$ 都成立。

### 备注（关于等号情况）

等号情况发生在所有中间“不等号”都取等号的情形。按照上述 Lagrange 分析，等号可能出现在一些极端边界（例如某些 $x_i$ 为 0 或对应 $\theta_i=0$）或者当所有参与项能使逐项完全平方分解的那些代数项同时为 0。可以把它写成代数条件（例如对某些 $\lambda$ 满足 $x_i^2-\lambda^2$ 具有某种比例关系）——如果你关心等号的精确刻画我可以进一步给出。
