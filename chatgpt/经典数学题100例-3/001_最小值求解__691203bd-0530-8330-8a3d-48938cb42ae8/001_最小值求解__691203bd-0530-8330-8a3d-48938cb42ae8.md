# 最小值求解

- Conversation ID: `691203bd-0530-8330-8a3d-48938cb42ae8`

## Question

给定正实数a,b，设z为复数。求|(z-a)^2 + b^2| + |(z+a)^2 + b^2|的最小值。
友情提示：需要对a,b的大小关系分情况讨论。

## Answer

先把代数式因式分解并利用对称性来化简问题。

注意

```math
(z-a)^2+b^2=(z-a-ib)(z-a+ib),
\qquad
(z+a)^2+b^2=(z+a-ib)(z+a+ib).
```

因此

```math
F(z)\equiv |(z-a)^2+b^2|+|(z+a)^2+b^2|
=|z-(a+ib)|\,|z-(a-ib)|+|z-(-a+ib)|\,|z-(-a-ib)|.
```

由对称性可见 $F(z)=F(-z)$ 且对共轭保持不变，所以在 $z=x+iy$ 固定 $y$ 时，关于 $x$ 的函数
$g(x)=F(x+iy)$ 是偶函数。对任意固定的 $y$，$g(x)$ 关于 $x$ 的图像是凸的（每一项为复二次多项式模的和，其关于实部的二阶导数非负），因此在固定 $y$ 的情况下 $g(x)$ 在 $x=0$ 处取得最小值。于是全局最小值可在虚轴上搜索，即可令 $x=0$，只需研究

```math
F(iy)=2\sqrt{\big(a^2+b^2-y^2\big)^2+(2ay)^2}\,.
```

把 $t=y^2\ge0$，定义

```math
S(t)=\big(a^2+b^2-t\big)^2+(2a\sqrt t)^2
= t^2+2(a^2-b^2)t+(a^2+b^2)^2.
```

这是关于 $t$ 的二次多项式。当 $b^2-a^2\ge0$（即 $b\ge a$）时，二次式在

```math
t_0=b^2-a^2
```

处取得最小值；代回得

```math
S_{\min}=4a^2b^2,
```

从而

```math
\min_{z}F(z)=2\sqrt{S_{\min}}=2\cdot(2ab)=4ab\qquad(\text{当 }b\ge a).
```

当 $b^2-a^2<0$（即 $b<a$）时，二次式在 $t=0$ 取得最小值，对应 $y=0$（即 $z$ 取实数），此时

```math
S(0)=(a^2+b^2)^2,\qquad \min_{z}F(z)=2\sqrt{(a^2+b^2)^2}=2(a^2+b^2)\qquad(\text{当 }b\lt{}a).
```

综上所述：

```math
\boxed{\min_{z\in\mathbb C}\Big(|(z-a)^2+b^2|+|(z+a)^2+b^2|\Big)=
\begin{cases}
4ab,& b\ge a,\\[4pt]
2(a^2+b^2),& b\lt{}a.
\end{cases}}
```

并且取等号的点为：

当 $b\ge a$ 时，可取 $z=\pm i\sqrt{\,b^2-a^2\,}$（在这些点上达到值 $4ab$）；

当 $b<a$ 时，在 $z=0$ 处达到 $2(a^2+b^2)$（且这是固定的全局最小点；当 $b=a$ 时两者相等，均为 $4a^2$）。
