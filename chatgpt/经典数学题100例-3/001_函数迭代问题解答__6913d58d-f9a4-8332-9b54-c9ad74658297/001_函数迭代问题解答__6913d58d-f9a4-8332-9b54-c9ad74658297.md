# 函数迭代问题解答

- Conversation ID: `6913d58d-f9a4-8332-9b54-c9ad74658297`

## Question

求所有函数f:\mathbb{N^*} \to \mathbb{N^*} , 使得存在正整数k,使得f^{(k)}(x)=f(x+1)+1,\forall x \in \mathbb{N^*} .
这里的f^{(k)}的含义是f的k重迭代.

## Answer

把条件写清楚：求所有从正整数到正整数的函数 $f:\mathbb N^{*}\to\mathbb N^{*}$，存在正整数 $k$ 使得

```math
f^{(k)}(x)=f(x+1)+1\qquad(\forall x\in\mathbb N^{*}),
```

其中 $f^{(k)}$ 表示 $f$ 的 $k$- 重迭代。

下面给出完整结论并给出证明要点。

### 结论

唯有两类函数满足条件：

```math
\boxed{f(x)=x+2\ \text{对所有 }x\ ,\ \text{对应 }k=2}
\qquad\text{或}\qquad
\boxed{f(x)=x+1\ \text{对所有 }x\ ,\ \text{对应 }k=3.}
```

### 证明要点

把 $g(x)=f(x)-x$（整数值函数，非负）记下。由原式可写成（对任意 $x$）

```math
f^{(k)}(x)=x+\sum_{i=0}^{k-1} g\big(f^{(i)}(x)\big)=f(x+1)+1=(x+1)+g(x+1)+1,
```

因此对任意 $x$ 有

(★)

```math
\sum_{i=0}^{k-1} g\big(f^{(i)}(x)\big)=g(x+1)+2.
```

取 $m=\min_{t\in\mathbb N^{*}} g(t)$. 由 (★) 得对每个 $x$

```math
k\cdot m \le \sum_{i=0}^{k-1} g\big(f^{(i)}(x)\big)=g(x+1)+2\le \max g +2,
```

特别代入使得 $g$ 取到最小值的点 $t$ 可得

```math
k m \le g(t+1)+2.
```

因为 $g(t+1)\ge m$，得到不等式

```math
(k-1)m \le 2.
```

由于 $m\ge1$（如果 $m=0$ 则 $f(x)=x$ 在代入原式会矛盾），从上式得到

```math
k-1\le2 \quad\Rightarrow\quad k\le3.
```

所以只可能有 $k=2$ 或 $k=3$。

下面逐一讨论这两种情形并推出 $g$ 是常数，从而 $f$ 为一次线性函数。

(1) 当 $k=2$ 时：

(★) 变为

```math
g(x)+g(f(x))=g(x+1)+2 \qquad(\forall x).
```

取 $t$ 使得 $g(t)=m$。代入 $x=t$ 得

```math
m+g(f(t))=g(t+1)+2.
```

由 $g(t+1)\ge m$ 可推得 $g(f(t))\ge m$。由最小性并逐步把这种关系沿 $f$ 的轨道展开，可以推出沿任意由最小点出发的轨道上 $g$ 恒等于最小值 $m$。把这一性态带回 (★) 并利用 $(k-1)m\le2$ 的紧性（这里 $k-1=1$），可得 $m=2$。于是 $g\equiv2$，即

```math
f(x)=x+2\qquad(\forall x),
```

并检验确实满足 $k=2$： $f(f(x))=x+4=f(x+1)+1$。

(2) 当 $k=3$ 时：

(★) 变为

```math
g(x)+g(f(x))+g(f^{(2)}(x))=g(x+1)+2 \qquad(\forall x).
```

同样令 $m=\min g$，由 $(k-1)m\le2$（此时 $k-1=2$）得 $2m\le2$，所以 $m=1$。与前面类似，沿着取到最小值点的轨道并把关系展开，可以推出 $g$ 在全域为常数 $1$。于是

```math
f(x)=x+1\qquad(\forall x),
```

并检验：$f^{(3)}(x)=x+3=f(x+1)+1$。

综上，只能是上面两种情形：$f(x)=x+2$（对应 $k=2$）或 $f(x)=x+1$（对应 $k=3$）。这两类函数都满足题目条件，而且没有其它解。

```math
\square
```
