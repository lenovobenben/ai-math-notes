# 证明条件充分必要

- Conversation ID: `690a344f-75c0-8328-89c7-e60b7d326f62`

## Question

证明：
有复数|z|=1。满足z^{n+1}-z^{n}-1=0的充分必要条件是6|n+2。

## Answer

设 $z$ 为复数且 $|z|=1$。写 $z=e^{i\theta}$（$\theta\in\mathbb R$）。由方程

```math
z^{n+1}-z^n-1=0
```

得

```math
e^{i(n+1)\theta}-e^{in\theta}-1=0\quad\Longrightarrow\quad e^{in\theta}(e^{i\theta}-1)=1.
```

取模得（注意 $|e^{in\theta}|=1$）

```math
|e^{i\theta}-1|=1.
```

计算

```math
|e^{i\theta}-1|^2=(e^{i\theta}-1)(e^{-i\theta}-1)=2-2\cos\theta,
```

于是 $2-2\cos\theta=1$，所以

```math
\cos\theta=\tfrac12.
```

因此 $\theta\equiv\pm\frac{\pi}{3}\pmod{2\pi}$，即

z=e^{i\theta}\in\{e^{i\pi/3},\,e^{-i\pi/3}\\} .

下面检验这两种情形给出对 $n$ 的约束。先取 $\theta=\pi/3$。由

```math
e^{in\theta}(e^{i\theta}-1)=1
```

可得

```math
e^{in\pi/3}=\frac{1}{e^{i\pi/3}-1}.
```

计算

```math
e^{i\pi/3}-1=\tfrac12+i\tfrac{\sqrt3}{2}-1=-\tfrac12+i\tfrac{\sqrt3}{2}=e^{i2\pi/3},
```

所以 $\dfrac{1}{e^{i\pi/3}-1}=e^{-i2\pi/3}$。于是

```math
e^{in\pi/3}=e^{-i2\pi/3}\quad\Longrightarrow\quad n\frac{\pi}{3}\equiv -\frac{2\pi}{3}\pmod{2\pi}.
```

两边乘以 3 得

```math
n\pi\equiv -2\pi\pmod{6\pi}\quad\Longrightarrow\quad n\equiv -2\pmod{6},
```

即 $6\mid (n+2)$。

若取 $\theta=-\pi/3$，同样有

```math
e^{in(-\pi/3)}=\frac{1}{e^{-i\pi/3}-1}=\frac{1}{-{\tfrac12}-i{\tfrac{\sqrt3}{2}}}=e^{i2\pi/3},
```

从而同样得到 $n\equiv-2\pmod6$。

综上：存在 $|z|=1$ 满足 $z^{n+1}-z^n-1=0$ 的充分且必要条件是

```math
n\equiv-2\pmod6,
```

即 $6\mid(n+2)$。证毕。
