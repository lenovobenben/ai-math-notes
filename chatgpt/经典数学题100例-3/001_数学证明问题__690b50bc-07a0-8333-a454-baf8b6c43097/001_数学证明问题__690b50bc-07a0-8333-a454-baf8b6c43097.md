# 数学证明问题

- Conversation ID: `690b50bc-07a0-8333-a454-baf8b6c43097`

## Question

令a_1, \dots ,a_n 是一组整数。请证明：
\prod_{1 \le i \lt  j \le n}(j-i) \mid \prod_{1 \le i \lt  j \le n}(a_j-a_i)

## Answer

把这个结论看作范德蒙行列式（Vandermonde determinant）与二项式基（binomial basis）之间的换基整性就很自然了。下面给出常见的、简洁的证明。

令矩阵 $M$ 的第 $i$ 行第 $j$ 列为

```math
M_{ij}=\binom{a_i}{\,j-1\,},\qquad i,j=1,\dots,n,
```

其中 $\binom{x}{k}=\dfrac{x(x-1)\cdots(x-k+1)}{k!}$（当 $k=0$ 时取 1）。由于 $a_i$ 是整数，矩阵 $M$ 的每个元素都是整数，因此 $\det M$ 是整数。

另一方面，考虑多项式列

```math
p_j(x)=\binom{x}{\,j-1\,},\qquad j=1,\dots,n.
```

$p_j(x)$ 是次数为 $j-1$ 的多项式，其首项系数为 $\dfrac{1}{(j-1)!}$. 因此把这组多项式在点 $x=a_1,\dots,a_n$ 处组成的行列式与用单项式基 $1,x,x^2,\dots$ 组成的范德蒙行列式相比较得到

```math
\det\big(\binom{a_i}{\,j-1\,}\big)_{1\le i,j\le n}
=\Big(\prod_{j=1}^{n}\frac{1}{(j-1)!}\Big)\cdot
\det\big(a_i^{\,j-1}\big)_{1\le i,j\le n}.
```

而范德蒙行列式的值是众所周知的

```math
\det\big(a_i^{\,j-1}\big)=\prod_{1\le i\lt{}j\le n}(a_j-a_i).
```

因此

```math
\det\big(\binom{a_i}{\,j-1\,}\big)
=\frac{\displaystyle\prod_{1\le i\lt{}j\le n}(a_j-a_i)}
{\displaystyle\prod_{k=0}^{\,n-1}k! }.
```

左边是整数，所以分数右边也是整数，即

```math
\prod_{k=0}^{n-1}k!\ \Bigm|\ \prod_{1\le i\lt{}j\le n}(a_j-a_i).
```

最后注意恒等

```math
\prod_{k=0}^{n-1}k!=\prod_{k=1}^{n-1}k!
=\prod_{1\le i\lt{}j\le n}(j-i),
```

（因为对固定的差 $t\in\{1,\dots,n-1\}$，因子 $t$ 出现在左边的乘积中恰好 $n-t$ 次，与右边统计差为 $t$ 的所有 $i<j$ 个数一致），从而得到所需结论：

```math
\prod_{1\le i\lt{}j\le n}(j-i)\ \Bigm|\ \prod_{1\le i\lt{}j\le n}(a_j-a_i).
```

证毕。
