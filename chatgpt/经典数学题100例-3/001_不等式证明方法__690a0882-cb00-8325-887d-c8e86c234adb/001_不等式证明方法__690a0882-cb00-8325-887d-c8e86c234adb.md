# 不等式证明方法

- Conversation ID: `690a0882-cb00-8325-887d-c8e86c234adb`

## Question

对于n个正实数a_{i}满足\sum_{i=1}^{n} {\frac{1}{1+a_{i}} }=1。
证明：\sum_{i=1}^{n}\sqrt{a_{i}} \ge (n-1) \sum_{i=1}^{n}{\frac{1}{\sqrt[]{a_{i}}}}

## Answer

把题设做代换并用排列（Chebyshev / rearrangement）思想可以很短地解决。下面给出一个清晰的证明思路（来自 Math.StackExchange 上的一个标准解法，步骤已作整理）。Mathematics Stack Exchange

证明： 设

```math
p_i=\frac{1}{1+a_i}\qquad(0\lt{}p_i\lt{}1),\qquad\sum_{i=1}^n p_i=1.
```

由此有

```math
a_i=\frac{1-p_i}{p_i},\qquad\sqrt{a_i}=\frac{\sqrt{1-p_i}}{\sqrt{p_i}},\qquad\frac{1}{\sqrt{a_i}}=\frac{\sqrt{p_i}}{\sqrt{1-p_i}}.
```

把原不等式写成 $p_i$ 的形式，等价于

```math
\sum_{i=1}^n\frac{\sqrt{1-p_i}}{\sqrt{p_i}}\;\ge\;(n-1)\sum_{i=1}^n\frac{\sqrt{p_i}}{\sqrt{1-p_i}}.
```

两边同除以 $\prod_{i=1}^n\sqrt{p_i(1-p_i)}$ 并整理，可写为

```math
\sum_{i=1}^n\frac{1-np_i}{\sqrt{p_i(1-p_i)}}\;\ge\;0.
```

注意到我们有 $\sum_{i=1}^n(1-np_i)=0$。此时应用排列不等式（rearrangement / Chebyshev）：把两个数列

```math
\bigl(1-np_1,\dots,1-np_n\bigr)\quad\text{和}\quad\Bigl(\tfrac{1}{\sqrt{p_1(1-p_1)}},\dots,\tfrac{1}{\sqrt{p_n(1-p_n)}}\Bigr)
```

按同样的顺序排列（可以证明在本条件下它们的次序是一致的），由 Chebyshev 不等式

```math
\frac{1}{n}\sum_{i=1}^n x_i y_i \ge\Bigl(\frac{1}{n}\sum_{i=1}^n x_i\Bigr)\Bigl(\frac{1}{n}\sum_{i=1}^n y_i\Bigr)
```

可得

```math
\sum_{i=1}^n\frac{1-np_i}{\sqrt{p_i(1-p_i)}}\ge 0.
```

因此原不等式成立。且当 $n=2$（或等价地存在使得所有项取等的情形）时可达等号。
