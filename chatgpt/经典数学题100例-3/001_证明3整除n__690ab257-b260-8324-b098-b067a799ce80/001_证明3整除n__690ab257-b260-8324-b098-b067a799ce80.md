# 证明3整除n

- Conversation ID: `690ab257-b260-8324-b098-b067a799ce80`

## Question

n \times n 方阵A,B满足A^2+B^2=AB，并且AB-BA可逆。
证明3 \mid n

## Answer

下面给出完整证明（用复数域或任意包含原始三次单位根 $\omega$ 的代数闭域即可；若开始的基域不包含 $\omega$，对系数域取代数闭包后不改变可逆性与行列式的整性，证明仍然成立）。

记 $\omega$ 为原始三次单位根，满足 $\omega^2+\omega+1=0$（且 $\omega^3=1,\ \omega\neq1$）。设

```math
C=AB-BA,
```

假设题中条件 $A^2+B^2=AB$ 与 $C$ 可逆成立。

定义矩阵

```math
M_1=A+\omega B,\qquad M_2=A+\omega^2 B.
```

计算它们的乘积（注意非交换性，需要保留 $AB$ 与 $BA$）：

```math
\begin{aligned}
M_1M_2
&=(A+\omega B)(A+\omega^2 B)\\
&=A^2+\omega^2 AB+\omega BA+\omega^3 B^2\\
&=A^2+B^2+\omega^2 AB+\omega BA\quad(\omega^3=1)\\
&=AB+\omega^2 AB+\omega BA\qquad(\text{由 }A^2+B^2=AB)\\
&=\omega(BA-AB)=-\omega C.
\end{aligned}
```

同理

```math
M_2M_1=-\omega^2 C.
```

由 $C$ 可逆可知 $-\omega C$ 与 $-\omega^2 C$ 也可逆，因而 $M_1M_2$ 与 $M_2M_1$ 可逆。对方阵而言，若积 $XY$ 可逆则 $X$ 与 $Y$ 都可逆，因此 $M_1,M_2$ 可逆。

取行列式，得到

```math
\det(M_1)\det(M_2)=\det(M_1M_2)=\det(-\omega C)=(-\omega)^n\det(C),
```

同时也有

```math
\det(M_2)\det(M_1)=\det(M_2M_1)=(-\omega^2)^n\det(C).
```

左右两式左端相同，故

```math
(-\omega)^n\det(C)=(-\omega^2)^n\det(C).
```

因 $\det(C)\neq0$ 可约去，得到

```math
(-\omega)^n =(-\omega^2)^n \quad\Longrightarrow\quad \left(\frac{-\omega}{-\omega^2}\right)^n=1.
```

化简得 $\left(\omega^{\, -1}\right)^n=1$。由于 $\omega^{-1}=\omega^2$ 且 $\omega$ 的阶为 $3$，所以 $\omega^{2n}=1$ 当且仅当 $3\mid 2n$。由于 $\gcd(2,3)=1$，这等价于 $3\mid n$。

因此 $n$ 必被 $3$ 整除。∎
