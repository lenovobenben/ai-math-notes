# 多项式非负充分必要条件

- Conversation ID: `690a0a8e-ffa0-8330-bee4-d60f0463700f`

## Question

证明：实系数多项式P(x) \ge 0 的充分必要条件是：有多项式q(x),r(x)，使得P(x)=q^2(x)+r^2(x)

## Answer

下面给出完整证明（中文）：

命题。 设 $P(x)$ 是系数为实数的多项式。则

```math
P(x)\ge0\ \text{对所有实数 }x
```

当且仅当存在实系数多项式 $q(x),\,r(x)$，使得

```math
P(x)=q(x)^2+r(x)^2 .
```

必要性（若能表示为两项平方和，则恒非负）。 若存在实多项式 $q,r$ 使得 $P=q^2+r^2$，则对任意实 $x$ 有 $q(x)^2\ge0,\ r(x)^2\ge0$，所以 $P(x)\ge0$。这部分显然。

充分性（非负的多项式可写成两项平方和）。 设 $P(x)$ 在实轴上恒非负且系数为实数。先作因式分解（在复数域）：

```math
P(x)=c\prod_{j=1}^m (x-\alpha_j)^{e_j}\prod_{k=1}^n (x-z_k)^{f_k}(x-\overline{z_k})^{f_k},
```

其中 $c>0$（若 $c<0$ 则 $P$ 不可能恒非负），实根 $\alpha_j\in\mathbb R$ 的幂次 $e_j$ 必为偶数（否则在该实根附近符号会改变），而复根成共轭对出现。于是可以把所有实根的偶次幂合并成一个平方多项式：

```math
s(x)=\sqrt{\,c\,}\prod_{j}(x-\alpha_j)^{e_j/2},
```

其中 $s(x)$ 是一个实系数多项式（取常数因子正平方根）。把剩下的因子写成共轭对：

```math
P(x)=s(x)^2\prod_{k=1}^n p_k(x),
\qquad p_k(x)=(x-z_k)^{f_k}(x-\overline{z_k})^{f_k},
```

每个 $p_k(x)$ 都是首项系数为正的实系数、多项式且在实轴上恒正（实际上当 $f_k=1$ 时是不可约的正定二次式，若 $f_k>1$ 则是该二次式的幂）。

下面给出两种常见且等价的构造方法来把这些正二次因子合并成两平方和。

方法一（利用复多项式的实部与虚部）。
将 $P$ 的复因子配对并定义复多项式

```math
A(x)=\sqrt{c}\prod_{k=1}^n (x-z_k)^{f_k}\,,
```

注意 $A(x)$ 的系数一般为复数，但把 $A(x)$ 与其共轭相乘得到

```math
A(x)\overline{A(x)}=|A(x)|^2=P(x).
```

写 $A(x)=U(x)+\mathrm i V(x)$（其中 $U,V$ 为实系数多项式），则

```math
P(x)=A(x)\overline{A(x)}=(U(x)+\mathrm i V(x))(U(x)-\mathrm i V(x))=U(x)^2+V(x)^2.
```

这正是所需的表示，取 $q=U,\ r=V$。

方法二（逐因子构造 + 乘法不变性）。
只需证明：不可约的正定二次式（形式 $ax^2+bx+c,\ a>0,\ b^2-4ac<0$）可以表示为两项线性多项式的平方和；以及两项平方和在乘法下封闭（即两个“和平方”的乘积仍可写成两项平方和）。

对 $ax^2+bx+c$，取

```math
u=\sqrt a,\quad v=\frac b{2\sqrt a},\quad t=\sqrt{\,c-\dfrac{b^2}{4a}\,},
```

则

```math
ax^2+bx+c=(u x+v)^2 + t^2,
```

右边是两个一次多项式的平方和（第二项 $t^2$ 也看作常数多项式的平方）。因为判别式为负，上式中的 $t$ 为实数，成立。

乘法不变性：若 $A_1^2+B_1^2$ 与 $A_2^2+B_2^2$ 是两对实多项式的平方和，则它们的乘积可用恒等式

```math
(A_1^2+B_1^2)(A_2^2+B_2^2)=(A_1A_2-B_1B_2)^2+(A_1B_2+B_1A_2)^2
```

表示为两项平方和。由此可以把所有正定二次式以及它们的幂的乘积合成为两项平方和。再乘上前面的 $s(x)^2$（一个完全平方）只会把两项平方和中的每一项乘以 $s(x)$，仍得到两项平方和。

综上，用任一方法都能构造出实多项式 $q(x),r(x)$ 使得 $P=q^2+r^2$。

结论。 命题成立：实系数多项式 $P(x)$ 在实轴上恒非负当且仅当它可写为两个实多项式的平方和 $q(x)^2+r(x)^2$。

∎
