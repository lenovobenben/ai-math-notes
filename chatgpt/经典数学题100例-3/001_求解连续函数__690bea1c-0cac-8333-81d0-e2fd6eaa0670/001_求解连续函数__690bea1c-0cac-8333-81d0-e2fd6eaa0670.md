# 求解连续函数

- Conversation ID: `690bea1c-0cac-8333-81d0-e2fd6eaa0670`

## Question

有实数范围内的连续函数f(x)，满足f(ab+1)=f(a)f(b)-f(b)-a+2，且f(0)=1。求f(x)

## Answer

把 $b=0$ 代入原式得

```math
f(a\cdot0+1)=f(1)=f(a)f(0)-f(0)-a+2.
```

又知 $f(0)=1$，代入得

```math
f(1)=f(a)-1-a+2 \implies f(a)=a+(f(1)-1).
```

于是存在常数 $c=f(1)-1$，使得对任意实数 $x$，都有

```math
f(x)=x+c.
```

把 $f(x)=x+c$ 代回原方程：

```math
\text{左边 } f(ab+1)=ab+1+c,
```

```math
\text{右边 } f(a)f(b)-f(b)-a+2=(a+c)(b+c)-(b+c)-a+2
```

化简得右边为

```math
ab+(a+b)(c-1)+(c^2-c+2).
```

两边关于 $a,b$ 的系数比较可得 $c-1=0$，即 $c=1$。代入常数项也吻合。

因此唯一解为

```math
\boxed{\,f(x)=x+1\,}
```

（满足连续性及给定条件，且检验通过）。
