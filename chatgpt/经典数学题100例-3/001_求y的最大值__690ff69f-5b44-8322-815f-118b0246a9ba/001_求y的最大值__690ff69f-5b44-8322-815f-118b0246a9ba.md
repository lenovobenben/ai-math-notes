# 求y的最大值

- Conversation ID: `690ff69f-5b44-8322-815f-118b0246a9ba`

## Question

x,y \in \mathbb{R} 且满足x^xy^y = 1。求\max y。

## Answer

先说明定义域：由于出现 $x^x,y^y$（实数底数的实数幂通常要求底数为正），我们取 $x>0,\;y>0$。

令函数

```math
f(t)=t\ln t\qquad(t\gt{}0).
```

由条件 $x^x y^y=1$，取对数得

```math
x\ln x+y\ln y=0\quad\Longleftrightarrow\quad f(x)+f(y)=0.
```

函数 $f(t)$ 在 $t>0$ 上的最小值为 $-\dfrac{1}{e}$（发生在 $t=\dfrac{1}{e}$），且 $f((0,\infty))=[-1/e,\infty)$。因此存在 $x>0$ 使得 $f(x)=-f(y)$ 的必要且充分条件是

```math
-f(y)\in[-1/e,\infty)\quad\Longleftrightarrow\quad f(y)\le\frac{1}{e}.
```

要使 $y$ 最大，应取等号，即解

```math
y\ln y=\frac{1}{e}.
```

设 $y=e^{w}$，则 $w e^{w}=1/e$。由 Lambert $W$ 函数得

```math
w=W\!\left(\frac{1}{e}\right),\qquad y=e^{W(1/e)}.
```

用 $e^{W(z)}=z/W(z)$ 也可写成

```math
y=\frac{1}{e\,W(1/e)}.
```

数值上

```math
y\_{\max}=e^{W(1/e)}\approx 1.321099762.
```

答：最大值为 $y_{\max}=e^{W(1/e)}\approx1.321099762$（假定 $x,y>0$）。
