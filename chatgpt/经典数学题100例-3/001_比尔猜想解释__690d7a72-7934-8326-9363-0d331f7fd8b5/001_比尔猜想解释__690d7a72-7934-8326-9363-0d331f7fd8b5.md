# 比尔猜想解释

- Conversation ID: `690d7a72-7934-8326-9363-0d331f7fd8b5`

## Question

数论中的比尔猜想是什么

## Answer

“比尔猜想”（Beal Conjecture）是数论中一个非常著名但尚未被证明的猜想。它是对费马大定理的一个推广，由美国银行家兼业余数学家 Andrew Beal 在 1993 年提出的。

##### 🧮 猜想内容

若 $A^x + B^y = C^z$，其中
$A, B, C, x, y, z$ 都是 正整数，且 $x, y, z > 2$，
那么 A、B、C 必须有一个大于 1 的公因子。

也就是说，只有当 $A, B, C$ 有公因子时，方程

```math
A^x + B^y = C^z
```

才可能成立。

##### 🔍 举几个例子

有公因子的情况（符合猜想）

```math
2^3 + 2^3 = 4^3
```

这里 $A=2, B=2, C=4$，显然有公因子 2。

若要没有公因子

```math
\gcd(A, B, C) = 1
```

比尔猜想认为：这种情况下方程没有解。

##### 📚 与费马大定理的关系

费马大定理是比尔猜想的一个特例。
费马大定理说：

```math
A^n + B^n = C^n \quad (n\gt{}2)
```

没有正整数解。
这正对应于比尔猜想中最对称的一种情况 $x=y=z>2$，且显然三者互质时无解。

所以：

```math
\text{费马大定理} \Rightarrow \text{比尔猜想的特例}
```

##### 💰 奖励与现状

Andrew Beal 为此设立了一个奖励：

证明或反驳比尔猜想的人可获得 100万美元奖金。

该奖金由美国数学学会（AMS）管理。

截至目前（2025年），比尔猜想仍未被证明或反驳。

##### 🧠 总结一句话

比尔猜想：

```math
A^x + B^y = C^z, \ x, y, z \gt{} 2 \implies \gcd(A, B, C) \gt{} 1
```

是对费马大定理的推广，目前尚未解决。
