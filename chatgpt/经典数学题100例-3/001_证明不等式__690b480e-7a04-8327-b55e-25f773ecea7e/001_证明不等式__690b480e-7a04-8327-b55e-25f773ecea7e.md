# 证明不等式

- Conversation ID: `690b480e-7a04-8327-b55e-25f773ecea7e`

## Question

给定一组正实数a_1,a_2,\dots,a_n。证明不等式
(1+G)^n \le (1+a_1) \dots (1+a_n) \le (1+A)^n。
其中A,G分别为算术平均值和几何平均值。

## Answer

设 $a_1,\dots,a_n>0$。记算术平均数 $A=\dfrac{a_1+\cdots+a_n}{n}$，几何平均数 $G=\bigl(a_1\cdots a_n\bigr)^{1/n}$。对不等式两端分别取对数，等价于证明

```math
n\ln(1+G)\le\sum_{i=1}^n\ln(1+a_i)\le n\ln(1+A).
```

下面分别证明左右两边。

右不等式：$\displaystyle\sum_{i=1}^n\ln(1+a_i)\le n\ln(1+A)$.

函数 $f(x)=\ln(1+x)$ 在 $(0,\infty)$ 上二阶导为

```math
f''(x)=-\frac{1}{(1+x)^2}\lt{}0,
```

所以 $f$ 是凹函数。由 Jensen 不等式（凹函数的情形）得

```math
\frac{1}{n}\sum_{i=1}^n\ln(1+a_i)\le \ln\!\Bigl(1+\frac{1}{n}\sum_{i=1}^n a_i\Bigr)=\ln(1+A).
```

两边乘以 $n$ 并取指数，得到

```math
\prod_{i=1}^n(1+a_i)\le(1+A)^n.
```

左不等式：$\displaystyle n\ln(1+G)\le\sum_{i=1}^n\ln(1+a_i)$.

记 $a_i=e^{x_i}$（因 $a_i>0$ 可行），则 $G=\exp\!\bigl(\frac{1}{n}\sum_{i=1}^n x_i\bigr)$。定义 $g(x)=\ln(1+e^{x})$。计算二阶导：

```math
g''(x)=\frac{e^{x}}{(1+e^{x})^2}\gt{}0,
```

因此 $g$ 在 $\mathbb R$ 上为凸函数。由 Jensen 不等式（凸函数的情形）得

```math
\frac{1}{n}\sum_{i=1}^n\ln(1+e^{x_i})\ge \ln\!\Bigl(1+\exp\!\bigl(\frac{1}{n}\sum_{i=1}^n x_i\bigr)\Bigr).
```

代回 $a_i=e^{x_i}$ 和 $G=\exp\!\bigl(\frac{1}{n}\sum x_i\bigr)$，两边乘以 $n$ 并取指数，得到

```math
\prod_{i=1}^n(1+a_i)\ge(1+G)^n.
```

综上合并两端不等式，得

```math
(1+G)^n \le \prod_{i=1}^n(1+a_i)\le(1+A)^n,
```

证毕。 ∎
