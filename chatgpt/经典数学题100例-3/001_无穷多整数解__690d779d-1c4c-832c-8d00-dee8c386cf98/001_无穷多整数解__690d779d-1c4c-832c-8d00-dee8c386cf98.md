# 无穷多整数解

- Conversation ID: `690d779d-1c4c-832c-8d00-dee8c386cf98`

## Question

求证下列方程有无穷多的整数解(x,y,z)。
x^3+3y^3+9z^3-9xyz=1

## Answer

好的——给你一个干净、构造性的证明：这个丙次方程

```math
x^3+3y^3+9z^3-9xyz=1
```

确实有无限多个整数解，而且可以给出一个显式的递推（代数）办法来生成它们。

### 关键想法（代数数论 + 具体乘法律）

把整数三元组 $(x,y,z)$ 看作代数整数

```math
A(x,y,z)=x+y\theta+z\theta^2
```

在环 $\mathbb Z[\theta]$ 中，其中 $\theta$ 满足

```math
\theta^3=3.
```

在这个环里，对任意 $A=x+y\theta+z\theta^2$ 定义其代数范数（乘积范数）

```math
N(A)=\prod_{i=1}^3 (x+y\theta_i+z\theta_i^2),
```

这里 $\theta_1,\theta_2,\theta_3$ 是 $\theta$ 的三个共轭（即三个复立方根）。可以直接算出（或通过乘法矩阵的行列式计算）得到恒等式

```math
\boxed{\,N(x+y\theta+z\theta^2)=x^3+3y^3+9z^3-9xyz\,}.
```

因此原方程等价于要求元素 $A\in\mathbb Z[\theta]$ 有范数 $N(A)=1$，即找单位元（范数为 $1$ 的代数整数）。

### 乘法公式（显式的整数双线性映射）

在$\theta^3=3$ 的约化下，两元素相乘仍给出整数系数。对

```math
A_1=(x_1,y_1,z_1),\qquad A_2=(x_2,y_2,z_2)
```

分别表示 $x_1+y_1\theta+z_1\theta^2$ 和 $x_2+y_2\theta+z_2\theta^2$，它们的乘积在基 $1,\theta,\theta^2$ 下的系数为（可用多项式乘法并把 $\theta^3$ 用 $3$ 约化得到）：

```math
\begin{aligned}
x_3&=x_1x_2+3(y_1z_2+y_2z_1),\\[4pt]
y_3&=x_1y_2+x_2y_1+3 z_1 z_2,\\[4pt]
z_3&=x_1 z_2 + x_2 z_1 + y_1 y_2.
\end{aligned}
```

这说明：三元组在这种乘法下闭合，且范数是可乘的（即 $N(A_1A_2)=N(A_1)N(A_2)$），这正是我们需要的代数结构。

### 给出一个非平凡的范数为 1 的元 — 从而得到无限多解

直接找一个整数解，算一下范数：三元组 $(4,3,2)$ 满足方程（代入可验证）：

```math
4^3+3\cdot 3^3+9\cdot 2^3-9\cdot 4\cdot 3\cdot 2 = 64 + 81 + 72 - 216 = 1.
```

所以对应的代数元素

```math
\alpha = 4+3\theta+2\theta^2
```

满足 $N(\alpha)=1$；也就是说 $\alpha$ 是 $\mathbb Z[\theta]$ 中的一个单位。

既然 $\alpha$ 是单位，则其任意整数次幂 $\alpha^n$ 仍是单位，且

```math
N(\alpha^n)=N(\alpha)^n=1.
```

把 $\alpha^n$ 用基 $1,\theta,\theta^2$ 展开，会得到整数系数三元组 $(x_n,y_n,z_n)$，而这些三元组都满足原方程。于是我们获得了一列无限多的整数解：

```math
(x_n,y_n,z_n) \quad\text{来自}\quad \alpha^n=(x_n)+ (y_n)\theta + (z_n)\theta^2,\qquad n\in\mathbb Z.
```

### 更具体：把幂运算写成显式递推

把固定单位 $\alpha=(4,3,2)$ 与任意当前三元组 $(x,y,z)$ 相乘，按上面的乘法公式得到下一个三元组 $(x',y',z')$：

```math
\begin{aligned}
x'&=4x+3(2y+3z)=4x+6y+9z,\\
y'&=3x+4y+3\cdot 2 z=3x+4y+6z,\\
z'&=2x+3y+4z.
\end{aligned}
```

从基元 $(1,0,0)$（对应常数 $1$）出发反复用上面的线性递推乘以 $(4,3,2)$，即得到

```math
(x_{n+1},y_{n+1},z_{n+1}) = \big(4x_n+6y_n+9z_n,\;3x_n+4y_n+6z_n,\;2x_n+3y_n+4z_n\big),
```

且每一步都保持方程的右端为 $1$。于是 $\{(x_n,y_n,z_n)\}_{n\ge0}$ 就是一列无限不同的整数解（因为对应代数元 $\alpha^n$ 在实嵌入下绝对值会增长，所以不会相等为循环，$\alpha$ 不是根单位）。

举几个前几项：

$n=0$ 对应 $(1,0,0)$（显然是解）；

$n=1$ 得到 $(4,3,2)$（我们已经验证是解）；

$n=2$ 代入递推给出 $(52,36,25)$，同样满足方程；

继续可以得到任意多的整数解。

### 结论

方程

```math
x^3+3y^3+9z^3-9xyz=1
```

在整数域中有无限多解。构造法：取 $\theta^3=3$，把解与环 $\mathbb Z[\theta]$ 中范数为 $1$ 的单位对应；具体单位 $\alpha=4+3\theta+2\theta^2$ 的任意幂 $\alpha^n$（在基 $1,\theta,\theta^2$ 下的系数）给出无限多个整数解；并给出显式的线性递推，便于实际生成解序列。
