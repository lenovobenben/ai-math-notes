# 凸多边形角度不等式证明

- Conversation ID: `6909b8d9-5b20-8324-adaf-e15f49d1a21c`

## Question

设P为凸多边形A_{1}A_{2}\dots A_{n}内一点。证明有\min_{1 \le i \le n} \angle PA_{i}A_{i+1} \le \frac{\pi}{2} - \frac{\pi}{n}

## Answer

把结论用几何语言说清楚：对于凸多边形 $A_1A_2\cdots A_n$ 内的一点 $P$，记

```math
\beta_i:=\angle PA_iA_{i+1}\qquad(i=1,2,\dots,n),
```

下标模 $n$ 周期。要证 $\min_{1\le i\le n}\beta_i\le \dfrac\pi2-\dfrac\pi n$。

证明思路：把多边形分成以 $P$ 为公共顶点的 $n$ 个三角形 $PA_iA_{i+1}$。记

```math
\alpha_i:=\angle A_iPA_{i+1}\quad\text{（即第 \(i\) 个以 \(P\) 为顶点的三角形在 \(P\) 的角）},
```

则 $\displaystyle\sum_{i=1}^n\alpha_i=2\pi$。所以存在某个指标 $k$ 使得

```math
\alpha_k\ge\frac{2\pi}{n}.
```

看三角形 $PA_kA_{k+1}$。在这个三角形中三内角和为 $\pi$，因此其余两角之和为

```math
\beta_k+\gamma_k=\pi-\alpha_k\le\pi-\frac{2\pi}{n},
```

其中我们把 $\gamma_k$ 记为三角形 $PA_kA_{k+1}$ 在点 $A_{k+1}$ 处的角（即 $\gamma_k=\angle PA_{k+1}A_k$）。于是

```math
\min\{\beta_k,\gamma_k\}\le\frac{\beta_k+\gamma_k}{2}\le\frac{\pi-\tfrac{2\pi}{n}}{2}
=\frac\pi2-\frac\pi n.
```

如果在这个三角形里较小的角是 $\beta_k$，那就直接得到所要不等式。

只剩下当较小角是 $\gamma_k$ 的情形。此时 $\gamma_k\le\dfrac\pi2-\dfrac\pi n$。注意 $\gamma_k$ 是顶点 $A_{k+1}$ 处与边 $A_{k+1}A_k$ 所成的角；而题中要求的角集合 $\{\beta_i\}$ 包含了每个顶点与它的后一条边所成的角。若当前较小角恰好不是我们已写成 $\beta_i$ 形式的那个（即它是与前一条边成角），把顶点编号顺序沿着多边形方向向前或向后整体平移一个单位并不改变问题（只是循环下标），于是总能在某一步把该最小角写成某一项 $\beta_i$ 的形式。更直观地看：多边形的顶点是循环排列的，若某条边的两个端点中有一个端点给出的角 $\le\frac\pi2-\frac\pi n$，那么以合适的循环编号，该角就是某个 $\beta_i$。因此存在某个 $i$ 使得 $\beta_i\le\dfrac\pi2-\dfrac\pi n$。

综上，必有

```math
\min_{1\le i\le n}\angle PA_iA_{i+1}=\min_{1\le i\le n}\beta_i\le\frac\pi2-\frac\pi n,
```

证毕。 ∎
