# 最大值求解

- Conversation ID: `690ff446-fd6c-8324-b4f3-e39ace91fc50`

## Question

a^2 + b^2 = 1，a,b \in \mathbb{R} 。求 \max ((a + 1)(b + 2))

## Answer

把 $a=\cos t,\; b=\sin t$ 带入或用拉格朗日乘子法都可以。这里用拉格朗日乘子法：

要极值的目标函数

```math
F(a,b)= (a+1)(b+2)
```

在约束 $g(a,b)=a^2+b^2-1=0$ 下，解

```math
\nabla F=\lambda \nabla g
```

得到方程组

```math
\begin{cases}
b+2=2\lambda a,\\[4pt]
a+1=2\lambda b,\\[4pt]
a^2+b^2=1.
\end{cases}
```

解出实数解为两组：$(-1,0)$ 和

```math
(a,b)\approx(0.8171826465,\;0.5763788010).
```

在这两点处的函数值分别为 $0$ 和

```math
\max (a+1)(b+2)\approx 4.681750847566336.
```

所以最大值为约 $4.6817508476$，在
$(a,b)\approx(0.8171826465,\,0.5763788010)$ 处取得。
