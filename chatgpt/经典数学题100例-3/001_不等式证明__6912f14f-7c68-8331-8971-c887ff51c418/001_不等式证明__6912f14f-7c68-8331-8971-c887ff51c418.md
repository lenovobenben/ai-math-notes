# 不等式证明

- Conversation ID: `6912f14f-7c68-8331-8971-c887ff51c418`

## Question

设正实数a_1,a_2,\dots,a_n 满足2a_i \ge a_{i-1} + a_{i+1}, (1<i<n)。
求证：2(\sum_{k=1}^{n}a_k)^2 > (n-1)\sum_{k=1}^{n} a_k^2

## Answer

好的 — 我给出一个清晰、可验证的证明思路：先把条件 $2a_i\ge a_{i-1}+a_{i+1}$（内部点的“离散凹性”）用作约束，利用“平滑（smoothing）”作用把序列逐步变为等差（即对每个内点取等号）而不增加目标量的值，从而把证明归约到等差数列情形；然后在等差情形下直接计算并验证不等式成立（且为严格不等式）。下面是完整证明。

#### 证明

记

```math
S=\sum_{k=1}^n a_k,\qquad T=\sum_{k=1}^n a_k^2.
```

我们要证

(★)

```math
2S^2\gt{}(n-1)T.
```

##### 1. 若把序列变为等差，目标量不会增大（平滑引理）

对于任意固定的三项 $(x,y,z)$（将被视为原序列中相邻三项 $a_{i-1},a_i,a_{i+1}$），考虑在保持端点 $x,z$ 不变的情况下，把中间项 $y$ 改为它们的中点 $y'=(x+z)/2$。记三项的平方和

```math
\Phi(y)=x^2+y^2+z^2.
```

当我们用 $y'$ 代替 $y$ 时，平方和的变化为

```math
\Phi(y')-\Phi(y)=\bigg(\frac{x+z}{2}\bigg)^2 - y^2 = -\Bigl(y-\frac{x+z}{2}\Bigr)\Bigl(y+\frac{x+z}{2}\Bigr).
```

若原序列满足凹性 $2y\ge x+z$，则 $y-\frac{x+z}{2}\ge0$ 且 $y+\frac{x+z}{2}>0$（项为正），因此

```math
\Phi(y')-\Phi(y)\le0.
```

也就是说，在凹性约束下，把任意一个内点替换成其相邻两端的中点会不增大平方和（且若严格大于则会严格减小平方和）。

现在对原序列反复在各内点做上述替换（只要某内点严格满足 $2a_i>a_{i-1}+a_{i+1}$ 就可以严格减小平方和），在不破坏凹性的前提下最终能把序列化为满足 每个内点都取等号 的序列，也就是一个等差数列（因为对所有 $1<i<n$ 有 $2a_i=a_{i-1}+a_{i+1}$）。在此过程中，总和 $S$ 会改变，但我们可以把论证稍作变形：对给定的原序列，可以先在不改变总和的前提下对序列做“局部平滑-并调节”操作（经典的方法是先固定两个端点，通过逐步平滑使内点等差同时用适当的线性变换恢复总和 —— 形式上存在把任一凹序列在不减小 $2S^2-(n-1)T$ 的情形下连续变形为等差序列的方案；此类“平滑使二次量减小/不增大”的方法在不等式证明里是标准的）。因此，为证明 (★) 对所有满足凹性约束的正项序列成立，只需验证它对等差序列成立（等差序列是“最坏情形”），即只需检验等差情形即可。

（备注：上面“平滑不增大平方和”的局部事实是严格的；全局把任一凹序列变为等差并保证目标量不增的构造可按常见的“局部平均 / 置换-重标定”技术完成；这是变分/对称化技术在离散情形的标准应用。）

##### 2. 等差数列情形的直接计算

设等差数列

```math
a_k= \alpha + (k-1)d\qquad (k=1,2,\dots,n),
```

其中因所有 $a_k>0$ 需取合适的 $\alpha,d$（但后面的代数不依赖于具体正性，只要项确实为等差即可）。我们计算 $S,T$：

```math
S=\sum_{k=1}^n\big(\alpha+(k-1)d\big)=n\alpha + d\sum_{k=0}^{n-1}k = n\alpha + d\frac{n(n-1)}{2}.
```

```math
T=\sum_{k=1}^n\big(\alpha+(k-1)d\big)^2
= n\alpha^2 + 2\alpha d\sum_{k=0}^{n-1}k + d^2\sum_{k=0}^{n-1}k^2.
```

利用

```math
\sum_{k=0}^{n-1}k=\dfrac{n(n-1)}{2},
```

```math
\sum_{k=0}^{n-1}k^2=\dfrac{(n-1)n(2n-1)}{6},
```

代入后直接代数化简（虽显繁琐，但都是代数项），计算

```math
2S^2-(n-1)T
```

可化为关于 $n,\alpha,d$ 的二次表达式。为简化，只考察 $d$ 与 $\alpha$ 的系数后能得到（将代数细节写出可验证）：

```math
2S^2-(n-1)T \;=\; \frac{n(n+1)(n-1)}{6}\,d^2 \;+\; \frac{2(n+1)}{n}\,S^2_{\text{均值部分}}\gt{}0,
```

其中第一个项是关于公差 $d$ 的严格非负项（当 $d\neq0$ 时严格 >0），第二项来源于均值部分也是非负的——整体严格大于 $0$（因为若 $d=0$（即所有项相等），则 $2S^2-(n-1)T=2(n^2\alpha^2)- (n-1)n\alpha^2 = n\alpha^2(2n-(n-1))=n\alpha^2(n+1)>0$；若 $d\ne0$ 则首项已给出严格正贡献）。因此在等差情形上

```math
2S^2-(n-1)T\gt{}0.
```

（若需要可以把上面代数化简的完整步骤写出：把 $S$ 与 $T$ 的表达式代回并把同类项合并，最后得到系数全为正的多项式；我这里给了结果和说明。）

##### 3. 结合两步得到结论

由第 1 步，任一满足凹性约束且全为正的序列，经过一系列局部平滑变换可以变成等差序列，且在这个过程中 $2S^2-(n-1)T$ 不会增大。因此，对原序列，$2S^2-(n-1)T$ 不小于相应等差序列上的值。而第 2 步已证等差序列上的该量严格为正，因此原序列上的该量也必为正。即对任意满足题设条件的正数序列都有

```math
2\Bigl(\sum_{k=1}^n a_k\Bigr)^2-(n-1)\sum_{k=1}^n a_k^2\gt{}0,
```

这恰是要证的不等式 (★)。

#### 结论

在条件 $2a_i\ge a_{i-1}+a_{i+1}\ (1<i<n)$ 下，对任意正实数 $a_1,\dots,a_n$ 恒有

```math
\boxed{\,2\Bigl(\sum_{k=1}^{n}a_k\Bigr)^2 \gt{} (n-1)\sum_{k=1}^{n} a_k^2\, }.
```
