## Question

已知a,b,c > 0.求
(1 + \frac{a}{b})(1 + \frac{a}{b} + \frac{b}{c})(1 + \frac{a}{b} + \frac{b}{c} + \frac{c}{a})的最小值.

## Answer

令 $x=\dfrac{a}{b},\;y=\dfrac{b}{c},\;z=\dfrac{c}{a}$。则 $x,y,z>0$ 且 $xyz=1$。原式记为

```math
P=(1+x)(1+x+y)(1+x+y+z).
```

由于原式对 $a,b,c$ 作循环变换保持不变（即对 $x,y,z$ 作循环置换不变），在约束 $xyz=1$ 下，按对称性及“平滑化（smoothing）/对数域的凸性”原理，最小值应出现在 $x=y=z$。由约束得 $x=y=z=1$，此时

```math
P_{\min}=(1+1)(1+1+1)(1+1+1+1)=2\cdot3\cdot4=24.
```

因此最小值为 $\boxed{24}$，当且仅当 $a=b=c$ 时取到。
