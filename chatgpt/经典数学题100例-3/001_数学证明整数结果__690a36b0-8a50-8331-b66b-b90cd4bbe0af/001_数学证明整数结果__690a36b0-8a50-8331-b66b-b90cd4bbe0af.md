# 数学证明整数结果

- Conversation ID: `690a36b0-8a50-8331-b66b-b90cd4bbe0af`

## Question

a,b,c为正整数。证明：如果\frac{\sqrt[]{3}a+b }{\sqrt[]{3}b+c } 为有理数，那么 \frac{a^2+b^2+c^2}{a+b+c} 为整数

## Answer

设

```math
r=\frac{\sqrt3\,a+b}{\sqrt3\,b+c}\in\mathbb Q.
```

由此有

```math
\sqrt3\,a+b=r(\sqrt3\,b+c).
```

移项得

```math
\sqrt3\,(a-rb)=rc-b.
```

注意到 $r$ 是有理数，且 $a,b,c$ 为整数，故右端 $rc-b$ 为有理数；若 $a-rb\neq0$ 则右端除以左端得到 $\sqrt3=\dfrac{rc-b}{a-rb}$ 为有理数，与 $\sqrt3$ 无理矛盾。因此必须同时有

```math
a-rb=0,\qquad rc-b=0.
```

由此 $r=\dfrac a b=\dfrac b c$，所以

```math
b^2=ac.
```

现在用 $b^2=ac$ 来计算

```math
\frac{a^2+b^2+c^2}{a+b+c}.
```

令 $a=du^2,\; b=duv,\; c=dv^2$（可取正整数 $d,u,v$，并可令 $\gcd(u,v)=1$），则

```math
\frac{a^2+b^2+c^2}{a+b+c}
=\frac{d^2(u^4+u^2v^2+v^4)}{d(u^2+uv+v^2)}
=\frac{d\,(u^4+u^2v^2+v^4)}{u^2+uv+v^2}.
```

但注意恒等式

```math
(u^2+uv+v^2)(u^2-uv+v^2)=u^4+u^2v^2+v^4.
```

因此

```math
\frac{u^4+u^2v^2+v^4}{u^2+uv+v^2}=u^2-uv+v^2
```

为整数，进而原式等于 $d(u^2-uv+v^2)$，也是整数。

综上，当 $\dfrac{\sqrt3\,a+b}{\sqrt3\,b+c}$ 为有理数时，必有 $b^2=ac$，从而 $\dfrac{a^2+b^2+c^2}{a+b+c}$ 为整数。证毕。 □
