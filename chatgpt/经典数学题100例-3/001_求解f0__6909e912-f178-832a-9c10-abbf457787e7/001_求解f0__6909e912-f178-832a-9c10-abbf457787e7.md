# 求解f(0)

- Conversation ID: `6909e912-f178-832a-9c10-abbf457787e7`

## Question

对于一个连续函数 f:\mathbb{R} \to \mathbb{R}，满足对任何的x都有性质f(f(x))=af(x)+bx，其中a,b\in (0,\frac{1}{2} )。求f(0)=?

## Answer

令 $x_0=0,\;x_1=f(0)$，并定义迭代 $x_{n+1}=f(x_n)$.
由题给关系对任意 $x$ 成立，代入 $x=x_n$ 得

```math
x_{n+2}=f(f(x_n))=a f(x_n)+b x_n = a x_{n+1}+b x_n.
```

于是 $(x_n)$ 满足二阶线性递推

```math
x_{n+2}=a x_{n+1}+b x_n,\qquad x_0=0,\;x_1=f(0).
```

特征方程为 $r^2-ar-b=0$，其根为

```math
r_{1,2}=\frac{a\pm\sqrt{a^2+4b}}{2}.
```

因为 $a,b\in(0,\tfrac12)$，有 $b<1-a$，从而

```math
r_1=\frac{a+\sqrt{a^2+4b}}{2}\lt{}1,
```

且

```math
r_2=\frac{a-\sqrt{a^2+4b}}{2}\gt{}-1.
```

因此 $|r_1|<1$ 且 $|r_2|<1$，所以对任意初值 $x_1=f(0)$，由通解

```math
x_n=\alpha r_1^n+\beta r_2^n
```

可得 $x_n\to0$（当 $n\to\infty$）。

因为 $x_n\to0$ 且 $f$ 在实数上连续，故

```math
f(0)=\lim_{n\to\infty}f(x_n)=\lim_{n\to\infty}x_{n+1}=0.
```

因此 $\boxed{f(0)=0}$。
