# 最小值求解

- Conversation ID: `690be778-8af8-8333-8c61-17997ba6316c`

## Question

x,y,z为正实数，且满足x+y^2+3z^3=8。求1/x+16/y+9/z的最小值

## Answer

把这个最值问题用拉格朗日乘数法来做。

设

```math
F=\frac{1}{x}+\frac{16}{y}+\frac{9}{z}+\lambda\bigl(x+y^{2}+3z^{3}-8\bigr).
```

对 $x,y,z$ 求偏导并令其为0：

```math
\frac{\partial F}{\partial x}=-\frac{1}{x^{2}}+\lambda=0\quad\Rightarrow\quad\lambda=\frac{1}{x^{2}},
```

```math
\frac{\partial F}{\partial y}=-\frac{16}{y^{2}}+2\lambda y=0\quad\Rightarrow\quad 2\lambda y^{3}=16\Rightarrow y^{3}=\frac{8}{\lambda},
```

```math
\frac{\partial F}{\partial z}=-\frac{9}{z^{2}}+9\lambda z^{2}=0\quad\Rightarrow\quad \lambda z^{4}=1\Rightarrow z^{4}=\frac{1}{\lambda}.
```

由上述可得（用 $\lambda$ 表示）：

```math
x=\lambda^{-1/2},\qquad y=2\lambda^{-1/3},\qquad z=\lambda^{-1/4}.
```

代入约束条件 $x+y^{2}+3z^{3}=8$：

```math
\lambda^{-1/2}+4\lambda^{-2/3}+3\lambda^{-3/4}=8.
```

显然 $\lambda=1$ 满足上式，此时

```math
x=1,\quad y=2,\quad z=1.
```

代回目标函数得

```math
\frac{1}{x}+\frac{16}{y}+\frac{9}{z}=1+8+9=18.
```

因为当任一变量趋近 0 时目标函数趋于 $+\infty$，且可行域是有界的，因此最小值在内点取得，拉格朗日得到的临界点即为全局最小点。

结论：最小值为 $\boxed{18}$，当且仅当 $(x,y,z)=(1,2,1)$ 时取得。
