# 正整数n求解

- Conversation ID: `6915d58a-f53c-8326-b4b4-3c073b505d40`

## Question

求所有正整数n,使得存在正偶数a满足\prod_{k=1}^{n}(a^k-1)是完全平方数.

## Answer

答案：只有 $n=1$ 或 $n=2$。

证明：

$n=1,2$ 可行。

$n=1$：取任意偶数 $a$ 使得 $a-1$ 为完全平方数即可（比如 $a=10$，有 $a-1=9=3^2$），因此 $n=1$ 存在正偶数 $a$ 满足条件。

$n=2$：

```math
\prod_{k=1}^{2}(a^k-1)=(a-1)(a^2-1)=(a-1)^2(a+1).
```

因此只要 $a+1$ 为完全平方数就行。取 $a=8$，有 $(8-1)^2(8+1)=7^2\cdot9=21^2$，所以 $n=2$ 也可行。

若 $n\ge3$ 不可能。

设存在偶数 $a$ 使得

```math
P_n=\prod_{k=1}^n (a^k-1)
```

为完全平方。对 $k\ge1$ 写 $a^k-1=(a-1)\,S_k$ 其中 $S_k=a^{k-1}+a^{k-2}+\cdots+1$。于是

```math
P_n=(a-1)^n\prod_{k=1}^n S_k.
```

取一个 $k$ 满足 $3\le k\le n$。根据 Zsigmondy 定理（Bang—Zsigmondy），对于每个 $k\ge3$，存在一个素数 $p$（称为 $a^k-1$ 的原始素因子），使得 $p\mid a^k-1$ 且对所有 $j<k$ 都有 $p\nmid a^j-1$。因此这个素数 $p$ 只出现在第 $k$ 项 $a^k-1$ 中，不出现在其他因子里。

设 $f(x)=x^k-1$，则 $p\mid f(a)$。若 $p$ 在 $f(a)$ 中的重数大于 $1$，则同时有 $p\mid f'(a)$。但 $f'(x)=k x^{k-1}$，所以若 $p\mid f'(a)$ 则 $p\mid k$（因为 $p\nmid a$——若 $p\mid a$ 则 $p\mid a^k-1$ 不可能）。然而 Zsigmondy 的原始素因子满足 $p>k$，从而不可能整除 $k$。因此 $p\nmid f'(a)$，说明 $p$ 在 $a^k-1$ 中的幂次为 $1$。

综上，存在一个素数 $p$ 在整积 $P_n$ 中的幂次为 $1$（奇数），这与 $P_n$ 为完全平方矛盾。因此不可能存在 $n\ge3$ 满足条件。

因此所有满足条件的正整数 $n$ 仅为 $1,2$。▢
