# 旋转后函数变化

- Conversation ID: `690f37d2-331c-8320-b52d-cfd0b7caa075`

## Question

\sin x 顺时针旋转45度后，表达式是什么？还是函数吗？

## Answer

非常好的问题 👍

我们来仔细分析一下：

##### 一、题意解析

你给的函数是

```math
y = \sin x
```

所谓“顺时针旋转45°”，指的是在平面直角坐标系中，将整条曲线绕原点 顺时针旋转45°。

旋转后我们要找出新坐标系 $(x', y')$ 下的表达式 $y' = f(x')$。

##### 二、坐标变换公式

若点 $(x, y)$ 绕原点 顺时针旋转角度 $\theta$，则有：

```math
\begin{cases}
x' = x\cos\theta + y\sin\theta \\
y' = -x\sin\theta + y\cos\theta
\end{cases}
```

这里 $\theta = 45° = \pi/4$。

##### 三、代入原函数 $y = \sin x$

```math
\begin{cases}
x' = x\cos\frac{\pi}{4} + \sin x \sin\frac{\pi}{4} = \frac{\sqrt{2}}{2}(x + \sin x) \\
y' = -x\sin\frac{\pi}{4} + \sin x \cos\frac{\pi}{4} = \frac{\sqrt{2}}{2}(-x + \sin x)
\end{cases}
```

##### 四、是否还能写成 $y' = f(x')$？

注意：
上式是一个参数方程，以 $x$ 为参数，描述旋转后的曲线。

要想写成 $y' = f(x')$，必须能把 $x'$ 表达为单值函数的 $x$，但实际上 $\sin x$ 是周期函数，经过旋转后，曲线在某些区间中会出现“竖直段”或“多值性”——也就是说，它 不再是函数（不满足单值性）。

##### 五、结论

旋转45°后的曲线表达式为：

```math
\begin{cases}
x' = \frac{\sqrt{2}}{2}(x + \sin x) \\
y' = \frac{\sqrt{2}}{2}(\sin x - x)
\end{cases}
```

这是一条以 $x$ 为参数的曲线。

⚠️ 它不再是函数形式 $y' = f(x')$，因为旋转后每个 $x'$ 可能对应多个 $y'$。

##### 六、补充说明

若只旋转很小角度（比如几度），局部仍可近似看作函数；

但严格意义上，除了 0°、180°、360° 这类特殊旋转外，$\sin x$ 旋转后都不再是函数。

## Question

它依然是函数，而且是非周期的，严格递减的函数。你可以证明这一点。

## Answer

你说得没错 — 旋转后这条曲线确实可以看成一个（全局）函数，并且非周期、严格单调（递减）。下面给出一个完整且尽量简洁的证明。

##### 1) 参数方程（已知）

以原参数 $x$（原曲线上点的横坐标）表示旋转后点的坐标（顺时针旋转 $\pi/4$）：

```math
\begin{cases}
X(x)=x'=\dfrac{\sqrt2}{2}\bigl(x+\sin x\bigr),\\[6pt]
Y(x)=y'=\dfrac{\sqrt2}{2}\bigl(\sin x - x\bigr).
\end{cases}
```

我们要证明：$X$ 关于参数 $x$ 是严格单调增且可逆（从而可以把 $x$ 作为 $x'$ 的单值逆像），并且 $Y$ 关于参数 $x$ 是严格单调减。由此得到旋转曲线可以表示成全域单值函数 $y'=G(x')$（严格递减、非周期）。

##### 2) $X(x)$ 单调性（可逆性）

计算导数：

```math
X'(x)=\frac{\sqrt2}{2}\bigl(1+\cos x\bigr).
```

注意 $1+\cos x\ge0$，且 $1+\cos x=0$ 的点恰为 $x=(2k+1)\pi$（孤立点）。因此 $X'(x)\ge0$ 且不可能在任何有长度的区间上恒为 0（因为 $1+\cos x$ 只是离散点为 0，在这些点以外严格 >0）。

从微积分基本事实：若连续函数导数 $X'(x)\ge0$ 且在任意长度正的区间内并非处处为 0，则对任意 $a<b$，

```math
X(b)-X(a)=\int_a^b X'(t)\,dt\gt{}0.
```

所以当 $a<b$ 时 $X(a)<X(b)$。即 $X$ 在整个实线是严格递增的，故它是单射并且在 $\mathbb R$ 上可逆：存在唯一的逆函数 $X^{-1}:\mathbb R\to\mathbb R$。

（直观上，虽然 $X'(x)=0$ 在离散点出现，但这不会破坏整体严格单调性——导数为 0 的点没有“扁平成段”的区间，所以整体仍严格上升。）

##### 3) $Y(x)$ 单调性（关于参数 $x$）

计算 $Y'(x)$：

```math
Y'(x)=\frac{\sqrt2}{2}\bigl(\cos x-1\bigr)\le 0,
```

且 $\cos x-1=0$ 的点是 $x=2k\pi$（同样是离散点）。由与上面相同的理由：$Y'(x)\le0$ 且不在任何有长度区间内恒为 0，所以对任意 $a<b$，

```math
Y(b)-Y(a)=\int_a^b Y'(t)\,dt\lt{}0.
```

因此 $Y$ 关于参数 $x$ 是严格递减的：若 $a<b$ 则 $Y(a)>Y(b)$。

##### 4) 从参数表示得到 $y'$ 关于 $x'$ 的函数且严格递减

既然 $X$ 在 $\mathbb R$ 上严格递增并可逆，我们可以把参数 $x$ 用 $x'$ 表示：

```math
x = X^{-1}(x').
```

于是旋转后的曲线可以表示为单值函数

```math
y' = G(x') := Y\bigl(X^{-1}(x')\bigr).
```

因为 $X^{-1}$ 是严格递增且 $Y$ 是严格递减，所以复合 $G$ 是严格递减的（增的逆 + 减的原 -> 减）。因此旋转曲线确实是一个全域定义的、严格递减的函数。

##### 5) 非周期性

若 $G$ 周期为 $T\ne0$，则回到参数层面会产生矛盾。观察 $X(x)=\tfrac{\sqrt2}{2}(x+\sin x)$ 中的线性项 $x$ 导致 $X(x+T)-X(x)=\tfrac{\sqrt2}{2}\bigl(T+\sin(x+T)-\sin x\bigr)$。若这一差与 $x$ 无关并为 0（即产生周期），则必须有 $T+\sin(x+T)-\sin x\equiv0$ 对所有 $x$ 成立。令 $x$ 变化，这不可能（左边为恒等式则必须 $T=0$ 且 $\sin(x+T)=\sin x$ 恒成立），故无非零周期。更直接：由于包含不可消去的线性项 $x$，函数不能周期化。因此 $G$（以及参数表达的 $X,Y$）均非周期。

##### 6) 总结（结论）

旋转后曲线可写为全域单值函数 $y' = G(x')$。

这个函数 $G$ 在整个实线上 严格递减。

$G$ 不周期（非周期函数）。

## Question

你起初为什么会想当然的认为它不是函数？是偷懒了吗？

## Answer

哈哈，这个问题问得非常好，而且非常到点子上。
我来认真回答一下——不是“偷懒”，而是一种常见的视觉错觉 + 几何直觉误导造成的。

##### 🧠 一、我最初的“错觉”来源

当我们听到“把一个函数图像旋转一个非 90° 的角度”，脑子里一般会自动浮现出：

“旋转后，会出现竖直段，所以不再是一条函数曲线（对某个 $x'$，可能对应多个 $y'$）。”

这种印象是由最常见的例子造成的，比如：

把 $y = x^2$ 顺时针旋转 45°，确实不再是函数；

把 $y = \sin x$ 逆时针旋转 90°，也不再是函数。

于是我按照这个经验直接套用，没有立刻去分析具体导数的单调性 —— 这确实是一种**“想当然”式的几何偷懒**。

##### 🧩 二、为什么这个例子其实是特例

然而，$\sin x$ 这个函数有一个特别的性质：

它的导数在绝大多数地方小于 1；

当你顺时针旋转 $45°$ 时，本质上是把曲线投影到某个严格单调方向上；

这使得旋转后的横坐标 $x' = \frac{\sqrt2}{2}(x + \sin x)$ 仍然严格单调增，因此它不会出现“竖直段”。

也就是说，这个例子恰好落在一个非常特殊的区间：旋转角度不大、斜率的绝对值没有超过旋转角度对应的临界值，从而保持了函数性。

##### 🔍 三、总结反思

所以你说得对：

我起初的判断是基于惯性思维，而没有立刻去验证单调性。
严格来说，这属于**“几何直觉的偷懒”**，而非推理上的严谨。

如果我第一步就写出参数式并看一眼 $dx'/dx = \frac{\sqrt2}{2}(1+\cos x) > 0$，我就能立即发现它依然是函数、而且是严格递减的非周期函数。

##### 💡 四、这其实是一个很好的“陷阱题”

因为：

绝大多数函数旋转后确实不再是函数；

但 $\sin x$ 顺时针旋转 45° 是极少数例外；

所以这道题非常棒：可以区分“几何直觉”与“严格分析”。
