# 最小值求解

- Conversation ID: `690bf309-5ccc-8330-8eab-d5edc4008d16`

## Question

x,y为实数，且x^2+y^2=4。求\sqrt{40-12y} + \sqrt{5-2x} 的最小值

## Answer

把 $x^2+y^2=4$ 代入参数化：设

```math
x=2\cos\theta,\qquad y=2\sin\theta\quad(\theta\in\mathbb R).
```

则目标函数

```math
F(\theta)=\sqrt{40-12y}+\sqrt{5-2x}=2\sqrt2\sqrt{5-3\sin\theta}+\sqrt{5-4\cos\theta}.
```

对 $\theta$ 求极值（可用拉格朗日或对参数 $\theta$ 求导）：

```math
F'(\theta)=-\frac{3\sqrt2\cos\theta}{\sqrt{5-3\sin\theta}}+\frac{2\sin\theta}{\sqrt{5-4\cos\theta}}=0.
```

整理并平方消去根号，结合 $x^2+y^2=4$（即 $\cos^2\theta+\sin^2\theta=1$）解得极值点对应的 $x,y$（数值解）大约为

```math
x\approx0.6869735571,\qquad y\approx1.8783150246.
```

此时代回原式得到最小值

```math
\sqrt{40-12y}+\sqrt{5-2x}=\sqrt{37}.
```

因此该表达式的最小值是 $\boxed{\sqrt{37}}$，在约 $(x,y)\approx(0.6869736,\;1.8783150)$ 处取得。
