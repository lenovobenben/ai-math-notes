# 面积比计算

- Conversation ID: `6910ba8f-9d88-832c-b068-b721c24a0bff`

## Question

已知\triangle ABC。D在BC上，且BD=\frac{1}{3}BC。E在CA上，且CE=\frac{1}{3} CA。F在AB上，且AF=\frac{1}{3}AB。 AD，BE，CF三条线相交于G，H，I三点。求\frac{S_{\triangle GHI}}{S_{\triangle ABC}}

## Answer

把题目做成坐标几何最直接。取
$A=(0,0),\;B=(3,0),\;C=(0,3)$（这样各边长度取 3，便于按三等分取点）。

由条件得

```math
F\in AB,\ AF=\tfrac13AB\Rightarrow F=(1,0),
```

```math
D\in BC,\ BD=\tfrac13BC\Rightarrow D=(2,1),
```

```math
E\in CA,\ CE=\tfrac13CA\Rightarrow E=(0,2).
```

三条直线方程：

```math
AD:\ y=\tfrac12 x,
\quad BE:\ y=-\tfrac23 x+2,
\quad CF:\ y=-3x+3.
```

求两两交点（即题中 $G,H,I$）：

```math
G=BE\cap CF=\Big(\tfrac{3}{7},\tfrac{12}{7}\Big),\quad
H=CF\cap AD=\Big(\tfrac{6}{7},\tfrac{3}{7}\Big),\quad
I=AD\cap BE=\Big(\tfrac{12}{7},\tfrac{6}{7}\Big).
```

用行列式（或向量叉积）算面积：

```math
S_{\triangle ABC}=\frac{1}{2}\cdot3\cdot3=\frac{9}{2},
```

```math
S_{\triangle GHI}=\frac{9}{14}.
```

因此

```math
\frac{S_{\triangle GHI}}{S_{\triangle ABC}}=\frac{\tfrac{9}{14}}{\tfrac{9}{2}}=\frac{1}{7}.
```

答案：$\displaystyle \boxed{\dfrac{1}{7}}$。
