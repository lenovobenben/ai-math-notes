# 正三角形最小面积

- Conversation ID: `6911361b-53f4-8332-ac28-f996296cb8a9`

## Question

在边长为2的正方形ABCD中，其内接正三角形的面积最小值为多少？

## Answer

把正方形放在坐标系上，取 $A(0,0),B(2,0),C(2,2),D(0,2)$。设内接（且三个顶点落在三条不同边上）的正三角形的三个顶点分别在下、右、上三条边上，记为

```math
P=(x,0),\quad Q=(2,y),\quad R=(u,2),
```

其中 $0\le x,y,u\le2$。令三角形边长为 $s$。

以 $P$ 与 $Q$ 为一条边，向外旋转向量 $\overrightarrow{PQ}$ $+60^\circ$ 得到第三点应为 $R$，所以由旋转分量的 $y$-坐标得（用 $\sqrt3$ 表示 $\sqrt3$）：

```math
\frac{\sqrt3}{2}(2-x)+\frac{1}{2}y=2.
```

两边乘 2 得

```math
\sqrt3(2-x)+y=4,
```

所以

```math
y=4-2\sqrt3+\sqrt3 x.
```

边长的平方为

```math
s^2=(2-x)^2+y^2=(2-x)^2+\bigl(4-2\sqrt3+\sqrt3 x\bigr)^2.
```

对 $x$ 求导并令导数为零，可得唯一驻点

```math
x=2-\sqrt3.
```

把它代入上式计算 $s^2$：

```math
s^2=(2-(2-\sqrt3))^2+\bigl(4-2\sqrt3+\sqrt3(2-\sqrt3)\bigr)^2
= (\sqrt3)^2 + (2)^2 = 3+1=4,
```

因此 $s=2$。

正三角形面积为 $\dfrac{\sqrt3}{4}s^2$，代入 $s^2=4$ 得最小面积

```math
\boxed{\sqrt3}
```

约等于 $1.7320508$。

（通过对称性，若把三顶点落在其它三条边的情形，会得到同样的结论。）
