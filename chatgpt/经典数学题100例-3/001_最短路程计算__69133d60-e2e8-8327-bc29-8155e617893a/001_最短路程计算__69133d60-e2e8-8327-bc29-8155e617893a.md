# 最短路程计算

## Question

如图，荆州古城河在 CC' 处直角转弯，河宽均为5米，从A处到达B处，须经两座桥: DD'，EE'（桥宽不计）。设护城河以及两座桥都是东西、南北方向的，A,B在东西方向上相距 65 米，南北方向上相距 85 米。求：ADD'E'EB 的最短路程是多少米？

## Answer

最短路长是 110 米。

做法（反射/平移法，简要说明）：

设从 $A$ 到 $B$ 的折线路径为

```math
A\to D( d,5)\to D'(d,0)\to(0,e)\to(-5,e)\to B,
```

其中两个过河段 $D D'$ 与 $E E'$ 各为 5 米，所以总长

```math
L=\underbrace{|A-(d,5)|+| (d,0)-(0,e)|+|(-5,e)-B|}_{S}+5+5.
```

把整个折线的中间两段用反射/平移消去：把点 $B$ 向东北方向平移 $(5,5)$（相当于把穿过两条护城河的“空隙”补回去），记为 $B' = B+(5,5)$。则上面可变部分的最短值恰好等于 $A$ 到 $B'$ 的直线距离，即

```math
S_{\min} = |A-B'|.
```

题中东西方向相差 $65$ m，南北方向相差 $85$ m，平移后东西差为 $65-5=60$，南北差为 $85-5=80$，所以

```math
S_{\min}=\sqrt{60^2+80^2}=100.
```

再加上两座桥的固定宽度 $5+5=10$，得到最短总路程

```math
L_{\min}=100+10=110\ \text{米}.
```
