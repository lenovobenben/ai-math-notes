# 四数替换证明

- Conversation ID: `6909df0a-57b0-8333-96d2-3179148bcd07`

## Question

给出四个正整数(a,b,c,d)不断做替换(|a-b|,|b-c|,|c-d|,|d-a|)。证明：在有限的替换次数以后一定止于全部是零。

## Answer

下面给出一个干净的证明（中文）。

把一次替换写成映射 $P:\mathbb Z^4_{\ge0}\to\mathbb Z^4_{\ge0}$：

```math
P(a,b,c,d)=(|a-b|,\;|b-c|,\;|c-d|,\;|d-a|).
```

注意两点关键性质：

齐次性（线性放缩）：
对任意非负整数 $\lambda$ 和向量 $v=(a,b,c,d)$ 有

```math
P(\lambda v)=\lambda P(v),
```

因为 $|\lambda a-\lambda b|=\lambda|a-b|$。

模 $2$ 的线性化：把运算对每一坐标取模 $2$，有

```math
|x-y|\equiv x+y \pmod2
```

（在模 $2$ 下减法和加法相同，且绝对值的符号影响在模 $2$ 下消失）。因此，对模 $2$ 的向量空间来说，$P$ 相当于线性算子 $M$：

```math
M(a,b,c,d)=(a-b,b-c,c-d,d-a)
```

（在 $\mathbb F_2$ 中写成加法形式）。

下面用这两点做归纳提升（Hensel 型思路）：

步骤 A（基步）：在模 $2$ 上把 $M$ 的第四次迭代为零。计算或观察可知（可直接算矩阵幂或在多项式环上分析）$M^4$ 的所有系数都是偶数，因此

```math
M^4\equiv 0\pmod2.
```

于是对任意整数向量 $v$,

```math
P^4(v)\equiv 0\pmod2,
```

即 $P^4(v)$ 的每一坐标都是偶数。

步骤 B（归纳提升）：假设对某个 $k\ge0$ 已知经过 $4k$ 步替换后，向量的每个坐标都能被 $2^k$ 整除（这是成立的，$k=0$ 时显然）。写初态为 $v$。令 $v^{(4k)}=P^{4k}(v)$，由归纳假设可写成

```math
v^{(4k)}=2^k w
```

对某整数向量 $w$。由于 $P$ 齐次，

```math
P^{4}(v^{(4k)})=P^4(2^k w)=2^k P^4(w).
```

由步骤 A，$P^4(w)$ 的各坐标是偶数，因此 $P^{4}(v^{(4k)})$ 的各坐标可被 $2^{k+1}$ 整除。也就是说经过再多 $4$ 步（总共 $4(k+1)$ 步）后，所有坐标都被 $2^{k+1}$ 整除。由数学归纳法，命题对所有 $k$ 都成立：经过 $4k$ 步以后，四个数都能被 $2^k$ 整除。

结论：设初始四数的最大值为 $N$。取 $k$ 使得 $2^k>N$（这能做到，因为 $2^k\to\infty$）。按上面结论，经过 $4k$ 步以后，四个数同时被 $2^k$ 整除，但它们又都不超过 $N$，故只能全为 $0$。因此序列在有限步后必然达到全零向量。

（补充说明：这就是著名的 Ducci 序列的结论——当项数 $n$ 为 $2^m$ 的时候，任意整数初值的 Ducci 序列最终都会落入全零序列；这里 $n=4$ 是 $2^2$ 的情形。）✔️

## Question

```math
如果是n=3呢
```

## Answer

当 $n=3$ 时，情况和 $n=4$ 完全不同。让我们仔细分析：

我们考虑三元组 $(a,b,c)$ 的 Ducci 序列：

```math
(a,b,c) \mapsto (|a-b|, |b-c|, |c-a|).
```

##### 关键观察

模 2 的性质：同样可以考虑模 2。

对 $(a,b,c)$ 在 $\mathbb{F}_2^3$ 上的映射

```math
(a,b,c) \mapsto (a+b, b+c, c+a) \pmod 2
```

是线性的。

这个线性映射的特征值并不保证经过有限步后归零。

反例：

取 $(a,b,c) = (1,2,3)$：

```math
(1,2,3) \mapsto (1,1,2) \mapsto (0,1,1) \mapsto (1,0,1) \mapsto (1,1,0) \mapsto (0,1,1)\dots
```

可以看到它进入了一个循环 非零序列：

```math
(0,1,1) \to (1,0,1) \to (1,1,0) \to (0,1,1) \to \dots
```

永远不会达到全零。

##### 结论

对 $n=3$ 的 Ducci 序列，不保证最终全为零。

可能进入一个非零循环，或者最终到达零，这取决于初值。

与 $n=4$ 不同，只有当 $n$ 是 $2^m$ 时，才有“最终必为零”的保证。
