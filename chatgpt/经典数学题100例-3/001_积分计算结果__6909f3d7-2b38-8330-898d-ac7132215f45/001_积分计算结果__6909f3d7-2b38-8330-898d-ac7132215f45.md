# 积分计算结果

- Conversation ID: `6909f3d7-2b38-8330-898d-ac7132215f45`

## Question

求积分：\int_{0}^{1} \frac{\ln (1+x)\ln (1-x)}{x} dx

## Answer

设

```math
F(a)=\int_0^1\frac{\ln(1+ax)\ln(1-x)}{x}\,dx,
```

则 $F(0)=0$ 且

```math
F'(a)=\int_0^1\frac{\ln(1-x)}{1+ax}\,dx.
```

用幂级数 $1/(1+ax)=\sum_{k\ge0}(-a)^k x^k$ 并代入得

```math
F'(a)=\sum_{k\ge0}(-a)^k\int_0^1 x^k\ln(1-x)\,dx.
```

已知
$\displaystyle\int_0^1 x^k\ln(1-x)\,dx=-\frac{H_{k+1}}{k+1}$（$H_n$ 为调和数），因此

```math
F'(a)=-\sum_{k\ge0}(-a)^k\frac{H_{k+1}}{k+1}.
```

对 $a$ 从 $0$ 积分到 $1$：

```math
F(1)= -\sum_{k\ge0}\frac{H_{k+1}}{(k+1)^2}(-1)^k
=\sum_{n=1}^\infty\frac{(-1)^n H_n}{n^2}.
```

利用已知级数值（经典恒等式）

```math
\sum_{n=1}^\infty\frac{(-1)^{\,n-1}H_n}{n^2}=\frac{5}{8}\zeta(3),
```

可得

```math
\int_0^1\frac{\ln(1+x)\ln(1-x)}{x}\,dx
=\sum_{n=1}^\infty\frac{(-1)^n H_n}{n^2}
=-\frac{5}{8}\zeta(3).
```

数值上约为 $-0.7512855644$。

## Question

这里面的ζ函数是黎曼ζ函数吗？

## Answer

是的 ✅，这里的 $\zeta(3)$ 指的正是 黎曼 ζ 函数（Riemann zeta function） 在 $s=3$ 处的取值：

```math
\zeta(s) = \sum_{n=1}^{\infty} \frac{1}{n^s}, \quad \text{当 } \mathrm{Re}(s) \gt{} 1.
```

即：

```math
\zeta(3) = \sum_{n=1}^{\infty} \frac{1}{n^3} \approx 1.2020569\ldots
```

它又被称为 阿佩里常数（Apéry's constant），因为 Roger Apéry 在 1978 年证明了 $\zeta(3)$ 是无理数。

所以你的积分结果：

```math
\int_{0}^{1} \frac{\ln(1+x)\ln(1-x)}{x}\,dx = -\frac{5}{8}\zeta(3)
```

中，$\zeta(3)$ 就是黎曼 ζ 函数在 3 处的值。
