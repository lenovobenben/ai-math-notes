# 最大值计算

- Conversation ID: `6915ef99-8ca4-832e-b1c5-6d745233fe06`

## Question

已知X是随机变量,满足0 \le X \le 1,且\mathbb{E}(X-\mathbb{E}X)^3=0.求\mathbb{E}(X-\mathbb{E}X)^4的最大值.

## Answer

答案：最大值为 $\displaystyle \frac{1}{16}$，且当且仅当 $P(X=0)=P(X=1)=\tfrac12$（即 $X$ 在 $\{0,1\}$ 上各以 $1/2$ 取值）时取到。

说明（要点）：

令 $\mu=\mathbb{E}X$。对任意分布有

```math
\mathbb{E}(X-\mu)^4=\int_0^1 (x-\mu)^4\,dF(x),
```

且约束条件 $\mathbb{E}(X-\mu)^3=0$ 可看作对分布的一个约束。

在有界区间上、在这些线性／多项式矩约束下，把质量集中到最少的点会使高阶矩达到极值（极端分布原理）。因此寻找极大值时可只考虑至多三个（事实上两个）点的分布。为了满足三阶中心距为零、并把四阶中心距尽可能做大，最优应为两个关于均值对称的质量点（这保证三阶矩为零），且要使它们之间距离尽量大以增大第四矩。

在区间 $[0,1]$ 上，两点关于均值对称且距离最大的情况是均值为 $\tfrac12$、质量点为 $0$ 和 $1$，各自概率 $\tfrac12$。此时

```math
\mathbb{E}(X-\mu)^4=\tfrac12\big(\tfrac12\big)^4+\tfrac12\big(-\tfrac12\big)^4=\big(\tfrac12\big)^4=\frac{1}{16}.
```

可直接验证该分布满足 $\mathbb{E}(X-\mu)^3=0$ 且使四阶中心距达到上界 $\tfrac{1}{16}$。

因此最大值为 $\boxed{\dfrac{1}{16}}$。
