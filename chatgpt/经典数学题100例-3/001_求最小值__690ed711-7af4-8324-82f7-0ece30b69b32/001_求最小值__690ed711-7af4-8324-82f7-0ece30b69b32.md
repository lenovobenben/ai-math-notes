# 求最小值

- Conversation ID: `690ed711-7af4-8324-82f7-0ece30b69b32`

## Question

求 \sqrt{x^2+1} + \sqrt{(2-x)^2+4}的最小值

## Answer

把函数记作

```math
f(x)=\sqrt{x^2+1}+\sqrt{(2-x)^2+4}
```

几何解释：令点 $A(0,1)$，变量点为 $P(x,0)$，点 $B(2,2)$。则 $\sqrt{x^2+1}=|PA|$，$\sqrt{(2-x)^2+4}=|PB|$。要求的是点 $P$ 在 $x$-轴上使 $PA+PB$ 最小。

用反射法：把 $B(2,2)$ 关于 $x$-轴反射得到 $B'(2,-2)$。则对任意 $P$ 有 $PA+PB=PA+PB' \ge |AB'|$，且当 $P$ 为线段 $AB'$ 与 $x$-轴的交点时取等号。

计算

```math
|AB'|=\sqrt{(2-0)^2+(-2-1)^2}=\sqrt{4+9}=\sqrt{13}.
```

交点对应的 $x$ 坐标：直线 $AB'$ 参数形式 $x=2t,\ y=1-3t$，令 $y=0$ 得 $t=\tfrac13$，于是 $x=\tfrac{2}{3}$。

结论：最小值为 $\boxed{\sqrt{13}}$（约 $3.6056$），在 $x=\tfrac{2}{3}$ 处取得。
