# 最小周长条件

- Conversation ID: `6911f899-6958-8325-827d-d88d8625090a`

## Question

一个锐角\triangle ABC，有内接\triangle DEF。求使\triangle DEF 周长最小的条件是什么？

## Answer

答案：当且仅当 $D,E,F$ 分别是三角形 $ABC$ 的三条高的垂足（即 $DEF$ 为 $ABC$ 的直心三角形 / 正交三角形，orthic triangle）时，内接三角形 $DEF$ 的周长取得最小值。（注意：这里要求 $ABC$ 为锐角三角形，才能保证三条高的垂足都落在对应边上。）

证明（变分/反射思想，常见且简洁的证明途径）：

记欲极小化的周长为

```math
P=DE+EF+FD,
```

其中点 $D\in BC,\;E\in CA,\;F\in AB$。

在极小情形下，若把例如点 $E$ 沿边 $CA$ 做一个微小移动，则 $P$ 的一阶变分应为 0。对点 $E$ 的位移作一阶展开可得到必要条件：经过 $E$ 的两条边段 $DE$ 和 $EF$ 与直线 $CA$ 所成角度相等（这正是“反射定律”或“等角条件”）——换言之，

```math
\angle(\,DE,\,CA)=\angle(\,EF,\,CA).
```

同理，对点 $F$ 与点 $D$ 也得到对应的等角条件：

```math
\angle(\,EF,\,AB)=\angle(\,FD,\,AB),\qquad
\angle(\,FD,\,BC)=\angle(\,DE,\,BC).
```

把上述“等角”用反射理解：将线段 $DE$ 关于 $CA$ 反射得到一条射线，等角条件说明这条反射射线与 $EF$ 共线。对其它两条边作同样反射，三个反射后的直线互相延长后会在同一点相交（反射-折线构造保证共点）。把这交点记为 $H$。

进一步观察反射关系可看出：$H$ 到三边 $BC,CA,AB$ 的垂线分别通过 $D,E,F$。即 $HD\perp BC,\;HE\perp CA,\;HF\perp AB$。因此 $H$ 是三角形 $ABC$ 的垂心，且 $D,E,F$ 恰好是三条高在边上的垂足 —— $DEF$ 正是 $ABC$ 的直心三角形（orthic triangle）。

由步骤 2 的必要条件可以逆向验证：当 $D,E,F$ 为三条高的垂足时，上述等角条件成立，从而局部变化无法使周长减小，因此它给出局部极小；因为该构造在锐角三角形中唯一，所以为全局最小。

结论：在锐角三角形 $ABC$ 中，所有内接且每个顶点分别位于三条边上的三角形中，周长最小者唯一为三角形 $ABC$ 的直心三角形（垂足三角形）。
