# 最大值求解

- Conversation ID: `6915ea78-873c-8330-9249-bbf3b6955ae3`

## Question

已知非负实数x_1 + x_2 + \dots + x_{2024} = 2024.求\prod_{i=1}^{2024}(x_{i}^2 + x_{i+1})的最大值.约定下标 \mod 2024.

## Answer

设 $n=2024$. 记

```math
P=\prod_{i=1}^{n}\bigl(x_i^2+x_{i+1}\bigr),\qquad x_i\ge0,\quad \sum_{i=1}^n x_i=n,
```

下标按模 $n$ 周期。

答案：最大值为

```math
\boxed{8^{\,n/2}=8^{1012}=2^{3036}},
```

且取等号的构造例：当 $n$ 为偶数时令

```math
x_1=2,\;x_2=0,\;x_3=2,\;x_4=0,\;\dots,\;x_{n-1}=2,\;x_n=0
```

（或任意循环移位、把 2 和 0 互换的等价排布），此时每一对相邻因子交替为 $4$ 与 $2$，每对的乘积为 $4\cdot2=8$，共有 $n/2$ 对，所以

```math
P=8^{\,n/2}=8^{1012}
```

下面给出说明（要点）——说明最大值出现在边界的极端分配，且交替分配为最优。

对此问题做拉格朗日乘数法可以得到内部驻点（所有 $x_i>0$）满足方程系

```math
\frac{2x_k}{x_k^2+x_{k+1}}+\frac{1}{x_{k-1}^2+x_k}=\lambda\quad(k=1,\dots,n).
```

显然 $x_1=\cdots=x_n=1$（即平均分配）是一个解，对应 $\lambda=\tfrac32$。但数值试验和二阶性状分析可看出这是一个驻点而非全局最大（事实上该点给出 $P=2^n$ 只是一个中等值）。

通过把质量在邻近位置上做“搬运”——即对某些相邻分量在保持和不变的条件下作微小调整——可以证明函数 $\ln P$ 在许多方向上是向外增大的，说明最优在可行域的边界。更直接的直观是：把总量集中到某些位置而让它们与相邻的零交替，会把各因子中的某些项 $x_i^2$ 放大（产生 $4$），同时相邻因子中的 $+x_{i+1}$ 项对零处仍给出常数（产生 $2$），交替排列能把“放大”和“保持”为每对贡献 $8$，从而整体乘积最大化。

在所有边界分配中（即有部分分量为 0）进行枚举/局部优化可以发现：若要求和为 $n$ 且 $x_i\ge0$，最佳是取一半分量等于 $2$ 且交替分布，其它为 $0$。这是因为在固定邻三元或成对约束下，把质量从两个相邻非零点挪到同一位置可以提高该段对应因子乘积，直到一个点变为 0；重复此类操作最终得到交替的 $2,0,2,0,\dots$。

因此结论成立：最大值为 $8^{1012}=2^{3036}$，由交替 $2,0,\dots$ 构造取得。
