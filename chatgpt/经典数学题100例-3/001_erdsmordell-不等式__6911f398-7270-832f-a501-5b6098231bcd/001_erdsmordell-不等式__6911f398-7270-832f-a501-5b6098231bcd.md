# Erdős–Mordell 不等式

- Conversation ID: `6911f398-7270-832f-a501-5b6098231bcd`

## Question

已知P在\triangle ABC的内部，P到三边的距离为PD,PE,PF。
求证：PA+PB+PC \ge 2(PD+PE+PF)

## Answer

这是著名的 Erdős–Mordell 不等式。我们用一个常见的代数/面积法证明（思路与 Wikipedia 的证明一致）。记号先统一：

设三角形 $ABC$ 的边长为

```math
a=BC,\quad b=CA,\quad c=AB,
```

点 $P$ 到顶点的距离为

```math
PA=p,\quad PB=q,\quad PC=r,
```

到三边的垂足分别为 $D\in BC,\;E\in CA,\;F\in AB$，并设垂距

```math
PD=x,\quad PE=y,\quad PF=z.
```

目标证明

```math
p+q+r \ge 2(x+y+z).
```

第一步（建立三条基本不等式）。
先证明

```math
c\,r \ge a x + b y.
```

把三角形 $ABC$ 的面积表示成三块小三角形面积之和：

```math
[ABC]=[PBC]+[PCA]+[PAB]=\tfrac12(ax+by+cz).
```

另一方面，考虑以边 $AB$（长度 $c$）为底的三角形，其对应的高不超过 $r+z$（可以通过对 $P$ 关于角平分线的反射等常规构造看到这一点——反射后得到的点与顶点 $C$ 的连线高度给出界），因此

```math
[ABC]\le \tfrac12\,c(r+z).
```

合并得到

```math
\tfrac12(ax+by+cz)\le \tfrac12 c(r+z)\quad\Rightarrow\quad cr\ge ax+by.
```

用同样的方法对轮换的顶点得到三条不等式：

```math
bq\ge az+cx,\qquad ap\ge bz+cy.
```

第二步（解出 $p,q,r$ 的下界并相加）。
由上面三不等式解出

```math
r\ge \frac{a}{c}x+\frac{b}{c}y,\qquad
q\ge \frac{a}{b}z+\frac{c}{b}x,\qquad
p\ge \frac{b}{a}z+\frac{c}{a}y.
```

把三式相加得

```math
p+q+r\ge
\Big(\frac{b}{c}+\frac{c}{b}\Big)x
+\Big(\frac{c}{a}+\frac{a}{c}\Big)y
+\Big(\frac{a}{b}+\frac{b}{a}\Big)z.
```

第三步（用算术–几何不等式完成）。
对任意正数 $t$ 有 $t+\dfrac1t\ge2$（AM–GM 的一个直接推论），所以每个系数满足

```math
\frac{b}{c}+\frac{c}{b}\ge 2,\quad
\frac{c}{a}+\frac{a}{c}\ge 2,\quad
\frac{a}{b}+\frac{b}{a}\ge 2.
```

因此

```math
p+q+r \ge 2x+2y+2z = 2(x+y+z).
```

这就完成了所要证明的

```math
PA+PB+PC \ge 2(PD+PE+PF).
```

（等号情形只在三角形为等边且 $P$ 为重心时取得。）

参考（此不等式与证明的常见版本）：Erdős–Mordell inequality（参见 Wikipedia 说明与证明）。en.wikipedia.org
