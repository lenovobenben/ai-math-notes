# 求极限计算

## Question

求极限 \lim_{n \to \infty} \sum_{k=1}^{n} (\frac{k}{n})^n

## Answer

我们要求的是：

```math
\lim_{n \to \infty} \sum_{k=1}^{n} \left(\frac{k}{n}\right)^n
```

##### 第一步：观察项的性质

对固定 $k$，当 $n \to \infty$ 时：

```math
\left(\frac{k}{n}\right)^n = e^{n \ln(\frac{k}{n})} = e^{n(\ln k - \ln n)} = \frac{k^n}{n^n} \to 0
```

因为 $\frac{k}{n} < 1$，且指数 $n \to \infty$。

但注意：对于接近 $k=n$ 的项，$\frac{k}{n}$ 接近 1，因此衰减得不快。

##### 第二步：分析大项（靠近 $k=n$ 的部分）

令 $k = n - j$，其中 $j = 0, 1, 2, \dots, n-1$。
则：

```math
\left(\frac{k}{n}\right)^n = \left(1 - \frac{j}{n}\right)^n \approx e^{-j}
```

当 $n \to \infty$ 时。

##### 第三步：近似和求和

因此：

```math
\sum_{k=1}^{n} \left(\frac{k}{n}\right)^n = \sum_{j=0}^{n-1} \left(1 - \frac{j}{n}\right)^n \approx \sum_{j=0}^{n-1} e^{-j}
```

而：

```math
\sum_{j=0}^{\infty} e^{-j} = \frac{1}{1 - e^{-1}} = \frac{e}{e-1}
```

##### 第四步：结论

当 $n \to \infty$ 时，

```math
\boxed{\lim_{n \to \infty} \sum_{k=1}^{n} \left(\frac{k}{n}\right)^n = \frac{e}{e - 1}}
```

## Question

如果变成 求极限 \lim_{n \to \infty} \sum_{k=1}^{n} (\frac{k}{n})^k 呢？

## Answer

好的，我们来算

```math
\lim_{n\to\infty}\sum_{k=1}^{n}\left(\frac{k}{n}\right)^k.
```

##### 观察靠近 $k=n$ 的项

令 $k=n-j$（$j=0,1,2,\dots$），对固定的 $j$，当 $n\to\infty$：

```math
\left(\frac{n-j}{n}\right)^{\,n-j}=\left(1-\frac{j}{n}\right)^{n-j}
=\left(1-\frac{j}{n}\right)^{n}\left(1-\frac{j}{n}\right)^{-j}\to e^{-j}\cdot 1=e^{-j}.
```

所以靠近末尾的若干项（固定个数的尾项）按 $j$ 给出极限序列 $e^{-j}$。

因此，若把和按距离末尾的距离 $j$ 展开：

```math
\sum_{k=1}^n\Big(\frac{k}{n}\Big)^k=\sum_{j=0}^{n-1}\Big(1-\frac{j}{n}\Big)^{\,n-j},
```

对固定有限的 $J$ 有

```math
\lim_{n\to\infty}\sum_{j=0}^{J}\Big(1-\frac{j}{n}\Big)^{\,n-j}=\sum_{j=0}^{J}e^{-j}.
```

##### 其它（离末尾较远的）项可以忽略

考虑 $k\le n-M$（即 $x=k/n\le 1-\delta$，取 $\delta>0$），令 $f(x)=x\ln x$（对 $x\in(0,1)$，有 $f(x)<0$），在区间 $[0,1-\delta]$ 上有上界 $-c<0$。于是对应的每一项

```math
\left(\frac{k}{n}\right)^k = e^{k\ln(k/n)} = e^{\,n f(k/n)} \le e^{-c n},
```

这类项最多有 $n$ 个，因此这一部分总和最多为 $n e^{-c n}\to0$（指数压过多项的线性增长）。

##### 汇总极限

把和拆成三部分：前面（很小 $k$）+ 中间（$k$ 与 $n$ 有固定距离以外）+ 尾部 $j=0,1,\dots$。前两部分在极限中都可使贡献趋于 $0$，尾部对固定 $J$ 收敛到 $\sum_{j=0}^J e^{-j}$，然后 $J\to\infty$ 得到完整尾和

```math
\sum_{j=0}^{\infty} e^{-j}=\frac{1}{1-e^{-1}}=\frac{e}{e-1}.
```

因此极限为

```math
\boxed{\displaystyle \lim_{n\to\infty}\sum_{k=1}^{n}\Big(\frac{k}{n}\Big)^k=\frac{e}{\,e-1\,}.}
```

数值上约等于 $1.582$。

## Question

这两个为什么是相同的？按理说k次幂应该比n次幂大

## Answer

非常好的问题！👏
你非常敏锐——确实，从表面上看，

```math
\left(\frac{k}{n}\right)^k \quad \text{vs} \quad \left(\frac{k}{n}\right)^n
```

因为 $k \le n$，所以幂次更小的 $k$ 理应使数值更大。
那为什么两个极限最后都等于 $\frac{e}{e-1}$ 呢？
我们仔细分析一下它们背后的结构。

#### 🧩 一、两者的主要贡献都来自“尾部”（$k$ 接近 $n$）

先看两者的形式：

```math
S_1(n) = \sum_{k=1}^{n} \left(\frac{k}{n}\right)^n
```

```math
S_2(n) = \sum_{k=1}^{n} \left(\frac{k}{n}\right)^k
```

我们换成 $k = n - j$，其中 $j = 0, 1, 2, \ldots$。

##### 对 $S_1(n)$

```math
\left(\frac{n-j}{n}\right)^n = \left(1 - \frac{j}{n}\right)^n \to e^{-j}
```

\to 靠近末尾的项贡献大。

##### 对 $S_2(n)$

```math
\left(\frac{n-j}{n}\right)^{n-j} = \left(1 - \frac{j}{n}\right)^{n-j}
= \left(1 - \frac{j}{n}\right)^n \cdot \left(1 - \frac{j}{n}\right)^{-j}
\to e^{-j} \cdot 1 = e^{-j}.
```

\to 靠近末尾的项同样趋近于 $e^{-j}$。

##### 对于较小的 $k$（离 $n$ 远的项）

当 $k \ll n$，$\frac{k}{n}$ 很小。

不论幂是 $k$ 还是 $n$，$(k/n)^k$ 和 $(k/n)^n$ 都极小，几乎可以忽略。

例如：

```math
\left(\frac{1}{n}\right)^n \text{ 和 } \left(\frac{1}{n}\right)^1
```

虽然后者大很多，但前者只有在靠近 $k=n$ 的部分才有显著贡献，前者的“中段”和“前段”几乎为零。

#### 🧠 二、两者极限为何一致

虽然幂不同，但：

对于主要贡献的部分（$k$ 接近 $n$ 的若干项），两者几乎相同，极限值都给出 $e^{-j}$。

对于其余部分，值小到可以忽略不计。

因此整个和的极限都由那一串 $e^{-j}$ 决定：

```math
\sum_{j=0}^{\infty} e^{-j} = \frac{e}{e-1}.
```

#### ✅ 三、小结对比

| 表达式 | 主要贡献来自 | 极限中单项近似 | 极限和 |
| --- | --- | --- | --- |
| $\sum (\frac{k}{n})^n$ | $k \approx n$ | $e^{-j}$ | $e/(e-1)$ |
| $\sum (\frac{k}{n})^k$ | $k \approx n$ | $e^{-j}$ | $e/(e-1)$ |

它们局部极限结构一致，所以最终结果相同。

如果我们想区分二者，可以考虑更细的近似项（例如加入 $1/n$ 级别修正），那时两者的差异会体现出来，但在极限（$n\to\infty$）下，它们收敛到同一个常数。
