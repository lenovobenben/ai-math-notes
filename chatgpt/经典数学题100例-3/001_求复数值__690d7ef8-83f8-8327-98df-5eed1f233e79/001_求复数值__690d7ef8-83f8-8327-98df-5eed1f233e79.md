# 求复数值

- Conversation ID: `690d7ef8-83f8-8327-98df-5eed1f233e79`

## Question

三个复数模长都是1，其积与和也都是1。求这三个复数的值

## Answer

我们一步一步来推理这个题。

##### 题设条件

设三个复数为

```math
z_1, z_2, z_3
```

满足：

```math
|z_1| = |z_2| = |z_3| = 1
```

且

```math
z_1 z_2 z_3 = 1
```

以及

```math
z_1 + z_2 + z_3 = 1
```

##### 第一步：用单位圆表示

因为模长都是 1，所以可以设

```math
z_k = e^{i\theta_k}, \quad k=1,2,3
```

于是：

```math
z_1 z_2 z_3 = e^{i(\theta_1 + \theta_2 + \theta_3)} = 1
```

说明

```math
\theta_1 + \theta_2 + \theta_3 \equiv 0 \pmod{2\pi}
```

##### 第二步：求解和为 1 的条件

有

```math
e^{i\theta_1} + e^{i\theta_2} + e^{i\theta_3} = 1
```

把 $z_3$ 用前两个表示：

```math
z_3 = \frac{1}{z_1 z_2}
```

于是

```math
z_1 + z_2 + \frac{1}{z_1 z_2} = 1
```

##### 第三步：设 $a = z_1, b = z_2$

则

```math
a + b + \frac{1}{ab} = 1
```

且 $|a|=|b|=1$。

##### 第四步：用 $a+b$ 和 $ab$ 表示

设

```math
s = a+b, \quad p = ab
```

则有

```math
s + \frac{1}{p} = 1 \quad \Rightarrow \quad \frac{1}{p} = 1 - s
```

即

```math
p = \frac{1}{1 - s}
```

另一方面，$a,b$ 是方程

```math
x^2 - s x + p = 0
```

的根。

##### 第五步：|a|=|b|=1 的条件

若 $|a|=|b|=1$，则 $b = 1/\overline{a}$。
同时因为 $ab=p$，所以

```math
a \cdot \frac{1}{\overline{a}} = p \Rightarrow p = \frac{a}{\overline{a}} = e^{2i\arg(a)}
```

即 $|p|=1$。

所以从上面 $p = \frac{1}{1-s}$，我们得到：

```math
|1-s| = 1
```

##### 第六步：几何解释

这个条件 $|1-s|=1$ 说明复数 $s$ 在以 1 为圆心、半径 1 的圆上。

##### 第七步：确定 s 取值

因为 $s = a+b$，且 $a,b$ 都在单位圆上，$s$ 是单位圆上两点的和。

$a+b$ 的可能范围是半径为 2 的圆盘中的所有复数，但我们还要满足上一步条件 $|1-s|=1$。

这两个圆：

圆 $C_1$：以原点为圆心，半径 2（$s=a+b$ 的最大长度）

圆 $C_2$：以 1 为圆心，半径 1

我们需要 $s$ 同时能写成 $a+b$ 且满足 $|1-s|=1$。

实际上直接用几何或代数法求解更明确。

##### 第八步：尝试对称形式

假设三个复数在单位圆上等角分布，但旋转了某个角度 $\phi$：

```math
z_1 = e^{i\phi}, \quad z_2 = e^{i(\phi+ \tfrac{2\pi}{3})}, \quad z_3 = e^{i(\phi+ \tfrac{4\pi}{3})}
```

那么：

```math
z_1+z_2+z_3 = e^{i\phi}(1 + e^{i 2\pi/3} + e^{i4\pi/3}) = 0
```

不满足。

所以不是等间距。

##### 第九步：设对称形式 $z_1 = e^{i\theta}, z_2 = e^{-i\theta}$，则 $z_3 = 1/(z_1 z_2) = 1$。

于是

```math
z_1+z_2+z_3 = e^{i\theta} + e^{-i\theta} + 1 = 2\cos\theta + 1
```

我们要求它等于 1：

```math
2\cos\theta + 1 = 1 \Rightarrow \cos\theta = 0
\Rightarrow \theta = \frac{\pi}{2} \text{ 或 } \frac{3\pi}{2}
```

##### 第十步：检验结果

若 $\theta = \frac{\pi}{2}$，则

```math
z_1 = i, \quad z_2 = -i, \quad z_3 = 1
```

验证：

模长：都为 1 ✅

积：$i \cdot (-i) \cdot 1 = (i\cdot -i)=1$ ✅

和：$i - i + 1 = 1$ ✅

完全符合。

✅ 最终答案：

```math
\boxed{z_1 = i, \quad z_2 = -i, \quad z_3 = 1}
```

或者顺序交换（对称的）都成立。
