# 证明同余关系

- Conversation ID: `69149f84-2688-8333-b1d7-7d7266ea5768`

## Question

对于正整数n,定义n \otimes 1 = n, n \otimes (k+1) = n^{n \otimes k}.
求证：对任何正整数m,都存在正整数a>b使得 n \otimes a \equiv n \otimes b(\mod m)对任何n都成立.

## Answer

把题目稍微重写一下（便于证明的归纳）：

对正整数 $n$ 定义

```math
n\otimes1=n,\qquad n\otimes(k+1)=n^{\,n\otimes k}.
```

要证明：对任意模数 $m\ge1$，存在正整数 $a>b$（只依赖于 $m$，与 $n$ 无关），使得对任意正整数 $n$ 都有

```math
n\otimes a\equiv n\otimes b\pmod m .
```

下面给出一个常见且清楚的证明思路（用欧拉函数/卡迈克尔函数与中国剩余分解，按模的阶层归纳）。

把模 $m$ 分解为互素的素幂因子。由中国剩余定理，证明模 $m$ 的结论等价于对每个素幂 $p^{\alpha}$ 都成立。于是只需固定一个素幂模 $M=p^{\alpha}$ 证明存在统一的高度 $a>b$ 使得对任意 $n$ 有

```math
n\otimes a\equiv n\otimes b\pmod{p^{\alpha}}.
```

引入欧拉函数的迭代。定义 $\varphi^{0}(M)=M,\ \varphi^{1}(M)=\varphi(M),\ \varphi^{2}(M)=\varphi(\varphi(M)),\dots$。由于对任意正整数迭代欧拉$\varphi$ 最终会到达 $1$，存在最小整数 $t$ 使得 $\varphi^{t}(M)=1$。取

```math
b=t+1,\qquad a=t+2.
```

我们要证明对这个 $a,b$ 上面的同余成立。

以 $j$ 从 $t$ 向下归纳，证明更强的分层命题：

对任意 $0\le j\le t$，设 $M_j=\varphi^{\,j}(M)$。我们断言：
对任意正整数 $n$ 有

```math
n\otimes (t+2-j)\equiv n\otimes (t+1-j)\pmod{M_j}.
```

当 $j=0$ 时正好是我们要的结论（模 $M_0=M$）。

归纳基：当 $j=t$ 时 $M_t=\varphi^{t}(M)=1$，模 $1$ 的同余恒成立，基底成立。

归纳步（从 $j+1$ 推到 $j$）：
假设对模 $M_{j+1}$ 有

```math
n\otimes (t+2-(j+1))\equiv n\otimes (t+1-(j+1))\pmod{M_{j+1}},
```

也就是

```math
n\otimes (t+1-j)\equiv n\otimes (t-j)\pmod{M_{j+1}}.
```

现在考察模 $M_j$。

若 $p\mid n$（即底数与模 $M_j=p^{\alpha_j}$ 有共同素因子），记 $v_p(n)=u\ge1$。那么 $n^{e}$ 的 $p$-幂次为 $e\cdot u$。于是当指数 $e$ 足够大使得 $e\cdot u\ge\alpha_j$ 时，$n^{e}\equiv0\pmod{p^{\alpha_j}}$。在我们的情形，作为指数的量是形如 $n\otimes (t+1-j)$ 与 $n\otimes (t-j)$，对于 $n\ge2$ 这些“塔”都非常大（对于 $n=1$ 问题显然成立，因为 $1\otimes k=1$）。因此两边模 $p^{\alpha_j}$ 都为 $0$，同余成立。

若 $\gcd(n,M_j)=1$，则可以用欧拉定理（或卡迈克尔函数）把指数模 $\varphi(M_j)=M_{j+1}$ 化简：对任意整数 $x\ge M_{j+1}$ 有

```math
n^{x}\equiv n^{\,x'\,}\pmod{M_j}\quad\text{当且仅当}\quad x\equiv x'\pmod{M_{j+1}}.
```

因为根据归纳假设 $n\otimes (t+1-j)\equiv n\otimes (t-j)\pmod{M_{j+1}}$，并且这两个指数显然都大于或等于 $M_{j+1}$（当 $n\ge2$ 时“塔”迅速增大；而 $n=1$ 已处理），因此把它们作为指数代入模 $M_j$ 的幂同余可得

```math
n^{\,n\otimes (t+1-j)}\equiv n^{\,n\otimes (t-j)}\pmod{M_j},
```

即

```math
n\otimes (t+2-j)\equiv n\otimes (t+1-j)\pmod{M_j}.
```

上述两种情况覆盖了所有可能的 $n$，所以归纳步完成。

从 $j=0$ 得到结论：取 $b=t+1,\ a=t+2$（这里 $t$ 是使 $\varphi^{t}(m)=1$ 的最小整数），则对所有正整数 $n$ 有

```math
n\otimes a\equiv n\otimes b\pmod m.
```

备注与直观说明：

本证明的要点在于用欧拉函数把“指数的指数”一路压下去：为了比较两层高的幂塔模 $m$ 的值，我们只需比较其顶端的指数模 $\varphi(m)$；而 $\varphi(m)$ 又可用同样方法进一步化简到 $\varphi(\varphi(m))$，如此迭代直到 1，形成有限层的归纳链。

对于底数与模不互素的情况，用 $p$-进制估计（当指数足够大时幂为 $0$）处理即可。

因此对每个 $m$ 可以选择固定的高度差 $a-b=1$（即取 $a=t+2,\ b=t+1$）就能保证对所有 $n$ 同时成立。（有时可取更小或不同的 $a,b$，但上面的选择很自然且统一。）

因此命题成立：对任意正整数 $m$ 存在 $a>b$（例如上面给出的 $a=t+2,\ b=t+1$）使得对任意正整数 $n$ 都有 $n\otimes a\equiv n\otimes b\pmod m$。 □
