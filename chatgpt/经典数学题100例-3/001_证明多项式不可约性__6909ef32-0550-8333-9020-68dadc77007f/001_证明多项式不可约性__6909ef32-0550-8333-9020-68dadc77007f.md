# 证明多项式不可约性

- Conversation ID: `6909ef32-0550-8333-9020-68dadc77007f`

## Question

证明：若p为素数，则多项式f(x) = x^{p-1} + 2x^{p-2} + \dots + p 在\mathbb{Q} 上不可约。

## Answer

令

```math
f(x)=x^{p-1}+2x^{p-2}+\cdots+(p-1)x+p.
```

考虑多项式

```math
h(x)=(x-1)f(x)=x^p+x^{p-1}+\cdots+x-p=\sum_{i=1}^p x^i-p.
```

注意 $h(1)=p-p=0$，于是 $x-1$ 是 $h(x)$ 的因子，且

```math
f(x)=\frac{h(x)}{x-1}.
```

令 $y=x-1$，则

```math
h(y+1)=\sum_{i=1}^p (y+1)^i-p.
```

展开并按 $y^j$ 收集系数（使用组合数的“冰棒”恒等式）

```math
\sum_{i=j}^p\binom{i}{j}=\binom{p+1}{j+1},
```

可得（常数项被抵消掉）

```math
h(y+1)=\sum_{j=1}^{p}\binom{p+1}{\,j+1\,}y^j.
```

因此可以把 $y$ 提出来：

```math
h(y+1)=y\cdot H(y),\qquad H(y)=\sum_{k=0}^{p-1}\binom{p+1}{k+2}y^k.
```

注意 $H(y)$ 是首项系数为 $1$ 的整系数多项式。对 $1\le m\le p-1$ 有
$\binom{p+1}{m}$ 被素数 $p$ 整除（由组合数的整除性质或模 $p$ 的二项式系数性质可见），而
$\binom{p+1}{2}=\dfrac{p(p+1)}{2}$ 整除 $p$ 但不被 $p^2$ 整除（因为 $p+1$ 与 $p$ 互素）。于是 $H(y)$ 的系数满足：

最高次项系数不被 $p$ 整除（等于 $1$）；

除最高次项外的所有系数均被 $p$ 整除；

常数项 $\binom{p+1}{2}$ 被 $p$ 整除但不被 $p^2$ 整除。

因此 $H(y)$ 满足爱森斯坦判别法（以素数 $p$），所以 $H(y)$ 在 $\mathbb{Q}$ 上不可约。

最后回到原多项式，注意

```math
f(x)=H(x-1).
```

代换 $y=x-1$ 不改变不可约性（有理系数的可约/不可约性质在变量平移下不变），因此 $f(x)$ 在 $\mathbb{Q}$ 上也不可约。

证毕。
