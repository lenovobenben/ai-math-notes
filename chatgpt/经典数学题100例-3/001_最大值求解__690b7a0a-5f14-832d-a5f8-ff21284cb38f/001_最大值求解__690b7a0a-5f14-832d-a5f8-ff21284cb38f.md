# 最大值求解

- Conversation ID: `690b7a0a-5f14-832d-a5f8-ff21284cb38f`

## Question

已知a,b,c为正实数，且满足 a+b+c=1。
求 a + \sqrt{b} + \sqrt[4]{c} 的最大值。

## Answer

用拉格朗日乘子法求极大值（并注意目标函数是凹函数，在单纯形上驻点即为全局最大值）。

设

```math
F(a,b,c)=a+\sqrt b+\sqrt[4]{c},\qquad a+b+c=1,\ a,b,c\gt{}0.
```

构造拉格朗日函数

```math
\mathcal L=a+\sqrt b+c^{1/4}+\lambda(1-a-b-c).
```

对$a,b,c$求偏导并令其为0：

```math
\frac{\partial\mathcal L}{\partial a}=1-\lambda=0\implies\lambda=1,
```

```math
\frac{\partial\mathcal L}{\partial b}=\frac{1}{2\sqrt b}-\lambda=0\implies\frac{1}{2\sqrt b}=1\implies b=\frac14,
```

```math
\frac{\partial\mathcal L}{\partial c}=\frac{1}{4}c^{-3/4}-\lambda=0\implies\frac{1}{4}c^{-3/4}=1
```

```math
\implies c^{3/4}=\frac14\implies c=\Big(\frac14\Big)^{4/3}=4^{-4/3}=2^{-8/3}.
```

于是

```math
a=1-b-c=1-\tfrac14-4^{-4/3}=\tfrac34-4^{-4/3}.
```

因为函数 $a+\sqrt b+c^{1/4}$ 在可行域上为凹函数，该驻点即为全局最大点。因此最大值为

```math
\begin{aligned}
F_{\max}&=a+\sqrt b+c^{1/4}
=\Big(\tfrac34-4^{-4/3}\Big)+\tfrac12+4^{-1/3}\\
&=\tfrac54+4^{-1/3}-4^{-4/3}
=\tfrac54+3\cdot4^{-4/3}.
\end{aligned}
```

也可写成

```math
F_{\max}=\frac54+\frac{3}{2^{8/3}}\approx 1.72246.
```

当且仅当

```math
(b,c)=\Big(\tfrac14,\;4^{-4/3}\Big),\quad a= \tfrac34-4^{-4/3}
```

时达成。
