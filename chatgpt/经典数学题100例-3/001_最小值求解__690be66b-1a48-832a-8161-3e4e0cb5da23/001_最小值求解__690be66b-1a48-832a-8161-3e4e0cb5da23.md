# 最小值求解

- Conversation ID: `690be66b-1a48-832a-8161-3e4e0cb5da23`

## Question

\triangle ABC为锐角三角形。求5\tan A + 2\tan B + \tan C的最小值

## Answer

设 $x=\tan A,\;y=\tan B,\;z=\tan C$. 由 $A+B+C=\pi$ 得

```math
\tan A+\tan B+\tan C=\tan A\cdot\tan B\cdot\tan C,
```

即

(1)

```math
x+y+z=xyz.
```

由 (1) 可得

```math
z=\frac{x+y}{xy-1}\qquad(\text{因为三角形为锐角，}xy\gt{}1).
```

于是要最小化的量变为

```math
f(x,y)=5x+2y+\frac{x+y}{xy-1},\qquad x\gt{}0,\;y\gt{}0,\;xy\gt{}1.
```

对 $x,y$ 求偏导：

```math
f_x=5-\frac{1+y^2}{(xy-1)^2},\qquad
f_y=2-\frac{1+x^2}{(xy-1)^2}.
```

令 $f_x=f_y=0$，得

```math
\frac{1+y^2}{(xy-1)^2}=5,\qquad \frac{1+x^2}{(xy-1)^2}=2.
```

由此消去公因子得到

```math
5(1+x^2)=2(1+y^2)\Longrightarrow 5x^2-2y^2=-3.
```

联立上面两式解得正数解

```math
x=1,\qquad y=2.
```

此时由 (1) 得 $z=3$。

检查二阶充分条件：海森矩阵在 $(1,2)$ 处为
$\displaystyle \begin{pmatrix}20&6\\6&4\end{pmatrix}$，行列式 $=80-36=44>0$ 且迹 $>0$，故该驻点为严格极小值点。又在边界 $xy\to1^+$ 或 $x,y\to\infty$ 时 $f\to+\infty$，因此全局最小值在此驻点取得。

代回得到最小值

```math
5\tan A+2\tan B+\tan C=5\cdot1+2\cdot2+3=12.
```

答：最小值为 $\boxed{12}$，当且仅当 $\tan A=1,\;\tan B=2,\;\tan C=3$（即对应的角满足此三者时）取得。
