# 积分计算解析

- Conversation ID: `690d7e46-190c-832c-ae7a-0c4eed60d1d5`

## Question

若f(x)e^{f(x)}=x，x \in [0,e]。求\int_{0}^{e}f(x)dx

## Answer

注意到 $f(x)$ 满足

```math
f(x)e^{f(x)}=x,
```

即 $f(x)=W(x)$（Lambert W 函数）。令 $t=f(x)$，则 $x=t e^{t}$，且

```math
dx=e^{t}(1+t)\,dt.
```

所以

```math
\int_{0}^{e}f(x)\,dx=\int_{x=0}^{x=e} t\;dx
=\int_{t=0}^{t=1} t\cdot e^{t}(1+t)\,dt
=\int_{0}^{1}\big(t e^{t}+t^{2}e^{t}\big)\,dt.
```

不定积分可得（逐项计算）：

```math
\int t e^{t}\,dt=(t-1)e^{t},\qquad
\int t^{2}e^{t}\,dt=(t^{2}-2t+2)e^{t},
```

因此

```math
\int t\cdot e^{t}(1+t)\,dt=(t^{2}-t+1)e^{t}.
```

代入上下限 $t=0$ 到 $t=1$：

```math
\Big[(t^{2}-t+1)e^{t}\Big]_{0}^{1}= (1-1+1)e^{1}-(0-0+1)e^{0}=e-1.
```

答案：

```math
\displaystyle \int_{0}^{e}f(x)\,dx=e-1.
```
