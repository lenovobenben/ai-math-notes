# 证明勒贝格正测度集

- Conversation ID: `690cafc5-b818-832d-a62a-58feb7e21d30`

## Question

设X\subset \mathbb{R} 为勒贝格正测度集，证明X包含一个长度为n（n \gt 1）的等差数列

## Answer

下面给出一个简洁而标准的证明（利用特征函数与受控极限／支配收敛定理）。

记 $1_X$ 为集合 $X$ 的指示函数（特征函数），即

```math
1_X(x)=\begin{cases}1,&x\in X,\\0,&x\notin X.\end{cases}
```

因为 $X\subset\mathbb R$ 的勒贝格测度 $m(X)>0$，所以 $1_X\in L^1(\mathbb R)$ 且

```math
\int_{\mathbb R}1_X(x)\,dx=m(X)\gt{}0.
```

对任意实数 $d$ 定义函数

```math
c(d)\;=\;\int_{\mathbb R}\prod_{k=0}^{n-1}1_X\bigl(x+k d\bigr)\,dx
\;=\;m\Bigl(\bigcap_{k=0}^{n-1}\bigl(X-kd\bigr)\Bigr).
```

注意 $c(d)$ 的几何含义：它就是那些 $x$ 的测度，使得点
$x,x+d,\dots,x+(n-1)d$ 全都落在 $X$ 中。若某个 $d$ 使得 $c(d)>0$，
则必存在 $x$（事实上测度为正的一组 $x$）满足 $x+k d\in X$ 对一切 $k=0,\dots,n-1$ 成立，
这正是我们要的长度为 $n$（公差为 $d$）的等差数列。

现在考察 $c(d)$ 在 $d$ 越来越接近 $0$ 时的极限。对每个固定 $x$，
当 $d\to0$ 时， $1_X(x+k d)\to 1_X(x)$（几乎处处成立）。于是

```math
\prod_{k=0}^{n-1}1_X(x+k d)\longrightarrow\bigl(1_X(x)\bigr)^n=1_X(x)
\quad\text{当 }d\to0.
```

并且对所有 $d$ 与 $x$ 有

```math
0\le \prod_{k=0}^{n-1}1_X(x+k d)\le 1,
```

故可用支配收敛定理交换极限与积分，得到

```math
\lim_{d\to0} c(d)
=\lim_{d\to0}\int_{\mathbb R}\prod_{k=0}^{n-1}1_X(x+k d)\,dx
=\int_{\mathbb R}\lim_{d\to0}\prod_{k=0}^{n-1}1_X(x+k d)\,dx
=\int_{\mathbb R}1_X(x)\,dx=m(X)\gt{}0.
```

因此存在一个足够小的 $d$（可以取正的）使得 $c(d)>0$。由此存在 $x$ 满足

```math
x,\;x+d,\;x+2d,\;\dots,\;x+(n-1)d\in X,
```

即 $X$ 包含一个长为 $n$ 的等差数列（公差为该 $d$）。

这完成证明。 ∎
