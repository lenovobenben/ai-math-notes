# 最大化乘积问题

- Conversation ID: `69160b09-6814-8329-afbd-e28800c78731`

## Question

给定奇数 n \ge 3,已知 a_i \in [n^{-1},n],1 \le i \le n,且 \prod_{i=1}^{n} a_i = 1. 求 \prod_{i=1}^{n}(a_i - a_{i+1}) 的最大可能值.
下标\mod n理解.

## Answer

结论（答案）

```math
\boxed{\displaystyle \max\prod_{i=1}^n (a_i-a_{i+1})
=\Bigl(n-\frac1n\Bigr)^{\,n-2}\cdot\frac{(n-1)^2}{n},\qquad(n\ \text{为奇数}\ge3).}
```

构造性说明（取到该值的例子）
令 $k=\dfrac{n-1}{2}$. 取恰好 $k$ 个项等于 $n$、恰好 $k$ 个项等于 $1/n$、其余唯一一项等于 $1$（这样总体乘积为 $n^k\cdot(1/n)^k\cdot1=1$ 且都在区间 $[1/n,n]$ 内）。把它们按循环次序交替排列，例如

```math
\underbrace{\tfrac1n,\,n,\,\tfrac1n,\,n,\dots,\tfrac1n,\,n}_{2k\ \text{项}},\,1
```

（也可从 $n$ 开头交替，或经循环位移），此时相邻差绝大多数为

```math
D:=n-\frac1n,
```

剩下两个差分别为 $1/n-1$ 和 $1-n$。因此乘积为

```math
P=D^{\,2k-1}\cdot\Bigl( \frac{1}{n}-1\Bigr)\cdot(1-n)
= \Bigl(n-\frac1n\Bigr)^{\,n-2}\cdot\frac{(n-1)^2}{n},
```

（注意 $2k-1=n-2$），且为正。上面给出的数列实际达到了该值。

最优性要点（说明为何不能更大）
把问题看作在约束 $\prod a_i=1$, $a_i\in[1/n,n]$ 下最大化多项式 $\prod(a_i-a_{i+1})$。固定其它变量时，将某一对变量在保持它们乘积不变的情形下向区间端点推进，会使相邻差的绝对值变大（直观上：在正数上，固定乘积时两数之差 $|x-y|$ 随两数更分散而增大），因此全局最优必出现在边界取值上。由乘积约束和取值区间可知边界情形最终只能是上文所构造的那类——恰有 $k$ 个取 $n$、$k$ 个取 $1/n$、一个取 $1$。在这些边界点中按交替排列能最大化每个相邻差的模，从而得到上面的封闭式值；任意其他排列要么使某两个相邻相等（乘积为 $0$），要么某些差的绝对值变小，导致总体乘积变小。

（可用对偶性或对每一对变量做代换并用单峰性/凸性推理把上面的“直观”论证严格化；此处给出构造与要点以说明最大值及其可取性。）
