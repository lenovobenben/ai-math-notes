# 证明不等式

- Conversation ID: `6910c3b3-1210-8330-899f-47b15afca71b`

## Question

a \gt b \gt 0，且 a^b = b^a。证明ab \gt e^2

## Answer

设 $a>b>0$ 且 $a^b=b^a$。

因为 $a>b$，可写成 $a=b^k$ 且 $k>1$。把它代入等式得

```math
(b^k)^b=b^{b^k}\quad\Longrightarrow\quad b^{kb}=b^{b^k}\,,
```

于是 $kb=b^k$。两边同时除以 $b$（注意 $b>0$），得到

```math
k=b^{\,k-1}\quad\Longrightarrow\quad b=k^{\frac{1}{\,k-1}}.
```

于是

```math
a=b^k=\Big(k^{\frac{1}{\,k-1}}\Big)^k=k^{\frac{k}{\,k-1}}.
```

所以

```math
ab=k^{\frac{k}{k-1}}\cdot k^{\frac{1}{k-1}}=k^{\frac{k+1}{\,k-1}}.
```

两边取对数，需证明

```math
\ln(ab)=\frac{k+1}{k-1}\ln k\gt{}2.
```

等价地证明函数

```math
\Phi(k)\coloneqq (k+1)\ln k-2(k-1)
```

在 $k>1$ 时为正。计算导数得

```math
\Phi'(k)=\ln k-1+\frac{1}{k},\qquad
\Phi''(k)=\frac{1}{k}-\frac{1}{k^2}=\frac{k-1}{k^2}\gt{}0\quad(k\gt{}1).
```

因此 $\Phi'$ 在 $k>1$ 上严格增加，而 $\lim_{k\to1}\Phi'(k)=0$，于是对 $k>1$ 有 $\Phi'(k)>0$。再加上 $\Phi(1)=0$，可得对所有 $k>1$ 有 $\Phi(k)>0$。因此 $\ln(ab)>2$，即

```math
ab\gt{}e^2.
```

证毕。
