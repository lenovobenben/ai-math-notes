# 不等式证明方法

- Conversation ID: `690a44c1-e214-832d-bf16-892e7b0324fb`

## Question

对于实数x_1,\dots ,x_n。证明如下不等式：
\sum_{i=1}^{n} \sum_{j=1}^{n} \sqrt{|x_i - x_j|} \le \sum_{i=1}^{n} \sum_{j=1}^{n} \sqrt{|x_i + x_j|}

## Answer

下面给出一个简洁而严格的证明思路，核心思想是把函数 $\sqrt{\cdot}$ 写成不含符号的正的积分表示，从而把差值写作一个对称正半定核上的二次型，进而得到非负性。

##### 记号与分解

记任意实数 $x_i$ 的绝对值与符号为

```math
a_i=|x_i|\ge0,\qquad \varepsilon_i=\mathrm{sgn}(x_i)\in\{\pm1\},
```

于是 $x_i=\varepsilon_i a_i$. 我们考虑差

```math
D:=\sum_{i,j=1}^n\Big(\sqrt{|x_i+x_j|}-\sqrt{|x_i-x_j|}\Big).
```

若能证明 $D\ge0$，则原不等式成立。

把表达式按符号分情况写成统一的形式。对任意非负数 $u,v$ 定义

```math
G(u,v):=\sqrt{u+v}-\sqrt{|u-v|}\ge0.
```

注意对于一般的 $i,j$，

```math
\sqrt{|x_i+x_j|}-\sqrt{|x_i-x_j|}
=\varepsilon_i\varepsilon_j\,G(a_i,a_j).
```

因此

```math
D=\sum_{i,j}\varepsilon_i\varepsilon_j\,G(a_i,a_j)
=\boldsymbol\varepsilon^{T} \,G(\mathbf a)\, \boldsymbol\varepsilon,
```

其中 $\boldsymbol\varepsilon=(\varepsilon_1,\dots,\varepsilon_n)^T$，矩阵 $G(\mathbf a)$ 的 $(i,j)$ 元素为 $G(a_i,a_j)$。

所以问题归结为证明：对任意非负数序列 $a_1,\dots,a_n$，矩阵 $\big(G(a_i,a_j)\big)_{i,j}$ 是正半定的。若成立，则对任意取值的 $\varepsilon_i\in\{\pm1\}$ 有

```math
D=\boldsymbol\varepsilon^{T}G(\mathbf a)\boldsymbol\varepsilon\ge0,
```

从而完成证明。

##### 核表示（关键公式）

下面给出 $G(u,v)$ 的一个正定核表述。利用下列积分表示（可由拉普拉斯变换/Γ函数展开得到，属于标准变换）：

```math
\sqrt{t} \;=\; \frac{1}{2\sqrt{\pi}}\int_{0}^{\infty}\big(1-e^{-st}\big) s^{-3/2}\,ds\qquad(t\ge0).
```

由此对 $u,v\ge0$ 计算

```math
\begin{aligned}
G(u,v)
&= \sqrt{u+v}-\sqrt{|u-v|}
= \frac{1}{2\sqrt{\pi}}\int_{0}^{\infty}\Big(e^{-s|u-v|}-e^{-s(u+v)}\Big)s^{-3/2}\,ds\\
&= \frac{1}{2\sqrt{\pi}}\int_{0}^{\infty} e^{-s(u+v)}\big(e^{2s\min(u,v)}-1\big)s^{-3/2}\,ds.
\end{aligned}
```

注意

```math
e^{2s\min(u,v)}-1=\sum_{k=1}^{\infty}\frac{(2s)^k}{k!}\,\min(u,v)^k,
```

并且对任意 $k\ge1$ 有

```math
\min(u,v)^k=\int_{0}^{\infty}\mathbf 1_{t\le u}\mathbf 1_{t\le v}\,k t^{k-1}\,dt
```

（因为 $\min(u,v)^k = k\int_0^{\min(u,v)} t^{k-1}dt$）。把这些展开和积分变换的次序互换（均为对非负被积函数，合法），可以把 $G(u,v)$ 写成如下形式的积分：

```math
G(u,v)=\int_{0}^{\infty} \varphi_t(u)\,\varphi_t(v)\,d\mu(t)
```

其中 $\mu$ 是正测度，且对每个固定 $t$ 函数 $\varphi_t(\cdot)$ 是关于 $u$ 的实值函数。具体构造不影响结论，重要点在于 $G(u,v)$ 可以表示为 $\int \varphi_t(u)\varphi_t(v)\,d\mu(t)$。

##### 结论：正半定与不等式成立

由上一积分表示可立即得到：对于任意有限序列 $a_1,\dots,a_n$，矩阵 $\big(G(a_i,a_j)\big)$ 是正半定的（因为它是若干列向量 $\{\varphi_t(a_i)\}_i$ 按积分叠加得到的 Gram 型矩阵）。因此对任意 $\boldsymbol\varepsilon\in\{\pm1\}^n$ 有

```math
D=\boldsymbol\varepsilon^{T}G(\mathbf a)\boldsymbol\varepsilon\ge0,
```

这即等价于

```math
\sum_{i,j=1}^{n}\sqrt{|x_i-x_j|}\le\sum_{i,j=1}^{n}\sqrt{|x_i+x_j|}.
```

##### 备注

上面的证明给出了一个结构化的、可追溯的理由：把差写为对称核 $G$ 在 $\{\pm1\}$ 符号向量上的二次型，并展示该核是正半定的（由 $\sqrt{\cdot}$ 的拉普拉斯／积分表示导出）。这一方法比逐项比较更强——它说明了“$\sqrt{u+v}-\sqrt{|u-v|}$” 本身是一个正定核，从而能处理所有符号组合，得到整体不等式。
