# 求解三角恒等式

- Conversation ID: `690a3d6e-2cc0-832b-b07a-3fc7808707d2`

## Question

实数a,b,c满足\cos (b-c) + \cos (c-a) + \cos (a-b) = -\frac{3}{2} 。求 \cos a + \cos b + \cos c

## Answer

把三个单位向量 $(\cos a,\sin a),(\cos b,\sin b),(\cos c,\sin c)$ 看成平面向量。注意恒等式

```math
\begin{aligned}
&\cos(b-c)+\cos(c-a)+\cos(a-b) \\
&= (\cos a\cos b+\sin a\sin b)+(\cos b\cos c+\sin b\sin c)+(\cos c\cos a+\sin c\sin a)\\
&=\frac{(\cos a+\cos b+\cos c)^2+(\sin a+\sin b+\sin c)^2-3}{2}.
\end{aligned}
```

设 $S=\cos(b-c)+\cos(c-a)+\cos(a-b)$。由上式得

```math
(\cos a+\cos b+\cos c)^2+(\sin a+\sin b+\sin c)^2=3+2S.
```

题中给出 $S=-\tfrac32$，代入得右端为 $3+2(-\tfrac32)=0$。因此

```math
(\cos a+\cos b+\cos c)^2+(\sin a+\sin b+\sin c)^2=0,
```

所以

```math
\cos a+\cos b+\cos c=0,\qquad \sin a+\sin b+\sin c=0.
```

结论：$\boxed{0}$。
