# 求最小值

- Conversation ID: `690c2903-5074-8324-b096-9d5dc49021a2`

## Question

若|x^2+2y|-|y^2+2x|=2。
```math
求|x|_{min}
```

## Answer

先把题目写成等式

```math
|x^2+2y|-|y^2+2x|=2.
```

把绝对值按被取号内的正负情况分四类讨论。

记 $A=x^2+2y,\;B=y^2+2x$。

$A\ge0,\;B\ge0$：去掉绝对号得

```math
x^2+2y-(y^2+2x)=2\iff x^2-y^2+2y-2x=2,
```

整理为

```math
(x-y)(x+y-2)=2.
```

把上式化为关于 $y$ 的二次方程：

```math
y^2-2y-(x^2-2x-2)=0,
```

判别式 $\Delta=4+4(x^2-2x-2)=4(x^2-2x-1)\ge0$，所以必须有

```math
x^2-2x-1\ge0\iff |x-1|\ge\sqrt2,
```

即 $x\le 1-\sqrt2$ 或 $x\ge 1+\sqrt2$。因此在这一类中 $|x|$ 的最小可能值是当 $x=1-\sqrt2$ 时的 $|x|= \sqrt2-1$。且此时对应的 $y$ 由判别式为 0 得 $y=1$。检验：

```math
A=(1-\sqrt2)^2+2\cdot1=5-2\sqrt2\gt{}0,\quad B=1+2(1-\sqrt2)=3-2\sqrt2\gt{}0,
```

并且 $A-B=2$，满足原式。

$A\ge0,\;B<0$：得

```math
x^2+2y+y^2+2x=2\iff (x+1)^2+(y+1)^2=4
```

（圆心 $(-1,-1)$，半径 2），并且还需满足 $B<0$。这一类解的 $x$ 落在 $[-3,1]$ 内，并且由于 $B<0$ 必然需要 $x<0$。用代入检验可知这一类中不可能得到比 $\sqrt2-1$ 更小的 $|x|$（可以用代数或逐点检验得到相同结论），所以不影响最小值。

$A<0,\;B\ge0$：变为 $-A-B=2$ 等价于 $(x+1)^2+(y+1)^2=0$，得唯一点 $(x,y)=(-1,-1)$，但此点使得 $B<0$，与 $B\ge0$ 矛盾，故无解。

$A<0,\;B<0$：类似第1类，得到

```math
(x-y)(x+y-2)=-2,
```

这一类允许的 $x$ 集合与第1类不同，检验可知也不会产生比第1类更小的 $|x|$。

因此所有可能中 $|x|$ 的最小值由第1类的临界点给出：

```math
\boxed{|x|_{\min}=\sqrt2-1.}
```

并且一个对应的解为 $x=1-\sqrt2,\;y=1$。
