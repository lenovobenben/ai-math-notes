# 证明不等式

- Conversation ID: `6909e703-4f44-8327-9b87-821710fd0742`

## Question

令a,b,c为正实数，证明：
\frac{2}{a+b} + \frac{2}{b+c} + \frac{2}{c+a} \ge \frac{9}{a+b+c}

## Answer

用 Engel 形式的柯西（又称 Titu 引理）很短的证明如下。

把每项写成平方除以数：

```math
\frac{2}{a+b}=\frac{1^2}{\frac{a+b}{2}}.
```

由 Titu 引理，

```math
\sum_{\text{cyc}}\frac{1^2}{\frac{a+b}{2}}
\ge\frac{(1+1+1)^2}{\sum_{\text{cyc}\ }\frac{a+b}{2}}
=\frac{9}{\frac{(a+b)+(b+c)+(c+a)}{2}}
=\frac{9}{a+b+c}.
```

因此

```math
\frac{2}{a+b}+\frac{2}{b+c}+\frac{2}{c+a}\ge\frac{9}{a+b+c},
```

等号当且仅当 $a=b=c$。证毕。
