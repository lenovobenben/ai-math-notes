\section*{题目}

设抛物线

```math
C:\; y^2 = 4x
```

的焦点为 \(F(1,0)\)。

任取一条过焦点 \(F\) 的直线，与抛物线相交于两点 \(A,B\)。
再作过 \(F\) 且垂直于 \(AB\) 的直线，与抛物线相交于两点 \(D,E\)。

记 \(M,N\) 分别为弦 \(AB\) 与弦 \(DE\) 的中点。
又作直线 \(AE\) 与 \(BD\) 相交于点 \(G\)。

求三角形 \(GMN\) 的面积 \(S\) 的表达式，并求其最小值。

## Answer

好——我们把题目按参数化计算，得到三角形 $GMN$ 面积的解析式并求最小值。下面给出完整且尽量简洁的推导。

取过焦点 $F(1,0)$ 的任意直线，设其斜率为 $k$（$k\ne0$），方程为

```math
\ell:\; y=k(x-1).
```

与抛物线 $C:\;y^2=4x$ 联立得

```math
k^2(x-1)^2=4x \quad\Longrightarrow\quad k^2x^2-(2k^2+4)x+k^2=0.
```

记两交点为 $A(x_1,y_1),\;B(x_2,y_2)$。由二次方程，根的和为

```math
x_1+x_2=\frac{2k^2+4}{k^2}=2+\frac{4}{k^2}.
```

因此弦 $AB$ 的中点 $M$ 的坐标为

```math
M\Big(\frac{x_1+x_2}{2},\ \frac{y_1+y_2}{2}\Big)
= \Big(1+\frac{2}{k^2},\ k\big(1+\tfrac{2}{k^2}-1\big)\Big)
= \Big(1+\frac{2}{k^2},\ \frac{2}{k}\Big).
```

记与 $AB$ 垂直并过 $F$ 的直线斜率为 $-1/k$，其方程

```math
y=-\frac{1}{k}(x-1).
```

同样与抛物线联立得到两交点 $D,E$，其弦的中点 $N$ 类似计算得

```math
N\Big(1+2k^2,\ -2k\Big).
```

接下来求点 $G$：按定义，$G$ 为直线 $AE$ 与直线 $BD$ 的交点。对上述参数化点代入并解线交，我们得到（可直接计算或用代数软件验证）

```math
G=\big(-1,\; \frac{2(1-k)}{k+1}\big).
```

值得注意的是 $G$ 的横坐标恒为 $-1$，与 $k$ 无关。

现在计算三角形 $GMN$ 的面积。用行列式公式（向量叉积）：

```math
S=\frac12\big|\det(\overrightarrow{GM},\overrightarrow{GN})\big|.
```

把 $G,M,N$ 的坐标代入并化简（可验算），得到

```math
\boxed{\,S=2\!\left(k^2+\frac{1}{k^2}+2\right)\,}.
```

进一步化为便于分析的形式：

```math
S=2\Big(\Big(k-\frac{1}{k}\Big)^2+4\Big)=2\Big(k-\frac{1}{k}\Big)^2+8.
```

于是面积的最小值出现在 $k-\dfrac{1}{k}=0$，即 $k^2=1$（即 $k=1$ 或 $k=-1$）时，此时最小面积为

```math
\boxed{S_{\min}=8}.
```

总结答案：

三角形 $GMN$ 面积关于参数 $k$ 的表达式为

```math
S=2\left(k^2+\frac{1}{k^2}+2\right)=2\Big(k-\frac{1}{k}\Big)^2+8.
```

最小值为 $S_{\min}=8$，在 $k=\pm1$（即通过焦点且斜率为 $1$ 或 $-1$ 的直线）时取到。
