# 有理数三平方和

- Conversation ID: `68fa1270-53b8-8321-9ce9-403958f00dd9`

## Question

是否存在x,y,z\inℚ，使得x²+y²+z²=7？

## Answer

你问的是：

是否存在 $x, y, z \in \mathbb{Q}$ 使得

```math
x^2 + y^2 + z^2 = 7
```

我们可以仔细分析这个问题。

##### 第一步：有理数转整数

假设存在有理数解 $x, y, z \in \mathbb{Q}$，那么可以写成分数形式：

```math
x = \frac{a}{d}, \quad y = \frac{b}{d}, \quad z = \frac{c}{d}, \quad a,b,c,d \in \mathbb{Z}, \quad d \neq 0
```

代入方程：

```math
\left(\frac{a}{d}\right)^2 + \left(\frac{b}{d}\right)^2 + \left(\frac{c}{d}\right)^2 = 7
```

两边同时乘以 $d^2$：

```math
a^2 + b^2 + c^2 = 7 d^2
```

所以问题转化为：是否存在整数 $a, b, c, d$ 满足

```math
a^2 + b^2 + c^2 = 7 d^2
```

##### 第二步：模 8 分析

我们知道一个重要的整数性质：

任何整数的平方模 8 只能是 $0,1,4$。

因此 $a^2+b^2+c^2 \pmod{8}$ 只能是 $0,1,2,3,4,5,6,7$ 中可以由 0,1,4 三个数相加得到的值。我们列一下所有可能和：

| a² mod 8 | b² mod 8 | c² mod 8 | sum mod 8 |
| --- | --- | --- | --- |
| 0,1,4 | 0,1,4 | 0,1,4 | 可以得到 0,1,2,3,4,5,6,7 |

仔细检查 7 的情况：

三个平方和为 7 mod 8，可能组合是：

(4,1,2)? 不行，因为平方 mod 8 只能是 0,1,4。

真正组合：$1+1+1 = 3$, $4+1+1 = 6$, $4+4+1=1$, $0+1+4 =5$... 试验所有组合，无法得到 7。

所以 不存在整数解 $a^2+b^2+c^2 = 7 d^2$ 对于任意整数 $d \neq 0$。（模 8 可知 7 的形式无法被三个平方数整除）

##### 第三步：结论

因此：

```math
\boxed{\text{不存在有理数 } x, y, z \text{ 使得 } x^2+y^2+z^2=7}
```
