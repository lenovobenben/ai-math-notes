# 阴影四边形面积

- Conversation ID: `68f35581-fec0-8322-b913-1a1b3ed8518c`

## Question

在一个边长为 10 的大正方形内部，有一个倾斜的、被阴影覆盖的不规则四边形。
这个阴影四边形的四个顶点分别位于大正方形的四条边上。已知：
阴影四边形的上下两个顶点的横坐标差值为 2。
阴影四边形的左右两个顶点的纵坐标差值为 3。

问题是：求这个阴影部分的面积。

## Answer

把题目按坐标来解最清楚——结论很简洁：面积不是唯一的，而是两种可能之一，分别是 47 或 53。下面是推导。

设大正方形为单位坐标系：左下角 $(0,0)$，右上角 $(10,10)$。

设阴影四边形四顶点分别在左、上、右、下四边上，坐标为

```math
L=(0,y_L),\; T=(x_T,10),\; R=(10,y_R),\; B=(x_B,0)
```

题给条件是 $|x_T-x_B|=2,\;|y_R-y_L|=3$。

用轮廓面积（Shoelace）公式计算四边形面积，按上面顺序：

```math
S=\tfrac12\big(100+(x_B-x_T)(y_R-y_L)\big).
```

（代数化简可得上式。）

由于只知道差的绝对值，$(x_B-x_T)(y_R-y_L)$ 的可能值为 $\pm(2\cdot3)=\pm6$。于是

```math
S=\tfrac12(100\pm6)=50\pm3,
```

即 $S=47$ 或 $S=53$。

举例验证：

若 $x_B=4,\;x_T=6$（$x_B-x_T=-2$），且 $y_L=3,\;y_R=6$（$y_R-y_L=3$），则代入得到 $S=47$。

若方向相反（例如 $x_B=6,\;x_T=4$ 且 $y_R=6,\;y_L=3$），则 $S=53$。

所以除非题目另外给出“上点在右/左边”或“左点在上/下”这类方向信息，否则只有这两个可能面积：47 或 53（平方单位）。需要我用图形画出来并标出两种情况吗？

## Question

假如：上点在左。左点在下。

## Answer

好——你给出的额外方向信息等价于：

“上点在左” ⇒ 上顶点的横坐标小于下顶点的横坐标，因此 $x_T<x_B$，所以 $x_B-x_T=+2$；

“左点在下” ⇒ 左顶点的纵坐标小于右顶点的纵坐标，因此 $y_R-y_L=+3$。

由我之前推导的通式（取顶点顺序为 左 $L=(0,y_L)$、上 $T=(x_T,10)$、右 $R=(10,y_R)$、下 $B=(x_B,0)$）：

```math
S=\tfrac12\big(100+(x_B-x_T)(y_R-y_L)\big).
```

代入 $x_B-x_T=2$、$y_R-y_L=3$ 得

```math
S=\tfrac12(100+2\cdot3)=\tfrac12(106)=53.
```

举一组满足条件的具体坐标以便验证（都在正方形 $[0,10]\times[0,10]$ 内）：

```math
L=(0,2),\;T=(4,10),\;R=(10,5),\;B=(6,0)
```

用对角和法（Shoelace）逐项算一次（按要求逐位计算）：

```math
x_1y_2+x_2y_3+x_3y_4+x_4y_1 = 0\cdot10 + 4\cdot5 + 10\cdot0 + 6\cdot2 = 0+20+0+12 = 32
```

```math
y_1x_2+y_2x_3+y_3x_4+y_4x_1 = 2\cdot4 + 10\cdot10 + 5\cdot6 + 0\cdot0 = 8+100+30+0 = 138
```

面积 $S=\tfrac12|32-138|=\tfrac12\cdot106=53$。

所以在你指定的方向条件下，阴影部分的面积为 53 平方单位。需要我把这组顶点画成图并标注面积过程吗？
