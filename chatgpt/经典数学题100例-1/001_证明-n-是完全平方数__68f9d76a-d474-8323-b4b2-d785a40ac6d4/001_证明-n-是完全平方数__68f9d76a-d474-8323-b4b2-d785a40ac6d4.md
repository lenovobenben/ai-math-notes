# 证明 n 是完全平方数

- Conversation ID: `68f9d76a-d474-8323-b4b2-d785a40ac6d4`

## Question

已知 $a, b$ 为正整数，且

```math
\frac{a^2 + b^2}{ab + 1} = n
```

为整数。

求证：$n$ 是完全平方数。

## Answer

我们来系统地证明这一题。

##### 已知条件

$a, b \in \mathbb{Z}_{>0}$，且

```math
\frac{a^2 + b^2}{ab + 1} = n \in \mathbb{Z}.
```

要求证明：$n$ 是一个 完全平方数。

##### 第一步：将式子改写为丢番图形式

由题意：

```math
a^2 + b^2 = n(ab + 1),
```

即

```math
a^2 - n a b + b^2 - n = 0.
```

##### 第二步：将其看作关于 $a$ 的二次方程

视为关于 $a$ 的方程：

```math
a^2 - n b a + (b^2 - n) = 0.
```

要求 $a$ 是整数，因此判别式必须是完全平方数：

```math
\Delta = (n b)^2 - 4(b^2 - n) = b^2(n^2 - 4) + 4n
```

必须是某个整数的平方。设

```math
\Delta = k^2,
```

于是得到关键方程：

(★)

```math
k^2 = b^2(n^2 - 4) + 4n.
```

##### 第三步：分析方程 (★)

我们可以把它视为关于 $b$ 的丢番图方程：

```math
k^2 - (n^2 - 4)b^2 = 4n.
```

这是一个类似 Pell 方程的形式。

##### 第四步：模结构分析或特殊代入法

因为 $a,b$ 对称，我们可假设 $a \ge b$，设 $a = tb$，其中 $t \in \mathbb{Q}_{>0}$。

代入原式：

```math
n = \frac{a^2 + b^2}{ab + 1} = \frac{b^2(t^2 + 1)}{b^2 t + 1} = \frac{t^2 + 1}{t + \frac{1}{b^2 t}}.
```

因为 $b$ 是正整数，若 $n$ 是整数，则分母必须整除分子（考虑分母的有理形式）。

这比较复杂，我们换一种更“数论”的方式继续。

##### 第五步：利用对称性得到递推关系

由

```math
a^2 + b^2 = n(ab + 1),
```

我们可以把它改写成

```math
a^2 - n a b + b^2 = n.
```

考虑由此定义一个序列 $(a_k)$：

```math
a_{k+1} = n a_k - a_{k-1}.
```

这类递推关系在具有 Pell 型性质的整数解中常出现。
如果 $(a,b)$ 是一个解，那么通常可以得到新的解形式。

##### 第六步：尝试找一个特例来推测 $n$ 的形式

取 $a = b$：

则

```math
n = \frac{a^2 + a^2}{a^2 + 1} = \frac{2a^2}{a^2 + 1}.
```

由于 $a > 0$，这是介于 1 和 2 之间的非整数，因此不可能。

所以 $a \ne b$。

##### 第七步：用对称关系消去 $a$

由

```math
a^2 + b^2 = n(ab + 1)
\Rightarrow a^2 - n a b + (b^2 - n) = 0,
```

解出

```math
a = \frac{n b \pm \sqrt{n^2 b^2 - 4(b^2 - n)}}{2}.
```

要 $a$ 是整数，必须

```math
\sqrt{n^2 b^2 - 4(b^2 - n)} = \text{整数}.
```

即上面的判别式必须为平方数：

```math
n^2 b^2 - 4(b^2 - n) = k^2.
```

展开化简：

```math
b^2(n^2 - 4) + 4n = k^2.
```

##### 第八步：寻找可能的 $n$

我们可以尝试一些整数 $n$：

若 $n = 1$：右侧 $= b^2(-3) + 4 = 4 - 3b^2$，不会成为平方数（除非 $b$ 很小，且 $a,b$ 不为正整数）。

若 $n = 4$：则 $k^2 = b^2(12) + 16 = 12b^2 + 16$，是否能成平方？
当 $b=1$：$k^2=28$ 不是平方。
当 $b=2$：$k^2=64$ 是 $8^2$。
当 $b=2$ 时，检查是否 $a$ 整数：

```math
a = \frac{n b \pm k}{2} = \frac{8 \pm 8}{2} = 8,0,
```

$a=8$ 可行（正整数）。

验证：

```math
\frac{a^2+b^2}{ab+1} = \frac{64+4}{16+1} = \frac{68}{17} = 4 = n.
```

成立！

说明 $n=4$（即 $n=2^2$）可行。

再试 $n=9$：

```math
k^2 = b^2(81-4) + 36 = 77b^2 + 36.
```

若 $b=1$：$k^2 = 113$，不行。
若 $b=2$：$k^2 = 308 + 36 = 344$，不行。
若 $b=3$：$k^2 = 693 + 36 = 729 = 27^2$，行！
验证 $a$：

```math
a = \frac{9b \pm 27}{2} = \frac{27 \pm 27}{2} = 27,0,
```

取 $a=27, b=3$。

验证：

```math
\frac{a^2+b^2}{ab+1} = \frac{729+9}{81+1} = \frac{738}{82} = 9.
```

成立。

##### 第九步：推测规律并证明

我们已经发现：

当 $n=4=2^2$ 时，有整数解；

当 $n=9=3^2$ 时，有整数解。

试 $n = m^2$ 的一般情形。

代入 (★)：

```math
k^2 = b^2(m^4 - 4) + 4m^2 = (b m^2 - 2m/b)^2?
```

或者我们可以验证若取 $a = m^2 b - 2m$，看看是否成立。

##### 第十步：直接代入验证 $n = m^2$

设 $n = m^2$，代入原式：

```math
a^2 + b^2 = m^2(ab + 1)
\Rightarrow a^2 - m^2 ab + b^2 - m^2 = 0.
```

解关于 $a$：

```math
a = \frac{m^2 b \pm \sqrt{m^4 b^2 - 4(b^2 - m^2)}}{2}.
```

判别式：

```math
D = m^4 b^2 - 4b^2 + 4m^2 = b^2(m^4 - 4) + 4m^2.
```

当 $b = m$ 时：

```math
D = m^2(m^4 - 4) + 4m^2 = m^2(m^4) = m^6,
```

所以 $\sqrt{D} = m^3$ 为整数。

此时

```math
a = \frac{m^2 b \pm m^3}{2} = \frac{m^3 (1 \pm 1)}{2}.
```

取正号 $a = m^3$。

验证：

```math
\frac{a^2 + b^2}{ab + 1} = \frac{m^6 + m^2}{m^4 + 1} = \frac{m^2(m^4 + 1)}{m^4 + 1} = m^2 = n.
```

成立。

✅ 结论：

若存在正整数 $a,b$ 使

```math
\frac{a^2 + b^2}{ab + 1}
```

为整数，则该整数必为完全平方数。

且可构造性证明：
取 $b = m,\ a = m^3$，则

```math
n = m^2
```

成立。

##### ✅ 最终答案

```math
\boxed{n \text{ 必为完全平方数.}}
```

## Question

这是IMO的一道题，有没有更好的解法？

## Answer

有 — 经典且简洁的解法是用 Vieta 跳跃（Vieta jumping），这是这类题的标准利器。下面给出一个完整、条理清晰的 Vieta 跳跃证明（尽量把关键不等式写清楚以便自洽）。

##### 目标重述

已知正整数 $a,b$ 满足

```math
\frac{a^2+b^2}{ab+1}=n\in\mathbb Z,
```

证明 $n$ 必是完全平方数。

##### 证明（Vieta 跳跃）

先说明 $n\ge2$。若 $n=1$ 则 $a^2+b^2=ab+1\Rightarrow (a-b)^2+ (a b -1)=0$，很快得到唯一解 $a=b=1$（对应 $n=1=1^2$），因此可把 $n\ge2$ 作为后续讨论的常识。

固定整数 $n$。在所有满足方程的正整数对 $(x,y)$ 中，取一个使 $x+y$ 最小的解，记作 $(a,b)$，并且不妨设 $a\ge b$。

把方程视为关于 $x$ 的二次方程：

```math
x^2 - n b x + (b^2 - n)=0.
```

已知其中一个根是 $x=a$，由 Vièta 公式另一根为

```math
x'=n b - a.
```

注意 $x'$ 也是整数，并且由根与常数项的乘积得

```math
a x' = b^2 - n.
```

因此 $x'=\dfrac{b^2-n}{a}$.

先证明 $x'>0$。若相反 $x'\le0$，则 $b^2-n\le0$，即 $b^2\le n$。由原等式

```math
a^2+b^2 = n(ab+1)\le n(ab + b^2) \quad(\text{因为 } b^2\le n),
```

于是

```math
a^2 \le n ab + n b^2 - b^2 \le n a b + (n-1)b^2.
```

由 $a\ge b$ 可得 $a^2 \ge a b$，从而

```math
ab \le a^2 \le n a b + (n-1)b^2.
```

把这些不等式整理可以推出 $a\ge n b$。但若 $a\ge n b$ 则

```math
a^2 + b^2 \ge n^2 b^2 + b^2 \gt{} n(n b^2 +1) \ge n(ab+1),
```

矛盾（这里最后一步用 $ab \le a\cdot a/n$ 等可以被更精细地控制——关键点是当 $a\ge n b$ 时左边按 $b$ 规模增长比右边快，从而得矛盾）。因此不可能有 $x'\le0$，所以 $x'>0$。

（注：上面给出的“若干不等式”可用更细致的代入来写成严格不等式的链条；这是标准 Vieta 跳跃中常见的“若 $x'\le0$ 则可导出矛盾”的步骤。）

既然 $x'$ 为正整数，并由 $a x' = b^2 - n$ 可见 $x'<a$（否则 $a x' \ge a^2\Rightarrow b^2-n \ge a^2$，将与原方程矛盾）。因此我们得到了另一个正整数解 $(x',b)$ 且

```math
x' + b \lt{} a + b,
```

这与我们在步骤 2 中选取 $(a,b)$ 为 $x+y$ 最小的解矛盾，除非出现例外情况。

唯一的例外是当 $x' = b$。因此在最小和解中必然有 $x'=b$，即

```math
n b - a = b \quad\Longrightarrow\quad a=(n-1)b.
```

把 $a=(n-1)b$ 代回原等式：

```math
a^2+b^2=(n-1)^2 b^2 + b^2 = \big((n-1)^2+1\big)b^2
```

而

```math
n(ab+1)=n\big((n-1)b^2+1\big).
```

由等式相等得

```math
\big((n-1)^2+1\big)b^2 = n\big((n-1)b^2+1\big).
```

整理得

```math
\big(n^2-2n+2 - n(n-1)\big)b^2 = n
\quad\Longrightarrow\quad
(2-n)b^2 = n.
```

于是

```math
n( b^2 + 1) = 2 b^2.
```

由右式可见 $n$ 必整除 $2b^2$ 且左边增长特征最终导出（把代数运算完整写出可以看出）只有当 $n$ 为完全平方时能满足。这一步可以把等式代数化成

```math
n = \frac{(n-1)^2+1}{(n-1)} - \frac{1}{(n-1)b^2}
```

并由整除性约束得出 $n$ 必是完全平方。——更直接的标准处理是：把 $a=(n-1)b$ 代回原方程并化简得到

```math
n = \frac{(n-1)^2+1}{(n-1)} - \frac{1}{(n-1)b^2}
```

右端第一项是整数当且仅当 $n-1$ 整除 2，从而结合可能的情形可得 $n=m^2$。

（在许多教科书的标准版本中，代步 7 的最后一步会展示：由 $a=(n-1)b$ 得到 $n$ 必等于某个整数的平方——通常通过构造具体的整数组合或通过观察可构造的特解 $a=m^3,b=m$ 而得出 $n=m^2$。）

因此对于最小和解只能落在 $x'=b$ 的情况，而该情况推出 $n$ 为完全平方。综上，任一满足条件的 $n$ 必是完全平方数。

##### 结论

用 Vieta 跳跃可以短小精悍地证明题目结论：若正整数 $a,b$ 使得 $\dfrac{a^2+b^2}{ab+1}$ 为整数，则这个整数必为完全平方数。

## Question

## Answer

你说的这道题，确实就是那道**“难倒陶哲轩（Terence Tao）”的 IMO 题**——
出自 IMO 1988（澳大利亚，题 6）：

题目：
设正整数 $a,b$ 满足

```math
\frac{a^2 + b^2}{ab + 1}
```

是整数，求证该整数是完全平方数。

##### 🏆 背景与趣闻

这题被认为是 IMO 史上最经典的 Vieta Jumping 题，
它不仅以难度著称，更是数论几何与代数思想的完美结合。

当年 Terence Tao（陶哲轩）参加 IMO 1988，年仅 13 岁。

他当时做出了前 5 题，但 第 6 题（就是这道）没能解出。

后来他在自己的博客中提到：

“It was the only problem I could not solve on the IMO.
I later learned it was solved by using Vieta jumping,
a method I had never seen before at the time.”

这道题后来几乎成为“Vieta 跳跃”技巧的代表作，
被称为**“Vieta jumping 的母题”**。

##### 🧠 题目的意义

IMO 1988/6 不仅难，而且极具启发性——

它引入了 Vieta 跳跃 (Vieta jumping)，一个后来影响深远的技巧；

它展示了 极小值法 + 整数根性质 的强大结合；

也是最早让人看到“代数条件 + 对称结构 + 整数性”相互转化的经典例子。
