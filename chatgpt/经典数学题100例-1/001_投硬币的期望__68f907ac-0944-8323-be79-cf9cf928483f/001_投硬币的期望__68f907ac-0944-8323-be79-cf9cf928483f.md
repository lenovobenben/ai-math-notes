# 投硬币的期望

- Conversation ID: `68f907ac-0944-8323-be79-cf9cf928483f`

## Question

假设你在投一枚普通的硬币（正反两面的概率各为1/2）。达到连续n次正面所需的投掷次数的期望是多少？

## Answer

好的，这个问题是经典的马尔可夫期望问题，通常称为“连续出现 $n$ 次正面所需的投掷次数的期望”。我们可以一步一步推导。

##### 1️⃣ 定义期望

设 $E_n$ 表示达到连续 n 次正面所需的期望投掷次数。我们考虑递推：

如果我们投一次硬币：

正面（概率 1/2）：我们就进入“已连续出现 1 个正面”的状态。

反面（概率 1/2）：连续正面次数归零，需要重新开始。

更一般地，我们可以定义状态期望：

$E_k$：当前已经连续出现了 $k$ 个正面，还需要多少次投掷才能达到连续 $n$ 个正面。

显然：

```math
E_n = 0 \quad (\text{因为已经有连续 } n \text{ 个正面})
```

##### 2️⃣ 状态递推公式

对于 $k < n$ 的情况：

```math
E_k = 1 + \frac{1}{2} E_{k+1} + \frac{1}{2} E_0
```

解释：

投一次硬币，消耗 1 次投掷（+1）；

正面（概率 1/2）：连续数加 1，期望为 $E_{k+1}$；

反面（概率 1/2）：连续数归零，期望为 $E_0$。

最终我们需要求的是 $E_0$。

##### 3️⃣ 求解递推

这个递推有已知解法。它可以通过线性方程求解得到：

```math
E_0 = 2^n + 2^{n-1} + \cdots + 2 = 2^{n+1} - 2
```

更一般地，公式为：

```math
\boxed{E_n = 2^{n+1} - 2}
```

##### 4️⃣ 验证几个小例子

$n=1$：期望投掷次数 = $2^{1+1}-2 = 2$，正确，因为达到 1 次正面的期望次数就是 2。

$n=2$：期望 = $2^{3}-2 = 6$，符合常见结论。

$n=3$：期望 = $2^{4}-2 = 14$。

✅ 结论：

```math
\text{连续 } n \text{ 次正面所需的期望投掷次数 } = 2^{n+1}-2
```
