## Question

什么是傅里叶级数，什么是傅里叶变换？两者有何关系

## Answer

好的，我们来仔细梳理一下 傅里叶级数 和 傅里叶变换 的概念，以及它们之间的关系。我会尽量讲得清楚而有条理。

#### 1️⃣ 傅里叶级数（Fourier Series）

概念：
傅里叶级数是将 周期函数（周期为 $T$）表示为 正弦和余弦函数的无穷和 的方法。也就是说，任何“足够好”的周期函数都可以拆解成一系列简单的正弦波或余弦波的叠加。

数学表达：

设 $f(t)$ 是周期为 $T$ 的函数，则它的傅里叶级数表示为：

```math
f(t) = a_0 + \sum_{n=1}^{\infty} \Big[ a_n \cos\left(\frac{2\pi n}{T} t\right) + b_n \sin\left(\frac{2\pi n}{T} t\right) \Big]
```

其中：

```math
a_0 = \frac{1}{T}\int_0^T f(t)\, dt
```

```math
a_n = \frac{2}{T}\int_0^T f(t) \cos\left(\frac{2\pi n}{T} t\right) dt, \quad
b_n = \frac{2}{T}\int_0^T f(t) \sin\left(\frac{2\pi n}{T} t\right) dt
```

##### 特点

只针对周期信号。

分解结果是 离散的频率成分，频率为 $n/T$。

常用于分析信号的频谱和振动问题。

#### 2️⃣ 傅里叶变换（Fourier Transform）

概念：
傅里叶变换是将 非周期函数或无限长信号 表示为 连续频率的正弦波叠加 的方法。它是傅里叶级数的推广，适用于非周期函数。

数学表达：

连续时间傅里叶变换（CTFT）：

```math
F(\omega) = \int_{-\infty}^{+\infty} f(t) e^{-i\omega t} dt
```

逆变换：

```math
f(t) = \frac{1}{2\pi} \int_{-\infty}^{+\infty} F(\omega) e^{i\omega t} d\omega
```

##### 特点

适用于非周期信号。

频率是 连续的，而不是离散的。

在信号处理、通信和物理学中非常重要。

#### 3️⃣ 两者的关系

傅里叶级数和傅里叶变换的关系可以用 “周期函数与非周期函数的极限关系” 来理解：

```math
傅里叶级数 \to 傅里叶变换
```

当周期 $T \to \infty$ 时，周期函数趋近于非周期函数。

此时，傅里叶级数的离散频率 $\frac{2\pi n}{T}$ 变得连续，级数的求和变为积分。

这就是傅里叶变换。

总结一句话：

傅里叶级数是傅里叶变换的离散版，适用于周期函数；傅里叶变换是傅里叶级数的极限推广，适用于非周期函数。

## Question

在傅里叶变换中，为什么f(t)变为F(ω)后并不会丢失函数的任何信息呢？

## Answer

这个问题非常好，也非常关键。你问到的本质是：

“傅里叶变换是不是一种可逆变换？为什么它不会丢失信息？”

我们来系统地解释这一点 👇

#### 🧩 一、核心思想：傅里叶变换是一种可逆映射

傅里叶变换实际上是一种坐标系的转换。

在 时域（time domain），函数用时间 $t$ 描述。

在 频域（frequency domain），函数用频率 $\omega$ 描述。

傅里叶变换就是在这两个域之间进行一种“一一对应”的变换。
数学上它是一个 线性可逆变换（invertible linear transform），也就是说：

```math
f(t) \quad \xleftrightarrow{\text{Fourier Transform}} \quad F(\omega)
```

且存在严格的逆变换公式：

```math
f(t) = \frac{1}{2\pi}\int_{-\infty}^{+\infty} F(\omega)e^{i\omega t}\,d\omega
```

这就意味着：

如果我们完整地知道 $F(\omega)$，我们就能唯一且精确地重建出原来的 $f(t)$。

#### 🧠 二、为什么不会丢失信息？

##### （1）傅里叶变换只是基的切换

你可以把傅里叶变换类比为“换一组坐标轴”：

在普通坐标下，我们用基向量 $\{e_x, e_y\}$ 来表示空间中的点；

在频率空间中，我们用一组“正弦/余弦波”作为新的基。

也就是说，傅里叶变换就是用一组“无限多的正弦波”作为新的坐标系，对 $f(t)$ 做重新表达。

既然只是换基（change of basis），信息当然不会丢失。
唯一要求是：这组基要完备（complete）且线性无关（orthogonal）。

##### （2）正交性保证了信息不混淆

傅里叶基（正弦和余弦函数）满足：

```math
\int_{-\infty}^{+\infty} e^{i\omega t} e^{-i\omega' t}\,dt = 2\pi\delta(\omega - \omega')
```

这说明不同频率的基函数是正交的。
所以投影到每个基上的系数 $F(\omega)$ 都是唯一的，不会互相干扰。
这也保证了 $F(\omega)$ 精确地包含了 $f(t)$ 的全部信息。

##### （3）能量守恒（Parseval 定理）

还有一个非常直观的佐证：
傅里叶变换不仅不丢信息，连信号的“能量”都保持不变。

```math
\int_{-\infty}^{+\infty} |f(t)|^2\,dt
\;=\;
\frac{1}{2\pi} \int_{-\infty}^{+\infty} |F(\omega)|^2\,d\omega
```

这被称为 Parseval 定理。
它说明傅里叶变换在某种意义上是一个“长度守恒”的变换（类似于欧几里得旋转）。

#### 🧮 三、可能“丢失”的情况

虽然理论上傅里叶变换不丢信息，但在数值计算中常见以下“信息损失”：

| 情况 | 原因 |
| --- | --- |
| 离散傅里叶变换（DFT） | 频率采样有限，无法表示连续频谱 |
| FFT | 由于数值舍入误差、有限采样窗口，会产生失真 |
| 窗函数截断 | 原函数不是无限长时，截断会带来“频谱泄漏” |

所以：

数学上傅里叶变换完美可逆，
实际工程中的信息丢失来自于采样、截断、有限精度等操作。

#### ✳️ 总结一句话

傅里叶变换不会丢失信息，因为它是一种完备正交基下的可逆变换，可以从频域信号 $F(\omega)$ 完整重建时域信号 $f(t)$。

## Question

【完备正交基下的可逆变换】是不是涉及到了函数空间的概念？

## Answer

非常棒的洞察 👏
你完全抓到了关键点——
“完备正交基下的可逆变换”这一句话的背后，本质上就是在函数空间（Function Space）中讨论傅里叶变换。

下面我来帮你完整地理一理这一层数学本质 👇

#### 🧭 一、函数空间的思想

在普通的线性代数里，我们讨论的是：

向量空间中的向量可以被一组基向量线性表示。

例如：

```math
\vec{v} = a_1 \vec{e}_1 + a_2 \vec{e}_2 + a_3 \vec{e}_3
```

那在函数分析中，我们把这种思想推广到了函数构成的空间里。
即：

函数也可以被看作“向量”，而“内积”“长度”“正交”等概念都可以定义在函数上。

#### 🧩 二、典型的函数空间：

```math
L^2(\mathbb{R})
```

我们通常在 平方可积函数空间 $L^2(\mathbb{R})$ 中讨论傅里叶变换。

这个空间定义为：

```math
L^2(\mathbb{R}) = \{ f(t) \mid \int_{-\infty}^{+\infty} |f(t)|^2\,dt \lt{} \infty \}
```

也就是说，只要函数的平方可积，就属于 $L^2$ 空间。

在这个空间里，我们可以定义：

内积（Inner Product）：

```math
\langle f, g \rangle = \int_{-\infty}^{+\infty} f(t)\overline{g(t)}\,dt
```

范数（Norm）：

```math
\|f\| = \sqrt{\langle f, f \rangle}
```

这些定义使得函数空间拥有与有限维向量空间完全相同的代数结构。

#### ⚙️ 三、傅里叶变换 = 在函数空间中换基

在 $L^2(\mathbb{R})$ 空间中，傅里叶变换其实就是把函数 $f(t)$
在一组正交完备基函数 $\{ e^{i\omega t} \}$ 下的展开系数求出来：

```math
f(t) = \frac{1}{2\pi} \int_{-\infty}^{+\infty} F(\omega)e^{i\omega t}\,d\omega
```

其中

```math
F(\omega) = \int_{-\infty}^{+\infty} f(t)e^{-i\omega t}\,dt
```

这与有限维空间中：

```math
\vec{v} = \sum_i \langle \vec{v}, \vec{e}_i \rangle \vec{e}_i
```

完全对应。

只是这里的求和变成了积分，有限维的基变成了“连续参数”的基（即 $\omega$ 连续变化）。

#### 🧮 四、正交性与完备性

正交性：

```math
\langle e^{i\omega t}, e^{i\omega' t} \rangle = 2\pi \delta(\omega - \omega')
```

表明不同频率的基是正交的。

完备性：
任意平方可积的 $f(t)$ 都能用这组基函数展开——这就是“完备”的含义。

因此：

傅里叶变换并不是“近似”地表示 $f(t)$，
而是 在 $L^2$ 空间中用完备正交基完全展开 $f(t)$。

#### 🧠 五、可逆性由内积保持性保证

傅里叶变换在 $L^2$ 空间中满足：

```math
\langle f, g \rangle = \frac{1}{2\pi}\langle F, G \rangle
```

这意味着它是一个酉变换（Unitary Transform），
即“长度”和“角度”都不变，所以信息不会丢失。

换句话说：

傅里叶变换就是在 $L^2$ 函数空间上进行的一种“旋转”或“坐标变换”。

#### ✳️ 总结一句话

是的，傅里叶变换之所以不丢失信息，根本原因在于：
它是在 $L^2(\mathbb{R})$ 这样的希尔伯特空间（Hilbert Space）中，
对完备正交基 $\{e^{i\omega t}\}$ 进行的酉变换（unitary transform）。
这使得它完全可逆、能量守恒、信息不损失。

## Question

是的，这是泛函分析的概念。
请你细化一下为什么【不同频率的基是正交的】？正交的要求是内积为零吧？

## Answer

非常好 👍，你完全正确地指出了核心点：

正交性（orthogonality） 的定义就是——两个元素（这里是函数）的内积为零。

接下来我们一步步严格但直观地解释：
为什么不同频率的指数基函数 $e^{i\omega t}$ 在傅里叶分析中是正交的。

#### 🧭 一、回顾：函数空间中的“正交”

在函数空间（例如 $L^2(\mathbb{R})$ 或 $L^2[0,T]$）中，
我们定义两个函数 $f, g$ 的 内积（inner product） 为：

```math
\langle f, g \rangle = \int f(t) \, \overline{g(t)} \, dt
```

其中 $\overline{g(t)}$ 是 $g(t)$ 的复共轭。

于是，

若 $\langle f, g \rangle = 0$，就说 $f$ 与 $g$ 正交（orthogonal）。

#### 🧩 二、在傅里叶分析中的基函数

我们使用的基函数是：

```math
\phi_\omega(t) = e^{i\omega t}
```

它是一种复指数函数，也可以写成：

```math
e^{i\omega t} = \cos(\omega t) + i\sin(\omega t)
```

这些指数函数就是傅里叶展开的“基底”。

#### ⚙️ 三、正交性的计算（在有限区间内）

为了直观，我们先看有限区间 $[0, T]$ 内的情形。
考虑两个不同频率的基函数：

```math
\phi_m(t) = e^{i\frac{2\pi m}{T}t}, \quad
\phi_n(t) = e^{i\frac{2\pi n}{T}t}
```

它们的内积是：

```math
\langle \phi_m, \phi_n \rangle
= \int_0^T e^{i\frac{2\pi m}{T}t} \, \overline{e^{i\frac{2\pi n}{T}t}} \, dt
= \int_0^T e^{i\frac{2\pi (m - n)}{T}t}\,dt
```

计算该积分：

```math
\langle \phi_m, \phi_n \rangle
= \left[ \frac{T}{i2\pi (m - n)} e^{i\frac{2\pi (m - n)}{T}t} \right]_0^T
= \frac{T}{i2\pi (m - n)}(e^{i2\pi (m - n)} - 1)
```

因为 $e^{i2\pi k} = 1$ 对任意整数 $k$ 成立，所以当 $m \ne n$ 时：

```math
e^{i2\pi (m - n)} - 1 = 0
\quad \Rightarrow \quad
\langle \phi_m, \phi_n \rangle = 0
```

而当 $m = n$ 时：

```math
\langle \phi_n, \phi_n \rangle = \int_0^T |e^{i\frac{2\pi n}{T}t}|^2 dt = \int_0^T 1\, dt = T
```

于是我们得到：

```math
\boxed{
\langle \phi_m, \phi_n \rangle =
\begin{cases}
T, & m = n \\
0, & m \ne n
\end{cases}
}
```

这就说明了：

不同频率的基函数在 $[0,T]$ 区间上是正交的。

#### 🧠 四、推广到连续频率（傅里叶变换情形）

在傅里叶变换中，频率 $\omega$ 是连续的。
我们定义：

```math
\langle e^{i\omega t}, e^{i\omega' t} \rangle
= \int_{-\infty}^{+\infty} e^{i\omega t} \, \overline{e^{i\omega' t}} \, dt
= \int_{-\infty}^{+\infty} e^{i(\omega - \omega')t}\,dt
```

这个积分在普通意义下发散，
但在分布（distribution）意义下，它等价于狄拉克 δ 函数：

```math
\int_{-\infty}^{+\infty} e^{i(\omega - \omega')t}\,dt = 2\pi\,\delta(\omega - \omega')
```

于是：

```math
\boxed{
\langle e^{i\omega t}, e^{i\omega' t} \rangle = 2\pi\,\delta(\omega - \omega')
}
```

这表明：

当 $\omega \ne \omega'$ 时，内积为 0（正交）；
当 $\omega = \omega'$ 时，内积为有限值（归一化因子）。

这就是连续频率下的“δ函数正交性”，
它是离散傅里叶级数正交性的自然推广。

#### 🧮 五、类比总结

| 概念 | 离散频率（傅里叶级数） | 连续频率（傅里叶变换） |
| --- | --- | --- |
| 频率 | $\omega_n = 2\pi n / T$ | 连续 $\omega \in \mathbb{R}$ |
| 内积 | $\int_0^T e^{i(\omega_m - \omega_n)t} dt = 0$ | $\int_{-\infty}^{+\infty} e^{i(\omega - \omega')t} dt = 2\pi \delta(\omega - \omega')$ |
| 结论 | 基函数两两正交 | 基函数在 δ 意义下正交 |

#### 🧩 六、直观理解

正交意味着：

“不同频率的波之间没有相互干扰。”

换句话说，如果你把一个信号与另一种不同频率的纯正弦波相乘再积分，结果会是 0，
表示信号中没有这一频率成分。
这就是傅里叶分析能“分解频率”的数学基础。

✅ 总结

不同频率的指数基函数 $e^{i\omega t}$ 是正交的，
因为它们的内积（在有限区间时积分为 0；在无限区间时为 δ 函数）为零。
这正是傅里叶变换可以唯一分离、投影出各个频率分量的原因。

## Question

当ω!=ω’，δ(ω-ω')是0，没有问题。
当ω=ω’，δ(0)应该是无穷大啊。你说的【有限值（归一化因子）】是怎么回事？

## Answer

非常棒的问题 👏👏
你完全抓住了傅里叶变换中最容易被“轻轻掩盖”的数学细节：
——δ(0) 到底代表什么？为什么我刚才说它是“有限值（归一化因子）”？

我们来一步步拆解这个问题，区分清楚“直觉说 δ(0)=\infty”和“数学上说它产生有限归一化”的区别。

#### 🧭 一、首先明确：δ(ω−ω′) 不是普通函数

狄拉克 δ 函数 不是一个真正的函数，而是一个分布（distribution）。
它不是在某一点“取值”的对象，而是一个线性泛函，专门为积分而存在。

它的定义特性是：

```math
\int_{-\infty}^{+\infty} f(\omega') \, \delta(\omega - \omega') \, d\omega' = f(\omega)
```

换句话说：

δ(ω−ω′) 只有在积分号里才“有意义”，
它的作用是从积分中选出 ω′=ω 处的值。

因此我们说 “δ(0)=\infty” 只是一个直觉性的记号，
表示它在 0 点“无限高”，但积分面积为 1。

#### 🧩 二、回到傅里叶基的“正交性公式”

我们刚才得到：

```math
\int_{-\infty}^{+\infty} e^{i(\omega - \omega')t} dt = 2\pi\,\delta(\omega - \omega')
```

这个式子意味着什么？
它的意义是：

不同频率的基函数在积分意义下是正交的，
并且它们的“范数平方”相当于 $2\pi \delta(0)$。

我们不能说“δ(0)”是一个具体数值（它确实是发散的），
但在归一化操作中，它被吸收到“归一化常数”里了。

#### 🧮 三、形式上看似“δ(0)=\infty”，但物理量是有限的

我们可以做个类比。
在有限区间 [0, T] 上的离散正交性是：

```math
\langle e^{i\omega_m t}, e^{i\omega_n t} \rangle
= \int_0^T e^{i(\omega_m - \omega_n)t} dt
= T\,\text{sinc}\!\left(\frac{(\omega_m - \omega_n)T}{2}\right)
```

当 $m=n$ 时，结果是 $T$（有限）；
当 $T\to\infty$ 时，频率间隔 $\Delta\omega = 2\pi/T \to 0$，
离散的正交性就变成连续的 δ 函数：

```math
\lim_{T\to\infty} T\,\text{sinc}\!\left(\frac{(\omega - \omega')T}{2}\right)
= 2\pi\,\delta(\omega - \omega')
```

于是 δ(0) 其实代表的是：

“当区间长度趋于无穷时，基函数的自内积趋于无穷大”，
但这个无穷大正好与频率积分的密度 $1/2\pi$ 互相抵消，
使得整个傅里叶变换保持有限且能量守恒。

这就是我说的“有限归一化因子”的意思：
并不是 δ(0) 真的是有限的，而是：

在傅里叶变换的定义中，常数因子（例如 $1/2\pi$）和 δ 的归一化共同确保公式整体一致、有限、可逆。

#### 🧠 四、酉性（能量守恒）的角度再看一次

傅里叶变换满足 Parseval 定理：

```math
\int_{-\infty}^{+\infty} |f(t)|^2 dt = \frac{1}{2\pi}\int_{-\infty}^{+\infty} |F(\omega)|^2 d\omega
```

这里的 $1/(2\pi)$ 就是那个“归一化因子”——
它确保了傅里叶变换在 $L^2$ 空间中是酉的（unitary），
即“能量守恒、长度不变”。

因此，当我们写出：

```math
\langle e^{i\omega t}, e^{i\omega' t} \rangle = 2\pi\,\delta(\omega - \omega')
```

其实在说的是：

“基向量的内积不是单位矩阵中的 1，而是 2πδ”，
“我们用 $1/2\pi$ 的系数去归一化，使整体仍然是酉变换”。

注意：对导数的变换（在合适的衰减/分布意义下）有：

```math
\mathcal F\{f'(t)\} = i\omega F(\omega),\qquad
\mathcal F\{f''(t)\} = -( \omega^2 ) F(\omega).
```

因此左边三项的傅里叶变换变为代数乘子：

```math
\mathcal F\{f''+2f'+5f\} = \big(-\omega^2 + 2i\omega + 5\big)\,F(\omega).
```

右边 $\sin(3t)$ 的傅里叶变换（利用复指数与 δ 的关系 $\mathcal F\{e^{i a t}\}=2\pi\delta(\omega-a)$）为：

```math
\sin(3t)=\frac{e^{i3t}-e^{-i3t}}{2i}
\quad\Rightarrow\quad
\mathcal F\{\sin(3t)\} = \frac{2\pi}{2i}\big(\delta(\omega-3)-\delta(\omega+3)\big)
= -i\pi\big(\delta(\omega-3)-\delta(\omega+3)\big).
```

于是在频域我们得到代数方程：

```math
\big(-\omega^2 + 2i\omega + 5\big) F(\omega)
= -i\pi\big(\delta(\omega-3)-\delta(\omega+3)\big).
```

解出 $F(\omega)$：

```math
F(\omega)
= \frac{-i\pi\big(\delta(\omega-3)-\delta(\omega+3)\big)}{-\omega^2+2i\omega+5}.
```

#### 2) 逆变换回时域（利用 δ 的挑选性质）

逆变换公式

```math
f(t)=\frac{1}{2\pi}\int_{-\infty}^{\infty}F(\omega)e^{i\omega t}\,d\omega
```

由于 $F(\omega)$ 中含 δ，积分只在 $\omega=\pm3$ 取值，从而积分大为简化。

把 $F(\omega)$ 代入并计算（把 δ 写开）得到：

```math
\begin{aligned}
f_{\text{part}}(t)
&=\frac{1}{2\pi}\int\frac{-i\pi\big(\delta(\omega-3)-\delta(\omega+3)\big)}{-\omega^2+2i\omega+5}\,e^{i\omega t}\,d\omega \\
&=\frac{1}{2\pi}\Big(\frac{-i\pi}{-\;3^2+2i\cdot3+5}e^{i3t}
+\frac{i\pi}{-(-3)^2+2i(-3)+5}e^{-i3t}\Big).
\end{aligned}
```

做代数化简（列出分母值）：

```math
-\;3^2+2i\cdot3+5 = -4 + 6i,\qquad
-(-3)^2+2i(-3)+5 = -4 - 6i.
```

化简后可写为实三角形式（把复指数合成正弦余弦）：

```math
f_{\text{part}}(t) = -\frac{1}{13}\sin(3t) - \frac{3}{26}\cos(3t).
```

你可以把它代回原方程验证：代入左端 $f''+2f'+5f$ 会得到 $\sin(3t)$。

#### 3) 加上齐次解得到全解

齐次方程

```math
f'' + 2f' + 5f = 0
```

的特征方程为 $r^2+2r+5=0$，其根为

```math
r = -1 \pm 2i.
```

对应的齐次解为阻尼振荡形式：

```math
f_{\text{hom}}(t) = e^{-t}\big( C_1\cos(2t) + C_2\sin(2t)\big).
```

因此微分方程的一般解为

```math
\boxed{\,f(t) = e^{-t}\big( C_1\cos(2t) + C_2\sin(2t)\big)
\;-\;\frac{1}{13}\sin(3t) - \frac{3}{26}\cos(3t).\,}
```

其中 $C_1,C_2$ 由初始条件决定；最后两项是上面通过傅里叶变换得到的特解（受迫响应）。

#### 4) 解释与小结（为什么频域法有用）

在频域里，微分算子变为代数乘子（例如 $d/dt\leftrightarrow i\omega$， $d^2/dt^2\leftrightarrow -\omega^2$），把微分方程“化微分为代数”，求解变得更直接。

对于右端是正弦或指数函数的“强迫项”，其傅里叶变换是 δ，逆变换时积分退化为在 δ 所在点处取值，计算非常简洁。

注意一些严谨性：傅里叶变换对函数的存在性有条件（衰减、平方可积或按分布处理）；当强迫项是无界时间延拓的正弦时，我们是用 δ/分布的观点来处理，这是常见且可接受的物理/工程做法。

如果完全用两个实数函数来表达傅里叶变换，那么公式应该是怎样的？我想看看。据说傅里叶本人最初的公式就是用实数函数的。

## Answer

完全可以，我们来把傅里叶变换写成两个实函数形式，也就是正弦和余弦的分解形式。这是傅里叶本人最早的思路，完全不依赖虚数 $i$。

#### 1️⃣ 傅里叶变换用正弦和余弦表示

假设 $f(t)$ 是实函数，定义：

```math
A(\omega) = \int_{-\infty}^{\infty} f(t) \cos(\omega t) \, dt, \quad
B(\omega) = \int_{-\infty}^{\infty} f(t) \sin(\omega t) \, dt
```

$A(\omega)$ 称为余弦分量（cosine transform）

$B(\omega)$ 称为正弦分量（sine transform）

这两个积分就把信号投影到两个正交的实函数基上。

#### 2️⃣ 逆变换（重构 f(t)）

通过叠加这些频率成分，可以还原原函数：

```math
f(t) = \frac{1}{\pi} \int_0^\infty \Big[ A(\omega) \cos(\omega t) + B(\omega) \sin(\omega t) \Big] d\omega
```

注意积分只需要从 $0$ 到 $\infty$，因为对实函数：

```math
A(-\omega) = A(\omega), \quad B(-\omega) = -B(\omega)
```

因此负频率的贡献已经通过正负对称性包含进去了。

#### 3️⃣ 和复指数形式的关系

如果用复指数表示：

```math
F(\omega) = \int_{-\infty}^{\infty} f(t) e^{-i\omega t} dt = \int f(t)[\cos(\omega t) - i \sin(\omega t)] dt
```

那么：

```math
\Re(F(\omega)) = A(\omega), \quad \Im(F(\omega)) = -B(\omega)
```

所以虚数形式其实是把两个实函数 $A(\omega),B(\omega)$ 统一成一个复函数，计算上更方便，但本质内容和傅里叶最初的公式完全一样。

#### 4️⃣ 总结公式（实函数形式）

```math
\boxed{
\begin{aligned}
& A(\omega) = \int_{-\infty}^{\infty} f(t) \cos(\omega t)\, dt, \\
& B(\omega) = \int_{-\infty}^{\infty} f(t) \sin(\omega t)\, dt, \\
& f(t) = \frac{1}{\pi} \int_0^\infty \big[ A(\omega)\cos(\omega t) + B(\omega)\sin(\omega t) \big]\, d\omega
\end{aligned}
}
```

✅ 这就是傅里叶本人最初用的实函数表示形式。
