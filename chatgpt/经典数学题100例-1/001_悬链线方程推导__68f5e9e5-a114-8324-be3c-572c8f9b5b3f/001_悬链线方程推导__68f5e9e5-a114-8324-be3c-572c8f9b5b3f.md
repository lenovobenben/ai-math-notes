# 悬链线方程推导

- Conversation ID: `68f5e9e5-a114-8324-be3c-572c8f9b5b3f`

## Question

悬链线方程是如何推导出来的呢？

## Answer

悬链线（catenary）的方程推导其实是一个经典的力学与微积分问题，涉及静力平衡和变分法。我给你完整梳理一下推导过程，并尽量条理清晰：

#### 1️⃣ 问题描述

假设一条均匀柔软的链子悬挂在两个固定点之间，在重力作用下达到平衡形状。我们希望求出链子的形状函数 $y(x)$。

链子的线密度：$\lambda$（单位长度重量）

重力加速度：

```math
g
```

水平坐标：$x$，垂直坐标：$y(x)$

#### 2️⃣ 力学分析

取链子上的一点 $P(x, y)$，考虑平衡条件：

链子的张力 $T$ 沿切线方向。

链子在垂直方向受到重力：$\lambda g \, ds$，其中 $ds = \sqrt{1 + (y')^2} \, dx$ 是弧长微元。

水平方向平衡：
张力的水平方向分量 $T_x$ 是常数 $H$，因为没有水平外力。

```math
T_x = T \cos \theta = H
```

垂直方向平衡：

```math
\frac{d}{dx}(T \sin \theta) = \lambda g \sqrt{1 + (y')^2}
```

注意：

```math
\tan \theta = y' \quad \Rightarrow \quad \sin \theta = \frac{y'}{\sqrt{1 + (y')^2}}, \quad \cos \theta = \frac{1}{\sqrt{1 + (y')^2}}
```

所以：

```math
T_x = T \cos \theta = H \quad \Rightarrow \quad T = H \sqrt{1 + (y')^2}
```

垂直力平衡：

```math
\frac{d}{dx}(T \sin \theta) = \frac{d}{dx}\left(H \sqrt{1 + (y')^2} \cdot \frac{y'}{\sqrt{1 + (y')^2}}\right) = \frac{d}{dx}(H y') = \lambda g \sqrt{1 + (y')^2}
```

得到：

```math
H y'' = \lambda g \sqrt{1 + (y')^2} \quad \Rightarrow \quad y'' = \frac{\lambda g}{H} \sqrt{1 + (y')^2}
```

#### 3️⃣ 微分方程求解

设：

```math
y' = p \quad \Rightarrow \quad y'' = p \frac{dp}{dy} \text{ 或 } y'' = p' \text{ 看作对 } x
```

直接使用 $y'' = \frac{\lambda g}{H} \sqrt{1 + (y')^2}$：

```math
\frac{d y'}{dx} = k \sqrt{1 + (y')^2}, \quad k = \frac{\lambda g}{H}
```

分离变量：

```math
\frac{dy'}{\sqrt{1 + (y')^2}} = k \, dx
```

对两边积分：

```math
\sinh^{-1}(y') = k x + C_1
```

取反双曲正弦：

```math
y' = \sinh(k x + C_1)
```

再积分一次求 $y(x)$：

```math
y = \frac{1}{k} \cosh(k x + C_1) + C_2
```

#### 4️⃣ 最终悬链线方程

```math
\boxed{y(x) = a \cosh\Big(\frac{x - b}{a}\Big) + c}
```

```math
a = \frac{H}{\lambda g}
```

$b$ 与 $c$ 由链子端点位置决定

#### ✅ 结论

悬链线是双曲余弦曲线，与抛物线不同，抛物线只是近似于小跨度悬链线。
核心思想：

水平张力恒定

垂直力平衡

构建微分方程并积分
