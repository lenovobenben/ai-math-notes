# 正三角形面积求解

- Conversation ID: `68fcd149-dec4-8323-b367-ac6d92c83be0`

## Question

已知正三角形内有一点P。 P到三个顶点的距离分别是3,4,5。
求这个正三角形的面积。

## Answer

我们来严谨地一步步推导。

##### 已知条件

在边长为 $a$ 的正三角形 $ABC$ 内有一点 $P$，
设：

```math
PA = 3,\quad PB = 4,\quad PC = 5
```

求：正三角形的面积。

##### 关键结论：维维安尼定理 (Viviani’s Theorem) 的推广

对于 正三角形：

点到三边距离之和恒等于高。
但这里给的是 到顶点的距离，不是到边的距离，需用坐标法处理。

##### 建立坐标系

设正三角形边长为 $a$，放置如下：

```math
A(0,0)
```

```math
B(a,0)
```

```math
C\left(\frac{a}{2}, \frac{\sqrt{3}}{2}a\right)
```

```math
P(x,y)
```

则：

```math
\begin{cases}
PA^2 = x^2 + y^2 = 9 \\
PB^2 = (x - a)^2 + y^2 = 16 \\
PC^2 = \left(x - \frac{a}{2}\right)^2 + \left(y - \frac{\sqrt{3}}{2}a\right)^2 = 25
\end{cases}
```

##### 解方程组

1️⃣ 由前两式相减：

```math
(x - a)^2 - x^2 = 16 - 9 = 7
```

```math
x^2 - 2ax + a^2 - x^2 = 7 \implies -2ax + a^2 = 7
```

```math
x = \frac{a^2 - 7}{2a}
```

2️⃣ 将 $x$ 代入第一式求 $y^2$：

```math
y^2 = 9 - x^2 = 9 - \left(\frac{a^2 - 7}{2a}\right)^2 = 9 - \frac{a^4 - 14a^2 + 49}{4a^2}
```

```math
\Rightarrow y^2 = \frac{36a^2 - (a^4 - 14a^2 + 49)}{4a^2} = \frac{-a^4 + 50a^2 - 49}{4a^2}
```

3️⃣ 代入第三式：

```math
\left(x - \frac{a}{2}\right)^2 + \left(y - \frac{\sqrt{3}}{2}a\right)^2 = 25
```

代入 $x$ 和 $y^2$。

计算：

```math
x - \frac{a}{2} = \frac{a^2 - 7}{2a} - \frac{a}{2} = \frac{a^2 - 7 - a^2}{2a} = -\frac{7}{2a}
```

```math
(x - \frac{a}{2})^2 = \frac{49}{4a^2}
```

展开第三式：

```math
\frac{49}{4a^2} + y^2 - \sqrt{3} a y + \frac{3a^2}{4} = 25
```

代入 $y^2 = \frac{-a^4 + 50a^2 - 49}{4a^2}$：

```math
\frac{49}{4a^2} + \frac{-a^4 + 50a^2 - 49}{4a^2} - \sqrt{3} a y + \frac{3a^2}{4} = 25
```

先合并前两项：

```math
\frac{-a^4 + 50a^2}{4a^2} - \sqrt{3} a y + \frac{3a^2}{4} = 25
```

```math
\Rightarrow \frac{-a^2}{4} + \frac{50}{4} - \sqrt{3} a y + \frac{3a^2}{4} = 25
```

```math
\Rightarrow \frac{2a^2}{4} + 12.5 - \sqrt{3} a y = 25
```

```math
\Rightarrow \frac{a^2}{2} - \sqrt{3} a y = 12.5
```

4️⃣ 代入 $y^2$：

从 $y^2 = \frac{-a^4 + 50a^2 - 49}{4a^2}$，取正的 $y$：

```math
y = \sqrt{\frac{-a^4 + 50a^2 - 49}{4a^2}} = \frac{\sqrt{-a^4 + 50a^2 - 49}}{2a}
```

代入上式：

```math
\frac{a^2}{2} - \sqrt{3} a \cdot \frac{\sqrt{-a^4 + 50a^2 - 49}}{2a} = 12.5
```

```math
\Rightarrow a^2 - \sqrt{3}\sqrt{-a^4 + 50a^2 - 49} = 25
```

##### 化简求 $a$

```math
\sqrt{3}\sqrt{-a^4 + 50a^2 - 49} = a^2 - 25
```

两边平方：

```math
3(-a^4 + 50a^2 - 49) = (a^2 - 25)^2
```

展开：

```math
-3a^4 + 150a^2 - 147 = a^4 - 50a^2 + 625
```

合并：

```math
-4a^4 + 200a^2 - 772 = 0
```

```math
a^4 - 50a^2 + 193 = 0
```

##### 令 $x = a^2$

```math
x^2 - 50x + 193 = 0
```

```math
x = \frac{50 \pm \sqrt{2500 - 772}}{2} = \frac{50 \pm \sqrt{1728}}{2} = \frac{50 \pm 24\sqrt{3}}{2} = 25 \pm 12\sqrt{3}
```

##### 取合理解

$a^2 = 25 - 12\sqrt{3}$ 会太小（约 4.216），边长 ≈ 2.05，不可能容纳点距顶点 3、4、5；
故取较大值：

```math
a^2 = 25 + 12\sqrt{3}
```

```math
a = \sqrt{25 + 12\sqrt{3}}
```

##### 求面积

正三角形面积：

```math
S = \frac{\sqrt{3}}{4} a^2 = \frac{\sqrt{3}}{4}(25 + 12\sqrt{3}) = \frac{25\sqrt{3}}{4} + 9
```

✅ 最终答案：

```math
\boxed{S = 9 + \frac{25\sqrt{3}}{4}}
```

若要近似值：

```math
\sqrt{3} \approx 1.732
```

```math
S \approx 9 + \frac{25 \times 1.732}{4} = 9 + 10.825 = 19.825
```

✅ 结论

正三角形的面积为：

```math
\boxed{S = 9 + \dfrac{25\sqrt{3}}{4} \approx 19.83}
```
