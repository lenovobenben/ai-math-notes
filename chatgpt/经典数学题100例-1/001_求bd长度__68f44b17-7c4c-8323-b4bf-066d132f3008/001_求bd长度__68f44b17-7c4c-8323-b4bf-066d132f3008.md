# 求BD长度

- Conversation ID: `68f44b17-7c4c-8323-b4bf-066d132f3008`

## Question

圆内接四边形ABCD。 AB=3,BC=4,CD=2,DA=2。求BD的长度。

## Answer

用余弦定理和圆内接四边形对角互补的性质来解。

设 $BD=x$。在三角形 $ABD$ 中，余弦定理得

```math
\cos\angle BAD=\frac{AB^2+AD^2-BD^2}{2\cdot AB\cdot AD}
=\frac{3^2+2^2-x^2}{2\cdot3\cdot2}=\frac{13-x^2}{12}.
```

在三角形 $BCD$ 中，

```math
\cos\angle BCD=\frac{BC^2+CD^2-BD^2}{2\cdot BC\cdot CD}
=\frac{4^2+2^2-x^2}{2\cdot4\cdot2}=\frac{20-x^2}{16}.
```

因为 $ABCD$ 为圆内接四边形，$\angle BAD$ 与 $\angle BCD$ 互为补角，所以

```math
\cos\angle BAD=-\cos\angle BCD.
```

代入上式得到

```math
\frac{13-x^2}{12}=-\frac{20-x^2}{16}.
```

两边去分母并整理：

```math
16(13-x^2)=-12(20-x^2)\Rightarrow 208-16x^2=-240+12x^2,
```

```math
448=28x^2\Rightarrow x^2=16\Rightarrow x=4.
```

因此 $BD=4$。（检验：用 Ptolemy 定理 $AC\cdot BD=AB\cdot CD+BC\cdot AD=6+8=14$，得 $AC=14/4=3.5$，与几何一致。）
