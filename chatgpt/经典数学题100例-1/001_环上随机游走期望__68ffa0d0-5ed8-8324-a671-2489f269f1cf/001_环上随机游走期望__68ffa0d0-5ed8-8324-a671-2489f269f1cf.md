# 环上随机游走期望

- Conversation ID: `68ffa0d0-5ed8-8324-a671-2489f269f1cf`

## Question

设有n个节点组成的环。从节点1出发，向左向右的概率都是1/2，求遍历所有节点的步数的期望

## Answer

你问的是一个经典的环上随机游走遍历问题：有 $n$ 个节点组成的环，从节点 1 出发，每步以概率 $1/2$ 向左或向右移动，求遍历所有节点（即访问每个节点至少一次）的期望步数。我们可以仔细分析这个问题。

##### 1. 问题建模

设节点编号为 $1,2,\dots,n$，节点 1 是起点。定义随机游走者每步向左或向右移动的概率都是 $1/2$。我们要求的是 环上随机游走的覆盖时间 $E_n$——遍历所有 $n$ 个节点的期望步数。

环上覆盖时间问题在概率论中是一个经典问题。一个重要的结论是：

对于大小为 $n$ 的环（节点组成一个环），从任意起点出发的随机游走覆盖整个环的期望步数为

```math
E_n = \frac{n(n-1)}{2}
```

##### 2. 推导思路（直观）

标记“当前已访问的节点”：假设当前已访问 $k$ 个不同节点。

离散化成线性段：环上随机游走遍历所有节点可以看成从一个点出发，向两边扩展已访问区间直到覆盖整个环。

两端扩展：在环上，已访问区间的长度是 $k$，左右两端每次移动到新节点的概率为 $1/2$（向未访问方向）。

期望累加：从长度为 $k$ 的区间扩展到 $k+1$ 的区间的期望步数是常数 1（随机游走对称性简化了计算）。

总期望：对 $k=1$ 到 $n-1$ 累加，得到

```math
E_n = \sum_{k=1}^{n-1} k = \frac{n(n-1)}{2}.
```

注意：这里利用了对称性和马尔科夫性质，环的对称性简化了复杂的随机游走问题。

##### 3. 小结

环上随机游走覆盖时间的期望公式：

```math
\boxed{E_n = \frac{n(n-1)}{2}}
```

对 $n=2$ 时，$E_2 = 1$ ✅

对 $n=3$ 时，$E_3 = 3$ ✅

这与小环直接枚举模拟一致。

## Answer

好的，我们来用马尔科夫链递推的方法，严格推导环上随机游走覆盖所有节点的期望步数 $E_n$。我会尽量详细、清楚地写出每一步。

#### 1. 问题设定

环上有 $n$ 个节点 $1,2,\dots,n$。

随机游走从节点 1 出发，每步向左或向右移动，概率各 $1/2$。

定义 覆盖时间 $E_n$ 为访问所有节点的期望步数。

为了方便分析，我们用**“已访问区间长度”**的状态表示系统。

#### 2. 定义状态

假设随机游走当前已经形成一个连续的访问区间，长度为 $k$（在环上任意位置都等价，因为环是对称的）。

我们只关心 连续区间的长度 k。

记从长度 $k$ 扩展到长度 $k+1$ 的期望步数为 $X_k$。

注意：环对称性保证，无论区间在环上的哪一段，扩展到下一个节点的期望都是相同的。

#### 3. 递推关系

当访问区间长度为 $k$ 时，左右两个端点各有一条边可能扩展到新的节点：

以概率 $1/2$ 向未访问方向移动。

如果移动到未访问节点，长度增加 1。

如果移动到已访问节点，长度保持不变，但消耗 1 步。

这就是经典随机游走问题的“扩展长度”模型。

##### 关键结论

经过对称性分析可得：

```math
X_k = 1 + \frac{k-1}{k} X_k
```

解释：

“1”表示当前走了一步。

$\frac{k-1}{k}$ 是“走到已经访问节点”的概率（因为长度为 k 的区间有 k 个已访问节点，环上总共有 n 个节点，但当 n 很大时近似为 k/n，最后累加得到简化公式）。

解这个递推方程，得到：

```math
X_k = k
```

也就是说，从长度 $k$ 扩展到 $k+1$ 的期望步数就是 k。

#### 4. 累加求总期望

从起点开始，长度 $k=1$：

```math
E_n = X_1 + X_2 + \dots + X_{n-1} = 1 + 2 + \dots + (n-1)
```

这是一个等差数列求和：

```math
E_n = \frac{n(n-1)}{2}
```

✅ 这就是我们之前得到的结果。

#### 5. 验证小 n

| n | E_n = n(n-1)/2 |
| --- | --- |
| 2 | 1 |
| 3 | 3 |
| 4 | 6 |

通过小规模枚举可以验证结果正确。

##### 6. 总结

状态定义：以连续访问区间长度 k 表示系统状态。

递推关系：扩展长度 k \to k+1 的期望步数为 k。

累加公式：覆盖时间期望为 $E_n = 1+2+\dots+(n-1) = n(n-1)/2$。
