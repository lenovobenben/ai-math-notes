# 解方程 3^x + 4^x = 5^x

- Conversation ID: `68ff7ab3-ffac-8323-8e8c-5c5f098d0550`

## Question

设 \(x \in \mathbb{R}\)，解方程：

```math
3^x + 4^x = 5^x
```

## Answer

我们来解方程：

```math
3^x + 4^x = 5^x.
```

##### 第一步：两边同除以 $5^x$

假设 $x \in \mathbb{R}$，且 $5^x > 0$，可以安全地两边除以 $5^x$：

```math
\left(\frac{3}{5}\right)^x + \left(\frac{4}{5}\right)^x = 1.
```

##### 第二步：设 $a = \frac{3}{5}$，$b = \frac{4}{5}$，注意 $0 < a < b < 1$

方程变为：

```math
a^x + b^x = 1.
```

##### 第三步：分析函数单调性

定义函数：

```math
f(x) = a^x + b^x.
```

因为 $0<a,b<1$，所以当 $x$ 增大时，$a^x, b^x$ 都递减，因此 $f(x)$ 严格递减。

##### 第四步：计算一些特值

当 $x = 0$ 时：

```math
f(0) = 1 + 1 = 2 \gt{} 1.
```

当 $x = 1$ 时：

```math
f(1) = \frac{3}{5} + \frac{4}{5} = \frac{7}{5} = 1.4 \gt{} 1.
```

当 $x = 2$ 时：

```math
f(2) = \left(\frac{3}{5}\right)^2 + \left(\frac{4}{5}\right)^2 = \frac{9}{25} + \frac{16}{25} = 1.
```

##### 第五步：结论

由于 $f(x)$ 单调递减，且 $f(2) = 1$，因此方程

```math
3^x + 4^x = 5^x
```

仅在 $x = 2$ 时成立。

✅ 最终答案：

```math
\boxed{x = 2.}
```

## Question

解是唯一的，对吧

## Answer

是的，✅ 解是唯一的。

理由如下更严格地说明：

##### 1️⃣ 方程

```math
3^x + 4^x = 5^x
```

等价于

```math
\left(\frac{3}{5}\right)^x + \left(\frac{4}{5}\right)^x = 1.
```

##### 2️⃣ 定义函数

```math
f(x) = \left(\frac{3}{5}\right)^x + \left(\frac{4}{5}\right)^x.
```

因为 $0 < \frac{3}{5}, \frac{4}{5} < 1$，所以：

```math
f'(x) = \ln\!\left(\tfrac{3}{5}\right)\left(\tfrac{3}{5}\right)^x + \ln\!\left(\tfrac{4}{5}\right)\left(\tfrac{4}{5}\right)^x.
```

注意到 $\ln(\tfrac{3}{5}) < 0$、$\ln(\tfrac{4}{5}) < 0$，且各项都为正数乘负数，因此：

```math
f'(x) \lt{} 0,
```

即 严格单调递减。

##### 3️⃣ 端点行为

当 $x \to -\infty$，$(3/5)^x, (4/5)^x \to +\infty$，所以 $f(x) \to +\infty$；

当 $x \to +\infty$，$(3/5)^x, (4/5)^x \to 0$，所以 $f(x) \to 0$。

因此 $f(x)$ 从 $+\infty$ 连续单调递减到 $0$。

##### 4️⃣ 解的唯一性

因为 $f(x)$ 连续且严格单调递减，又 $f(2)=1$，所以

```math
f(x)=1
```

只在 $x=2$ 时成立。

✅ 最终结论：

```math
\boxed{x = 2 \text{ 且唯一。}}
```
