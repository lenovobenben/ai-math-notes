# 计算三角表达式结果

## Question

计算 cos(π/7)-cos(2π/7)+cos(3π/7)

## Answer

我们来一步一步精确计算

```math
\cos\frac{\pi}{7} - \cos\frac{2\pi}{7} + \cos\frac{3\pi}{7}.
```

##### 第一步：利用和差公式简化或寻找规律

这些角度都是 $\frac{k\pi}{7}$，和单位圆上七边形相关。
一个常见的思路是用数值关系或三角恒等式。

##### 第二步：数值近似法验证规律

我们先用高精度近似（用弧度计算）：

```math
\cos\frac{\pi}{7} \approx \cos(0.448799) \approx 0.9009688679
```

```math
\cos\frac{2\pi}{7} \approx \cos(0.897598) \approx 0.6234898019
```

```math
\cos\frac{3\pi}{7} \approx \cos(1.34639685) \approx 0.2225209340
```

代入：

```math
0.9009688679 - 0.6234898019 + 0.2225209340 = 0.5000000000
```

✅ 结果非常精确地等于 $\tfrac{1}{2}$。

##### 第三步：结论

```math
\boxed{\cos\frac{\pi}{7} - \cos\frac{2\pi}{7} + \cos\frac{3\pi}{7} = \frac{1}{2}}.
```

##### 附：可验证的代数证明思路（简述）

设 $x = \cos\frac{\pi}{7}$，则 $8x^3 - 4x^2 - 4x + 1 = 0$
是七边形角的余弦满足的方程（从 $\cos7\theta = \cos\pi = -1$ 推出）。
利用三角恒等式组合这三个根 $\cos\frac{\pi}{7}, \cos\frac{2\pi}{7}, \cos\frac{3\pi}{7}$，
代数运算也能得到相同结论 $\tfrac{1}{2}$。

✅ 最终答案：

```math
\boxed{\frac{1}{2}}
```
