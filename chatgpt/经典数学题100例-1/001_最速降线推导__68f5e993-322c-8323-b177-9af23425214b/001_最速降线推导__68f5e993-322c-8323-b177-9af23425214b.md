# 最速降线推导

- Conversation ID: `68f5e993-322c-8323-b177-9af23425214b`

## Question

经典的最速降线是如何推导出来的呢？

## Answer

你问的是最速降线（Brachistochrone）问题的推导，这是一个经典的变分法问题，我来给你从头到尾系统讲清楚。我们一步步推导，不跳步。

#### 1️⃣ 问题描述

假设有两点 $A=(x_1, y_1)$ 和 $B=(x_2, y_2)$（假设 $y$ 竖直向下），我们要找一条光滑曲线 $y(x)$，使一个质点从 $A$ 沿这条曲线下滑到 $B$ 所用时间最短。
已知重力加速度 $g$，忽略摩擦。

目标：求最短时间的曲线 $y(x)$。

#### 2️⃣ 写出时间函数

质点从 $A$ 到 $B$ 的时间为

```math
T = \int_A^B \frac{ds}{v}
```

```math
ds = \sqrt{dx^2 + dy^2} = \sqrt{1 + (y')^2} dx
```

根据能量守恒，滑下高度 $y$ 的速度为

```math
v = \sqrt{2 g y}
```

因此时间积分变为

```math
T[y(x)] = \int_{x_1}^{x_2} \frac{\sqrt{1 + (y')^2}}{\sqrt{2 g y}} \, dx
```

#### 3️⃣ 应用变分法

这是一个标准的泛函最小化问题：

```math
T[y] = \int_{x_1}^{x_2} F(y, y') \, dx, \quad F(y, y') = \frac{\sqrt{1 + (y')^2}}{\sqrt{2 g y}}
```

变分法给出的欧拉–拉格朗日方程为：

```math
\frac{\partial F}{\partial y} - \frac{d}{dx} \frac{\partial F}{\partial y'} = 0
```

#### 4️⃣ 计算偏导

```math
F(y, y') = \frac{\sqrt{1 + (y')^2}}{\sqrt{2 g y}}
```

```math
\frac{\partial F}{\partial y} = - \frac{\sqrt{1 + (y')^2}}{2 \sqrt{2 g} y^{3/2}}
```

```math
\frac{\partial F}{\partial y'} = \frac{y'}{\sqrt{2 g y} \sqrt{1 + (y')^2}}
```

把它代入欧拉–拉格朗日方程，得到一个二阶微分方程。

#### 5️⃣ 使用“没有显式 x 的情况”

注意 $F(y, y')$ 不显含 x，于是可以用Beltrami 恒等式：

```math
F - y' \frac{\partial F}{\partial y'} = \text{常数 } C
```

计算：

```math
F - y' \frac{\partial F}{\partial y'} = \frac{\sqrt{1 + (y')^2}}{\sqrt{2 g y}} - y' \cdot \frac{y'}{\sqrt{2 g y} \sqrt{1 + (y')^2}} = \frac{1}{\sqrt{2 g y}} \cdot \frac{1 + (y')^2 - (y')^2}{\sqrt{1 + (y')^2}} = \frac{1}{\sqrt{2 g y} \sqrt{1 + (y')^2}}
```

令其等于常数 $\frac{1}{2g a}$（常数可换成其他形式，方便表示）：

```math
\frac{1}{\sqrt{2 g y} \sqrt{1 + (y')^2}} = \frac{1}{2 g a} \implies 1 + (y')^2 = \frac{4 g^2 a^2}{2 g y} = \frac{2 g a^2}{y}
```

#### 6️⃣ 分离变量并积分

得到：

```math
y' = \sqrt{\frac{2 g a^2}{y} - 1}
```

可以引入参数 $\theta$ 方便积分：

```math
\begin{cases}
x = a(\theta - \sin\theta) + x_0 \\
y = a(1 - \cos\theta) + y_0
\end{cases}
```

这就是**圆摆线（Cycloid）**的参数方程。

#### ✅ 7️⃣ 结论

最速降线是一条圆摆线，不是直线也不是抛物线。

参数 $a$ 与曲线的“大小”有关

起点 $A$ 决定 $x_0, y_0$

下滑时间是最短的
