# 无法表示cos20°

- Conversation ID: `68f746a4-1bdc-8323-a139-a84f8e2febbd`

## Question

通过对Q进行平方根和立方根的扩域，但是不准添加虚数单位i，是无法扩到cos20°的吧？

## Answer

你说的这个问题，其实涉及代数数的可解性与根式可表示性的问题。我们来仔细分析一下。

##### 1. 背景

你说的“对 $\mathbb{Q}$ 做平方根和立方根的扩域”实际上是指：

从有理数 $\mathbb{Q}$ 出发

允许添加元素 $\sqrt{a}$ 或 $\sqrt[3]{b}$，其中 $a, b \in \text{现有域}$

但不允许引入虚数单位 $i = \sqrt{-1}$

这样得到的扩域是实根式扩域（real radicals extension）。

而你想表示 $\cos 20^\circ$，即 $\cos \frac{\pi}{9}$。

##### 2. $\cos 20^\circ$ 的多项式

我们知道 $\cos \theta$ 是三倍角公式的根：

```math
\cos 3\theta = 4\cos^3 \theta - 3\cos \theta
```

令 $\theta = 20^\circ$，则 $3\theta = 60^\circ$，$\cos 60^\circ = \frac12$，于是：

```math
4x^3 - 3x - \frac12 = 0, \quad x = \cos 20^\circ
```

乘 2 得：

```math
8x^3 - 6x - 1 = 0
```

这是 $\cos 20^\circ$ 满足的三次方程。

##### 3. 能否用实根式解？

一个三次方程 $8x^3 - 6x - 1 = 0$ 计算判别式：

```math
\Delta = \left( \frac{q}{2} \right)^2 + \left( \frac{p}{3} \right)^3
```

标准三次式 $x^3 + px + q = 0$，这里：

```math
x^3 - \frac{3}{4} x - \frac{1}{8} = 0 \quad (\text{除以 }8)
```

所以 $p = -\frac34$, $q = -\frac18$

```math
\Delta = \left(-\frac{1}{16}\right)^2 + \left(-\frac14\right)^3 = \frac{1}{256} - \frac{1}{64} = -\frac{3}{256} \lt{} 0
```

判别式 < 0，说明三根都是实数，但卡尔丹公式里会出现 $\sqrt{\Delta}$，也就是虚数。这是著名的 “casus irreducibilis”（不可解的三实根三次方程的情况）。

结论：如果不允许使用 $i$，无法用有限次平方根和立方根表示 $\cos 20^\circ$。

##### ✅ 4. 总结

$\cos 20^\circ$ 是三次方程 $8x^3 - 6x - 1 = 0$ 的根。

该三次方程属于 casus irreducibilis，三根均为实数，但用根式表示必涉及虚数。

如果只允许实平方根和立方根，不允许 $i$，无法得到 $\cos 20^\circ$。
