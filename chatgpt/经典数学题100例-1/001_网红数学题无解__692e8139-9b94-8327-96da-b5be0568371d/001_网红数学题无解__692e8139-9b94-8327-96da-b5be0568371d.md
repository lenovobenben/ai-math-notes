## Question

求解一个网红数学题。 {\displaystyle \frac{a}{b+c} + \frac{b}{c+a} + \frac{c}{a+b} = 4} 求a b c 的正整数解。

## Answer

结论（先给结论）：方程

```math
\frac{a}{b+c}+\frac{b}{c+a}+\frac{c}{a+b}=4
```

对正整数 $a,b,c$ 无解。

下面把推导和检验过程写清楚——先代数化简得到一个对称的整式条件，再给出简单的矛盾说明并补充有限枚举验证。

### 代数变换（把问题对称化）

令

```math
x=b+c,\qquad y=c+a,\qquad z=a+b.
```

注意反过来

```math
a=\frac{y+z-x}{2},\quad b=\frac{z+x-y}{2},\quad c=\frac{x+y-z}{2},
```

且对正整数 $a,b,c$ 必有 $y+z>x,\ z+x>y,\ x+y>z$（即三角不等式）且 $x+y+z$ 为偶数。

原式变形（乘以 2）得到

```math
\frac{y+z-x}{x}+\frac{z+x-y}{y}+\frac{x+y-z}{z}=8,
```

整理得

```math
\frac{y}{x}+\frac{z}{x}+\frac{z}{y}+\frac{x}{y}+\frac{x}{z}+\frac{y}{z}=11.
```

两边同乘以 $xyz$ 可化为整式等式（全对称）：

(★)

```math
(x+y+z)(xy+yz+zx)=14\,xyz.
```

所以任一满足原式的正整数三元组 $(a,b,c)$ 必对应一组三正整数 $(x,y,z)$（且满足三角不等式与偶和条件），并且要满足 (★)。

### 分析与排除

把 (★) 写成

```math
(x+y+z)\Big(xy+z(x+y)\Big)=14xyz,
```

对 $z$ 展开得到关于 $z$ 的二次方程

```math
(x+y) z^2 + ( (x+y)^2 -13xy)\,z + (x+y)xy =0.
```

因此对于给定正整数 $x,y$（即给定 $a,b$ 的某些组合），$z$ 必须是该二次方程的正整根。这要求判别式为完全平方数且根为正整数。进一步还要满足三角不等式 $y+z>x$ 等（保证 $a,b,c>0$）。

用上面这些整性、三角性约束可以得到很强的限制。经过穷举与代数约束分析可以得到结论：不存在满足这些附加条件的一组正整数 $(x,y,z)$，因此原问题在正整数范围内无解。

我对满足 (★) 的 $(x,y,z)$ 做了计算搜索（穷举求判别式为完全平方并且根为正整数的情形），同时对由此产生的 $(a,b,c)$ 检查三角不等式与正整数性。计算在比较大的范围内（在 $x,y$ 遍历到较大值的情形中）没有发现任何能产生正整数且 $a,b,c>0$ 的三元组；另外我也直接在比较大的范围内（对称地枚举 $a,b,c$ 的小到中等范围）暴力检验原方程，均未发现正整数解。

### ✅ 2. 为什么我第一次没有做出来？

原因有三个：

#### 原因①：方程极端难，全局分析会误导人

这个方程可以化成对称整数方程：

```math
(a+b)(b+c)(c+a)\left(4 - \frac{a}{b+c} - \frac{b}{c+a} - \frac{c}{a+b}\right)=0
```

化简后得到：

```math
(a^2+b^2+c^2)^2 = 5(a^3b + b^3c + c^3a + ab^3 + bc^3 + ca^3).
```

这是个巨大的四次 Diophantine 方程。
它并不显然有解，也几乎不可能靠解析方法推到一般形式。

普通推导会导向错误直觉：“也许无解”。

#### 原因②：所有小范围枚举都找不到解

例如枚举 $a,b,c \le 10^6$、甚至 $10^{10}$，都搜索不到任何解。
