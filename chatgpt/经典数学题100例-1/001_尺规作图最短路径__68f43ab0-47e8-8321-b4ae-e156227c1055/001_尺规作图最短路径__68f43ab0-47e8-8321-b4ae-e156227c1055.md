# 尺规作图最短路径

- Conversation ID: `68f43ab0-47e8-8321-b4ae-e156227c1055`

## Question

尺规作图问题。圆外两点A,B。求圆上一点P，使得|AP|+|BP|最短。怎么做？

## Answer

把问题做成尺规作图并给出理由，这里给出一个方便构造且常见的方法（用圆心为反演中心的反演 — 反演半径取为给定圆的半径；反演可以用直尺+圆规构造）。

### 结论（便于记忆）

令 $O$ 为给定圆的圆心，半径为 $R$。对点 $A,B$ 做以 $O$ 为中心、$R$ 为半径的反演，得到像点 $A',B'$。令直线 $A'B'$ 与原圆的交点为 $P$（注意可能有两个交点，取使 $AP+BP$ 更小的那个——通常是靠近 $A$ 和 $B$ 的那个交点）。该点 $P$ 即为在圆上使 $AP+BP$ 最小的点。

### 具体尺规作图步骤

已知圆 $\Gamma(O,R)$，已知圆外两点 $A,B$。

用直尺+圆规构造点 $A'$：在射线 $OA$ 上，构造满足 $OA\cdot OA' = R^2$ 的点（即以 $O$ 为反演中心、$R$ 为反演半径把 $A$ 反演到 $A'$）。同理构造 $B'$（标准的反演作图：在直线 $OA$ 上作长度 $\dfrac{R^2}{OA}$）。

连接 $A'B'$ 作直线，求它与原圆 $\Gamma$ 的交点（最多两个）。把这些交点记为候选点。

在候选交点中计算或比较（用尺规可比较距离之和或用三角不等式判断）哪一个使 $AP+BP$ 更小；取该点为所求 $P$。

（如果只需要一种可作出的点，把 $A'B'$ 与圆的交点作为答案即可——一般这两点中有且只有一个是全局最小点。）

### 说明与证明要点（直观）

以 $O$ 为中心、半径 $R$ 的反演把圆 $\Gamma$ 上的点逐点不动（圆上点是反演不变点），把圆外的点 $A,B$ 变为圈内点 $A',B'$。

反演把从 $A$ 到圆上点 $P$ 的有向线段与从 $A'$ 到同一点 $P$ 的线段成一一对应关系，且满足 $PA\cdot PA' = R^2$（相应关系可用于变形总距离的单调性论证）。

关键观察：在反演后问题等价于在圆上寻找点 $P$ 使得某种（与距离和单调相关的）函数在 $P$ 上最小，而直线 $A'B'$ 与圆的交点给出了极值（可以从变分/反射性质或微分条件看出），因此取 $A'B'$ 与圆的交点即得到原问题的最优点。

另一个直观解释：在反演平面内，$A'B'$ 的直线与圆交于点，反演回去对应满足“角度对称/梯度”条件的点，从而在原问题中使 $AP+BP$ 达到最小。

### 备注

反演中心选为圆心、半径取为该圆的半径是技巧性的选择：这样圆上的点在反演中保持不动，便于做出不动点上的最优化问题转化，且反演是用圆规与直尺可以构造的（在尺规体系内可作出与给定比例相关的点）。

实际作图时 $A'B'$ 与圆常交两点，按常理应比较两者的 $AP+BP$（或观察几何位置判断哪一个是更近的极小点）。

你说得对 —— 我先前的“反演可尺规作图”的说法是不恰当的。更精确的结论是：

一般情况下（对一般给定的圆和圆外两点 $A,B$），在圆上使 $AP+BP$ 最小的点 $P$ 不能用尺规作图得到。

下面给出简要、清晰的理由与说明（尽量把要点讲明，不做冗长的技术证明）。

### 关键思路（直观）

把目标函数记为

```math
f(P)=|AP|+|BP|,\qquad P\in\Gamma
```

其中 $\Gamma$ 是给定圆。取参数化 $P=(x,y)$ 满足圆方程（例如单位圆 $x^{2}+y^{2}=1$），最小化 $f$ 等价于在圆上解方程由一阶条件（拉格朗日乘子或几何的“切线反射/等角”条件）得到的一组代数方程。去掉根号、消去乘子后，会得到以 $x$（或以 $y$）为未知数的代数多项式方程，其次数通常为 4（或更高），即代数次数为四的代数方程是“典型情形”。

### 为什么这与尺规作图矛盾

已知代数数是否能用尺规作图构造，与它们所在的扩域阶数有关：

用尺规能构造的点的坐标属于从有理数出发按有限次取平方根扩张得到的域——也就是说相应代数扩域的阶数必须是 $2^k$ 的形式（仅含 2 的幂因子）。

而一般的四次（quartic）方程的根的伽罗瓦群在多数“泛型”情形下是 $S_4$（阶 24），这不是 $2$ 的幂，所以这类根（作为代数数）通常不是通过仅用平方根可达的扩张得到的，从而不是尺规可构造的。

因此，对一般位置的 $A,B$ 和圆，最优点 $P$ 的坐标会生成一个非 $2$-幂阶的扩域（比如产生 $S_4$ 的情形），从而不能用尺规构造。

（换句话说：尽管四次方程在代数上“可用代数根式表示”，但尺规作图允许的运算只限于二次扩张，四次方程的根在一般情形下会需要不是由嵌套平方根生成的扩张，所以不可构造。）

### 更具体的说明（不严谨但足够说明“泛型不可构造”的事实）

将拉格朗日乘子条件（或几何等角条件）与圆的方程联立，消去根号后得到一个多项式方程（在一般坐标取值下为四次）。

代数几何/伽罗瓦论的常识告诉我们：四次方程的伽罗瓦群对参数来说是“开集”（generic）情形为 $S_4$。因此存在开集的参数（即“几乎所有”的 $A,B$ 相对位置）使得生成的伽罗瓦群为 $S_4$ —— 那样的解不是尺规可构造的。

结论：并非所有实例都可尺规作图；且“几乎所有”实例是不可能用尺规作图的。也就是说结论不是个别病例，而是一个一般性的不可构造结论。

### 注释与对之前错误的解释

我之前给出的“以圆心为反演中心，把 $A,B$ 反演到 $A',B'$，连 $A'B'$ 与圆交点即为最优点”的说法并不成立为一般命题（也不是一个可行的尺规构造法则）。那种反演变换会把问题转换成另一种方程，但并不保证相交点就是全局最小点（也不保证能用尺规得到）。

简单几何直觉（如“反射/等角”）能给出必要条件，但并不能保证这些条件对应的方程一定落在可尺规解的类。

明白，你想要明确**“在一般坐标系中，圆心在原点、半径为 1 的单位圆上，从圆外两点 $A,B$ 到圆上点 $P$ 的最短距离问题，什么时候 $P$ 可尺规作图”**，并排除掉一些显然退化的情况。我们可以从代数与伽罗瓦角度分析。

#### 1️⃣ 基本设定

圆：

```math
x^2+y^2=1
```

圆外点：$A=(a_x,a_y)$，$B=(b_x,b_y)$，且 $\sqrt{a_x^2+a_y^2}>1$、$\sqrt{b_x^2+b_y^2}>1$

圆上点：

```math
P=(\cos\theta,\sin\theta)
```

目标函数：

```math
f(\theta)=|AP|+|BP|=\sqrt{(a_x-\cos\theta)^2+(a_y-\sin\theta)^2} + \sqrt{(b_x-\cos\theta)^2+(b_y-\sin\theta)^2}
```

一阶必要条件（拉格朗日乘子）：

```math
\frac{d}{d\theta} f(\theta) = 0 \quad \implies \quad
\frac{(\cos\theta - a_x)\sin\theta - (\sin\theta - a_y)\cos\theta}{\sqrt{(a_x-\cos\theta)^2+(a_y-\sin\theta)^2}} + \frac{(\cos\theta - b_x)\sin\theta - (\sin\theta - b_y)\cos\theta}{\sqrt{(b_x-\cos\theta)^2+(b_y-\sin\theta)^2}} = 0
```

化简得到一个关于 $\cos\theta, \sin\theta$ 的代数方程。用代数消元消掉 $\sin\theta=\pm\sqrt{1-\cos^2\theta}$，最终得到 一个四次方程：

```math
c_4 x^4 + c_3 x^3 + c_2 x^2 + c_1 x + c_0 = 0, \quad x = \cos\theta
```

对于一般 $A,B$，四次方程的伽罗瓦群为 $S_4$，不可尺规作图。

可尺规作图的必要条件：四次方程的伽罗瓦群必须是 $2$-幂阶群（即它必须可由嵌套平方根解出）。

#### 2️⃣ 四次方程可尺规作图的几何条件

根据伽罗瓦理论，四次方程只有在“退化”或“对称”情形才可能可尺规作图：

方程降阶到三次或二次：

四次方程退化为三次方程（可由平方根解出）或二次方程（直接平方根解出），那么根式只含平方根，尺规可作图。

这种退化对应于几何上 A,B 与圆存在特殊对称：

$A,B$ 关于圆心对称（O为中点）

$A,B$ 关于某条直径对称

$A,B$ 与 O 共线

$A,B$ 关于圆上某条切线或直径对称

特殊比例条件：

方程可以分解成两个二次方程的乘积（例如两点关于圆心形成特殊矩形或正方形配置）

这要求 $(a_x,b_x,a_y,b_y)$ 满足特定代数关系，使四次方程的伽罗瓦群降阶为 $V_4$ 或小于 4 的二次扩张群

#### 3️⃣ 总结性结论（非平凡情形）

在排除显然退化的几何对称后（如共线、关于直径对称、共圆心、坐标轴对称等），一般圆外两点 $A,B$ 的最短距离点 $P$ 是不可尺规作图的。

可尺规作图的必要几何条件（非平凡）大致为：

$A,B$ 关于圆心或某条直径对称；

$A,B$ 构成的配置使得产生的四次方程可分解为两个二次方程（即方程的伽罗瓦群为 $2$-幂阶）；

或者 A,B 的坐标满足特定代数条件，使得 $\cos\theta$ 可由嵌套平方根表示。

换句话说，除了这些高度对称或代数“巧合”的情况，泛型的圆外两点配置，最短点 P 是不可尺规作图的。

可以做到，但这个判别式会非常复杂，因为本质上是一个 四次方程的伽罗瓦群是否为 2-幂阶 的判别问题。我们可以严格从代数出发，把几何最短点问题化为四次方程，然后利用四次方程的判别式和特解条件构造一个代数判别式。下面我给出完整思路：

#### 1️⃣ 参数化问题

圆：单位圆 $x^2+y^2=1$

圆外点：$A=(x_a,y_a)$, $B=(x_b,y_b)$

圆上点：$P=(x,y)$，且 $x^2+y^2=1$

目标函数：

```math
f(P) = \sqrt{(x-x_a)^2+(y-y_a)^2} + \sqrt{(x-x_b)^2+(y-y_b)^2}
```

一阶条件（对圆上的点最小距离）可写成向量形式：

```math
\frac{\vec{OP}-\vec{OA}}{|\vec{OP}-\vec{OA}|} + \frac{\vec{OP}-\vec{OB}}{|\vec{OP}-\vec{OB}|} = \lambda \vec{OP}
```

其中 $\lambda$ 为拉格朗日乘子。

#### 2️⃣ 化为代数方程

令 $P=(\cos\theta,\sin\theta)$，则

```math
f'(\theta) = 0 \implies \frac{(x_a-\cos\theta)\sin\theta-(y_a-\sin\theta)\cos\theta}{\sqrt{(x_a-\cos\theta)^2+(y_a-\sin\theta)^2}} + \frac{(x_b-\cos\theta)\sin\theta-(y_b-\sin\theta)\cos\theta}{\sqrt{(x_b-\cos\theta)^2+(y_b-\sin\theta)^2}} = 0
```

消掉 $\sin\theta = \pm\sqrt{1-\cos^2\theta}$，得到一个关于 $x=\cos\theta$ 的四次方程：

```math
c_4 x^4 + c_3 x^3 + c_2 x^2 + c_1 x + c_0 = 0
```

其中 $c_0,\dots,c_4$ 都是 $x_a,y_a,x_b,y_b$ 的多项式。

#### 3️⃣ 四次方程可尺规作图的代数条件

四次方程的根可以通过尺规构造 当且仅当：

四次方程退化成两个二次方程的乘积，或者

四次方程的伽罗瓦群为 2-幂阶（即 $V_4$ 或更小的子群）

对于一般四次方程：

```math
x^4 + p x^3 + q x^2 + r x + s = 0
```

可尺规作图的条件可以写成判别式形式：

令 $y = x^2 + \frac{p}{2}x + m$，代入四次方程，得到配平方程。

构造三次解的判别式 $\Delta_3$：

若 $\Delta_3 = 0$，则四次方程可分解为二次方程乘积，根可尺规作图。

更通用的条件是“quartic resolvent cubic 的判别式为完全平方”：

具体公式（基于 Ferrari 法）：

```math
\text{quartic } x^4 + p x^3 + q x^2 + r x + s = 0 \text{ 可尺规作图} \iff \text{resolvent cubic 有有理平方根解}
```

在你的问题中：

将 $c_4 x^4 + c_3 x^3 + c_2 x^2 + c_1 x + c_0 = 0$ 归一化为 $x^4 + a x^3 + b x^2 + c x + d = 0$

构造 Ferrari cubic resolvent：

```math
y^3 - b y^2 + (ac-4d) y + (4bd - c^2 - a^2 d) = 0
```

判定这个 cubic 是否有平方根解即可。

所以最终判别式就是：

```math
\exists y \in \mathbb{R} \text{ 满足 } y^3 - b y^2 + (ac-4d) y + (4bd - c^2 - a^2 d) = z^2 \text{ 对某个实数 } z
```

这个判别式完全用 $x_a,y_a,x_b,y_b$ 表示，因为 $a,b,c,d$ 都是 $x_a,y_a,x_b,y_b$ 的多项式。

你可以直接用 $x_a,y_a,x_b,y_b$ 算出四次方程系数 $c_4,c_3,c_2,c_1,c_0$

#### 1️⃣ 核心几何性质

已知圆心 $O=(0,0)$、圆上点 $P=(x,y)$、圆外两点 $A=(x_a,y_a), B=(x_b,y_b)$

目标：$|AP|+|BP|$ 最小

角平分线性质（经典结论）：

在圆外两点到圆上点的最短路径 $AP+BP$ 时，圆心 $O$ 的连线 $OP$ 恰好是角 $\angle APB$ 的平分线。

代入坐标系后：如果直线 $OP$ 与向量 $\vec{PA},\vec{PB}$ 的方向满足角平分线条件，就能得到一个代数等式，完全用斜率表示。

#### 2️⃣ 斜率表示

设：

```math
s_1 = \text{斜率 } AP = \frac{y-y_a}{x-x_a}, \quad s_2 = \text{斜率 } BP = \frac{y-y_b}{x-x_b}
```

设 $OP$ 的斜率：

```math
s_0 = \frac{y}{x}
```

角平分线条件（利用斜率公式）：

若 $l$ 是两直线 $l_1,l_2$ 的角平分线，且斜率分别为 $s_1,s_2$，则满足

```math
\frac{s_1 - s_0}{1 + s_1 s_0} = - \frac{s_2 - s_0}{1 + s_2 s_0}
```

这是因为平分线斜率公式（tan α 法则）消除了根号。

#### 3️⃣ 建立多项式方程

将上式化为代数形式：

```math
(s_1 - s_0)(1 + s_2 s_0) + (s_2 - s_0)(1 + s_1 s_0) = 0
```
