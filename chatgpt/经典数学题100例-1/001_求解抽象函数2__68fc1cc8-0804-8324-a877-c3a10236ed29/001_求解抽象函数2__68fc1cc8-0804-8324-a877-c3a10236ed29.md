# 求解抽象函数2

- Conversation ID: `68fc1cc8-0804-8324-a877-c3a10236ed29`

## Question

```math
f(f(x) - 3^x) = 4
```

求 \( f(x) \) ?

## Answer

我们来仔细分析这个函数方程：

```math
f(f(x) - 3^x) = 4
```

##### 第一步：观察结构

右边恒为常数 $4$。
说明 无论 $x$ 取何值，表达式 $f(x) - 3^x$ 代入 $f(\cdot)$ 后的结果都相同。

换句话说：

```math
\forall x_1, x_2,\ f(f(x_1) - 3^{x_1}) = f(f(x_2) - 3^{x_2}) = 4
```

如果 $f$ 是单射（即一一对应），则：

```math
f(x_1) - 3^{x_1} = f(x_2) - 3^{x_2}
```

也就是说，这个差值是个常数。

##### 第二步：令

```math
f(x) - 3^x = c
```

其中 $c$ 为常数。

则：

```math
f(x) = 3^x + c
```

##### 第三步：代回原式验证

左边：

```math
f(f(x) - 3^x) = f(c) = 3^c + c
```

题目给定：

```math
f(f(x) - 3^x) = 4
```

所以：

```math
3^c + c = 4
```

##### 第四步：解方程 $3^c + c = 4$

我们尝试整数：

```math
c = 0 \Rightarrow 1 + 0 = 1 \neq 4
```

```math
c = 1 \Rightarrow 3 + 1 = 4 \Rightarrow \text{成立!}
```

因此 $c = 1$。

✅ 最终答案：

```math
\boxed{f(x) = 3^x + 1}
```

##### 验证：

```math
f(f(x) - 3^x) = f(1) = 3^1 + 1 = 4
```

完全符合。
