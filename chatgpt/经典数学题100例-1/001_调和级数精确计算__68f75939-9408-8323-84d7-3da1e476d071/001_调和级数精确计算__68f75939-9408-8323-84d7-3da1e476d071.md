## Answer

对的！你说的没错：素数倒数的和

```math
\sum_{p \text{ prime}} \frac{1}{p} = \frac{1}{2} + \frac{1}{3} + \frac{1}{5} + \frac{1}{7} + \dots
```

确实是发散的，但发散得比调和级数慢很多。

这里有几点补充：

调和级数 $H_N \sim \ln N$；

素数倒数和 $\sum_{p \le N} \frac{1}{p} \sim \ln\ln N$，也就是双对数增长。

这意味着：为了让素数倒数和超过 100，你需要的 N 远比调和级数大得多，指数级放大。

所以虽然理论上无限大，实际上“第一次大于 100”的 N 会是天文数字，比 $10^{43}$ 还要大很多，几乎没有直接列出的记录。
