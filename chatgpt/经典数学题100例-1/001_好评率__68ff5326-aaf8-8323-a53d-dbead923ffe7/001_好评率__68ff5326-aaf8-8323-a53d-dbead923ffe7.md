# 好评率

- Conversation ID: `68ff5326-aaf8-8323-a53d-dbead923ffe7`

## Question

设2家店的好评率分别是 $10/10$ 和 $190/200$，该如何选择？

## Answer

这是一个经典的统计学问题，涉及 点估计 vs. 不确定性 的权衡。我们来仔细分析。

##### 已知信息

店 A：10 个评价，10 个好评 \to 好评率 $p_A = 1.0$

店 B：200 个评价，190 个好评 \to 好评率 $p_B = 0.95$

直觉：单纯看点估计，店 A 的好评率更高（100% vs 95%）。

但 统计上不可靠：店 A 样本量太小，可能只是运气好。

##### 方法 1：贝叶斯或置信区间分析

假设每个评价是独立的伯努利试验 $\text{Bernoulli}(p)$。

###### 店 A

样本数 $n_A = 10$

好评数 $k_A = 10$

用 Beta 分布做贝叶斯后验（假设均匀先验 Beta(1,1)）：

```math
p_A \sim \text{Beta}(1+10, 1+0) = \text{Beta}(11,1)
```

95% 置信区间约：

```math
[0.74, 1.0]
```

###### 店 B

样本数 $n_B = 200$

好评数 $k_B = 190$

后验：

```math
p_B \sim \text{Beta}(1+190,1+10) = \text{Beta}(191,11)
```

95% 置信区间约：

```math
[0.91, 0.98]
```

##### 分析

店 A 的好评率虽然是 100%，但区间很宽 (0.74–1.0)，存在较大不确定性。

店 B 好评率略低 (95%)，但样本大，估计稳定 (0.91–0.98)。

结论：如果你追求 更可靠的选择，应该选 店 B。
如果你只看点估计且忽略不确定性，店 A 看起来最好，但风险较大。

##### 方法 2：贝叶斯期望值

用 Beta 后验均值：

```math
\text{E}[p] = \frac{\alpha}{\alpha+\beta}
```

店 A：

```math
\frac{11}{11+1} = 0.9167
```

店 B：

```math
\frac{191}{191+11} = 0.9458
```

这个指标下，店 B 也更高。

✅ 结论总结

样本少的“完美评分”不可靠。

样本大、稍低评分的店铺更稳妥。

实际选择应偏向 店 B。
