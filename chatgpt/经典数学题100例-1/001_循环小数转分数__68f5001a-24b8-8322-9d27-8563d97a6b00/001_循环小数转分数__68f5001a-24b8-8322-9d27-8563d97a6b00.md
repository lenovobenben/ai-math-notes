# 循环小数转分数

- Conversation ID: `68f5001a-24b8-8322-9d27-8563d97a6b00`

## Question

一个数论问题。假如我给出一个循环小数。比如 0.ab。 a是前置的数字（可以是0或者没有），b是其循环节。比如 a=123,b=78，则数字为0.123787878...。现在我已知a和b。如何求他是哪两个互质的整数之商？

## Answer

好——这是经典的循环小数转分数问题。给定非循环部分 $a$（可能为空或为“0”）和循环节 $b$，令

$m$ = $a$的位数（若 $a$ 为空或“0”，视情况取 $m=0$ 或 $m$ 为你给出的位数），

$k$ = $b$ 的位数。

把 $a$ 与 $b$ 连接成整数记作 $\overline{ab}$（例如 $a=123,b=78$ 时 $\overline{ab}=12378$），把 $a$ 单独当作整数记作 $\overline{a}$（若为空则为 0）。

结论与公式：

```math
0.\,a\overline{b} \;=\; \frac{\overline{ab}-\overline{a}}{10^{m}\,(10^{k}-1)}
```

最后把分子分母除以它们的最大公约数（gcd）得到最简（互质）分数。

【证明（常用代数法）】设 $x=0.a\overline{b}$。
乘以 $10^{m+k}$ 得 $10^{m+k}x=\overline{ab}.\overline{b}\overline{b}\dots$（小数部分仍为循环），
乘以 $10^{m}$ 得 $10^{m}x=\overline{a}.\overline{b}\overline{b}\dots$。
两式相减消去循环小数得：

```math
10^{m+k}x-10^{m}x=(10^{m+k}-10^{m})x=\overline{ab}-\overline{a},
```

所以

```math
x=\frac{\overline{ab}-\overline{a}}{10^{m}(10^{k}-1)}.
```

【示例】 $a=123,\; b=78$：

$m=3,\; k=2$；$\overline{ab}=12378,\; \overline{a}=123$。

分数为 $(12378-123)/(10^{3}(10^{2}-1))=12255/99000$。
求最大公约数 $\gcd(12255,99000)=15$，约去得最简分数 $\displaystyle\frac{817}{6600}$。
因此 $0.1237878\ldots=\dfrac{817}{6600}$。

【特殊/注意事项】

若 $a$ 为空（纯循环小数 0.$\overline{b}$），则公式退化为 $b/(10^{k}-1)$。例如 $0.\overline{78}=78/99$（可约）。

$a$ 或 $b$ 若包含前导零，请按实际位数 $m,k$ 计算（例如 $a=0$ 与 $a$ 为空的区别在 $m$ 上）。

最后一步必须把分子和分母同时除以 gcd，得到互质的两个整数。

需要的话我可以帮你把任意一对 $a,b$ 计算成最简分数——直接把 $a$ 和 $b$ 发来就行。

## Question

```math
假如是a=空（没有a） ，b=123456789 呢
```

## Answer

纯循环小数 $0.\overline{123456789}$ 的分数写法是

```math
\frac{b}{10^{k}-1}=\frac{123456789}{10^9-1}=\frac{123456789}{999\,999\,999}.
```

因为 $1+2+\dots+9=45$，分子能被 $9$ 整除；分母 $999\,999\,999$ 也能被 $9$ 整除。两者的最大公约数为 $9$，约分得最简分数

```math
\boxed{\dfrac{13717421}{111111111}}.
```
