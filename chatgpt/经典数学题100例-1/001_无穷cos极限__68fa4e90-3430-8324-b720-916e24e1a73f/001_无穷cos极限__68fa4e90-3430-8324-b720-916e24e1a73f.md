# 无穷cos极限

- Conversation ID: `68fa4e90-3430-8324-b720-916e24e1a73f`

## Question

无穷的cos，它等于几？
cos(cos(cos(cos(cos(...cos(x))))))=?

## Answer

无限嵌套的 $\cos$（即把 $\cos$ 不断作用下去）

```math
x_{n+1}=\cos(x_n),\qquad x_0=x
```

对任意实数 $x$ 都收敛到唯一的定点 $y$ ，满足

```math
y=\cos(y).
```

证明要点（简短）：

$\cos:\mathbb{R}\to[-1,1]$，因此第二次及以后迭代都落在 $[-1,1]$。

$\cos([-1,1])=[\cos1,1]\approx[0.5403,1]$。在该区间上，导数的绝对值有上界：$|\cos'(t)|=|\!-\sin t|\le\sin1\approx0.84147<1$，因此 $\cos$ 在这个闭区间上是压缩映射。

由压缩映射原理（Banach）可得存在且唯一的定点，且从任意初值迭代都会收敛到该定点。

数值解（称为 Dottie number）：

```math
\boxed{y\approx 0.7390851332151607.}
```

所以无穷嵌套的 $\cos$ 的极限就是这个常数：约 $0.7390851332151607$。
