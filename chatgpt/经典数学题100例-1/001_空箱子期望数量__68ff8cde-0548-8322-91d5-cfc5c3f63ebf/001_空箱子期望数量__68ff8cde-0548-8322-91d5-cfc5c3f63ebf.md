# 空箱子期望数量

- Conversation ID: `68ff8cde-0548-8322-91d5-cfc5c3f63ebf`

## Question

100 个球随机放入 100 个箱子中，求最终空箱子的数量（期望）。

## Answer

把问题用指标随机变量和期望线性化一下，算法很直接。

设对第 $j$ 个箱子定义指示变量

```math
I_j=\begin{cases}1,&\text{若箱子 }j\ \text{为空},\\0,&\text{否则}.\end{cases}
```

则每个球独立均匀放入 100 个箱子中的一个，箱子 $j$ 为空的概率为

```math
\Pr(I_j=1)=\Big(1-\frac{1}{100}\Big)^{100}.
```

空箱子总数为 $X=\sum_{j=1}^{100}I_j$，由线性期望

```math
\mathbb{E}[X]=\sum_{j=1}^{100}\mathbb{E}[I_j]=100\Big(1-\frac{1}{100}\Big)^{100}.
```

数值计算得

```math
\mathbb{E}[X]=100\Big(1-\frac{1}{100}\Big)^{100}\approx 36.603234\,
```

（约为 $36.60$）。另外，对于一般的 $n$ 球 $n$ 箱，当 $n\to\infty$ 极限接近 $n/e$。
