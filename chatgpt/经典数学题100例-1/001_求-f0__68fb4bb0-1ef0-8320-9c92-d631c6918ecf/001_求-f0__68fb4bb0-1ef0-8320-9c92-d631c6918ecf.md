# 求 f(0)

- Conversation ID: `68fb4bb0-1ef0-8320-9c92-d631c6918ecf`

## Question

```math
f(f(x)) = x^2 - x + 1
```

求 \( f(0) \)？

## Answer

我们一步步仔细推理。

##### 已知：

```math
f(f(x)) = x^2 - x + 1
```

我们要求 $f(0)$。

##### 第一步：令 $x = 0$

```math
f(f(0)) = 0^2 - 0 + 1 = 1
```

因此：

```math
f(f(0)) = 1 \quad \Rightarrow \quad f(a) = 1 \text{ 其中 } a = f(0)
```

##### 第二步：令 $x = a$

```math
f(f(a)) = a^2 - a + 1
```

但从上一步我们知道 $f(a) = 1$，所以：

```math
f(f(a)) = f(1)
```

于是：

```math
f(1) = a^2 - a + 1
```

##### 第三步：再令 $x = 1$

```math
f(f(1)) = 1^2 - 1 + 1 = 1
```

代入刚刚求的 $f(1) = a^2 - a + 1$，得到：

```math
f(f(1)) = f(a^2 - a + 1) = 1
```

于是我们得到：

```math
f(a^2 - a + 1) = 1
```

##### 观察到：

我们已经知道 $f(a) = 1$。

若函数 $f$ 是单射（injective），那么：

```math
f(a^2 - a + 1) = f(a) \Rightarrow a^2 - a + 1 = a
```

于是：

```math
a^2 - 2a + 1 = 0 \Rightarrow (a - 1)^2 = 0
```

```math
\boxed{a = 1}
```

##### 验证：

若 $f(0) = 1$，则：

```math
f(f(0)) = f(1)
```

又已知 $f(f(0)) = 1$，所以 $f(1) = 1$，
完全符合 $f(f(x)) = x^2 - x + 1$ 对 $x=0,1$ 的情况。

✅ 最终答案：

```math
\boxed{f(0) = 1}
```
