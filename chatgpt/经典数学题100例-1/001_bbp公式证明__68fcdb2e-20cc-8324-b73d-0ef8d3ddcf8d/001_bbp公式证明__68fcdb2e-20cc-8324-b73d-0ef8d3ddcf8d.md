# BBP公式证明

- Conversation ID: `68fcdb2e-20cc-8324-b73d-0ef8d3ddcf8d`

## Question

{\displaystyle \pi =\sum _{k=0}^{\infty }{\frac {1}{16^{k}}}\left({\frac {4}{8k+1}}-{\frac {2}{8k+4}}-{\frac {1}{8k+5}}-{\frac {1}{8k+6}}\right)}

这是著名的BBP圆周率公式，请证明它。

## Answer

好的——下面给出一个完整且常见的证明，思路是把级数写成一个定积分（利用 $1/(8k+m)=\int_0^1 t^{8k+m-1}\,dt$），交换求和与积分，再用有理函数的部分分式分解计算该积分，最后得到 $\pi$。

定理（BBP 公式）.

```math
\pi=\sum_{k=0}^{\infty}\frac{1}{16^{k}}\!\left(\frac{4}{8k+1}-\frac{2}{8k+4}-\frac{1}{8k+5}-\frac{1}{8k+6}\right).
```

证明：

用积分表述分母项。对任意正整数 $m$,

```math
\frac{1}{8k+m}=\int_{0}^{1} t^{8k+m-1}\,dt.
```

因此把级数项写成积分形式（交换求和与积分将在下文说明）：

```math
\begin{aligned}
S &:=\sum_{k=0}^{\infty}\frac{1}{16^{k}}\Big(\frac{4}{8k+1}-\frac{2}{8k+4}-\frac{1}{8k+5}-\frac{1}{8k+6}\Big)\\
&=\sum_{k=0}^{\infty}\frac{1}{16^{k}}\int_{0}^{1}\big(4t^{8k}-2t^{8k+3}-t^{8k+4}-t^{8k+5}\big)\,dt\\
&=\int_{0}^{1}\sum_{k=0}^{\infty}\Big(\frac{t^{8}}{16}\Big)^{k}\big(4-2t^{3}-t^{4}-t^{5}\big)\,dt,
\end{aligned}
```

这里交换求和与积分是合法的，因为对 $t\in[0,1]$，幂级数 $\sum (t^{8}/16)^k$ 在闭区间上一致收敛（绝对收敛），从而可交换。

求和出闭式并化简。对内项求和得几何级数：

```math
\sum_{k=0}^{\infty}\Big(\frac{t^{8}}{16}\Big)^{k}=\frac{1}{1-t^{8}/16}=\frac{16}{16-t^{8}}.
```

于是

```math
S=\int_{0}^{1}\frac{4-2t^{3}-t^{4}-t^{5}}{1-t^{8}/16}\,dt
=\int_{0}^{1}\frac{64-32t^{3}-16t^{4}-16t^{5}}{16-t^{8}}\,dt.
```

对有理函数做部分分式分解。对被积函数做代数分解（可直接计算或用计算代数工具辅助）可得：

```math
\frac{64-32t^{3}-16t^{4}-16t^{5}}{16-t^{8}}
=\frac{4t}{t^{2}-2}-\frac{4(t-2)}{t^{2}-2t+2}.
```

（这个分解可以通过标准的部分分式或多项式整除得到；读者可自行检验两边相等。）

计算定积分。由上式

```math
S=\int_0^1\frac{4t}{t^{2}-2}\,dt-\int_0^1\frac{4(t-2)}{t^{2}-2t+2}\,dt.
```

分别求不定积分：

```math
\int \frac{4t}{t^{2}-2}\,dt=2\ln|t^{2}-2|+C,
```

```math
\int \frac{4(t-2)}{t^{2}-2t+2}\,dt
=\;2\ln\big((t-1)^{2}+1\big)-4\arctan(t-1)+C.
```

（第二个可通过令 $u=t-1$ 并分拆 $\dfrac{u-1}{u^{2}+1}$ 得到对数项与反正切项。）

于是一个原函数为

```math
F(t)=2\ln|t^{2}-2|-2\ln\big((t-1)^{2}+1\big)+4\arctan(t-1).
```

计算定积分 $S=F(1)-F(0)$。

代入：

$t=1$ 时： $t^{2}-2=-1,\ (t-1)^{2}+1=1,\ \arctan(0)=0$，所以 $F(1)=0$.

$t=0$ 时： $t^{2}-2=-2,\ (t-1)^{2}+1=2,\ \arctan(-1)=-\pi/4$，所以

```math
F(0)=2\ln 2-2\ln 2+4\big(-\tfrac{\pi}{4}\big)=-\pi.
```

因此

```math
S=F(1)-F(0)=0-(-\pi)=\pi.
```

这正是我们要证明的等式：$S=\pi$。

```math
\square
```

备注：上面关键步骤是（1）把每一项用 $\int_0^1 t^{8k+m-1}\,dt$ 表示并交换和与积分，（2）把和求出变为有理函数的定积分，（3）通过部分分式和标准初等不定积分（对数与反正切）计算出值 $\pi$。这就是 BBP 公式的经典解析证明。

## Question

这个公式计算圆周率为什么可以直接计算第N位，而不用计算前面的数字？

## Answer

非常好的问题 👏——这其实正是 BBP（Bailey–Borwein–Plouffe）公式 最令人震惊和著名的地方：
它允许在 不计算前面所有数字的情况下，直接计算圆周率的第 N 位（在十六进制或二进制下）。
下面我会分三层来解释它为什么能做到这一点。

#### 一、BBP 公式的特殊结构

BBP 公式如下：

```math
\pi = \sum_{k=0}^{\infty}\frac{1}{16^k}\left(\frac{4}{8k+1}-\frac{2}{8k+4}-\frac{1}{8k+5}-\frac{1}{8k+6}\right)
```

这个公式的关键特征是：

每一项都有一个 明确的权重 $16^{-k}$，即 $1/16^k$；

$16 = 2^4$，所以它在 二进制和十六进制下极其自然；

每项的分母 $8k+m$ 都是奇数整数，使得模运算下可逆。

这意味着我们可以写：

```math
\pi = \sum_{k=0}^{\infty} \frac{a_k}{16^k}, \quad a_k \in \mathbb{Q}.
```

在十六进制表示下：

```math
\pi = (d_0.d_1d_2d_3\ldots)_{16},
```

其中 $d_n$ 就是第 $n$ 位十六进制数字。

#### 二、计算第 N 位的核心思想：模算术 + 分拆求和

如果想要计算第 $n$ 位十六进制数字，可以考虑：

```math
16^n \pi = \sum_{k=0}^{\infty} \frac{16^n}{16^k} \left(\frac{4}{8k+1} - \cdots \right)
= \sum_{k=0}^{n-1} \frac{16^{n-k}}{8k+m} + \sum_{k=n}^{\infty} \frac{1}{16^{k-n}(8k+m)}.
```

然后我们只关心 小数部分（fractional part）：

```math
\{16^n \pi\} = 16^n \pi - \lfloor 16^n \pi \rfloor.
```

这个小数部分的前几位，就是 $\pi$ 从第 $n+1$ 位开始的十六进制数字。

##### 1️⃣ 前半部分：有限项求和（k < n）

对于 $k < n$，项 $\frac{16^{n-k}}{8k+m}$ 是整数加上一个小数。
我们可以在模 1 意义下计算小数部分：

```math
\left\{\frac{16^{n-k}}{8k+m}\right\} = \frac{16^{n-k} \bmod (8k+m)}{8k+m}.
```

这是有限精度整数运算，不需要大数乘法，也不涉及浮点误差。

因为只取模 $8k+m$，所以即使 $n$ 很大，也可以通过快速模幂算法（modular exponentiation）高效算出。

##### 2️⃣ 后半部分：无限尾项（k \ge n）

当 $k \ge n$ 时，

```math
\frac{1}{16^{k-n}(8k+m)}
```

已经非常小，可以直接用高精度浮点或有限位计算，几乎不会影响前几位数字。

因此我们可以近似求和这部分，用很少的项达到足够精度。

#### 三、结果：无需前面位数

总结来说：

传统算法：需要计算所有前面的数字（因为进位影响会一直传播）；

BBP 公式：每项的贡献是以 $1/16^k$ 缩放的，可以独立定位；

利用模算术，我们能直接算出第 $n$ 位开始的小数部分；

所以可以直接计算第 $n$ 位十六进制或二进制数字，而不需要前面数字。

#### 四、示意总结

| 步骤 | 作用 | 说明 |
| --- | --- | --- |
| ① | 把 π 表示为 $\sum a_k / 16^k$ | 基于 BBP 公式 |
| ② | 乘上 $16^n$ 并取小数部分 | 定位第 n 位起的数字 |
| ③ | 对前半部分做模运算 | 不需要高精度 |
| ④ | 对后半部分做有限求和 | 收敛很快 |
| ✅ | 结果 | 得到 π 的第 n 位十六进制数 |

#### 五、延伸：二进制与其他常数

BBP 公式的结构是基于 $1/b^k$ 的形式，其中 $b$ 是进制基；

这种结构在 二进制（b=2）或 十六进制（b=16）下特别容易定位；

之后人们发现 很多常数（如 log(2)、π²、ζ(3) 等） 都存在类似的 BBP 型公式；

这些公式的发现，推动了数值计算与数学常数理论的研究。

✅ 总结一句话：

BBP 公式之所以能直接计算 π 的第 N 位，是因为它的结构是一个以 16 为底的幂级数。利用模算术，我们可以独立计算第 N 项的贡献，从而直接定位到任意位的小数，而无需依赖前面的结果。
