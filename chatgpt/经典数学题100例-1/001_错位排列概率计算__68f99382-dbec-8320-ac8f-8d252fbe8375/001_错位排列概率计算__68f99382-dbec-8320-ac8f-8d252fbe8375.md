# 错位排列概率计算

- Conversation ID: `68f99382-dbec-8320-ac8f-8d252fbe8375`

## Question

n个人戴着n顶不同的帽子参加聚会，散场时每人随机拿走一顶帽子，最终没有人戴上正确帽子的概率是多少？

## Answer

这是著名的 错位排列（derangement） 问题。

设 $D_n$ 表示把 $n$ 个不同物品放回 $n$ 个不同位置且没有任何一个放回原位的排列数（即“完全错位数”或子阶乘，记作 $!n$ 或 $D_n$）。问题要的概率为

```math
P_n=\frac{D_n}{n!}.
```

有几种常用表达：

容斥公式（闭式）

```math
D_n = n!\sum_{k=0}^{n}\frac{(-1)^k}{k!}
\quad\Rightarrow\quad
P_n=\sum_{k=0}^{n}\frac{(-1)^k}{k!}.
```

递推式

```math
D_0=1,\; D_1=0,\qquad D_n=(n-1)\big(D_{n-1}+D_{n-2}\big)\ (n\ge2).
```

给出几个小 $n$ 的具体例子：

$n=1$：$P_1=0$.

$n=2$：$P_2=\tfrac{1}{2}=0.5$.

$n=3$：$P_3=\tfrac{2}{6}=\tfrac{1}{3}\approx0.3333$.

$n=4$：$P_4=\tfrac{9}{24}=0.375$.

$n=5$：$P_5=\tfrac{44}{120}\approx0.3666667$.

$n=6$：$P_6=\tfrac{265}{720}\approx0.3680556$.

当 $n$ 很大时，$P_n$ 收敛到

```math
\lim_{n\to\infty}P_n=\frac{1}{e}\approx0.3678794412.
```

因此答案：概率为 $\displaystyle P_n=\frac{D_n}{n!}=\sum_{k=0}^n\frac{(-1)^k}{k!}$，当 $n$ 很大时近似为 $1/e$。需要我帮你算某个具体 $n$ 的值吗？ 😊

是等价的。证明很短——把你的式子化一化：

```math
1-\sum_{i=1}^{n}\frac{(-1)^{\,i-1}}{i!}
=1-\sum_{i=1}^{n}\left(-\frac{(-1)^i}{i!}\right)
=1+\sum_{i=1}^{n}\frac{(-1)^i}{i!}
=\sum_{i=0}^{n}\frac{(-1)^i}{i!},
```

而我之前给出的概率就是 $\displaystyle P_n=\sum_{k=0}^{n}\frac{(-1)^k}{k!}$。两者完全相同。

举个小例子（$n=4$）验证：
你的式子： $1-\big(\frac{(-1)^0}{1!}+\frac{(-1)^1}{2!}+\frac{(-1)^2}{3!}+\frac{(-1)^3}{4!}\big)=1-(1-\tfrac12+\tfrac16-\tfrac1{24})=\tfrac{9}{24}$.
