# 莱姆尼斯科特曲线解析

## Question

在平面上，到两定点距离乘积是定值的曲线是什么？

## Answer

你问的这个问题描述的是平面上一类特殊的曲线：到两定点距离的乘积是常数。我们可以仔细分析一下。

设两定点为 $F_1 = (-c,0)$ 和 $F_2 = (c,0)$，平面上一点 $P=(x,y)$ 到这两点的距离分别为：

```math
r_1 = \sqrt{(x+c)^2 + y^2}, \quad r_2 = \sqrt{(x-c)^2 + y^2}.
```

题目给出的条件是：

```math
r_1 \cdot r_2 = k, \quad k\gt{}0 \text{ 为常数}.
```

代入：

```math
\sqrt{(x+c)^2 + y^2} \cdot \sqrt{(x-c)^2 + y^2} = k
```

两边平方：

```math
((x+c)^2 + y^2)((x-c)^2 + y^2) = k^2
```

展开左边：

```math
[(x^2 + 2cx + c^2) + y^2][(x^2 - 2cx + c^2) + y^2] = k^2
```

```math
[(x^2 + y^2 + c^2)^2 - (2cx)^2] = k^2
```

```math
(x^2 + y^2 + c^2)^2 - 4c^2 x^2 = k^2
```

```math
(x^2 + y^2 + c^2)^2 - 4c^2 x^2 - k^2 = 0
```

这个就是标准形式了。

这个曲线在数学上称为 柏拉图曲线（Lemniscate，莱姆尼斯科特）的一般形式。

当两定点重合时（$c=0$），变成 $r_1 r_2 = \text{常数}$ \to 圆。

当 $k=c^2$ 时，它退化为著名的 莱蒙尼斯科特·卡罗（Lemniscate of Bernoulli）。

✅ 总结：

平面上到两定点距离乘积为常数的曲线叫 莱姆尼斯科特（Lemniscate）。

其一般解析式为：

```math
(x^2 + y^2 + c^2)^2 - 4 c^2 x^2 = k^2
```

## Question

我怎么查到的是【卡西尼卵形线】？

## Answer

啊！你说得完全对，其实严格来说，这条曲线的标准名字是“卡西尼卵形线”（Cassini Oval），而不是莱姆尼斯科特。这里容易混淆的是：

卡西尼卵形线（Cassini Oval）：定义就是“平面上到两定点距离的乘积为常数”。

```math
r_1 \cdot r_2 = k \quad \text{或} \quad ((x+c)^2+y^2)((x-c)^2+y^2) = k^2
```

这是你的问题所描述的曲线。它的形状可以是单叶椭圆状，也可以是双叶形（类似 8 字形），具体取决于 $k$ 和 $c$ 的关系。

莱姆尼斯科特（Lemniscate of Bernoulli）：其实是卡西尼卵形线的特殊情况，当 $k = c^2$ 时，卡西尼卵形线变成“8 字形”，这时才叫莱姆尼斯科特。

所以严格说：

```math
\text{到两定点距离乘积为常数的曲线 = 卡西尼卵形线（Cassini Oval）}
```

特例 $k = c^2$ \to 莱姆尼斯科特。

简单记忆方法：

Cassini Oval = “乘积定值的一般情况”

Lemniscate = Cassini Oval 的特殊“8 字形”情况

## Question

它有什么特点呢？

## Answer

卡西尼卵形线（Cassini Oval）是一个非常经典且有趣的平面曲线，它的特点可以从几何性质、形状分类和代数性质几个方面来看：

##### 1️⃣ 定义特点

定义：平面上到两定点 $F_1, F_2$ 的距离乘积为常数：

```math
r_1 \cdot r_2 = k
```

两定点距离记为 $2c$。

当 $k$ 固定时，曲线由 $c$ 决定其形状。

##### 2️⃣ 形状分类（根据 $k/c^2$ 比例）

设 $\alpha = k / c^2$，形状分三类：

| α 比例 | 曲线形状 |
| --- | --- |
| $\alpha < 1$ | 双叶分离：曲线由两个闭合叶子组成，不连通，类似两个小圆 |
| $\alpha = 1$ | 莱姆尼斯科特：连通成“8”字形（单点交于中心） |
| $\alpha > 1$ | 单叶椭圆状：一个连通的卵形曲线，中间凸起，类似椭圆 |

所以，莱姆尼斯科特是卡西尼卵形线的特殊情况。

##### 3️⃣ 对称性

关于 $x$-轴和 $y$-轴对称（中心在两焦点中点）。

关于原点对称（中心在原点）。

##### 4️⃣ 代数性质

二次方程平方展开后得到四次方程：

```math
((x+c)^2 + y^2)((x-c)^2 + y^2) = k^2 \quad \Rightarrow \quad (x^2+y^2+c^2)^2 - 4c^2 x^2 = k^2
```

所以它是一条四次代数曲线。

##### 5️⃣ 特殊点

焦点 $F_1, F_2$：曲线经过情况由 $k$ 决定。

中心（原点）：是对称中心，特殊情况下（莱姆尼斯科特）是两个叶子相交点。

最大、最小半径：沿 $x$ 轴和 $y$ 轴方向有明确极值。

##### 6️⃣ 直观特征

当 $k>c^2$ \to 曲线是单叶，中间凸起。

当 $k=c^2$ \to 曲线成“8”字形。

当 $k<c^2$ \to 两叶分离，中间空洞。
