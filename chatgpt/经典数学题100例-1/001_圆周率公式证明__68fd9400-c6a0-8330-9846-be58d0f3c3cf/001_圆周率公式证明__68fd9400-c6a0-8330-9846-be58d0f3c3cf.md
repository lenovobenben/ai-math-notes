# 圆周率公式证明

- Conversation ID: `68fd9400-c6a0-8330-9846-be58d0f3c3cf`

## Question

圆周率公式证明：

{\displaystyle {\frac {2}{\pi }}={\frac {\sqrt {2}}{2}}\cdot {\frac {\sqrt {2+{\sqrt {2}}}}{2}}\cdot {\frac {\sqrt {2+{\sqrt {2+{\sqrt {2}}}}}}{2}}\cdots }

## Answer

这是著名的 Viète（维耶特）公式。我给出一个简明且严格的证明思路：用正弦的二倍角恒等式反复展开并用小角极限 $\lim_{t\to0}\dfrac{\sin t}{t}=1$，并用半角根式表示余弦，得到题中嵌套根号的形式。

##### 1. 从二倍角公式出发

对任意 $x$ 有

```math
\sin x = 2\sin\frac{x}{2}\cos\frac{x}{2}.
```

对右边再把 $\sin\frac{x}{2}$ 展开，反复应用 $n$ 次，可得到

```math
\sin x = 2^n \sin\!\Big(\frac{x}{2^n}\Big)\prod_{k=1}^{n}\cos\!\Big(\frac{x}{2^k}\Big).
```

两边同时除以 $2^n \sin\!\big(\frac{x}{2^n}\big)$，得

```math
\prod_{k=1}^{n}\cos\!\Big(\frac{x}{2^k}\Big)
=\frac{\sin x}{2^n \sin\!\big(\frac{x}{2^n}\big)}.
```

##### 2. 取特殊角 $x=\dfrac{\pi}{2}$

令 $x=\dfrac{\pi}{2}$，注意 $\sin\frac{\pi}{2}=1$，得到

```math
\prod_{k=1}^{n}\cos\!\Big(\frac{\pi}{2^{k+1}}\Big)
=\frac{1}{2^n \sin\!\big(\frac{\pi}{2^{n+1}}\big)}.
```

（注意下标：当 $k=1$ 时因子是 $\cos(\pi/4)=\tfrac{\sqrt2}{2}$，与题中第一个因子一致。）

##### 3. 令 $n\to\infty$ 并用小角极限

当 $n\to\infty$ 时，角 $\dfrac{\pi}{2^{n+1}}\to0$。利用极限

```math
\lim_{t\to0}\frac{\sin t}{t}=1
```

可得

```math
\lim_{n\to\infty}2^n \sin\!\Big(\frac{\pi}{2^{n+1}}\Big)
=\lim_{n\to\infty}2^n\cdot\frac{\pi}{2^{n+1}}\cdot\frac{\sin(\pi/2^{n+1})}{\pi/2^{n+1}}
=\frac{\pi}{2}\cdot 1=\frac{\pi}{2}.
```

因此取极限得

```math
\prod_{k=1}^{\infty}\cos\!\Big(\frac{\pi}{2^{k+1}}\Big)
=\frac{1}{\pi/2}=\frac{2}{\pi}.
```

##### 4. 将余弦写成嵌套根号形式

半角公式

```math
\cos\frac{\theta}{2}=\sqrt{\frac{1+\cos\theta}{2}}
```

给出余弦的嵌套根号表达。以 $\theta=\frac{\pi}{2}$ 开始：

```math
\cos\frac{\pi}{4}=\frac{\sqrt2}{2},
\qquad
\cos\frac{\pi}{8}=\sqrt{\frac{1+\cos\frac{\pi}{4}}{2}}=\frac{\sqrt{2+\sqrt2}}{2},
```

依此类推，便得到题中因子序列

```math
\frac{\sqrt{2}}{2},\quad \frac{\sqrt{2+\sqrt2}}{2},\quad \frac{\sqrt{2+\sqrt{2+\sqrt2}}}{2},\ \dots
```

所以把第 $k$ 项写成 $\cos(\pi/2^{k+1})$ 的嵌套根号形式，连乘的极限就是 $\dfrac{2}{\pi}$。

##### 结论

因此

```math
\boxed{\displaystyle \frac{2}{\pi}
=\frac{\sqrt{2}}{2}\cdot\frac{\sqrt{2+\sqrt{2}}}{2}\cdot\frac{\sqrt{2+\sqrt{2+\sqrt{2}}}}{2}\cdots}
```

这就是所求的 Viète 公式证明。
