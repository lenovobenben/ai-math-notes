# 积分可积性与值

- Conversation ID: `68fc4efa-8ec0-832b-a52b-150c5621f4b9`

## Question

\int_{0}^{\frac{\pi}{2}} \sqrt[\infty]{\sin^{\infty} x + \cos^{\infty} x}  dx

是可积的吗？值是多少？

## Answer

我们来仔细分析这个积分：

```math
\int_{0}^{\frac{\pi}{2}} \sqrt[\infty]{\sin^{\infty} x + \cos^{\infty} x} \, dx.
```

##### 第一步：理解 $\sqrt[\infty]{\cdot}$ 和 $\cdot^\infty$

我们考虑极限：

```math
f(x) = \lim_{n \to \infty} \bigl( \sin^n x + \cos^n x \bigr)^{1/n}.
```

这是一个经典的“极大值型极限”，因为对于非负数 $a,b \ge 0$：

```math
\lim_{n\to\infty} (a^n + b^n)^{1/n} = \max(a,b).
```

证明直观：

如果 $a > b$，那么 $b^n / a^n \to 0$，所以

```math
(a^n + b^n)^{1/n} = a (1 + (b/a)^n)^{1/n} \to a
```

同理，如果 $b > a$，结果是 $b$。

如果 $a = b$，结果就是 $a = b$。

##### 第二步：应用到 $\sin^n x + \cos^n x$

于是

```math
\lim_{n\to\infty} (\sin^n x + \cos^n x)^{1/n} = \max(\sin x, \cos x).
```

所以积分变为：

```math
\int_0^{\pi/2} \max(\sin x, \cos x) \, dx.
```

##### 第三步：分段积分

我们知道在 $[0, \pi/2]$ 上：

$\sin x = \cos x$ 当且仅当 $x = \pi/4$

在 $0 \le x \le \pi/4$，$\cos x \ge \sin x$

在 $\pi/4 \le x \le \pi/2$，$\sin x \ge \cos x$

所以：

```math
\int_0^{\pi/2} \max(\sin x, \cos x)\,dx = \int_0^{\pi/4} \cos x \, dx + \int_{\pi/4}^{\pi/2} \sin x \, dx.
```

##### 第四步：计算积分

```math
\int_0^{\pi/4} \cos x \, dx = \sin x \Big|_0^{\pi/4} = \sin(\pi/4) - \sin(0) = \frac{\sqrt{2}}{2} - 0 = \frac{\sqrt{2}}{2}.
```

```math
\int_{\pi/4}^{\pi/2} \sin x \, dx = -\cos x \Big|_{\pi/4}^{\pi/2} = (-\cos(\pi/2)) - (-\cos(\pi/4)) = 0 - (-\frac{\sqrt{2}}{2}) = \frac{\sqrt{2}}{2}.
```

加起来：

```math
\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2} = \sqrt{2}.
```

##### ✅ 结论

积分是可积的。

积分值为：

```math
\boxed{\sqrt{2}}.
```

这个问题的关键是认识到 $\lim_{n\to\infty} (a^n + b^n)^{1/n} = \max(a,b)$，把原来的复杂极限问题变成了一个简单的分段积分。
