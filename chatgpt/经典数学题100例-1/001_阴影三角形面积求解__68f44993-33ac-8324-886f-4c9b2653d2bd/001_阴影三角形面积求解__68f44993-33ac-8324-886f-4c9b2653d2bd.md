# 阴影三角形面积求解

- Conversation ID: `68f44993-33ac-8324-886f-4c9b2653d2bd`

## Question

有一个矩形，从左上角逆时针四个顶点分别为ABCD。其内部有一个阴影三角形，其中一个顶点为A（与矩形共用）。另外两个顶点 E F 分别位于BC和CD上。
三角形ABE面积为2，三角形ECF面积为3，三角形FDA面积为4。
求阴影三角形AEF的面积。

## Answer

设矩形 $ABCD$ 的宽为 $w$、高为 $h$。取坐标

```math
A(0,0),\;B(w,0),\;C(w,h),\;D(0,h)
```

设 $E$ 在 $BC$ 上，令 $E(w,y)$；$F$ 在 $CD$ 上，令 $F(x,h)$。

由已知面积得：

(1)

```math
[ABE]=\tfrac12 w y=2\quad\Rightarrow\quad w y=4.
```

(2)

```math
[ECF]=\tfrac12 (h-y)(w-x)=3\quad\Rightarrow\quad (h-y)(w-x)=6.
```

(3)

```math
[FDA]=\tfrac12 x h=4\quad\Rightarrow\quad x h=8.
```

把(1)(3)代入(2)的展开式：

```math
(h-y)(w-x)=hw-hx-wy+xy=6.
```

用 $hx=8,\;wy=4$ 得

(4)

```math
hw-8-4+xy=6\quad\Rightarrow\quad hw+xy=18.
```

又由(1)(3)相乘得到

(5)

```math
(wy)(xh)=4\cdot8=32\quad\Rightarrow\quad (hw)(xy)=32.
```

设 $A=hw,\;B=xy$，由(4)(5)有

```math
A+B=18,\quad AB=32.
```

解得 $A=16,\;B=2$（另一根 $A=2$ 不合矩形尺寸），即 $hw=16,\;xy=2$。

阴影三角形 $AEF$ 面积为

```math
[AEF]=\tfrac12\big(hw-xy\big)=\tfrac12(16-2)=7.
```

答：阴影三角形 $AEF$ 的面积为 $\boxed{7}$。
