# cos(x+y) 方程分析

- Conversation ID: `68f50504-2208-8323-ad9f-463cd6bace4a`

## Question

cos(x+y)=x²+y²是否是椭圆？如果不是，为什么

## Answer

我们来仔细分析方程

```math
\cos(x+y) = x^2 + y^2
```

是否表示椭圆。

##### 一、先看椭圆的一般形式

平面上一条椭圆（或圆）的标准形式是：

```math
\frac{(x - x_0)^2}{a^2} + \frac{(y - y_0)^2}{b^2} = 1
```

或其一般形式：

```math
Ax^2 + Bxy + Cy^2 + Dx + Ey + F = 0
```

其中 $B^2 - 4AC < 0$。

这些形式都是二次曲线方程，即关于 $x, y$ 的二次多项式方程。

##### 二、对比我们的方程

```math
\cos(x+y) = x^2 + y^2
```

两边不是同类型的函数：

左边 $\cos(x+y)$ 是三角函数，不是多项式；

右边 $x^2 + y^2$ 是二次多项式。

因此，这个方程不是二次曲线方程，从形式上就不可能是椭圆。

##### 三、进一步几何直观分析

$\cos(x+y)$ 的取值范围是 $[-1, 1]$，
所以必须满足：

```math
x^2 + y^2 \in [-1, 1]
```

即：

```math
x^2 + y^2 \le 1
```

说明所有可能的解点都在单位圆内。

对于每个固定的 $x+y = c$，$\cos(c)$ 是常数，所以方程变成：

```math
x^2 + y^2 = \cos(c)
```

若 $\cos(c) > 0$，这是一个半径为 $\sqrt{\cos(c)}$ 的圆；
若 $\cos(c) < 0$，则无实数解；
若 $\cos(c) = 0$，则只有原点解。

因为 $\cos(x+y)$ 随 $x+y$ 变化周期振荡，这条曲线是若干条圆弧的周期性组合，呈波浪状分布，而非一个封闭的单一椭圆。

##### ✅结论

```math
\boxed{\cos(x+y) = x^2 + y^2 \text{ 不是椭圆。}}
```

原因：

它不是二次多项式方程（包含三角函数）；

几何上它表示的是一族由 $\cos(x+y)$ 控制的周期性圆弧曲线，而不是一个封闭的椭圆形。

## Question

但是它的图像和椭圆非常像

## Answer

你说得对——从远处或在原点附近看，这条曲线确实看起来很像椭圆。下面把“为什么会像”用数学方式说明，并解释什么时候、为什么会偏离椭圆。

##### 1) 在原点附近做泰勒展开（直观来源）

当 $x$ 和 $y$ 都比较小时，$x+y$ 也小，三角函数可以用泰勒展开近似：

```math
\cos(x+y)=1-\frac{(x+y)^2}{2}+\frac{(x+y)^4}{24}-\cdots
```

把它代入方程 $\cos(x+y)=x^2+y^2$，并忽略四阶及更高阶项，得到二次近似：

```math
1-\frac{(x+y)^2}{2}\approx x^2+y^2.
```

移项得一个二次多项式方程：

```math
x^2+y^2+\frac{(x+y)^2}{2}=1.
```

这是关于 $x,y$ 的二次曲线方程，因此在原点附近（高阶项可以忽略时）它就是一条椭圆（实际上是带中心在原点的椭圆）。

##### 2) 写成矩阵形式看主轴（求半轴和方向）

把上式展开：

```math
x^2+y^2+\frac{x^2+2xy+y^2}{2}=1
```

```math
\Rightarrow \; \tfrac32 x^2 + xy + \tfrac32 y^2 =1.
```

对应的二次形式矩阵为

```math
A=\begin{pmatrix}1.5 & 0.5\\[4pt]0.5 & 1.5\end{pmatrix}
```

其特征值为 $\lambda_1=2$ 与 $\lambda_2=1$（可验算或直接解特征方程），对应特征向量分别为 $[1,1]$（$\lambda=2$）和 $[1,-1]$（$\lambda=1$）。

对于方程 $v^T A v =1$，在特征方向上的半轴长度为 $1/\sqrt{\lambda}$。因此近似椭圆的半轴长度为

```math
a_1=\frac{1}{\sqrt{2}}\approx0.707,\qquad a_2=\frac{1}{\sqrt{1}}=1,
```

方向分别沿 $x=y$（较短半轴）和 $x=-y$（较长半轴）。这给出一个沿 $\pm45^\circ$ 方向的椭圆——这就是你看到“很像椭圆”的具体原因。

##### 3) 为什么不是全局椭圆（何时偏离）

上面的椭圆只是二阶近似（只保留了 $\cos$ 的常数项和二次项）。真实的 $\cos(x+y)$ 有高阶项（四次、六次……）以及周期性，所以当 $|x+y|$ 不足够小（比如离原点较远或者沿某些方向增长），高阶项会变得不可忽略，曲线会偏离那个椭圆。

此外 $\cos(x+y)$ 的取值被限制在 $[-1,1]$，因此解只存在于单位圆 $x^2+y^2\le1$ 内；而二次近似的椭圆可能完全落在单位圆内，但全局形状会因为 $\cos$ 的周期性（当 $x+y$ 增大到使 $\cos$ 变小或负）出现缺口、分支或额外的弧段——不是一个单一的光滑闭合二次曲线。

具体来说，当 $|x+y|$ 接近 $\pi/2$ 时 $\cos(x+y)$ 变为 0，方程仅在原点或无解；当 $|x+y|>\pi/2$（模$\pi$的位置）时 $\cos$ 负值，$x^2+y^2$ 无法等于负数，因此在这些区域没有实解。于是全局图形不是一个连续的椭圆，而是局部近似椭圆的若干弧/区域。

##### 结论（简短）

不是全局椭圆，因为原方程含有三角函数，非二次多项式；全局上它表现出周期性和高阶非线性，不满足二次曲线的一般形式。

但在原点附近，因为 $\cos(x+y)$ 的二阶泰勒近似，方程可以近似化为一个二次方程，得到一条椭圆 —— 这就是图像“看起来很像椭圆”的原因。偏离程度随 $|x+y|$ 的增大而加剧。
