# 变分法经典例题

- Conversation ID: `68ffcc54-ed10-8323-bc05-2acaddb9a0a4`

## Question

设光滑曲线y=f(x) 的起点为原点，长度为1，求当f(x) 绕 x 轴旋转产生旋转体的体积最大时，f(x)的表达式和最大体积。

## Answer

好题 — 我们把它作为变分问题来解。下面给出完整推导（要点与结果），并给出显式的隐式表达式与数值结果。

设定。 曲线用弧长参数 $s\in[0,1]$ 描述（因为曲线长度已知为 1），记点坐标为 $(x(s),y(s))$，起点为原点：$x(0)=0,\;y(0)=0$。设切线方向角为 $\theta(s)$，于是

```math
x'(s)=\cos\theta,\qquad y'(s)=\sin\theta,\qquad\text{且}\; s\ \text{即弧长}.
```

旋转体体积为

```math
V=\pi\int y^2\,dx=\pi\int_{0}^{1} y^2\,x'(s)\,ds=\pi\int_{0}^{1} y^2\cos\theta\,ds.
```

我们在弧长参数下用变分法对函数 $\theta(s)$（或等价地对 $y(s)$）变分，得约束方程 $y'=\sin\theta$。

变分与欧拉—拉格朗日。 构造广义拉格朗日量

```math
\mathcal L = y^2\cos\theta+\mu(s)\big(y'-\sin\theta\big),
```

对 $\mu,\;y,\;\theta$ 变分，消去 $\mu$ 后可得到（略去中间代数步骤）

```math
\boxed{\cos\theta = A\,y^{2}}
```

其中 $A$ 为常数（由全长 $1$ 的条件决定）。这个关系即为必要条件：切线余弦（即 $x'$）与 $y^2$ 成正比。

由此得到曲线的隐式表达式。 利用关系

```math
\frac{dx}{dy}=\frac{x'}{y'}=\frac{\cos\theta}{\sin\theta}=\frac{A y^2}{\sqrt{1-A^2 y^4}},
```

于是从 $y=0$ 积分得到

```math
\boxed{\,x(y)=\int_{0}^{y}\frac{A t^2}{\sqrt{1-A^2 t^4}}\,dt\,.}
```

这给出了 $y\mapsto x$ 的隐式表达（即 $y=f(x)$ 的反函数形式）。曲线的起点 $y(0)=0$ 对应参数 $\theta=\pi/2$（初始切线竖直），终点满足 $\theta=0$ 时 $y_{\max}^2=1/A$。

由长度条件确定常数 $A$。 弧长 $1$ 的条件

```math
1=\int_0^1 ds=\int_{y=0}^{y_{\max}}\frac{dy}{\sqrt{1-A^2 y^4}}
```

等价于（换参 $\cos\theta=A y^2$，并化简）

```math
1=\frac{1}{2\sqrt{A}}\int_0^{\pi/2}\cos^{-1/2}\theta\,d\theta.
```

令

```math
I=\int_{0}^{\pi/2}\cos^{-1/2}\theta\,d\theta,
```

则

```math
\sqrt{A}=\frac{I}{2},\qquad A=\Big(\frac{I}{2}\Big)^2.
```

利用伽玛函数表达可得精确形式（可选）：

```math
I=\frac{\sqrt\pi\,\Gamma(1/4)}{2\,\Gamma(3/4)},\quad
\Rightarrow\; A=\frac{\pi\,\Gamma(1/4)^2}{16\,\Gamma(3/4)^2}.
```

最大体积（显式表达与数值）。 代入上面的关系并计算体积（通过参数化或直接变换）可得

```math
V_{\max}=\frac{\pi}{2\,A^{3/2}}\int_0^{\pi/2}\cos^{3/2}\theta\,d\theta,
```

其中积分同样可用伽玛函数表示。把 $A$ 与这些积分代入并做数值计算，得到

```math
\boxed{A\approx 1.7187964339688735,}
\qquad
\boxed{V_{\max}\approx 0.6092621189785.}
```

总结（最终答案，供引用）。

满足体积极大化条件的曲线由关系 $\displaystyle \cos\theta=A\,y^2$ 给出（等价于对 $y(x)$ 的微分方程）

```math
\frac{dx}{dy}=\frac{A y^2}{\sqrt{1-A^2 y^4}},\qquad y(0)=0,
```

或隐式地

```math
x(y)=\int_0^y\frac{A t^2}{\sqrt{1-A^2 t^4}}\,dt,
```

其中常数 $A$ 由总弧长 $1$ 决定，数值 $A\approx1.71879643397$。

对应的最大体积为 $V_{\max}\approx 0.60926211898$（体积单位同曲线长度单位的立方）。

## Question

Vmax用Γ函数表示

## Answer

好的——把之前的积分表达化成伽玛函数，得到一个整洁的解析形式。

从推导我们有

```math
V_{\max}=\frac{\pi}{2A^{3/2}}\int_{0}^{\pi/2}\cos^{3/2}\theta\,d\theta,
```

且

```math
A=\Big(\frac{1}{2}\int_{0}^{\pi/2}\cos^{-1/2}\theta\,d\theta\Big)^2.
```

利用积分与贝塔／伽玛函数的常用公式

```math
\int_0^{\pi/2}\cos^{\alpha}\theta\,d\theta=\frac{\sqrt\pi\,\Gamma\!\big(\tfrac{\alpha+1}{2}\big)}{2\,\Gamma\!\big(\tfrac{\alpha}{2}+1\big)},
```

代入并化简（并用 $\Gamma(z+1)=z\Gamma(z)$）可得最终简洁形式：

```math
\boxed{\,V_{\max}=\frac{16}{3}\Big(\frac{\Gamma(3/4)}{\Gamma(1/4)}\Big)^{2}\,.}
```

数值近似为

```math
V_{\max}\approx 0.6092621080592848\ (\text{与之前数值一致}).
```
