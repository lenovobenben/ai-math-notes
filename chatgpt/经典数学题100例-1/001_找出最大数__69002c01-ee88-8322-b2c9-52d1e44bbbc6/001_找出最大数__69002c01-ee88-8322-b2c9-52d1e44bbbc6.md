# 找出最大数

- Conversation ID: `69002c01-ee88-8322-b2c9-52d1e44bbbc6`

## Question

1^100,2^99,3^98,......,99^2,100^1 这些数中的最大数是哪个？

## Answer

把第 $k$ 项记作 $a_k=k^{101-k}$（因为 $1^{100},2^{99},\dots,100^1$ 对应 $k=1,\dots,100$）。取对数考察

```math
g(k)=\ln a_k=(101-k)\ln k.
```

把 $k$ 当作连续变量求极值：

```math
g'(x)=-\ln x+\frac{101-x}{x}=0\iff x(\ln x+1)=101,
```

解得 $x\approx24.14026486$。因此在整数点附近的极大值应在 $k=24$ 或 $k=25$ 处。直接比较可得

```math
g(23)\lt{}g(24)\gt{}g(25),
```

数值上 $g(24)=(101-24)\ln24$ 最大。所以原序列中最大的数是

```math
\boxed{24^{77}}.
```
