# 正态分布与高斯积分

- Conversation ID: `68fe4d1d-9050-8322-9b13-5940ca98222e`

## Question

为什么二项式分布的极限是正态分布呢？是如何证明的？

## Answer

非常好的问题，这是概率论中的一个经典结论——二项分布在适当条件下的极限是正态分布，也称为 De Moivre–Laplace 定理，是中心极限定理（CLT, Central Limit Theorem）的一个特例。
我们来系统地理解并证明它。

#### 一、背景与结论

设随机变量

```math
X_n \sim \text{Binomial}(n, p)
```

则它的期望与方差为：

```math
\mathbb{E}[X_n] = np, \quad \text{Var}(X_n) = np(1-p)
```

当 $n \to \infty$ 时，若我们定义标准化变量：

```math
Z_n = \frac{X_n - np}{\sqrt{np(1-p)}}
```

则有：

```math
Z_n \xrightarrow{d} N(0, 1)
```

即：

```math
\frac{X_n - np}{\sqrt{np(1-p)}} \Longrightarrow \mathcal{N}(0, 1)
```

也就是说，二项分布在 $n$ 很大时，可以用正态分布近似。

#### 二、直观解释

二项分布是 $n$ 个独立伯努利随机变量的和：

```math
X_n = \sum_{i=1}^{n} Y_i, \quad Y_i \sim \text{Bernoulli}(p)
```

而中心极限定理（CLT）告诉我们：

独立同分布随机变量的标准化和在 $n \to \infty$ 时服从标准正态分布。

因此，二项分布是正态分布的自然极限：
二项分布 = 伯努利和 \to 应用中心极限定理 \to 正态分布。

#### 三、严格证明（De Moivre–Laplace 定理）

##### 1️⃣ 二项分布的概率质量函数（PMF）

```math
P(X_n = k) = \binom{n}{k} p^k (1-p)^{n-k}
```

令：

```math
k = np + x\sqrt{np(1-p)} \quad \text{(标准化偏离)}
```

##### 2️⃣ 使用 Stirling 公式近似阶乘：

```math
n! \approx \sqrt{2\pi n}\left(\frac{n}{e}\right)^n
```

带入组合数近似：

```math
\binom{n}{k} \approx \frac{1}{\sqrt{2\pi n p(1-p)}} \exp\left(-\frac{(k - np)^2}{2np(1-p)}\right)
```

于是：

```math
P(X_n = k) \approx \frac{1}{\sqrt{2\pi np(1-p)}} \exp\left(-\frac{(k - np)^2}{2np(1-p)}\right)
```

这正是正态分布 $N(np, np(1-p))$ 的密度形式。

#### 四、结论（De Moivre–Laplace 近似）

因此：

```math
P(a \le X_n \le b) \approx \int_{a}^{b} \frac{1}{\sqrt{2\pi np(1-p)}}
\exp\!\left(-\frac{(x - np)^2}{2np(1-p)}\right) dx
```

等价地：

```math
P\left(\frac{a - np}{\sqrt{np(1-p)}} \le Z_n \le \frac{b - np}{\sqrt{np(1-p)}}\right)
\approx \int_{\frac{a - np}{\sqrt{np(1-p)}}}^{\frac{b - np}{\sqrt{np(1-p)}}}
\frac{1}{\sqrt{2\pi}} e^{-t^2/2} dt
```

#### 五、与中心极限定理的关系

其实 De Moivre–Laplace 定理是中心极限定理的一个最早形式：

若 $X_1, X_2, \dots, X_n$ 独立同分布，$\mathbb{E}[X_i]=\mu, \text{Var}(X_i)=\sigma^2 < \infty$，
则

```math
\frac{\sum_{i=1}^n X_i - n\mu}{\sqrt{n\sigma^2}} \xrightarrow{d} N(0,1)
```

二项分布的情形只是 $X_i \sim \text{Bernoulli}(p)$ 的特例。

#### 六、数值例子

例如 $n=1000, p=0.5$，则：

```math
X \sim \text{Binomial}(1000, 0.5), \quad \mu=500, \sigma=\sqrt{250}=15.81
```

则

```math
P(X=520) \approx \frac{1}{\sqrt{2\pi}\cdot15.81} e^{-(20)^2/(2\times 250)} \approx 0.014
```

与正态近似几乎一致。

✅ 总结：

| 内容 | 公式/意义 |
| --- | --- |
| 二项分布 | $X_n = \sum_{i=1}^n Y_i, \, Y_i\sim \text{Bernoulli}(p)$ |
| 标准化变量 | $Z_n = \frac{X_n - np}{\sqrt{np(1-p)}}$ |
| 极限分布 | $Z_n \xrightarrow{d} N(0,1)$ |
| 关键工具 | Stirling 近似 或 中心极限定理 |

## Question

正态分布和高斯积分有什么关系？

## Answer

非常好的问题！🌟
“正态分布”和“高斯积分”其实是一体两面的概念——正态分布的概率密度函数（PDF）本质上就是高斯函数，而高斯积分是它能够成为“分布”的数学基础。
我们一步步讲清楚它们的关系。

#### 一、正态分布的形式

标准正态分布 $N(0, 1)$ 的概率密度函数是：

```math
f(x) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}
```

更一般的正态分布 $N(\mu, \sigma^2)$ 是：

```math
f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x - \mu)^2}{2\sigma^2}}
```

要使 $f(x)$ 成为一个“概率密度函数”，就必须满足：

```math
\int_{-\infty}^{+\infty} f(x) \, dx = 1
```

这就需要用到高斯积分。

#### 二、高斯积分的定义与结果

所谓“高斯积分”是指：

```math
I = \int_{-\infty}^{+\infty} e^{-x^2} \, dx
```

这个积分没有初等函数的原函数，但它的值是一个著名的常数：

```math
I = \sqrt{\pi}
```

这就是高斯积分的核心结论。

#### 三、如何求出高斯积分（经典推导）

关键技巧：平方再转到极坐标。

1️⃣ 令

```math
I = \int_{-\infty}^{+\infty} e^{-x^2} dx
```

那么：

```math
I^2 = \left(\int_{-\infty}^{+\infty} e^{-x^2} dx\right)
\left(\int_{-\infty}^{+\infty} e^{-y^2} dy\right)
= \int_{\mathbb{R}^2} e^{-(x^2 + y^2)} dxdy
```

2️⃣ 改成极坐标：

```math
x^2 + y^2 = r^2, \quad dxdy = r\,dr\,d\theta
```

于是：

```math
I^2 = \int_0^{2\pi} \int_0^{\infty} e^{-r^2} r \, dr \, d\theta
```

3️⃣ 积分：

```math
I^2 = 2\pi \int_0^{\infty} e^{-r^2} r\,dr
```

令 $t = r^2 \Rightarrow dt = 2r\,dr$，得：

```math
I^2 = \pi \int_0^{\infty} e^{-t} dt = \pi
```

所以：

```math
I = \sqrt{\pi}
```

#### 四、与正态分布的联系

我们现在知道：

```math
\int_{-\infty}^{+\infty} e^{-x^2} dx = \sqrt{\pi}
```

因此，对于标准正态分布：

```math
\int_{-\infty}^{+\infty} \frac{1}{\sqrt{2\pi}} e^{-x^2/2} dx = 1
```

推导如下：

令 $t = x / \sqrt{2} \Rightarrow dx = \sqrt{2}\,dt$，则：

```math
\int_{-\infty}^{+\infty} e^{-x^2/2} dx = \sqrt{2}\int_{-\infty}^{+\infty} e^{-t^2} dt = \sqrt{2\pi}
```

所以：

```math
\frac{1}{\sqrt{2\pi}} \int_{-\infty}^{+\infty} e^{-x^2/2} dx = 1
```

✅ 这说明了为什么正态分布的系数必须是 $\frac{1}{\sqrt{2\pi}}$：

它正是高斯积分的归一化常数，使得总概率为 1。

#### 五、因此两者的关系总结如下：

| 概念 | 数学形式 | 含义 |
| --- | --- | --- |
| 高斯积分 | $\int_{-\infty}^{+\infty} e^{-x^2} dx = \sqrt{\pi}$ | 纯数学积分，计算指数平方的面积 |
| 高斯函数 | $e^{-x^2/2}$ | 正态分布的核心形状 |
| 正态分布 | $f(x) = \frac{1}{\sqrt{2\pi}} e^{-x^2/2}$ | 高斯函数乘以归一化系数（由高斯积分决定） |

✅ 一句话总结：

正态分布的概率密度函数就是高斯函数，
而高斯积分 $\int e^{-x^2} dx = \sqrt{\pi}$ 使得它能被“归一化”为一个真正的概率分布。
