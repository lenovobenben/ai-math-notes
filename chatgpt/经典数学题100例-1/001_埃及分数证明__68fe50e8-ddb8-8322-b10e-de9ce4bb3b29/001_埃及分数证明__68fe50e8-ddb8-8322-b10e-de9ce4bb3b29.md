# 埃及分数证明

- Conversation ID: `68fe50e8-ddb8-8322-b10e-de9ce4bb3b29`

## Question

证明：任何有理数x（0<x<1）都可表为有限个互不相同整数的倒数和

## Answer

### 证明（埃及分数/贪心算法）

设 $x=\dfrac{a}{b}$ 是一个互质的有理数，且 $0<a<b$。我们用如下构造性方法（也称埃及分数的贪心算法）把它写成有限个互不相同整数的倒数和。

取第一个单位分数的分母为

```math
n=\left\lceil\frac{b}{a}\right\rceil.
```

若 $b$ 恰好等于 $a n$，则 $\dfrac{a}{b}=\dfrac{1}{n}$，结束。否则考虑差

```math
\frac{a}{b}-\frac{1}{n}=\frac{a n-b}{b n}.
```

记新的分子为 $a_1=a n-b$，新的分母为 $b_1=b n$。注意

```math
0\lt{}a_1=a n-b\lt{} a n-a(n-1)=a,
```

因此 $a_1$ 是一个正整数，且严格小于原来的分子 $a$。

若 $a_1=1$，则 $\dfrac{a_1}{b_1}=\dfrac{1}{b_1}$，把它加入表示中，算法结束。若 $a_1>1$，重复步骤1：取

```math
n_1=\left\lceil\frac{b_1}{a_1}\right\rceil,
```

并用 $\dfrac{1}{n_1}$ 去减去 $\dfrac{a_1}{b_1}$，得到新的分子 $a_2<a_1$。

因为分子在每一步都严格下降且始终为正整数，经过有限步必然到达分子为1 的情形，进而在下一步把剩余部分表示为一个单位分数，算法终止。

于是我们把最初的 $\dfrac{a}{b}$ 写成有限个互不相同的单位分数之和：

```math
\frac{a}{b}=\frac{1}{n}+\frac{1}{n_1}+\cdots+\frac{1}{n_k}.
```

举个小例子：$\dfrac{5}{7}$：

```math
\left\lceil\frac{7}{5}\right\rceil=2\Rightarrow \frac{5}{7}-\frac{1}{2}=\frac{3}{14},
```

```math
\left\lceil\frac{14}{3}\right\rceil=5\Rightarrow \frac{3}{14}-\frac{1}{5}=\frac{1}{70},
```

因此 $\dfrac{5}{7}=\dfrac{1}{2}+\dfrac{1}{5}+\dfrac{1}{70}$。

综上，任意 $0<x<1$ 的有理数都能写成有限个互不相同的整数倒数之和。□

## Answer

好的，我们用上面的埃及分数贪心算法来计算这三个例子。逐步展示过程：

##### 1️⃣ $\frac{23}{97}$

第一步：

```math
n_1 = \lceil 97/23 \rceil = \lceil 4.217 \rceil = 5
```

```math
\frac{23}{97} - \frac{1}{5} = \frac{23\cdot5 - 97}{97\cdot5} = \frac{115 - 97}{485} = \frac{18}{485}
```

第二步：

```math
n_2 = \lceil 485/18 \rceil = \lceil 26.94 \rceil = 27
```

```math
\frac{18}{485} - \frac{1}{27} = \frac{18\cdot27 - 485}{485\cdot27} = \frac{486-485}{13095} = \frac{1}{13095}
```

✅ 所以

```math
\frac{23}{97} = \frac{1}{5} + \frac{1}{27} + \frac{1}{13095}
```

##### 2️⃣ $\frac{33}{49}$

第一步：

```math
n_1 = \lceil 49/33 \rceil = 2
```

```math
\frac{33}{49} - \frac{1}{2} = \frac{33\cdot2 - 49}{49\cdot2} = \frac{66-49}{98} = \frac{17}{98}
```

第二步：

```math
n_2 = \lceil 98/17 \rceil = 6
```

```math
\frac{17}{98} - \frac{1}{6} = \frac{17\cdot6 - 98}{98\cdot6} = \frac{102-98}{588} = \frac{4}{588} = \frac{1}{147}
```

✅ 所以

```math
\frac{33}{49} = \frac{1}{2} + \frac{1}{6} + \frac{1}{147}
```

##### 3️⃣ $\frac{51}{80}$

第一步：

```math
n_1 = \lceil 80/51 \rceil = 2
```

```math
\frac{51}{80} - \frac{1}{2} = \frac{51\cdot2 - 80}{80\cdot2} = \frac{102-80}{160} = \frac{22}{160} = \frac{11}{80}
```

第二步：

```math
n_2 = \lceil 80/11 \rceil = 8
```

```math
\frac{11}{80} - \frac{1}{8} = \frac{11\cdot8 - 80}{80\cdot8} = \frac{88-80}{640} = \frac{8}{640} = \frac{1}{80}
```

✅ 所以

```math
\frac{51}{80} = \frac{1}{2} + \frac{1}{8} + \frac{1}{80}
```

## Question

这种分解是唯一的吗？

## Answer

非常好的问题 👍——不是唯一的。
埃及分数（即有限个互不相同正整数倒数之和的表示）在一般情况下是非唯一的。

##### ✅ 一、非唯一性的例子

例如：

```math
\frac{2}{3}
```

一种分解是

```math
\frac{2}{3} = \frac{1}{2} + \frac{1}{6}
```

但也可以写成

```math
\frac{2}{3} = \frac{1}{3} + \frac{1}{4} + \frac{1}{12}
```

这两个分解都正确，分母集合 $\{2,6\}$ 与 $\{3,4,12\}$ 完全不同。

##### ✅ 二、为什么不唯一？

因为分数之间可以用恒等式进一步分解，例如：

```math
\frac{1}{n} = \frac{1}{n+1} + \frac{1}{n(n+1)}
```

这个恒等式可以无限使用。

比如从 $\frac{2}{3}=\frac{1}{2}+\frac{1}{6}$ 出发，
把 $\frac{1}{6}$ 替换为 $\frac{1}{7}+\frac{1}{42}$，就得到另一种表示：

```math
\frac{2}{3} = \frac{1}{2} + \frac{1}{7} + \frac{1}{42}
```

因此可以得到无穷多种不同分解。

##### ✅ 三、但“贪心算法分解”是唯一的

虽然一般分解不唯一，但我们前面使用的贪心算法（取最小可能的单位分数）产生的那一种分解是唯一的。

也就是说：

对于任意给定的有理数 $0 < x < 1$，按照贪心算法规则

```math
n = \left\lceil \frac{1}{x} \right\rceil
```

一步步生成的埃及分数分解是唯一确定的。

##### ✅ 总结

| 性质 | 结论 |
| --- | --- |
| 一般的埃及分数分解 | 不唯一 |
| 贪心算法生成的分解 | 唯一 |
| 能否分解为有限项 | 总是可以（对于有理数） |
