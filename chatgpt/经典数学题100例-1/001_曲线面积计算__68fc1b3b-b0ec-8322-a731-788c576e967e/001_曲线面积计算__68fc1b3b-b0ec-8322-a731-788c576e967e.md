# 曲线面积计算

- Conversation ID: `68fc1b3b-b0ec-8322-a731-788c576e967e`

## Question

笛卡尔坐标系内，x^2+y^2=|x|+|y| 。 求这个曲线围成的面积

## Answer

先把几何对称性用上：因为方程关于坐标轴对称，只需计算第一象限内满足不等式

```math
x^2+y^2\le |x|+|y|
```

的面积再乘以 4。第一象限 $x,y\ge0$ 时等价于

```math
x^2+y^2\le x+y.
```

用极坐标（以原点为极点，第一象限对应 $\theta\in[0,\tfrac{\pi}{2}]$），设
$x=\rho\cos\theta,\;y=\rho\sin\theta$。不等式变为

```math
\rho^2\le\rho(\cos\theta+\sin\theta)\quad\Rightarrow\quad 0\le\rho\le\cos\theta+\sin\theta.
```

第一象限面积为

```math
A_{Q}=\int_{0}^{\pi/2}\int_{0}^{\cos\theta+\sin\theta}\rho\,d\rho\,d\theta
=\tfrac12\int_{0}^{\pi/2}(\cos\theta+\sin\theta)^2\,d\theta.
```

计算积分：

```math
(\cos\theta+\sin\theta)^2=1+\sin2\theta,
```

所以

```math
A_{Q}=\tfrac12\int_{0}^{\pi/2}(1+\sin2\theta)\,d\theta
=\tfrac12\Big(\tfrac{\pi}{2}+1\Big)=\frac{\pi}{4}+\frac12.
```

总面积为四象限之和：

```math
A=4A_{Q}=4\Big(\frac{\pi}{4}+\frac12\Big)=\pi+2.
```

答案：$\boxed{\pi+2}$（约为 $5.1416$）。
