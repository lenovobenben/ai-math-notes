# 取值范围求解

- Conversation ID: `68f9ec8a-6098-8324-bf94-4cc56aa04486`

## Question

已知 $x, y, z$ 均是正数，求

```math
\frac{x}{x+y} + \frac{y}{y+z} + \frac{z}{z+x}
```

的取值范围。

## Answer

答案是

```math
\boxed{1\lt{}\frac{x}{x+y}+\frac{y}{y+z}+\frac{z}{z+x}\lt{}2}.
```

证明（要点）：

下界：因为对正数 $x,y,z$ 有

```math
\frac{x}{x+y}\gt{}\frac{x}{x+y+z},\qquad
\frac{y}{y+z}\gt{}\frac{y}{x+y+z},\qquad
\frac{z}{z+x}\gt{}\frac{z}{x+y+z},
```

将三式相加得

```math
\frac{x}{x+y}+\frac{y}{y+z}+\frac{z}{z+x}\gt{}\frac{x+y+z}{x+y+z}=1,
```

所以总和 $>1$。

上界：注意三个余项

```math
\frac{y}{x+y},\ \frac{z}{y+z},\ \frac{x}{z+x}
```

也满足与上面的同样不等式之和 $>1$。而这三项之和等于 $3-\big(\frac{x}{x+y}+\frac{y}{y+z}+\frac{z}{z+x}\big)$，因此

```math
3-\Big(\frac{x}{x+y}+\frac{y}{y+z}+\frac{z}{z+x}\Big)\gt{}1,
```

即

```math
\frac{x}{x+y}+\frac{y}{y+z}+\frac{z}{z+x}\lt{}2.
```

端点不可取但可任意逼近：例如

令 $y\to\infty$、$x$ 固定而 $z\ll x$，可使和逼近 $1$；

令 $x\to\infty$、$y$ 固定而 $z\ll1$，可使和逼近 $2$。

综上，取值范围为开区间 $(1,2)$。
