# 证明P^2+26不是质数

- Conversation ID: `68f9d9cd-ed64-8322-ac80-8b93dd6fface`

## Question

```math
已知P是质数，求证P^2+26不是质数
```

## Answer

好的——很快证明如下：

若 $p$ 是质数，分两种情况讨论：

$p=3$。此时 $p^2+26=9+26=35=5\times7$，合数。

$p\ne3$。因为 $p$ 不是 3，故 $p$ 与 3 互素，模 3 下只有两种可能：
$p\equiv1\pmod3$ 或 $p\equiv2\pmod3$。两者平方都同余于 1：

```math
p^2\equiv1\pmod3.
```

因此

```math
p^2+26\equiv 1+26\equiv 27\equiv0\pmod3.
```

即 $3\mid(p^2+26)$。且当 $p\ge2$ 时 $p^2+26\ge30>3$，所以被 3 整除的不可能是质数，必为合数。

综上：对任一质数 $p$，$p^2+26$ 都是合数。证明完毕。 ✔️
