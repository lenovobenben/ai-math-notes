# 化简表达式结果

- Conversation ID: `68fcdff8-9444-832b-b098-d01e2d1d052a`

## Question

化简   \left( \frac{1 - \sqrt{13}}{2} \right)^7 + \left( \frac{1 + \sqrt{13}}{2} \right)^7

## Answer

我们要化简

```math
\left( \frac{1 - \sqrt{13}}{2} \right)^7 + \left( \frac{1 + \sqrt{13}}{2} \right)^7
```

##### 第一步：设定符号

令

```math
a = \frac{1 + \sqrt{13}}{2}, \quad b = \frac{1 - \sqrt{13}}{2}.
```

则 $a + b = 1$，$ab = \frac{1 - 13}{4} = -3$。

我们要求：

```math
S_n = a^n + b^n.
```

##### 第二步：递推关系

由对称多项式的性质（因为 $a,b$ 是二次方程根）：

$a,b$ 是方程

```math
x^2 - (a+b)x + ab = 0 \Rightarrow x^2 - x - 3 = 0
```

的两个根。

于是 $S_n$ 满足递推：

```math
S_{n+2} = (a+b) S_{n+1} - (ab) S_n = 1 \cdot S_{n+1} + 3 S_n.
```

即：

```math
S_{n+2} = S_{n+1} + 3 S_n.
```

初值：

```math
S_0 = 2, \quad S_1 = a + b = 1.
```

##### 第三步：逐步计算

```math
\begin{aligned}
S_0 &= 2, \\
S_1 &= 1, \\
S_2 &= S_1 + 3S_0 = 1 + 6 = 7, \\
S_3 &= S_2 + 3S_1 = 7 + 3 = 10, \\
S_4 &= S_3 + 3S_2 = 10 + 21 = 31, \\
S_5 &= S_4 + 3S_3 = 31 + 30 = 61, \\
S_6 &= S_5 + 3S_4 = 61 + 93 = 154, \\
S_7 &= S_6 + 3S_5 = 154 + 183 = 337.
\end{aligned}
```

✅ 最终结果：

```math
\boxed{337}.
```

验证：因为 $a,b$ 是实数且 $b < 1 < a$，所以 $a^7 + b^7$ 应为整数且略大于 $a^7 \approx 2.3^7 \approx 340$，结果 $337$ 符合数值预期。
