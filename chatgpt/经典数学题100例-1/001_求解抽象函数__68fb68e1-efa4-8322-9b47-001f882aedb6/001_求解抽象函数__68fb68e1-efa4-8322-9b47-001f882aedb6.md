## Question

f(x) + f\left(\frac{1}{1-x}\right) = x 

求：\( f(x) \)

## Answer

我们来仔细求解这个函数方程：

方程是

```math
f(x) + f\Big(\frac{1}{1-x}\Big) = x.
```

这是一个函数方程问题。通常思路是通过迭代变量变换来寻找函数的规律。

##### 第 1 步：引入迭代变量

定义

```math
y = \frac{1}{1-x}.
```

那么方程变成

```math
f(x) + f(y) = x, \quad \text{其中 } y = \frac{1}{1-x}.
```

接下来计算 $y$ 的下一个变换：

z = \frac{1}{1-y} = \frac{1}{1 - \frac{1}{1-x}}} = \frac{1}{\frac{(1-x)-1}{1-x}}} = \frac{1}{\frac{-x}{1-x}} = -\frac{1-x}{x} = \frac{x-1}{x}.

于是有

```math
f(y) + f(z) = y = \frac{1}{1-x}.
```

再一次迭代：

```math
w = \frac{1}{1-z} = \frac{1}{1 - \frac{x-1}{x}} = \frac{1}{1 - 1 + \frac{1}{x}} = \frac{1}{1/x} = x.
```

所以我们得到一个三步循环：

```math
x \to y = \frac{1}{1-x} \to z = \frac{x-1}{x} \to w = x.
```

##### 第 2 步：建立线性方程组

由三步循环得到三个方程：

```math
f(x) + f(y) = x
```

```math
f(y) + f(z) = y = \frac{1}{1-x}
```

```math
f(z) + f(x) = z = \frac{x-1}{x}
```

将三者表示为线性方程组：

```math
\begin{cases}
f(x) + f(y) = x \\
f(y) + f(z) = \frac{1}{1-x} \\
f(z) + f(x) = \frac{x-1}{x}
\end{cases}
```

##### 第 3 步：解线性方程组

先用前两个方程消去 $f(y)$：

```math
f(y) = x - f(x)
```

代入第二个方程：

```math
(x - f(x)) + f(z) = \frac{1}{1-x} \implies f(z) - f(x) = \frac{1}{1-x} - x
```

再用第三个方程：

```math
f(z) + f(x) = \frac{x-1}{x}.
```

联立两式求 $f(x)$：

```math
f(z) - f(x) = \frac{1}{1-x} - x
```

```math
f(z) + f(x) = \frac{x-1}{x}
```

把两式相加：

```math
2 f(z) = \left(\frac{1}{1-x} - x\right) + \frac{x-1}{x} = \frac{1}{1-x} - x + \frac{x-1}{x}.
```

于是

```math
f(z) = \frac{1}{2} \left( \frac{1}{1-x} - x + \frac{x-1}{x} \right)
```

再用 $f(z) + f(x) = \frac{x-1}{x}$ 求 $f(x)$：

```math
f(x) = \frac{x-1}{x} - f(z) = \frac{x-1}{x} - \frac{1}{2}\left( \frac{1}{1-x} - x + \frac{x-1}{x} \right)
```

整理：

```math
f(x) = \frac{x-1}{x} - \frac{1}{2} \frac{1}{1-x} + \frac{1}{2} x - \frac{1}{2} \frac{x-1}{x}
```

```math
f(x) = \frac{1}{2} \frac{x-1}{x} + \frac{1}{2} x - \frac{1}{2} \frac{1}{1-x}
= \frac{x-1}{2x} + \frac{x}{2} - \frac{1}{2(1-x)}
```

合并成一个分式也可以：

```math
f(x) = \frac{x-1}{2x} + \frac{x}{2} - \frac{1}{2(1-x)}.
```

##### ✅ 第 4 步：最终答案

```math
\boxed{f(x) = \frac{x-1}{2x} + \frac{x}{2} - \frac{1}{2(1-x)}}
```

这是原方程的解。

如果需要，也可以进一步合并成一个分式：

```math
f(x) = \frac{x^2 - x + 1}{2x} - \frac{1}{2(1-x)} \quad \text{或类似形式。}
```

## Answer

```math
f(x) = \frac{x-1}{2x} + \frac{x}{2} - \frac{1}{2(1-x)}
```

整理成一个最简分式形式。

##### 第 1 步：统一分母

三个分式分别是：

```math
\frac{x-1}{2x}
```

```math
\frac{x}{2} = \frac{x}{2} \cdot \frac{x}{x} = \frac{x^2}{2x}
```

```math
-\frac{1}{2(1-x)} = \frac{-1}{2(1-x)}
```

先合并前两项：

```math
\frac{x-1}{2x} + \frac{x^2}{2x} = \frac{x^2 + x - 1}{2x}.
```

所以：

```math
f(x) = \frac{x^2 + x - 1}{2x} - \frac{1}{2(1-x)}
= \frac{x^2 + x - 1}{2x} - \frac{1}{2(1-x)}.
```

##### 第 2 步：合并成单一分式

分母 $2x(1-x)$，于是：

```math
f(x) = \frac{(x^2 + x - 1)(1-x) - x}{2x(1-x)}.
```

展开分子：

```math
(x^2 + x -1)(1-x) = x^2(1-x) + x(1-x) -1(1-x) = x^2 - x^3 + x - x^2 -1 + x = -x^3 + x -1 + x = -x^3 + 2x -1
```

再减去 $x$（因为分子还有 $-x$）：

```math
-x^3 + 2x -1 - x = -x^3 + x -1
```

所以分子是 $-x^3 + x -1$，分母是 $2x(1-x)$。

##### ✅ 第 3 步：最终最简分式形式

```math
\boxed{f(x) = \frac{-x^3 + x - 1}{2x(1-x)}}.
```

这个就是单一分式形式的解。
