## Question

已知 a_1, a_2, \ldots, a_n \in (-1, 1)

证明 \prod_{1 \leq i,j \leq n} \left(\frac{1 + a_i a_j}{1 - a_i a_j}\right) \geq 1

## Answer

好的，我们仔细分析这个不等式问题。已知

```math
a_1, a_2, \dots, a_n \in (-1,1)
```

要求证明

```math
\prod_{1 \le i,j \le n} \frac{1 + a_i a_j}{1 - a_i a_j} \ge 1.
```

##### 第一步：对数转换

由于所有因子都是正的（因为 $|a_i a_j|<1$），可以对不等式取自然对数：

```math
\ln \prod_{i,j} \frac{1 + a_i a_j}{1 - a_i a_j} = \sum_{i,j} \ln\frac{1 + a_i a_j}{1 - a_i a_j}.
```

定义函数

```math
f(x) = \ln\frac{1+x}{1-x}, \quad x \in (-1,1).
```

则不等式变为

```math
\sum_{i,j} f(a_i a_j) \ge 0.
```

##### 第二步：函数性质

注意到

```math
f(x) = \ln(1+x) - \ln(1-x),
```

并且 $f(x)$ 在 $(-1,1)$ 上是 严格单调增函数，而且是 奇函数（$f(-x) = -f(x)$）。

更重要的是，$f(x)$ 可以展开为幂级数：

```math
f(x) = 2\left(x + \frac{x^3}{3} + \frac{x^5}{5} + \cdots\right),
```

这是因为

```math
\ln\frac{1+x}{1-x} = 2 \sum_{k=0}^{\infty} \frac{x^{2k+1}}{2k+1}.
```

因此 $f(x)$ 是x的奇次幂的正系数幂级数。

##### 第三步：代入 $x = a_i a_j$

我们有

```math
\sum_{i,j} f(a_i a_j) = \sum_{i,j} 2 \sum_{k=0}^\infty \frac{(a_i a_j)^{2k+1}}{2k+1} = 2 \sum_{k=0}^\infty \frac{1}{2k+1} \sum_{i,j} (a_i a_j)^{2k+1}.
```

交换求和顺序：

```math
\sum_{i,j} (a_i a_j)^{2k+1} = \sum_{i,j} a_i^{2k+1} a_j^{2k+1} = \left(\sum_{i=1}^n a_i^{2k+1}\right)^2 \ge 0.
```

于是每一项都是非负的平方，因此整个和非负：

```math
\sum_{i,j} f(a_i a_j) = 2 \sum_{k=0}^\infty \frac{1}{2k+1} \left(\sum_{i=1}^n a_i^{2k+1}\right)^2 \ge 0.
```

##### 第四步：结论

由于对数和非负，因此原来的乘积满足：

```math
\prod_{1\le i,j\le n} \frac{1 + a_i a_j}{1 - a_i a_j} \ge 1.
```

✅ 结论：不等式成立，等号当且仅当 $a_1 = a_2 = \dots = a_n = 0$ 时取到。

## Answer

好的，我们用一个更直观、基于向量和 Cauchy 型不等式的方法来证明这个不等式。

##### 直观证明思路（向量/内积法）

我们需要证明

```math
\prod_{1 \le i,j \le n} \frac{1 + a_i a_j}{1 - a_i a_j} \ge 1, \quad a_i \in (-1,1).
```

###### 第一步：对数转换

和之前一样，先对不等式取自然对数：

```math
\sum_{i,j} \ln \frac{1 + a_i a_j}{1 - a_i a_j} \ge 0.
```

令函数

```math
f(x) = \ln \frac{1+x}{1-x} = 2 \, \mathrm{artanh}(x), \quad x \in (-1,1),
```

那么我们要证明

```math
\sum_{i,j} f(a_i a_j) \ge 0.
```

###### 第二步：用 $\mathrm{artanh}$ 转换

因为 $\mathrm{artanh}$ 是单调增函数，定义

```math
b_i = \mathrm{artanh}(a_i) \in \mathbb{R}, \quad \text{即} a_i = \tanh(b_i).
```

于是

```math
a_i a_j = \tanh(b_i)\tanh(b_j), \quad f(a_i a_j) = 2\,\mathrm{artanh}(\tanh(b_i)\tanh(b_j)).
```

###### 第三步：双曲函数恒等式

利用恒等式：

```math
\mathrm{artanh}(x) + \mathrm{artanh}(y) = \mathrm{artanh}\left(\frac{x+y}{1+xy}\right), \quad |x|,|y|\lt{}1,
```

或者更适合我们的形式（双曲正切）：

```math
\tanh(b_i) \tanh(b_j) = \frac{\cosh(b_i-b_j)-\cosh(b_i+b_j)}{\cosh(b_i-b_j)+\cosh(b_i+b_j)} \quad \text{（稍复杂，但可化简为平方和形式）}
```

为了简单，我们可以采用更直接的方法：

###### 第四步：幂级数/内积法（向量表示）

回到幂级数展开（更直观地理解为向量平方和）：

```math
f(a_i a_j) = 2\sum_{k=0}^\infty \frac{(a_i a_j)^{2k+1}}{2k+1}.
```

于是

```math
\sum_{i,j} f(a_i a_j) = 2 \sum_{k=0}^\infty \frac{1}{2k+1} \sum_{i,j} a_i^{2k+1} a_j^{2k+1} = 2 \sum_{k=0}^\infty \frac{1}{2k+1} \Big(\sum_{i=1}^n a_i^{2k+1}\Big)^2.
```

###### 第五步：向量内积的解释

把向量定义为

```math
\mathbf{v}^{(k)} = \big(a_1^{2k+1}, a_2^{2k+1}, \dots, a_n^{2k+1}\big) \in \mathbb{R}^n,
```

那么

```math
\sum_{i,j} a_i^{2k+1} a_j^{2k+1} = \langle \mathbf{v}^{(k)}, \mathbf{v}^{(k)} \rangle = \|\mathbf{v}^{(k)}\|^2 \ge 0.
```

所以每一项都是向量长度的平方，自然非负，因此整个和非负。

##### ✅ 第六步：结论

因此

```math
\sum_{i,j} f(a_i a_j) = 2 \sum_{k=0}^\infty \frac{\|\mathbf{v}^{(k)}\|^2}{2k+1} \ge 0 \implies \prod_{i,j} \frac{1+a_i a_j}{1-a_i a_j} \ge 1.
```

等号成立当且仅当 $\mathbf{v}^{(k)} = 0 \implies a_i = 0, \forall i$。

这个解释就像把原来的乘积写成“无限维向量的平方和”，每一层都是非负的，直观又优雅。
