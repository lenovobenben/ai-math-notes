# 随机数生成

- Conversation ID: `68ff8e0b-0488-8322-8a81-6eb3f633a8b2`

## Question

随机生成[1,n]中的整数，出现第1个重复数字前有几个数（即求期望）？

## Answer

设从 $[1,n]$ 中独立且等概率地重复抽取整数，令随机变量 $T$ 表示“第一次出现重复时所抽取的次数”（即包含第一个重复的那一次抽取）。要求 $\mathbb{E}[T]$。

精确表示（尾和公式）
记事件“前 $k$ 次抽取都互不相同”的概率为

```math
P(\text{no repeat in first }k)=\frac{n(n-1)\cdots(n-k+1)}{n^k}=\prod_{i=0}^{k-1}\Big(1-\frac{i}{n}\Big),\quad 0\le k\le n.
```

由尾和公式（tail-sum）有

```math
\mathbb{E}[T]=\sum_{k\ge 0} P(T\gt{}k)=\sum_{k=0}^{n}\prod_{i=0}^{k-1}\Big(1-\frac{i}{n}\Big).
```

（注意当 $k>n$ 时概率为 0，所以上界写到 $n$ 即可。）

渐近近似（大 $n$）
当 $n$ 很大，可用近似

```math
\prod_{i=0}^{k-1}\Big(1-\frac{i}{n}\Big)\approx\exp\Big(-\frac{k(k-1)}{2n}\Big)\approx\exp\Big(-\frac{k^2}{2n}\Big)
```

把离散和近似为积分，得到主项

```math
\mathbb{E}[T]\sim\int_{0}^{\infty} e^{-k^2/(2n)}\,dk=\sqrt{\frac{\pi n}{2}}.
```

更精细的渐近展开给出

```math
\mathbb{E}[T]=\sqrt{\frac{\pi n}{2}}+\frac{2}{3}+o(1)\qquad(n\to\infty).
```

数值例子

$n=365$ 时，精确和（可由上面的表达数值计算）约为

```math
\displaystyle \mathbb{E}[T]\approx 24.6166.
```

当 $n$ 很大时，可用 $\sqrt{\pi n/2}$ 作为快速近似（并可加上约 $2/3$ 的常数校正以提高精度）。
