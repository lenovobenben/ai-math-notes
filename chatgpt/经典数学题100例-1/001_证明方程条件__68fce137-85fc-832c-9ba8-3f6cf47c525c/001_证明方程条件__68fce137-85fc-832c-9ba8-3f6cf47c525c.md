# 证明方程条件

- Conversation ID: `68fce137-85fc-832c-9ba8-3f6cf47c525c`

## Question

```math
a + b + c = 1
```

\frac{1}{a} + \frac{1}{b} + \frac{1}{c} = 1

\textbf{证明：\( a, b, c \) 至少有一个等于1}

## Answer

先说明一点：由题中 $\dfrac1a+\dfrac1b+\dfrac1c$ 存在可知 $a,b,c\neq0$。

设

```math
S_1=a+b+c,\qquad S_2=ab+bc+ca,\qquad S_3=abc.
```

题给 $S_1=1$，且

```math
\frac1a+\frac1b+\frac1c=\frac{ab+bc+ca}{abc}=\frac{S_2}{S_3}=1,
```

因此 $S_2=S_3$。

考虑多项式 $(1-a)(1-b)(1-c)$ 的展开：

```math
(1-a)(1-b)(1-c)=1-(a+b+c)+(ab+bc+ca)-abc=1-S_1+S_2-S_3.
```

代入 $S_1=1$ 与 $S_2=S_3$，得

```math
(1-a)(1-b)(1-c)=0.
```

因此至少有一个因子为零，即至少有一个 $1-a,1-b,1-c$ 为 0，等价于至少有一个 $a,b,c$ 等于 $1$。

证毕。 ✦
