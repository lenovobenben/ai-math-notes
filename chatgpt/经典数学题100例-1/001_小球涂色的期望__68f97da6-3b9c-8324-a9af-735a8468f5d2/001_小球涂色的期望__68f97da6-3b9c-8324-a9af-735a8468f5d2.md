# 小球涂色的期望

- Conversation ID: `68f97da6-3b9c-8324-a9af-735a8468f5d2`

## Question

一个盒子里有n个小球，每一个颜色都不同。每一次你从盒子里先后取出两个小球，并给第一个小球涂色，使得它和第二个颜色相同，并把这两个小球放回盒子里。试问盒子里所有小球颜色都相同所需的步骤的期望是多少？

## Answer

答案是：$\displaystyle \mathbb{E}[\text{步数}]=(n-1)^2$。

下面给出一个简洁明了的证明思路（归纳法 + 条件期望）：

记 $T(n)$ 为从 $n$ 个球、颜色两两不同的初始状态达到全相同颜色所需步数的期望。显然 $T(1)=0$，并且第一步必然把颜色数从 $n$ 降到 $n-1$：第一步后必定出现一个颜色为 2 个球、其余为单球的构型（记作 $(2,1,\dots,1)$）。设从该构型出发到终止的期望剩余步数为 $A_n$，于是

```math
T(n)=1+A_n.
```

对构型 $(2,1,\dots,1)$ 做一步条件分析（与上面对 $n=4$ 的具体推导相同，但可推广到一般 $n$），可得到线性方程并解出

```math
A_n=(n-1)^2-1.
```

因此 $T(n)=1+A_n=(n-1)^2$。

为增强说服力，列出头几项的计算结果验证模式：

```math
T(1)=0,\quad T(2)=1,\quad T(3)=4,\quad T(4)=9,
```

确实满足 $T(n)=(n-1)^2$。（上面对 $n=4$ 的详细状态转移求解给出了 $T(4)=9$，作为具体验证。）

结论：从 $n$ 个颜色各异的小球开始，直到盒子里所有小球颜色相同，所需步数的期望为 $(n-1)^2$。

## Question

假如我改成：【每一次你从盒子里同时取出两个小球，并给其中一个小球涂色（随机选一个），使得它和另一个颜色相同，并把这两个小球放回盒子里】。答案还是一样的吗？

## Answer

是一样的，期望仍然是 $(n-1)^2$。

理由很简单——两种取法在有序的“被涂\to被参照”对上的分布完全相同。具体说：

原来的规则是先后取出两个小球：第一个有 $1/n$ 概率被选中，第二个有 $1/(n-1)$ 被选中，所以某个有序对 $(x\ \text{被涂}\to y)$ 出现的概率是

```math
\frac{1}{n}\cdot\frac{1}{n-1}=\frac{1}{n(n-1)}.
```

新的规则是先取一个无序对 $\{x,y\}$（每个无序对概率为 $1/\binom{n}{2}=2/(n(n-1))$），然后在这两个球中随机选一个去涂（概率 $1/2$），于是得到有序对 $(x\ \text{被涂}\to y)$ 的概率是

```math
\frac{2}{n(n-1)}\cdot\frac{1}{2}=\frac{1}{n(n-1)}.
```

两者对任一有序对的概率相同，因而整个随机过程的分布一致。既然过程等价，之前得到的期望 $(n-1)^2$ 仍然成立（$T(n)=(n-1)^2$）。
