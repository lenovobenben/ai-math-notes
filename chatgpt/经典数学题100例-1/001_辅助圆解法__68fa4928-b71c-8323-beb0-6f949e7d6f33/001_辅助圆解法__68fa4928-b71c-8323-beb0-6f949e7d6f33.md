# 辅助圆解法

- Conversation ID: `68fa4928-b71c-8323-beb0-6f949e7d6f33`

## Question

正三角形ABC。BC上有一点E，CA上有一点F，BE=CF, AE和BF相较于D点。求CD的最小值

## Answer

把边长记作 $a$（若题中已假定边长为 1，代入 $a=1$ 即可）。

取坐标方便计算：
令等边三角形 $ABC$ 的顶点为

```math
A=(0,0),\; B=(a,0),\; C=(\tfrac a2,\tfrac{\sqrt3}{2}a)
```

在 $BC$ 上取点 $E$，在 $CA$ 上取点 $F$，记 $BE=CF=ta$（$0\le t\le1$）。于是

```math
E=B+t(C-B),\qquad F=C+t(A-C).
```

设 $D$ 为直线 $AE$ 与 $BF$ 的交点，计算可得

```math
CD=\frac{\sqrt{3t^4-6t^3+7t^2-4t+1}}{\,t^2-t+1\,}\,a\quad(\text{长度按比例乘 }a) .
```

为求最小值，考察 $CD^2$ 关于 $t$ 的导数，化简后得到

```math
\frac{d}{dt}(CD^2)=\frac{2(2t-1)}{(t^2-t+1)^2}\,a^2,
```

因此唯一驻点为 $t=\tfrac12$。代入并检查二阶导数为正，可知这是极小值点。于是

```math
\min CD=CD\big|_{t=1/2}=\frac{a}{\sqrt3}.
```

结论：最小值为 $\displaystyle \frac{a}{\sqrt3}$。若三角形边长 $a=1$，则最小值为 $\boxed{\dfrac{1}{\sqrt3}}$。此时 $E,F$ 为边的中点，$D$ 为重心。

## Question

我看有一种初中的解法，是做一个辅助圆。
因为角ADB恒为120°。D虽为动点，但是它的轨迹在一个隐藏圆上。

## Answer

很好——你说的“辅助圆”法是一个更几何、更直观的解法。我把思路按初中几何的风格把步骤写清楚（不用坐标），并把关键的计算展示出来，验证最后仍然得到 $\min CD=\dfrac{a}{\sqrt3}$（若边长取 $a=1$，就是 $\dfrac{1}{\sqrt3}$）。

思路要点

先证明 $\angle ADB=120^\circ$（这是你提到的恒等角）。
（证明略写一条常见路线：由条件 $BE=CF$ 可证出在三角形 $ABC$ 中线/平移关系，使得四边形 $A, E, B, F$ 的某些角或向量关系成立，进而得到 $\angle ADB$ 恒为 $120^\circ$。这里我们接受题中或习题给出的结论：$\angle ADB=120^\circ$。）

由第 1 步：点 $D$ 在动但满足“从弦 $AB$ 所对的圆周角为 $120^\circ$”的圆弧上。
—— 对任何在固定弦 $AB$ 上看到弦 $AB$ 的圆周角为常角 $120^\circ$ 的点，其轨迹是一条过 $A,B$ 的圆（弧）。换言之，所有满足 $\angle ADB=120^\circ$ 的点 $D$ 都位于常圆 $\Omega$ 上。

求这个圆 $\Omega$ 的半径 $R$。
设等边三角形边长为 $a$。弦长 $AB=a$。记圆心为 $O$。
弦 $AB$ 在圆上的圆周角为 $120^\circ$，对应的中心角为 $2\times 120^\circ=240^\circ$。
弦长与半径的关系： $AB=2R\sin(\tfrac{\text{中心角}}{2})=2R\sin(120^\circ)$。
而 $\sin(120^\circ)=\sin(60^\circ)=\tfrac{\sqrt3}{2}$，所以

```math
a=2R\cdot\frac{\sqrt3}{2}=R\sqrt3\quad\Rightarrow\quad R=\frac{a}{\sqrt3}.
```

计算点 $C$ 到圆心 $O$ 的距离 $OC$。
把等边三角形的几何位置想象一下：弦 $AB$ 在三角形的一边上，圆心 $O$ 位于 $AB$ 的垂直平分线上且在 $AB$ 的“下方”（或“上方”，选择使圆周角为 $120^\circ$ 的那一边）。用坐标或直接用长度差也可算出

```math
OC=\frac{2a}{\sqrt3}.
```

（在坐标计算中，取 $A=(0,0),\;B=(a,0),\;C=(\tfrac a2,\tfrac{\sqrt3}{2}a)$，可得 $O=(\tfrac a2,-\tfrac{a}{2\sqrt3})$，从而 $OC= \tfrac{2a}{\sqrt3}$。）

点 $D$ 在圆 $\Omega$ 上，要求 $CD$ 的最小值，就是求点 $C$ 到圆上点的最短距离。几何上这等于点 $C$ 到圆心 $O$ 的距离减去半径：

```math
\min CD = OC - R = \frac{2a}{\sqrt3}-\frac{a}{\sqrt3}=\frac{a}{\sqrt3}.
```

结论与取得情形：最小值为 $\displaystyle \frac{a}{\sqrt3}$。当且仅当 $D$ 为圆 $\Omega$ 上距 $C$ 最近的点时取到该最小值（在等边三角形的对称位置上这时对应 $E,F$ 为边的中点，即你最初用坐标法得到的 $t=\tfrac12$ 情形一致）。

所以用“辅助圆”法的关键是：把恒定角 $\angle ADB=120^\circ$ 转化为“D 在过 $A,B$ 的定圆上”的事实，再用“点到圆上点的最短距离 = 点到圆心距离 − 半径”来立即得到最小值。这条思路既直观又简洁。要不要我把证明“$\angle ADB=120^\circ$”的那一步也用向量/等角线段的初中几何步骤完整写出来？
