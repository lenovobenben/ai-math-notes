# 最小值求解

- Conversation ID: `696f3244-6880-832b-9122-80e11123fab1`

## Question

给定正整数n，定义a_k=1-\frac{k-1}{n},1 \le k \le n，求 \sum_{j=1}^{n}(-1)^{j-1}\prod_{k=1}^{j}a_k 的最小值。

## Answer

把和式稍作整理，可以得到一个很清晰的概率解释，从而很容易得出最小值。

先把每一项写成阶乘形式。
对 $1\le j\le n$，

```math
\prod_{k=1}^j a_k=\prod_{k=1}^j\Big(1-\frac{k-1}{n}\Big)
=\frac{n(n-1)\cdots (n-j+1)}{n^j}
=\frac{n!}{(n-j)!\,n^j}.
```

因此

```math
S(n):=\sum_{j=1}^{n}(-1)^{j-1}\prod_{k=1}^{j}a_k
=\sum_{j=1}^{n}(-1)^{j-1}\Pr(\text{前 }j\text{ 次抽样互不相同}).
```

把概率意义写清楚：在 $n$ 种颜色中“有放回地”连续抽样，令 $T$ 为“首次出现重复”的抽样次数（即第 $T$ 次抽到的球与之前某次相同），则

```math
\Pr(T\gt{}j)=\Pr(\text{前 }j\text{ 次互不相同})=\prod_{k=1}^j\Big(1-\frac{k-1}{n}\Big).
```

所以

```math
S(n)=\sum_{j=1}^{n}(-1)^{j-1}\Pr(T\gt{}j).
```

交换求和次序（把按 $T$ 的分布展开），注意对固定的 $t$ 有
$\displaystyle\sum_{j=1}^{t-1}(-1)^{j-1}=
\begin{cases}1,&t\ \text{为偶数},\\[2pt]0,&t\ \text{为奇数},\end{cases}$
于是得到一个极简的结论：

```math
\boxed{\,S(n)=\Pr(T\ \text{为偶数})\,.}
```

由此很容易看出最小值：当 $n=2$ 时，第一次重复发生在第 $2$ 次的概率为 $1/2$，所以 $S(2)=1/2$。对 $n\ge3$ 的情形可以证明（或直接计算几项并用上面概率配对的思路）$S(n)>1/2$，且当 $n\to\infty$ 时 $S(n)\downarrow\frac12$。因此在所有正整数 $n$ 中

```math
\boxed{\min_{n\in\mathbb Z_{\gt{}0}} S(n)=\tfrac12,}
```

且最小值在 $n=2$ 处取得（极限值 $\tfrac12$ 也可由 $n\to\infty$ 得到）。

（补充代数式：用排列表示可写成

```math
S(n)=(-1)^{\,n-1}\frac{n!}{n^n}\sum_{m=0}^{n-1}\frac{(-n)^m}{m!},
```

但上面的“首次重复时刻 $T$”的概率解释更直观，也便于判断极值与单调行为。）
