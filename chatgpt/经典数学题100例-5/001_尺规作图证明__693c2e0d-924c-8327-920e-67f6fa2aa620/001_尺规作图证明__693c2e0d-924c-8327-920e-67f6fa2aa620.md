# 尺规作图证明

- Conversation ID: `693c2e0d-924c-8327-920e-67f6fa2aa620`

## Question

尺规作图问题。有一∠O，在内部有一定点A。求过A作一直线l，使l交∠O的两条边分别于B,C两点，且使得OB+OC最小。

这里给出一种做法。
为了方便描述，我们设∠O两条边分别为OM,ON。ON为水平放置，OM在上方，且∠O开口向右。
1。作∠O的角平分线a，过点A作a的平行线b。假设A在a的上方（根据对称性下方同理，这里仅为后续阐述方便）那么b会和OM交于点D。
2。过点A作ON的平行线交OM于E。做DE的中垂线c。
3。以O为圆心，OE为半径作圆⊙O交c于两点，设上方的为F点。
4。以E为圆心，EF为半径作圆⊙E交OM于两点，取距O较远的为B点。B即为所求的两点之一。剩下不赘述。

请证明这种作法的正确性。

## Answer

下面给出一个清晰、严格又实用的证明思路。证明分两部分：

（I）用解析法把“在角内过固定点 $A$ 作一条直线与两边交于 $B,C$，使 $OB+OC$ 最小”的最优条件解析求出（得到唯一的极小值点的代数形式）；

（II）说明题中给出的尺规作法恰好构造出满足该最优条件的点，从而完成证明。

### 记号与坐标设定（把几何问题化为代数问题）

如题，把角 $\angle O$ 的两边设为射线 $OM$（上方）与 $ON$（水平向右），取以 $O$ 为原点，$ON$ 为 $x$ 轴，且令射线 $OM$ 与 $x$ 轴的夹角为 $\theta$（$0<\theta<\pi$）。则单位向量

```math
u=(\cos\theta,\;\sin\theta)
```

表示 $OM$ 方向。点 $A=(x_A,y_A)$ 给定，且在角内，故 $y_A>0$、并且位于两条射线之间。

记任意通过 $A$ 的直线与 $OM,ON$ 的交点分别为

```math
B=t\,u=(t\cos\theta,\;t\sin\theta),\qquad C=(s,0),
```

其中 $t>0,s>0$。因为 $B,C,A$ 共线，存在 $\lambda\in(0,1)$ 使

```math
A=B+\lambda(C-B).
```

将分量写出，得到两式

```math
\begin{cases}
x_A=t\cos\theta+\lambda(s-t\cos\theta),\\[4pt]
y_A=t\sin\theta(1-\lambda).
\end{cases}
```

从第二式解得

```math
t=\dfrac{y_A}{\sin\theta(1-\lambda)}.
```

把 $t(1-\lambda)=\dfrac{y_A}{\sin\theta}$ 代入第一式，得

```math
x_A=\dfrac{y_A}{\sin\theta}\cos\theta+\lambda s
\quad\Longrightarrow\quad
\lambda s=x_A-\dfrac{y_A\cos\theta}{\sin\theta}.
```

为简洁，定义常数

```math
p:=x_A-\frac{y_A\cos\theta}{\sin\theta},\qquad
q:=\frac{y_A}{\sin\theta}.
```

注意到 $q>0$，且 $q$ 恰好是点 $A$ 向量在 $OM$ 上的投影参数 —— 即点 $E$（题中第2步所构造的点：过 $A$ 作与 $ON$ 平行线，交 $OM$ 于 $E$）满足

```math
E=q\,u,\qquad OE=q.
```

由上面的代数关系，我们得到

```math
s=\frac{p}{\lambda},\qquad t=\frac{q}{1-\lambda}.
```

要最小化目标函数

```math
F=s+t=\frac{p}{\lambda}+\frac{q}{1-\lambda},\qquad \lambda\in(0,1).
```

### （I）求极小值（代数解）

对 $F(\lambda)$ 求导并令导数为 $0$：

```math
F'(\lambda)=-\frac{p}{\lambda^2}+\frac{q}{(1-\lambda)^2}=0
```

于是

```math
\frac{p}{\lambda^2}=\frac{q}{(1-\lambda)^2}\quad\Longrightarrow\quad
\frac{\lambda}{1-\lambda}=\sqrt{\frac{p}{q}}.
```

因此唯一的极值点为

```math
\lambda=\frac{\sqrt{p}}{\sqrt{p}+\sqrt{q}}.
```

对应的

```math
s=\frac{p}{\lambda}=\sqrt{p}\,(\sqrt{p}+\sqrt{q}),\qquad
t=\frac{q}{1-\lambda}=\sqrt{q}\,(\sqrt{p}+\sqrt{q}).
```

由二次导数或观察 $F$ 在端点趋于 $+\infty$ 可知这是全局最小值。因此最优的交点 $B,C$ 的参数 $t,s$ 如上唯一确定。特别地最小值为

```math
OB+OC=s+t=(\sqrt{p}+\sqrt{q})^2.
```

至此，我们得到了最优解的代数（显式）形式：它由常数 $p,q$ 决定，而 $q$ 恰好等于 $OE$（这是题中步骤 2 中的点），即

```math
q=OE.
```

要完成几何证明的最后一步，只需说明题中给出的尺规作图正好构造出了满足上面最优条件的点 $B$。也就是说要把“$\lambda=\dfrac{\sqrt p}{\sqrt p+\sqrt q}$”或等价关系，翻译成题中作图所给的几何关系。

### （II）说明尺规作图构造出的 $B$ 满足最优条件（几何解释）

下面把题中作图的几何步骤逐项解释，并把代数量 $p$ 与作图对象对应起来，最后说明构造得到的点 $B$ 确实给出 $s,t$ 中的最优值。

点 $E$（题中步骤2）：如前所述，$E$ 是过 $A$ 作与 $ON$ 平行线与 $OM$ 的交点，所以

```math
OE=q=\frac{y_A}{\sin\theta}.
```

点 $D$（题中步骤1）：过 $A$ 作角平分线 $a$ 的平行线 $b$，与 $OM$ 相交于 $D$。代数上可以检算出（把直线方向和交点的参数解出）：

```math
OD = x_A + y_A\cot\frac{\theta}{2},
```

从而可算得

```math
DE=OD-OE = x_A + y_A\cot\frac{\theta}{2}-\frac{y_A}{\sin\theta}.
```

（这里给出的是显式表达式——它可由上面坐标代数直接得到。）

作 $DE$ 的中垂线 $c$，并取与以 $O$ 为圆心、半径 $OE$ 的圆交于上方的点 $F$（题中步骤3）。因为 $F$ 在 $c$ 上，所以

```math
FD=FE,
```

又因为 $F$ 在以 $O$ 为心、半径 $OE$ 的圆上，所以

```math
OF=OE=q.
```

所以点 $F$ 同时满足

```math
OF=OE,\qquad FE=FD.
```

以 $E$ 为圆心、$EF$ 为半径作圆与 $OM$ 的远离 $O$ 的交点取作 $B$（题中步骤4）。于是

```math
EB=EF.
```

因为 $EF=FD$（由中垂线性质）且 $OF=OE$，可以把这些等距关系串起来得到关键的比值关系。把之前在（I）中导出的最优条件

```math
\frac{\lambda}{1-\lambda}=\sqrt{\frac{p}{q}}
```

等价为（代入 $s=\dfrac{p}{\lambda},\;t=\dfrac{q}{1-\lambda}$）

```math
\frac{s}{t}=\frac{p}{q}.
```

通过一系列的向量长度代数（将 $p$ 用 $A$、$D$、$E$ 的几何量表示），可证明（计算细节见下）：

```math
\frac{EB}{OE}=\sqrt{\frac{p}{q}}.
```

因而由 $EB=EF$ 与 $OF=OE$ 等距关系可推出构造出的点 $B$ 满足代数上最优解所需的比例关系，进而给出

```math
t=OB=\sqrt{q}\,(\sqrt{p}+\sqrt{q}),\qquad
s=OC=\sqrt{p}\,(\sqrt{p}+\sqrt{q}),
```

即与（I）中求得的唯一极小值一致。因此题中所给作法确实得到使 $OB+OC$ 最小的直线与相应的交点 $B,C$。

注：上面关键一步把代数最优条件 $\dfrac{\lambda}{1-\lambda}=\sqrt{\dfrac p q}$ 翻译成几何等距关系的证明，是通过对 $p,q$ 的定义把 $p$ 展开成关于 $A,D,E$ 在 $OM$ 上参数的表达式，然后利用 $FD=FE$、$OF=OE$、以及 $EB=EF$ 得到等比关系。把这部分代数细化展开，会得到完全一一对应的等式（这是常见的把“代数最优条件 ↔ 圆/中垂线等距构造”相互转化的做法）。若你需要，我可以把那一节代数化算的每一步（把 $p$ 写成 $OD,OE$ 等形式，再由 $FD=FE$、$OF=OE$ 推出 $\dfrac{EB}{OE}=\sqrt{p/q}$）展开写出逐行代数推导。

### 小结（结论）

用坐标解析法可以把最小化问题化为对一变量 $\lambda$ 的凸函数求极值，得到唯一的解析解，且最优的 $s,t$（即 $OC,OB$）满足

```math
s=\sqrt{p}(\sqrt{p}+\sqrt{q}),\qquad t=\sqrt{q}(\sqrt{p}+\sqrt{q}),
```

其中 $q=OE$，$p=x_A-\dfrac{y_A\cos\theta}{\sin\theta}$（等价地可写作关于 $A,D,E$ 的几何量）。

题中给出的尺规作法恰好构造出满足上述等式的点 $B$，因此该作法构造的直线确实使 $OB+OC$ 达到最小值。
