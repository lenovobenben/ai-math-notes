# 凸多面体边数证明

- Conversation ID: `69342d19-d9bc-8328-9836-b0796e4c5bfe`

## Question

证明：每一个凸多面体，都至少有两个面的边数是相等的。

## Answer

证明如下（反证法）：

设凸多面体有 $V$ 个顶点、$E$ 条棱、$F$ 个面。若所有面的边数两两不同，则这些面的边数至少为

```math
3,4,5,\dots,F+2
```

（因为每个面至少是三边形，且有 $F$ 个不同的正整数）。因此所有面边数之和至少为

```math
3+4+\cdots+(F+2)=\frac{F(3+(F+2))}{2}=\frac{F(F+5)}{2}.
```

另一方面，按“把每条棱计入它相邻的两个面”的计数法，面边数之和恰好等于 $2E$。于是

(1)

```math
2E\ge \frac{F(F+5)}{2}\quad\Rightarrow\quad E\ge\frac{F(F+5)}{4}.
```

再利用平面图（或多面体）的一般不等式。每个顶点的度数至少为 3，所以

```math
2E\ge 3V \quad\Rightarrow\quad V\le\frac{2E}{3}.
```

由欧拉公式 $V-E+F=2$ 得

```math
\frac{2E}{3}-E+F\ge 2 \quad\Rightarrow\quad -\frac{E}{3}+F\ge 2
```

即

(2)

```math
E\le 3F-6.
```

将 (1) 与 (2) 联立得到

```math
\frac{F(F+5)}{4}\le E\le 3F-6,
```

即

```math
F(F+5)\le 12F-24\quad\Rightarrow\quad F^2-7F+24\le 0.
```

但二次式 $F^2-7F+24$ 的判别式为 $49-96=-47<0$，所以该二次式对任意整数 $F$ 都为正，以上不等式不可能成立。这与“所有面边数两两不同”的假设矛盾。

因此假设不成立，结论成立：任一凸多面体中，必有至少两个面的边数相等。▢
