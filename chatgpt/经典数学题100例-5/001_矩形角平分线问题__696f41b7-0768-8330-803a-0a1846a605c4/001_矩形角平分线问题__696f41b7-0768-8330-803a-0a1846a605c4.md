# 矩形角平分线问题

- Conversation ID: `696f41b7-0768-8330-803a-0a1846a605c4`

## Question

对于任意矩形ABCD，BC上有一点E，且AE平分\angle BAC。O为矩形的中心点，连接OE。\frac{OE}{AE}是否存在最小值？如果存在，这个值是多少？

## Answer

存在最小值。下面给出简洁推导和结果。

取坐标系：令 $A=(0,0),\;B=(1,0),\;C=(1,k),\;D=(0,k)$（矩形可按比例缩放，比例不影响比值），矩形中心 $O=(\tfrac12,\tfrac{k}{2})$。由角平分线向量法可得 $AE$ 与 $x=1$（即 $BC$）的交点

```math
E=(1,\;y),\qquad y=\frac{k}{1+\sqrt{1+k^2}}.
```

记

```math
AE=\sqrt{1+y^2},\qquad OE=\sqrt{\Big(\tfrac12\Big)^2+(y-\tfrac{k}{2})^2},
```

要极小化的量为

```math
r=\frac{OE}{AE}=\sqrt{\frac{\tfrac14+(y-\tfrac{k}{2})^2}{1+y^2}}.
```

将 $y=\dfrac{k}{1+\sqrt{1+k^2}}$ 代入，化简并对 $k>0$ 求极值。解一阶条件得到

```math
k^2=\frac54\quad\Rightarrow\quad k=\frac{\sqrt5}{2}.
```

此时

```math
r_{\min}=\frac{\sqrt{14}}{8}\approx0.4677071733.
```

结论：存在最小值，最小值为 $\boxed{\dfrac{\sqrt{14}}{8}}$（约 $0.46770717$），当且仅当矩形的高宽比为 $h/w=\dfrac{\sqrt5}{2}$ 时取得。
