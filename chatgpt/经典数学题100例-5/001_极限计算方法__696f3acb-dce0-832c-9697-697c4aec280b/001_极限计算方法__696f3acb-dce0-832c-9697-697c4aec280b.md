# 极限计算方法

- Conversation ID: `696f3acb-dce0-832c-9697-697c4aec280b`

## Question

求极限 \lim_{x \to e} \frac{e^x-x^e}{x^x-e^e}

## Answer

我们求

```math
\lim_{x\to e}\frac{e^x-x^e}{x^x-e^e}.
```

方法一（洛必达法）：设分子 $f(x)=e^x-x^e$，分母 $g(x)=x^x-e^e$。有 $f(e)=g(e)=0$，可用洛必达：

```math
f'(x)=e^x-e x^{e-1},\qquad g'(x)=x^x(\ln x+1).
```

代入 $x=e$ 得

```math
f'(e)=e^e-e\cdot e^{\,e-1}=e^e-e^e=0,\qquad g'(e)=e^e(1+1)=2e^e\neq0.
```

因此

```math
\lim_{x\to e}\frac{e^x-x^e}{x^x-e^e}=\lim_{x\to e}\frac{f'(x)}{g'(x)}=\frac{0}{2e^e}=0.
```

（也可用泰勒展开：分子首阶项相消，阶为 $(x-e)^2$，分母首阶不消，为 $(x-e)$，所以比值趋于 $0$。）

答案：$\boxed{0}$。
