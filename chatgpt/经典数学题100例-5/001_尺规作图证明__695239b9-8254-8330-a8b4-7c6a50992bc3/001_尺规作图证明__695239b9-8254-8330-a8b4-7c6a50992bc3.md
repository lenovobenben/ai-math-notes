# 尺规作图证明

- Conversation ID: `695239b9-8254-8330-a8b4-7c6a50992bc3`

## Question

尺规作图问题。已知有一正方形ABCD。内部有一点P。过P求作ABCD的内接正方形（P点的位置确保有解）。

这里给出一种做法。
首先假设A,B,C,D四点中，D距离P最近。
1。连接AC,BD交于E。
2。以P为圆心，PE为半径画圆⊙P。
3。过D做BD的垂线交⊙P于两点，取其中一点为F（另一个交点F'可得到另一个解）。
4。作EF的中垂线交AD和CD分别为M,N。
5。M,N即为目标正方形的两个顶点。剩下的不赘述。

请证明这种作法的正确性。

## Answer

下面给出一个坐标化且严谨的证明（算术每步都写明），把几何构造转化为坐标运算后可直接验证每一条性质。

设定坐标系（等价变换）
不妨把正方形 $ABCD$ 放在平面上使得

```math
D=(0,0),\; C=(1,0),\; B=(1,1),\; A=(0,1).
```

于是对角线交点（大正方形中心）为

```math
E=(\tfrac12,\tfrac12).
```

设给定内部点为 $P=(p_x,p_y)$。题中已假设 $D$ 是离 $P$ 最近的顶点，所以

```math
p_x^2+p_y^2\le p_x^2+(p_y-1)^2\quad\Rightarrow\quad p_y\le\tfrac12,
```

以及类似地 $p_x\le\tfrac12$。因此 $0\le p_x,p_y\le \tfrac12$。

按构造求出 $F$、$M$、$N$ 的坐标（代数运算）

对角线 $BD$ 的直线方程是 $y=x$。与之垂直并过 $D$ 的直线是 $y=-x$。因此构造步骤中取的点 $F$ 在直线 $y=-x$ 上，可设

```math
F=(t,-t)
```

（某个实数 $t$）。

$F$ 是以 $P$ 为圆心、以 $PE$ 为半径的圆与直线 $y=-x$ 的交点之一。于是 $F$ 满足

```math
|F-P|^2=|E-P|^2.
```

代入坐标：

```math
(t-p_x)^2+(-t-p_y)^2=(\tfrac12-p_x)^2+(\tfrac12-p_y)^2.
```

化简可得（略去中间代数步骤）关于 $t$ 的方程，解得某个实数 $t$。（不需要显式写出 $t$ 的解析式，后面的代数仅用到 $t$ 的存在性与其代入结果。）

作 $EF$ 的中垂线并与边 $AD$（即 $x=0$）与边 $CD$（即 $y=0$）相交，得到点 $M$ 和 $N$。计算可得（将中点与垂直斜率代入解交点）：

```math
M=\bigl(0,\; \tfrac12 - t\bigr),\qquad N=\bigl(\tfrac12 + t,\; 0\bigr).
```

（代数化简细节：中点 $G=\bigl(\tfrac{1}{2}(0.5+t),\tfrac{1}{2}(0.5-t)\bigr)$，中垂线斜率为 $\dfrac{t-0.5}{t+0.5}$，代入 $x=0$ 与 $y=0$ 得到上面结果；计算可检验无误。）

注意：由于 $p_x,p_y\le\tfrac12$ 可推出交点 $F$ 对应的 $t$ 满足 $-\tfrac12<t<\tfrac12$，于是

```math
0\le \tfrac12-t\le1,\qquad 0\le \tfrac12+t\le1,
```

即 $M$ 在段 $AD$ 上，$N$ 在段 $CD$ 上 —— 与构造一致（也正好体现了“取 D 为最近顶点”以保证解在边段上）。

验证 $M,N$ 与另外两个点构成内接正方形

令另外两个边上的点定义为

```math
M'=\bigl(1,\; \tfrac12 + t\bigr)\quad(\text{在 }BC),\qquad N'=\bigl(\tfrac12 - t,\; 1\bigr)\quad(\text{在 }AB).
```

我们将验证四点 $M\,(0,\tfrac12 - t),\; N(\tfrac12 + t,0),\; M'(1,\tfrac12 + t),\; N'(\tfrac12 - t,1)$ 正好为一个正方形的四个顶点，且四个顶点各落在 $AD,CD,BC,AB$ 上（即内接于大正方形）。

计算向量：

```math
\overrightarrow{MN}=\bigl(\tfrac12+t,\; t-\tfrac12\bigr),\qquad
\overrightarrow{NP'}=\bigl(\tfrac12 - t,\; \tfrac12 + t\bigr).
```

它们的点积为

```math
(\tfrac12+t)(\tfrac12 - t)+(t-\tfrac12)(\tfrac12 + t)=\tfrac14-t^2+\tfrac14-t^2- \tfrac14 +\tfrac14 =0,
```

（按项收敛可见结果为 $0$），所以 $\overrightarrow{MN}\perp\overrightarrow{NP'}$。

再计算模的平方：

```math
|\overrightarrow{MN}|^2=(\tfrac12+t)^2+(t-\tfrac12)^2=2t^2+\tfrac12,
```

```math
|\overrightarrow{NP'}|^2=(\tfrac12-t)^2+(\tfrac12+t)^2=2t^2+\tfrac12,
```

两者相等，说明相邻两边长度相等且垂直。因此 $M,N,P',N'$ 构成一个正方形。另外检查它们确实分别位于四条边上（坐标的第一分量或第二分量为 $0$ 或 $1$），所以这是落在 $ABCD$ 四边上的一个内接正方形。

证明构造中 $P$ 在所得正方形的一条边上（“过 P”）

由构造，$M$ 与 $N$ 是 $EF$ 的中垂线与 $AD,CD$ 的交点。因为我们取的 $E$ 与 $F$ 都在以 $P$ 为圆心的同一圆上（圆心为 $P$，半径 $PE$），所以 $PE=PF$，从而 $P$ 必在线段 $MN$（即 $EF$ 的中垂线上）。因此 $P$ 位于正方形的边 $MN$ 上 —— 正好满足“过 $P$ 作内接正方形”的要求（这里理解为使该正方形的一条边经过点 $P$）。

总结与注记

我们把构造转为坐标，显式求出 $M=(0,\tfrac12-t)$ 与 $N=(\tfrac12+t,0)$，并由此写出其余两顶点，直接验证了四点为正方形且各自位于 $AD,CD,BC,AB$ 上，因此构造正确。

中垂线一定通过 $P$（因为 $PE=PF$），所以构造得到的正方形确实“过 $P$”（在 $MN$ 边上）。

选择交点 $F'$ （另一交点）会给出关于 $t$ 的另一个解，对应同样方法得到另一个内接正方形 —— 这与题中备注一致。

“D 为最近顶点”的条件保证了 $t\in(-\tfrac12,\tfrac12)$，从而四个构造出的顶点落在四条边的线段上（而不是延长线上），即构造在 $ABCD$ 内有效。
