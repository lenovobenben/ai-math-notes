# 尺规作图证明

- Conversation ID: `69450073-f448-8320-9231-5196c6c41222`

## Question

尺规作图问题。已知一个凸四边形ABCD。求作出其作为均匀薄板的质心（注意：不是四个顶点的质心）。

这里给出一种作法。
1。连接AC,BD相交于E。分别作AC,BD的中垂线相交于F。AC,BD的中点分别为G,H。
2。以F为圆心FE为半径作圆⊙F。分别交AC,BD于I,J。
3。连接HI和GJ交于K。K即为所求。

请证明作法的正确性。

## Answer

下面给出一个清晰、严格的证明（把题中构造与向量/坐标表示结合，步骤尽量直观可验证）。

设定与符号
把对角线 $AC$ 与 $BD$ 的交点设为原点 $E$。取单位向量 $\mathbf u$ 沿着线 $AC$（方向由 $E$ 指向 $C$），取单位向量 $\mathbf v$ 沿着线 $BD$（方向由 $E$ 指向 $B$）。设

```math
A=-\alpha\,\mathbf u,\quad C=\beta\,\mathbf u\quad(\alpha,\beta\gt{}0),
```

```math
B=s\,\mathbf v,\quad D=-t\,\mathbf v\quad(s,t\gt{}0).
```

（这样 $AE=\alpha,\;EC=\beta,\;BE=s,\;ED=t$。）

中点

```math
G=\frac{A+C}{2}=\frac{\beta-\alpha}{2}\,\mathbf u,\qquad
H=\frac{B+D}{2}=\frac{s-t}{2}\,\mathbf v.
```

题中点 $F$ 是 $AC$ 与 $BD$ 的中垂线的交点，所以 $F\!-\!E=\mathbf f$ 满足

```math
\mathbf f\cdot\mathbf u = G\cdot\mathbf u=\frac{\beta-\alpha}{2},\qquad
\mathbf f\cdot\mathbf v = H\cdot\mathbf v=\frac{s-t}{2}.
```

（因为 $F$ 在通过 $G$ 的垂线，因而 $(F-G)\perp\mathbf u$，从而 $\mathbf f\cdot\mathbf u=G\cdot\mathbf u$；对 $\mathbf v$ 同理。）

按构造，取以 $F$ 为圆心、$FE$ 为半径的圆，圆与 $AC$ 的另一个交点记为 $I$，与 $BD$ 的另一个交点记为 $J$。由圆的等距方程（用向量表示）可得在以 $E$ 为原点、沿 $\mathbf u,\mathbf v$ 展开时：

$I$ 在 $AC$ 上且满足 $|F-I|=|F-E|$。把 $I$ 写成 $I = \tau\,\mathbf u$，代入得（计算略）$\tau = 2(\mathbf f\cdot\mathbf u)$。因此

```math
I = 2(\mathbf f\cdot\mathbf u)\,\mathbf u = 2\cdot\frac{\beta-\alpha}{2}\,\mathbf u = (\beta-\alpha)\,\mathbf u.
```

（更直接：$E$ 与 $I$ 在同一条直线上，且 $EI:EG = 2:1$ 的关系可由代数解出；上面用内积写法直接给出坐标。）

同理 $J$ 在 $BD$ 上，写成 $J=\sigma\,\mathbf v$，得到

```math
J=2(\mathbf f\cdot\mathbf v)\,\mathbf v = 2\cdot\frac{s-t}{2}\,\mathbf v = (s-t)\,\mathbf v.
```

（注意：上面得到的 $I=(\beta-\alpha)\mathbf u$ 与我们起初把 $A=-\alpha\mathbf u,\;C=\beta\mathbf u$ 一致 —— 即相对于 $E$ 来说 $I$ 恰是 $A$ 与 $C$ 位置坐标之和：$I=A+C$。同理 $J=B+D$。这一步可以通过解等距方程直接得到。）

计算薄板质心 $K$
把四边形以对角线 $AC$ 分成两三角形 $\triangle ABC$ 与 $\triangle ADC$。它们的面积比等于在 $BD$ 上对点 $E$ 的划分比：

```math
\frac{[ABC]}{[ADC]}=\frac{BE}{ED}=\frac{s}{t},
```

因为两三角形有相同底边 $AC$ 而高成比例（高与 $BE$ 或 $ED$ 成正比）。

三角形质心分别为

```math
U=\frac{A+B+C}{3}=\frac{-\alpha+\beta}{3}\,\mathbf u+\frac{s}{3}\,\mathbf v,
```

```math
V=\frac{A+C+D}{3}=\frac{-\alpha+\beta}{3}\,\mathbf u-\frac{t}{3}\,\mathbf v.
```

整体薄板（两三角形并起）的质心按面积加权为

```math
K=\frac{sU+tV}{s+t}.
```

代入并化简（逐项计算）：

$\mathbf u$ 分量：$\dfrac{s(-\alpha+\beta)+t(-\alpha+\beta)}{3(s+t)}=\dfrac{-\alpha+\beta}{3}$.

$\mathbf v$ 分量：$\dfrac{s\cdot\frac{s}{3}+t\cdot\left(-\frac{t}{3}\right)}{s+t}=\dfrac{s^2-t^2}{3(s+t)}=\dfrac{s-t}{3}$.

所以

```math
K=\frac{\beta-\alpha}{3}\,\mathbf u+\frac{s-t}{3}\,\mathbf v .
```

把 $G,H,I,J$ 写出（相对于原点 $E$）：

```math
G=\frac{\beta-\alpha}{2}\,\mathbf u,\qquad H=\frac{s-t}{2}\,\mathbf v,
```

```math
I=(\beta-\alpha)\,\mathbf u,\qquad J=(s-t)\,\mathbf v.
```

于是可以直接验证

```math
K-G=\frac{1}{3}(J-G),\qquad K-H=\frac{1}{3}(I-H).
```

确实，用上面的坐标：

```math
K-G=\Big(\frac{\beta-\alpha}{3}-\frac{\beta-\alpha}{2}\Big)\mathbf u+\frac{s-t}{3}\mathbf v
       =\frac{1}{3}\Big(-\frac{\beta-\alpha}{2}\mathbf u+ (s-t)\mathbf v\Big)
       =\frac{1}{3}(J-G),
```

```math
K-H=\frac{\beta-\alpha}{3}\mathbf u+\Big(\frac{s-t}{3}-\frac{s-t}{2}\Big)\mathbf v
       =\frac{1}{3}\Big((\beta-\alpha)\mathbf u-\frac{s-t}{2}\mathbf v\Big)
       =\frac{1}{3}(I-H).
```

因此 $K$ 同时在直线 $GJ$ 与直线 $HI$ 上，且分别满足从 $G$ 到 $J$ 的三等分位置（在 $G$ 到 $J$ 段上距 $G$ 长的 $1/3$）与从 $H$ 到 $I$ 的三等分位置（在 $H$ 到 $I$ 段上距 $H$ 长的 $1/3$）。所以构造的交点 $K=GJ\cap HI$ 正好等于薄板质心。

结论：按照题中构造得到的点 $K$ 等于四边形作为均匀薄板的质心。证明的要点是（1）用对角线把四边形分成两个三角形，质心是两个三角形质心按面积加权的点；（2）圆心 $F$ 在两条中垂线上的条件推出 $I=A+C$（关于 $E$ 的坐标）与 $J=B+D$ 的简洁表示；（3）代数化简后得到 $K$ 在 $GJ$ 与 $HI$ 上且恰为两段的 $1/3$ 分点，从而与构造交点一致。
