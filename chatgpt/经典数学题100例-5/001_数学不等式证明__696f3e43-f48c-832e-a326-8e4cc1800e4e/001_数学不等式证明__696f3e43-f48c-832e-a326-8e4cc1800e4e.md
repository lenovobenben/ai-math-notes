# 数学不等式证明

- Conversation ID: `696f3e43-f48c-832e-a326-8e4cc1800e4e`

## Question

已知实数a,b,c满足|a|<|b-c|，|b|<|c-a|。请证明|c|>|a-b|

## Answer

证明如下。

由题设有

```math
|a|\lt{}|b-c|,\qquad |b|\lt{}|c-a|.
```

两式同乘方得

```math
a^2\lt{}(b-c)^2,\qquad b^2\lt{}(c-a)^2.
```

把两式相加：

```math
a^2+b^2\lt{}(b-c)^2+(c-a)^2
= a^2+b^2+2c^2-2c(a+b).
```

两边同时减去 $a^2+b^2$，得到

```math
0\lt{}2c^2-2c(a+b)=2c\bigl(c-(a+b)\bigr),
```

即

(1)

```math
c\bigl(c-(a+b)\bigr)\gt{}0.
```

再用第一式减去第二式：

```math
a^2-b^2\lt{}(b-c)^2-(c-a)^2.
```

右端化简得

```math
(b-c)^2-(c-a)^2=(a-b)\bigl(2c-(a+b)\bigr),
```

因此

```math
(a-b)(a+b-c)\lt{}0,
```

等价于

(2)

```math
(a-b)\bigl(c-(a+b)\bigr)\gt{}0.
```

由 (1) 和 (2) 可得

```math
\bigl(c-(a-b)\bigr)\bigl(c-(a+b)\bigr)=c\bigl(c-(a+b)\bigr)-(a-b)\bigl(c-(a+b)\bigr)\gt{}0,
```

以及

```math
\bigl(c+(a-b)\bigr)\bigl(c-(a+b)\bigr)=c\bigl(c-(a+b)\bigr)+(a-b)\bigl(c-(a+b)\bigr)\gt{}0.
```

因为 $c-(a+b)\neq0$（若等于 0 会使 (1) 成立不可能），由上两式可知 $c-(a-b)$ 与 $c+(a-b)$ 与 $c-(a+b)$ 同号，因而两者之积为正：

```math
\bigl(c-(a-b)\bigr)\bigl(c+(a-b)\bigr)\gt{}0.
```

即

```math
c^2-(a-b)^2\gt{}0,
```

所以

```math
|c|\gt{}|a-b|.
```

证毕。
