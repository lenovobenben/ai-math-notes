# 凸n边形证明

- Conversation ID: `6934129f-e924-832e-ace5-78da39b16635`

## Question

平面内n个小乌龟，从各自的初始位置以恒定相同的速度沿着各自不同的方向爬行。证明：存在某个时刻使得n个小乌龟的位置点必成凸n边形。

## Answer

证明思路：把时间 $t$ 拉大并做一次缩放，把运动的位置序列归一化到速度（单位向量）附近，利用单位圆上的速度向量们本身处于凸位置且这是个开性质，从而得到存在足够大的时刻使得所有乌龟的位置也处于凸位置。

下面给出详细证明（中文）：

设第 $i$ 只乌龟的初始位置为向量 $a_i\in\mathbb R^2$，运动方向为单位向量 $v_i$（题中速度相等，故只需写方向单位向量），且假设这些方向两两不同： $v_i\neq v_j$（$i\neq j$）。则第 $i$ 只乌龟在时间 $t$ 的位置为

```math
p_i(t)=a_i+t v_i.
```

引入归一化位置

```math
q_i(t)=\frac{p_i(t)}{t}=v_i+\frac{a_i}{t}\qquad(t\gt{}0).
```

当 $t\to\infty$ 时显然

```math
q_i(t)\to v_i.
```

注意到 $v_1,\dots,v_n$ 都是单位向量且两两不同，它们位于单位圆上。单位圆是严格凸的，所以这 $n$ 个点必然处于“凸位置”——即每个 $v_i$ 都是点集 $\{v_1,\dots,v_n\}$ 的极点（位于凸包的顶点），并且不存在三个 $v_i$ 共线。且“处于凸位置”是对点位置的开条件：因为每个顶点 $v_i$ 在其处存在一条支撑直线（或支持半平面）仅与该顶点相切，不触及其它点；对有限集合可以取这些支撑直线与其它点之间的最小正距离 $\delta>0$。因此存在 $\varepsilon>0$ 使得任取集合 $\{r_i\}$ 满足 $|r_i-v_i|<\varepsilon$（对所有 $i$），则 $\{r_i\}$ 仍然处于凸位置（即所有点仍为凸包的顶点）。

因为 $q_i(t)\to v_i$，所以存在 $T>0$ 使得当 $t>T$ 时对所有 $i$ 都有 $|q_i(t)-v_i|<\varepsilon$。因此当 $t>T$ 时，点集 $\{q_1(t),\dots,q_n(t)\}$ 处于凸位置，进而把这些点缩放回去（乘以正数 $t$）得到 $\{p_1(t),\dots,p_n(t)\}$ 也处于凸位置（正同胚保持凸性与极点性质）。换言之，存在某时刻 $t>T$ 使得 $n$ 个乌龟的位置正好是一个凸 $n$-边形的 $n$ 个顶点。

这完成证明。 ∎
