# 尺规作图证明-相切圆

- Conversation ID: `693a1f75-dfd4-8323-92c3-c2e6d75ff8df`

## Question

尺规作图问题。有一个直角∠A，在内部有一点P，求作两个大小相同的圆C1和C2，C1,C2相切于P，且C1和直角的一边相切，C2和直角的另一边相切。

我这里给出一种作法。
先设∠A的两条边，一条是水平向右的AX，另一条是向上的AY。且P点到AX比AY更近一点。
1。以P为圆心，PA为半径作圆⊙P交AX于B，交AY于C。
2。以A为圆心，AC为半径作圆⊙A交AX于D，交⊙P于E。连接ED延长交⊙P于F点。连接CD。
3。以B为圆心，BF为半径作圆⊙B交CD于G。
4。以G为圆心，GP为半径作圆即为所求的圆之一。延长GP作另一个圆即可。

请证明作法的正确性。

## Answer

好——你的作法是对的。下面给出一个清晰的解析（坐标化）证明，尽量把每一步写得简洁且能看出构造的几何意义。

坐标设定（把直角放在原点）
设直角顶点 $A=(0,0)$，边 $AX$ 为 $x$-轴正方向，边 $AY$ 为 $y$-轴正方向。设内部点 $P=(u,v)$（$u>0,v>0$），且按你的备注 $P$ 到 $AX$ 比到 $AY$ 近一点不用特别用到，只需 $u,v>0$。

按你的构造：

以 $P$ 为圆心、半径 $PA=\sqrt{u^2+v^2}$ 的圆与坐标轴相交得

```math
B=(2u,0),\qquad C=(0,2v).
```

（计算：在 $y=0$ 代入得 $x( x-2u)=0$，非零解为 $x=2u$，类似得 $C$。）

以 $A$（原点）为心、半径 $AC=2v$ 的圆与 $x$-轴交点为

```math
D=(2v,0).
```

把圆 $(x- u)^2+(y-v)^2=u^2+v^2$（以 $P$ 为心、半径 $PA$）和圆 $x^2+y^2=(2v)^2$ 相交，易得它们的另一个交点（除去 $C=(0,2v)$）为

```math
E=\Big(\frac{4uv^2}{u^2+v^2},\ \frac{2v(v^2-u^2)}{u^2+v^2}\Big).
```

（此处可以通过联立两圆方程消去二次项得到 $ux+vy=2v^2$，再解出具体坐标。）

连接 $E$ 与 $D=(2v,0)$，将这条直线与以 $P$ 为心的圆再作第二交点，记为 $F$。用参数消去法（或解联立）可得到一个简洁的结果：

```math
F=(u+v,\;u+v).
```

（代入检验：点 $(u+v,u+v)$ 满足 $(x-u)^2+(y-v)^2=u^2+v^2$ 且在直线 $DE$ 的另一交点上——代数运算可直接验证。）

以 $B=(2u,0)$ 为心、半径 $BF$ 作圆，交线 $CD$（注意 $C=(0,2v),D=(2v,0)$，所以 $CD$ 的方程是 $x+y=2v$）于点 $G$。解联立方程

```math
(x-2u)^2+y^2=BF^2,\qquad x+y=2v,
```

可求出 $G$ 的两个解（对应两个交点）。取位于角内的那个解（构作时自然选择在角内的交点），可写为

```math
G=\Big(u+v-\sqrt{2uv},\;v-u+\sqrt{2uv}\Big)
```

（另一个解在几何上不在角内，丢弃）。

关键等式（代数恒等）——验证 $GP$ 与 $G$ 到 $y$-轴（即到直角的另一边 $AY$）的距离相等：
直接计算可得代数恒等

```math
GP^2=(G_x-u)^2+(G_y-v)^2=G_x^2,
```

因而（取正平方根）

```math
GP=G_x.
```

这里 $G_x$ 就是点 $G$ 的横坐标，等于它到 $y$-轴的距离。因此圆心为 $G$、半径 $GP$ 的圆恰好经过 $P$ 且与 $AY$ 相切（切点在 $x=0$ 的垂足处），正好符合“一个圆与直角的一条边相切且与另一个圆相切于 $P$”的要求。

最后一步“延长 $GP$ 作另一个圆”的含义是：在直线 $GP$ 上把 $G$ 关于点 $P$ 作对称点 $G'$（即 $G'$ 是以 $P$ 为中点的 $G$ 的反射点），那么

```math
G' = 2P - G,
```

且显然 $G'P=GP$。于是以 $G'$ 为心、半径 $GP$ 的圆与以 $G$ 为心的圆半径相等、在 $P$ 处相切。因为 $G$ 的半径等于其到 $y$-轴的距离，反射过去的 $G'$ 的半径等于其到 $x$-轴的距离，所以这第二个圆正好与 $AX$ 相切。于是两圆分别与两条直角边相切，并在 $P$ 处互相相切，且半径相等。

##### 几何意义小结

你构造出的点 $B,C,D,F$ 都有简单的坐标表达（尤其 $B=(2u,0),\ C=(0,2v),\ D=(2v,0),\ F=(u+v,u+v)$），这让后续的代数关系很干净。

交点 $G$ 满足 $GP=G_x$，即圆心到 $y$-轴的距离等于半径，所以圆与 $AY$ 相切。反射得到的另一个中心与 $AX$ 相切。

因此你的构法确实得到一对等圆，分别切于两条直角边并在 $P$ 处互相相切。
