# 方程正实解求解

- Conversation ID: `696f2ce4-e218-8332-b4cf-df50fb61598a`

## Question

求方程： x^{\ln_{}{3}} + x^{\ln_{}{4}} = x^{\ln_{}{5}} 的正实数解。

## Answer

令 $t=\ln x$（$x>0$），则

```math
x^{\ln 3}+x^{\ln 4}=x^{\ln 5}
```

可改写为

```math
3^{t}+4^{t}=5^{t}.
```

两边同时除以 $5^{t}$，得

```math
\left(\frac{3}{5}\right)^{t}+\left(\frac{4}{5}\right)^{t}=1.
```

设 $q(t)=\left(\frac{3}{5}\right)^{t}+\left(\frac{4}{5}\right)^{t}$。注意到

```math
q'(t)=\left(\frac{3}{5}\right)^{t}\ln\frac{3}{5}+\left(\frac{4}{5}\right)^{t}\ln\frac{4}{5}\lt{}0,
```

因此 $q(t)$ 对 $t$ 单调递减。又 $q(2)=\frac{9}{25}+\frac{16}{25}=1$，故方程有且只有一解 $t=2$。

于是 $\ln x=2$，得 $x=e^{2}$。

答：唯一的正实数解是 $\boxed{e^{2}}$。
