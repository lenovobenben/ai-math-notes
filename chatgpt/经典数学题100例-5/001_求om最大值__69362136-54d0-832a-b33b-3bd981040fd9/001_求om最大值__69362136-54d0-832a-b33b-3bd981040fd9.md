# 求OM最大值

- Conversation ID: `69362136-54d0-832a-b33b-3bd981040fd9`

## Question

平面内有一个正方形ABCD，O为其中心点。在BC的外侧有一点M，MC=1,MB=3,求OM_{max}。

## Answer

把正方形中心 $O$ 设在坐标原点，设半边长为 $r>0$。取顶点为

```math
A(-r,r),\; B(r,r),\; C(r,-r),\; D(-r,-r),
```

于是边 $BC$ 对应直线 $x=r$，在 $BC$ 外侧的点满足 $x>r$。设所求点为 $M=(x,y)$，由题给条件

```math
MC=1,\qquad MB=3,
```

即

```math
(x-r)^2+(y+ r)^2=1^2,\qquad (x-r)^2+(y-r)^2=3^2.
```

两式相减得

```math
(y-r)^2-(y+r)^2=9-1=8\ \Rightarrow\ -4ry=8\ \Rightarrow\ y=-\frac{2}{r}.
```

把 $y$ 代回任一圆方程（取与 $C$ 的方程）得

```math
(x-r)^2+\Big(r-\frac{2}{r}\Big)^2=1.
```

整理得到

```math
(x-r)^2=\frac{(r^2-1)(4-r^2)}{r^2},
```

因此需有 $(r^2-1)(4-r^2)\ge0$，即 $1\le r^2\le4$（即 $r\in[1,2]$）。且 $M$ 在 $x>r$ 侧，取正号：

```math
x=r+\sqrt{\frac{(r^2-1)(4-r^2)}{r^2}}.
```

于是

```math
OM^2=x^2+y^2
= \Big(r+\sqrt{\frac{(r^2-1)(4-r^2)}{r^2}}\Big)^2+\frac{4}{r^2}.
```

设 $t=r^2$（$t\in[1,4]$），代入并化简可得一个简洁表达：

```math
OM^2=5+2\sqrt{(t-1)(4-t)}.
```

要使 $OM$ 最大，需使 $\sqrt{(t-1)(4-t)}$ 最大；在区间 $[1,4]$ 上该表达在中点 $t=\dfrac{1+4}{2}=2.5$ 处取最大值，值为 $\sqrt{(2.5-1)(4-2.5)}=\sqrt{1.5\cdot1.5}=1.5$。因此

```math
OM_{\max}^2=5+2\cdot1.5=8\quad\Rightarrow\quad OM_{\max}=\sqrt{8}=2\sqrt2.
```

答案：$\displaystyle OM_{\max}=2\sqrt2$。
