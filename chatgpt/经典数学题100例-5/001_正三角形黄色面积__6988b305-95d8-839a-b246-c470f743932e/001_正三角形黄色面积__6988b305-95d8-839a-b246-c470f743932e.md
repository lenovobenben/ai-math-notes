# 正三角形黄色面积

- Conversation ID: `6988b305-95d8-839a-b246-c470f743932e`

## Question

如图，已知一个正三角形的面积为6，现由4个相同的正三角形连成一排，求图中黄色部分的面积。

## Answer

设每个正三角形边长为 $s$，高为 $h=\frac{\sqrt3}{2}s$，则单个正三角形面积

```math
S_{\triangle}=\frac12 sh=6.
```

把四个三角形的底边放在 $x$-轴上：第1个底边为 $[0,s]$，第2个为 $[s,2s]$，第3个为 $[2s,3s]$，第4个为 $[3s,4s]$。第4个的顶点为

```math
P_4\left(\frac{7s}{2},\,h\right),
```

斜线从 $(0,0)$ 连到 $P_4$，所以其方程为

```math
y=\frac{h}{7s/2}\,x=\frac{2h}{7s}x.
```

下面分别求这条直线在第1、2、3个三角形内“上方”部分（黄色）的面积。

#### ① 第1个三角形

它的右边边是从 $(s,0)$ 到 $(\frac{s}{2},h)$。设斜线与这条边交于点 $Q_1$，令

```math
Q_1=(s,0)+u\Big(\big(\tfrac{s}{2},h\big)-(s,0)\Big)=\Big(s-\tfrac{u}{2}s,\,uh\Big).
```

代入斜线方程 $uh=\frac{2h}{7s}\big(s-\frac{u}{2}s\big)$，约去 $h$ 得

```math
u=\frac{2}{7}\left(1-\frac{u}{2}\right)\ \Rightarrow\ u=\frac14.
```

所以 $Q_1$ 的高度是 $h/4$。斜线下方在第1个三角形里正好是三角形 $\triangle (0,0)(s,0)Q_1$，其面积为

```math
\frac12\cdot s\cdot \frac{h}{4}=\frac14\cdot\frac12 sh=\frac14 S_{\triangle}.
```

故黄色面积

```math
S_1=S_{\triangle}-\frac14S_{\triangle}=\frac34S_{\triangle}.
```

#### ② 第2个三角形

顶点 $P_2=(\frac{3s}{2},h)$，左右边分别是 $(s,0)\to P_2$、$(2s,0)\to P_2$。

左边交点 $L_2$：设

```math
L_2=(s,0)+u\Big(P_2-(s,0)\Big)=\Big(s+\tfrac{u}{2}s,\,uh\Big).
```

代入斜线：$u=\frac{2}{7}\left(1+\frac{u}{2}\right)\Rightarrow u=\frac13$。

右边交点 $R_2$：设

```math
R_2=(2s,0)+v\Big(P_2-(2s,0)\Big)=\Big(2s-\tfrac{v}{2}s,\,vh\Big).
```

代入斜线：$v=\frac{2}{7}\left(2-\frac{v}{2}\right)\Rightarrow v=\frac12$。

黄色部分就是顶端三角形 $\triangle P_2L_2R_2$。用坐标行列式算面积（或直接记结论）可得

```math
S_2=\frac{sh}{6}=\frac13\cdot\frac12 sh=\frac13 S_{\triangle}.
```

#### ③ 第3个三角形

同理可算（过程完全一样）：

左边交点参数 $u=\frac23$，右边交点参数 $v=\frac34$，顶端黄色三角形面积为

```math
S_3=\frac{sh}{24}=\frac{1}{12}\cdot\frac12 sh=\frac{1}{12}S_{\triangle}.
```

#### ④ 总黄色面积

```math
S_{\text{yellow}}=S_1+S_2+S_3
=\left(\frac34+\frac13+\frac1{12}\right)S_{\triangle}
=\frac{9+4+1}{12}S_{\triangle}
=\frac{14}{12}\cdot 6=7.
```

答案：黄色部分总面积 $=7$。
