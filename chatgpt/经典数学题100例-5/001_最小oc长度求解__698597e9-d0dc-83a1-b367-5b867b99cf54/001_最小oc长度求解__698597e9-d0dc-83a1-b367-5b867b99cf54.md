# 最小OC长度求解

- Conversation ID: `698597e9-d0dc-83a1-b367-5b867b99cf54`

## Question

点 $A$ 在直线 $y = -x - 6$ 上运动，点 $B$ 在以 $M(-3, 3)$ 为圆心、$1$ 为半径的圆上运动。以 $AB$ 为底边作等腰直角 $\triangle ABC$ ($A$、$B$、$C$ 按逆时针顺序排列)，连接 $OC$，求线段 $OC$ 的最小值。

## Answer

设 $O$ 为原点。因“以 $AB$ 为底边作等腰直角 $\triangle ABC$”且 $A,B,C$ 逆时针，说明 $\angle ACB=90^\circ,\ AC=BC$，即 $AB$ 为斜边，点 $C$ 由向量 $\overrightarrow{AB}$ 旋转 $+90^\circ$ 得到。

令

```math
A=(t,-t-6)\quad (t\in\mathbb R),\qquad B=(x,y)
```

且 $B$ 在圆 $(x+3)^2+(y-3)^2=1$ 上。

对等腰直角三角形（斜边为 $AB$）有坐标公式：

```math
C=\frac{A+B}{2}+\frac{R_{90^\circ}(B-A)}{2},\quad R_{90^\circ}(u,v)=(-v,u).
```

于是

```math
C=\frac{1}{2}\bigl(A+B+(- (y-(-t-6)),\ x-t)\bigr)
=\left(\frac{x-y-6}{2},\ \frac{x+y-2t-6}{2}\right).
```

关键观察：$x_C=\dfrac{x-y-6}{2}$ 与 $t$ 无关，而 $y_C$ 随 $t$ 可取任意实数。

因此对固定的 $B$，点 $C$ 在一条竖直直线

```math
x=\frac{x-y-6}{2}
```

上运动，故

```math
\min_{t\in\mathbb R} |OC|=\left|\frac{x-y-6}{2}\right|
```

（取 $t=\frac{x+y-6}{2}$ 使 $y_C=0$ 即可达到）。

所以问题化为只对 $B$ 取最小：

```math
\min |OC|=\frac12\min_{(x+3)^2+(y-3)^2=1} |x-y-6|.
```

令 $B=M+(\cos\theta,\sin\theta)$，其中 $M(-3,3)$。则

```math
x-y-6 = (-3+\cos\theta)-(3+\sin\theta)-6
= -12+\cos\theta-\sin\theta.
```

而 $\cos\theta-\sin\theta\in[-\sqrt2,\sqrt2]$，所以

```math
| -12+\cos\theta-\sin\theta|\ \text{的最小值} = 12-\sqrt2
```

（取 $\cos\theta-\sin\theta=\sqrt2$ 即可）。

故

```math
\boxed{\min |OC|=\frac{12-\sqrt2}{2}=6-\frac{\sqrt2}{2}.}
```
