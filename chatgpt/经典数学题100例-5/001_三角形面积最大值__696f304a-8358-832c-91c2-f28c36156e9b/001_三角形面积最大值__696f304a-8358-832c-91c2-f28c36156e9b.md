# 三角形面积最大值

- Conversation ID: `696f304a-8358-832c-91c2-f28c36156e9b`

## Question

有\triangle ABC，D,E分别是BC,AC中点。AD=1,BE=2。求\triangle ABC面积的最大值。

## Answer

把点放坐标系里最直接。

设 $A=(0,0)$， $C=(c,0)$， $B=(x,y)$.
则 $E$ 为 $AC$ 中点： $E=(\tfrac c2,0)$， $D$ 为 $BC$ 中点： $D=(\tfrac{x+c}{2},\tfrac y2)$。

由条件

```math
BE^2=(x-\tfrac c2)^2+y^2=4,
\qquad
AD^2=(\tfrac{x+c}{2})^2+(\tfrac y2)^2=1,
```

即

```math
(x-\tfrac c2)^2+y^2=4,\qquad (x+c)^2+y^2=4.
```

两式相减得

```math
(x+c)^2-(x-\tfrac c2)^2=0\implies 3cx+\tfrac34 c^2=0\implies x=-\tfrac c4.
```

代回得

```math
y^2=4-\Big(\tfrac{3c}{4}\Big)^2=4-\tfrac{9}{16}c^2.
```

三角形面积

```math
S=\tfrac12\cdot AC\cdot\text{height}=\tfrac12\cdot c\cdot|y|=\tfrac12\,c\sqrt{4-\tfrac{9}{16}c^2}.
```

为求最大值，令 $S^2=c^2-\tfrac{9}{64}c^4$。对 $c>0$ 求导：

```math
\frac{d}{dc}S^2=2c-\tfrac{9}{16}c^3=2c\Big(1-\tfrac{9}{32}c^2\Big)=0
```

得非零解 $c^2=\tfrac{32}{9}$，即 $c=\tfrac{4\sqrt2}{3}$。此时

```math
y^2=4-\tfrac{9}{16}\cdot\tfrac{32}{9}=4-2=2\implies |y|=\sqrt2,
```

所以最大面积

```math
S_{\max}=\tfrac12\cdot \tfrac{4\sqrt2}{3}\cdot\sqrt2=\tfrac{4}{3}.
```

达到等号时可取例如

```math
A=(0,0),\quad C=\Big(\tfrac{4\sqrt2}{3},0\Big),\quad B=\Big(-\tfrac{\sqrt2}{3},\ \sqrt2\Big),
```

检验满足 $AD=1,\,BE=2$。

答：三角形面积的最大值为 $\boxed{\tfrac{4}{3}}$。
