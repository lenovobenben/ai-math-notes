# 无斐波那契等差数列

- Conversation ID: `6936e79b-a1f4-832b-818c-5b51e3db79e4`

## Question

是否存在不包括任何Fibonacci numbers的无穷递增等差数列？

## Answer

存在。给出一个简单的构造与证明。

考虑斐波那契数列对模 $8$ 的剩余。前几项：

```math
F_0=0,\ F_1=1,\ F_2=1,\ F_3=2,\ F_4=3,\ F_5=5,\ F_6=8\equiv0,\ F_7=13\equiv5,\ F_8=21\equiv5,\ F_9=34\equiv2,\ F_{10}=55\equiv7,\ F_{11}=89\equiv1,
```

并且 $F_{12}\equiv0,\ F_{13}\equiv1$，于是模 $8$ 的周期为 $12$（Pisano 周期），在这个周期内出现的残余仅为

```math
\{0,1,2,3,5,7\}.
```

注意模 $8$ 下没有出现 $4$（也没有出现 $6$）。

因此任取以 $4$ 为模 $8$ 同余类的等差数列（例如

```math
4,\,12,\,20,\,28,\dots
\quad\text{即}\quad a_n=4+8n,\ n=0,1,2,\dots
```

），其每一项都恒为 $4\pmod8$。而斐波那契数对 $8$ 的剩余从不为 $4$，所以该等差数列中不会出现任何斐波那契数。由此得到了一个不包含任何斐波那契数的无穷递增等差数列。

（同样的方法还能构造出许多此类等差数列：只需找到一个模 $m$ 和一个在斐波那契数模 $m$ 值域外的剩余类，再取该同余类组成的等差数列即可。）
