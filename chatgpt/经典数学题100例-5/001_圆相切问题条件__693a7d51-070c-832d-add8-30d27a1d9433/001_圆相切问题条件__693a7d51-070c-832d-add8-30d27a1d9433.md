# 圆相切问题条件

- Conversation ID: `693a7d51-070c-832d-add8-30d27a1d9433`

## Question

在直角坐标系的第一象限内部（不能在坐标轴上）有一点P，求作两个大小相同的圆C1和C2，C1,C2相切于P，且C1和x正半轴相切于M，C2和y正半轴相切于N。我们来看是否可作的问题。当P在某些位置时，是无法作出两个圆的。比如(100,1)这个点不可以。请问可作的充要条件是什么？

注意：圆C1,C2可以部分超出第一象限。

## Answer

把点记为 $P=(x,y)$（第一象限内，$x>0,y>0$）。设两个半径相等的圆为 $C_1,C_2$，半径为 $r>0$。由题意：

$C_1$ 与正 $x$ 轴相切，说明其圆心形如 $A=(a,r)$，且切点为 $M=(a,0)$，要求 $a>0$。

$C_2$ 与正 $y$ 轴相切，说明其圆心形如 $B=(r,b)$，且切点为 $N=(0,b)$，要求 $b>0$。

两圆在 $P$ 处相切。对于半径相等且外切的两圆，相切点正好是两圆心的中点，所以

```math
P=\frac{A+B}{2}=\Big(\frac{a+r}{2},\;\frac{r+b}{2}\Big).
```

由此得

```math
a=2x-r,\qquad b=2y-r.
```

两圆外切且半径为 $r$ 等价于两心距为 $2r$，代入上式等价于

```math
(x-r)^2+(r-y)^2=r^2,
```

整理为关于 $r$ 的二次方程

```math
r^2-2(x+y)r+(x^2+y^2)=0,
```

解得两个根

```math
r=(x+y)\pm\sqrt{2xy}.
```

较小的根

```math
r_-=x+y-\sqrt{2xy}
```

是我们感兴趣的（另一根更大，不能同时满足切点在正半轴上的正坐标条件）。

还要保证切点在正半轴上，即

```math
a=2x-r\gt{}0,\qquad b=2y-r\gt{}0,
```

即 $r<2x$ 且 $r<2y$。把 $r=r_-$ 代入并化简，得到必要且充分的约束（两不等式合并后可写成一个等价的简单形式）：

```math
|x-y|\lt{}\sqrt{2xy}.
```

等价写法有

```math
(x-y)^2\lt{}2xy\quad\Longleftrightarrow\quad x^2+y^2\lt{}4xy.
```

因此结论：
存在且仅存在 两个所求相同大小的圆（且切点 $M,N$ 在正半轴的正部分）当且仅当

```math
(x-y)^2\lt{}2xy\quad(\text{即 }x^2+y^2\lt{}4xy).
```

当等号成立时（$(x-y)^2=2xy$）会导致切点位于原点（不属于正半轴的正部分），故按题要求应视为不可作。

补充：若条件满足，则可取半径

```math
r=x+y-\sqrt{2xy},
```

圆心分别为 $A=(2x-r,r)$ 和 $B=(r,2y-r)$。

举例：$P=(100,1)$ 时 $100^2+1^2=10001\ge 4\cdot100\cdot1=400$，不满足条件，故不可作。而 $P=(2,1)$ 满足条件（可作，半径为 $r=1$）。

## Question

正确答案是  2 - \sqrt{3} < \frac{p_x}{p_y} < 2 + \sqrt{3}

## Answer

你给出的式子是正确的。下面把我先前写的条件 $(x-y)^2<2xy$ 代数化简成关于比值 $t=\dfrac{x}{y}$ 的等价形式并解出界。

设 $t=\dfrac{x}{y}>0$。不等式

```math
(x-y)^2\lt{}2xy
```

两边除以 $y^2>0$ 得

```math
\Big(\frac{x}{y}-1\Big)^2\lt{}2\frac{x}{y},
```

即

```math
(t-1)^2\lt{}2t.
```

展开并整理：

```math
t^2-2t+1\lt{}2t \quad\Longrightarrow\quad t^2-4t+1\lt{}0.
```

解方程 $t^2-4t+1=0$：

```math
t=\frac{4\pm\sqrt{16-4}}{2}=\frac{4\pm\sqrt{12}}{2}=\frac{4\pm 2\sqrt3}{2}=2\pm\sqrt3.
```

二次式开口向上，小于零的解集正是两个根之间，所以

```math
2-\sqrt3\lt{}t\lt{}2+\sqrt3.
```

数值上 $2-\sqrt3\approx0.2679,\;2+\sqrt3\approx3.7321$。因此等价的充分必要条件就是

```math
2-\sqrt3\lt{}\frac{p_x}{p_y}\lt{}2+\sqrt3,
```

与您给出的结果一致。例如 $P=(100,1)$ 时 $\dfrac{100}{1}=100>2+\sqrt3$，故不可作。

## Question

如果圆C1,C2不能超出第一象限，则范围是[1/2,2]

## Answer

对的 — 如果还要求两个圆不得越出第一象限（可以与坐标轴相切，但圆的任何点 $x$ 或 $y$ 坐标都不能为负），充要条件就是

```math
\frac{1}{2}\le\frac{p_x}{p_y}\le 2.
```

推导（简短）：
记 $P=(x,y)\,(x>0,y>0)$，仍取小的根

```math
r=x+y-\sqrt{2xy}
```

为圆的半径。要使两个圆都不越过第一象限，等价于
两圆分别相比左边最小横坐标和最小纵坐标不小于 $0$，即
对 $C_1$ 要求 $a-r\ge0$，代入 $a=2x-r$ 得 $x\ge r$；
对 $C_2$ 要求 $b-r\ge0$，代入 $b=2y-r$ 得 $y\ge r$。

以 $x\ge r$ 为例：

```math
x\ge x+y-\sqrt{2xy}\ \Longrightarrow\ y\le\sqrt{2xy}\ \Longrightarrow\ y^2\le2xy\ \Longrightarrow\ \frac{y}{x}\le2,
```

即 $\dfrac{x}{y}\ge \tfrac12$。同理由 $y\ge r$ 得 $\dfrac{x}{y}\le2$。合并得 $\tfrac12\le x/y\le2$。

端点也允许（等号时某一圆关于对应轴正好“贴到”轴的非负边界，但不越出第一象限），因此可作的比值范围为闭区间 $[1/2,2]$。
