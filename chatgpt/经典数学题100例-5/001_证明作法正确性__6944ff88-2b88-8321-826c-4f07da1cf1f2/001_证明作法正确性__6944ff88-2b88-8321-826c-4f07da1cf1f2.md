# 证明作法正确性

- Conversation ID: `6944ff88-2b88-8321-826c-4f07da1cf1f2`

## Question

尺规作图问题。平面上有一定角A，角A内部有一定点P，问如何过点P作一直线交角A于B,C，使△ABC的面积最小？

我这里给一种作法。
假设∠A的两条边分别为AM,AN。
1。连接AP并延长至A'，使得PA'=AP。
2。过A'作AM的平行线交AN于B。过A'作AN的平行线交AM于C。
3。连接BC即所求。

请证明此作法的正确性。

## Answer

很好的问题，你的作法是对的。下面给出一个简洁而严格的证明（用向量/坐标语言，会更清楚）。

### 设定与符号

设角 $A$ 的两边沿射线 $AM$ 和 $AN$，令两单位方向向量为 $\mathbf u$（沿 $AM$）和 $\mathbf v$（沿 $AN$），它们线性无关，且夹角为 $\theta=\angle A$。把点 $A$ 作为原点。
任意在 $AM$ 上的点可表示为 $\alpha\mathbf u$（$\alpha>0$），在 $AN$ 上的点可表示为 $\beta\mathbf v$（$\beta>0$）。设给定点 $P$ 在角内部，则可以唯一表示为

```math
\mathbf p = a\mathbf u + b\mathbf v,\qquad a\gt{}0,\; b\gt{}0.
```

我们考虑过 $P$ 的一条直线与两边的交点记为 $C=\alpha\mathbf u$（在 $AM$ 上）和 $B=\beta\mathbf v$（在 $AN$ 上）。三角形 $ABC$ 的面积为

```math
S(\alpha,\beta)=\tfrac12\;|AC|\;|AB|\;\sin\theta=\tfrac12\,\alpha\beta\sin\theta.
```

要在所有通过 $P$ 并与两边相交的直线上取使面积最小的那一条。

### 通过 $P$ 的直线与参数关系

点 $P$ 在线段 $BC$ 上，故存在参数 $t\in(0,1)$ 使

```math
\mathbf p=(1-t)\,C+t\,B=(1-t)\alpha\mathbf u + t\beta\mathbf v.
```

比较 $\mathbf u,\mathbf v$ 的分量，得到

```math
\alpha(1-t)=a,\qquad \beta t=b.
```

因此

```math
\alpha=\frac{a}{1-t},\qquad \beta=\frac{b}{t}.
```

代入面积表达式，得关于参数 $t$ 的面积

```math
S(t)=\tfrac12\sin\theta\;\alpha\beta
=\tfrac12\sin\theta\;\frac{a}{1-t}\cdot\frac{b}{t}
=\frac{ab\sin\theta}{2}\cdot\frac{1}{t(1-t)}.
```

### 极值分析

函数 $f(t)=\frac{1}{t(1-t)}$ 在 $t\in(0,1)$ 上是对称的，且容易计算得导数可知其在 $t=\tfrac12$ 取得最小值（也可用不等式 $t(1-t)\le\tfrac14$ 得到最小值 $\frac{1}{t(1-t)}\ge4$ ，等号当且仅当 $t=\tfrac12$ 时成立）。因此

```math
S(t)\ \text{最小时}\ \Longleftrightarrow\ t=\tfrac12.
```

当 $t=\tfrac12$ 时：

```math
\alpha= \frac{a}{1/2}=2a,\qquad \beta=\frac{b}{1/2}=2b.
```

于是

```math
C=2a\,\mathbf u,\quad B=2b\,\mathbf v,\quad A'=\alpha\mathbf u+\beta\mathbf v=2(a\mathbf u+b\mathbf v)=2\mathbf p.
```

注意 $A'$ 就是沿 $AP$ 向外延长，使得 $PA'=AP$（因为 $\mathbf a'=2\mathbf p$ 意味着 $P$ 是 $AA'$ 的中点）。且由计算可见当 $t=\tfrac12$ 时 $P$ 恰为线段 $BC$ 的中点（把参数代入线段表达式即可），即 $P$ 为 $BC$ 的中点。

### 与你的作法等价

你的作法是：沿 $AP$ 延长到 $A'$ 使 $PA'=AP$（即 $\mathbf a'=2\mathbf p$），过 $A'$ 作与 $AM$ 平行的直线交 $AN$ 于 $B$，过 $A'$ 作与 $AN$ 平行的直线交 $AM$ 于 $C$。在向量展式里，这正给出

```math
A'=\alpha\mathbf u+\beta\mathbf v,\quad B=\beta\mathbf v,\quad C=\alpha\mathbf u,
```

且由求交条件得出的 $\alpha=2a,\ \beta=2b$。这正是上面导出的唯一使 $t=\tfrac12$ 的 $\alpha,\beta$。因此你的构造恰好给出了使面积最小的 $B,C$。

### 结论

以上证明表明：在角 $A$ 内给定点 $P$，过 $P$ 作直线交角的两边于 $B,C$ 使得三角形 $ABC$ 面积最小，当且仅当 $P$ 为线段 $BC$ 的中点。你给出的作法（沿 $AP$ 延长取 $A'$ 使 $PA'=AP$，然后通过 $A'$ 分别作与两边平行求 $B,C$）正是构造出这样一个 $BC$，因此方法正确且唯一。
