# C_4子图证明

- Conversation ID: `696f2aaf-5474-8332-b77f-ce40c225c50d`

## Question

设n阶无向简单图G=(V,E)满足\sum_{v \in V} \binom{d(v)}{2} > \binom{n}{2} ，其中对每个v \in V，d(v)为v的度。
证明：G必含子图C_4（4个顶点的圈）。

## Answer

好——用“数计数 + 鸽巢原理”可以很短地证明。

对每个顶点 $v$，$\binom{d(v)}{2}$ 是以 $v$ 为中心的“夹角”数（即一对不同邻居 $\{x,y\}$），所以

```math
\sum_{v\in V}\binom{d(v)}{2}
```

等于所有无序顶点对 $\{x,y\}$ 被共同邻居计数的总和。把这个计数按顶点对 $\{x,y\}$ 重新分组，记

```math
c_{xy}=|N(x)\cap N(y)|
```

为顶点对 $\{x,y\}$ 的公共邻居数，则

```math
\sum_{v\in V}\binom{d(v)}{2}=\sum_{\{x,y\}\subset V} c_{xy}.
```

题设给出左边大于 $\binom{n}{2}$（即所有无序顶点对的数量），于是均值

```math
\frac{1}{\binom{n}{2}}\sum_{\{x,y\}} c_{xy} \gt{}1.
```

因此存在某个顶点对 $\{x,y\}$ 使得 $c_{xy}\ge 2$，即 $x$ 和 $y$ 至少有两个不同的公共邻居，记作 $u$ 和 $v$。

于是在顶点集合 $\{u,v,x,y\}$ 上，存在边 $u x,u y,v x,v y$，这四条边构成一个 $K_{2,2}$（或包含 $K_{2,2}$ 的图）。而 $K_{2,2}$ 中必有环 $u-x-v-y-u$（长度为4），即存在一个 $C_4$。

因此在给定不等式成立时，图 $G$ 必含 $C_4$。◻️
