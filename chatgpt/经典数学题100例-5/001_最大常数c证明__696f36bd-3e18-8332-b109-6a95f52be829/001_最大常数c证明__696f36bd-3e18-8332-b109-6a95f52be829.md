# 最大常数C证明

- Conversation ID: `696f36bd-3e18-8332-b109-6a95f52be829`

## Question

给定正整数n，求最大的实数C，使得对任意实数a_1,a_2,\dots,a_n，都存在实数\beta ，
使得 \left | \prod_{k=1}^{n} \sin(a_k + \beta)  \right | \ge C。

## Answer

结论：最大的常数是

```math
C=2^{\,1-n}.
```

证明（要点清楚、简洁）：

记对任意固定的实数列 $a_1,\dots,a_n$ ，把关于 $\beta$ 的函数

```math
S(\beta)=\prod_{k=1}^n\sin(a_k+\beta)
```

考虑起来。为方便尺度，令

```math
F(\beta)=2^{\,n-1}S(\beta)=2^{\,n-1}\prod_{k=1}^n\sin(a_k+\beta).
```

把每个 $\sin$ 用指数形式展开：

```math
\sin x=\frac{e^{ix}-e^{-ix}}{2i},
```

于是 $F(\beta)$ 是一个频率不超过 $n$ 的三角多项式，其傅里叶展开中 $e^{i n\beta}$ 项的系数可以从展开中直接读出——对应把每个因子都取上标 $+$ 的那一项，得到

```math
\text{coeff of }e^{i n\beta}
=2^{\,n-1}\cdot(2i)^{-n}e^{i\sum_{k=1}^n a_k}
=\tfrac12\,i^{-n}e^{i\sum a_k}.
```

因此其傅里叶系数（按常规归一化）

```math
c_n=\frac{1}{2\pi}\int_0^{2\pi}F(\beta)e^{-in\beta}\,d\beta
```

满足 $|c_n|=\tfrac12$。同理对负频率也有 $|c_{-n}|=\tfrac12$。

由三角不等式与积分估计，

```math
|c_n|+|c_{-n}|
=\frac{1}{2\pi}\Big|\int F e^{-in\beta}\Big|+\frac{1}{2\pi}\Big|\int F e^{in\beta}\Big|
\le\frac{1}{2\pi}\int |F|\,(|e^{-in\beta}|+|e^{in\beta}|)\,d\beta
=\frac{1}{\pi}\int_0^{2\pi}|F(\beta)|\,d\beta.
```

又因为 $\displaystyle\sup_\beta|F(\beta)|\ge\frac{1}{2\pi}\int_0^{2\pi}|F(\beta)|\,d\beta$，由上得

```math
\sup_\beta|F(\beta)|\ge\frac{|c_n|+|c_{-n}|}{2}=\frac{1/2+1/2}{2}=\frac12.
```

于是

```math
\sup_\beta\big|\prod_{k=1}^n\sin(a_k+\beta)\big|
=\frac{1}{2^{\,n-1}}\sup_\beta|F(\beta)|\ge\frac{1}{2^{\,n-1}}\cdot\frac12=2^{\,1-n}.
```

这说明对于任意 $a_1,\dots,a_n$ 总存在某个 $\beta$ 使得 $|\prod\sin(a_k+\beta)|\ge 2^{1-n}$。

最后给出可达性（即该下界是最优的）：取

```math
a_k=\frac{(k-1)\pi}{n}\quad(k=1,\dots,n).
```

利用恒等式（易由复指数展开验证）

```math
\prod_{k=0}^{n-1}\sin\Big(\beta+\frac{k\pi}{n}\Big)=2^{\,1-n}\sin(n\beta),
```

可见对于这组 $a_k$ 有

```math
\sup_\beta\Big|\prod_{k=1}^n\sin(a_k+\beta)\Big|=2^{\,1-n},
```

因此常数 $2^{1-n}$ 无法再增大。于是它就是题目所求的最大的常数。

——证毕。
