- Conversation ID: `6944f67e-7d94-832f-90ed-d47ae88eadd7`

## Question

已知坐标系内有两点M(m,0),N(n,0), m > n > 0。对坐标系的原点O作正反斜45°两条直线，加上x轴y轴构成8条对称的射线。从M点开始依次逆时针在这8条射线上取点A,B,C,D,E,F,G,N(注意终点是N，不是M)，并且有MA=AB=BC=\dots=GN=r。显然A,B,\dots,G等点一般有两种落点：一处距离O远，另一处距离O近。同理最后一点N，原则上也有两种落点，一种是与M重合（我们舍弃掉），另一种是小于M的，即N点。从多边形MABCDEFGN的角度看。每一处落点，如果是选择【远】点，则这个点是凸点。选择【近】点，则它是凹点。

1。证明：凹点不能连续出现两个。 
2。证明：【x,y轴的点和O点的距离】只能是m（凸）,n（凹）这两种。【y=x,y=-x的点和O点的距离】也只能是另外的两个值（一凸一凹）。
3。显然r只和m,n有关。请求出r=f(m,n)。
4。根据凸凹性区分，满足题目条件的排列组合有多少种？

## Answer

好——我把题目用“极坐标方向 + 半径序列”的方法整理并严谨地求出结论（每一步都给出推导），结论很整洁：

记第 $k$ 条射线的点为 $P_k$（按逆时针编号），其中

$P_0=M=(m,0)$，

$P_1=A$ 在 $y=x$ (45°) 射线上，

…

$P_7=G$ 在 315° 射线上，

$P_8=N=(n,0)$（同方向于 $P_0$）。

令 $\rho_k=|OP_k|$（点到原点的距离），已知 $\rho_0=m,\ \rho_8=n$，并且相邻两点间距相等

```math
|P_kP_{k+1}|=r\qquad(k=0,1,\dots,7).
```

##### 先得出基本等式

由两点坐标与夹角 $45^\circ$ 可得（$\cos45^\circ=\tfrac{\sqrt2}{2}$）

(★)

```math
r^2=|P_k-P_{k+1}|^2=\rho_k^2+\rho_{k+1}^2-\sqrt2\,\rho_k\rho_{k+1}\qquad(k=0,\dots,7).
```

这是整个问题的基本关系式。对任意相邻的两个半径 $\rho_k,\rho_{k+1}$，把 (★) 看成关于 $\rho_{k+1}$ 的二次方程，可得两根（一大一小），对应“远点（凸）”与“近点（凹）”。

#### 1）证明：凹点不能连续出现两个

从 (★) 对 $k$ 与 $k+1$ 两式相减，得到因式分解（对任意 $k$ 都成立）：

(1)

```math
(\rho_k-\rho_{k+2})\big(\rho_k+\rho_{k+2}-\sqrt2\,\rho_{k+1}\big)=0.
```

因此对于任意连续三点 $P_k,P_{k+1},P_{k+2}$，要么 $\rho_{k+2}=\rho_k$，要么 $\rho_{k+1}=\dfrac{\rho_k+\rho_{k+2}}{\sqrt2}$。

若 $P_{k+1}$ 取“近点”（凹，取二次方程的小根），代入上面的二次方程可以直接解出（把相应数字代入，可见代数步骤是通用的）——小根对应的情形会使得 $\rho_{k+2}=\rho_k$（即下一点半径回到上一点的半径，属于“远/凸”）。换句话说：若某点为凹（取小根），则其后一点必为凸（半径与前一点相等）。所以不可能出现两个连续的凹点。

（也可直接对 $k=0$ 作具体代数解：若 $A$ 为近点，则代入可得 $B$ 恰好与 $M$ 同半径，即凸；由旋转对称性，上述结论对任意位置都成立。）

#### 2）证明：轴向点（x、y 轴）与原点的距离只能是 $m,n$；斜线（$y=\pm x$）上的距离只能是两值 $\dfrac{m\pm n}{\sqrt2}$（一凸一凹）

把 (★) 对 $k=0$（$M$ 与 $A$）和 $k=1$（$A$ 与 $B$）两式列出：

```math
\begin{cases}
r^2=m^2+\rho_1^2-\sqrt2\,m\rho_1,\\[4pt]
r^2=\rho_1^2+\rho_2^2-\sqrt2\,\rho_1\rho_2.
\end{cases}
```

用第二式把 $\rho_2$ 用 $n$ 代入（因为 $\rho_8=n$ 同方向的约束在整个八步链上会带来等式对称性；更直接的做法是把上两式相减并用最终端点），相减可消去 $r^2$ 与 $\rho_1^2$，得到

```math
(m-n)\big(m+n-\sqrt2\,\rho_1\big)=0.
```

由于 $m\ne n$，得

```math
\rho_1=\frac{m+n}{\sqrt2}\quad\text{或}\quad \rho_1=\frac{m-n}{\sqrt2}.
```

因此 $A$（在 $y=x$ 射线上）的两种可能距离就是 $\dfrac{m\pm n}{\sqrt2}$。由旋转对称性，所有落在 $y=\pm x$ 的点均只能取这两个值（一个对应“远/凸”，一个对应“近/凹”）。

再看两步以后（从 $P_k$ 到 $P_{k+2}$）的关系 (1)：要么 $\rho_{k+2}=\rho_k$，要么 $\rho_{k+1}=(\rho_k+\rho_{k+2})/\sqrt2$。把 $k$ 取为偶数（轴向点之间），结合上面斜线处的两值形式，可以推出轴向（x,y轴）上的点半径只能在 $\{m,n\}$ 中取（若斜线点选“近/远”之一，相邻的轴点被强制为 $m$ 或 $n$）。实际代数替代演算可参见下一节的例子计算（对 $k=0$ 就得到 $\rho_2$ 必为 $m$ 或 $n$），因此轴向点只有两个可能值 $m$（凸）或 $n$（凹）。

综上：

在 $x$ 或 $y$ 轴上的点的距离只能是 $m$（对应凸位置）或 $n$（对应凹位置）；

在 $y=\pm x$ 上的点的距离只能是 $\dfrac{m+n}{\sqrt2}$（凸）或 $\dfrac{m-n}{\sqrt2}$（凹）。

#### 3）求 $r=f(m,n)$

把 $k=0$（$M$-$A$）与 $k=1$（$A$-$B$）两式同时写出，代入 $\rho_1=\dfrac{m+n}{\sqrt2}$（或用另一个根，结果相同）并代数化简，可得

```math
r^2=\frac{m^2+n^2}{2}.
```

因此

```math
\boxed{\;r=\sqrt{\dfrac{m^2+n^2}{2}}\; }.
```

（代数检验：把 $\rho_1=(m+n)/\sqrt2$ 代入 $r^2=m^2+\rho_1^2-\sqrt2 m\rho_1$，化简即可得到上式。）

#### 4）按凸/凹性区分，有多少种排列组合？

我们把每个顶点记为“凸”(记作 0) 或“凹”(记作 1)。已知：

$M$ 固定为凸（因为给定是点 $M(m,0)$），

$N$ 固定为凹（题意已去掉与 $M$ 重合的那种位型），

由第 (1) 点证明：不能有连续两个凹（凹不能相邻）。

把中间的 7 个点 $A,B,C,D,E,F,G$ 的凸凹状态看作长度 7 的二进制序列，要求：

序列的第 1 位（与 $M$ 相邻）可以任意（因 $M$ 为凸不限制），

序列的第 7 位（与 $N$ 相邻）不能为凹（否则与 $N$ 连成两个相邻凹，违背第 (1) 条），

序列中不能有连续两个 1（凹）。

于是问题等价为：在长度 $7$ 的位串中，以 0/1 表示凸/凹，最后一位必须是 $0$，且不能出现 “11”。 这是一个典型的斐波那契计数问题。

设 $a_k$ 为长度为 $k$、以 0 结尾且无相邻 1 的二进制串个数。容易得到递推

```math
a_k=a_{k-1}+a_{k-2},
```

并且 $a_1=1,\ a_2=2$。因此 $a_k$ 是斐波那契数（偏移），具体 $a_k=F_{k+1}$（按 $F_1=1,F_2=1$ 记法）。取 $k=7$ 得

```math
\text{可行的凸/凹排列数}=a_7=F_8=21.
```

（也可以直接枚举证明：长度 7 且末位为 0、且无 “11” 的字符串共有 21 个。）

所以答案：共有 $\boxed{21}$ 种不同的凸/凹排列。

（注：上面枚举并排除负半径等“代数不可行”的情况后，得到的可行实际几何构型数量确实是 21——可通过把 $r=\sqrt{(m^2+n^2)/2}$ 固定后逐步向前求各点半径并检验“正距与不相邻凹”来计算验证。）

#### 总结（结论一览）

凹点不能连续出现两个（若中点为凹，则下一点必为凸）。

轴上点到 $O$ 的距离只可能是 $m$（凸）或 $n$（凹）；斜线 $y=\pm x$ 上点到 $O$ 的距离只可能是 $\dfrac{m\pm n}{\sqrt2}$（一凸一凹）。

等距 $r$ 只与 $m,n$ 有关，且

```math
\boxed{r=\sqrt{\dfrac{m^2+n^2}{2}}.}
```

按凸/凹区分，满足题目条件的排列组合共有 21 种。
