# 求解符合条件的三元组

- Conversation ID: `6936ea39-e46c-832f-995d-fa82d47284b7`

## Question

已知x,y,z \in \mathbb{N},x < y < z,\gcd(x,y) = \gcd(y,z) = \gcd(z,x) = 1,x \mid (y+z), y\mid (x+z), z\mid (x+y)。求符合条件的所有解。

## Answer

答案是唯一的一组：$(x,y,z)=(1,2,3)$。

证明如下（中文）：

设

```math
s=x+y+z
```

由题意有

```math
\frac{y+z}{x},\ \frac{x+z}{y},\ \frac{x+y}{z}\in\mathbb{Z},
```

等价于 $x\mid(y+z),\ y\mid(x+z),\ z\mid(x+y)$。因此

```math
x\mid s,\qquad y\mid s,\qquad z\mid s.
```

又因为 $x,y,z$ 两两互素，所以它们的最小公倍数是 $xyz$，因此

```math
xyz\mid s.
```

于是有不等式

```math
xyz\le s = x+y+z \le 3z,
```

即

```math
xy\le 3.
```

因为 $x<y$ 且 $x,y\in\mathbb{N}$ 且互素，满足的可能只有 $xy=2$（即 $x=1,y=2$）或 $xy=3$（即 $x=1,y=3$）。

若 $x=1,y=2$，则 $xyz=2z$ 必须整除 $s=1+2+z=z+3$。于是 $2z\le z+3$，得 $z\le3$。又 $z>y=2$，故 $z=3$。检验可行：$1\mid(2+3),\ 2\mid(1+3),\ 3\mid(1+2)$。

若 $x=1,y=3$，则 $xyz=3z$ 整除 $s=1+3+z=z+4$，故 $3z\le z+4$，得 $z\le2$，与 $z>y=3$ 矛盾。

因此唯一解为 $(1,2,3)$。 ✔
