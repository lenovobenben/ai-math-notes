# 经典力学两体问题轨道

- Conversation ID: `699fe77a-b544-8332-a4fa-051cbd23c59d`

## Question

请证明：在经典力学中，两个质点在万有引力作用下的运动只有有限几种可能，且全部属于圆锥曲线（及其退化形式）。

## Answer

下面在牛顿经典力学（点质量、只有万有引力、无其他外力与相对论效应）框架内，证明两体问题的所有轨道只有有限几类，并且全部是圆锥曲线（及其退化形式）。

#### 1. 两体问题化为一体问题（约化质量）

两质点质量 $m_1,m_2$，位置 $\mathbf r_1,\mathbf r_2$。万有引力满足

```math
m_1\ddot{\mathbf r}_1=-Gm_1m_2\frac{\mathbf r_1-\mathbf r_2}{\|\mathbf r_1-\mathbf r_2\|^3},\qquad
m_2\ddot{\mathbf r}_2=+Gm_1m_2\frac{\mathbf r_1-\mathbf r_2}{\|\mathbf r_1-\mathbf r_2\|^3}.
```

定义质心

```math
\mathbf R=\frac{m_1\mathbf r_1+m_2\mathbf r_2}{M},\quad M=m_1+m_2,
```

与相对位矢

```math
\mathbf r=\mathbf r_1-\mathbf r_2,\quad r=\|\mathbf r\|.
```

把两式相加得

```math
M\ddot{\mathbf R}=0\ \Rightarrow\ \mathbf R(t)=\mathbf R_0+\mathbf V t
```

（质心做匀速直线运动）。

把两式相减（或直接标准推导）得相对运动满足

```math
\ddot{\mathbf r}=-GM\frac{\mathbf r}{r^3}.
```

等价地，也可写为约化质量 $\mu=\frac{m_1m_2}{M}$ 形式

```math
\mu\ddot{\mathbf r}=-Gm_1m_2\frac{\mathbf r}{r^3},
```

它们本质相同（只是把常数写法不同）。

因此：两体问题 = 质心匀速直线 + 一个粒子在中心力 $\propto 1/r^2$ 下的运动。轨道分类只需研究 $\mathbf r(t)$。

#### 2. 守恒量：角动量与能量

因为力始终沿 $\mathbf r$ 方向，是中心力，故角动量

```math
\mathbf L=\mu\,\mathbf r\times \dot{\mathbf r}
```

守恒（$\dot{\mathbf L}=0$）。因此运动在与 $\mathbf L$ 垂直的一个固定平面内（若 $\mathbf L=\mathbf 0$，则为纯径向直线运动，是退化情形）。

同时能量守恒：

```math
E=\frac12\mu\|\dot{\mathbf r}\|^2-\frac{Gm_1m_2}{r}
```

为常数。

#### 3. 在极坐标中得到轨道方程（Binet 方程）

在运动平面取极坐标 $(r,\theta)$。设

```math
h:=\frac{\|\mathbf L\|}{\mu},
```

则由角动量守恒

```math
\|\mathbf L\|=\mu r^2\dot\theta \ \Rightarrow\ \dot\theta=\frac{h}{r^2}.
```

相对运动的径向方程为（中心力标准形式）

```math
\ddot r-r\dot\theta^2=-\frac{GM}{r^2}.
```

令 $u(\theta)=1/r$。利用 $\dot\theta=hu^2$ 可计算出（这是经典 Binet 变换）

```math
u''+u=\frac{GM}{h^2},
```

其中 $u'=\frac{du}{d\theta}$。

这是常系数线性方程，解为

```math
u(\theta)=\frac{GM}{h^2}\Bigl(1+e\cos(\theta-\theta_0)\Bigr),
```

其中 $e\ge 0,\theta_0$ 为积分常数。

于是轨道满足

```math
r(\theta)=\frac{p}{1+e\cos(\theta-\theta_0)},\qquad p:=\frac{h^2}{GM}\gt{}0.
```

#### 4. 这就是圆锥曲线的极坐标方程

把坐标轴旋转使 $\theta_0=0$（只改变参考方向，不改变曲线类型），得

```math
r=\frac{p}{1+e\cos\theta}.
```

记 $x=r\cos\theta,\ y=r\sin\theta,\ r=\sqrt{x^2+y^2}$，则

```math
r=\frac{p}{1+e\frac{x}{r}}
\ \Longleftrightarrow\
r+ex=p.
```

两边平方：

```math
(x^2+y^2)+2ex\sqrt{x^2+y^2}+e^2x^2=p^2.
```

将 $\sqrt{x^2+y^2}=r=p-ex$ 代回更直接：由 $r=p-ex$ 得

```math
x^2+y^2=(p-ex)^2=p^2-2pex+e^2x^2,
```

整理：

```math
(1-e^2)x^2+y^2-2px+p^2=0,
```

这是一个二次曲线（圆锥曲线的统一代数形式）。并且：

$0\le e<1$：$1-e^2>0$，是椭圆（$e=0$ 时是圆）。

$e=1$：系数退化为抛物线型，是抛物线。

$e>1$：$1-e^2<0$，是双曲线。

所以只要 $\mathbf L\neq 0$，轨道必为某条圆锥曲线，焦点在力心（原点）。

#### 5. “有限几种可能”的分类来自两个守恒量

关键不只是“是圆锥曲线”，还要说明“只有有限几类”。这来自：

角动量是否为 0（$\mathbf L=0$ or $\mathbf L\ne 0$），

能量符号（$E<0,\,E=0,\,E>0$）。

并且偏心率与能量有确定关系（可由能量表达式推得）：

```math
e^2=1+\frac{2Eh^2}{\mu(Gm_1m_2)^2}
\quad\Bigl(\text{或等价写法，常数取决于你用 }GM\text{还是 }Gm_1m_2\Bigr).
```

因此：

##### (A) $\mathbf L\neq 0$（非径向，一般情形）

$E<0 \iff e<1$：椭圆（含圆），有界轨道。

$E=0 \iff e=1$：抛物线，临界逃逸。

$E>0 \iff e>1$：双曲线，散射轨道。

##### (B) $\mathbf L=0$（纯径向运动，退化情形）

此时 $\theta$ 恒定，轨迹在几何上是一条直线（穿过力心），对应退化圆锥曲线：

$E<0$：从某有限 $r$ 处落向 $r=0$（若允许“穿过/反弹”则往返），几何轨迹为线段型退化椭圆。

$E=0$：从无穷远径向落入或恰好逃逸，几何为射线型退化抛物线。

$E>0$：从无穷远来、穿过（或碰撞后）再到无穷远，几何为直线型退化双曲线。

因此，按“能量三类 \times 角动量两类（径向/非径向）”只得到有限种定性类型；在非径向情形中又精确落在椭圆/抛物线/双曲线三大类（圆为椭圆特例），径向情形落在其退化形式。

#### 6. 回到原来的两质点轨道

在质心系（$\mathbf R$ 固定）里

```math
\mathbf r_1=\mathbf R+\frac{m_2}{M}\mathbf r,\qquad
\mathbf r_2=\mathbf R-\frac{m_1}{M}\mathbf r.
```

也就是说，每个质点都沿着与 $\mathbf r(t)$ 相似缩放的圆锥曲线绕质心运动（再叠加质心在一般惯性系中的匀速平移，仍是同一条曲线整体平移）。

结论：在牛顿经典力学的万有引力两体问题中，所有可能的相对轨道由守恒能量与角动量完全定性分类，非径向时必为以力心为焦点的椭圆/抛物线/双曲线（圆为特例），径向时为其退化形式。因此“只有有限几种可能，且全部属于圆锥曲线（及其退化形式）”得证。

## Question

在经典力学的场景下，选取参考系很重要。如果我们以其中一个质点为参考系和以第三方的视角为参考系，那么他们的运动轨迹会有什么区别？

为了使讨论简单。假设是椭圆这种情况。

## Answer

把两体问题写成质心运动 + 相对运动之后，这个问题就非常清楚了。

设两质点位置分别为 $\mathbf r_1(t),\mathbf r_2(t)$，质量 $m_1,m_2$，总质量 $M=m_1+m_2$。

定义

质心：

```math
\displaystyle \mathbf R=\frac{m_1\mathbf r_1+m_2\mathbf r_2}{M}
```

相对位矢（2 相对 1）：

```math
\displaystyle \mathbf r=\mathbf r_2-\mathbf r_1
```

则经典结论是：

质心做匀速直线运动

```math
\ddot{\mathbf R}=0 \quad\Rightarrow\quad \mathbf R(t)=\mathbf R_0+\mathbf V t .
```

相对位矢满足开普勒方程（中心反平方力）

```math
\ddot{\mathbf r}=-GM\frac{\mathbf r}{\|\mathbf r\|^3}.
```

因此相对轨道必是圆锥曲线；你这里假设是椭圆，就意味着 $\mathbf r(t)$ 在某个平面内走一条椭圆。

#### 1）以其中一个质点为参考系（例如“以 1 为原点”）

如果你的参考系的原点随质点 1 平移、轴不随时间转动，那么在这个（非惯性）参考系里，质点 2 的坐标正是

```math
\mathbf r_{(1)}(t)=\mathbf r_2-\mathbf r_1=\mathbf r(t).
```

而 $\mathbf r(t)$ 满足上面的开普勒方程，所以：

质点 2 在此系中的轨迹：一条椭圆

焦点在哪里：焦点在原点，也就是质点 1 的位置

半长轴：这条椭圆的半长轴就是“相对轨道”的 $a$（记为 $a_{\rm rel}$ 也行）

直观上：你“坐在 1 上看 2”，2 就沿着一条以 1 为焦点的椭圆绕。

注意：严格说这个参考系是加速的，所以会出现“惯性力”；但由于我们直接取的是相对矢量 $\mathbf r_2-\mathbf r_1$，两体加速度差恰好把它整理成一个仍然是中心反平方的方程，因此轨道仍是椭圆。

#### 2）以“第三方视角”的惯性参考系看（最典型：质心系）

##### （A）如果第三方选的是质心静止的惯性系（最常用、最干净）

令 $\mathbf V=0$，也就是选择让质心静止：$\mathbf R=\mathbf 0$。则有

```math
m_1\mathbf r_1+m_2\mathbf r_2=0
\quad\Rightarrow\quad
\mathbf r_1=-\frac{m_2}{M}\mathbf r,\qquad
\mathbf r_2=\frac{m_1}{M}\mathbf r.
```

这告诉你：

两个质点都走椭圆；

两条椭圆有相同的偏心率 $e$、同一条长轴方向、同一周期（同步相位）；

但大小不同：若相对椭圆半长轴为 $a$，则

```math
a_1=\frac{m_2}{M}a,\qquad a_2=\frac{m_1}{M}a .
```

这两条椭圆的共同焦点在质心（原点）。

直观上：你站在远处的惯性系里看，会看到“两个物体绕着共同质心跳舞”：质量大的那条椭圆更小，质量小的那条更大。

##### （B）如果第三方是任意惯性系，质心不一定静止

一般惯性系下 $\mathbf R(t)=\mathbf R_0+\mathbf Vt$ 不为常数，于是

```math
\mathbf r_1(t)=\mathbf R(t)-\frac{m_2}{M}\mathbf r(t),\qquad
\mathbf r_2(t)=\mathbf R(t)+\frac{m_1}{M}\mathbf r(t).
```

这意味着：

在该第三方惯性系里，每个质点的运动 = “绕质心的椭圆运动” + “整体匀速漂移 $\mathbf Vt$”。

所以轨迹通常不再是闭合椭圆，而是一条“椭圆被匀速搬运”的曲线（周期性摆动叠加直线漂移）。

直观上：你会看到整套系统一边直线飞走，一边内部做椭圆绕转。

#### 3）一句话对比（你要的“椭圆情况”）

以质点 1 为参考系：质点 2 走一条以质点 1 为焦点的椭圆（闭合）。

以质心静止的第三方惯性系：两个质点分别走以质心为焦点的两条相似椭圆，半长轴按质量比缩放。

以任意第三方惯性系：在上面的基础上再叠加一个整体匀速直线运动，轨迹一般不闭合。

如果你愿意，我也可以把三种参考系下的轨道用同一个参数方程（偏近点角 $E$）写出来，你会看到它们之间就是“缩放 + 平移 + 匀速漂移”的关系。
