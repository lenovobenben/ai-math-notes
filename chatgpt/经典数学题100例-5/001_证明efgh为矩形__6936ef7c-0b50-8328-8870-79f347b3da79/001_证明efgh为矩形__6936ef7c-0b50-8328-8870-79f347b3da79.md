# 证明EFGH为矩形

- Conversation ID: `6936ef7c-0b50-8328-8870-79f347b3da79`

## Question

已知圆内接四边形ABCD，设\triangle ABC，\triangle BCD，\triangle CDA，\triangle DAB的内心分别是E,F,G,H。证明：EFGH是矩形。

## Answer

把四个三角形的内心分别记作

```math
E=I_{ABC},\;F=I_{BCD},\;G=I_{CDA},\;H=I_{DAB}.
```

我们用三角形内心的一个常用角公式：
在三角形 $XYZ$ 中，其内心 $I$ 满足

```math
\angle X I Y = 90^\circ+\frac{\angle Z}{2},
```

也就是说内心看到相对两顶点的角等于 $90^\circ$ 加上第三个顶点角的一半。

下面做角度追赶（angle–chasing）。

先看点列 $B,E,F,C$。
在三角形 $ABC$ 中有

```math
\angle B E C=90^\circ+\frac{\angle A}{2}.
```

在三角形 $BCD$ 中有

```math
\angle B F C=90^\circ+\frac{\angle D}{2}.
```

但因为 $ABCD$ 是内接四边形，$\angle A$ 与 $\angle C$ 互补，$\angle B+\angle D=180^\circ$（或更直接用圆周角同弧定理得到 $\angle BDC=\angle BAC$），所以 $\angle B F C=\angle B E C$。因此四点 $B,E,F,C$ 共圆（同弧对应角相等 -> 共圆）。

用同样的方法可以得到若干四点共圆的结论，例如点列 $A,H,E,B$、$C,F,G,D$、$D,G,H,A$ 都是共圆的（把上面论证按循环顶点应用于相应的三角形）。

接着把这些共圆关系拼在一起做角和。按 Cut-the-knot 上的标准推理（见参考文献），由前面的共圆性可得

```math
\angle I_{ABD}I_{ABC}I_{BCD}=90^\circ,
```

即

```math
\angle H E F=90^\circ.
```

（上式中把具体的三角形内心代入即得到 $\angle HEF=90^\circ$。）

同理（循环顶点）可得到

```math
\angle E F G=90^\circ,\quad \angle F G H=90^\circ,\quad \angle G H E=90^\circ.
```

因此四边形 $EFGH$ 的每个内角都是直角，所以它是一个矩形。

总结：利用“内心看两顶点的角等于 $90^\circ$ 加第三角的一半”这一事实和内接四边形的圆周角关系，可以证明若干点列共圆，并由此做角和得出 $EFGH$ 四个内角均为直角，故为矩形。
