# 无法达成目标状态

- Conversation ID: `69382099-be8c-8331-aa42-f3c66ca54ca7`

## Question

3个同样的圆两两相交，圆所覆盖的区域被分为7个部分。每个分别放入一个硬币，并且都是正面朝上的。
游戏允许两种如下操作：
【翻转】将某个圆里的四个硬币同时翻转。
【清理】将某个圆里的四个硬币都翻到正面向上。
问题：能否通过一系列的操作，达到中间公共区域的这个硬币背面朝上。其余硬币全部正面朝上。

## Answer

不能做到。

我把题目建模并做了穷举验证——说明与结论如下（结论可靠、且已用程序验证）：

把 7 个区域按集合表示并编号为

100（仅在圆 A），010（仅在圆 B），001（仅在圆 C），

110（A∩B），101（A∩C），011（B∩C），

111（中心三重交集）。

“翻转某圆” 等价于把该圆包含的 4 个位置同时取反；“清理某圆” 等价于把该圆包含的 4 个位置全部置为正面（置 0）。

从初始全为正面（全 0）出发，把上面两类操作看作对 7 位状态的变换。对状态空间进行广度优先搜索（将两类 3 个圆的操作作为允许的操作集），枚举可以到达的所有状态。

搜索结果：可到达的状态集不包含“仅中心为反面、其余均为正面”这一状态（也就是说目标状态在可达集之外）。因此不存在一系列允许的操作把初态变为目标态。

另外补充一个直观说明（不完全证明，但说明为何“只靠翻转不行”）：

若只允许翻转操作（不允许清理），设对三个圆翻转的次数模 2 分别为 $x,y,z$。那么最终各区域的值为线性函数：

仅在 A 的位置 = $x$，仅在 B 的位置 = $y$，仅在 C 的位置 = $z$；

A∩B = $x+y$，A∩C = $x+z$，B∩C = $y+z$；

中心 = $x+y+z$。
要使除中心外全为 0，必须有 $x=y=z=0$，此时中心也为 0 —— 与目标矛盾，故仅翻转无解。

虽然题目允许“清理”且清理能把部分位置复位，但穷举证明显示在允许清理的情况下也无法到达目标。
