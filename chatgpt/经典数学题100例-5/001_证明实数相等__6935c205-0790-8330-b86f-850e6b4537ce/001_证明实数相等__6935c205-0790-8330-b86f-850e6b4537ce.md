# 证明实数相等

- Conversation ID: `6935c205-0790-8330-b86f-850e6b4537ce`

## Question

已知2n+1个实数a_1,a_2,\dots,a_{2n+1}，具有下面的性质：任意去掉其中的一个数，剩下的2n个实数都可以分为两组A,B（A,B个数可以不同），使得这两组的和相等。证明：a_1=a_2=\dots=a_{2n+1}

## Answer

证明（线性代数法）：

记向量 $a=(a_1,a_2,\dots,a_{2n+1})^T$，全体实数之和为 $S=\sum_{j=1}^{2n+1}a_j$。

根据题意，对于每个 $i\in\{1,\dots,2n+1\}$ ，把去掉 $a_i$ 后的剩下 $2n$ 个数分成两组 $A_i,B_i$ 使得两组和相等。把 “两组和相等” 用符号表示：对每个 $i$ 存在符号序列

```math
\sigma^{(i)}=(\sigma^{(i)}_1,\sigma^{(i)}_2,\dots,\sigma^{(i)}_{2n+1}),
```

其满足

$\sigma^{(i)}_i=0$（第 $i$ 个被去掉），

对 $j\ne i$，$\sigma^{(i)}_j\in\{1,-1\}$，并且

```math
\sum_{j=1}^{2n+1}\sigma^{(i)}_j a_j=0.
```

（这里把属于 $A_i$ 的记为 $+1$，属于 $B_i$ 的记为 $-1$，两组和相等等价于上述带符号和为 0。）

我们得到 $2n+1$ 个线性方程

```math
\sigma^{(i)}\cdot a=0\qquad(i=1,\dots,2n+1),
```

把这些 $2n+1$ 个行向量作为矩阵 $M$ 的行，得到矩阵方程 $M a=0$。

下面说明矩阵 $M$ 的行空间维数为 $2n$。为此固定一个索引（例如取第 $2n+1$ 行），对任意 $i\le 2n$ 考查行向量差

```math
u^{(i)}=\frac{1}{2}\big(\sigma^{(i)}-\sigma^{(2n+1)}\big).
```

注意到 $\sigma^{(i)}$ 与 $\sigma^{(2n+1)}$ 在第 $i$ 和第 $2n+1$ 个坐标处的取值分别是 $0$ 与 $0$，而在其它坐标上都是 $\pm1$；因此 $u^{(i)}$ 在第 $i$ 和第 $2n+1$ 坐标处分别是 $\pm1$ 或 $0$，而在其它坐标上为 $\{ -1,0,1\}$ 中的值。更重要的是，直接计算可见 $u^{(i)}$ 的第 $i$ 个坐标为 $1$，第 $2n+1$ 个坐标为 $-1$，其它坐标某些可能为 $0$ 或 $\pm1$。因此可以用适当的线性组合（对不同的 $i$）把第 $i$ 个标准差向量 $e_i-e_{2n+1}$（第 $i$ 个分量为 $1$，第 $2n+1$ 个为 $-1$，其余为 $0$）表示出来。于是可以得到 $2n$ 个线性无关向量 $e_i-e_{2n+1}$（$i=1,\dots,2n$）都在由行向量 $\{\sigma^{(i)}\}$ 张成的行空间里，故该行空间的维数至少为 $2n$。 显然行空间不可能超过 $2n$（因为每行在一个坐标上为 0，矩阵最多丢失 1 维），因此行空间维数恰为 $2n$。

因此矩阵 $M$ 的秩为 $2n$，所以线性方程组 $M a=0$ 的解空间维数为 $(2n+1)-2n=1$：也就是说，所有满足这些方程的向量 $a$ 形成一维子空间，任意两个解向量互为常数倍。

现在注意到常数向量 $\mathbf{1}=(1,1,\dots,1)^T$ 显然满足所有等式 $\sigma^{(i)}\cdot \mathbf{1}=\sum_{j\ne i}\sigma^{(i)}_j$。但由于对每一个 $i$ 两组和相等，$\sigma^{(i)}$ 的正项和负项个数可以不同，$\sigma^{(i)}\cdot\mathbf{1}$ 不一定为 0——不过这一步不是必要的。我们已经知道解空间是一维，所以任意解向量 $a$ 必与某固定非零解成常数倍关系。取一组显然满足条件的向量 —— 如果把所有数都取同一个常数 $c$，那么对任意 $i$ 去掉一个数后剩下的数可以任意对分为两组且两组和相等（因为每组元素个数不论如何，和都等于相同常数的和），所以常向量 $(1,1,\dots,1)$ 对应的数列 $(c,c,\dots,c)$ 是满足题意的解，从而解空间的一维基向量可以取为全等分量向量。因此任意满足题意的 $a$ 必为全相等常向量的常数倍，即

```math
a_1=a_2=\cdots=a_{2n+1}.
```

综上所述，题中条件蕴含所有 $a_j$ 相等，得证。 □

（注：以上证明的核心思想是把“去掉一个数后能分成和相等的两组”转化为一组线性方程 $\sigma^{(i)}\cdot a=0$，再用行向量差构造出 $2n$ 个线性无关的向量，得到解空间维数为 1，从而得出所有坐标相等。）
