# 魔方操作序列周期

- Conversation ID: `69955344-cf08-839c-b99c-4a1cd809b990`

## Question

对于一个魔方（三阶），重复操作同一个操作序列（比如RUFU'），在一定次数后必然恢复到初始状态。注意，魔方不得在一次操作序列中途停止。
1，证明这个结论。
2，这个【一定次数】的最大值是多少？达到最大值的操作序列具有什么特征？

## Answer

下面把你的“重复同一个操作序列必然回到初始状态”用群论语言说清楚，并给出三阶魔方这个“最多要重复多少次”的精确最大值与达到最大值时序列的结构特征。

#### 1) 证明：重复同一段操作序列必然回到初始状态

把三阶魔方所有可达状态组成集合 $G$。把“做一次某个操作序列（例如 $RUFU'$ ）”看成对状态集合的一个变换。更标准地：

每个基本转动（$R,U,F,\dots$）都是一个可逆变换；

若把“先做 $A$ 再做 $B$”记为复合 $BA$，那么这些变换在复合下构成一个有限群（魔方群）$G$。其规模是有限的（事实上是 $43{,}252{,}003{,}274{,}489{,}856{,}000$ ）。Mathematics Stack Exchange+1

设你的固定操作序列对应群元素 $g\in G$。从初始状态 $x_0$ 出发，重复整段序列 $n$ 次得到

```math
x_n=g^n(x_0).
```

因为 $G$ 有限，所以序列 $x_0,x_1,x_2,\dots$ 只能取有限多个值。由抽屉原理，必存在 $0\le i<j$ 使得 $x_i=x_j$，即

```math
g^i(x_0)=g^j(x_0)\ \Rightarrow\ g^{j-i}(x_0)=x_0.
```

这说明重复整段序列 $j-i$ 次后回到初始状态。

更强的是：由于 $g$ 在有限群里必有有限阶（存在最小正整数 $m$ 使 $g^m=e$），所以对任意起始状态 $x$，都有 $g^m(x)=x$。stanford.garron.us

你强调“不得在一次操作序列中途停止”，等价于说我们只看 $g, g^2, g^3,\dots$ 这些“整段重复”的幂，不看序列的前缀；上面的证明正是针对“整段重复”。

#### 2) 这个“最大必要次数”是多少？达到最大值的序列有什么特征？

##### 结论：最大值是 1260

三阶魔方群中元素的最大阶（也就是“某个操作序列需要重复的最多次数”）是

```math
\boxed{1260}.
```

这是经典结论；维基百科与多处群论讨论都给出该最大值，并给出具体例子序列。维基百科+2维基百科+2

例如下面这个序列就具有 1260 的周期（重复 1260 次回到原状）：

```math
RU^2D'BD'.
```

##### 为什么最大只能到 1260（结构性原因的“证明思路”）

三阶魔方状态由两大部分决定：

8 个角块：位置排列 + 角块朝向（扭转，模 3）；

12 个棱块：位置排列 + 棱块朝向（翻转，模 2）；

并且满足著名的“三条可达性约束”（基本定理/第一定律之一）：角排列与棱排列奇偶性必须一致；角扭转总和 $\equiv 0 \pmod 3$；翻转棱块数为偶数。sfu.ca+2jaapsch.net+2

因此一个序列 $g$ 的“回到原状次数”（阶）可以理解为：角部分需要的次数 与 棱部分需要的次数 的最小公倍数，并且还要满足奇偶性等耦合约束。

已知（并可用“按循环分解 + 朝向约束”系统枚举/论证）：

角部分最大阶是 45：来自角块出现一个 5-循环和一个 3-循环，并且朝向在循环内“带扭转”，使得 3-循环部分的回归需要 9 次，从而角整体达到 45。Mathematics Stack Exchange+1

棱部分理论上可到 56（例如 7-循环与 4-循环再叠加翻转效应），但把它与角的 45 组合会触犯“角棱排列奇偶性一致”的硬约束，因此 45 与 56 的组合不可实现；于是全局最大降下来。Mathematics Stack Exchange

在满足奇偶性耦合后，与角 45 可兼容的棱最大阶为 28，于是全局最大

```math
\mathrm{lcm}(45,28)=1260.
```

SpeedSolving Puzzles Community+1

#### 达到 1260 的操作序列具有什么“特征”？

用一句话概括：必须同时在角与棱上制造“长循环 + 朝向不立即复原”的坏循环（bad cycle），并且满足角棱奇偶性一致等约束，使得阶达到 $\mathrm{lcm}(45,28)$。

更具体地（这是最常见的结构描述）：

角块：出现 5-循环 + 3-循环，而且为了满足“角扭转总和 $\equiv0\pmod 3$”的约束，两个循环都需要带上合适的扭转，使角部分阶达到 45。SpeedSolving Puzzles Community+2sfu.ca+2

棱块：出现一个 7-循环，并配合若干 2-循环（交换），且至少有一个循环带翻转效应，使棱部分阶达到 28，同时保证棱翻转总数为偶数、并与角排列奇偶性匹配。SpeedSolving Puzzles Community+2sfu.ca+2

全局耦合：角排列与棱排列的奇偶性必须相同，因此某些“看起来能更大”的角/棱最大阶组合（比如导致 2520 的组合）会被奇偶性直接排除。Mathematics Stack Exchange+1
