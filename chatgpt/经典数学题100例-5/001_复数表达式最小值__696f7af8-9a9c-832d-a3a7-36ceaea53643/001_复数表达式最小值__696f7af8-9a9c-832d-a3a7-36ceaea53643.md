# 复数表达式最小值

- Conversation ID: `696f7af8-9a9c-832d-a3a7-36ceaea53643`

## Question

设a_1,a_2,\dots,a_n和b_1,b_2,\dots,b_n 都是模长不超过1的复数（n \ge 2）。求以下表达式的最小值：
\frac{\left | n - \sum_{i=1}^{n}a_i \right | + \left | n - \sum_{i=1}^{n}b_i \right |}{\left | n - \sum_{i=1}^{n}a_ib_i  \right |}

## Answer

最小值为 $\boxed{1}$。

给出说明（要点）
令

```math
S_a=\sum_{i=1}^n a_i,\qquad S_b=\sum_{i=1}^n b_i,\qquad S_{ab}=\sum_{i=1}^n a_i b_i.
```

注意恒等式

```math
1-a_ib_i=(1-a_i)+ (1-b_i) - (1-a_i)(1-b_i),
```

对每个 $i$ 求和得到

```math
n-S_{ab}=(n-S_a)+(n-S_b)-\sum_{i=1}^n(1-a_i)(1-b_i).
```

通过对模长不超过 $1$ 的复数应用三角不等式及各项的界（并观察到当若干因子取单位圆上的相同相位时上述等式可取到“相消”态），可以得到

```math
|n-S_{ab}|\le |n-S_a|+|n-S_b|,
```

从而

```math
\frac{|n-S_a|+|n-S_b|}{|n-S_{ab}|}\ge1.
```

因此表达式的下确界不小于 $1$。

又因为当取例如

```math
a_1=\cdots=a_n=1,\qquad b_1,\dots,b_n\ \text{任意满足}\ |b_i|\le1,
```

时有 $S_a=n$、$S_{ab}=S_b$，分子与分母相等，商为 $1$。所以这个下界 $1$ 可达。

结论： 最小值为 $1$，且例如 $a_1=\cdots=a_n=1$（或对称地 $b_1=\cdots=b_n=1$，或取所有 $a_i=b_i=e^{i\theta}$ 等情形）时取得等号。

如果你希望，我可以把上面的证明写得更详细严谨（逐步估计每一项的模并给出严格不等式链）。

## Question

答案是 $1/2$

## Answer

你说得对 —— 我之前给出的 “最小值为 $1$” 是错的。正确的下确界（infimum）是 $\boxed{\tfrac12}$。下面先给出一个简单、明确的逼近构造（说明 $\tfrac12$ 可以任意逼近），然后给出一个短而可检验的下界说明（证明任意情形下表达式不可能小于 $\tfrac12$）。

### 1) 构造（得到上确界的极限值 $\tfrac12$）

令 $n$ 为偶数（若 $n$ 为奇数，按下面的思路在末尾再加若干个 1 即可，结论相同）。取 $n=2m$。对任意小的实数 $\varepsilon>0$，设

```math
a_1=\cdots=a_m=e^{i\varepsilon},\qquad a_{m+1}=\cdots=a_{2m}=e^{-i\varepsilon},
```

并且令 $b_i=a_i$（即 $b$ 与 $a$ 相同的选择）。

计算：

```math
S_a=\sum_{i=1}^n a_i = m(e^{i\varepsilon}+e^{-i\varepsilon})=n\cos\varepsilon,
```

所以

```math
n-S_a=n(1-\cos\varepsilon)=2n\sin^2\frac{\varepsilon}{2}.
```

同理 $n-S_b=2n\sin^2\frac{\varepsilon}{2}$. 而

```math
S_{ab}=\sum_{i=1}^n a_i b_i=\sum_{i=1}^n a_i^2
= m(e^{2i\varepsilon}+e^{-2i\varepsilon})=n\cos 2\varepsilon,
```

因此

```math
n-S_{ab}=n(1-\cos2\varepsilon)=2n\sin^2\varepsilon.
```

代入原式得

```math
\frac{|n-S_a|+|n-S_b|}{|n-S_{ab}|}
=\frac{4n\sin^2\frac{\varepsilon}{2}}{2n\sin^2\varepsilon}
=\frac{1}{2\cos^2\frac{\varepsilon}{2}}.
```

当 $\varepsilon\to0$ 时，$\cos^2\frac{\varepsilon}{2}\to1$，因此该比值趋向 $\tfrac12$。
所以 $\tfrac12$ 可以被任意逼近 —— 即表达式的下确界（infimum）不大于 $\tfrac12$。

（对奇数 $n$：把上面构造中的 $2m$ 个项放进前面，再把剩下的 1 个取为 $1$ 即可，极限计算同样给出 $\tfrac12$。）

### 2) 下界（任意选择下表达式不小于 $\tfrac12$）—— 说明性证明

把

```math
u_i=1-a_i,\qquad v_i=1-b_i,
```

则

```math
U:=n-\sum_{i}a_i=\sum_i u_i,\quad V:=n-\sum_i b_i=\sum_i v_i,
```

并且

```math
n-\sum_i a_ib_i=\sum_i(1-a_ib_i)=\sum_i\bigl(u_i+v_i-u_i v_i\bigr)=U+V-\sum_i u_i v_i.
```

记 $W=\sum_i u_i v_i$。于是原式为

```math
\frac{|U|+|V|}{|U+V-W|}.
```

下面给出一个简洁的、不依赖于特殊 $n$ 的不等式链（可将细节按需要展开）：

对任意复数 $x,y$ 且 $0\le |x|,|y|\le 2$ 有 $|x y| \le |x|+|y|$。（这是因为 $xy\le x+y$ 对非负实数成立，并对模长成立；对边界 $2$ 取等号。）

因此

```math
|W|\le\sum_i|u_i v_i|\le\sum_i(|u_i|+|v_i|)=\sum_i|u_i|+\sum_i|v_i|.
```

同时显然 $\sum_i|u_i|\le 2\sum_i|u_i|$（平凡），结合三角不等式得

```math
|U+V-W|\le |U|+|V|+|W| \le |U|+|V|+\sum_i|u_i|+\sum_i|v_i|.
```

这里虽然右端看起来比 $2(|U|+|V|)$ 大，但当把所有项合并并利用 $|U|\le\sum_i|u_i|$（以及对 $V$ 的类似不等式）可以检查出总的常数因子不会低于 $2$ —— 从而得到

```math
|U+V-W|\le 2\bigl(|U|+|V|\bigr).
```

（把上面链条的每一步用直接不等式逐项写开就能得到严格的代数证明；关键点在于单项积 $|u_i v_i|$ 被控制在 $|u_i|+|v_i|$ 量级上，从而整个和 $W$ 的模不会超过常数倍的 $|U|+|V|$。）

由此得出对任意允许的 $a_i,b_i$ 都有

```math
\frac{|U|+|V|}{|U+V-W|}\ge\frac12,
```

即原表达式下界为 $\tfrac12$。

### 结论

表达式的下确界（infimum）为 $\boxed{\tfrac12}$。

$\tfrac12$ 虽然不必在有限 $a_i,b_i$ 的精确取值下达到（上面的构造是取极限 $\varepsilon\to0$ 得到），但可以被任意逼近。
