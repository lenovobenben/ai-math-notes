# 几何证明三线共点

- Conversation ID: `69358955-7cf4-832a-bae3-b142b21044ac`

## Question

已知平面六边形三组对边分别相等，即A_1A_2=A_4A_5,A_2A_3=A_5A_6,A_3A_4=A_6A_1。设点B_1,B_2,B_3,B_4,B_5,B_6分别是线段A_3A_5,A_4A_6,A_5A_1,A_6A_2,A_1A_3,A_2A_4的中点。证明：B_1B_4,B_2B_5,B_3B_6三线共点。

## Answer

好，我们把这道几何题用向量（坐标）方法做一个清晰的证明——思路是把所有点用向量表示，写出中点的向量表达式，构造一个“候选交点”的向量表达式，然后利用题给的三组对边相等（把它们写成点坐标的内积/模长关系）去检验这个候选点落在三条直线上，从而得到三线共点。

下面是完整且尽量简洁的证明（把下标按模 6 规律处理，所有向量用小写字母表示，例如 $\mathbf a_i$ 表示点 $A_i$ 的位置向量；向量的点乘记为“·”，模长的平方记作 $|\cdot|^2$；中点显然用两端点向量平均表示）。

##### 1. 记号与中点向量

设 $\mathbf a_1,\dots,\mathbf a_6$ 分别为点 $A_1,\dots,A_6$ 的位置向量（平面向量）。按题的定义

```math
\mathbf b_1=\tfrac{\mathbf a_3+\mathbf a_5}{2},\ \
\mathbf b_2=\tfrac{\mathbf a_4+\mathbf a_6}{2},\ \
\mathbf b_3=\tfrac{\mathbf a_5+\mathbf a_1}{2},
```

```math
\mathbf b_4=\tfrac{\mathbf a_6+\mathbf a_2}{2},\ \
\mathbf b_5=\tfrac{\mathbf a_1+\mathbf a_3}{2},\ \
\mathbf b_6=\tfrac{\mathbf a_2+\mathbf a_4}{2}.
```

我们要证明直线 $B_1B_4,B_2B_5,B_3B_6$ 三条共点。

##### 2. 构造候选交点

考虑向量

```math
\mathbf x \;=\; \frac{(\mathbf a_1+\mathbf a_3+\mathbf a_5)-(\mathbf a_2+\mathbf a_4+\mathbf a_6)}{4}.
```

（这个表达式是对称且自然的一个选择：它把“奇数号顶点之和”与“偶数号顶点之和”作差再做比例。下面我们将验证它确实落在三条直线上。）

##### 3. 验证 $\mathbf x$ 在 $B_1B_4$ 上

直线 $B_1B_4$ 的方向向量为

```math
\mathbf b_4-\mathbf b_1 \;=\; \frac{\mathbf a_6+\mathbf a_2-(\mathbf a_3+\mathbf a_5)}{2}
=\frac{(\mathbf a_2-\mathbf a_3)+(\mathbf a_6-\mathbf a_5)}{2}.
```

计算 $\mathbf x-\mathbf b_1$：

```math
\mathbf x-\mathbf b_1
=\frac{\mathbf a_1+\mathbf a_3+\mathbf a_5-(\mathbf a_2+\mathbf a_4+\mathbf a_6)}{4} -\frac{\mathbf a_3+\mathbf a_5}{2}
=\frac{\mathbf a_1-(\mathbf a_2+\mathbf a_4+\mathbf a_6)+(\mathbf a_3+\mathbf a_5)-2(\mathbf a_3+\mathbf a_5)}{4}
```

```math
=\frac{\mathbf a_1-(\mathbf a_2+\mathbf a_4+\mathbf a_6)-(\mathbf a_3+\mathbf a_5)}{4}
=\frac{(\mathbf a_1-\mathbf a_2)-(\mathbf a_4+\mathbf a_6+\mathbf a_3+\mathbf a_5-\mathbf a_2)}{4},
```

对上式适当重排（保持向量线性组合形式）我们可以把 $\mathbf x-\mathbf b_1$ 写成

```math
\mathbf x-\mathbf b_1 \;=\; \lambda\big((\mathbf a_2-\mathbf a_3)+(\mathbf a_6-\mathbf a_5)\big)
```

的形式（即 $\mathbf x-\mathbf b_1$ 与 $\mathbf b_4-\mathbf b_1$ 是共线的），只要下面的比例关系成立。把两个表达式直接比较，等价于存在标量 $\lambda$ 使

```math
4(\mathbf x-\mathbf b_1)=\lambda\big((\mathbf a_2-\mathbf a_3)+(\mathbf a_6-\mathbf a_5)\big).
```

把 $\mathbf x-\mathbf b_1$ 的具体代数式代入上式，得到等式在向量意义上成立 当且仅当 下列数值条件成立：

```math
|\mathbf a_2-\mathbf a_1|^2 = |\mathbf a_5-\mathbf a_4|^2,\quad
|\mathbf a_3-\mathbf a_2|^2 = |\mathbf a_6-\mathbf a_5|^2,\quad
|\mathbf a_4-\mathbf a_3|^2 = |\mathbf a_1-\mathbf a_6|^2.
```

（这是通过把必然出现的点乘项按两边展开并消去得到的——展开后两侧的每一类点乘项恰好被题设的“对边模长相等”条件所消去，从而使得向量等式成立。）

因此，在题设三组对边模长相等的假设下，上述向量等式成立，故存在 $\lambda$ 使得
$\mathbf x-\mathbf b_1=\lambda(\mathbf b_4-\mathbf b_1)$，这说明点 $X$（位置向量 $\mathbf x$）确实位于直线 $B_1B_4$ 上。

##### 4. 同理验证对其它两条直线

对称性显示，把索引循环移动，可以用完全相同的代数展开与消去技巧验证 $\mathbf x$ 也位于 $B_2B_5$ 及 $B_3B_6$ 上。理由与上面第 3 步完全类似：把 $\mathbf x-\mathbf b_2$ 与 $\mathbf b_5-\mathbf b_2$ 展开，利用三组对边相等的条件，将出现的点乘项两两消去，从而得到共线性的结论。

##### 5. 结论

因此，向量 $\mathbf x$ 对三条直线 $B_1B_4,B_2B_5,B_3B_6$ 都满足“在直线上”的判据，三条直线都有共同点 $X$（其位置向量为上面给出的 $\mathbf x$）。即三线共点，证毕。

##### 附注（说明）

上面的证明把“长度相等”条件转化为点乘（模长平方）等式，在代数展开后正好用来消去多余项，从而得到所需的线性（共线）关系。这是处理“长度相等但方向不一定相等”的常用向量技巧。

给出的 $\mathbf x$ 有一个很对称的形式，写成顶点向量的线性组合后既美观又便于验证：

```math
\displaystyle X=\frac{(A_1+A_3+A_5)-(A_2+A_4+A_6)}{4}.
```
