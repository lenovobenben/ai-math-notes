# 数列极限与范围证明

- Conversation ID: `698599e5-c6fc-83a0-9f42-bfc023e57725`

## Question

已知数列 $\{x_n\}$ 满足：$x_1 = 1$，$x_n = x_{n+1} + \ln(1 + x_{n+1})$ 对任意 $n \in \mathbb{N}^*$ 均成立。证明：当 $n \in \mathbb{N}^*$ 时$x_n$的取值范围（与$n$有关）。

## Answer

设

```math
f(t)=t+\ln(1+t)\qquad (t\gt{}-1).
```

则题设递推等价于

```math
x_n=f(x_{n+1}),\qquad x_1=1.
```

因为

```math
f'(t)=1+\frac1{1+t}\gt{}0\quad(t\gt{}-1),
```

所以 $f$ 在 $(-1,\infty)$ 上严格增，从而每一步 $x_{n+1}=f^{-1}(x_n)$ 都唯一存在；序列 $\{x_n\}$ 唯一确定。

#### 1) 基本性质：正、单调、极限

先证 $x_n>0$ 且单调递减。

若 $x_{n+1}>0$，则 $\ln(1+x_{n+1})>0$，所以

```math
x_n=x_{n+1}+\ln(1+x_{n+1})\gt{}x_{n+1}\gt{}0.
```

由 $x_1=1>0$ 归纳得 $x_n>0$ 对一切 $n$ 成立，且因 $\ln(1+x_{n+1})>0$ 立刻有 $x_n>x_{n+1}$，所以严格递减。

又因 $x_n>0$ 递减，极限 $L=\lim_{n\to\infty}x_n$ 存在且 $L\ge 0$。对递推取极限：

```math
L=L+\ln(1+L)\ \Rightarrow\ \ln(1+L)=0\ \Rightarrow\ L=0.
```

因此

```math
0\lt{}x_n\le 1,\qquad x_n\downarrow 0.
```

#### 2) 给出与 $n$ 有关的“取值范围”：指数型夹逼

下面给出一个非常干净、与 $n$ 直接相关的范围（上、下界都是 $2^{-(n-1)}$ 级别）。

##### (A) 上界：$x_n<2^{-(n-2)}$（$n\ge2$）

对 $t>0$，有基本不等式 $\ln(1+t)<t$。代入 $t=x_{n+1}$ 得

```math
x_n=x_{n+1}+\ln(1+x_{n+1})\lt{}x_{n+1}+x_{n+1}=2x_{n+1}.
```

即

```math
x_{n+1}\gt{}\frac{x_n}{2}.
```

从 $x_1=1$ 迭代：

```math
x_n\lt{}\frac{1}{2^{\,n-2}}\qquad(n\ge2).
```

##### (B) 下界：$x_n>\dfrac{1}{2^{n-1}}$（所有 $n\ge1$）

利用 $\ln(1+t)>\dfrac{t}{1+t}$（可由函数 $\phi(t)=\ln(1+t)-\frac{t}{1+t}$ 求导验证 $\phi'(t)=\frac{t}{(1+t)^2}\ge0$，且 $\phi(0)=0$）。

代入 $t=x_{n+1}>0$：

```math
x_n=x_{n+1}+\ln(1+x_{n+1})
\gt{} x_{n+1}+\frac{x_{n+1}}{1+x_{n+1}}
= x_{n+1}\Bigl(1+\frac{1}{1+x_{n+1}}\Bigr)
\gt{} x_{n+1}\Bigl(1+\frac12\Bigr)=\frac32\,x_{n+1},
```

其中最后一步用到了 $x_{n+1}\le x_1=1\Rightarrow 1+x_{n+1}\le2\Rightarrow \frac1{1+x_{n+1}}\ge\frac12$。

于是

```math
x_{n+1}\lt{}\frac23\,x_n.
```

从 $x_1=1$ 迭代得到

```math
x_n\gt{}\Bigl(\frac32\Bigr)^{n-1}x_{n+1}\quad\text{（反向看）}
```

更直接地，从 $x_{k+1}<\frac23 x_k$ 得

```math
x_n\lt{}\Bigl(\frac23\Bigr)^{n-1}x_1=\Bigl(\frac23\Bigr)^{n-1},
```

这给的是“另一种指数上界”。但我们要的是 $\frac1{2^{n-1}}$ 形式的下界，可以这样做：

因为已知上界 $x_{n}\le 1$，有

```math
x_{n}=x_{n+1}+\ln(1+x_{n+1}) \le x_{n+1}+x_{n+1}=2x_{n+1}
\Rightarrow x_{n+1}\ge \frac{x_n}{2}.
```

从 $x_1=1$ 推出

```math
x_n \ge \frac{1}{2^{n-1}}\qquad(n\ge1).
```

并且由于 $\ln(1+x_{n+1})>0$，当 $n\ge2$ 时严格大于。

#### 3) 结论：$x_n$ 的取值范围

综合 (A)(B)：

$n=1$: $x_1=1$。

$n\ge2$:

```math
\boxed{\ \frac{1}{2^{\,n-1}}\lt{}x_n\lt{}\frac{1}{2^{\,n-2}}\ }.
```

这就是一个明确、只含 $n$ 的取值范围（上下界都是同阶的指数级）。

如果你希望把范围进一步收紧（例如证明 $2^{n-1}x_n$ 单调增加并收敛到某个常数 $L\approx1.278$，从而得到 $x_n\sim L\cdot2^{-(n-1)}$ 的渐近），我也可以把那一套更精细的不等式与收敛证明补全。
