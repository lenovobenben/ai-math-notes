# 最小平方和条件

- Conversation ID: `693bcc10-3024-8330-a0aa-0b9ccf917a92`

## Question

给定一个锐角三角形ABC，有一个它的内接三角形DEF。使得DEF各边的平方和最小的充要条件是什么？

## Answer

结论（充要条件）：

当且仅当 $D,E,F$ 分别是三角形 $ABC$ 的三条高线在边 $BC,CA,AB$ 上的垂足（即 $DEF$ 为 $ABC$ 的直心三角形 / 垂足三角形）时，$\;DE^2+EF^2+FD^2$ 取到所有内接三角形中的最小值。

下面给出一个清晰的变分/最优性说明（思路证明——若要严格的坐标代数证明我可以接着写出完整计算）：

设 $D$ 在边 $BC$ 上可变，$E$ 在 $CA$ 上可变，$F$ 在 $AB$ 上可变。记目标函数

```math
S(D,E,F)=|DE|^2+|EF|^2+|FD|^2.
```

把三个点独立变动做一阶必要条件：固定 $E,F$ 考虑沿边 $BC$ 移动 $D$。把 $D$ 在边 $BC$ 上参数化（任意参数化都可），对参数取导并令导数为零，得到

```math
\bigl(\overrightarrow{DE}+\overrightarrow{DF}\bigr)\cdot \mathbf{t}_{BC}=0,
```

其中 $\mathbf{t}_{BC}$ 是边 $BC$ 的切向量。也就是说，向量 $\overrightarrow{DE}+\overrightarrow{DF}$ 与 $BC$ 垂直。

对 $E$ 在 $CA$ 上可变、$F$ 在 $AB$ 上可变做同样分析，分别得到类似条件：

```math
\bigl(\overrightarrow{EF}+\overrightarrow{ED}\bigr)\cdot \mathbf{t}_{CA}=0,\qquad
\bigl(\overrightarrow{FD}+\overrightarrow{FE}\bigr)\cdot \mathbf{t}_{AB}=0.
```

以上三个向量等式合起来可以等价地表述为三条连接顶点与对应内接点的线（即 $A\!-\!D,\;B\!-\!E,\;C\!-\!F$）都与对边对应的内接边互为垂直关系。更直观地说，最优点阵必须满足每一条从顶点到其对应内接点的连线都垂直于内接三角形对应的对边——这正是“垂足三角形”的特征：若 $D$ 是 $A$ 的垂足则 $AD\perp BC$ 且 $D\in BC$，类似地对 $E,F$。

因为题中假设 $ABC$ 为锐角三角形，所以三条高线都落在各自对应的对边上，垂足三角形 $DEF$ 是一个合法的内接三角形；并且上面的一阶必要条件在 $DEF$ 为垂足三角形时被满足，从而它是候选极小点。再进一步（可通过二阶变分或直接坐标代数展开二次形式检查）可证该极值为全局最小值（直观上：向任一方向小幅移动任一顶点都会增大平方和）。

因此结论成立：只有当 $D,E,F$ 是三条高的垂足（即 $DEF$ 为直心三角形）时，$\;DE^2+EF^2+FD^2$ 达到最小值；反之若该平方和在所有内接三角形中最小，则 $DEF$ 必为垂足三角形。

## Question

如果是DEF各边的和最小呢？

## Answer

结论（充要条件）：

当且仅当 $D,E,F$ 分别是 $ABC$ 三条高在边 $BC,CA,AB$ 上的垂足（即 $DEF$ 为 $ABC$ 的正交三角形 / 直心三角形）时，内接三角形 $DEF$ 的周长 $DE+EF+FD$ 在所有内接三角形中取得最小值。

下面给出一个清晰的变分/几何证明要点（要点式、易于把握；需要完整坐标推导我可以接着写）：

固定两个点，沿边移动第三点——反射原理（光路最短 / 菲涅尔原理）。
设 $E\in CA,\;F\in AB$ 固定，令 $D\in BC$ 可变。要使 $DE+DF$ 关于 $D$ 取最小，$D$ 必须满足“入射角等于反射角”条件：把点 $E$ 关于直线 $BC$ 作镜像 $E'$，则最短路径 $E\rightarrow D\rightarrow F$ 等价于直线段 $E'F$ 与 $BC$ 的交点。因此在极值点处有

```math
\angle( DE,\;BC)=\angle( DF,\;BC).
```

即 $DE$ 与 $DF$ 关于边 $BC$ 对称（角度相等、方向对称）。

对三个顶点做同样的变动分析，分别得到三条条件：

```math
\angle( DE,\;BC)=\angle( DF,\;BC),\quad
\angle( EF,\;CA)=\angle( ED,\;CA),\quad
\angle( FD,\;AB)=\angle( FE,\;AB).
```

这三个条件一起限制了 $D,E,F$ 的位置。

将上述角度条件展开可得：边 $BC$ 是角 $\angle EDF$ 的外（或内）角平分线所在直线；同理 $CA$ 与角 $\angle FED$ 满足平分关系，$AB$ 与角 $\angle DFE$ 也满足平分关系。把这三条边作为相应角的（外）平分线联立，可推出三角形 $DEF$ 的三个角的平分线分别通过 $A,B,C$——换言之，线 $AD,\,BE,\,CF$ 与 $BC,CA,AB$ 成特定对称关系。对锐角三角形 $ABC$ 进一步分析（角和关系、垂直关系代入）可得

```math
AD\perp BC,\quad BE\perp CA,\quad CF\perp AB,
```

即 $D,E,F$ 必为三高的垂足。

反过来，如果 $D,E,F$ 是三高的垂足（正交三角形），则上面“反射/等角”条件显然成立（在垂足处两条连接内接点的边相对于对应边呈对称角），所以这样的 $DEF$ 满足一阶最优条件。再检查二阶变分或利用几何单调性可证该极值为全局最小，从而成立“当且仅当”。

因此，周长最小时刻的内接三角形就是 $ABC$ 的直心（垂足）三角形。

## Question

这两个的结论为什么是一致的？

## Answer

之所以 “最小周长” 和 “最小平方和” 得到 同一个最优内接三角形（垂足三角形），本质原因是：

对于三角形 $DEF$ 在锐角三角形 $ABC$ 内移动的几何约束下，周长函数 $P = DE+EF+FD$ 与平方和函数 $Q = DE^2+EF^2+FD^2$ 的极值点满足同一组几何最优性条件，而这组条件唯一地指向垂足三角形。

下面分三层解释：（1）直观几何原因；（2）解析原因；（3）特殊性：锐角＋内接三角形。

### 1. 直观几何原因：两个目标函数的“最优性条件”一致

考虑一个点（如 $D$）在一条边（如 $BC$）上滑动。

##### 周长最小的必要条件：

对 $D$ 来说，要求

```math
DE+DF \quad\text{对 }D\text{ 最小}
```

利用反射法得到：

```math
\angle( DE , BC ) = \angle( DF , BC ).
```

即 角度对称。

##### 平方和最小的必要条件：

```math
DE^2+DF^2 \quad\text{对 }D\text{ 最小}
```

对 $D$ 做导数，会要求：

```math
\overrightarrow{DE}+\overrightarrow{DF}\perp BC.
```

这等价于：

```math
\angle( DE , BC ) = \angle( DF , BC ).
```

也就是说，平方和最小时，得到的也是同一个“角度对称”条件。

因此：
两个优化问题对 $D,E,F$ 的一阶必要条件完全一样。

而对内接三角形来说，这三个条件联立只有一个解：

```math
D,E,F\text{ 是三条高的垂足。}
```

### 2. 解析原因：两个函数是同一个几何量的凸变换

设 $x = (DE, EF, FD)$，则

周长 $P(x) = x_1 + x_2 + x_3$ 是线性函数

平方和 $Q(x) = x_1^2 + x_2^2 + x_3^2$ 是严格凸函数

在几何约束（点必须在边上）下，两者的极小值必然出现在满足一阶条件的唯一可行点。

因为：

在允许的区域内，满足一阶条件的点只有一个

凸函数（包括线性、平方）在该区域的极小点也只能是这个点

因此两者极值点一致。

### 3. 为什么限定“锐角三角形”很重要？

若 $ABC$ 有钝角：

高线的垂足会落在边的延长线上，不是合法内接点

于是该“唯一解”不再属于可行解域

两个优化问题可能得到不同的最优点

但题目中明确 $ABC$ 为锐角，所以：

三个垂足全部在边上

是合法内接三角形

因此两题恰好同向指向同一个唯一可行极小点

### 总结：为什么两个问题的结论一致？

因为在锐角三角形中，

“周长最小”

“平方和最小”

这两个目标函数的极值必要条件完全一致（都要求三边关于边呈角度对称），而该条件的唯一几何解就是垂足三角形。

因此：

两个问题的极小点完全相同：就是三条高的垂足构成的内接三角形。
