# 解方程 3^n-2^n

- Conversation ID: `693825d0-edac-832f-bc97-d4132ee83518`

## Question

```math
解方程 3^n-2^n=65（n为正整数）
```

## Answer

解方程：

```math
3^n - 2^n = 65,\quad n \in \mathbb{Z}_{\gt{}0}
```

从小的正整数 $n$ 开始代入：

```math
n=1: 3^1-2^1 = 1
```

```math
n=2: 9-4 = 5
```

```math
n=3: 27-8 = 19
```

```math
n=4: 81-16 = 65
```

正好满足方程，因此：

```math
\boxed{n=4}
```

这是唯一的正整数解。
