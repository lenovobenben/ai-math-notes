# 求函数f

- Conversation ID: `694505af-b3c0-8322-bedd-7fab411e653f`

## Question

平面中，依次有四条平行线a,b,c,d（它们两两之间的距离分别为x,y,z）。a上任取一点A作垂线l与d的垂足为D。求b,c上各一点B,C使得AB=BC=CD=f(x,y,z)。求函数f

## Answer

把四条平行线看作水平线，垂直方向为 $y$-轴。为方便设坐标：令线 $a$ 的高度为 $y=0$，从 $a$ 上任取点 $A=(0,0)$（可任意平移），则

```math
b:\ y=-x,\quad c:\ y=-(x+y),\quad d:\ y=-(x+y+z).
```

设所求的等长值为 $s$，在 $b,c$ 上的点为

```math
B=(b,\,-x),\qquad C=(c,\,-(x+y)),
```

并有

```math
AB^2=b^2+x^2=s^2,\quad BC^2=(c-b)^2+y^2=s^2,\quad CD^2=c^2+z^2=s^2.
```

令 $S=s^2$。由上三式得

```math
b^2=S-x^2,\quad c^2=S-z^2,\quad (c-b)^2=S-y^2.
```

利用 $(c-b)^2=c^2+b^2-2bc$ 可得

```math
S-y^2=(S-x^2)+(S-z^2)-2bc,
```

即

```math
bc=\tfrac{1}{2}\bigl(S-(x^2+z^2)+y^2\bigr).
```

又 $b^2c^2=(S-x^2)(S-z^2)$。把 $bc$ 平方代入可消去 $b,c$，得到关于 $S$ 的二次方程，解出

```math
S=\frac{x^2+y^2+z^2}{3}+\frac{2}{3}\sqrt{\,x^4+y^4+z^4-x^2y^2-y^2z^2-z^2x^2\,}.
```

因此所求函数为

```math
\boxed{\,f(x,y,z)=\sqrt{\frac{x^2+y^2+z^2}{3}+\frac{2}{3}\sqrt{x^4+y^4+z^4-x^2y^2-y^2z^2-z^2x^2}\,}\,}.
```

说明：上式取“+”号的根（较大的一根），保证 $S\ge x^2,y^2,z^2$ 以使 $b,c$ 实数。检验特殊情况：当 $x=y=z=t$ 时，公式化简得 $f=t$，与直观一致。
