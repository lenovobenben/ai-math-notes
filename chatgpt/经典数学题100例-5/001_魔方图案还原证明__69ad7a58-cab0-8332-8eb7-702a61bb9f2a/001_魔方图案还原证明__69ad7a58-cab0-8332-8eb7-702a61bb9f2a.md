# 魔方图案还原证明

- Conversation ID: `69ad7a58-cab0-8332-8eb7-702a61bb9f2a`

## Question

有一种魔方带图案。那么按照传统魔方还原的步骤复原后，每个面的图案不一定是对的。那么有如下两个问题：

1，有错误的面的个数n有几种情况？
2，如果一个面有错误。那么错误的细节是什么？我听说只有中心块方向会错，周围八个都是同一个方向的。

请给出数学上的证明。而不是简单的陈述结论。

## Answer

我先把这个问题当成“3\times3 图案魔方 / supercube”的数学模型来处理：颜色已经还原，但每个可见小块上的图案方向也要算。接下来我会先确定“一个面错了”到底对应什么群论量，再推出错误面的个数只能取哪些值。

把这种“带方向图案的 3\times3 魔方”抽象成 supercube / picture cube 来看最清楚：
普通 3\times3 只关心“块在哪、边是否翻、角是否扭”；而图案魔方还要关心每个中心块的朝向。公开资料里常见的结论是：这类魔方在普通 3\times3 还原后，额外要处理的是中心朝向；并且存在“单个中心转 $180^\circ$”和“两个中心分别转 $\pm90^\circ$”这两类中心算法，而“单个中心单独转 $90^\circ$”是不可能的。SpeedSolving Puzzles Community+2jaapsch.net+2

下面给出严格证明。

#### 一、模型

固定一个“完全图案还原”的参考状态。
设六个中心面的朝向分别为

```math
c_U,c_D,c_F,c_B,c_L,c_R\in \mathbb Z_4,
```

其中 $0$ 表示正确，$1,2,3$ 分别表示顺时针转了 $90^\circ,180^\circ,270^\circ$。

我们把“按普通魔方已经还原”理解为：

8 个角块都在正确位置且角向正确；

12 个棱块都在正确位置且棱向正确；

允许六个中心块的朝向还没对。

也就是说，普通 3\times3 已解后的残余自由度，只可能来自中心朝向。

#### 二、为什么“周围八块不会再额外出错”

这一步是关键。

##### 1）角块没有额外自由度

一个角块在普通魔方里一旦“位置正确且角向正确”，它的三个贴纸分别落在正确的三个面上。
而一个刚体旋转若同时固定三个两两不共线的外法向方向，那么这个旋转只能是恒等。

换句话说：
角块在普通意义上解对以后，它在三维空间中的姿态已经唯一确定。

因此它露在每个面上的图案方向，也已经唯一确定；不存在“颜色对了但图案还能再转一点”的额外自由度。

##### 2）棱块也没有额外自由度

同理，一个棱块一旦“位置正确且未翻转”，它的两个贴纸落在正确的两个面上。
一个保向刚体旋转若同时固定两条互相垂直的法向，也只能是恒等。

所以：
棱块在普通意义上解对以后，它的图案方向也唯一确定。

##### 3）只有中心块不同

中心块始终待在自己的面上，但它绕该面法向还可以转 $0,90,180,270$ 四种角度。
这正是普通魔方忽略、而图案魔方必须额外处理的自由度。SpeedSolving Puzzles Community+1

于是得到：

定理 A
当普通 3\times3 已经还原时，若图案仍未完全还原，那么所有错误都只可能出现在六个中心块上；周围八个块并不存在额外的“图案朝向错误”。

所以你听到的那句“如果一个面错了，只有中心块方向会错”，在数学上是对的。

但“周围八个都是同一个方向”这句话要区分两层意思：

若你说的是 箭头 supercube（每个小贴纸都是箭头），那因为这八块已经是正确姿态，所以它们确实会呈现成解状态该有的统一方向。

若你说的是 一般图片魔方，那“周围八块是否看起来同方向”取决于图片本身；数学上能保证的不是“同方向”，而是“它们已经完全正确，没有额外错误”。

#### 三、中心朝向满足什么约束

现在分析

```math
(c_U,c_D,c_F,c_B,c_L,c_R)\in(\mathbb Z_4)^6
```

哪些是可能出现的。

##### 1）必要条件：奇数个 $90^\circ$ 型错误不可能

把一个解普通魔方的过程拆成若干个 四分之一转（半转算两个四分之一转）。

每做一次某个面的四分之一转：

那个面的中心朝向会加 $1\pmod 4$；

同时，这一步对角块置换是奇排列，对棱块置换也是奇排列。jaapsch.net

设总共做了 $Q$ 次四分之一转。
因为最后普通魔方被还原了，所以角块总排列、棱块总排列都必须是偶排列，于是

```math
Q\equiv 0 \pmod 2.
```

另一方面，每一次四分之一转都会把“某一个中心的朝向奇偶性”翻转一次。
因此六个中心朝向之和的奇偶性就是 $Q$ 的奇偶性：

```math
c_U+c_D+c_F+c_B+c_L+c_R \equiv Q \equiv 0 \pmod 2.
```

于是必有

```math
\boxed{c_U+c_D+c_F+c_B+c_L+c_R\equiv 0\pmod 2.}
```

这等价于：

奇数个中心处于 $\pm90^\circ$ 状态是不可能的。

特别地，单独一个中心转 $90^\circ$ 不可能。
这和资料中的结论一致。SpeedSolving Puzzles Community+1

#### 四、这个条件也是充分的

现在要证明：除了上面的奇偶限制外，没有别的限制。

公开资料里给出了两种中心操作：

一个中心转 $180^\circ$ 的算法；

两个中心分别转 $+90^\circ$ 与 $-90^\circ$ 的算法。SpeedSolving Puzzles Community+2jaapsch.net+2

把它们记成两类生成元：

$2e_i$：第 $i$ 个中心加 $2$（即转 $180^\circ$）；

$e_i-e_j$：第 $i$ 个中心加 $1$，第 $j$ 个中心减 $1$。

这些操作都不会破坏普通 3\times3 的已解状态。

现在任取一个满足

```math
c_U+\cdots+c_R\equiv 0\pmod 2
```

的六元组 $c=(c_i)$。

把其中所有奇数项（即 $1$ 或 $3$）两两配对。因为奇数项个数必为偶数，所以总能配对。
对每一对，用若干个 $e_i-e_j$ 就能把它们做出来；做完以后，剩下的各项只可能是 $0$ 或 $2$。
再对每个值为 $2$ 的中心使用一次 $2e_i$。

于是任意满足偶和条件的 $c$ 都能实现。

所以残余中心状态的全集正是

```math
\boxed{
K=\Bigl\{(c_U,\dots,c_R)\in(\mathbb Z_4)^6:\;
c_U+\cdots+c_R\equiv 0\pmod 2
\Bigr\}.
}
```

这就是“普通魔方已还原后，图案还可能剩下的全部错误状态”。

#### 五、问题 1：错误面的个数 $n$ 有几种情况？

一个面错误，当且仅当该面的中心朝向 $c_i\neq 0$。

因此

```math
n=\#\{i:\;c_i\neq 0\}.
```

现在问 $n$ 能取哪些值。

由于“单个中心转 $180^\circ$”是允许的，所以对任意 $k=0,1,\dots,6$，都可以取

```math
(c_1,\dots,c_6)=
(\underbrace{2,\dots,2}_{k\text{ 个}},0,\dots,0),
```

它显然满足总和为偶数，因此是可实现的。

于是：

```math
\boxed{n\in\{0,1,2,3,4,5,6\}.}
```

也就是说，0 到 6 全部都可能。

例如：

$n=1$：恰有一个中心转 $180^\circ$；

$n=2$：可以是两个中心都转 $180^\circ$，也可以是一个 $+90^\circ$、一个 $-90^\circ$；

$n=3$：三个中心各转 $180^\circ$；

…

$n=6$：六个中心全都不对。

#### 六、问题 2：如果一个面错了，具体会怎么错？

现在可以精确回答：

定理 B
在普通 3\times3 已还原的前提下，一个错误面只可能表现为：该面的中心块朝向错误；该面周围八个块的图案方向已经完全正确，不会再有额外错误。

而中心块的错误只有三种非零可能：

```math
90^\circ,\quad 180^\circ,\quad 270^\circ.
```

但它们还受全局约束：

$180^\circ$ 错误可以单独出现；

$90^\circ$ / $270^\circ$ 错误必须成“总奇偶为偶”的方式出现；特别是单独一个 $90^\circ$ 面不可能。SpeedSolving Puzzles Community+1

所以更完整地说：

一个错误面可能只是中心转了 $180^\circ$；

若某面中心转了 $90^\circ$ 或 $270^\circ$，那么必然还有别的面也带着相应的 $90^\circ/270^\circ$ 型错误。

#### 七、最终结论

在“普通 3\times3 已经还原，只差图案方向”的意义下：

```math
\boxed{\text{错的只会是中心，不会是周围八块。}}
```

并且六个中心朝向构成的状态恰好满足

```math
\boxed{c_U+c_D+c_F+c_B+c_L+c_R\equiv 0\pmod 2.}
```

因此：

错误面的个数

```math
\boxed{n=0,1,2,3,4,5,6\ \text{都可能}.}
```

一个错误面的细节

严格地说，只有中心块方向会错；

周围八块在数学上已经完全正确；

“周围八块都是同一个方向”只是在箭头 supercube 这类特殊贴纸下呈现出来的视觉现象，不是一般性约束。
