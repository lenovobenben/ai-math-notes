# 正整数m,n

- Conversation ID: `6906dfad-17fc-8321-8206-152847328c78`

## Question

设正整数m,n均不大于2021，且 m/(n+1) < sqrt(2) < (m+1)/n 。则这样的数组(m,n)的个数为？

## Answer

先把不等式变形（$m,n$ 均为正整数，且不超过 2021）：

```math
\frac{m}{n+1}\lt{}\sqrt2\lt{}\frac{m+1}{n}
```

等价于

```math
\sqrt2\,n-1 \lt{} m \lt{} \sqrt2\,(n+1).
```

所以对每个 $n=1,2,\dots,2021$，可取的 $m$ 是区间 $(\sqrt2 n-1,\;\sqrt2(n+1))$ 中的整数，且还要满足 $1\le m\le 2021$。把所有 $n$ 的可行整数 $m$ 数目加起来就得到所求对数。

用该方法逐一统计（对每个 $n$ 计算区间内的整数个数并作上界 $m\le2021$ 的截断）得到总数为

```math
\boxed{3449}.
```

额外说明（可作检查）：在 $n=1,\dots,2021$ 中，得到 0 个 $m$ 的有 592 个 $n$，得到 2 个 $m$ 的有 838 个 $n$，得到 3 个 $m$ 的有 591 个 $n$，总对数为 $592\cdot0+838\cdot2+591\cdot3=3449$。
