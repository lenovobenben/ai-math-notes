# 四面体顶点证明

- Conversation ID: `6907534a-5290-8322-8551-5290193a678a`

## Question

求证：任意一个四面体必有一个顶点满足从它出发的三条棱可以做三角形的三边

## Answer

设四面体为 $ABCD$，记各条棱的长度为对应顶点字母间的线段长度（例如 $AB$ 表示棱 $AB$ 的长）。取六条棱中最长的一条，不妨记为 $AB$（即 $AB\ge$ 其它五条棱的长度）。

我们要证明：在顶点 $A$ 与顶点 $B$ 中至少有一个顶点，其出发的三条棱能构成三角形的三边。
用反证法：假设 A、B 两个顶点都不满足。那么

在顶点 $A$ 的三条棱 $AB,AC,AD$ 中，若不能构成三角形，则最长的那条（此处为 $AB$）满足

```math
AB\ge AC+AD.
```

同理，在顶点 $B$ 的三条棱 $BA,BC,BD$ 中（注意 $BA=AB$），若也不能构成三角形，则

```math
AB\ge BC+BD.
```

将上面两式相加，得

(1)

```math
2AB \ge (AC+AD)+(BC+BD).
```

另一方面，考虑三角形 $ABC$ 与三角形 $ABD$ 的三角不等式（四面体的任一面都是非退化三角形，故不等号为严格号）：

```math
AC+BC\gt{}AB,\qquad AD+BD\gt{}AB.
```

将二者相加得到

(2)

```math
(AC+AD)+(BC+BD) \gt{} 2AB.
```

(1) 与 (2) 明显矛盾。因此假设不成立：不可能两端点 $A$ 和 $B$ 都不满足条件。

由此得出结论：对于最长棱 $AB$ 的两个端点 $A$ 与 $B$ 中至少有一个顶点，其出发的三条棱可以构成三角形的三边。因为任意四面体中总有一条最长棱，所以上述结论对任意四面体成立。证毕。
