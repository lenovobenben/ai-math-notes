# 解常微分方程

- Conversation ID: `69023eaa-caec-8321-9cc1-a3d656931049`

## Question

解常微分方程。y+y'+y''+y'''+…=0

## Answer

把方程写成无穷级数

```math
\sum_{n=0}^{\infty} y^{(n)}(x)=0\qquad(\ast)
```

假设对每个 $x$ 该级数收敛（于是各项必趋于 0，即 $y^{(n)}(x)\to0$ 当 $n\to\infty$）。记部分和

```math
S_N(x)=\sum_{n=0}^{N} y^{(n)}(x).
```

对有限 $N$ 有

```math
(1-D)S_N(x)=S_N(x)-S_N'(x)=y(x)-y^{(N+1)}(x),
```

其中 $D=\dfrac{d}{dx}$。对 $N\to\infty$ 取极限，由假设 $S_N(x)\to0$（因为总和为 0）且 $y^{(N+1)}(x)\to0$，得到

```math
(1-D)\underbrace{0}_{\lim S_N}=y(x)-0,
```

因此 $y(x)=0$ 对所有 $x$ 成立。

结论：在要求无穷和对每个 $x$ 收敛（即各阶导数项趋于 0）的情形下，方程只有平凡解

```math
\boxed{y\equiv0.}
```

（补充说明：若放宽收敛条件或考虑广义函数/分布，情形可能不同；但在常规光滑函数且级数收敛的框架下只有零解。）
