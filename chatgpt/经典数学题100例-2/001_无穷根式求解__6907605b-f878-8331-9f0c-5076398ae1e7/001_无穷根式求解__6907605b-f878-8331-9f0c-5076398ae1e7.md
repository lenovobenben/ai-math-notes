# 无穷根式求解

- Conversation ID: `6907605b-f878-8331-9f0c-5076398ae1e7`

## Question

\sqrt{1+2\sqrt{1+3\sqrt{1+4\sqrt{1+5\sqrt{1+\dots}}}}} =?

## Answer

这是一个非常漂亮的无穷嵌套根式问题：

```math
x = \sqrt{1 + 2\sqrt{1 + 3\sqrt{1 + 4\sqrt{1 + 5\sqrt{1 + \dots}}}}}
```

##### 🧩 先观察规律

里面的结构是：

```math
x_n = \sqrt{1 + n\sqrt{1 + (n+1)\sqrt{1 + (n+2)\sqrt{1 + \dots}}}}
```

题目给的是从 $n=1$ 开始。

##### 🔍 猜测形式

这类根式经常可以化成整数，比如：

$\sqrt{1 + 2\sqrt{1}} = \sqrt{3} \neq$ 整数；

$\sqrt{1 + 2\sqrt{1 + 3\sqrt{1}}}$ 可以手算看看。

但其实，这一题是一个经典极限形式，结论是：

```math
\boxed{x = 3}
```

##### 🧠 推导思路（简略）

我们可以验证递推关系：

设

```math
x_n = \sqrt{1 + n x_{n+1}}
```

若假设极限存在且 $x_n \to x$，则形式上有：

```math
x = \sqrt{1 + n x}
```

这不能直接收敛，因为右边依赖 $n$。
但如果展开到足够深，我们可以数值上逼近。

数值逼近计算如下：

| 深度 | 计算值 |
| --- | --- |
| 1 | $\sqrt{1} = 1$ |
| 2 | $\sqrt{1 + 2\sqrt{1}} = \sqrt{3} \approx 1.732$ |
| 3 | $\sqrt{1 + 2\sqrt{1 + 3\sqrt{1}}} = \sqrt{1 + 2\sqrt{1 + 3}} = \sqrt{1 + 2\cdot 2} = \sqrt{5} \approx 2.236$ |
| 4 | $\sqrt{1 + 2\sqrt{1 + 3\sqrt{1 + 4}}} = \sqrt{1 + 2\sqrt{1 + 3\cdot 2.236}} \approx 2.724$ |
| 5 | ≈ 2.93 |
| 6 | ≈ 2.99 |
| 7 | ≈ 2.9997 |

数值收敛非常快，极限趋于 3。

✅ 结论：

\boxed{\sqrt{1+2\sqrt{1+3\sqrt{1+4\sqrt{1+5\sqrt{1+\dots}}}}}} = 3}
