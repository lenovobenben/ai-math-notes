# 旋转后重合函数

- Conversation ID: `6900d095-f9d8-8321-b528-18cbc3e80494`

## Question

是否存在ℝ上的函数f(x)，图象绕原点旋转90°后与原函数重合？

## Answer

我们来仔细分析这个问题：

##### ❓问题

是否存在一个实函数 $f:\mathbb{R}\to\mathbb{R}$，其图像绕原点旋转 $90^\circ$ 后，与原来的图像重合？

##### 🧭分析

原函数图像：
图像集合为：

```math
G = \{(x, f(x)) \mid x\in\mathbb{R}\}
```

绕原点旋转 $90^\circ$（逆时针）后的图像：
对任意点 $(x, y)$，旋转 $90^\circ$ 得：

```math
(x', y') = (-y, x)
```

因此，旋转后的图像集合为：

```math
G' = \{(-f(x), x) \mid x\in\mathbb{R}\}
```

重合条件：
若旋转后图像与原图像重合，则应有：

```math
G' = G
```

即：对任意 $x$，存在某个 $t\in\mathbb{R}$，使

```math
(-f(x), x) = (t, f(t))
```

于是得到方程组：

```math
\begin{cases}
t = -f(x) \\
f(t) = x
\end{cases}
```

将第一式代入第二式，得：

```math
f(-f(x)) = x, \quad \forall x\in\mathbb{R}.
```

##### 🔁 这意味着什么？

这个条件说明 $f$ 必须满足：

```math
f(-f(x)) = x
```

即

```math
f^{-1}(x) = -f(x)
```

所以 $f$ 是一个可逆函数，并且它的反函数等于自身的负函数。

##### 💡 寻找满足条件的函数

设 $f$ 是单调递增或单调递减的可逆函数，且满足：

```math
f^{-1}(x) = -f(x)
```

两边同时对 $f$ 作用：

```math
f(f^{-1}(x)) = f(-f(x)) = f(f^{-1}(x)) = x
```

与前面条件一致。

现在我们来猜测几种可能形式。

###### 尝试 1：线性函数 $f(x)=k x$

代入：

```math
f(-f(x)) = f(-k x) = k(-k x) = -k^2 x
```

要求 $f(-f(x))=x$，则

```math
- k^2 = 1 \quad \Rightarrow \quad k^2 = -1
```

没有实数解。

所以 线性函数不行。

###### 尝试 2：奇函数或反函数对称性？

若 $f$ 可逆且 $f^{-1}(x) = -f(x)$，则对于所有 $x$：

```math
f(f(x)) = -x
```

（将 $x$ 替换为 $f(x)$）。

这是一个重要等价条件。

所以我们需要找到满足：

```math
f(f(x)) = -x
```

的实函数。

##### 🔍 检查是否可能存在这样的函数

考虑对称关系：

第一次作用 $f$：

```math
x \mapsto f(x)
```

第二次作用 $f$：

```math
f(f(x)) = -x
```

再作用两次：

```math
f(f(f(f(x)))) = f(f(-x)) = -f(f(x)) = -(-x) = x
```

于是 $f$ 的四次迭代为恒等变换，即：

```math
f^{(4)}(x) = x
```

说明 $f$ 是一个“周期 4”的自同构。

##### 🔨 构造一个这样的函数

我们可以分段定义满足此条件的函数。例如：

定义

```math
f(x) =
\begin{cases}
\phantom{-}\sqrt{x^2+1} - 1, & x \ge 0,\\[4pt]
-\sqrt{x^2+1} + 1, & x \lt{} 0,
\end{cases}
```

这不满足 $f(f(x))=-x$。
所以我们要更系统地构造。

##### 🧩 代数结构角度

一个简单构造方法是：
取平面上的旋转 $90^\circ$ 视为映射 $(x,y)\mapsto (-y,x)$，
若要该图形在 $\mathbb{R}^2$ 上仍是一条函数图像 $y=f(x)$，
则旋转后的集合仍需是某个函数图像，即每个 $x$ 对应唯一 $y$。

但当图像与旋转后图像重合时，意味着在同一点上垂直方向和水平方向一一对应。
这种图像无法在实数上单值化（例如圆 $x^2+y^2=r^2$ 就能做到旋转后重合，但它不是函数图像）。

##### 🚫 结论

不存在这样的实函数 $f:\mathbb{R}\to\mathbb{R}$，
其图像绕原点旋转 $90^\circ$ 后仍与自身重合。

理由：

若存在，则必须满足 $f(f(x))=-x$；

这要求 $f$ 的定义域、值域完全对称地互换；

然而此时 $f$ 既要为实函数（单值），又要全定义在 $\mathbb{R}$ 上；

这样的连续实函数在实数范围内不存在（只能在复平面上成立，例如 $f(x)=i x$）。

✅ 最终结论：

在实数域 $\mathbb{R}$ 上，不存在使其图像绕原点旋转 $90^\circ$ 后与原图像重合的函数 $f(x)$。
若扩展到复平面，则 $f(z) = i z$ 是一个满足条件的复函数。

## Question

存在的。有一种分段函数，样子很像大风车

## Answer

对——你说得对，确实存在这样的函数，而且可以用“风车”式的分段构造给出一个很简单明白的例子。下面给出一个清晰的存在性构造（并验证），同时说明为什么它的图像绕原点旋转 $90^\circ$ 后与原图重合。

#### 关键等价关系

设 $G=\{(x,f(x))\}$ 为函数的图像。将点 $(x,y)$ 绕原点逆时针旋转 $90^\circ$ 得 $(-y,x)$。要使旋转后的图像等于原图像，等价于对任意 $x$ 存在 $t$ 使

```math
(-f(x),x)=(t,f(t)),
```

即等价于

```math
f(f(x))=-x\qquad(\forall x\in\mathbb R).
```

所以我们要构造一个满足 $f\circ f=-\mathrm{id}$ 的双射 $f:\mathbb R\to\mathbb R$（并令 $f(0)=0$）。

#### 构造（“大风车”式，分块定义）

把除 $0$ 外的实数分成四个互不相交的集合

```math
\mathbb R\setminus\{0\}=A\ \cup\ B\ \cup\ (-A)\ \cup\ (-B),
```

其中 $A$ 与 $B$ 是任意选定的互不相交集合，且它们与它们的取负 $-A,-B$ 一起覆盖了除 $0$ 外的所有实数。
（这样的划分很容易做：例如先把正实数分成两部分 $A_+$ 与 $B_+$，再把负实数设为对应的 $-A_+$ 与 $-B_+$。）

再任取一个双射（双向一一对应）$\varphi:A\to B$。现在定义 $f$ 如下：

```math
f(0)=0
```

对任意 $a\in A$： $f(a)=\varphi(a)\in B$.

对任意 $b\in B$： $f(b)=-\varphi^{-1}(b)\in -A$.

对任意 $-a\in -A$： $f(-a)=-\varphi(a)\in -B$.

对任意 $-b\in -B$： $f(-b)=\varphi^{-1}(-b)\in A$.

也就是说，沿着 $A\to B\to -A\to -B\to A$ 做一个 4 周期的置换（每一步用 $\varphi$ 或 $\varphi^{-1}$ 并在适当步骤加上取负）。

#### 验证 $f(f(x))=-x$

逐类检查（只列一类，其他类似）：

若 $x=a\in A$，则

```math
f(f(a))=f(\varphi(a))=-\varphi^{-1}(\varphi(a))=-a.
```

若 $x=b\in B$，则

```math
f(f(b))=f(-\varphi^{-1}(b))=\varphi^{-1}(b)\ (= -b\text{ 的负）} = -b.
```

对 $-A,-B,0$ 同样成立。

因此对任意 $x\in\mathbb R$ 都有 $f(f(x))=-x$。

#### 为什么图像像“风车”？

由 $f(f(x))=-x$ 可逆推出图像在平面上满足旋转关系：若 $(x,f(x))$ 在图上，则旋转后得 $(-f(x),x)$；但因为 $f(f(x))=-x$，点 $(-f(x),x)$ 仍然是某点 $(t,f(t))$（取 $t=f(x)$），因此旋转后的点还在原图上。所以图像在绕原点旋转 $90^\circ$ 后不变——视觉上就是四片“翼”轮流对应，像一个大风车（或四臂漩涡）。

#### 关于连续性、可构造性

这个构造通常产生不连续的函数（大多数自然的划分和任取双射不会是连续的）。若要求 $f$ 连续（或可微），则不存在非平凡的连续解（因为连续解会带来矛盾：$f(f(x))=-x$ 会使某些周期与单值性冲突）。所以此处的可行解通常是分段/不连续的——正是你说的“像大风车”的那种分段风格。

构造中用了任取分割与任取双射，基于这些可以显式写出很多具体例子；也可以用更“算法化”的分割（例如按对数的小数部分、按二进制首位等）来得到更易描述的具体 $A,B$ 与 $\varphi$。

#### 小结

是的，存在这样的实函数；一种自然的构造是把 $\mathbb R\setminus\{0\}$ 划成四个互不相交的部分 $A,B,-A,-B$，再在 $A$ 与 $B$ 之间选一个双射 $\varphi$，按上面规则定义 $f$。

这样的函数的图像就是“风车”样的——每旋转 $90^\circ$ 对应函数在四个“分区”上的循环。

请给出一个简单的示例

## Answer

好 —— 我给你一个可写出明确公式、结构很“风车”且尽量简单的示例。思路：把正实数按尾数 $r\in[1,2)$ 分成两份（靠 $\sqrt2$ 划分），再把负实数当作正实数的取负副本，沿着 $A\to B\to -A\to -B\to A$ 做四步循环。

##### 定义（完整且显式）

把任意非零实数写成

```math
x=\mathrm{sgn}(x)\,2^{n}r,
```

其中 $\mathrm{sgn}(x)=\pm1$、$n\in\mathbb Z$、尾数 $r\in[1,2)$（这是把正数按二的幂分解得到的标准表示）。

定义集合

```math
A=\{x\gt{}0:\;r\in[1,\sqrt2)\},\quad B=\{x\gt{}0:\;r\in[\sqrt2,2)\},
```

以及 $-A,\,-B$ 为它们的取负。显然 $\mathbb R\setminus\{0\}=A\cup B\cup(-A)\cup(-B)$。

现在定义函数 $f:\mathbb R\to\mathbb R$：

```math
\boxed{f(0)=0}
```

若 $x\ne0$，写 $x=\mathrm{sgn}(x)\,2^{n}r$（$r\in[1,2)$），则

```math
f(x)=
\begin{cases}
\phantom{-}\mathrm{sgn}(x)\,2^{n}\,(r\sqrt2), & r\in[1,\sqrt2),\\[6pt]
-\,\mathrm{sgn}(x)\,2^{n}\,\dfrac{r}{\sqrt2}, & r\in[\sqrt2,2).
\end{cases}
```

（换句话说：若 $x\in A$ 就把尾数乘以 $\sqrt2$ 映到 $B$；若 $x\in B$ 就把尾数除以 $\sqrt2$ 并取负，映到 $-A$；对 $-A,-B$ 按自然规则得到的定义与上式一致。）

##### 验证 $f(f(x))=-x$

示例检验两种情况（其余同理）：

若 $x\in A$，即 $x=\mathrm{sgn}(x)\,2^{n}r$ 且 $r\in[1,\sqrt2)$。
那么 $f(x)=\mathrm{sgn}(x)\,2^{n}(r\sqrt2)$，此时尾数变为 $r\sqrt2\in[\sqrt2,2)$，所以 $f(x)\in B$。对它再应用 $f$，按第二条得

```math
f(f(x)) = -\,\mathrm{sgn}(x)\,2^{n}\,\frac{r\sqrt2}{\sqrt2} = -\,\mathrm{sgn}(x)\,2^{n}r = -x.
```

若 $x\in B$，即尾数 $r\in[\sqrt2,2)$。
首次应用 $f$ 给出 $f(x)=-\mathrm{sgn}(x)\,2^{n}(r/\sqrt2)$，这个数属于 $-A$（负数且尾数在 $[1,\sqrt2)$）；再对它应用 $f$，按 $-A$ 的规则会得到负号再乘以 $\sqrt2$，最后回到 $-x$。

因此对任意 $x$ 都有 $f(f(x))=-x$。

##### 直观与性质

图像看起来像四臂旋转的“风车”：每个点经一次 $f$ 跳到下一臂，二次 $f$ 得到相反点（对应 90° 旋转关系），所以图像绕原点旋转 $90^\circ$ 后不变。

这个 $f$ 是双射且满足 $f\circ f = -\mathrm{id}$，但一般 处处不连续（在尾数为 $\sqrt2$ 的边界处有跳变），这是典型“风车”构造的特征。

如果需要，可以把上面的“按二的幂分解 + 按尾数分区”的办法替换成其它可计算规则（例如用对数的小数部分分区），得到同样类型的明确例子。
