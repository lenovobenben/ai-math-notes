# 计算矩阵交换子

- Conversation ID: `69087523-f348-8322-b74a-f39ce7eea214`

## Question

复方阵A,B满足A^2-2AB-B^2=0。请计算|AB-BA|=？

## Answer

答案：$|AB-BA|=0$。

证明思路（简洁版）：

记交换子 $C:=AB-BA$。由条件

```math
A^2-2AB-B^2=0
```

对 $B$ 取对易得

```math
[B,\,A^2-2AB-B^2]=0,
```

展开并整理得到一个关于 $C$ 的恒等式

(1)

```math
CA+AC=2CB.
```

同样对 $A$ 作对易并整理可以得到

(2)

```math
BC+CB=0.
```

由 (2) 可得 $B^2$ 与 $C$ 对易，从而由原方程可写出

```math
C=(A-B)^2-2B^2,
```

可见 $C$ 与 $A-B$ 对易。于是 $C$ 与 $A,B$ 生成的代数中的元素相容，进一步可用代数方法（例如 Engel/上三角化论断）得出 $C$ 是幂零元（即存在 $k$ 使得 $C^k=0$），所以 $C$ 的特征值全为 $0$。因此

```math
\det(AB-BA)=\det C=0.
```

（数值检验和若干代数推导都支持上述结论；若需要，我可以把中间代数恒等式的逐步展开写得更详细。）
