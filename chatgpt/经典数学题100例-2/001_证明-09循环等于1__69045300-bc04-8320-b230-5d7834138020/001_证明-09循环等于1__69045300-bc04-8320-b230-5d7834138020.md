# 证明 0.9循环等于1

- Conversation ID: `69045300-bc04-8320-b230-5d7834138020`

## Question

0.\overline{9} = 1 如何严格证明？

## Answer

你问的是经典问题：为什么 $0.\overline{9} = 1$。我来给你一个严格数学证明，可以从几种角度来看。我们用最常见的几种方法：

##### 方法 1：利用等比数列

记：

```math
0.\overline{9} = 0.9999\ldots
```

这是一个无限小数，可以表示为无穷级数：

```math
0.9 + 0.09 + 0.009 + \dots
```

这是一个等比数列，首项 $a = 0.9$，公比 $r = 0.1$。无限等比数列的和公式是：

```math
S = \frac{a}{1-r}, \quad \text{当 } |r|\lt{}1
```

代入：

```math
S = \frac{0.9}{1-0.1} = \frac{0.9}{0.9} = 1
```

所以严格得出：

```math
0.\overline{9} = 1
```

✅ 这是严格的分析证明。

##### 方法 2：利用代数方法

设：

```math
x = 0.\overline{9}
```

两边同时乘以 10：

```math
10x = 9.\overline{9}
```

两式相减：

```math
10x - x = 9.\overline{9} - 0.\overline{9} \implies 9x = 9 \implies x = 1
```

所以：

```math
0.\overline{9} = 1
```

这个方法也严格，基于极限和无穷循环小数的定义。

##### 方法 3：利用极限定义

定义 $0.\overline{9}$ 为序列：

```math
0.9, \, 0.99, \, 0.999, \dots
```

记作 $(s_n)$，其中 $s_n = 1 - 10^{-n}$。显然：

```math
\lim_{n\to\infty} s_n = \lim_{n\to\infty} (1 - 10^{-n}) = 1
```

所以根据极限定义：

```math
0.\overline{9} = \lim_{n\to\infty} s_n = 1
```

这是最严格的分析学证明。
