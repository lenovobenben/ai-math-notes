# 二次函数旋转

- Conversation ID: `6901fef0-6d70-8322-b352-d222b4cf8123`

## Question

y=x²绕原点旋转多少度后仍然是函数？
nπ这种特殊情况除外。

## Answer

我们来仔细分析这个问题：

题目：函数 $y = x^2$ 绕原点旋转多少度后仍然是函数？排除 $n\pi$ 的情况。

##### 第一步：旋转公式

将平面上的点 $(x, y)$ 绕原点逆时针旋转 $\theta$ 角后的坐标为：

```math
\begin{cases}
x' = x \cos \theta - y \sin \theta \\
y' = x \sin \theta + y \cos \theta
\end{cases}
```

##### 第二步：代入 $y = x^2$

旋转后的点 $(x', y')$：

```math
x' = x \cos \theta - x^2 \sin \theta
```

```math
y' = x \sin \theta + x^2 \cos \theta
```

旋转后想保持“函数”，要求 对于每个 $x'$ 只有唯一的 $y'$，即满足垂直线测试。

换句话说，如果我们能解 $x$ 关于 $x'$ 的表达式，旋转后的图像才是函数。

##### 第三步：分析函数性

考虑旋转后：

```math
x' = x \cos \theta - x^2 \sin \theta = -\sin\theta x^2 + \cos\theta x
```

这是一个二次方程关于 $x$ 的式子：

```math
-\sin\theta x^2 + \cos\theta x - x' = 0
```

若旋转后的曲线仍是函数，要求这个二次方程 关于 x 至多有一个解。

二次方程判别式：

```math
\Delta = (\cos\theta)^2 - 4(-\sin\theta)(-x') = \cos^2\theta - 4 \sin\theta x'
```

对于任意 $x'$ 都成立“至多一个解”的条件：

$\Delta$ 必须恒小于等于 0（否则可能有两个不同的 x 对应同一个 x'），

或者方程退化为一阶，即二次项系数为 0：$-\sin\theta = 0 \Rightarrow \theta = n\pi$（题目已排除这种情况）。

如果 $\theta \neq n\pi$，二次项不为零，$\Delta$ 会随 $x'$ 变化而变化。

当 $x'$ 很小或很大时，$\Delta > 0$ 的情况必然出现。

所以不存在其他角度能保持函数性。

##### ✅ 第四步：结论

排除 $n\pi$ 的情况，没有其他角度可以使 $y = x^2$ 绕原点旋转后仍是函数。

答案：

```math
\boxed{\text{没有，只有 } \theta = n\pi \text{ 保持函数性}}
```
