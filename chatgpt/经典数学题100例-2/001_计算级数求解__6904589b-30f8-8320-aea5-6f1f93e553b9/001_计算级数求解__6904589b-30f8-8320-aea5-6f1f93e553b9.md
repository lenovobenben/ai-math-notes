# 计算级数求解

- Conversation ID: `6904589b-30f8-8320-aea5-6f1f93e553b9`

## Question

计算级数：

```math
\sum_{n=0}^{\infty} \frac{n^{2^2}}{2^n}
```

## Answer

你想计算的级数是

```math
\sum_{n=0}^{\infty} \frac{n^{2^2}}{2^n} = \sum_{n=0}^{\infty} \frac{n^4}{2^n}.
```

这是一个指数生成函数类型的求和问题。我们可以用公式：

对于 $|x|<1$ 的几何级数：

```math
\sum_{n=0}^{\infty} n^k x^n
```

可以通过对几何级数求导得到。

##### 第一步：基础公式

已知几何级数：

```math
\sum_{n=0}^{\infty} x^n = \frac{1}{1-x}, \quad |x|\lt{}1
```

对 $x$ 求导：

```math
\sum_{n=0}^{\infty} n x^{n-1} = \frac{1}{(1-x)^2} \implies \sum_{n=0}^{\infty} n x^n = \frac{x}{(1-x)^2}.
```

进一步求导可以得到更高次幂的公式：

```math
\sum_{n=0}^{\infty} n^2 x^n = \frac{x + x^2}{(1-x)^3}, \quad
\sum_{n=0}^{\infty} n^3 x^n = \frac{x + 4x^2 + x^3}{(1-x)^4}, \quad
\sum_{n=0}^{\infty} n^4 x^n = \frac{x + 11x^2 + 11x^3 + x^4}{(1-x)^5}.
```

##### 第二步：代入 $x = \frac{1}{2}$

```math
\sum_{n=0}^{\infty} \frac{n^4}{2^n} = \sum_{n=0}^{\infty} n^4 \left(\frac{1}{2}\right)^n = \frac{\frac{1}{2} + 11 (\frac{1}{2})^2 + 11 (\frac{1}{2})^3 + (\frac{1}{2})^4}{(1-\frac{1}{2})^5}.
```

先计算分子：

```math
\frac{1}{2} + 11 \cdot \frac{1}{4} + 11 \cdot \frac{1}{8} + \frac{1}{16}
= \frac{1}{2} + \frac{11}{4} + \frac{11}{8} + \frac{1}{16}.
```

通分成 16：

```math
\frac{8}{16} + \frac{44}{16} + \frac{22}{16} + \frac{1}{16} = \frac{75}{16}.
```

分母：

```math
(1 - \frac{1}{2})^5 = \left(\frac{1}{2}\right)^5 = \frac{1}{32}.
```

所以整个级数：

```math
\sum_{n=0}^{\infty} \frac{n^4}{2^n} = \frac{75/16}{1/32} = \frac{75}{16} \cdot 32 = 150.
```

✅ 答案：

```math
\boxed{150}
```
