# Sard定理解释

- Conversation ID: `6908a207-4eb4-8331-b36e-2b8aca538601`

## Question

Sard定理是什么？

## Answer

Sard 定理（Sard’s Theorem） 是微分拓扑中的一个非常重要的结果，主要研究光滑映射的“临界值”集合的测度性质。它揭示了一个深刻的事实：尽管光滑函数可能在很多点上导数退化（即存在临界点），但这些“坏点”对应的像在目标空间中几乎是“零测”的。

##### 🧠 一、定理的基本内容

设

```math
f: \mathbb{R}^n \to \mathbb{R}^m
```

是一个 $C^k$（$k \ge \max(n - m + 1, 1)$）的光滑函数。

定义：

临界点（critical point）：导数（Jacobian矩阵）$Df(x)$ 的秩小于 $m$ 的点；

临界值（critical value）：这些临界点在 $f$ 下的像的集合。

📜 Sard 定理表述：

对于 $f: \mathbb{R}^n \to \mathbb{R}^m$ 的 $C^k$ 映射（其中 $k > \max(n - m, 0)$），
它的所有临界值组成的集合在 $\mathbb{R}^m$ 中的 Lebesgue 测度为 0。

换句话说：

```math
\text{Lebesgue measure}(f(\text{Crit}(f))) = 0.
```

##### 🧩 二、直观理解

“临界点”是导数退化的点（比如极值点、鞍点、拐点）。

Sard 定理告诉我们：虽然这些点可能很多（甚至是无穷多），但它们在像空间中“占的地方非常小”，几乎可以忽略不计。

也就是说：几乎所有值都是正则值（regular values），即其原像都是流形（由隐函数定理保证）。

##### 🧭 三、一个常见例子

设

```math
f(x) = x^3, \quad f: \mathbb{R} \to \mathbb{R}.
```

临界点：$f'(x) = 3x^2 = 0 \Rightarrow x = 0$；

临界值：$f(0) = 0$。

\to 临界值集合只有一个点 $\{0\}$，其测度为 0。
这与 Sard 定理一致。

##### 🧮 四、几何与拓扑意义

Sard 定理是许多重要理论的基础：

隐函数定理 + Sard 定理 ⇒ 正则值的原像是流形；

Morse 理论 中，关键在于研究临界点；

微分同胚 与 横截性理论（Transversality） 的证明都依赖 Sard 定理；

代数几何、微分几何 中用于说明“几乎所有”参数下某性质都成立。

##### 📘 五、扩展版本

在流形之间的映射

```math
f: M^n \to N^m
```

若 $f \in C^k$ 且 $k > \max(n - m, 0)$，则同样成立：临界值集在 $N$ 中测度为 0。

##### ⚙️ 六、小结

| 概念 | 含义 |
| --- | --- |
| 临界点 | Jacobian 秩小于满秩的点 |
| 临界值 | 临界点的像 |
| Sard 定理结论 | 临界值的测度为 0 |
| 含义 | “几乎所有值”都是正则的 |
| 要求 | $f \in C^k, \; k > \max(n - m, 0)$ |
