# 求a+b+c解法

- Conversation ID: `6906c145-5144-8320-aa58-f3dc9756ab55`

## Question

a,b,c为复数。满足：
```math
a^2=b-c
b^2=c-a
c^2=a-b
```
求 a+b+c

## Answer

设 $S=a+b+c$. 由题给条件

```math
\begin{cases}
a^2=b-c,\\[4pt]
b^2=c-a,\\[4pt]
c^2=a-b,
\end{cases}
```

两边相加得

(1)

```math
a^2+b^2+c^2=0.
```

两式分别乘以 $a,b,c$ 后相加得

(2)

```math
a^3+b^3+c^3=0.
```

记 $p_1=a+b+c=S,\; p_2=a^2+b^2+c^2,\; p_3=a^3+b^3+c^3$，以及初等对称式 $e_1=S,\; e_2=ab+bc+ca,\; e_3=abc$。
由 Newton 关系有

```math
p_2=S^2-2e_2,\quad p_3=S^3-3Se_2+3e_3.
```

将已知 $p_2=0,\; p_3=0$ 代入得

```math
0=S^2-2e_2\implies e_2=\frac{S^2}{2},
```

```math
0=S^3-3S\cdot\frac{S^2}{2}+3e_3\implies e_3=\frac{S^3}{6}.
```

再用 Newton 第四式

```math
p_4=a^4+b^4+c^4 = S p_3 - e_2 p_2 + e_3 p_1 = 0- e_2\cdot0 + e_3 S = e_3 S = \frac{S^4}{6}.
```

另一方面，由差的平方恒等式并利用 $b-c=a^2,\; c-a=b^2,\; a-b=c^2$ 可得

```math
(a-b)^2+(b-c)^2+(c-a)^2 = c^4+a^4+b^4 = a^4+b^4+c^4.
```

但左边又等于

```math
2(a^2+b^2+c^2)-2(ab+bc+ca)= -2e_2 = -2\cdot\frac{S^2}{2} = -S^2.
```

于是

```math
a^4+b^4+c^4 = -S^2.
```

将 $a^4+b^4+c^4=\dfrac{S^4}{6}$ 与上式合并得到

```math
\frac{S^4}{6} = -S^2 \quad\Longrightarrow\quad S^2(S^2+6)=0.
```

因此

```math
S= a+b+c \in\{\,0,\;+\mathrm{i}\sqrt{6},\;-\mathrm{i}\sqrt{6}\,\}.
```

（其中 $S=0$ 对应例如 $a=b=c=0$；也存在使得 $S=\pm i\sqrt6$ 的非平凡复数解。）
