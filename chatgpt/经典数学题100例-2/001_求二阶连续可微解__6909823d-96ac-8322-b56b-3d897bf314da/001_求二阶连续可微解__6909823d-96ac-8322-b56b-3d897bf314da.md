# 求二阶连续可微解

- Conversation ID: `6909823d-96ac-8322-b56b-3d897bf314da`

## Question

求所有二阶连续可微函数f，使得f满足f(\sqrt{2} t) = f^2(t)

## Answer

答案：所有解为

```math
f\equiv0\qquad\text{或}\qquad f(t)=\exp(a t^2)\ (a\in\mathbb{R}).
```

下面给出完整证明（假设方程对所有实数 $t$ 成立，且 $f\in C^2(\mathbb R)$）。

符号情况：由方程

```math
f(\sqrt2\,t)=f(t)^2
```

右端为非负数，故左端对所有 $t$ 亦非负，所以 $f(t)\ge0$ 对任意 $t$ 成立。若存在某点 $t_0$ 使得 $f(t_0)=0$，则

```math
f(\sqrt2\,t_0)=f(t_0)^2=0.
```

由方程可向两侧“扩展”零点：若 $f(\sqrt2 s)=0$ 则 $f(s)=0$（取平方根方向），于是由 $f(t_0)=0$ 得出对任意整数 $k$，$f((\sqrt2)^k t_0)=0$。把序列向负方向扩展可得到 $f(0)=0$。对任意固定 $t$ 有迭代关系

```math
f(t)=\bigl(f(t/(\sqrt2)^n)\bigr)^{2^n}.
```

当 $n\to\infty$ 时，$t/(\sqrt2)^n\to0$ 而 $f(0)=0$，所以右侧极限为 $0$，因此 $f(t)=0$ 对任意 $t$ 成立。即：要么 $f$ 处处为 $0$。

严格正值的情况：若 $f(t)>0$ 对所有 $t$ 成立，则可取对数令 $h(t)=\ln f(t)$。则方程等价于

```math
h(\sqrt2\,t)=2h(t)\qquad(\forall t).
```

对两边关于 $t$ 求导（$h$ 由 $f$ 的 $C^2$ 性质得到 $C^2$）：

```math
\sqrt2\,h'(\sqrt2 t)=2h'(t)\quad\Rightarrow\quad h'(\sqrt2 t)=\sqrt2\,h'(t),
```

再对一次求导得

```math
2\,h''(\sqrt2 t)=2\,h''(t)\quad\Rightarrow\quad h''(\sqrt2 t)=h''(t).
```

因此 $h''$ 在乘以 $\sqrt2$ 的缩放下保持不变。对任意固定 $y\ne0$，有

```math
h''(y)=h''\bigl(y(\sqrt2)^n\bigr)\quad(\forall n\in\mathbb Z).
```

当 $n\to-\infty$ 时，$y(\sqrt2)^n\to0$，由 $h''$ 的连续性得

```math
h''(y)=\lim_{n\to-\infty}h''\bigl(y(\sqrt2)^n\bigr)=h''(0).
```

于是 $h''$ 在全体 $\mathbb R$ 上为常数，记为 $2a$。故 $h$ 为二次多项式

```math
h(t)=a t^2 + b t + c.
```

将其代回方程 $h(\sqrt2 t)=2h(t)$：

```math
a\cdot 2 t^2 + b\sqrt2 t + c =2(a t^2 + b t + c).
```

比较各项系数得 $b=0$、$c=0$，而 $a$ 任取实数均满足。故

```math
h(t)=a t^2,\qquad f(t)=e^{h(t)}=e^{a t^2},\quad a\in\mathbb R.
```

汇总：结合两种情况，所有 $C^2$ 解为 $f\equiv0$ 或 $f(t)=\exp(a t^2)$（包含常数 $1$ 当 $a=0$）。

（检验：直接代入可见 $\exp(a(\sqrt2 t)^2)=\exp(2a t^2)=(\exp(a t^2))^2$。）

## Question

嗯。这里的a其实是(f''(0))/2

## Answer

完全正确！我们在上面的推导里得到 $h(t) = a t^2$，而 $h(t) = \ln f(t)$。由于 $h''(t) = 2a$ 是常数，所以

```math
h''(0) = 2a \quad\implies\quad a = \frac{h''(0)}{2} = \frac{f''(0)}{2},
```

因为在 $t=0$ 处，$f(0) = e^{h(0)} = 1$（假设 $f$ 非零且 $f(0)>0$），由链式法则有

```math
f''(0) = \bigl(e^{h(t)}\bigr)''\big|_{t=0} = h''(0) e^{h(0)} + (h'(0))^2 e^{h(0)} = h''(0),
```

因为 $h'(0)=0$。所以确实可以把指数里的系数 $a$ 写成

```math
f(t) = \exp\Bigl(\frac{f''(0)}{2} t^2\Bigr).
```

这个写法更直接地和函数在 $0$ 点的二阶导数联系起来。
