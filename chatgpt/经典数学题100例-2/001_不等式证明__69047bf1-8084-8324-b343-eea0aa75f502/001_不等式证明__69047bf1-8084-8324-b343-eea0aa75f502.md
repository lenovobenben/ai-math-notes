# 不等式证明

- Conversation ID: `69047bf1-8084-8324-b343-eea0aa75f502`

## Question

```math
\frac{a^2 + b^2}{a+b}
\ \ge\
\sqrt{\frac{a^2 + b^2}{2}}
\ \ge\
\frac{2(a^2+ab+b^2)}{3(a+b)}
\ \ge\
\frac{a + b}{2}
\ \ge\
\frac{a + \sqrt{ab} + b}{3}
\ \ge\
\sqrt{ab}
\ \ge\
\frac{2}{\dfrac{1}{a} + \dfrac{1}{b}}
```

a,b均大于0。请证明不等式

## Answer

下面逐项证明（设 $a,b>0$）。对每一项我都给出等价变形并说明用到的不等式；所有等号同时成立的情形为 $a=b$。

记

```math
S=a+b,\qquad Q=a^2+b^2,\qquad G=\sqrt{ab}.
```

(1) $\displaystyle\frac{a^2+b^2}{a+b}\ \ge\ \sqrt{\frac{a^2+b^2}{2}}$.

等价于

```math
\frac{Q}{S}\ge \sqrt{\tfrac{Q}{2}}
\iff \frac{Q^2}{S^2}\ge \frac{Q}{2}
\iff \frac{Q}{S^2}\ge\frac12
\iff 2Q\ge S^2.
```

但 $S^2=a^2+2ab+b^2=Q+2ab$，所以 $2Q\ge Q+2ab\iff Q\ge2ab$，即
$(a-b)^2\ge0$。因此成立（且当且仅当 $a=b$ 取等号）。

(2) $\displaystyle\sqrt{\frac{a^2+b^2}{2}}\ \ge\ \frac{2(a^2+ab+b^2)}{3(a+b)}$.

两边平方并通分，等价于

```math
\frac{9}{2}(a+b)^2(a^2+b^2)\ \ge\ 4(a^2+ab+b^2)^2.
```

将左减右并因式分解（或直接展开整理）得

```math
\frac{9}{2}S^2Q-4(a^2+ab+b^2)^2=\frac{1}{2}(a-b)^2\bigl(a^2+4ab+b^2\bigr)\ge0,
```

因此原不等式成立。（等号当且仅当 $a=b$）

(3) $\displaystyle\frac{2(a^2+ab+b^2)}{3(a+b)}\ \ge\ \frac{a+b}{2}$.

移项得

```math
4(a^2+ab+b^2)\ge 3(a+b)^2.
```

左减右等于

```math
4(a^2+ab+b^2)-3(a+b)^2=(a-b)^2\ge0,
```

所以成立（等号当且仅当 $a=b$）。

(4) $\displaystyle\frac{a+b}{2}\ \ge\ \frac{a+G+b}{3}$.

两边同乘以 $6$ 得等价形式

```math
3(a+b)\ge 2(a+G+b)\iff a+b\ge 2G,
```

这是算术-几何不等式（AM–GM），成立（等号当且仅当 $a=b$）。

(5) $\displaystyle\frac{a+G+b}{3}\ \ge\ G$.

把 $a,\;G,\;b$ 作为三正数应用 AM–GM：

```math
\frac{a+G+b}{3}\ge \sqrt[3]{a\cdot G\cdot b}=\sqrt[3]{a\cdot\sqrt{ab}\cdot b}
=\sqrt[3]{a^{3/2}b^{3/2}}=\sqrt{ab}=G.
```

等号当且仅当 $a=G=b$，即 $a=b$。

(6) $\displaystyle G\ \ge\ \frac{2}{\frac{1}{a}+\frac{1}{b}}$.

右边是两数 $a,b$ 的调和平均 $H=\dfrac{2ab}{a+b}$。不等式等价于

```math
\sqrt{ab}\ge \frac{2ab}{a+b}\iff a+b\ge 2\sqrt{ab},
```

这又是 AM–GM。故成立（等号当且仅当 $a=b$）。

综上，链式不等式

```math
\frac{a^2 + b^2}{a+b}
\ \ge\
\sqrt{\frac{a^2 + b^2}{2}}
\ \ge\
\frac{2(a^2+ab+b^2)}{3(a+b)}
\ \ge\
\frac{a + b}{2}
\ \ge\
\frac{a + \sqrt{ab} + b}{3}
\ \ge\
\sqrt{ab}
\ \ge\
\frac{2}{\dfrac{1}{a} + \dfrac{1}{b}}
```

对任意正 $a,b$ 都成立；且每一处等号同时成立的唯一情况是 $a=b$。

## Question

这7个表达式哪些是经常用到的？

## Answer

非常好的问题 👍。
这 7 个量虽然都能按大小排出链式不等式，但它们的“常用程度”差别很大。下面我按使用频率和场景给出说明（假设 $a,b>0$）：

##### ✅ 最常用的 3 个

这些几乎是数学竞赛、物理、数据分析、工程计算等都常出现的：

| 名称 | 表达式 | 常见缩写 | 常用原因 |
| --- | --- | --- | --- |
| 算术平均 (Arithmetic Mean, AM) | $\displaystyle \frac{a+b}{2}$ | AM | 基本平均值，几乎所有加权或不等式的起点。 |
| 几何平均 (Geometric Mean, GM) | $\displaystyle \sqrt{ab}$ | GM | 出现在 AM–GM 不等式、比例增长、对称表达式、调和级数中。 |
| 调和平均 (Harmonic Mean, HM) | $\displaystyle \frac{2}{\frac{1}{a}+\frac{1}{b}}$ | HM | 常用于“速率–时间”问题、等效电阻/电导等。 |

👉 它们三者的经典不等式关系是：

```math
\text{AM} \ge \text{GM} \ge \text{HM},
```

即

```math
\frac{a+b}{2} \ge \sqrt{ab} \ge \frac{2}{\frac{1}{a}+\frac{1}{b}}.
```

这条是极常用的标准链式不等式。

##### ⚙️ 中等常用的 2 个

主要用于竞赛题、不等式强化、数学分析或对称函数中：

| 表达式 | 名称或来源 | 说明 |
| --- | --- | --- |
| $\displaystyle \sqrt{\frac{a^2+b^2}{2}}$ | 平方平均（RMS, Root Mean Square） | 在统计、信号分析、误差分析中常见，比如 RMS 电压、电流。与 AM、GM 的关系：$\text{RMS} \ge \text{AM} \ge \text{GM}$。 |
| $\displaystyle \frac{a+b+\sqrt{ab}}{3}$ | 有时称为“算几均”或“混合平均” | 在一些不等式链或竞赛题中出现，不是常规定义，但形式简洁、对称。 |

##### 🧮 较少用但有特殊意义的 2 个

这两个形式比较“人造”，用于构造长链、比较函数性质时会见到：

| 表达式 | 特点 |
| --- | --- |
| $\displaystyle \frac{2(a^2+ab+b^2)}{3(a+b)}$ | 很少单独定义，有时作为“加权二次平均”，在不等式练习中出现。 |
| $\displaystyle \frac{a^2+b^2}{a+b}$ | 偶尔出现在对称分式题中，但不是常见平均。用于证明链首端的放大性。 |

##### 🔚 总结排序（按常见程度）

| 级别 | 表达式 | 名称 |
| --- | --- | --- |
| ⭐⭐⭐ | $\frac{a+b}{2}$、$\sqrt{ab}$、$\frac{2}{\frac{1}{a}+\frac{1}{b}}$ | AM, GM, HM |
| ⭐⭐ | $\sqrt{\frac{a^2+b^2}{2}}$、$\frac{a+b+\sqrt{ab}}{3}$ | RMS, 混合平均 |
| ⭐ | $\frac{2(a^2+ab+b^2)}{3(a+b)}$、$\frac{a^2+b^2}{a+b}$ | 人造加权平均 |

是否希望我给出这 7 个“平均数”的几何意义（比如在数轴或坐标平面上对应点的相对位置和解释）？那样可以非常直观地看到为什么它们会按这个顺序排列。
