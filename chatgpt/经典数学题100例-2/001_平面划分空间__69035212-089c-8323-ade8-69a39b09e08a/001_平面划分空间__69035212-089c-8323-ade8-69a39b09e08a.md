# 平面划分空间

- Conversation ID: `69035212-089c-8323-ade8-69a39b09e08a`

## Question

n个平面最多可以将空间划分成多少份？

## Answer

设有 $n$ 个平面，最多能把三维空间分成的区域数为

```math
R(n)=1+\binom{n}{1}+\binom{n}{2}+\binom{n}{3}.
```

把组合数展开成多项式形式：

```math
R(n)=1+n+\frac{n(n-1)}{2}+\frac{n(n-1)(n-2)}{6}
=\frac{n^{3}+5n+6}{6}.
```

推导要点（快速说明）：
每增加第 $k$ 个平面，它在已有 $k-1$ 个平面所形成的交线上被划分成最多

```math
1+\binom{k-1}{1}+\binom{k-1}{2}
```

个面片，因此新增的区域数等于该面片数，对所有 $k$ 求和得上式。

举例：

$n=0$：$R(0)=1$（整个空间）

$n=1$：

```math
R(1)=2
```

$n=2$：

```math
R(2)=4
```

$n=3$：

```math
R(3)=8
```

$n=4$：

```math
R(4)=15
```

## Question

好的。假设d维空间。n个超平面。

## Answer

在 $d$ 维空间中，若 $n$ 个超平面处于一般位置（即任意 $m\le d$ 个超平面交成维数为 $d-m$ 的平面，且没有多余的退化重合），则它们把空间最多划分的区域数为

```math
R_d(n)=\sum_{i=0}^{d}\binom{n}{i}.
```

（当 $n<d$ 时，组合数在 $i>n$ 时为 0，公式仍然成立。）

常见特例：

$d=1:\ R_1(n)=\binom{n}{0}+\binom{n}{1}=1+n$（把直线上成 $n+1$ 段）。

$d=2:\ R_2(n)=1+n+\binom{n}{2}=\frac{n(n+1)}{2}+1$（平面上最多分成的区域数）。

$d=3:\ R_3(n)=1+n+\binom{n}{2}+\binom{n}{3}$，与你之前的结果一致。

证明要点（递推法）：
加入第 $n$ 个超平面时，它与已有的 $n-1$ 个超平面的交集在该超平面内构成一个 $(d-1)$-维超平面安排，这个安排最多把该超平面分成 $R_{d-1}(n-1)$ 个区域。于是有递推关系

```math
R_d(n)=R_d(n-1)+R_{d-1}(n-1).
```

以 $R_d(0)=1$ 为初始条件递推并用组合数恒等式（帕斯卡恒等式）即可得闭式和 $\sum_{i=0}^d\binom{n}{i}$。

## Question

和这个等价吧？
C_{n}^{d} + C_{n}^{d-1} + C_{n}^{d-2} + \cdots + C_{n}^{1} + C_{n}^{0}

## Answer

是的，完全等价。

你写的

```math
C_n^d + C_n^{d-1} + \cdots + C_n^1 + C_n^0
```

就是

```math
\sum_{i=0}^d \binom{n}{i},
```

与我之前给出的 $R_d(n)=\sum_{i=0}^d\binom{n}{i}$ 表达式是同一个式子（$\binom{n}{i}=C_n^i$）。

补充几点说明：

该公式在假设 $n$ 个超平面处于一般位置（any $m\le d$ 个超平面交成维数 $d-m$，没有退化重合）时成立；退化情况会导致划分数变小。

若 $d\ge n$，右端求和到 $d$ 实际上等于求和到 $n$，因此

```math
R_d(n)=\sum_{i=0}^n\binom{n}{i}=2^n,
```

也就是在维度至少为 $n$ 时 $n$ 个一般位置的超平面最多把空间分成 $2^n$ 份（对应每个超平面的正负侧组合）。

递推关系仍然是

```math
R_d(n)=R_d(n-1)+R_{d-1}(n-1),
```

可用于证明该闭式。

## Question

C和()是两种书写方式对吧？

## Answer

完全正确。😊

$C_n^k$ 是组合数的另一种常见记法，读作 “从 $n$ 个元素中取 $k$ 个的组合数”。

$\binom{n}{k}$ 是 LaTeX / 数学教材中更标准的记法，读作 “$n$ 选 $k$”。

它们等价：

```math
C_n^k = \binom{n}{k} = \frac{n!}{k!(n-k)!}.
```

比如：

```math
C_5^2 = \binom{5}{2} = 10.
```

所以公式里写 $C_n^i$ 或 $\binom{n}{i}$ 都是一样的，只是风格不同。
