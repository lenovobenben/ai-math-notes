# 费马平方和定理

- Conversation ID: `69030304-5094-8321-8445-ece207afe990`

## Question

费马平方和定理是什么？是如何证明的？

## Answer

“费马平方和定理”（Fermat’s theorem on sums of two squares）是一个数论中的经典定理，内容简洁而深刻，结论如下：

##### 🌟 定理内容

费马平方和定理：
一个素数 $p$ 能表示为两个整数的平方和

```math
p = x^2 + y^2
```

当且仅当

```math
p = 2 \quad \text{或} \quad p \equiv 1 \pmod{4}.
```

##### 🔢 举例说明

```math
2 = 1^2 + 1^2
```

```math
5 = 2^2 + 1^2
```

```math
13 = 3^2 + 2^2
```

```math
17 = 4^2 + 1^2
```

```math
29 = 5^2 + 2^2
```

而：

```math
3, 7, 11, 19, 23, 31, \ldots \equiv 3 \pmod{4}
```

都不能表示为两个平方的和。

##### 🧩 更一般的形式

如果一个正整数 $n$ 的素因数分解为：

```math
n = 2^a \prod_{p_i \equiv 1 (4)} p_i^{b_i} \prod_{q_j \equiv 3 (4)} q_j^{c_j}
```

那么：

$n$ 能写成两个平方的和当且仅当所有 $q_j \equiv 3 \pmod{4}$ 的指数 $c_j$ 都是偶数。

例如：

$45 = 3^2 \times 5$，其中 $3 \equiv 3(4)$，指数为 2（偶数）\to ✅ 可以写成平方和：

```math
45 = 6^2 + 3^2
```

$21 = 3 \times 7$，两个都是 $\equiv 3(4)$，指数为奇数 \to ❌ 不能表示为平方和。

##### 🧠 证明思路（核心思想）

费马自己给出了结论，但没有留下证明。第一个完整证明是由欧拉（Euler）给出的，后人还发现了更代数化的证明。

下面是欧拉证明思路的核心要点（用高斯整数思想可以更简洁地理解）：

###### 1️⃣ 引入高斯整数环 $\mathbb{Z}[i]$

高斯整数定义为：

```math
\mathbb{Z}[i] = \{ a + bi \mid a, b \in \mathbb{Z} \}
```

其“模长”定义为：

```math
N(a+bi) = a^2 + b^2
```

而且这个模长有乘法性：

```math
N((a+bi)(c+di)) = (a^2 + b^2)(c^2 + d^2)
```

###### 2️⃣ 若 $p \equiv 1 \pmod{4}$，则存在 $a$ 使得 $a^2 \equiv -1 \pmod{p}$

这一步是关键。它说明模 $p$ 下有“虚数单位”存在。
也就是说，存在整数 $a$ 满足：

```math
p \mid (a^2 + 1)
```

###### 3️⃣ 在 $\mathbb{Z}[i]$ 中，$p$ 可以分解

由于 $p \mid (a^2 + 1)$，我们可以写为：

```math
a^2 + 1 = kp
```

即

```math
(a + i)(a - i) = kp
```

在 $\mathbb{Z}[i]$ 中，这意味着 $p$ 不是不可约元，它可以分解成两个非平凡因子。

###### 4️⃣ 由于 $\mathbb{Z}[i]$ 是唯一分解整环（UFD）

因此 $p$ 可以唯一地分解为：

```math
p = (a + bi)(a - bi)
```

其中 $N(a + bi) = p$，即：

```math
a^2 + b^2 = p
```

###### 5️⃣ 如果 $p \equiv 3 \pmod{4}$

则不存在整数 $a$ 使得 $a^2 \equiv -1 \pmod{p}$，
因此 $p$ 在 $\mathbb{Z}[i]$ 中保持不可约，不能写成两个平方和。

##### 🧮 总结证明结构

| 情况 | 性质 | 结论 |
| --- | --- | --- |
| $p = 2$ | $2 = 1^2 + 1^2$ | ✅ |
| $p \equiv 1 (4)$ | 可分解于 $\mathbb{Z}[i]$ | ✅ |
| $p \equiv 3 (4)$ | 不可分解 | ❌ |

##### 📚 相关延伸

复整数环 $\mathbb{Z}[i]$ 的唯一分解性质是证明的关键。

该定理是研究二次型理论、代数数论的起点。

高斯更进一步推广为“平方和定理”的多变量形式。
