# 证明无理数性质

- Conversation ID: `6901d644-7150-8324-bc7b-1a95487617e6`

## Question

证明 $\sqrt{2} + \sqrt{3} + \cdots + \sqrt{n}$ 是无理数

## Answer

我们要证明：

```math
S_n = \sqrt{2} + \sqrt{3} + \sqrt{4} + \cdots + \sqrt{n}
```

是无理数。

##### 💡 基本思路

先观察：

当 $n = 2$，$\sqrt{2}$ 无理。✅

当 $n = 3$，$\sqrt{2} + \sqrt{3}$ 是否可能是有理数？
若是，则 $\sqrt{3} = r - \sqrt{2}$，两边平方得到矛盾。
我们可以把这一思想推广。

##### ✅ 关键引理

若 $a_1, a_2, \dots, a_k$ 是正整数中不含平方因子的不同数（即平方不可约数），
那么 $\sqrt{a_1}, \sqrt{a_2}, \dots, \sqrt{a_k}$ 在有理数域 $\mathbb{Q}$ 上线性无关。

也就是说，如果

```math
q_1\sqrt{a_1} + q_2\sqrt{a_2} + \cdots + q_k\sqrt{a_k} = 0,\quad q_i \in \mathbb{Q},
```

则所有 $q_i = 0$。

##### ✳️ 应用到本题

注意：

```math
\sqrt{4} = 2 \in \mathbb{Q}
```

是有理的。
所以我们只要考虑那些不是平方数的项：

```math
S_n = (\text{整数部分}) + (\sqrt{2} + \sqrt{3} + \sqrt{5} + \sqrt{6} + \cdots)
```

其中每个被开方的数都是非平方数。

##### ⚙️ 证明（构造性）

设

```math
S_n = \sqrt{2} + \sqrt{3} + \sqrt{5} + \cdots + \sqrt{m_k}
```

其中 $m_1, m_2, \dots, m_k$ 是 $2,3,\dots,n$ 中的所有非平方数。

假设（为反证）$S_n$ 是有理数，即

```math
\sqrt{2} + \sqrt{3} + \cdots + \sqrt{m_k} = r, \quad r \in \mathbb{Q}.
```

则有

```math
\sqrt{m_k} = r - (\sqrt{2} + \sqrt{3} + \cdots + \sqrt{m_{k-1}}).
```

右边是有理数与若干无理根号的线性组合，根据引理，这不可能是单个根号的形式（因为左边的 $\sqrt{m_k}$ 在 $\mathbb{Q}$ 上与前面那些无理根号线性无关）。
故矛盾。

##### 🔍 举例验证（n=3）

若 $\sqrt{2} + \sqrt{3}$ 是有理数 $r$，
则 $\sqrt{3} = r - \sqrt{2}$。
平方：

```math
3 = r^2 - 2r\sqrt{2} + 2
```

```math
\Rightarrow 1 = r^2 - 2r\sqrt{2}.
```

```math
\Rightarrow \sqrt{2} = \frac{r^2 - 1}{2r} \in \mathbb{Q},
```

矛盾。

##### ✅ 结论

由此可知，对于任意整数 $n \ge 2$，

```math
\boxed{\sqrt{2} + \sqrt{3} + \cdots + \sqrt{n} \text{ 是无理数。}}
```

##### 🧠 附加说明（更强命题）

事实上，若 $a_1, \dots, a_k$ 为两两不同的正整数，且每个不是完全平方数，则

```math
\sum_{i=1}^{k} \sqrt{a_i}
```

不仅是无理数，而且是代数次数 $2^k$ 的代数数。
