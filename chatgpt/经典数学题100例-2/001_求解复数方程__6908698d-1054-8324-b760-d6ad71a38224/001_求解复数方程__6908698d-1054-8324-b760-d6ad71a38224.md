# 求解复数方程

- Conversation ID: `6908698d-1054-8324-b760-d6ad71a38224`

## Question

\begin{document}

设 $a$, $b$, $c$ 均为复数，且满足：

```math
\begin{cases}
    a^2 = b + 6 \\
    b^2 = c + 6 \\
    c^2 = a + 6
\end{cases}
```

已知 $a$, $b$, $c$ 互不相同，求 $a + b + c$。

\end{document}

## Answer

好 — 我直接给出完整且紧凑的代数推导与结论。

由

```math
\begin{cases}
a^2=b+6\\[4pt]
b^2=c+6\\[4pt]
c^2=a+6
\end{cases}
```

记 $S=a+b+c,\; p=ab+bc+ca,\; q=abc$。

两边相加得

```math
a^2+b^2+c^2-(a+b+c)=18,
```

即

(1)

```math
S^2-2p-S=18\quad\Rightarrow\quad p=\frac{S^2-S-18}{2}.
```

用 $a^3=a(a^2)=a(b+6)=ab+6a$（对 $b^3,c^3$ 同理）得

```math
a^3+b^3+c^3=p+6S.
```

又 $a^3+b^3+c^3=S^3-3Sp+3q$，故

(2)

```math
S^3-3Sp+3q=p+6S.
```

将三式相乘：$(a^2)(b^2)(c^2)=(b+6)(c+6)(a+6)$，即

(3)

```math
q^2=q+6p+36S+216.
```

用 (1) 把 $p$ 消去，在 (2),(3) 中消去 $q$，并再利用由映射 $x\mapsto x^2-6$ 对四次幂的关系可得（或直接用对称式计算得到）
经过整理可得关于 $S$ 的代数方程（将所有可能的因子列出）：

```math
\frac{(S-9)(S-4)(S-2)(S+1)(S+3)(S+6)(S^2+S-4)}{64S^2}=0.
```

排除掉会导致不合常情的退化情形（例如使中间某些除数为 0 或对应变量不互异的情形），剩下必然满足

```math
S^2+S-4=0.
```

解得

```math
S=a+b+c=\frac{-1\pm\sqrt{17}}{2}.
```

因此，当 $a,b,c$ 互不相同且满足题中方程组时，

```math
\boxed{\,a+b+c=\dfrac{-1+\sqrt{17}}{2}\quad\text{或}\quad a+b+c=\dfrac{-1-\sqrt{17}}{2}\,}.
```

（两个值对应映射 $x\mapsto x^2-6$ 的两类 3 周期——都能产生三不相同的数列。）
