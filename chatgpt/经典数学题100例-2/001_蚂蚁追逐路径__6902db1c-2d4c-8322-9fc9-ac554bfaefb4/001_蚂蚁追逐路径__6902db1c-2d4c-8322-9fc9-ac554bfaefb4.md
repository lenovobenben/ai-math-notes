# 蚂蚁追逐路径

- Conversation ID: `6902db1c-2d4c-8322-9fc9-ac554bfaefb4`

## Question

假设一个正方形的四个顶点ABCD处各有一只蚂蚁。 A追逐B，B追逐C，C追逐D，D追逐A。 他们速度相同，方向时刻指向要追逐的对象。最后追逐路径形成的曲线是什么样的？

## Answer

简短结论（先说答案）：
四只蚂蚁沿着**对数螺线（logarithmic spiral）**向正方形中心螺旋收敛，最终在有限时间内相遇于中心。

下面给出清晰的推导与结果（对称性＋解析计算）。

对称性观察
任取正方形中心为原点，四只蚂蚁始终保持互为正方形四顶点的位置（正方形在不停缩小并匀速旋转）。因此只需研究任意一只蚂蚁的位置随时间变化：它到中心的距离记为 $r(t)$，极角为 $\theta(t)$。

向量方程与常数分解
设每只蚂蚁速度大小恒为 $v$，且每只蚂蚁的追逐方向始终指向下一个顶点。利用复数表示或向量代数，可以得到（对任意正多边形已知的常规推导）：

```math
r'(t) = -v\sin\frac{\pi}{n},\qquad r(t)\,\theta'(t)=v\cos\frac{\pi}{n},
```

其中 $n$ 为多边形边数。对正方形 $n=4$，有 $\sin\frac{\pi}{4}=\cos\frac{\pi}{4}=\tfrac{\sqrt2}{2}$。

对正方形的具体公式

距离随时间的线性衰减：

```math
r(t)=r_0 - v\frac{\sqrt2}{2}\,t,
```

其中 $r_0$ 为初始时任一顶点到中心的距离。由正方形边长 $L$ 可得 $r_0=\dfrac{L}{\sqrt2}$。

相遇（到达中心）的时间：

```math
T=\frac{r_0}{v\sin(\pi/4)}=\frac{r_0}{v(\sqrt2/2)}=\frac{\sqrt2\,r_0}{v}.
```

代入 $r_0=L/\sqrt2$ 得出对正方形：

```math
T=\frac{L}{v}.
```

轨迹是对数螺线。由上面两式消去 $t$ 并积分可得极形式关系

```math
r(\theta)=r_0\exp\big(-\tan(\tfrac{\pi}{n})\;(\theta-\theta_0)\big).
```

对正方形 $\tan(\pi/4)=1$，于是

```math
\boxed{\,r(\theta)=r_0\,e^{-(\theta-\theta_0)}\,},
```

这正是标准的对数（等角）螺线。

直观解释
每只蚂蚁的速度对半径方向有固定向心分量（使半径线以恒定速率缩短），并有固定切向分量（使正方形匀速旋转）。半径以恒定速率减小意味着它们在有限时间内到达中心，而角度随半径的对数变化，因而轨迹是对数螺线。

## Question

这个和飞蛾扑火的案例一样。但是飞蛾扑火的光源是固定的。这个是变化的。为什么结果都一样呢？

## Answer

非常好的问题 👍
你指出了一个关键区别：

飞蛾扑火：目标（光源）是固定的。

蚂蚁追逐：目标（被追的蚂蚁）在不断移动。

然而，结果居然都是对数螺线（logarithmic spiral）。
这并不是巧合，而是因为两者都共享一个更深层次的几何约束：相对速度方向始终与径向方向夹一个固定角度。

##### 🧭 一、飞蛾扑火：固定目标 + 恒定角度追踪

飞蛾始终以固定角度（相对于视线方向）飞向光源。
在极坐标下，这意味着飞行速度在径向方向与切向方向的分量之比是常数：

```math
\frac{r\,d\theta/dt}{dr/dt} = -\cot \alpha = \text{常数}
```

积分得到：

```math
r = r_0 e^{-\tan\alpha(\theta - \theta_0)}
```

这正是对数螺线。

##### 🐜 二、四蚂蚁问题：移动目标 + 对称追踪

虽然每只蚂蚁的目标在移动，但整体保持严格对称。
这带来一个奇妙的结果：

对称性使得“正方形始终保持形状，只是缩小并旋转”。

对每只蚂蚁来说，“它追逐的蚂蚁的方向”与“自己到中心的径向方向”之间的夹角保持恒定。

换句话说，虽然目标在动，但角度关系是恒定的。
这恰恰是飞蛾追光方程的核心条件。

因此它们满足相同的微分方程：

```math
\frac{dr}{d\theta} = -r\tan\alpha
```

于是同样得到对数螺线轨迹。

##### 🧩 三、为什么夹角恒定？

这是对称性 + 匀速条件共同保证的：

每只蚂蚁速度大小相等；

追逐方向沿着连接自己与目标的连线；

四只蚂蚁的位置始终构成正方形；

因此速度方向与连线到中心的夹角固定为 $\pi/4$。

一旦夹角恒定，轨迹形式就被完全确定——那就是对数螺线。

##### 🔍 四、小结

| 特征 | 飞蛾扑火 | 四蚂蚁追逐 |
| --- | --- | --- |
| 目标 | 固定光源 | 运动的相邻蚂蚁 |
| 系统 | 单体 | 对称多体 |
| 关键条件 | 方向与径向夹角恒定 | 对称性保证夹角恒定 |
| 微分方程 | 相同 | 相同 |
| 轨迹 | 对数螺线 | 对数螺线 |
