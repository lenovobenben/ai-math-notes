# 无限乘积计算

- Conversation ID: `69094ec2-238c-8323-b68e-1e6aaabb5a0a`

## Question

计算 \prod_{n=1}^{\infty} (1 + \frac{1}{n^2})

## Answer

我们要求的是

```math
\prod_{n=1}^{\infty} \left(1 + \frac{1}{n^2}\right)
```

##### 第一步：改写为可识别形式

注意到：

```math
1 + \frac{1}{n^2} = \frac{n^2 + 1}{n^2} = \frac{(n + i)(n - i)}{n^2}
```

所以

```math
\prod_{n=1}^{\infty} \left(1 + \frac{1}{n^2}\right)
= \prod_{n=1}^{\infty} \frac{(n + i)(n - i)}{n^2}
= \left( \prod_{n=1}^{\infty} \frac{n + i}{n} \right) \left( \prod_{n=1}^{\infty} \frac{n - i}{n} \right)
```

##### 第二步：利用 Gamma 函数的无限乘积公式

Gamma 函数有恒等式：

```math
\frac{1}{\Gamma(z)} = z e^{\gamma z} \prod_{n=1}^{\infty} \left(1 + \frac{z}{n}\right) e^{-z/n}
```

整理可得：

```math
\prod_{n=1}^{\infty} \left(1 + \frac{z}{n}\right) = \frac{e^{-\gamma z}}{z \Gamma(z)} \prod_{n=1}^{\infty} e^{z/n}
```

不过更方便的是利用：

```math
\prod_{n=1}^{\infty} \frac{n + z}{n} = \frac{\Gamma(1)}{\Gamma(1 + z)} e^{\gamma z}
```

（这可由上式推导得出，但这里我们只需其相对比例。）

##### 第三步：取模平方抵消指数因子

因为我们要乘上 $z = i$ 与 $z = -i$ 的部分，所以指数项 $e^{\gamma z}$ 与 $e^{-\gamma z}$ 会相互抵消：

```math
\prod_{n=1}^{\infty} \frac{n + i}{n} \cdot \prod_{n=1}^{\infty} \frac{n - i}{n}
= \frac{1}{\Gamma(1+i)\Gamma(1-i)}
```

##### 第四步：化简 $\Gamma(1+i)\Gamma(1-i)$

利用恒等式：

```math
\Gamma(1+z)\Gamma(1-z) = \frac{\pi z}{\sin(\pi z)}
```

令 $z = i$，得到：

```math
\Gamma(1+i)\Gamma(1-i) = \frac{\pi i}{\sin(\pi i)}
```

而 $\sin(\pi i) = i \sinh(\pi)$，因为 $\sin(ix) = i \sinh(x)$。

所以：

```math
\Gamma(1+i)\Gamma(1-i) = \frac{\pi i}{i \sinh(\pi)} = \frac{\pi}{\sinh(\pi)}
```

##### 第五步：取倒数

因此

```math
\prod_{n=1}^{\infty} \left(1 + \frac{1}{n^2}\right) = \frac{1}{\Gamma(1+i)\Gamma(1-i)} = \frac{\sinh(\pi)}{\pi}
```

✅ 最终结果：

```math
\boxed{\displaystyle \prod_{n=1}^{\infty} \left(1 + \frac{1}{n^2}\right) = \frac{\sinh(\pi)}{\pi}}
```
