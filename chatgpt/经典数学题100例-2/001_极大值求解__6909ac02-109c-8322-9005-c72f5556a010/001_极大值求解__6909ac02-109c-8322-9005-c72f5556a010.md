# 极大值求解

- Conversation ID: `6909ac02-109c-8322-9005-c72f5556a010`

## Question

可积函数f(x)，满足|f(x)|\le 1，而且\int_{1}^{3} f(x)dx=0。求极大值 max(\int_{1}^{3} \frac{f(x)}{x} dx)

## Answer

把题目写成变分/线性规划的语言更容易看清：设权函数 $w(x)=\dfrac1x$（在 $[1,3]$ 上单调减），要最大化线性泛函

```math
I(f)=\int_{1}^{3}\frac{f(x)}x\,dx
```

在约束集合 $\{f:\ |f(x)|\le1,\ \int_{1}^{3}f(x)\,dx=0\}$ 上。

线性目标在凸集上的最大值必出现在极点上。对约束 $|f|\le1$ 的函数，极点是几乎处处取值 $\pm1$ 的函数。因此可令

```math
f(x)=\begin{cases}1,& x\in A,\\ -1,& x\in A^c,\end{cases}
```

其中 $A\subset[1,3]$ 为可测集。

由 $\int_{1}^{3}f(x)\,dx=0$ 得

```math
|A|-|(A^c)|=0\quad\Rightarrow\quad |A|=\tfrac12|[1,3]|=\tfrac12\cdot2=1,
```

即 $A$ 的测度必须为 $1$。

此时目标变为

```math
I(f)=\int_A\frac1x\,dx-\int_{A^c}\frac1x\,dx
=2\int_A\frac1x\,dx-\int_{1}^{3}\frac1x\,dx.
```

要最大化 $I(f)$，等价于在测度为 $1$ 的集合中最大化 $\int_A\frac1x\,dx$。因为 $1/x$ 在 $[1,3]$ 上单调递减，这个最大化由取最靠近左端的长度为 $1$ 的区间实现，即 $A=[1,2]$（或等价的几乎处处与之相同的集合）。

代入计算：

```math
\max I(f)=2\int_{1}^{2}\frac{dx}{x}-\int_{1}^{3}\frac{dx}{x}=2\ln2-\ln3=\ln\frac{4}{3}.
```

结论：最大值为 $\displaystyle \ln\frac{4}{3}$（约等于 $0.28768$），当且仅当几乎处处

```math
f(x)=\begin{cases}1,& x\in[1,2],\\ -1,& x\in(2,3]\end{cases}
```

时取得。
