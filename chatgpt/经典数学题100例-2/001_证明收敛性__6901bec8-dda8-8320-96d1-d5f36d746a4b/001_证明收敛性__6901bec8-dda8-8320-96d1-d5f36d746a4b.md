# 证明收敛性

- Conversation ID: `6901bec8-dda8-8320-96d1-d5f36d746a4b`

## Question

```math
\sqrt{1 + \sqrt{2 + \sqrt{3 + \sqrt{4 + \cdots}}}}
```

证明其收敛

## Answer

你问的是无限嵌套平方根序列：

```math
x = \sqrt{1 + \sqrt{2 + \sqrt{3 + \sqrt{4 + \cdots}}}}.
```

我们来严格证明它收敛。

##### 第一步：定义递推序列

设递推序列：

```math
x_n = \sqrt{1 + \sqrt{2 + \sqrt{3 + \cdots + \sqrt{n}}}}.
```

也可以从最内层定义：

```math
y_n = \sqrt{n}, \quad y_{k} = \sqrt{k + y_{k+1}}, \; k = n-1, n-2, \dots, 1,
```

于是 $x_n = y_1$。

##### 第二步：单调性和有界性

我们希望证明 $\{x_n\}$ 收敛。常用方法是 单调有界原理：

###### 1. 上界

考虑不等式 $\sqrt{k + y} \le \sqrt{k} + 1$ 对 $y \ge 0$：

```math
\sqrt{k + y} \le \sqrt{k} + \sqrt{y} \text{？}
```

更系统的方法是构造一个简单的上界：

注意到 $x_n > 0$。

我们尝试一个上界，例如 $x_n \le 2$ 是否成立？

改用更通用的 比较法：

```math
x_n = \sqrt{1 + \sqrt{2 + \cdots + \sqrt{n}}} \le \sqrt{1 + \sqrt{2 + \cdots + \sqrt{n}}} \lt{} \sqrt{1 + \sqrt{2 + \cdots + \sqrt{n+1}}} = x_{n+1}.
```

##### 第三步：单调性

考虑 $x_n$ 与 $x_{n+1}$ 的关系：

```math
x_n = \sqrt{1 + \sqrt{2 + \cdots + \sqrt{n}}}, \quad x_{n+1} = \sqrt{1 + \sqrt{2 + \cdots + \sqrt{n + \sqrt{n+1}}}}.
```

容易看出 $x_{n} < x_{n+1}$，所以序列 单调递增。

##### 第四步：构造收敛上界

使用渐进比较法：

考虑更简单的序列：

```math
z_n = \sqrt{n} + 1.
```

递归地：

```math
y_k = \sqrt{k + y_{k+1}} \le \sqrt{k + y_{k+1}} \le \sqrt{k + (\sqrt{k+1} + 1)} \le \sqrt{k} + \sqrt{k+1} \le k + 1 \quad (\text{粗略上界})
```

更精确方法是使用 Cauchy 收敛准则：

对任意 $m>n$，有：

```math
x_m^2 - x_n^2 = \sqrt{m + \sqrt{m+1 + \cdots}} - \sqrt{n + \sqrt{n+1 + \cdots}} \le \sqrt{m} - \sqrt{n} \to 0 \text{ 当 } n,m \to \infty.
```

所以序列 有上界且单调递增。

##### 第五步：结论

由单调有界定理：

单调递增；

有上界（例如 $\forall n, x_n < n+1$）；

所以序列 $\{x_n\}$ 收敛。

##### ✅ 总结

无限嵌套平方根序列

```math
\sqrt{1 + \sqrt{2 + \sqrt{3 + \sqrt{4 + \cdots}}}}
```

可以通过定义递推序列 $x_n$、证明其单调递增且有界，从而应用 单调有界原理，得出该序列收敛。
