# 单位正方形距离期望值

- Conversation ID: `6901fa43-be78-8323-99cf-603d1ac68b4d`

## Question

求：单位正方形上两个随机点距离的平均值

## Answer

你问的是：单位正方形上随机选择两个点，它们之间距离的期望值。这是一个经典几何概率问题。我们可以系统地分析一下。

##### 问题描述

单位正方形：边长 $1$。

随机点：$(x_1, y_1)$ 和 $(x_2, y_2)$，其中 $x_i, y_i \sim \text{Uniform}[0,1]$。

距离：$d = \sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2}$。

求期望值：

```math
E[d] = \iint_0^1 \iint_0^1 \sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2} \, dx_1\, dy_1\, dx_2\, dy_2
```

##### 已知结果

这个积分没有初等函数形式的解析解，但它有一个精确表达式：

```math
E[d] = \frac{1}{15} \left( \sqrt{2} + 2 + 5 \ln(1 + \sqrt{2}) \right)
```

数值计算约为：

```math
E[d] \approx 0.521405
```
