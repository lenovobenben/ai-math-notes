# 扑克牌位置不变分析

- Conversation ID: `69006732-aab4-8320-84e1-6f0ebffa2b5f`

## Question

扑克牌随机打乱，平均有几张牌的位置保持不变？

## Answer

结论很简短：平均会有 1 张牌位置保持不变（对任意 $n$ 张牌都成立）。
下面给出简单推导并补充几个有用事实。

推导（指示变量法）
把第 $i$ 张牌是否回到原位记作指示变量 $I_i$（回到原位则 $I_i=1$，否则为 0）。
$\Pr(I_i=1)=1/n$。于是总的固定点数 $X=\sum_{i=1}^n I_i$，期望

```math
\mathbb{E}[X]=\sum_{i=1}^n\mathbb{E}[I_i]=\sum_{i=1}^n \frac{1}{n}=1.
```

对 52 张牌：

```math
\;52\times\frac{1}{52}=1.
```

补充（分布与方差）

当 $n$ 很大时，固定点数近似 Poisson(1) 分布，所以

```math
\Pr(X=k)\approx e^{-1}/k!
```

因此没有固定点的概率约为 $e^{-1}\approx0.3679$，至少有一张固定的概率约为 $1-1/e\approx0.6321$。

方差也等于 1（可以类似用协方差计算得到）。
