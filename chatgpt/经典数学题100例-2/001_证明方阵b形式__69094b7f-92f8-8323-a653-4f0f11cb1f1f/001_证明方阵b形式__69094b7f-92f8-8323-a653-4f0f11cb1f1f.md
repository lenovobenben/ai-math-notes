# 证明方阵B形式

- Conversation ID: `69094b7f-92f8-8323-a653-4f0f11cb1f1f`

## Question

方阵A的特征值各不相同。假设有方阵B满足AB=BA。证明：B必然可以写成B=f(A)，其中f是多项式函数。

## Answer

设 $A$ 为 $n\times n$ 方阵，且其特征值两两不同。令 $B$ 为与 $A$ 对易的方阵（即 $AB=BA$）。下面证明存在多项式 $f$ 使得 $B=f(A)$。

证明：

由 $A$ 的特征值互不相同可知，$A$ 可对角化。即存在可逆矩阵 $P$ 与对角矩阵 $D=\mathrm{diag}(\lambda_1,\dots,\lambda_n)$（$\lambda_i$ 为 $A$ 的特征值，且两两不同），使得

```math
A=PDP^{-1}.
```

记 $C=P^{-1}BP$。由 $AB=BA$ 和相似变换可得

```math
D C = P^{-1} A P \, P^{-1} B P = P^{-1} (AB) P = P^{-1} (BA) P = C D.
```

即 $C$ 与对角矩阵 $D$ 对易。

写 $C=(c_{ij})$。比较 $(DC)_{ij}$ 与 $(CD)_{ij}$ 得

```math
\lambda_i c_{ij} = c_{ij}\lambda_j\quad\Rightarrow\quad (\lambda_i-\lambda_j)c_{ij}=0.
```

由于 $\lambda_i\neq\lambda_j$（当 $i\neq j$），所以 $c_{ij}=0$ 对所有 $i\neq j$ 成立。因此 $C$ 必为对角矩阵：

```math
C=\mathrm{diag}(\mu_1,\dots,\mu_n)
```

某些标量 $\mu_i$。

现在我们需要一个多项式 $f$ 使得 $f(\lambda_i)=\mu_i$ 对每个 $i$ 都成立。由插值多项式（如拉格朗日插值）可知，存在次数至多 $n-1$ 的多项式

```math
f(x)=\sum_{i=1}^n \mu_i\prod_{j\neq i}\frac{x-\lambda_j}{\lambda_i-\lambda_j},
```

使得对每个 $i$ 有 $f(\lambda_i)=\mu_i$。

因为 $f$ 是多项式，代入相似形式得

```math
f(A)=P f(D) P^{-1}=P\,\mathrm{diag}(f(\lambda_1),\dots,f(\lambda_n))\,P^{-1}
= P\,\mathrm{diag}(\mu_1,\dots,\mu_n)\,P^{-1}=P C P^{-1}=B.
```

这样就得到了所需的多项式 $f$，使得 $B=f(A)$。且可取 $\deg f\le n-1$。

```math
\square
```

（注：上面的证明隐含地假设所在域包含 $A$ 的全部特征值——常见情况下取复数域 $\mathbb C$。若在一般域上讨论，需要额外说明特征多项式是否可分裂等。）
