# 积分计算结果

- Conversation ID: `6908e2ee-88b8-8322-9c6f-26649182bf1c`

## Question

求 \int_{0}^{\frac{\pi}{2}} \cos {(\tan {x})}\mathrm{d}x = ?

## Answer

把 $t=\tan x$，则 $\mathrm{d}t=\sec^2x\,\mathrm{d}x=(1+t^2)\,\mathrm{d}x$，积分变为

```math
\int_{0}^{\frac{\pi}{2}}\cos(\tan x)\,\mathrm{d}x
=\int_{0}^{\infty}\frac{\cos t}{1+t^2}\,\mathrm{d}t.
```

已知（或用留数法可证）

```math
\int_{0}^{\infty}\frac{\cos(at)}{1+t^2}\,\mathrm{d}t=\frac{\pi}{2}e^{-|a|}\qquad(a\in\mathbb{R}),
```

代入 $a=1$ 得

```math
\int_{0}^{\infty}\frac{\cos t}{1+t^2}\,\mathrm{d}t=\frac{\pi}{2}e^{-1}.
```

因此原积分的值为

```math
\boxed{\dfrac{\pi}{2}e^{-1}}.
```
