# 求解a和b值

- Conversation ID: `69074f7d-2a28-8324-ba52-9677c2c07ee9`

## Question

已知f(x) = ax^b （a,b为正实数），且满足f'(x) = f^{-1}(x)。求a,b的值

## Answer

先说明 $f^{-1}(x)$ 是函数的反函数。
已知 $f(x)=a x^b$（$a,b>0$），则

```math
f'(x)=a b x^{\,b-1},\qquad f^{-1}(x)=\Big(\frac{x}{a}\Big)^{1/b}.
```

由题意有

```math
a b x^{b-1}=\Big(\frac{x}{a}\Big)^{1/b}\quad(\forall x\gt{}0).
```

将两边写成 $C x^{\text{幂}}$ 的形式，比较 $x$ 的指数和常数项：

指数相等：$b-1=\dfrac{1}{b}$，得

```math
b^2-b-1=0\implies b=\frac{1+\sqrt5}{2}\quad(\text{取正根}).
```

常数相等：$a b=a^{-1/b}$。整理得

```math
a^{(b+1)/b}=\frac{1}{b}\implies a=\Big(\frac{1}{b}\Big)^{\frac{b}{\,b+1}}=b^{-\frac{b}{\,b+1}}.
```

代入 $b=\dfrac{1+\sqrt5}{2}$ 可得数值近似

```math
b\approx1.6180339887,\qquad a\approx0.7427429446.
```

因此答案为

```math
b=\frac{1+\sqrt5}{2},\qquad a=b^{-\frac{b}{b+1}} \ (\approx0.7427429446).
```

## Question

a=(1/b)^(1/b)和你的是等价的吧？

## Answer

我们仔细核对一下常数关系。

我们有方程：

```math
a b = a^{-1/b} \quad \Rightarrow \quad a b = \frac{1}{a^{1/b}} \quad \Rightarrow \quad a^{1+1/b} = \frac{1}{b}.
```

所以确实得到：

```math
a = \left(\frac{1}{b}\right)^{\frac{b}{b+1}}.
```

你说的 $(1/b)^{1/b}$ 不是完全等价，除非 $b/(b+1) = 1/b$。检查一下：

```math
\frac{b}{b+1} \stackrel{?}{=} \frac{1}{b} \quad \Rightarrow \quad b^2 = b+1 \quad \Rightarrow \quad b = \frac{1+\sqrt5}{2}.
```

哦！对，这里正好是黄金比例 $b = \frac{1+\sqrt5}{2}$。所以在这个特殊值下，两者确实是等价的。

✅ 结论：在这个问题中，

```math
a = \left(\frac{1}{b}\right)^{\frac{b}{b+1}} = \left(\frac{1}{b}\right)^{1/b}.
```

## Question

这样答案显得整齐一些

## Answer

对，非常整齐！用这个形式写：

```math
b = \frac{1+\sqrt5}{2}, \quad a = \left(\frac{1}{b}\right)^{1/b}.
```

不仅简洁，而且数字美观——黄金比例果然到处显现。 😄

如果你愿意，还可以写出函数的显式形式：

```math
f(x) = \left(\frac{1}{b}\right)^{1/b} x^b, \quad b = \frac{1+\sqrt5}{2}.
```

这样整齐又直观。
