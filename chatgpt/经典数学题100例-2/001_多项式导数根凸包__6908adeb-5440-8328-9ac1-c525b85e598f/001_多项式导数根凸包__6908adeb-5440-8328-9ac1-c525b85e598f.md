# 多项式导数根凸包

- Conversation ID: `6908adeb-5440-8328-9ac1-c525b85e598f`

## Question

多项式导数的根都落在原多项式的根构成的凸包中吗？

## Answer

是的，✅ 多项式的导数的所有根都落在原多项式根的凸包（convex hull）中。
这就是一个著名的结论，称为 高斯–卢卡斯定理（Gauss–Lucas Theorem）。

##### 📘 定理陈述（Gauss–Lucas Theorem）

设 $P(z)$ 是一个复系数多项式，次数 $n \ge 2$，其根为

```math
z_1, z_2, \ldots, z_n.
```

则其导数 $P'(z)$ 的所有根都位于这些根 $z_1, z_2, \ldots, z_n$ 的凸包（convex hull）中。

也就是说：

```math
\text{Zeros}(P') \subseteq \text{Conv}\{ z_1, z_2, \ldots, z_n \}.
```

##### 💡 直观理解

如果把 $z_1, z_2, \ldots, z_n$ 看作复平面上的点：

这些点形成一个（可能退化的）多边形；

$P'(z) = 0$ 的点全部都在这个多边形内部或边界上；

它们不会“跑出”这个凸包之外。

这个结论可以看作：导数的零点“受原多项式零点的吸引”，不会离开这些零点的包络区域。

##### 🧮 一个简单例子

设

```math
P(z) = (z - 1)(z - 3) = z^2 - 4z + 3.
```

则

```math
P'(z) = 2z - 4 = 0 \implies z = 2.
```

原多项式的根为 1 和 3，导数的根 2 正好位于区间 [1, 3] 上（即凸包中）。

##### 📊 更多性质

如果所有根都实数，则 $P'(z)$ 的根也都实数，并且在原根之间（这其实是高斯–卢卡斯定理在实轴上的特例）。

如果有重根，则导数的根也可能与原根重合。

对于高次多项式，可以用几何方式证明：

```math
\frac{P'(z)}{P(z)} = \sum_{k=1}^n \frac{1}{z - z_k}
```

当 $P'(z) = 0$ 时，说明这些向量 $\frac{1}{z - z_k}$ 的向量和为 0，几何上意味着 $z$ 必在根的凸包内。
