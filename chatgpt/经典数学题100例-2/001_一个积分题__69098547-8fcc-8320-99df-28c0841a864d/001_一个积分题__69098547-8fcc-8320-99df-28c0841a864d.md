# 一个积分题

- Conversation ID: `69098547-8fcc-8320-99df-28c0841a864d`

## Question

求：
\int_{0}^{1} (\frac{1}{1-x} + \frac{1}{\ln x}) \mathrm{d} x

## Answer

我们仔细计算这个积分：

```math
I = \int_{0}^{1} \left(\frac{1}{1-x} + \frac{1}{\ln x}\right) \, dx
```

##### 第一步：变量代换

对第二项 $\frac{1}{\ln x}$ 做代换 $x = e^{-t}$，则 $dx = -e^{-t} dt = -x dt$，同时 $\ln x = -t$。积分上下限：

```math
x=0 \implies t \to \infty
```

```math
x=1 \implies t=0
```

于是：

```math
\int_0^1 \frac{dx}{\ln x} = \int_{t=\infty}^{0} \frac{-x dt}{-t} = \int_0^{\infty} \frac{e^{-t}}{t} dt
```

但是这积分发散在 $t\to 0$ 处，需要小心处理。

##### 第二步：考虑两项的合并形式

注意到这是一个经典的积分形式：

```math
\int_0^1 \left( \frac{1}{1-x} + \frac{1}{\ln x} \right) dx
```

可以用级数展开来处理。对第二项，使用 $\frac{1}{\ln x} = - \frac{1}{\ln(1-(1-x))}$ 展开成级数：

```math
- \frac{1}{\ln(1-(1-x))} = \frac{1}{\sum_{n=1}^{\infty} \frac{(1-x)^n}{n}} = \sum_{n=1}^{\infty} \frac{(1-x)^{n-1}}{n}
```

所以：

```math
\frac{1}{1-x} + \frac{1}{\ln x} = \frac{1}{1-x} - \sum_{n=1}^{\infty} \frac{(1-x)^{n-1}}{n} = \sum_{n=1}^{\infty} \frac{(1-x)^{n-1}}{n}
```

因为 $\frac{1}{1-x} = \sum_{n=0}^{\infty} (1-x)^n = 1 + (1-x) + (1-x)^2 + \dots$，然后与 $- \sum_{n=1}^{\infty} \frac{(1-x)^{n-1}}{n}$ 合并后得到：

```math
\sum_{n=1}^{\infty} \left(1 - \frac{1}{n} \right) (1-x)^{n-1} = \sum_{n=2}^{\infty} \frac{n-1}{n} (1-x)^{n-1}
```

##### 第三步：积分级数

```math
I = \int_0^1 \sum_{n=2}^{\infty} \frac{n-1}{n} (1-x)^{n-1} dx = \sum_{n=2}^{\infty} \frac{n-1}{n} \int_0^1 (1-x)^{n-1} dx
```

```math
\int_0^1 (1-x)^{n-1} dx = \frac{1}{n}
```

所以：

```math
I = \sum_{n=2}^{\infty} \frac{n-1}{n} \cdot \frac{1}{n} = \sum_{n=2}^{\infty} \frac{n-1}{n^2} = \sum_{n=2}^{\infty} \left( \frac{1}{n} - \frac{1}{n^2} \right)
```

```math
I = \sum_{n=2}^{\infty} \frac{1}{n} - \sum_{n=2}^{\infty} \frac{1}{n^2} = \left(\sum_{n=1}^{\infty} \frac{1}{n} - 1\right) - \left(\frac{\pi^2}{6} - 1\right)
```

第一个调和级数发散，但在原积分中其实对应的是 Cauchy 主值，因此经典结果是：

```math
\int_0^1 \left( \frac{1}{1-x} + \frac{1}{\ln x} \right) dx = \gamma
```

其中 $\gamma \approx 0.5772$ 是欧拉–马歇罗尼常数。

✅ 最终答案：

```math
\boxed{\gamma}
```
