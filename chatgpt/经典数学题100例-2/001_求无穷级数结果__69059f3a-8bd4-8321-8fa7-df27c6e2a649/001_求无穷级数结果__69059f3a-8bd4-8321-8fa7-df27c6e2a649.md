# 求无穷级数结果

- Conversation ID: `69059f3a-8bd4-8321-8fa7-df27c6e2a649`

## Question

S_k = \sum_{n=0}^\infty \frac{n^k}{n!}

求这个无穷级数

## Answer

### 结果

对于非负整数 $k$，

```math
S_k=\sum_{n=0}^\infty\frac{n^k}{n!}=e\,B_k,
```

其中 $B_k$ 是第 $k$ 个 Bell 数（即把 $k$ 个不同元素划分为若干非空子集的划分数），等于

```math
B_k=\sum_{j=0}^k\left\{\!\begin{matrix}k\\[2pt] j\end{matrix}\!\right\},
```

这里 $\left\{\begin{smallmatrix}k\\ j\end{smallmatrix}\right\}$ 为第二类 Stirling 数。

### 简短证明

用落阶（降阶）因子和 Stirling 数分解：

```math
n^k=\sum_{j=0}^k\left\{\!\begin{matrix}k\\ j\end{matrix}\!\right\}n^{\underline{j}},
\qquad n^{\underline{j}}=n(n-1)\cdots(n-j+1).
```

因此

```math
\sum_{n=0}^\infty\frac{n^k}{n!}
=\sum_{j=0}^k\left\{\!\begin{matrix}k\\ j\end{matrix}\!\right\}
\sum_{n=0}^\infty\frac{n^{\underline{j}}}{n!}
=\sum_{j=0}^k\left\{\!\begin{matrix}k\\ j\end{matrix}\!\right\}
\sum_{n=j}^\infty\frac{1}{(n-j)!}.
```

把索引 $m=n-j$ 变换后，每一项内和都是 $\sum_{m=0}^\infty\frac{1}{m!}=e$，于是得

```math
S_k=e\sum_{j=0}^k\left\{\!\begin{matrix}k\\ j\end{matrix}\!\right\}=e\,B_k.
```

### 举例

$B_0=1,B_1=1,B_2=2,B_3=5,B_4=15,\dots$，所以

```math
S_0=e,\; S_1=e,\; S_2=2e,\; S_3=5e,\; S_4=15e,\ \dots
```

（说明：当 $k>0$ 时 $0^k=0$；$k=0$ 时按常见约定 $0^0=1$，级数即 $\sum_{n=0}^\infty 1/n!=e$。）
