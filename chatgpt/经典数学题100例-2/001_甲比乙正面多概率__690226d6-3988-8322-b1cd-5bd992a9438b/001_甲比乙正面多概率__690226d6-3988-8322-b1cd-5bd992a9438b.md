# 甲比乙正面多概率

- Conversation ID: `690226d6-3988-8322-b1cd-5bd992a9438b`

## Question

甲有101枚硬币，乙有100枚硬币，同时抛掷，甲比乙正面多的概率？

## Answer

我们来一步步严谨分析这个题目：

##### 🧩 题目

甲有 $101$ 枚硬币，乙有 $100$ 枚硬币，同时抛掷，每枚硬币正反面概率相等。
问：甲比乙正面多的概率是多少？

##### 1️⃣ 设随机变量

设 $X \sim \text{Binomial}(101, \frac{1}{2})$，表示甲的正面数。

设 $Y \sim \text{Binomial}(100, \frac{1}{2})$，表示乙的正面数。

问题是求：

```math
P(X \gt{} Y)
```

##### 2️⃣ 对称性思考

如果甲和乙都只有 100 枚硬币，则 $P(X>Y) = P(X<Y)$，因为对称性。
所以：

```math
P(X\gt{}Y) = P(X\lt{}Y) = \frac{1 - P(X=Y)}{2}
```

##### 3️⃣ 增加一枚硬币的思路

现在甲比乙多一枚硬币，记这枚额外硬币的结果为 $Z$，其中：

```math
Z =
\begin{cases}
1 & \text{正面} \\
0 & \text{反面}
\end{cases}
```

且 $P(Z=1)=P(Z=0)=\frac{1}{2}$。

那么我们可以写：

```math
X = X' + Z,\quad \text{其中 } X' \sim \text{Binomial}(100, \tfrac{1}{2})
```

于是问题变为：

```math
P(X' + Z \gt{} Y)
```

##### 4️⃣ 按 $Z$ 分条件讨论

```math
P(X' + Z \gt{} Y) = \frac{1}{2} P(X' + 1 \gt{} Y) + \frac{1}{2} P(X' \gt{} Y)
```

##### 5️⃣ 用对称性计算两项

由于 $X'$ 与 $Y$ 分布完全相同，且独立：

```math
P(X' \gt{} Y) = \frac{1 - P(X'=Y)}{2}
```

```math
P(X' + 1 \gt{} Y) = P(X' \ge Y) = \frac{1 + P(X'=Y)}{2}
```

##### 6️⃣ 合并

```math
P(X' + Z \gt{} Y)
= \frac{1}{2}\left(\frac{1 + P(X'=Y)}{2}\right)
+ \frac{1}{2}\left(\frac{1 - P(X'=Y)}{2}\right)
= \frac{1}{2}
```

✅ 最终结果：

```math
\boxed{P(\text{甲比乙正面多}) = \frac{1}{2}}
```

##### 7️⃣ 直观解释

那多出的那一枚硬币（第 101 枚）刚好“打破平局”的概率正好对称。
因此，不论数量差 1 枚还是更复杂的形式，只要正反概率相同且独立，这种对称性就保证结果为 1/2。

答案：

```math
\boxed{P = \frac{1}{2}}
```
