# 计算矩阵幂零性

- Conversation ID: `690767eb-5ff8-8329-b499-cc498143085b`

## Question

两个n阶矩阵A,B满足 AB+A = BA+B。
计算 (A-B)^n=？

## Answer

设 $X=A-B$。由题意

```math
AB+A=BA+B\iff AB-BA=B-A\iff AB-BA=-X.
```

计算对易子得

```math
[A,X]=AX-XA= A(A-B)-(A-B)A= -AB+BA= X,
```

同理

```math
[B,X]=X.
```

于是对 $M:=A+B$ 有

```math
[M,X]=[A+B,X]=[A,X]+[B,X]=X+X=2X.
```

对任意实数 $t$，由指数映射与对易子关系（即 $e^{tM}Xe^{-tM}=e^{t\mathrm{ad}_M}(X)$）可得

```math
e^{tM}Xe^{-tM}=e^{2t}X.
```

两边为相似矩阵，谱集不变，因此矩阵 $X$ 的任一特征值 $\lambda$ 必满足
$\lambda=e^{2t}\lambda$ 对所有 $t$ 成立，故 $\lambda=0$。所以 $X$ 是幂零的（nilpotent），其指数不超过阶数 $n$。由 Cayley–Hamilton 定理或幂零矩阵的性质，

```math
X^n=(A-B)^n=0.
```

答案：$\boxed{(A-B)^n=0}$。
