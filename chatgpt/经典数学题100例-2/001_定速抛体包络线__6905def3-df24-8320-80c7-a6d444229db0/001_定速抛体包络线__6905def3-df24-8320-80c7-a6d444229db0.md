# 定速抛体包络线

- Conversation ID: `6905def3-df24-8320-80c7-a6d444229db0`

## Question

定速抛体的包络线是什么？
定速抛体：假设以固定的速度向不同方向抛出物体（简化为质点）。建立笛卡尔坐标系，抛出点在坐标原点。抛出的方向都在x轴的上半部分。

## Answer

好的，我们来仔细分析这个问题。你问的是 定速抛体的包络线，我们按数学物理严格推导。

#### 1️⃣ 问题建模

条件：

初速度大小固定 $v_0$。

发射角度 $\theta$ 可变，范围 $0 \leq \theta \leq \pi/2$。

重力加速度 $g$，沿 $y$ 方向向下。

发射点在原点 $(0,0)$。

抛体轨迹公式：

抛射角为 $\theta$ 时，质点运动的方程为：

```math
x(t) = v_0 \cos\theta \, t, \quad
y(t) = v_0 \sin\theta \, t - \frac{1}{2} g t^2
```

消去 $t$ 得到轨迹方程：

```math
y = x \tan\theta - \frac{g}{2 v_0^2 \cos^2\theta} x^2
```

这是标准的二次抛物线方程。

#### 2️⃣ 包络线的求法

给定一族曲线：

```math
y = f(x, \theta) = x \tan\theta - \frac{g}{2 v_0^2 \cos^2\theta} x^2
```

求包络线的方法是：对参数 $\theta$ 求偏导，并令偏导为零：

```math
\frac{\partial y}{\partial \theta} = 0
```

计算偏导：

```math
\frac{\partial y}{\partial \theta} = x \cdot \frac{1}{\cos^2 \theta} - \frac{g x^2}{2 v_0^2} \cdot \frac{2 \sin\theta}{\cos^3 \theta} = 0
```

整理：

```math
\frac{x}{\cos^2\theta} - \frac{g x^2 \sin\theta}{v_0^2 \cos^3\theta} = 0
```

```math
\frac{x}{\cos^2\theta} \left( 1 - \frac{g x \sin\theta}{v_0^2 \cos\theta} \right) = 0
```

显然 $x \neq 0$，所以：

```math
1 - \frac{g x \tan\theta}{v_0^2} = 0 \implies \tan\theta = \frac{v_0^2}{g x}
```

#### 3️⃣ 代回原方程

轨迹方程：

```math
y = x \tan\theta - \frac{g}{2 v_0^2 \cos^2\theta} x^2
```

使用恒等式：

```math
1 + \tan^2\theta = \frac{1}{\cos^2\theta} \implies \cos^2\theta = \frac{1}{1 + \tan^2\theta}
```

代入 $\tan\theta = \frac{v_0^2}{g x}$：

```math
\cos^2\theta = \frac{1}{1 + (v_0^2 / g x)^2} = \frac{g^2 x^2}{g^2 x^2 + v_0^4}
```

再代回 $y$：

```math
y = x \cdot \frac{v_0^2}{g x} - \frac{g x^2}{2 v_0^2} \cdot \frac{g^2 x^2 + v_0^4}{g^2 x^2} = \frac{v_0^2}{g} - \frac{g x^2}{2 v_0^2} \cdot \left(1 + \frac{v_0^4}{g^2 x^2}\right)
```

整理：

```math
y = \frac{v_0^2}{g} - \frac{g x^2}{2 v_0^2} - \frac{v_0^2}{2 g} = \frac{v_0^2}{2 g} - \frac{g}{2 v_0^2} x^2
```

#### ✅ 4️⃣ 包络线结论

包络线是一个开口向下的抛物线：

```math
\boxed{y = \frac{v_0^2}{2g} - \frac{g}{2 v_0^2} x^2}
```

顶点在原点正上方 $(0, v_0^2 / 2g)$

对称轴是 $x = 0$

这是所有以相同初速度抛出的抛体轨迹的上界。
