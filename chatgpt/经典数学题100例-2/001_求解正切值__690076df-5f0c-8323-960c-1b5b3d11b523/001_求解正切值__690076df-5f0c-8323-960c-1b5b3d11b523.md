# 求解正切值

- Conversation ID: `690076df-5f0c-8323-960c-1b5b3d11b523`

## Question

已知△ABC内角的正切值均为整数，求tanA,tanB,tanC

## Answer

先用正切的加法公式。设 $x=\tan A,\;y=\tan B,\;z=\tan C$。由 $A+B+C=\pi$ 得

```math
\tan(A+B)=\tan(\pi-C)=-\tan C=-z.
```

但

```math
\tan(A+B)=\frac{\tan A+\tan B}{1-\tan A\tan B}=\frac{x+y}{1-xy},
```

故

(1)

```math
\frac{x+y}{1-xy}=-z\quad\Longrightarrow\quad x+y+z=xyz.
```

现在求整数解。注意 $x,y,z$ 不能为 $0$（角不为 $0$ 或 $\pi$），且由三角形角度性质不可能出现无穷大（即不存在 $\pi/2$ 的角）。不妨令 $1\le x\le y\le z$（考虑绝对值及排列），由 (1) 可解出

```math
z=\frac{x+y}{xy-1}.
```

若 $x\ge2$，则 $xy-1\ge 3$，于是

```math
z=\frac{x+y}{xy-1}\le\frac{y+y}{3}=\frac{2y}{3}\lt{}y,
```

这与 $z\ge y$ 矛盾。因此必有 $x=1$。代入得

```math
z=\frac{1+y}{y-1}=1+\frac{2}{y-1},
```

所以 $y-1$ 为 $2$ 的因子，故 $y-1=1$ 或 $2$，即 $y=2$ 或 $3$。对应得到的整数三元组（按任意排列）是

```math
(1,2,3).
```

负数解（例如 $(-1,-2,-3)$）虽也满足方程 $x+y+z=xyz$，但在三角形中不可能出现三个角的正切值都为负（角和为 $\pi$），因此被排除。

结论：三角形 $ABC$ 中三角的正切值必是 $1,2,3$（顺序可以互换）。例如一种对应是

```math
\tan A=1,\quad \tan B=2,\quad \tan C=3,
```

对应角约为 $45^\circ,\;63.4349^\circ,\;71.5651^\circ$，和为 $180^\circ$。
