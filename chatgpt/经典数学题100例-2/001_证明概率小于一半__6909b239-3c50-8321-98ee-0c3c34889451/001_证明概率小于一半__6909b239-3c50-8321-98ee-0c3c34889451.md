# 证明概率小于一半

- Conversation ID: `6909b239-3c50-8321-98ee-0c3c34889451`

## Question

有相互独立的n个事件分别是A_{1},\dots,A_{n}。每个发生的概率都小于\frac{1}{2}。
证明：恰有一个发生的概率也小于\frac{1}{2}

## Answer

先说明结论的直观路线：把

```math
F(p_1,\dots,p_n)=\Pr(\text{恰有一个发生})
=\sum_{i=1}^n p_i\prod_{j\ne i}(1-p_j)
```

看作关于每个 $p_i=\Pr(A_i)$ 的函数（注意题中 $0\le p_i<\tfrac12$ 且事件相互独立）。证明的两步：

在约束 $0\le p_i<\tfrac12$ 下，函数 $F$ 的极大值出现在所有 $p_i$ 相等的点（也就是对称性 + 内部极值条件表明各分量相等）。

若所有 $p_i=p\in[0,\tfrac12)$，则

```math
F=n p(1-p)^{\,n-1}\le n\cdot\frac{1}{n}\Big(1-\frac{1}{n}\Big)^{n-1}=\Big(1-\frac{1}{n}\Big)^{n-1}\lt{}\tfrac12
```

（对 $n\ge2$ 上面的最大值发生在 $p=\frac{1}{n}$，值为 $(1-\frac{1}{n})^{\,n-1}$，当 $n=2$ 时该值为 $\tfrac12$ 的极限情况，但由于题中 $p<\tfrac12$ 是严格不等，实际取得的值严格小于 $\tfrac12$。对 $n=1$ 则显然 $F=p_1<\tfrac12$。）

下面给出稍详细的论证要点（省去非常冗长的初等变形）：

（A）内部临界点必然对称。
把 $q_i=1-p_i$。有

```math
F=\sum_{i=1}^n p_i\prod_{j\ne i}q_j
=\Big(\prod_{j=1}^n q_j\Big)\sum_{i=1}^n\frac{p_i}{q_i}.
```

对任一内部点（即所有 $0<p_i<\tfrac12$）求偏导并令偏导为 $0$，可以得到对任意 $k,\ell$：

```math
\frac{\partial F}{\partial p_k}=\frac{\partial F}{\partial p_\ell},
```

由此可推出 $p_k=p_\ell$。因此任一内部极值处所有分量相等。

（B）当所有 $p_i=p$ 时的最大值计算。
令 $p_i=p\in[0,\tfrac12)$，则

```math
F(n,p)=n p(1-p)^{n-1}.
```

把该函数看作 $p$ 的函数，求导得

```math
\frac{d}{dp}F(n,p)=n(1-p)^{n-2}(1-np),
```

所以在区间内的唯一驻点是 $p=\frac{1}{n}$（当 $n\ge2$ 时该点属于 $[0,\tfrac12]$），此处取得极大值，其值为

```math
F_{\max}=\Big(1-\frac{1}{n}\Big)^{\,n-1}.
```

当 $n=2$ 时该值为 $\tfrac12$，当 $n\ge3$ 时该值严格小于 $\tfrac12$。结合题中假设每个 $p_i<\tfrac12$（因此不会在 $n=2$ 时达到恰等于 $\tfrac12$ 的边界情况），得对于任意允许的 $(p_1,\dots,p_n)$，

```math
F(p_1,\dots,p_n)\lt{}\tfrac12.
```

综上：若 $A_1,\dots,A_n$ 相互独立且每个发生的概率都小于 $\tfrac12$，则“恰有一个发生”的概率严格小于 $\tfrac12$。 ∎
