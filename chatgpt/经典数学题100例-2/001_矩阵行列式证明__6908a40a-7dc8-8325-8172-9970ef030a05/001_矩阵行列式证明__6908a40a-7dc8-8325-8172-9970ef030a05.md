# 矩阵行列式证明

- Conversation ID: `6908a40a-7dc8-8325-8172-9970ef030a05`

## Question

实方阵A,B具有同样特征多项式，且有M使得AM=MB。证明|A-MX|=|B-XM|对于任何实方阵X都成立。

## Answer

下面给出简洁、标准的证明思路。

思路要点

用矩阵行列式恒等式（Sylvester 恒等式）$\det(I-UV)=\det(I-VU)$；

先把 $A,B$ 同时平移 $\lambda I$ 使其可逆，利用可逆时的分解与 Sylvester 恒等式得到等式；

观察两边关于 $\lambda$ 是多项式，在可逆的无限多个 $\lambda$ 上相等，由多项式恒等性在 $\lambda=0$ 处仍然相等，从而得结论。

详细证明

设对任意实数 $\lambda$ 定义

```math
A_\lambda:=A+\lambda I,\qquad B_\lambda:=B+\lambda I.
```

由于 $A$ 与 $B$ 有相同的特征多项式，故 $\det A_\lambda=\det B_\lambda$（作为 $\lambda$ 的多项式恒等）。并且由题设 $AM=MB$ 得

```math
A_\lambda M=(A+\lambda I)M=AM+\lambda M=MB+\lambda M=M(B+\lambda I)=M B_\lambda.
```

取任一使得 $A_\lambda$ 和 $B_\lambda$ 都可逆的 $\lambda$（这样的 $\lambda$ 有无限多个），由上式可左右乘逆得

```math
A_\lambda^{-1}M=M B_\lambda^{-1}.
```

现在对该 $\lambda$，分别展开两个行列式：

```math
\begin{aligned}
\det(A_\lambda-MX)
&=\det(A_\lambda)\,\det\big(I-A_\lambda^{-1}MX\big) \\
&=\det(B_\lambda)\,\det\big(I-A_\lambda^{-1}MX\big)
\end{aligned}
```

（用到了 $\det A_\lambda=\det B_\lambda$）。又由 $A_\lambda^{-1}M=M B_\lambda^{-1}$ 可得

```math
\det\big(I-A_\lambda^{-1}MX\big)=\det\big(I-MB_\lambda^{-1}X\big).
```

另一方面，

```math
\begin{aligned}
\det(B_\lambda-XM)
&=\det(B_\lambda)\,\det\big(I-B_\lambda^{-1}XM\big) \\
&=\det(B_\lambda)\,\det\big(I-MB_\lambda^{-1}X\big)
\end{aligned}
```

其中最后一步用到 Sylvester 恒等式 $\det(I-UV)=\det(I-VU)$（取 $U=B_\lambda^{-1}X,\ V=M$ 或等价变换）。

综上，对于所有使 $A_\lambda,B_\lambda$ 可逆的 $\lambda$ 有

```math
\det(A_\lambda-MX)=\det(B_\lambda-XM).
```

注意两边均为 $\lambda$ 的多项式，而且在无限多的 $\lambda$ 上相等，故作为多项式它们在所有 $\lambda$ 上恒等。于是取 $\lambda=0$ 得

```math
\boxed{\ \det(A-MX)=\det(B-XM)\ }
```

对任意实方阵 $X$ 都成立。证毕。
