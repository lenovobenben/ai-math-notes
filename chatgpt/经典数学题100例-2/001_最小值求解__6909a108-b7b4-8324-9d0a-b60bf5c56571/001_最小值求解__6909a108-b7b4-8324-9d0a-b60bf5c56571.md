# 最小值求解

- Conversation ID: `6909a108-b7b4-8324-9d0a-b60bf5c56571`

## Question

令a,b,c,d为正实数，且有ab^2c^4d^8=16。求a+b+c+d最小值。

## Answer

令 $a,b,c,d>0$ 且 $ab^2c^4d^8=16$。记约束的对数：

```math
\ln a+2\ln b+4\ln c+8\ln d=\ln 16=4\ln2.
```

用拉格朗日乘数法最小化 $f(a,b,c,d)=a+b+c+d$ （受上述约束）。设拉格朗日函数

```math
\mathcal{L}=a+b+c+d+\lambda(\ln a+2\ln b+4\ln c+8\ln d-4\ln2).
```

对各变量求偏导并令其为零：

```math
\frac{\partial\mathcal{L}}{\partial a}=1+\frac{\lambda}{a}=0\implies 1=\frac{\lambda}{a}\implies a=\lambda,
```

```math
\frac{\partial\mathcal{L}}{\partial b}=1+\frac{2\lambda}{b}=0\implies b=2\lambda,
```

```math
\frac{\partial\mathcal{L}}{\partial c}=1+\frac{4\lambda}{c}=0\implies c=4\lambda,
```

```math
\frac{\partial\mathcal{L}}{\partial d}=1+\frac{8\lambda}{d}=0\implies d=8\lambda.
```

所以 $a:b:c:d=1:2:4:8$。代入约束：

```math
\lambda^{1+2+4+8}\cdot 2^{2}\cdot4^{4}\cdot8^{8}
=\lambda^{15}\cdot 2^{2}\cdot(2^2)^4\cdot(2^3)^8
=\lambda^{15}\cdot 2^{2+8+24}=\lambda^{15}\cdot2^{34}=16=2^4.
```

得 $\lambda^{15}=2^{4-34}=2^{-30}$，所以 $\lambda=2^{-2}=\tfrac14$。

因此极小值处

```math
(a,b,c,d)=\Big(\tfrac14,\tfrac12,1,2\Big),
```

且

```math
a+b+c+d=\tfrac14+\tfrac12+1+2=\tfrac{15}{4}.
```

答：最小值为 $\boxed{\tfrac{15}{4}}$，当且仅当 $(a,b,c,d)=(\tfrac14,\tfrac12,1,2)$ 时取得。
