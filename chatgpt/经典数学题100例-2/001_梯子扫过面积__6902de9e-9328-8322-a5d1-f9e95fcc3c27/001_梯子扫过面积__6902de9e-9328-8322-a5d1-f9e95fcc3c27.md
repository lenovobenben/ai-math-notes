# 梯子扫过面积

- Conversation ID: `6902de9e-9328-8322-a5d1-f9e95fcc3c27`

## Question

在直角的墙角边有一个梯子（可理解为线段，长度为1）贴合在墙角的侧壁上。现在梯子滑落，最终贴合到地面上（线段原本的上端贴合到了墙角的顶点上）。求这个梯子扫过的面积。

## Answer

把墙面和地面看成直角坐标系的两条坐标轴（第一象限）。设梯子长为 $L=1$。当梯子两端分别贴在坐标轴上时，可用参数角 $\theta\in[0,\pi/2]$ 表示：两端坐标为

```math
(a,0)=(L\cos\theta,0),\quad (0,b)=(0,L\sin\theta).
```

这时梯子所在直线的方程是

```math
\frac{x}{L\cos\theta}+\frac{y}{L\sin\theta}=1.
```

把所有位置的这些线段的包络线求出来（即消去 $\theta$），对方程

```math
F(\theta,x,y)=\frac{x}{\cos\theta}+\frac{y}{\sin\theta}-L=0
```

同时令 $\partial F/\partial\theta=0$。解得

```math
x=L\cos^3\theta,\qquad y=L\sin^3\theta.
```

消去参数得到包络曲线的解析式（即一条四叶星形的 astroid）：

```math
x^{2/3}+y^{2/3}=L^{2/3}.
```

被扫过的区域就是第一象限内由坐标轴和该 astroid 围成的区域。已知参数化曲线
$x=L\cos^3 t,\; y=L\sin^3 t$（$t\in[0,2\pi]$）所围图形的总面积为 $\dfrac{3\pi L^2}{8}$，因此第一象限的面积为其四分之一：

```math
A=\frac{1}{4}\cdot\frac{3\pi L^2}{8}=\frac{3\pi L^2}{32}.
```

代入 $L=1$，得到

```math
\boxed{A=\frac{3\pi}{32}\approx 0.29452.}
```
