## Answer

### 二次互反律是什么？如何证明？

下面先给出精确表述、补充律（$(-1)$ 与 $2$ 的情况），然后给出Eisenstein 的格点计数证明（经典且相对易懂），并补充 Gauss 引理的说明与例子。

#### 1. 定义与定理（精确表述）

设 $p,q$ 是不同的奇素数。记勒让德符号为

```math
\left(\frac{a}{p}\right)=
\begin{cases}
0,& p\mid a,\\[4pt]
1,& a\ \text{为模}\ p\ \text{的二次剩余},\\[4pt]
-1,&\text{否则}.
\end{cases}
```

二次互反律（Quadratic reciprocity）：

```math
\boxed{\ \displaystyle
\left(\frac{p}{q}\right)\left(\frac{q}{p}\right)
=(-1)^{\frac{p-1}{2}\cdot\frac{q-1}{2}}.
\ }
```

也就是说，当 $p\equiv1\pmod4$ 或 $q\equiv1\pmod4$ 时符号为 $+1$，当 $p\equiv q\equiv3\pmod4$ 时符号为 $-1$。

补充律（常与互反律一起使用）：

对 $-1$：

```math
\left(\frac{-1}{p}\right)=(-1)^{\frac{p-1}{2}}=
\begin{cases}
1,& p\equiv1\pmod4,\\
-1,& p\equiv3\pmod4.
\end{cases}
```

对 $2$（第二补充律）：

```math
\left(\frac{2}{p}\right)=(-1)^{\frac{p^2-1}{8}}=
\begin{cases}
1,& p\equiv\pm1\pmod8,\\
-1,& p\equiv\pm3\pmod8.
\end{cases}
```

#### 2. Eisenstein（艾森斯坦）的格点计数证明（要点）

此证明用到对整数点网格的计数与取整函数的奇偶性。目标是把勒让德符号转化为某些取整和的奇偶性，从而比较 $\left(\frac{p}{q}\right)$ 与 $\left(\frac{q}{p}\right)$。

步骤：

对任意奇素数 $p$，把集合

```math
S_p=\{1,2,\dots,\tfrac{p-1}{2}\}.
```

考察所有点 $(x,y)$ ，其中 $x\in S_p,\ y\in S_q$，即矩形点格 $[1,(p-1)/2]\times[1,(q-1)/2]$。

画直线 $y=\frac{q}{p}x$。矩形内的格点被这条直线分为两部分：在直线下方的点和在直线上方的点。令

```math
A=\#\{(x,y)\in S_p\times S_q: y\lt{}\tfrac{q}{p}x\}.
```

由对称性或直接求和可得

```math
A=\sum_{x=1}^{(p-1)/2}\left\lfloor\frac{qx}{p}\right\rfloor.
```

交换角色（画直线 $x=\frac{p}{q}y$），得到

```math
B=\sum_{y=1}^{(q-1)/2}\left\lfloor\frac{py}{q}\right\rfloor.
```

注意到矩形中格点总数为 $\frac{p-1}{2}\cdot\frac{q-1}{2}$。直线不会通过任何格点（因为 $p,q$ 互素且系数是正有理数），所以每个格点要么在直线下方要么上方。因此

```math
A+B=\frac{p-1}{2}\cdot\frac{q-1}{2}.
```

所以

```math
A\equiv \frac{p-1}{2}\cdot\frac{q-1}{2}-B \pmod2,
```

因而 $A$ 与 $B$ 的奇偶性互相关联（但我们要具体与勒让德符号联系起来）。

现在引入 Gauss 引理（或直接证明）：对奇素 $p$ 与整数 $a$（且 $p\nmid a$），令集合

```math
T=\{a,2a,\dots,\tfrac{p-1}{2}a\}
```

在模 $p$ 下取规范代表为 $r_i\in\{1,2,\dots,p-1\}$。令 $n(a)$ 为这些代表中大于 $p/2$ 的个数。Gauss 引理断言

```math
\left(\frac{a}{p}\right)=(-1)^{\,n(a)}.
```

把 $a=q$ 带入并计算 $n(q)$ 与前面取整和的关系，可以得到

```math
n(q)=\sum_{x=1}^{(p-1)/2}\left\lfloor\frac{qx}{p}\right\rfloor = A.
```

同理 $n(p)=B$。

因此

```math
\left(\frac{q}{p}\right)\left(\frac{p}{q}\right)=(-1)^{n(q)+n(p)} = (-1)^{A+B}.
```

代入第 4 步 $A+B=\dfrac{p-1}{2}\cdot\dfrac{q-1}{2}$，得到

```math
\left(\frac{p}{q}\right)\left(\frac{q}{p}\right)=(-1)^{\frac{p-1}{2}\cdot\frac{q-1}{2}},
```

完成证明。

（关键点：把勒让德符号用 Gauss 引理转换为“>p/2 的倍数数量”，再用取整和表示该数量，最后用格点计数求和关系得到互反律。）

#### 3. Gauss 引理（补充说明）

Gauss 引理的证明通常如下：考虑集合 $\{a,2a,\dots,\frac{p-1}{2}a\}$ 在模 $p$ 下的规范代表 $r_1,\dots,r_{(p-1)/2}$，把这些代表写成

```math
r_i=\begin{cases}
s_i,& s_i\le \tfrac{p-1}{2},\\
p-s_i',& s_i'\gt{}\tfrac{p-1}{2},
\end{cases}
```

通过比较乘积 $\prod_{k=1}^{(p-1)/2} k a$ 与 $\prod r_i$（二者在模 $p$ 下相等，且前者与 $((p-1)/2)!$ 的乘积有关），可推出

```math
\left(\frac{a}{p}\right)=(-1)^{n(a)}.
```

详细证明可在标准代数数论教材中见到（例如 Ireland & Rosen，或经典的《初等数论》章节）。

#### 4. 举例说明

取 $p=3,\ q=11$。$\frac{p-1}{2}\cdot\frac{q-1}{2}=\;1\cdot5=5$，奇数，因此

```math
\left(\frac{3}{11}\right)\left(\frac{11}{3}\right)=-1.
```

计算 $\left(\frac{11}{3}\right)=\left(\frac{2}{3}\right)=-1$（因为 $2$ 在模 3 下不是二次剩余），因此 $\left(\frac{3}{11}\right)=1$。可以直接检验 $3$ 是否为模 $11$ 的二次剩余（$5^2=25\equiv3\pmod{11}$），确实是。

#### 5. 总结

二次互反律给出了两素数之间“互为二次剩余”的关系的简单判别式（只有当两者都 $\equiv3\pmod4$ 时符号为 $-1$）。

常与两条补充律一起使用，可以判定任意整数是否为给定素数的二次剩余。

常见证明有 Gauss 引理证明（多个变体）、Eisenstein 的格点证明、用二次高斯和的解析证明、以及代数数论中的代数方法（用二次域的规范与分裂行为）等。Eisenstein 的证明直观且经典，已在上面给出要点。
