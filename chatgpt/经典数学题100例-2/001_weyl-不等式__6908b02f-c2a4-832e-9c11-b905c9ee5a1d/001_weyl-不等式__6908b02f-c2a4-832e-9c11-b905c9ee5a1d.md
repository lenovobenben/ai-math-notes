# Weyl 不等式

- Conversation ID: `6908b02f-c2a4-832e-9c11-b905c9ee5a1d`

## Question

线性代数中的Weyl 不等式是什么？

## Answer

Weyl 不等式（Weyl’s inequality）是线性代数与矩阵谱理论中的一个重要结果，用于描述 两个 Hermitian（或实对称）矩阵相加时，其特征值之间的关系。它给出了矩阵特征值在相加运算下的“夹逼”关系，是研究矩阵扰动、谱估计、矩阵不等式的基础工具之一。

##### 🎯 一、背景

设：

$A, B \in \mathbb{C}^{n \times n}$ 为 Hermitian 矩阵（若为实矩阵则是对称矩阵）；

```math
C = A + B
```

记：

$\lambda_1(A) \ge \lambda_2(A) \ge \cdots \ge \lambda_n(A)$ 为 $A$ 的特征值（按降序排列）；

$\lambda_i(B), \lambda_i(C)$ 同理。

##### 🧮 二、Weyl 不等式的内容

对所有 $i, j$ 满足 $1 \le i, j \le n$ 且 $i+j-1 \le n$，有：

```math
\lambda_{i+j-1}(C) \le \lambda_i(A) + \lambda_j(B)
```

并且对所有 $i, j$ 满足 $i + j - n \ge 1$，有：

```math
\lambda_{i+j-n}(C) \ge \lambda_i(A) + \lambda_j(B)
```

##### 📘 三、几个重要特例

上界形式（最常用）
对每个 $k = 1, \dots, n$：

```math
\lambda_k(C) \le \lambda_i(A) + \lambda_{k-i+1}(B), \quad \forall\, i = 1, \dots, k
```

特别地：

```math
\lambda_k(C) \le \lambda_k(A) + \lambda_1(B)
```

下界形式
对每个 $k = 1, \dots, n$：

```math
\lambda_k(C) \ge \lambda_k(A) + \lambda_n(B)
```

极端情况（最大最小特征值）

```math
\lambda_{\max}(C) \le \lambda_{\max}(A) + \lambda_{\max}(B)
```

```math
\lambda_{\min}(C) \ge \lambda_{\min}(A) + \lambda_{\min}(B)
```

##### 💡 四、直观理解

Weyl 不等式可以理解为：

当你将两个对称矩阵 $A$ 和 $B$ 相加时，新矩阵的特征值不会“跑得太远”，它们被限制在由 $A$ 和 $B$ 的特征值所决定的范围内。

换句话说：

特征值的“上升”最多由 $B$ 的最大特征值决定；

特征值的“下降”最多由 $B$ 的最小特征值决定。

##### 🧠 五、在矩阵扰动理论中的应用

若 $B = E$ 是矩阵 $A$ 的一个小扰动，则：

```math
A' = A + E
```

则 Weyl 不等式告诉我们：

```math
|\lambda_i(A') - \lambda_i(A)| \le \|E\|_2
```

这就是 特征值扰动的上界（由 Weyl 不等式导出）。

##### 📖 六、与其他定理的关系

Weyl 不等式是 Courant–Fischer 极值原理 的推论。

它与 Lidskii 不等式、Ky Fan 不等式、Hoffman–Wielandt 不等式 等一同构成谱理论中的重要矩阵不等式体系。
