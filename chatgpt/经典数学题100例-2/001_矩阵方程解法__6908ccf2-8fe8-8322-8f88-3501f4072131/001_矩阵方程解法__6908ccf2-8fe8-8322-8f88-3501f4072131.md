# 矩阵方程解法

- Conversation ID: `6908ccf2-8fe8-8322-8f88-3501f4072131`

## Question

求所有满足下面等式的整数矩阵A。
2025 A ^ {2025} = A ^ {2024} + A ^ {2023} + \dots + A

## Answer

只存在平凡解 $A=0$。

证明（要点清楚、逐步说明）：

令 $n=2025$。方程是

```math
nA^{n}=A^{n-1}+A^{n-2}+\cdots+A.
```

把方程看作复数特征值问题。设 $\lambda$ 是 $A$ 的任一复特征值（取代特征向量不为零的情形），则 $\lambda$ 满足同样的代数方程

```math
n\lambda^{n}=\lambda^{n-1}+\lambda^{n-2}+\cdots+\lambda.
```

若 $\lambda=0$ 显然成立；下面说明不可能存在非零特征值。

把上式两边除以 $\lambda^{\,n-1}$（对 $\lambda\neq0$）得

```math
n\lambda=1+\lambda^{-1}+\lambda^{-2}+\cdots+\lambda^{-(n-1)}.
```

若 $|\lambda|\ge1$，则右边模的上界不超过 $n-1$，从而

```math
n|\lambda|\le n-1,
```

矛盾（因为 $n=2025$）。所以任一非零特征值都必须满足 $|\lambda|<1$。

另一方面，$A$ 是整数矩阵，其特征值都是代数整数（整系数首一多项式的根）。但代数整数若所有共轭（即代数共轭根）模都严格小于 $1$，唯有可能是 $0$（这是代数整数与根单位根/Kronecker 类型的常用结论：代数整数若模都 $\le1$ 则为根单位根，若严格小于 1 则只能是 0）。因此任何特征值 $\lambda$ 必为 $0$。

所以 $A$ 的所有特征值均为 $0$，即 $A$ 是幂零矩阵（nilpotent）：存在正整数 $s$ 使得 $A^{s}=0$。当 $s\le n$ 时，从原等式得

```math
0=nA^{n}=A+\cdots+A^{s-1},
```

即

```math
A\bigl(I+A+\cdots+A^{s-2}\bigr)=0.
```

但 $I+A+\cdots+A^{s-2}$ 在幂零矩阵情形下是可逆的（因为这是一个常数项为 $1$ 的多项式在幂零矩阵上的取值，其逆由有限项 Neumann 级数给出），因此上式蕴含 $A=0$。

（若 $s>n$，同样可由代数整数的论证推出所有特征值为 $0$，从而 $A$ 幂零，再重复上面步骤得到 $A=0$。）

综上，满足题目等式的整矩阵只有零矩阵 $A=0$。 ∎
