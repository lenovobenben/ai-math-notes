# 斐波那契相关的级数

- Conversation ID: `69046d1a-b97c-8324-b1c2-02fc5f2e5157`

## Question

```math
F_0 = 0, \quad F_1 = 1, \quad F_n = F_{n-1} + F_{n-2}, \, n \gt{} 1
```

请证明：

```math
\sum_{n=0}^\infty \frac{1}{F_{2^n}} = \frac{7 - \sqrt{5}}{2}
```

## Answer

把黄金分割数记为

```math
\varphi=\frac{1+\sqrt5}{2},\qquad \psi=\frac{1-\sqrt5}{2}.
```

注意常用恒等式 $\varphi\psi=-1,\ \psi=-1/\varphi,\ \varphi^2=\varphi+1$ 以及 Binet 公式

```math
F_m=\frac{\varphi^m-\psi^m}{\sqrt5}.
```

我们要计算

```math
S=\sum_{n=0}^\infty\frac{1}{F_{2^n}}.
```

把 $n=0$ 单独取出：$F_1=1$，所以第一项为 $1$。对 $n\ge1$（此时 $2^n$ 为偶数），有 $\psi^{2^n}=(-1/\varphi)^{2^n}=\varphi^{-2^n}$。因此对 $n\ge1$

```math
\frac{1}{F_{2^n}}=\frac{\sqrt5}{\varphi^{2^n}-\varphi^{-2^n}}
=\frac{\sqrt5\,\varphi^{-2^n}}{1-\varphi^{-2^{n+1}}}.
```

把分母用几何级数展开：

```math
\frac{\varphi^{-2^n}}{1-\varphi^{-2^{n+1}}}
=\sum_{k=0}^\infty \varphi^{-(2^n+k2^{n+1})}
=\sum_{j=0}^\infty \varphi^{-2^n(2j+1)}.
```

于是对 $n\ge1$ 的项之和为

```math
\sum_{n=1}^\infty\frac{1}{F_{2^n}}
=\sqrt5\sum_{n=1}^\infty\sum_{j=0}^\infty\varphi^{-2^n(2j+1)}.
```

观察每个正整数 $m\ge2$ 有且仅有一种写法 $m=2^n(2j+1)$ 且 $n\ge1$ 正好对应偶数 $m$。因此上式等于对所有偶数 $m\ge2$ 的求和：

```math
\sum_{n=1}^\infty\frac{1}{F_{2^n}}
=\sqrt5\sum_{k=1}^\infty\varphi^{-2k}.
```

于是整个和为

```math
S=1+\sqrt5\sum_{k=1}^\infty\varphi^{-2k}.
```

计算几何级数：

```math
\sum_{k=1}^\infty\varphi^{-2k}=\frac{\varphi^{-2}}{1-\varphi^{-2}}=\frac{1}{\varphi^2-1}.
```

利用 $\varphi^2=\varphi+1$ 得 $\varphi^2-1=\varphi$，所以

```math
S=1+\frac{\sqrt5}{\varphi}.
```

把 $\sqrt5$ 用 $\varphi$ 表示：由 $\varphi=(1+\sqrt5)/2$ 得 $\sqrt5=2\varphi-1$。代入并化简：

```math
S=1+\frac{2\varphi-1}{\varphi}=1+2-\frac{1}{\varphi}=3-\frac{1}{\varphi}.
```

又 $\dfrac{1}{\varphi}=\varphi-1$，所以

```math
S=3-(\varphi-1)=4-\varphi.
```

最后把 $\varphi$ 展开为 $(1+\sqrt5)/2$，得到

```math
S=4-\frac{1+\sqrt5}{2}=\frac{8-(1+\sqrt5)}{2}=\frac{7-\sqrt5}{2}.
```

因此

```math
\boxed{\displaystyle\sum_{n=0}^\infty\frac{1}{F_{2^n}}=\frac{7-\sqrt5}{2}}.
```
