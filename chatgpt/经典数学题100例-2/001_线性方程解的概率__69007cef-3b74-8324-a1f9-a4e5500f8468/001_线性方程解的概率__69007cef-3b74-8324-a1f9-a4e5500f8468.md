# 线性方程解的概率

- Conversation ID: `69007cef-3b74-8324-a1f9-a4e5500f8468`

## Question

有线性方程组Ax=b。 求其唯一解、无穷多解和无解的概率分别是多少？

## Answer

好——先给出结论（假设 $A$ 与 $b$ 的各分量都是从连续分布独立随机抽取的；这是常见的“随机矩阵/向量”概率模型）：

设 $A$ 为 $m\times n$ 矩阵，$x\in\mathbb R^n$，$b\in\mathbb R^m$。

若 $m<n$（未知数多于方程数）：几乎必然（概率 $1$）有无限多解；唯一解和无解的概率都是 $0$。

若 $m=n$（方程数等于未知数）：几乎必然（概率 $1$）有唯一解（因为方阵随机时满秩，$\mathrm{rank}A=n$），无解与无限解概率为 $0$。

若 $m>n$（方程数多于未知数）：几乎必然（概率 $1$）无解；唯一解与无限解的概率为 $0$。

下面给出直观理由及说明。

### 直观理由

关键是随机矩阵的秩与线性子空间的“测度”性质：

对于连续分布的随机矩阵，矩阵的秩等于 $\min(m,n)$ 的概率为 $1$（即“满秩”几乎必然）。

当 $\mathrm{rank}A=\min(m,n)$ 时：

若 $n>m$，则 $\mathrm{rank}A=m$，列空间为 $\mathbb R^m$，于是任意 $b\in\mathbb R^m$ 都在列空间内，方程必有解；因为 $n>m$ 又非满列可逆，解的维数为 $n-\mathrm{rank}A=n-m>0$，因此解集是无穷多解。

若 $n=m$，则 $\mathrm{rank}A=n$，矩阵可逆，方程对任意 $b$ 有且仅有一个解（唯一解）。

若 $m>n$，则 $\mathrm{rank}A=n$，列空间是 $n$ 维的子空间（严格小于 $\mathbb R^m$），对随机的 $b$ 来说，$b$ 落在该列空间的概率为 $0$，因此几乎必然无解（存在解的情形是 $b$ 恰巧落在那个子空间 —— 这是测度为 0 的事件）。

### 重要备注与例外

上述结论依赖于 “连续随机取数且独立” 的假设（例如每个分量服从正态或均匀分布等）。若 $A,b$ 来自离散分布（如取整数、小域上的数），这些“概率为0/1”的结论不再成立，具体概率需另行计算。

若 $b$ 是固定的（非随机），情况要看 $b$ 是否在 $A$ 的列空间：即使 $m>n$，也可能存在解（但对于随机 $A$ 与固定 $b$ 的联合分布，若 $A$ 随机且独立于固定 $b$，无解概率仍通常为 1 when $m>n$）。

也可从秩条件写出判定：

有解 ⇔ $\mathrm{rank}[A\mid b]=\mathrm{rank}A$；

唯一解 ⇔ 有解且 $\mathrm{rank}A=n$（即列秩等于未知数个数）；

无穷多解 ⇔ 有解且 $\mathrm{rank}A<n$。

## Question

这个结论是严格的吗（基于测度论）？

## Answer

非常好——你问到关键处了。
答案是：✅ 是严格的（在测度论意义下）。下面是严谨的说明。

#### 一、假设前提

我们在实数域 $\mathbb{R}$ 上讨论：

$A \in \mathbb{R}^{m\times n}$，其元素独立同分布于某个具有连续概率密度函数的分布（例如高斯、均匀分布等）。

$b \in \mathbb{R}^m$，独立于 $A$，也从一个连续分布抽取。

这意味着 $A$ 和 $b$ 的联合分布在 $\mathbb{R}^{mn+m}$ 上是绝对连续的（即对 Lebesgue 测度有密度）。

#### 二、关键的测度论事实

##### (1) 随机矩阵满秩几乎必然成立

集合

```math
S = \{ A \in \mathbb{R}^{m\times n} \mid \mathrm{rank}(A) \lt{} \min(m,n) \}
```

可以表示为某些多项式（所有 $\min(m,n)$ 阶子式的行列式）同时为 0 的集合。

这类集合是多项式方程组的解集，称为代数簇（algebraic variety），其在 $\mathbb{R}^{mn}$ 中的 Lebesgue 测度为 0。
因此：

```math
\mathbb{P}(\mathrm{rank}(A) = \min(m,n)) = 1.
```

##### (2) 随机 $b$ 落在给定列空间的概率为 0（若列空间是严格子空间）

若 $A$ 满秩，且 $m>n$，则列空间 $\mathcal{C}(A)\subset \mathbb{R}^m$ 是一个 $n$-维线性子空间。
对随机 $b\sim f_b$，其密度函数在整个 $\mathbb{R}^m$ 上绝对连续，因此：

```math
\mathbb{P}(b\in \mathcal{C}(A)) = 0,
```

因为 $\mathcal{C}(A)$ 的 Lebesgue 测度为 0。

##### (3) 综合上述两点

我们可以严格地写出：

```math
\begin{cases}
\mathbb{P}(\text{唯一解}) = 1 & \text{若 } m=n,\\[4pt]
\mathbb{P}(\text{无穷多解}) = 1 & \text{若 } m\lt{}n,\\[4pt]
\mathbb{P}(\text{无解}) = 1 & \text{若 } m\gt{}n.
\end{cases}
```

并且每种情形下其他两种情况的概率都为 0。

这就是一个严格的测度论意义下的结论（不是“近似成立”，而是“几乎处处成立”）。

#### 三、数学表达（更形式化）

设

```math
\Omega = \mathbb{R}^{mn+m}, \quad \mu = \text{Lebesgue 测度}, \quad \mathbb{P} \ll \mu
```

（即 $\mathbb{P}$ 对 Lebesgue 测度绝对连续）。定义事件：

```math
E_{\text{unique}} = \{ (A,b): \mathrm{rank}A = n = m \},\\
E_{\infty} = \{ (A,b): \mathrm{rank}A = m \lt{} n \},\\
E_{\text{none}} = \{ (A,b): \mathrm{rank}A = n \lt{} m, b \notin \text{Col}(A)\}.
```

则 $\mu(\Omega \setminus (E_{\text{unique}}\cup E_{\infty}\cup E_{\text{none}})) = 0$，从而：

```math
\mathbb{P}(E_{\text{unique}}) + \mathbb{P}(E_{\infty}) + \mathbb{P}(E_{\text{none}}) = 1.
```

✅ 结论：

对任何元素独立且连续分布的随机矩阵 $A$ 和随机向量 $b$，
线性方程组 $A x = b$ 的三种可能性概率为：

```math
\boxed{
\begin{aligned}
&P(\text{唯一解}) = 1 &&\text{当 } m=n,\\
&P(\text{无穷多解}) = 1 &&\text{当 } m\lt{}n,\\
&P(\text{无解}) = 1 &&\text{当 } m\gt{}n.
\end{aligned}
}
```

这是在测度论意义下严格成立的“几乎必然”结论。
