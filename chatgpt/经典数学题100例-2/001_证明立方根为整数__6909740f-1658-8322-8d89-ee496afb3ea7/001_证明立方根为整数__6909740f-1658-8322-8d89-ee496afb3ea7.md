# 证明立方根为整数

- Conversation ID: `6909740f-1658-8322-8d89-ee496afb3ea7`

## Question

设a,b,c \in \mathbb{Z} ，且均不为零。如果\frac{a}{b} + \frac{b}{c} + \frac{c}{a} \in \mathbb{Z}。
请证明：\sqrt [3]{abc} \in \mathbb{Z}

## Answer

令 $a,b,c\in\mathbb Z$ 且均非零，设

```math
S=\frac{a}{b}+\frac{b}{c}+\frac{c}{a}\in\mathbb Z.
```

将分母通为 $abc$ 得

```math
S=\frac{a^2c+ab^2+bc^2}{abc}\in\mathbb Z.
```

对任意素数 $p$，记 $\nu_p(x)$ 为 $x$ 中 $p$ 的幂次（$p$-adic valuation）。设

```math
\alpha=\nu_p(a),\qquad \beta=\nu_p(b),\qquad \gamma=\nu_p(c).
```

则分子三项的 $p$-次幂分别为

```math
\nu_p(a^2c)=2\alpha+\gamma,\qquad \nu_p(ab^2)=\alpha+2\beta,\qquad \nu_p(bc^2)=\beta+2\gamma,
```

分母的 $p$-次幂为 $\alpha+\beta+\gamma$。由于分数是整数，分子对 $p$ 的最低幂次至少要不小于分母的幂次，即

```math
\min(2\alpha+\gamma,\ \alpha+2\beta,\ \beta+2\gamma)\ge \alpha+\beta+\gamma.
```

两边同时减去 $\alpha+\beta+\gamma$，得到

```math
\min(\alpha-\beta,\ \beta-\gamma,\ \gamma-\alpha)\ge 0.
```

因此三者 $\alpha-\beta,\ \beta-\gamma,\ \gamma-\alpha$ 都不小于 $0$，于是必有

```math
\alpha=\beta=\gamma.
```

对每个素数 $p$ 都成立，说明对于任意 $p$，在 $abc$ 中 $p$ 的幂次为

```math
\nu_p(abc)=\alpha+\beta+\gamma=3\alpha,
```

是 $3$ 的倍数。因此 $abc$ 在素因子分解中每个素因子的幂次都是 $3$ 的倍数，即 $abc$ 是一个完全立方数。于是

```math
\sqrt[3]{abc}\in\mathbb Z.
```

（若 $abc<0$，也同样是一个负的完全立方数，其立方根为整数。） ∎
