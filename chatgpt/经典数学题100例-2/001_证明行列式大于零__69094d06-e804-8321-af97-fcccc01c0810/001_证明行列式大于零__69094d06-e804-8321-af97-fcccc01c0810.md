# 证明行列式大于零

- Conversation ID: `69094d06-e804-8321-af97-fcccc01c0810`

## Question

如果实矩阵A满足A^7 + A^5 + A^3 + A = I。请证明：\det A > 0

## Answer

令多项式 $f(x)=x^7+x^5+x^3+x-1$。由题意有

```math
f(A)=A^7+A^5+A^3+A-I=0,
```

因此 $A$ 的任一特征值 $\lambda$ 都满足标量方程

```math
f(\lambda)=\lambda^7+\lambda^5+\lambda^3+\lambda-1=0.
```

现在考察实数 $x\le0$ 时 $f(x)$ 的符号：当 $x\le0$ 时，$x^7,x^5,x^3,x$ 全为不大于 $0$ 的数，因此

```math
f(x)=x^7+x^5+x^3+x-1\le -1\lt{}0.
```

所以 $f(x)=0$ 不可能有非正实根。由此得出：任何实特征值必须严格为正；并且 $0$ 也不是特征值（因为 $f(0)=-1\neq0$）。

对于非实特征值，因 $A$ 为实矩阵，复特征值成共轭对出现，若 $\mu$ 是非实特征值，则 $\overline\mu$ 也是特征值。每对共轭特征值的乘积为 $|\mu|^2>0$。

因此行列式（为所有特征值的乘积）等于若干正实特征值的乘积与若干对共轭复特征值乘积（均为正实数）的乘积，故

```math
\det A\gt{}0.
```

此外，由于没有特征值为 $0$，矩阵 $A$ 可逆。证毕。
