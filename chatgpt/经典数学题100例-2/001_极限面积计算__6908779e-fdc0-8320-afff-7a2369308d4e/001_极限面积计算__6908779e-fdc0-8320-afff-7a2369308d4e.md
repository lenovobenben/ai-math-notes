# 极限面积计算

- Conversation ID: `6908779e-fdc0-8320-afff-7a2369308d4e`

## Question

令A_{n}为下面区域的面积x^{2n}+y^{2n} \leqslant 1。求极限\lim_{n \to \infty } A_{n}

## Answer

答案是 $\;4.$

证明（简洁）：

记

```math
E_n=\{(x,y)\in\mathbb R^2:\;|x|^{2n}+|y|^{2n}\le1\}.
```

当 $n\to\infty$ 时，对任意点 $(x,y)$ 有

```math
\mathbf 1_{E_n}(x,y)\to \mathbf 1_{[-1,1]^2}(x,y),
```

原因是：

若 $\max\{|x|,|y|\}<1$，则 $|x|^{2n}+|y|^{2n}\to0$，故对充分大的 $n$ 有 $(x,y)\in E_n$。

若 $\max\{|x|,|y|\}>1$，则某一坐标的 $2n$ 次幂发散，故对所有 $n$ 都有 $(x,y)\notin E_n$。

若 $\max\{|x|,|y|\}=1$ 且另一坐标 $\le1$，则为极限集合的边界点，指示函数也收敛到方形的指示函数。

由有界收敛（或受控收敛）定理，可把指示函数的逐点极限交换积分（面积），因此

```math
\lim_{n\to\infty} A_n=\lim_{n\to\infty}\iint \mathbf 1_{E_n} \,dxdy=\iint \mathbf 1_{[-1,1]^2}\,dxdy=\text{area}([-1,1]^2)=2\times2=4.
```

所以 $\displaystyle\lim_{n\to\infty}A_n=4.$
