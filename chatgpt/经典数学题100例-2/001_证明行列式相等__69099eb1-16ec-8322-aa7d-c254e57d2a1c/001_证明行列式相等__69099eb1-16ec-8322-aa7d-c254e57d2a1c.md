# 证明行列式相等

- Conversation ID: `69099eb1-16ec-8322-aa7d-c254e57d2a1c`

## Question

对于一个实方阵A满足A^{-1}+B^{-1}=(A+B)^{-1}。证明：|A|=|B|。

## Answer

设 $X=AB^{-1}$. 由题设

```math
A^{-1}+B^{-1}=(A+B)^{-1},
```

两边左乘 $A$，右乘 $B$，得

```math
A(A^{-1}+B^{-1})B=A(A+B)^{-1}B.
```

左边化简为 $A A^{-1}B + A B^{-1}B = B + A = A+B$，于是

```math
A+B = A(A+B)^{-1}B.
```

两边右乘 $B^{-1}$ 得

```math
(A+B)B^{-1} = A(A+B)^{-1}.
```

注意 $(A+B)B^{-1}=AB^{-1}+I=X+I$，于是代回原始等式的变形（也可直接从前一等式步骤得到）可得

```math
X(X+I)=-I,
```

即

```math
X^2+X+I=0.
```

由此可知 $X$ 的任一特征值 $\lambda$ 满足 $\lambda^2+\lambda+1=0$。解此方程得 $\lambda$ 是 $3$ 次单位根且 $\lambda\neq1$，因此 $\lambda^3=1$。于是所有特征值的乘积（即 $\det X$）满足

```math
(\det X)^3=1.
```

另一方面 $A,B$ 为实矩阵且可逆，所以 $\det X=\det A/\det B$ 是实数。实数中满足 $t^3=1$ 的唯一解为 $t=1$。因此 $\det X=1$，即

```math
\frac{\det A}{\det B}=1,
```

从而 $\boxed{\det A=\det B}$。证毕。
