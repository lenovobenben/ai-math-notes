# 不等式证明

- Conversation ID: `6908c927-e09c-8324-8560-897eabc0f336`

## Question

已知 $0 < x \leq \dfrac{1}{2}$，证明

```math
(1-x)\ln(1-x) \geq x\ln x.
```

## Answer

令

```math
f(x)=(1-x)\ln(1-x)-x\ln x,\qquad 0\lt{}x\le\frac12.
```

则

```math
f'(x)=-\ln(1-x)-1-(\ln x+1)=-\big(\ln x+\ln(1-x)+2\big),
```

再求二阶导得

```math
f''(x)=\frac{1}{1-x}-\frac{1}{x}=\frac{2x-1}{x(1-x)}\le0\quad\text{（当 }0\lt{}x\le\tfrac12\text{ 时）}.
```

因此在区间 $(0,\tfrac12]$ 上 $f$ 为凹函数。又

```math
\lim_{x\to0^+}f(x)=\lim_{x\to0^+}\big((1-x)\ln(1-x)-x\ln x\big)=0,
```

且 $f\big(\tfrac12\big)=(\tfrac12)\ln(\tfrac12)-(\tfrac12)\ln(\tfrac12)=0$。凹函数在端点连线之上，且两端点处值均为 $0$，故对所有 $0<x\le\tfrac12$ 有 $f(x)\ge0$。即

```math
(1-x)\ln(1-x)\ge x\ln x.
```

证毕。
