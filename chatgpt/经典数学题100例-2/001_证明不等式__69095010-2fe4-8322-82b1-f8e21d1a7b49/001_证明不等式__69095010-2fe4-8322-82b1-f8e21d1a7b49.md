# 证明不等式

- Conversation ID: `69095010-2fe4-8322-82b1-f8e21d1a7b49`

## Question

三维空间的凸多面体的体积为V，向三个垂直的平面投影以后的面积分别是S_{1},S_{2},S_{3}。请证明V^2 \le  S_{1}S_{2}S_{3}

## Answer

把问题放在一般的可测集合的情形下来证明更自然：对任意三维可测集合 $E\subset\mathbb R^3$（凸多面体是特殊情形）记体积 $|E|=V$，并把在三对互相垂直坐标平面上的投影面积记为

```math
S_{1}=|\pi_{yz}(E)|,\qquad S_{2}=|\pi_{xz}(E)|,\qquad S_{3}=|\pi_{xy}(E)|.
```

我们要证明

```math
|E|^{2}\le S_{1}S_{2}S_{3}.
```

下面给出基于Fubini 与两次Cauchy–Schwarz的不等式（即Loomis–Whitney 不等式在 $n=3$ 的专用形式）的证明。

记指示函数 $\mathbf1_E(x,y,z)$. 定义三个“断面长度/面积函数」

```math
f(x,y)=\int_{\mathbb R}\mathbf1_E(x,y,z)\,dz,\qquad
g(x,z)=\int_{\mathbb R}\mathbf1_E(x,y,z)\,dy,\qquad
h(y,z)=\int_{\mathbb R}\mathbf1_E(x,y,z)\,dx.
```

显然

```math
|E|=\iiint \mathbf1_E\,dx\,dy\,dz=\iint f(x,y)\,dx\,dy=\iint g(x,z)\,dx\,dz=\iint h(y,z)\,dy\,dz.
```

并且 $\mathrm{supp}f\subset\pi_{xy}(E)$，因此 $|\mathrm{supp}f|=S_{3}$（类似地对 $g,h$ 有 $|\mathrm{supp}g|=S_{2}$, $|\mathrm{supp}h|=S_{1}$）。

先对 $f$ 用一次Cauchy–Schwarz：

```math
|E|^{2}=\Big(\iint f(x,y)\,dx\,dy\Big)^{2}
\le\Big(\iint f(x,y)^{2}\,dx\,dy\Big)\Big(\iint 1_{\mathrm{supp}f}(x,y)^{2}\,dx\,dy\Big)
=\Big(\iint f^{2}\Big)S_{3}.
```

现在展开 $\iint f^{2}$：

```math
\iint f(x,y)^{2}\,dx\,dy
=\iint\Big(\int \mathbf1_E(x,y,z)\,dz\Big)\Big(\int \mathbf1_E(x,y,z')\,dz'\Big)\,dx\,dy
=\iiiint \mathbf1_E(x,y,z)\mathbf1_E(x,y,z')\,dx\,dy\,dz\,dz'.
```

对上述四重积分按变量重排并对 $y$ 再用一次Cauchy–Schwarz（或等价地按对称性对其它变量使用同样的步骤），可得

```math
\iint f^{2}\le S_{1}S_{2}.
```

把这两步合在一起得到

```math
|E|^{2}\le (\iint f^{2})S_{3}\le S_{1}S_{2}S_{3},
```

即

```math
V^{2}=|E|^{2}\le S_{1}S_{2}S_{3}.
```

这就是所需的不等式。对于凸多面体，上述可测性和Fubini 条件都满足，因此定理成立。该结论正是著名的Loomis–Whitney 不等式在三维的特例。若需要，我可以把上面“对 $y$ 的第二次Cauchy–Schwarz”的具体代数细节写得更展开。
