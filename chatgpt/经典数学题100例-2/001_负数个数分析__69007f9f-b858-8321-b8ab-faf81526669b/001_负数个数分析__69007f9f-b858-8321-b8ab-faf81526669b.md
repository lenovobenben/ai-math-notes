# 负数个数分析

- Conversation ID: `69007f9f-b858-8321-b8ab-faf81526669b`

## Question

已知a+b=1,c+d=1,ac+bd>1。问a,b,c,d至少有几个是负数？

## Answer

我们来仔细分析这个题目：

##### 已知条件：

```math
\begin{cases}
a + b = 1 \\
c + d = 1 \\
ac + bd \gt{} 1
\end{cases}
```

求：$a,b,c,d$ 至少有几个是负数？

##### 第一步：用 $b$ 和 $d$ 消元

由 $a + b = 1 \Rightarrow b = 1 - a$
由 $c + d = 1 \Rightarrow d = 1 - c$

代入 $ac + bd > 1$：

```math
ac + (1 - a)(1 - c) \gt{} 1
```

##### 第二步：化简

展开右边：

```math
ac + (1 - a - c + ac) \gt{} 1
```

```math
2ac - a - c + 1 \gt{} 1
```

```math
2ac - a - c \gt{} 0
```

##### 第三步：化为乘积形式

我们试着对 $2ac - a - c$ 进行整理：

```math
2ac - a - c = (2a - 1)(c - \tfrac{1}{2})
```

验证一下：

```math
(2a - 1)(c - \tfrac{1}{2}) = 2ac - a - c + \tfrac{1}{2}
```

差了一个 $\tfrac{1}{2}$，所以：

```math
2ac - a - c = (2a - 1)(c - \tfrac{1}{2}) - \tfrac{1}{2}
```

于是条件 $2ac - a - c > 0$ 等价于：

```math
(2a - 1)(c - \tfrac{1}{2}) \gt{} \tfrac{1}{2}
```

##### 第四步：分析符号关系

这个不等式告诉我们：
要使乘积 $(2a - 1)(c - 1/2)$ 大于 1/2，则：

两个因子必须同号；

并且绝对值都较大（至少有一个的绝对值 > 1）。

即：

```math
(2a - 1) \text{ 与 } (c - \tfrac{1}{2}) \text{ 同号且数值都较大}
```

##### 第五步：考虑可能的取值区间

###### 情况 1：都为正

若 $2a - 1 > 0 \Rightarrow a > 1/2$，
且 $c - 1/2 > 0 \Rightarrow c > 1/2$。

则两者都较大时，容易使乘积大于 1/2。
例如：$a = 1, c = 1$，则：

```math
ac + bd = 1\cdot1 + 0\cdot0 = 1
```

不行，还等于 1。

再取稍大一点的差异？其实不行，因为 $a,b,c,d$ 都线性相关。

我们换代入原式验证。

##### 第六步：代入数值试探

###### 例 1：

$a = 2, b = -1$（仍满足 $a+b=1$）
$c = 2, d = -1$（仍满足 $c+d=1$）

则：

```math
ac + bd = 2\times2 + (-1)\times(-1) = 4 + 1 = 5 \gt{} 1
```

✅ 满足条件。

此时负数有：$b, d$ 两个。

###### 例 2：

试试让某个只有一个负数：

```math
a = 2, b = -1, c = 1, d = 0
```

则：

```math
ac + bd = 2\times1 + (-1)\times0 = 2 \gt{} 1
```

仍满足，负数有 1 个（$b$）。

但注意！此时 $c+d=1$ 仍成立（1+0=1），没问题。

说明其实只要有一个负数也可以满足。

再试全为正数行不行：

###### 例 3：

```math
a = 0.8, b = 0.2, c = 0.8, d = 0.2
```

```math
ac + bd = 0.64 + 0.04 = 0.68 \lt{} 1
```

不满足。

所以必须让数的差距更大，而差距大往往意味着有负数。

##### 第七步：是否可能 0 个负数？

如果 $a,b,c,d > 0$，且 $a+b=c+d=1$，则每个都在 $(0,1)$ 内。
在此范围内，

```math
ac + bd \lt{} a + b = 1
```

因为 $ac + bd = a(c-d) + d(a+b) = a(c-d) + d$，
可以证明当 $0<a,b,c,d<1$ 时 $ac+bd < 1$。

所以全为正数不可能满足 $ac+bd>1$。

##### 第八步：最小负数个数

我们已找到例子 $a=2, b=-1, c=1, d=0$ 只有 1 个负数，并且满足条件。

所以：

✅ 至少 1 个是负数。

✅ 最终答案：至少有 1 个负数。
