# 证明数列收敛

- Conversation ID: `69076afe-9264-8331-9255-99103c695534`

## Question

设 $0 \le a_1 < 1$，且数列 $\{a_n\}$ 由递推关系

```math
a_{n+1} = a_n + \frac{a_n^2}{n^2}, \quad n \ge 1
```

定义。请证明 $\{a_n\}$ 收敛

## Answer

把题目翻译成要证明的命题并给出证明要点。

命题。 设 $0\le a_1<1$，且

```math
a_{n+1}=a_n+\frac{a_n^2}{n^2},\quad n\ge1.
```

则数列 $\{a_n\}$ 收敛。

证明。

（单调性）由于 $a_n\ge0$，得

```math
a_{n+1}-a_n=\frac{a_n^2}{n^2}\ge0.
```

所以 $\{a_n\}$ 是单调非减的。

（有界性）下面证明 $\{a_n\}$ 有上界，从而由单调有界性推出收敛。

对任意正整数 $k$，利用不等式 $1+t\le e^{t}$（对所有实 $t$ 成立）有

```math
a_{k+1}=a_k\bigl(1+\frac{a_k}{k^2}\bigr)\le a_k\exp\!\Big(\frac{a_k}{k^2}\Big).
```

对从 $1$ 到 $n-1$ 的乘积（递归展开）取对数，得到

```math
\ln a_n \le \ln a_1+\sum_{k=1}^{n-1}\frac{a_k}{k^2}.
```

设 $M_n=\max_{1\le k\le n}a_k$（注意 $M_n$ 随 $n$ 非减）。由上式并利用 $a_k\le M_n$（对所有 $k\le n$）得

```math
\ln a_n \le \ln a_1+M_n\sum_{k=1}^{n-1}\frac{1}{k^2}\le \ln a_1+M_n\sum_{k=1}^{\infty}\frac{1}{k^2}.
```

记常数 $C=\sum_{k=1}^{\infty}\frac{1}{k^2}=\frac{\pi^2}{6}$。则对任意 $n$ 有

```math
\ln M_n \le \ln a_1 + C M_n,
```

即

```math
\underbrace{\bigl(\ln x - Cx\bigr)}_{=:g(x)}\Big|_{x=M_n}\le \ln a_1.
```

函数 $g(x)=\ln x - Cx$ 在 $x>0$ 上连续，且当 $x\to0^+$ 或 $x\to\infty$ 时都趋于 $-\infty$，因此集合 $\{x>0:\; g(x)\le \ln a_1\}$ 是一个有限区间 $(0,R]$。由不等式可得 $M_n\le R$ 对所有 $n$ 成立，即 $\{a_n\}$ 有统一上界 $R$。

（若 $a_1=0$，递推式立即给出全体项都为 $0$，也显然收敛；故上面只需考虑 $a_1>0$ 的情况。）

（结论）由第 1 步和第 2 步，$\{a_n\}$ 为单调非减且有上界，故依单调有界原理必收敛。□

补充说明：证明的关键是用不等式 $1+t\le e^t$ 将乘积控制为对数形式，从而把对 $a_n$ 的上界问题转化为不等式 $\ln x -Cx\le$ 常数，后者天然限制了 $x$ 的大小，得到统一上界。
