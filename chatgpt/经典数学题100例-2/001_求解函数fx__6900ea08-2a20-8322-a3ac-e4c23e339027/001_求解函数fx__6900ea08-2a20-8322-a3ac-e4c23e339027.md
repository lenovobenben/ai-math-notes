# 求解函数f(x)

- Conversation ID: `6900ea08-2a20-8322-a3ac-e4c23e339027`

## Question

设f:R->R是连续双射。有不动点。满足对任意的x\inR都成立 f(2x-f(x))=x 。求f(x)

## Answer

我们仔细分析这个题目：

题目条件：

$f: \mathbb{R} \to \mathbb{R}$ 连续且双射（即单射且满射）。

$f$ 有不动点 $a$，即 $f(a) = a$。

对任意 $x \in \mathbb{R}$ 有

```math
f(2x - f(x)) = x.
```

我们要求 $f(x)$。

##### 第一步：做代换

令

```math
y = f(x).
```

那么方程变为：

```math
f(2x - y) = x.
```

因为 $y = f(x)$，所以

```math
f(2x - f(x)) = x
```

保持原式。

##### 第二步：尝试线性解

连续且双射的函数比较特殊，很多这类方程的解都是线性函数。设

```math
f(x) = \alpha x + \beta.
```

代入方程：

```math
f(2x - f(x)) = f(2x - (\alpha x + \beta)) = f((2 - \alpha)x - \beta) = \alpha((2-\alpha)x - \beta) + \beta = \alpha(2-\alpha)x - \alpha \beta + \beta
```

要求等于 $x$，所以：

```math
\alpha(2-\alpha)x - \alpha \beta + \beta = x \quad \forall x.
```

##### 第三步：系数比较

$x$ 的系数：

```math
\alpha(2-\alpha) = 1
```

常数项：

```math
-\alpha \beta + \beta = \beta(1-\alpha) = 0
```

##### 第四步：解方程

对 $\alpha$：

```math
\alpha(2-\alpha) = 2\alpha - \alpha^2 = 1 \implies \alpha^2 - 2\alpha + 1 = 0 \implies (\alpha-1)^2 = 0 \implies \alpha = 1
```

对 $\beta$：

```math
\beta(1-\alpha) = \beta(0) = 0 \implies \beta \text{ 可以任意实数。}
```

##### 第五步：考虑不动点

我们知道 $f$ 有不动点 $a$：

```math
f(a) = a \implies \alpha a + \beta = a \implies a + \beta = a \implies \beta = 0
```

##### 第六步：结论

因此

```math
\boxed{f(x) = x}.
```

✅ 检查：

```math
f(2x - f(x)) = f(2x - x) = f(x) = x,
```

完美符合条件。

答案：

```math
\displaystyle f(x) = x.
```

## Question

这是唯一解吗？

## Answer

让我们仔细验证“唯一性”。

##### 已知条件回顾

$f: \mathbb{R} \to \mathbb{R}$ 连续且双射（单射 + 满射）。

存在不动点 $a$（即 $f(a) = a$）。

对任意 $x \in \mathbb{R}$：

```math
f(2x - f(x)) = x
```

##### 第一步：考虑单射和连续性

已知 $f$ 是连续的双射 $\mathbb{R} \to \mathbb{R}$，那么 $f$ 是严格单调的。

连续 + 单射 \to 单调（严格递增或严格递减）。

##### 第二步：分析单调性

###### 情况 1：严格递增

假设 $f$ 是严格递增。

不动点 $a$ 存在 \to 这条直线与 $y=x$ 相交一次。

我们尝试线性猜测：$f(x) = x$ 已经满足方程。

###### 情况 2：严格递减

如果 $f$ 严格递减，设 $f(a) = a$。

严格递减 + 不动点 \to 对任意 $x > a$，有 $f(x) < x$，对 $x < a$，有 $f(x) > x$。

检查方程：

```math
f(2x - f(x)) = x
```

如果 $f$ 严格递减，则 $2x - f(x) > x$ 或 < x 会导致 $f(2x-f(x))$ 与单调性冲突。

更具体地，如果 $f$ 严格递减：

左边 $f(2x - f(x))$ 随 $x$ 变化严格递减

右边 $x$ 随 $x$ 增加严格递增

不可能恒等于 $x$

所以严格递减不可能。

##### 第三步：连续 + 严格递增 + 不动点

严格递增 + 不动点 + 方程 $f(2x-f(x)) = x$ \to 线性函数 $f(x) = x$ 唯一满足。

我们之前用线性猜测得到了唯一解 $f(x) = x$。

非线性严格递增连续函数若满足 $f(2x-f(x)) = x$ 将会导致二次形式或固定点冲突。

详细推导可以用 $g(x) = f(x)-x$：

```math
f(2x-f(x)) - x = 0 \implies g(x) = f(x)-x
```

方程变为：

```math
f(2x - f(x)) - x = g(2x - f(x)) - (2x - f(x) - x) = g(2x - f(x)) + g(x) = 0
```

即：

```math
g(2x - f(x)) = -g(x)
```

$g$ 连续，只有线性解 $g(x)=0$（即 $f(x)=x$）才不会产生矛盾。

##### ✅ 结论

结合连续性、双射（严格单调性）和存在不动点，唯一解是：

```math
\boxed{f(x) = x}.
```
