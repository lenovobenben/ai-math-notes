# AI Math Notes

这是一个用于收集、整理和展示数学题问答的仓库，当前内容主要来自 ChatGPT，后续也可能逐步扩展到其他大模型平台，例如 Gemini、Claude 等。

本仓库的目标不是简单堆放原始对话，而是尽量以更清晰、更易读的形式进行整理与呈现，使数学题目、图形、推导和解答过程能够更方便地查阅、分享与长期保存。

## 内容特点

- 以数学题问答为主，强调可读性与浏览体验。
- 题目中的图片能够正常展示，便于还原原始题意。
- 内容由开源工具 [chatgpt-math-exporter](https://github.com/lenovobenben/chatgpt-math-exporter) 导出并整理。

## 题目难度

题目难度覆盖范围较广，包括但不限于：

- 中学数学竞赛题
- 大学数学系本科层次题目
- 大学数学系研究生层次题目

其中不少题目表面上相对直观、易于理解，但真正完成求解、证明或构造往往并不容易，具有一定思维深度和训练价值。

## 目录组织

仓库内容按模型平台分层组织。

对于具有“项目”层级的平台，例如 ChatGPT，建议采用：

```text
chatgpt/项目/题目
```

对于没有“项目”层级的平台，例如部分 Gemini 内容，则可直接采用：

```text
gemini/题目
```

这样可以在保持整体结构清晰的同时，兼容不同模型平台的内容组织方式。
